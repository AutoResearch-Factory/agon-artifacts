# $\exists\mathbb{R}$-Completeness of Tensor Degeneracy and a Derandomization Barrier for Hyperdeterminants (2604.17061)

- **Year**: 2026.04
- **Authors**: Angshul Majumdar (tex 未标注机构)
- **Venue**: arXiv preprint
- **Keywords**: $\exists\mathbb{R}$-completeness, tensor degeneracy, hyperdeterminant, boundary format, projective bilinear feasibility, singular matrix-pencil feasibility, polynomial identity testing, derandomization
- **One-liner**: 证明实 3-张量退化 $\exists\mathbb{R}$-complete，并把超行列式硬度转移卡点定位为结构化 PIT。

## 精华提炼

### 管线/方法

1. 论文研究的对象不是 tensor rank、best rank-one approximation 或 singular value，而是张量的 intrinsic singularity：degeneracy。
2. 对 3-tensor \(T\in\mathbb{Q}^{n\times n\times m}\)，degeneracy 定义为存在非零 \(x,y,z\)，使三个 modewise contractions 同时为零。
3. 具体条件是 \(T(x,y,\cdot)=0\)、\(T(x,\cdot,z)=0\)、\(T(\cdot,y,z)=0\)。
4. 论文的主证明管线是：homogeneous quadratic feasibility -> projective bilinear feasibility -> singular bilinear pencil feasibility -> real 3-tensor degeneracy。
5. 起点是 homogeneous quadratic feasibility：给定对称 \(Q_1,\ldots,Q_m\in\mathbb{Q}^{n\times n}\)，问是否存在非零 \(u\) 使 \(u^\top Q_tu=0\)。
6. 这一步的 hardness 直接引用 $\exists\mathbb{R}$ compendium，论文不重新证明。
7. 第一层 bilinearization 引入 \(x,y\in\mathbb{R}^n\)，把 \(u^\top Q_tu=0\) 改成 \(x^\top Q_ty=0\)。
8. 为了强迫 \(x,y\) 表示同一个 projective point，加入所有 \(2\times2\) minor 约束 \(x_iy_j-x_jy_i=0\)。
9. 这些 minor 本身也是 bilinear constraints，可表示成 \(x^\top M_\ell y=0\) 的矩阵形式。
10. 若 \(x,y\neq0\) 且所有 minors 为零，则 \((x,y)\) 两列张成 rank 1，故 \(y=\lambda x\)，\(\lambda\neq0\)。
11. 代回 \(x^\top Q_ty=0\) 后得到 \(x^\top Q_tx=0\)，完成 quadratic feasibility 到 projective bilinear feasibility 的精确等价。
12. 第二层引入 singular bilinear pencil feasibility：给定 \(A_0,\ldots,A_r\)，问是否存在非零 \(x,y,z\) 使所有 slice bilinear forms 为零，并且 \(M(z)y=0\)、\(M(z)^\top x=0\)。
13. 其中 \(M(z)=\sum_{\ell=0}^rz_\ell A_\ell\)。
14. 从 projective bilinear feasibility 到 pencil feasibility 的 reduction 很直接：设 \(A_0=0\)，\(A_\ell=M_\ell\)。
15. 若原 bilinear system 有 \(x,y\) 解，则取 \(z=e_0\)，因为 \(A_0=0\)，两个 kernel constraints 自动成立。
16. 反向方向也直接：pencil feasibility 的第一组约束已经包含 \(x^\top M_\ell y=0\)。
17. 第三层把矩阵 pencil 编码成 3-tensor：令 \(T(:,:,\ell+1)=A_\ell\)。
18. 在这个 slice construction 下，\(T(x,y,\cdot)\) 正好列出所有 \(x^\top A_\ell y\)。
19. \(T(\cdot,y,z)\) 正好是 \(\sum_\ell z_\ell A_\ell y=M(z)y\)。
20. \(T(x,\cdot,z)\) 正好是 \(\sum_\ell z_\ell A_\ell^\top x=M(z)^\top x\)。
21. 因此 tensor degeneracy equations 与 singular bilinear pencil feasibility equations 完全同构。
22. membership in $\exists\mathbb{R}$ 也直接：所有 contraction equations 都是 degree-2 polynomial equations，非零性用 \(\sum_i x_i^2>0\)、\(\sum_i y_i^2>0\)、\(\sum_jz_j^2>0\) 表达。
23. 主定理因此是 real 3-tensor degeneracy is $\exists\mathbb{R}$-complete。
24. 论文第二部分转向 hyperdeterminant，并强调 degeneracy 与 hyperdeterminant vanishing 只在 boundary format 中等价。
25. boundary format 写成 \(n_i=k_i+1\)，要求某个模式满足 \(k_0=k_1+k_2\)，等价于 \(n_0=n_1+n_2-1\)。
26. 在 boundary format 中，GKZ hyperdeterminant 是 Segre dual hypersurface 的 defining equation，且 \(\Det(T)=0\iff T\) degenerate。
27. 因此如果想把 unrestricted tensor degeneracy 的 $\exists\mathbb{R}$-hardness 转移到 hyperdeterminant vanishing，缺的不是 degeneracy 定义，而是 deterministic boundary-format lifting。
28. 论文把这个缺口形式化为：给任意 \(T\)，构造 boundary-format \(\widehat{T}\)，满足 \(T\) degenerate iff \(\widehat{T}\) degenerate。
29. 已知 randomized completion 路线是把 \(T\) 放进更大的 boundary-format tensor \(\widehat{T}(U)\)，其中 \(U\) 是辅助 completion variables。
30. 定义 completion polynomial \(P_T(U)=\Det(\widehat{T}(U))\)。
31. 理想性质是：若 \(T\) degenerate，则 \(P_T\equiv0\)；若 \(T\) nondegenerate，则 \(P_T\not\equiv0\)。
32. randomized evaluation 用 Schwartz-Zippel 在高概率下找到 \(P_T(U^\star)\neq0\) 的点。
33. deterministic hardness transfer 卡在 deterministic hitting：对每个 nonzero \(P_T\)，显式找一个 \(U^\star\) 使 \(P_T(U^\star)\neq0\)。
34. 论文把这个卡点命名为 Completion-hitting / Completion-PIT。
35. 论文的 PIT barrier 不声称一般 PIT hard，只说明 deterministic lifting 至少需要解决这个 explicit hyperdeterminant-circuit family 的 identity/hitting 问题。
36. 论文还列出若干自然 deterministic embedding 策略失败的结构原因。
37. direct-sum embedding 失败，因为 auxiliary block 可以产生与原 \(T\) 无关的 degeneracy witness。
38. pairwise polynomial constraints 失败，因为三个 vector 可以有 disjoint supports，所有 pairwise products 消失但三者仍非零。
39. Vandermonde-weighted pairwise constraints 只能强迫 coordinatewise pairwise annihilation，仍允许 distributed-support witnesses。
40. fixed low-dimensional parametrization 失败，因为某个 nonzero \(P_T\) 的 zero set 可能完全包含该 parametrization image。
41. local hyperdeterminant gadgets 失败，因为组合后 global elimination determinant 产生跨 gadget mixed minors，不能保持局部分解。
42. boundary-format convolution 只能在已是 boundary format 的 tensors 上操作，不能把任意 tensor 变成 boundary format。
43. 全文没有实验、没有数值 benchmark、没有代码，核心证据是短而直接的 algebraic reduction。
44. tex 源码只有一个 `main.tex`，共 544 行，没有 appendix；证明链条也因此比较透明，但外部引用承担了部分基础事实。
45. 近期 arXiv 检索显示，2025-2026 年相关近邻包括 group/algebra hyperdeterminant 和 Cayley first hyperdeterminant identity，但没有发现直接证明 tensor degeneracy $\exists\mathbb{R}$-completeness 或 deterministic boundary lifting 的并行工作。

### 关键技术细节

1. 输入规模：homogeneous quadratic feasibility 的输入是 \(m\) 个 \(n\times n\) rational symmetric matrices。
2. bilinear constraints 数量：原 \(m\) 个 quadratic constraints 变成 \(m\) 个 \(x^\top Q_ty=0\)，再加 \(\binom{n}{2}\) 个 minor constraints。
3. minor constraints 的 matrix encoding 是 sparse skew-symmetric selector：\(x_iy_j-x_jy_i=x^\top(E_{ij}-E_{ji})y\)。
4. projective nonzero condition 要求 \(x\neq0\) 且 \(y\neq0\)，这避免了 trivial zero witness。
5. singular pencil reduction 使用 \(A_0=0\)，这是一个很强的 simplification：hardness 来自 slice bilinear constraints，而不是 pencil kernel constraints 本身。
6. tensor construction 的维度是 \(n\times n\times(r+1)\)，其中 \(r=m+\binom{n}{2}\)。
7. tensor entries 全部是 rational，因为只是把输入 matrices 作为 slices 堆叠。
8. degeneracy witness 的三个向量分别对应 bilinear feasibility 的 \(x,y\) 与 pencil coefficient vector \(z\)。
9. membership proof 没有 quantifier alternation，只有 existential real variables + degree-2 equations/inequalities。
10. boundary-format relation 依赖 GKZ：只有 dual variety 是 hypersurface 时，hyperdeterminant 才作为单个 polynomial certificate 存在。
11. completion polynomial \(P_T(U)\) 是 hyperdeterminant circuit family 的输入特化；论文引用 Joux-Narayanan 的 arithmetic circuit 描述来说明它不是纯几何抽象。
12. PIT connection 是 hitting-set style：能 deterministically 选到 \(P_T(U^\star)\neq0\) 的点，就能对该 family 做 identity testing。
13. 论文没有给 \(P_T\) 的具体 degree、variable count 或 circuit-size bounds；这些被留给引用工作和 future work。
14. 论文没有给 deterministic boundary lifting 的 positive construction，只给 barrier/失败模式。

### 核心结果

| 结果 | 形式 | 重要性 |
|------|------|--------|
| Projective bilinear feasibility $\exists\mathbb{R}$-complete | HQF + all \(2\times2\) minors | 把 quadratic feasibility 转成纯 bilinear normal form |
| Singular bilinear pencil feasibility $\exists\mathbb{R}$-hard | 设 \(A_0=0\), \(z=e_0\) | 给 tensor slice construction 准备完全匹配的中间问题 |
| Real 3-tensor degeneracy $\exists\mathbb{R}$-complete | \(T(:,:,l+1)=A_l\) | 主定理，说明 intrinsic tensor singularity 已达 full real-existential hardness |
| Boundary hyperdeterminant hardness未解决 | 需要 deterministic boundary-format lifting | 明确划出已证结论与 open problem 的边界 |
| Deterministic lifting -> Completion-PIT | 找 \(P_T(U^\star)\neq0\) | 把几何 completion 问题转成 algebraic derandomization 问题 |
| 多个 deterministic embedding 策略失败 | direct sum / pairwise / Vandermonde / gadgets / convolution | 支持“卡点不是小修小补”的论述，但不是一般 impossibility theorem |

反直觉点：

1. 证明 real 3-tensor degeneracy $\exists\mathbb{R}$-complete 不需要复杂 tensor gadgets，只需要把 bilinear forms 当作 tensor slices。
2. singular pencil feasibility 的 hardness 用 \(A_0=0\) 绕开了 kernel constraints 的复杂性；这使 reduction 非常干净，但也让中间问题偏 unrestricted。
3. hyperdeterminant 是 classical polynomial certificate，但它反而比 intrinsic degeneracy 更难直接做 deterministic hardness transfer。
4. 难点不是“随机 completion 不存在”，而是“确定性地选中一个不落在 completion polynomial zero set 的点”。
5. 论文把 hyperdeterminant vanishing 的复杂性问题从 tensor geometry 拉到 PIT/derandomization 语言里，这是本文第二个主要贡献。

### 明确局限

1. 本文没有证明 boundary-format hyperdeterminant vanishing 是 $\exists\mathbb{R}$-hard 或 $\exists\mathbb{R}$-complete。
2. PIT barrier 不是 lower bound；它只说明 deterministic lifting 会推出该 structured family 的 deterministic identity/hitting procedure。
3. “自然 deterministic embeddings 失败”的 propositions 更像结构化反例/失败模式整理，不是覆盖所有可能 construction 的 impossibility theorem。
4. 主 hardness reduction 使用 \(A_0=0\)，对更结构化、非退化的 matrix-pencil families 是否仍 hard 没有说明。
5. 没有 mechanized proof、没有 checker、没有补充材料，第三方验证需要手工 proof-check。
6. 没有展示具体小例子来说明 tensor degeneracy witness 与 bilinear feasibility witness 的 correspondence。
7. 没有给 completion polynomial family 的 explicit size/degree 参数，导致 PIT barrier 的精细复杂度还不够具体。
8. 论文只查/引用少量核心文献；对于 hyperdeterminant arithmetic circuit 和 randomized completion 的叙述依赖外部近期结果。
9. 对 FEX 或 symbolic regression 没有直接建模；它是 algebraic complexity prior，不是 SR/FEX 方法论文。

## Read by: 0616-fex (2026-06-19)

本次从 `0616-fex` 视角看，本文是一个有用的复杂度 theory prior，但不构成直接竞品。它没有研究 symbolic regression、FEX、PDE residual、RL controller、表达式树搜索或实值常数拟合；主贡献是把 intrinsic tensor degeneracy 证明为 $\exists\mathbb{R}$-complete，并把 boundary-format hyperdeterminant hardness 的 deterministic transfer 卡点表述为 completion-PIT。

对 `fex-search-complexity` 最有价值的是 reduction pattern。本文先把 quadratic feasibility bilinearize，再用 projective minors 保持非零 projective witness，最后把所有 bilinear constraints 堆成 tensor slices。这说明如果 FEX residual grammar 能承载足够自由的 bilinear/quadratic constraint family，就有机会把 bounded-ETR 或 ETR-INV 类约束映射进 fixed-structure coefficient feasibility。但这一步不能直接从本文推出；FEX 的 grammar、PDE residual evaluation、inner optimizer 和 coefficient domains 都要单独建模。

撞车评估：低。本文会撞到的是“tensor degeneracy 的 $\exists\mathbb{R}$-completeness”和“hyperdeterminant deterministic lifting/PIT barrier”这两个 claim；它不覆盖 FEX discoverability factorization、quotient-count theorem、blind sampler lower bound、PG gradient-domination condition 或 FEX coefficient feasibility reduction。若我们的 idea 后续写 Layer 4 hardness，应引用本文作为代数复杂性参考，并明确差异化在 FEX-specific residual grammar，而不是一般 tensor contraction system。
