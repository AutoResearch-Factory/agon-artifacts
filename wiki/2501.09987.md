# On understanding and overcoming spectral biases of deep neural network learning methods for solving PDEs (2501.09987)

- **Year**: 2025.01
- **Authors**: Zhi-Qin John Xu, Lulu Zhang (Shanghai Jiao Tong University); Wei Cai (Southern Methodist University)
- **Venue**: arXiv preprint; manuscript template targets Journal of Computational Physics
- **Keywords**: spectral bias, frequency principle, PDE neural solvers, PINN, PhaseDNN, MscaleDNN, Fourier features, neural operators
- **One-liner**: 综述并分类克服 PDE 神经解法谱偏置的技术路线。

## 精华提炼

### 管线/方法

1. 论文定位是 review，不提出单一新算法，而是围绕“DNN 学 PDE 解时低频先收敛、高频滞后”的 spectral bias 做方法谱系整理。
2. 输入问题族是含多频率解的 PDE 或 PDE operator learning，包括 Poisson/Helmholtz、Navier-Stokes、Boltzmann、interface problem、wave scattering、turbulence/wind field reconstruction 等。
3. 作者先定义标准 DNN：多层仿射变换加逐点激活，参数集合为 theta，输出作为 PDE 解的 surrogate。
4. PDE 求解训练入口分两类：residual-based loss，如 PINN/least-squares residual；variational energy loss，如 Deep Ritz。
5. PINN loss 使用 PDE residual 的均方项加边界条件损失，训练点在域内和边界随机采样；作者指出 residual 中高阶导数会在频域放大高频项，但也可能带来训练不稳定。
6. Deep Ritz 使用能量泛函，例如二阶椭圆问题的 Dirichlet energy；它降低了微分阶数，训练更稳定，但对高频收敛可能更慢。
7. spectral bias 经验演示用一维目标函数 `sin(x)+sin(3x)+sin(5x)`，比较 DNN 输出和目标函数的 DFT 峰值误差，显示低频先拟合。
8. qualitative theory 以一隐藏层 tanh 网络为例，写出输出的 Fourier transform；梯度贡献含 `exp(-|pi k/(2w_j)|)` 衰减项，解释高频梯度弱。
9. linear quantitative theory 用 NTK/linearized regime，把训练残差动力学写成 kernel flow；在 ReLU 两层宽网络中给出 LFP operator，其系数随 `||xi||` 衰减，说明高频误差更慢。
10. 方法分类第一类是 frequency shifting：把高频谱段平移到低频后训练，再乘回 phase factor。
11. PhaseDNN 将带限函数的频域区间 `[-K0,K0]` 切成子带 `A_j`，每个子带用 `exp(-i omega_j x)` 平移到低频，再由小 DNN 学习。
12. PhaseDNN 组合形式是 `sum_j exp(i omega_j x) T_j(x)`，实值写法是多组 `A_m(x) cos(omega_m x)+B_m(x) sin(omega_m x)`。
13. Helmholtz 示例设 `u'' + lambda^2 u = sin(mu x)`，边界 `u(-1)=u(1)=0`；PhaseDNN 选择频率 `{0, lambda, mu}`。
14. 该示例中每个 `A_m,B_m` 是 `1-40-40-40-40-1` DNN；在 `lambda=205, mu=3` 下 PhaseDNN 能拟合高频振荡，普通全连接 DNN 失败。
15. PhaseDNN 的直接局限是频率维数灾难：维度升高后需要沿多个坐标方向枚举 phase shifts，Fourier terms 数量指数增长。
16. 方法分类第二类是 frequency scaling：用输入/频域径向缩放把高频区域压到低频区域，代表是 MscaleDNN、Fourier features、adaptive activation、random feature method。
17. MscaleDNN 从带限函数 `f(x)` 的频域球 `B(Kmax)` 出发，把频域分成同心 annuli `A_i`。
18. 对第 `i` 个频带定义 `hat f_i^(scale)(k)=hat f_i(alpha_i k)`，物理域对应 `f_i^(scale)(x)=alpha_i^{-d} f_i(x/alpha_i)`。
19. 因而原函数可用 `sum_i alpha_i^d f_theta_i(alpha_i x)` 近似；尺度 `alpha_i` 可作为超参数或可训练参数。
20. MscaleDNN-1 在第一隐层把 neuron 分组，不同组接收 `a_i x`，常用 `a_i=i` 或 `2^(i-1)`。
21. MscaleDNN-2 是多个子网络求和，每个 scaled input 进入独立 subnetwork，后续权重矩阵等价于 block diagonal。
22. 作者报告 2D Poisson oscillatory solution 示例：普通 FC-DNN 为 `2-3200-3200-3200-1`，MscaleDNN 用 8 个 `2-400-400-400-1` 子网，scales 为 `{1,2,4,8,16,32,64,128}`。
23. 该示例中 MscaleDNN 能捕捉 `N=20` 的多频精确解，FC-DNN 在横截面上明显漏掉高频振荡。
24. MscaleDNN 的理论解释来自两层 sine activation 网络的 NTK/diffusion model；频域误差满足 diffusion-reaction 方程。
25. diffusion 系数 `A_s` 和反应系数 `B_s` 随 scale 数增加而扩大频域 support，解释为什么多尺度网络减弱 spectral bias。
26. Fourier feature networks 被解释为 MscaleDNN 的固定第一层版本：输入映射为 `[a_k cos(2 pi b_k^T v), a_k sin(2 pi b_k^T v)]`。
27. NeRF positional encoding 是特殊多尺度 sinusoidal embedding；随机 Fourier features 则从 Gaussian 采样 `b_k`，调带宽以改变 NTK 频率响应。
28. 文中连接了 MFFNN/MHDnet 等做法：在 embedding 中混合 `cos(2pi Bx)`, 原始 `x`, `sin(2pi Bx)` 或 scaled input。
29. activation optimization 方向包括 adaptive activation、reproducing activation functions、Kronecker/Rowdy networks。
30. adaptive activation 使用 `sigma(n a x)`，其中 `n` 是固定整数，`a` 是可训练超参数；其作用类似提高可见频率范围。
31. RAF 用多个基本激活函数的可学习组合，如 `{x, x^2, sin(beta x), exp(-(beta x)^2)}`，让网络重现多项式、Fourier、wavelet/RBF 风格基函数。
32. Rowdy-Net/Kronecker network 在每个 neuron 混入多种 `sin`/`cos` 高频激活项，理论上早期 loss 下降快于普通 feed-forward network。
33. Random Feature Method 用固定随机特征 `phi_m(x)=sigma(k_m dot x + b_m)` 做线性组合，训练只求解线性 least squares。
34. RFM 的 local PoU 版本把域分成 partition-of-unity patches，每个 patch 用 normalized coordinate `(x-x_n)/r_n`；`r_n` 起到局部尺度控制作用。
35. RFM 可加入 global component 加 local random features，最终求 overdetermined system 的 QR least-squares，而不是梯度下降，因此可在低维复杂域达到近 machine accuracy。
36. 方法分类第三类是 hybrid approach：利用传统迭代器擅长消高频误差、DNN 擅长低频近似的互补性。
37. Jacobi-DNN/Int-Deep 思路是先用 DNN 训练到一定阶段，再用 Jacobi/Gauss-Seidel 等迭代法从 DNN 初值继续平滑高频误差。
38. BLS-Jacobi 把 Broad Learning System 作为低频学习器，再交给 Jacobi 消高频。
39. HINTS 先离线训练 DeepONet 近似线性微分方程 solution operator，然后在迭代中按 `1:(n_r-1)` 比例交替使用 numerical iterator 和 DeepONet iterator。
40. HINTS 的关键是 DeepONet 负责低频近似，传统 relaxation 负责高频 mode，从而获得更均匀的谱收敛。
41. MgNet 把 multigrid 的 smoothing、restriction、prolongation 等操作视为 convolution，并替换成可训练 CNN 模块。
42. MgNet 利用粗细网格连接低/高频误差，但因为依赖 grid，主要适合低维 PDE。
43. multi-level learning 递归训练多个网络学残差；第 k 个网络补偿前 k-1 个网络的残差，类似多级 correction。
44. multi-grade learning 则让第 k 个网络接收第 k-1 个网络输出作为输入，逐级拟合目标输出。
45. neural operator 部分把 spectral bias 扩展到 operator learning：即使输入函数是常数，算子 `G[a](x)=sin(m a)` 对参数 `a` 也可呈高频依赖。
46. MscaleFNO 用多个 scaled input 分支学习 oscillatory operator：`u(x)=sum_i gamma_i FNO_theta([c_i x, c_i omega(x)])(x)`。
47. MscaleFNO Helmholtz 示例：`u'' + (lambda^2+c omega(x))u=f(x)`, `L=11`, `lambda=2`, `c=3.6`，source 频率 `mu_k=300+35k`。
48. 该示例中 MscaleFNO 用 8 个 scales：`1,80,160,200,240,280,360,400`；normal FNO 48 channels，MscaleFNO 每分支 16 channels。
49. 训练 500 epochs 后，MscaleFNO relative error 约 `4e-2`，normal FNO 约 `8e-1`，普通 FNO 只学到平均趋势，漏掉强振荡。
50. diffusion/generative operator learning 部分提到 HANO、diffusion model 与 neural operator 集成，用生成模型改善 turbulence/operator 的高频谱表示。
51. 结论把这些方法的传统数值分析根源对应起来：MscaleDNN/Fourier features 类似 wavelet/multiresolution，PhaseDNN 类似 wave-ray multigrid，RFM 类似 domain decomposition，MgNet 类似 multigrid。
52. 作者认为未来核心问题是建立适合 DNN 复合结构的数学分析工具，而不只是经验调参。
53. open problem 之一是训练过程机制：information bottleneck 的 fitting/diffusion phase 是否能解释不同抗谱偏置结构的有效性。
54. open problem 之二是 neuron condensation：两层 ReLU 在小初始化或 dropout 下会聚到有限方向，这可能解释网络先进入低复杂度、低频可表达状态。
55. open problem 之三是不同 PDE loss 的频域收敛理论：PINN、DeepRitz、Galerkin、SDE-based methods、operator learning 都需要各自的 spectral-bias 分析。
56. open problem 之四是从 linear regime 扩展到 nonlinear regime，因为函数复合使 Fourier 分析明显更难。
57. open problem 之五是 scale、activation、penalty constants、learning rate 等超参数选择仍缺少可靠原则。
58. 对读者最实用的 takeaway：若 PDE 解或算子含已知高频结构，优先考虑显式频率先验；若频率范围未知，adaptive scale 或 hybrid solver 比固定 phase dictionary 更稳。
59. 对实验设计的 takeaway：仅比较普通 PINN vs proposed method 不够，需要按频率分解误差，并报告低/中/高频 mode 的收敛曲线。
60. 对可靠性判断的 takeaway：这篇 review 的强项是分类和公式连接，弱项是缺少系统 review protocol、统一 benchmark、统一代码和跨方法同预算实验。

### 关键技术细节

- **核心对象**：多频 PDE 解、oscillatory operator、high-frequency approximation。
- **基线方法**：PINN residual loss、Deep Ritz variational loss、普通 fully-connected DNN、normal FNO。
- **谱偏置诊断**：对 DNN 输出和目标做 DFT/Fourier transform，跟踪各频率 relative error。
- **PhaseDNN 频率选择**：1D Helmholtz 示例用 `{0, lambda, mu}`，`lambda=205`, `mu=3`。
- **PhaseDNN 网络规模**：每个 `A_m,B_m` 使用 `1-40-40-40-40-1` DNN。
- **MscaleDNN scales**：Poisson 示例用 `{1,2,4,8,16,32,64,128}` 八个尺度。
- **MscaleDNN 网络规模**：8 个 `2-400-400-400-1` 子网，对比 `2-3200-3200-3200-1` FC-DNN。
- **MscaleFNO scales**：Helmholtz operator 示例用 `1,80,160,200,240,280,360,400`。
- **MscaleFNO 结果**：500 epochs 后 error 约 `4e-2`，normal FNO 约 `8e-1`。
- **理论工具**：NTK linearization、LFP operator、频域 diffusion-reaction equation、multigrid smoothing/coarse correction 类比。
- **常见失败模式**：高维 phase shift 维数灾难；固定 scales 需调参；传统迭代混合方法仍受低维网格限制；neural operator 对高频参数依赖也会失效。

### 核心结果

| 方向 | 代表方法 | 关键机制 | 论文给出的结果/判断 |
|------|----------|----------|--------------------|
| frequency shifting | PhaseDNN, PD-PINN | 先把高频子带移到低频 | Helmholtz `lambda=205` 示例中普通 DNN 失败，PhaseDNN 成功 |
| frequency scaling | MscaleDNN | scaled input/annular frequency decomposition | 2D Poisson 多频解中 MscaleDNN 捕捉高频，FC-DNN 漏掉 |
| fixed embedding | Fourier features | 固定 sinusoidal first layer 调 NTK 带宽 | 与 MscaleDNN 的 sinusoidal 第一层有结构等价性 |
| adaptive activation | `sigma(n a x)`, RAF, Rowdy-Net | 激活函数带可学习尺度或多频基 | 能改善部分高频拟合，但仍需选择结构和超参数 |
| random features | RFM + PoU | 固定随机基 + QR least squares | 低维复杂域可接近 machine accuracy，不走梯度下降 |
| hybrid solvers | DNN-Jacobi, HINTS, MgNet | DNN/DeepONet 学低频，传统迭代消高频 | 更接近频率均匀收敛，但常受 grid/维度限制 |
| operator learning | MscaleFNO, Mscale-DeepONet, HANO | 对输入函数和位置做多尺度算子学习 | MscaleFNO 在 oscillatory Helmholtz operator 上显著优于 normal FNO |

### 明确局限

- 这是一篇综述，没有提出新的统一 benchmark 或新算法。
- 不同方法的数字来自不同论文/场景，不能直接视为同预算公平比较。
- 很多示例仍是 1D/2D PDE 或受控合成问题；高维、复杂几何和真实工程数据仍未形成统一证据链。
- PhaseDNN 对频率先验依赖强，高维频率网格会指数膨胀。
- MscaleDNN/Fourier features 的 scale 选择高度影响表现，作者也承认需要理论或自适应原则。
- RFM 的高精度来自低维线性 least-squares 求解，扩展到高维或大规模 operator learning 仍有成本问题。
- Hybrid solver 需要传统数值器件，因此不能完全规避网格、离散化和低维限制。
- Neural operator 的抗谱偏置方法尚新，MscaleFNO、HANO、diffusion operator 之间缺少统一、长期稳定的 benchmark。
- 论文没有提供系统搜索协议；review 范围受作者社区和自家方法影响较明显。
- 对 0616-fex 若想主张 novelty，不能只说“克服谱偏置”，必须落到更窄 claim：新 scale 选择、新 operator architecture、新 benchmark 或新的频域诊断。

## Read by: 0616-fex (2026-06-19)

从 `0616-fex` 的审读角度，这篇不是一个可直接复现的新 baseline，而是 prior-work 地图。它最重要的价值是把“克服 DNN/PINN/FNO 高频困难”拆成几个已经被占据的 claim 家族：frequency shifting、frequency scaling、Fourier features、adaptive activation、random features、hybrid classical-neural solver、multiscale neural operator、diffusion/generative operator learning。

撞车风险很高的表述包括：用 Fourier feature 或 positional encoding 改善高频 PDE；用多尺度输入/多分支网络缓解 PINN 谱偏置；用 DeepONet/FNO 加传统迭代器做混合 solver；用 MscaleFNO/Mscale-DeepONet 处理 oscillatory operator。上述都已有明确文献链。

可利用的方法包括：以 DFT/per-frequency relative error 做主评测；用 MscaleDNN 的 annular/scaled input 解释模型设计；将 HINTS 的 low-frequency neural operator 与 high-frequency relaxation 互补性作为 hybrid baseline；把 scale selection 设计成真正的 novelty，而不是把固定 scale 当新方法。

差异化建议：`0616-fex` 若要立项，最好把 claim 压到更具体的未解决点，例如“未知频谱下自动选择 scale/feature bank”“跨 PDE family 的频域泛化”“统一低中高频 benchmark 和 veto-style 评测”“高维复杂域中不依赖网格的 hybrid solver”。这篇应作为背景引用，同时用其 open problems 反向定义我们的 novelty 边界。
