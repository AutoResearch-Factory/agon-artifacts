# Physics-Informed Inference Time Scaling for Solving High-Dimensional PDE via Defect Correction (2504.16172)

- **Year**: 2025.04
- **Authors**: Zexi Fan (Peking University), Yan Sun (Visa Inc.), Shihao Yang (Georgia Institute of Technology), Yiping Lu (Northwestern University)
- **Venue**: arXiv
- **Keywords**: inference-time scaling, high-dimensional PDE, defect correction, PINN, Gaussian Process PDE solver, Multilevel Picard iteration, Feynman-Kac simulation, SciML reliability
- **One-liner**: 用缺陷方程把预训练 PDE surrogate 在推理时校准，误差可乘法下降。

## 精华提炼

### 管线/方法

1. 输入是一类半线性抛物 PDE：时间区间 `[0,T]`，空间维度可到 160，算子含漂移、扩散和非线性项 `F(u, sigma^T grad u)`。
2. 先离线训练一个全局 surrogate `\hat u`，论文实验里主要用 PINN 和 Gaussian Process PDE solver。
3. PINN 通过 PDE residual、Dirichlet boundary loss、initial/terminal condition loss 训练，常用 5 层、每层 50 hidden units、`tanh` 激活。
4. GP surrogate 采用 RKHS/MAP 形式，把 PDE 约束写成有限维非线性约束优化，再用 sparse Cholesky 和 Newton 迭代求解。
5. 推理阶段不直接使用 `\hat u(t,x)`，而是把真实误差定义为 defect：`\breve u = u - \hat u`。
6. 核心推导是把原 PDE 与 surrogate residual 相减，得到新的 `Structural-preserving Law of Defect`。
7. 线性 warm-up 中，defect PDE 的源项就是 surrogate residual `epsilon`，终端条件是 `g - \hat u(T,.)`。
8. 半线性情形中，defect PDE 保持半线性结构，新的非线性项为 `F(\hat u+\breve u, grad \hat u + grad \breve u) - F(\hat u, grad \hat u) + epsilon`。
9. 因为结构不变，论文可继续用 Feynman-Kac、Bismut-Elworthy-Li 表示和 Multilevel Picard (MLP) 迭代求 defect。
10. 推理输出是 `u_SCaSML(t,x) = \hat u(t,x) + \breve u_est(t,x)`，只在用户关心的点或批量点上花额外算力。
11. 实现了两类 MLP：Quadrature MLP 用 Gauss-Legendre quadrature 处理时间积分；Full-history MLP 用 Monte Carlo 时间采样处理时间积分。
12. 论文主表使用 full-history MLP，设 level `n=2`、basis sample `M=10`，并在 scaling study 中增加 `M`。
13. 对强非线性问题，作者使用 thresholding/clipping 稳定 defect estimator；SCaSML 的 clipping 阈值通常比 naive MLP 小，因为 defect 幅度更小。
14. 对 HJB/LQG 问题，作者用 Hutchinson estimator 随机估计 Laplacian/divergence，抽样 `d/4` 个维度以降低内存和计算压力。
15. 对振荡 diffusion-reaction 问题，Hutchinson estimator 被报告为不稳定，因此改用完整 Laplacian。
16. 论文还做 fixed-budget 分析，比较把算力给更多训练、更大 PINN、naive MLP，还是给 SCaSML 推理校准。

### 关键技术细节

1. 目标 PDE 类别：半线性抛物 PDE，覆盖 linear convection-diffusion、viscous Burgers、HJB/LQG、diffusion-reaction。
2. 理论简化设定：`\mu=0`，`\sigma=s I_d`，重点证明 defect PDE 的 MLP 误差可被 surrogate error 乘法缩放。
3. 主要假设：surrogate residual 的 `L^\infty` 范数和 defect 的 `W^{1,\infty}` 范数都由 `e(\hat u)` 控制。
4. Full-history MLP 额外假设 residual 沿 Brownian path 的矩有限，并使用密度 `rho(s)=(1-alpha)s^{-alpha}` 做时间采样。
5. Quadrature MLP 额外要求 defect 对 `(\partial_t + sigma^2/2 Delta)^k` 有 Gevrey 类高阶正则界，因而条件更强。
6. 核心定理：SCaSML 的全局 `L^2` defect 估计误差 `<= E(M,N) * C_F * e(\hat u)`。
7. 改进 scaling law：若 surrogate error 随训练点数 `m` 为 `O(m^{-gamma})`，再分配 `m` 个推理样本，误差界变为 `O(m^{-gamma-1/2+o(1)})`。
8. Full-history 复杂度从 naive MLP 的 `O(d epsilon^{-(2+delta)})` 变为 `O(d e(\hat u)^{2+delta} epsilon^{-(2+delta)})`。
9. Quadrature 复杂度从 `O(d epsilon^{-(4+delta)})` 变为 `O(d e(\hat u)^{4+delta} epsilon^{-(4+delta)})`。
10. LCD 实验：`d in {10,20,30,60}`，域 `[0,0.5] x [0,0.5]^d`，PINN 训练 `10^4` iterations。
11. LCD PINN 训练点：每次 `2.5e3` interior 和 `1e2` boundary collocation points。
12. VB 实验：`d in {20,40,60,80}`，`sigma0=sqrt(2)`，PINN 与 GP 两种 surrogate 都测试。
13. VB-PINN 训练：`10^4` iterations，Adam `7e-4`，`beta1=0.9`，`beta2=0.99`，`2500` interior、`100` boundary、`160` terminal points。
14. VB-GP 训练：Newton method 20 iterations，`1000` interior、`200` boundary points。
15. LQG/HJB 实验：`d in {100,120,140,160}`，PINN 训练 `2.5e3` iterations，`100` interior、`1000` boundary points。
16. DR 实验：`d in {100,120,140,160}`，PINN 训练 `2.5e3` iterations，`1000` interior、`1000` boundary points。

### 核心结果

1. 主表中 SCaSML 在几乎所有设置下 `L^2`、`L^\infty`、`L^1` error 都低于 surrogate 和 naive MLP。
2. LCD full-history 主表相对 `L^2`：10d PINN `5.20E-02` 到 SCaSML `2.74E-02`；60d `3.13E-01` 到 `1.32E-01`。
3. VB-PINN 相对 `L^2`：20d `1.17E-02` 到 `4.03E-03`；80d `6.74E-02` 到 `5.64E-02`。
4. VB-GP 相对 `L^2`：20d `1.47E-01` 到 `6.23E-02`；80d `2.66E-01` 到 `1.52E-01`。
5. LQG 中 naive MLP 明显失败：100d relative `L^2=5.63E+00`，而 SCaSML 为 `5.53E-02`，PINN 为 `7.97E-02`。
6. LQG 160d：PINN `1.12E-01`，naive MLP `5.27E+00`，SCaSML `9.94E-02`。
7. DR 中 base PINN 已经较强，SCaSML 仍有小幅稳定提升：100d `1.41E-02` 到 `1.11E-02`；160d `3.45E-02` 到 `3.22E-02`。
8. 作者声称总体误差下降约 20-80%，但 DR 这种 base surrogate 已强的场景下降幅度较小，约 6.6-10.9%。
9. 统计分析做了 10 次重复实验、1200 test points，并给出 95% CI 和 paired t-test。
10. 多数配置的 SCaSML 相对 baseline 的 p 值远小于 `0.001`，支撑其不是单次随机种子偶然结果。
11. Inference scaling 曲线显示随着 `M` 增加，SCaSML error 通常下降；LQG 在 `M=14` 后出现 plateau，作者归因于 clipping threshold。
12. Fixed-budget 结果表明，在相同 wall-clock budget 下，把边际算力用于 defect correction 往往比继续训练 PINN 或扩大 PINN 更有效。
13. 反直觉点：小 surrogate 加推理校准可以在同等预算下超过更大 PINN，说明高频 residual 不一定适合靠继续训练消除。
14. 另一个关键点：naive MLP 在高维强非线性问题中可能完全失效，但以 surrogate 为 control variate 后可稳定工作。

### 明确局限

1. 理论证明依赖 surrogate residual 与 defect 范数可由 `e(\hat u)` 控制，这是强假设，不是自动保证。
2. Full-history 理论条件比 Quadrature 宽松，但仍要求 residual 沿 Brownian path 的矩条件。
3. Quadrature MLP 需要高阶时间/空间正则性，对真实复杂 PDE 可能很难验证。
4. 实验问题都带解析解或高精度参考解，尚未证明在真实无 ground truth 科学模拟中也能可靠校准。
5. 所有验证看起来仍由作者团队完成，没有独立复现报告。
6. 代码声称公开，但论文没有给出 artifact evaluation 或外部 benchmark 记录。
7. 方法主要针对半线性抛物 PDE；椭圆 PDE、双曲 PDE、不可微边界、多尺度随机系数等是否适配还不清楚。
8. 推理时需要可计算 surrogate residual、梯度、Hessian/Laplacian；黑盒 surrogate 或不可微 solver 不直接适用。
9. 高维问题中 thresholding/clipping 是关键稳定手段，但阈值选择带有经验性，会影响 bias/variance。
10. SCaSML 增加推理时间，主表中 SCaSML runtime 经常显著高于 surrogate 本身；适合高精度点查询，不适合所有点都低延迟的场景。
11. 论文用“first inference-time scaling”表述较强，需注意 classical defect correction、a posteriori error estimation、control variates 等已有思想。
