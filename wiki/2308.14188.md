# Bayesian Deep Operator Learning for Homogenized to Fine-Scale Maps for Multiscale PDE (2308.14188)

- **Year**: 2023.08
- **Authors**: Zecheng Zhang (FSU), Christian Moya (Purdue), Wing Tat Leung (CityU HK), Guang Lin (Purdue), Hayden Schaeffer (UCLA)
- **Venue**: arXiv preprint
- **Keywords**: operator learning, DeepONet, multiscale PDE, homogenization, Bayesian inference, multi-fidelity, mesh-free solver, uncertainty quantification
- **One-liner**: 用 Deep Operator Network 从 coarse-scale 齐次化解学习到 fine-scale 解的算子映射，引入 patch oversampling 和 Bayesian 不确定性量化。

## 精华提炼

### 管线/方法

整体 pipeline 分两条线：

**Vanilla (无 patch)**：
1. 用 coarse solver（MsFEM 或 NH-PINN）求解 multiscale PDE 得到 coarse-scale 解 $u_0(x)$
2. 在有限观测点 $\{x_i\}$ 获得 fine-scale 观测值 $u(x_i)$
3. 用 DON 学习映射 $G_\theta: u_0(x_i) \mapsto u(x_i)$：branch net 输入 $u_0(x_i)$（单点值），trunk net 输入 $x_i$，输出 fine-scale 预测
4. Loss: $\mathcal{L}(\theta) = \frac{1}{N_p} \sum \|G_\theta(u_0(x_i))(x_i) - u(x_i)\|^2$

**With patch**：
1. Coarse solver 在 patch $P_i$（以 $x_i$ 为中心的邻域网格）上求 coarse-scale 解 $\hat{u}_i = \{u_0(x)\}_{x \in P_i}$
2. Branch net 输入整个 patch 上的 coarse-scale 值（flattened vector），trunk net 输入 $x_i$
3. DON 学习映射 $G_\theta: \hat{u}_i \mapsto u(x_i)$
4. Patch 的直觉来自 asymptotic expansion $u_{\epsilon,1} = u_0 + \epsilon\chi_j\frac{\partial u_0}{\partial x_j}$：fine-scale 修正需要 $\nabla u_0$，patch 提供了用 finite difference 近似梯度的信息

**Bayesian DON (B-DON)**：
1. 假设 Gaussian likelihood: $p(G|(u_o,x),\theta) = \mathcal{N}(G|G_\theta(u_o)(x), \text{diag}(\Sigma^2))$
2. 用 SG-reLD（stochastic gradient replica exchange Langevin diffusion）采样 $M$-ensemble $\{\theta_k\}_{k=1}^M$ 来近似后验 $p(\theta|\mathcal{D})$
3. SG-reLD 使用两条 Langevin diffusion：高温链探索参数空间，低温链收敛到局部最优，swap 机制帮助逃离局部极小
4. 预测分布：用 ensemble 拟合 Gaussian $\mathcal{N}(\bar{\mu}(u_o)(X_{\text{test}}), \bar{\sigma}^2_e(u_o)(X_{\text{test}}))$，其中 $\bar{\mu} = \frac{1}{M}\sum G_{\theta_k}$, $\bar{\sigma}^2_e = \frac{1}{M}\sum(G_{\theta_k} - \bar{\mu})^2$

### 关键技术细节

**DON 架构**：
- Branch net: $K$ 个独立子网络，每个接收输入函数在 $N$ 个 sensor 上的采样值，输出 $(b_1,\ldots,b_K)^\top$
- Trunk net: 接收位置 $x$，输出 $(t_1,\ldots,t_K)^\top$
- 最终输出: $\sum_{k=1}^K b_k \cdot t_k$（内积）
- 基于 Chen & Chen (1995) 的算子通用逼近定理
- Key property: 可以在任意 $x$ 处预测 → mesh-free solver

**Homogenization 理论基础**：
- 从 multiscale elliptic PDE $-\nabla\cdot(a(x/\epsilon)\nabla u_\epsilon) = f$ 出发
- 做 asymptotic expansion: $u_\epsilon = u_0 + \epsilon u_1 + \epsilon^2 u_2 + \mathcal{O}(\epsilon^3)$
- 推导出 key equation: $u_{\epsilon,1} = u_0 + \epsilon\chi_j\frac{\partial u_0}{\partial x_j}$
- 因此存在算子 $G: u_0 \mapsto u_{\epsilon,1}$，这是本文要学的目标

**Coarse solver 选择**：
- 1D 实验：用 classical homogenization theory（精确解已知）
- 2D 单 fast variable：用 NH-PINN (Leung et al. 2022)，mesh-free PINN 求解 homogenized equation
- 2D 多 scale：用 MsFEM with 1 local basis
- RTE 实验：用 elliptic 解作为 low-accuracy 近似（$\epsilon \to 0$ 时 elliptic → RTE）

**实验设置**：
- 所有实验训练 100 个独立模型取平均
- 1D elliptic: $\epsilon=1/16$, $N_p=16$ 观测点, patch size 1→9
- 2D elliptic (1 fast var): $\epsilon=1/8$, patch size $1\times1$ → $9\times9$, 观测点 9→49
- 2D elliptic (multi-scale): 7 个不同 $\epsilon$ 值混合, patch size $1\times1$ → $16\times16$
- RTE: $\epsilon=0.001$, 高对比散射, 观测点 9→49, 测试 mesh $51\times51$
- 噪声实验: Gaussian noise $\sigma^2=0.005^2$ 加到 training labels

**SG-reLD 算法**：
- 基于作者前作 (Lin et al. 2021, 2022, 2023)
- 两条 Langevin diffusion + stochastic swap process
- 声称在非凸场景有理论保证，能处理大数据集，加速后验收敛
- 实践中 $M=100$ ensemble size

**Surprising/反直觉结果**：
- 即使 patch=1（只看单点），DON 也能学到有意义的 fine-scale 修正——说明 coarse solution 本身已经编码了足够的 PDE 信息（permeability 信息被隐式传递）
- Multi-scale 2D 问题中，patch $16\times16$（256 个输入点）相比 $1\times1$ 误差降低近 5 倍，说明 patch 信息极为关键
- B-DON 采样预测分布的均值比直接用 noisy DON 训练更准——ensemble averaging 起到了 denoising 效果

### 明确局限

1. **DON 不在输入离散化下不变**：所有 patch 必须有相同的离散化（同样的 grid size 和点数），这限制了方法的灵活性。Conclusion 中明确提到这是 future work。
2. **没有与其他 operator learning 方法比较**：完全没有对比 FNO、Neural Operators 等，只用了 DON。
3. **全部是合成数据**：没有真实物理实验数据，所有 fine-scale "observation" 都是数值仿真生成的。
4. **仅限 2D 问题**：没有扩展到 3D，实际 multiscale 应用（如 porous media）可能需要 3D。
5. **B-DON 的 Gaussian 假设**：预测分布假设为 Gaussian，对于强非线性 multiscale 问题可能不准确。
6. **没有讨论 coarse solver 误差的传播**：coarse solver（MsFEM/NH-PINN）本身有误差，但论文假设 coarse solution 总是可靠的。
7. **patch 大小与计算成本未分析**：更大的 patch 意味着更多 coarse solver 调用，但没有讨论这个 trade-off。
8. **SG-reLD 的调参细节缺失**：温度参数、swap rate、步长等超参数没有讨论。

**可迁移的核心思想**：
1. **Patch as local context token**：本文证明从函数局部邻域采样（而非单点）可以显著提升预测精度。对于 0616-fex，这意味着 function-space embedding 可能需要 capture 局部邻域信息，而非只 encode 单点值。具体来说，可以用 patch-based tokenization 作为 function encoder 的输入格式。
2. **Homogenization 提供的 inductive bias**：asymptotic expansion 显示 fine-scale 修正只需要 $u_0$ 和 $\nabla u_0$——这是一个强烈的结构先验。对于 function-space embedding，类似的数学结构（如 Taylor expansion）可以作为 design principle：embed 函数值 + 各阶导数信息。
3. **Multi-fidelity feature fusion**：本文的 coarse + fine 双 fidelity 训练框架可以推广为"廉价特征 + 稀缺标注"的表示学习范式。对于 0616-fex，可以把 coarse solver 替换为 cheap encoder，fine observation 替换为 expensive ground-truth annotation。

**差异化方向**：
- 本文用 DON 作为 black-box function-to-function mapper，0616-fex 可以探索更 structured 的 embedding space（如对比学习、autoencoder）
- 本文的 patch 是固定 grid，0616-fex 可以用 learnable patch（类似 ViT 的 patch embedding）
- 本文没有讨论不同函数之间的 representation sharing，这是 0616-fex 可以切入的点
