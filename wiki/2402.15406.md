# Conformalized-DeepONet: A Distribution-Free Framework for Uncertainty Quantification in Deep Operator Networks (2402.15406)

- **Year**: 2024.02
- **Authors**: Christian Moya (Purdue Math), Amirhossein Mollaali (Purdue ME), Zecheng Zhang (FSU Math), Lu Lu (Yale Stats & Data Science), Guang Lin (Purdue Math & ME)
- **Venue**: arXiv
- **Keywords**: Conformal Prediction, DeepONet, Uncertainty Quantification, Neural Operators, Split Conformal Prediction, Quantile Regression, Scientific Machine Learning
- **One-liner**: 将 split conformal prediction 应用于 DeepONet 回归，使 B-DeepONet 和 Prob-DeepONet 能输出带覆盖率保证的置信区间，并提出新的 Quantile-DeepONet 架构用于更自然的 conformalized 回归。

## 精华提炼

### 管线/方法

整体管线分三条路径，均基于 split conformal prediction：

**路径 1：Conformalized B/Prob-DeepONet**
1. 训练 B-DeepONet（Langevin diffusion 采样得到 M-ensemble）或 Prob-DeepONet（负对数似然训练，输出 μ 和 σ）
2. 用训练好的模型对 calibration set（n 条 triplet）计算 score：s = |G - μ(u)(x)| / σ(u)(x)
3. 取 scores 的 ⌈(n+1)(1-α)⌉/n 分位数作为 q̂
4. 对新样本构造 CI：[μ(u)(x) - q̂·σ(u)(x), μ(u)(x) + q̂·σ(u)(x)]
5. 此 CI 满足 P(G_test ∈ C) ≥ 1-α（Theorem 1）

**路径 2：Conformalized Quantile-DeepONet（CQR for operators）**
1. 设计 Quantile-DeepONet：branch + trunk 双分量，分别输出 t_{α/2} 和 t_{1-α/2}
2. 用 pinball loss 训练：ρ_γ(y, ŷ) = γ(y-ŷ) if y-ŷ>0 else (1-γ)(ŷ-y)
3. Score function：s = max{t_{α/2}(u)(x) - G, G - t_{1-α/2}(u)(x)}
4. 类似地计算 calibration quantile q̂
5. CI：[t_{α/2}(u)(x) - q̂, t_{1-α/2}(u)(x) + q̂]

**路径 3：Multi-Fidelity Conformalized Prob-DeepONet**
1. 先训练经典 DeepONet 逼近低精度解 y_LF
2. 再训练 Prob-DeepONet 逼近残差 (y_HF - y_LF)，输出 N(μ(u)(x), σ(u)(x))
3. 最终模型：N(μ(u)(x) + ŷ_LF(u)(x), σ(u)(x))
4. 对 high-fidelity 数据做 split conformal calibration

### 关键技术细节

- **网络架构**：全部使用 FNN（feed-forward neural network），ReLU 激活
  - Pendulum: trunk 3(1) × 100, branch 3(1) × 100（括号内为独立层数）
  - Diffusion-reaction: trunk 4(1) × 100, branch 4(1) × 100
  - Burgers: trunk 5(1) × 128, branch 5(1) × 128
- **优化器**：Prob/Quantile-DeepONet 用 Adam（lr=1e-3, default β），ReduceLROnPlateau scheduler
- **B-DeepONet 采样**：stochastic gradient replica-exchange sampling（双链 parallel tempering），温度 τ 控制
- **训练数据量**：Pendulum N=5000, Diffusion N=10000, Burgers N=30000
- **Calibration 数据量**：n = N/10（500, 1000, 3000）
- **测试**：100 trajectories × 100 evaluation points = 10000 test samples per experiment
- **miscoverage rate**：α=0.05（target 95% coverage）
- **输入离散化**：m=100 sensors
- **B-DeepONet M-ensemble**：需 forward M 次（M 未明确给出，但提到成本高）
- **Prob-DeepONet**：四个网络组件（branch mean, branch log-std, trunk mean, trunk log-std），共享部分层以减少参数
- **Quantile-DeepONet**：四个网络组件（两个 quantile 的 branch + trunk），共享部分层
- **Multi-fidelity**：N_LF=3800, N_HF=760, n_cal=380
- **Score function 设计原则**：
  - B/Prob 用的 score = |residual| / σ 是 normalized absolute error
  - Quantile 用的 score 是 max over-coverage violation，违反 quantile 越远 score 越大

### 核心结果

| Experiment | Conf. Prob | Conf. Quantile | Baseline Prob | Baseline Quantile |
|------------|-----------|---------------|---------------|-------------------|
| Pendulum | **94.69%** | **95.86%** | 3.80% | 55.31% |
| Diffusion | **96.26%** | **95.64%** | 8.99% | 83.09% |
| Burgers | **94.88%** | **95.96%** | 7.84% | 72.91% |
| Multi-Fidelity | **97.02%** | N/A | 82.07% | N/A |

关键发现：
1. Conformalized 方法在所有实验中均达到接近 target 95% 的覆盖率，证明 coverage guarantee 有效
2. Baseline Prob-DeepONet 覆盖率极低（3.8%-8.99%），说明其 heuristic σ 完全不能直接当 CI 用——需要大量超参调优才能达到可接受的覆盖率
3. Quantile-DeepONet baseline 比 Prob-DeepONet baseline 好很多（55%-83% vs 4%-9%），但距离 95% 仍有差距
4. Conformalized Quantile-DeepONet 在多数实验中略微优于 Conformalized Prob-DeepONet，因为其 baseline 起点更高
5. 即使 baseline 很差（Prob-DeepONet 只有 7.84%），conformal calibration 仍能将其纠正到 94.88%——证明 split conformal prediction 的鲁棒性
6. Ablation I：置信区间长度分布呈多峰（非单一长度），证明 conformalized intervals 是 adaptive 的
7. Ablation II：calibration set size n > 500 即可稳定达到 95% coverage，n=1000 收益饱和

**Surprising/Observations：**
- Prob-DeepONet baseline 覆盖率仅 3.8% 极低，近乎失效——但作者此前论文却声称它提供了 UQ。这说明在没有 coverage guarantee 的情况下，heuristic UQ 可能是完全的幻觉。
- B-DeepONet 的覆盖率数据完全缺失——虽然在引言中被介绍为一种 UQ 方法，但实验中未报告其 baseline coverage，且 Conformalized B-DeepONet 的结果也未出现在任何表格中（只在文字提及）。这是一个明显的 omission。

### 明确局限

1. **Exchangeability 假设**：split conformal prediction 要求 calibration 和 test 数据 i.i.d.。作者明确指出这不适用于 DeepONet extrapolation、非自治系统、Federated DeepONet 等场景
2. **无真机/真实数据**：所有实验都是合成 ODE/PDE（pendulum, diffusion-reaction, Burgers, jump function），完全没有真实世界数据验证
3. **未与其他 UQ 方法比较**：未与 ensemble methods（如 MC Dropout、Deep Ensemble）、Gaussian Process、或其他 conformal 变体（如 Jackknife+）比较
4. **B-DeepONet 结果缺失**：虽然声称 conformal prediction 增强了 B-DeepONet，但实验结果表格中完全缺失 B-DeepONet 和 Conformalized B-DeepONet 的 coverage 数据
5. **无 coverage conditional on X**：只保证 marginal coverage，不保证 conditional coverage。在某些输入区域可能 coverage 远低于 1-α
6. **计算成本未量化**：Prob-DeepONet 比 B-DeepONet 更高效（无需 M 次 forward），但未给出具体 wall-clock 对比
7. **仅支持 scalar output**：所有实验输出都是 1D scalar function，未展示对 multi-output operator 的支持
8. **α 固定为 0.05**：未探究不同 α 下的行为（如 α=0.01, 0.1）
9. **Quantile-DeepONet 的 pinball loss 训练已知不稳定**（尤其在极端 quantile），作者未讨论训练稳定性问题和解决方案
10. **无代码开源声明**：论文未提及代码是否公开

## Read by: 0616-fex (2026-06-24)

这篇论文的核心价值在于提供了一种 model-agnostic 的 UQ 方法（split conformal prediction），可以与任意 DeepONet 变体结合。对于 0616-fex 课题，如果工作涉及 operator learning 且需要不确定性估计，这篇提供的方法是即插即用的轻量方案。

**撞车风险**：极低。本文是纯 UQ 方法，不涉及 foundation model / large-scale pre-training / multi-task operator learning。如果 0616-fex 的侧重点在架构或预训练，本文可作为"如何为此类模型加 UQ"的参考。

**可复用组件**：
1. Split conformal calibration pipeline（Algorithm 1/2）—— ~20 行 Python 即可实现，只需 calibration set
2. Score function 设计模式可复用到任何有 heuristic uncertainty 的模型
3. Pinball loss + Quantile-DeepONet 架构可作为 foundation operator model 的 UQ head

**注意点**：本文的 exchangeability 假设在 federated / continual / non-i.i.d. 设置下不成立，如果 0616-fex 涉及此类场景需要更复杂的 conformal 变体（如 adaptive conformal）。
