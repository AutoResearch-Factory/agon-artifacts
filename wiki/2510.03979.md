# Beyond Softmax: A New Perspective on Gradient Bandits (2510.03979)

- **Year**: 2025.10
- **Authors**: Emerson Melo (Indiana U Bloomington, Economics), David Müller
- **Venue**: arXiv (targeting Publ. Inst. Math.)
- **Keywords**: discrete choice, generalized extreme value (GEV), generalized nested logit (GNL), multi-armed bandits, gradient bandit algorithm, differential consistency, regret bounds
- **One-liner**: 将 discrete choice 理论中的 GNL 模型引入 MAB，把 Exp3 和 Gradient Bandit 从 softmax/MNL 推广到允许 action 间相关性的 GEV 家族，并给出遗憾界。

## 精华提炼

### 管线/方法
**整体框架**：GBPA (Gradient-Based Prediction Algorithm) + GEV surplus function 作为 potential function。

**输入 → 输出流程（adversarial MAB）**：
1. 初始化 `Û₀ = 0`
2. 每轮 t：根据 `x_t = ∇Ẽ(Û_{t-1}; η)`（即 GNL choice probability vector）采样 arm `i_t`
3. 观察损失 `u_t^(i_t) ∈ [-1, 0]`
4. 估计增益向量 `û_t = u_t^(i_t)/x_t^(i_t) · e_{i_t}`
5. 更新累积估计 `Û_t = Û_{t-1} + û_t`

**Generalized Gradient Bandit 流程**：
1. 初始化偏好 `u_0 = 0`，`R̄_0 = 0`
2. 每轮按 GNL choice probabilities 采样 `i_t`
3. 观察奖励 `R_t`
4. 更新：`u_{t+1} = u_t + α · [(R_t - R̄_t) · ∂x_t^(i_t)/∂u_t(i) / x_t^(i_t)]`
5. 更新基线 `R̄_{t+1} = (R_t - R̄_t)/t`

### 关键技术细节

**GEV 生成函数**（Definition 1）：满足 (i) 非负 (ii) 1次齐次 (iii) 每参数 coercive (iv) 交叉偏导正负交替。

**GNL 生成函数**：`G(x)= Σ_{ℓ∈L} (Σ_i (σ_{iℓ}·x^{(i)})^{1/μ_ℓ})^{μ_ℓ/μ}`
- `σ_{iℓ}`：arm i 属于 nest ℓ 的份额，Σ_ℓ σ_{iℓ} = 1
- `μ_ℓ`：nest 内部 variance 参数，需满足 `μ_ℓ ≤ μ`
- `μ`：nest 间 variance 参数

**Surplus function**：`E(u) = μ ln G(e^u)`，其梯度 = choice probability vector（Williams-Daly-Zachary theorem）

**Choice probability**：`P^(i) = μ · ∂G(e^u)/∂x^(i) · e^{u^(i)} / G(e^u)` = 两阶段乘积（nest 概率 × nest 内条件概率）

**关键条件 — Differential Consistency**（Def 4）：`∇²_{ii} f(U) ≤ C · ∇_i f(U)`，保证 Bregman divergence 有界从而 regret 可控。

**Theorem 2**：若 G 满足 `∂²G/∂x^(i)² · x^(i) ≤ C̃ · ∂G/∂x^(i)`，则 E 是 C-differentially-consistent，C = C̃ + 1。

**Theorem 3**（主定理）：GNL surplus function 是 `1/min μ_ℓ`-differentially-consistent。

**Proposition 1**：微分一致性条件 ⇒ 强光滑性条件（反向不成立），即适用于 bandit 的 GEV 子类 ≤ 适用于 full-feedback 的 GEV 子类。

**遗憾界**：
- Full-feedback: `R_T ≤ 2√(α·L·T)·K`，其中 L 是 smoothness parameter
- Adversarial MAB (GNL): `η·E(0) + n·T/(min μ_ℓ·η)`
- MNL 的 `E(0) = μ·ln(n)`（优于 Abernethy 的 `2ln(2n)` 界）
- NL 的 α: `min μ_ℓ·ln(n) ≤ α ≤ ln(n)`
- MNL smoothness: `L = 1/(μ·η)`; NL smoothness: `L = 2/(min μ_ℓ·η)`（MNL 更优）

**NL Gradient Bandit 的三阶段更新公式**（Algorithm 4）：
- 被采样的 arm i（在 nest N_ℓ）：`u_{t+1}(i) = u_t(i) + α·(R_t-R̄_t)·(1/μ_ℓ)·[1-(1-μ_ℓ)·x_t^{(i|ℓ)}-μ_ℓ·x_t^{(i)}]`
- 同 nest 但未被采样的 arm k：`u_{t+1}(k) = u_t(k) - α·(R_t-R̄_t)·(x_t^{(k)}/x_t^{(i)})·[x_t^{(i)}+(1-μ_ℓ)/μ_ℓ·x_t^{(i|ℓ)}]`
- 不同 nest 的 arm j：`u_{t+1}(j) = u_t(j) - α·(R_t-R̄_t)·x_t^{(j)}`（等同于标准 Gradient Bandit）

当 μ_ℓ = 1 时，所有更新退化为标准 Gradient Bandit。

### 核心结果

1. **理论统一**：GNL 是包含 MNL、NL、Paired Combinatorial Logit 等模型的广阔家族，统一了已知的 bandit 算法。Exp3 是 MNL surplus function 的特例。

2. **Exp3 的 Gumbel smoothing 分析改进**：由 GEV 视角重分析 Exp3 得到更紧的 `E(0) = ln(n)` 而非 `2ln(2n)`。

3. **数值实验（stochastic 设定下）**：
   - **MNL 环境**（n=10，随机均值）：NL 与 MNL 性能相当；过度结构化（NL2, μ_ℓ=0.3-0.45）反而变差
   - **NL 环境**（n=9，3个好arm + 6个差arm，结构化 nest）：所有 NL 变体均优于 MNL，NL3（μ_ℓ=0.45）最优
   - **NL Large 环境**（n=25，1个最佳 + 24个随机）：NL 在识别最佳 arm 上显著优于 MNL
   - NL 通过牺牲同 nest 内的弱 arm 探索来加速最佳 arm 的 exploitation

4. **Surprising/Mildly counter-intuitive**：NL 的 smoothness 参数比 MNL 差（`L_MNL = 1/(μ·η)` vs `L_NL = 2/(min μ_ℓ·η)`），理论上 regret 界更差；但实验中通过更好的 exploration-exploitation 权衡实际表现更优。

### 明确局限

1. **无真实数据实验**：所有实验均为合成数据（Gaussian mean rewards、Bernoulli bandits），无推荐系统、定价、临床试验等实际场景验证。
2. **无标准 benchmark**：未与 UCB、Thompson Sampling 等标准 MAB baseline 对比（仅与 MNL/Exp3 比较）。
3. **nest 结构需先验知识**：nest 划分和 μ_ℓ 需要预先指定。论文未讨论 nest 结构的自动学习或错误指定（misspecification）的影响。
4. **仅限于 stochastic 实验**：理论的 adversarial 遗憾界未在实验中验证，所有数值实验均为 stochastic 设定。
5. **无代码开源**：论文未提供代码仓库链接。
6. **μ_ℓ 调参敏感**：实验显示 nest parameter 选择显著影响性能，但无系统性调参指导。
7. **与我们的关系**：论文被 suppress 的 appendix（\iffalse 块）包含 stochastic MAB 的更多实验和 local convergence 分析，这部分未正式发表。
8. **关键 gap**：仅讨论了从 surplus function 导出 sampling 概率，未讨论 nest 结构的在线学习（online structure learning）。

## Read by: 0616-fex (2026-06-22)

本次审读视角：从 feature exploration (fex) 课题出发评估这篇理论 MAB 论文。

**核心洞察**：本论文解决的是 "action space 中存在先验相关性结构时，如何设计更好的 bandit 探索策略"。GNL 框架允许通过 nest partition 和 μ_ℓ 参数编码"哪些 action 是相似的"这一先验知识，并在采样和偏好更新中利用它。

**对 fex 的启示**：
1. 如果我们的 feature exploration 涉及在结构化 action space 中搜索（如不同 skill/tool 的组合），GNL 提供了一种 principled way 来编码 action 相关性
2. "信息共享"机制（同 nest 内 arm 间传播奖励信号）可以加速对子空间的 exploration，类似于 feature-based exploration 中的泛化
3. 微分一致性概念可作为检验我们设计的 exploration policy 是否理论可靠的快速 check

**差距**：本文假设 nest 结构已知且固定。fex 场景的关键挑战是 nest 结构（哪些 feature/action 相似）本身是未知的、需要在线学习的。这构成了一个自然的研究延伸方向：online GNL structure learning for feature exploration。
