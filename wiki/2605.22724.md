# Multiple Neural Operators Achieve Near-Optimal Rates for Multi-Task Learning (2605.22724)

- **Year**: 2026.05
- **Authors**: Adrien Weihs, Hayden Schaeffer (Department of Mathematics, UCLA)
- **Venue**: arXiv preprint
- **Keywords**: Multi-Task Learning, Operator Learning, Neural Scaling Laws, Approximation Theory, Minimax Rates, Curse of Parametric Complexity, Multi-Operator Learning, DeepONet
- **One-liner**: Proves that multi-operator learning with MNO obeys the same near-optimal iterated-exponential scaling laws as single-operator learning, and establishes matching minimax lower bounds showing an unavoidable curse of parametric complexity.

## 精华提炼

### 管线/方法

**问题设定**：学习一个 multiple operator map `G: W → {G[α]: U → V}_{α∈W}`，即参数化的算子族。输入是算子标识符 α（函数空间的元素），输出是一个算子，该算子再将输入函数 u 映射为输出函数。

**MNO 架构**（可分离架构）：
```
MNO[α][u](x) = Σ_{p=1}^P Σ_{k=1}^H Σ_{ℓ=1}^N θ_{pkℓ} · l_p(M_W(α)) · b_k(M_U(u)) · τ_ℓ(x)
```
其中 `l_p`, `b_k`, `τ_ℓ` 分别编码对 α 的无限维依赖、对 u 的无限维依赖、对 x 的有限维评估。`M_W: W→ℝ^m`, `M_U: U→ℝ^q` 是线性映射（离散化采样）。

**证��技术——并行 vs 嵌套**：
- **旧方案（嵌套）**：Stage 1 近似 α 依赖产出 ~ε^{-d_W} 个项。Stage 2 对每个项做算子近似，误差逐项累加 → 总误差 = ε^{-d_W}·δ。为使总误差 ≤ ε，需要 δ ≈ ε^{d_W+1}（指数级收紧），导致 triple-exponential 参数爆炸。
- **新方案（并行）**：Stage 1 产出的 l_p 系数构成近似 partition of unity（Σ l_p ≈ 1）。Stage 2 误差以 θ_p 系数加权求和 → Σ θ_p·δ ≈ δ（因为 Σ θ_p ≈ 1）。因此只需 δ ≈ ε 即可，消除了嵌套带来的指数膨胀。

**构造步骤**：
1. 固定 u, x，将 `f_{u,x}(α) = G[α][u](x)` 视为 α 的泛函。对覆盖 Ω_W 的采样点 {y_m}，用网络族 `l_p` 逼近，得到 ε_0 近似误差的离散化表示。
2. 对每个固定的 α_p，将 `G[α_p]: U→V` 视为标准算子学习问题。覆盖 Ω_U 和 Ω_V，用 `b_k`, `τ_ℓ` 网络逼近。
3. 两阶段组合：利用 l_p 的 partition of unity 性质，总误差 ≤ ε_0 + ε_1·(1 + ε_0/η) ≤ ε。

### 关键技术细节

- **输入空间假设**：W, U, V 均为有界 Lipschitz 函数类，具体地 `W(d_W, γ_W, L_W, β_W)` 表示 d_W 维定义域、[−γ_W, γ_W]^d_W 上、L_W-Lipschitz、∥·∥_∞ ≤ β_W。
- **算子假设**：G 满足两部分 Lipschitz 条件 — (O1) G[α] 对 u 是 L_𝒢-Lipschitz（∥·∥_{r_𝒢} 范数）；(O2) G 对 α 是 L_G-Lipschitz（∥·∥_{r_G} 范数）。
- **子网络架构参数**（达到 ε 近似误差时）：
  - τ_ℓ（空间逼近）：# 网络 ~ ε^{-d_V}，depth ~ log(ε^{-1})，sparsity ~ log(ε^{-1})，参数幅度 ~ ε^{-(d_V+1)}。显著简化为有限维逼近。
  - b_k（输入函数逼近）：# 网络 ~ ε^{-ε^{-d_U}}，depth ~ ε^{-2d_U}，sparsity ~ ε^{-2d_U}，参数幅度 ~ ε^{-ε^{-d_U}}。
  - l_p（参数变量逼近）：# 网络 ~ ε^{-ε^{-d_W}}，depth ~ ε^{-2d_W}，sparsity ~ ε^{-2d_W}，参数幅度 ~ ε^{-ε^{-d_W}}。
- **总参数数**：N_# ≲ ε^{-d'' ε^{-max{d_W, d_U}}} → 反向求解得 ε ≲ (log N_# / log log N_#)^(-1/max{d_W, d_U})。
- **Lower bound 构造**：利用 Lanthaler & Stuart (2023) 的无限维超立方体框架。定义 [0,1]^∞ 上的系数衰减率 η，构造病态 Fréchet 可微泛函 F:W→ℝ，提升为 rank-1 多算子 G[α][u](x) = F(α)φ(x)。任何低复杂度逼近都会导出 F 的低复杂度逼近，矛盾。
- **超立方体嵌入条件**：对于 Assumption S.4 下的有界 Lipschitz 类，对任意 η > 1 + 1/d，均包含无限维超立方体 Q_η。这使用了 sin(κ·x) 作为正交基，并用紧凑性论证。
- **复制度量**：对可分离架构，`C(Φ) ≤ 2(∥Θ∥_0 + HK_2 + NK_1 + PK_3)`，通过归约为多重算子神经网络类型的对称定义。
- **学习设定**：层次化采样 — n_α 个算子标识符、每个对应 n_u 个输入函数、每个对应 n_x 个评估点。使用 clipped network class 和 empirical L² loss。
- **泛化误差分解**：总误差 ≤ 4ε² + η(8σ+6) + metric entropy 项 / √(n_α n_u n_x) + metric entropy 项 / (n_α n_u n_x) + metric entropy 项 / n_α。关键在于 metric entropy 由近似理论的架构参数控制。

### 核心结果

| 指标 | 旧结果 (weihs2025MOL) | 新结果 (本论文) |
|------|----------------------|----------------|
| 近似误差 vs N_# | ε ≲ (log log N_# / log log log N_#)^(-1/d_W) | ε ≲ (log N_# / log log N_#)^(-1/max{d_W, d_U}) |
| 参数数 vs ε | N_# ≲ ε^{-ε^{-ε^{-d_W}}} | N_# ≲ ε^{-d'' ε^{-max{d_W, d_U}}} |
| 泛化率 | (log log n_α / log log log n_α)^(-2/d_W) | (log n_α / log log n_α)^(-2/max{d_W, d_U}) |

**Minimax 率**（定理 3.4）：
```
exp(c · ε^{-1/[(η+1+δ)r]}) ≲ 𝕮(ε; 𝓗) ≲ exp(d · log(ε^{-1}) · ε^{-max{d_W, d_U}})
```
上下界同为 double-exponential 型，证明存在不可避免的 parametric complexity curse。

**Surprising/反直觉结果**：
1. MNO 和 concatenated DeepONet 在最坏情况 minimax 率上**无差别**：两者共享完全相同的 scaling law 形式。但 MNO 在实验上明显优于 concatenated DeepONet (weihs2025MOL Section 5)。这表明 empirical advantage 来自更精细的结构/几何/数据依赖特性，而非最坏情况复杂度。
2. 旧构造中 b_k 子网络占了几乎所有复杂度（triple-exponential），新构造中 l_p 和 b_k 复杂度均衡（均为 double-exponential），而 τ_ℓ 大幅简化。这意味着无限维依赖（α 和 u）是复杂度瓶颈，有限维 x 逼近是最简单的部分。

### 明确局限

1. **作者声明的局限**：
   - Lipschitz 假设（O1, O2）是 minmal regularity，但仅靠 Lipschitz/可微性无法克服 curse of parametric complexity。需要 holomorphic 结构或 PDE-specific regularity 才能获得更好的率。
   - 上下界在 exponent 层面不紧：上界有更大的 ε 幂次和额外的 log 因子。
   - 尚未研究 holomorphic 多算子或基于数值方案的架构设计在 multi-operator setting 中的潜力。
   - MNO vs concatenated DeepONet 的 minimax 等价性暗示需要超越最坏情况分析的理论框架。

2. **审读者发现的局限**：
   - 没有实验验证：纯理论论文，没有 numerical experiments 支撑 scaling law 的实际行为。
   - 假设空间限于有界 Lipschitz 类（W, U, V 均有紧支集且等度连续），这对 PDE 应用合理但不涵盖无界域或无界算子。
   - 离散化采样点 {y_s}, {c_s} 的构造是存在性的（通过覆盖数论证），实际如何最优选择不清楚。
   - 下界构造是"病态函数"论证（pathological construct），未说明这类坏函数是否在典型应用中常见。
   - Clipping 操作在统计部分引入但假设 boundedness（β_V），实际 loss 中如何选择 clip level 是超参数。
   - 泛化分析假设 noise 是 i.i.d. sub-Gaussian，未讨论异方差或分布偏移。
