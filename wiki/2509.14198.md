# A Variational Framework for Residual-Based Adaptivity in Neural PDE Solvers and Operator Learning (2509.14198)

- **Year**: 2025.09
- **Authors**: Juan Diego Toscano, Daniel T. Chen, Vivek Oommen, Jerome Darbon, George Em Karniadakis (Brown University, Division of Applied Mathematics)
- **Venue**: arXiv (elsarticle journal submission format)
- **Keywords**: physics-informed learning, PINNs, neural operators, adaptive sampling, PDEs, variational framework, residual-based attention, importance sampling, signal-to-noise ratio, L-infinity minimization
- **One-liner**: 用变分公式统一 SciML 中的残差自适应采样/加权方法，指数势对应 L∞ 最小化，二次势对应 L² 最小化，在 PINNs 和 operator learning 上都获得显著提升。

## 精华提炼

### 管线/方法

整体框架将 SciML 训练重构为变分优化问题：

1. **问题设定**：定义残差 r(x;θ) — 在 PINNs 中是 PDE 残差 |F[u]|，在 operator learning 中是预测与真值之差 |G_θ(v)(x) - G[v](x)|
2. **原始优化目标从 L² 提升到 L∞**：从 min_θ E_p[r²] 提升到 min_θ max_x r(x)（即 min_θ ||r||_L∞）
3. **对偶化**：利用 Laplace 原理 + Gibbs 变分公式，将 L∞ 最小化改写为 min_θ max_q {E_q[r] - ε H(q|p)}
4. **交替优化**：每轮迭代三步 — (a) 更新倾斜分布 q ∝ exp(r/ε) p，(b) 用 line search 更新 θ，(c) 用退火方案减小 ε
5. **离散化实现**：通过重要性采样（resample）或重要性加权（reweight）两种方式施加自适应分布
6. **扩展到 operator learning**：混合策略 — 函数空间用重要性采样，空间域用重要性加权

### 关键技术细节

**变分框架核心**：
- Laplace 原理 (Varadhan's lemma)：max_x r(x) = lim_{ε→0} ε log ∫ e^{r/ε} dp
- Gibbs 变分公式：log ∫ e^r dp = sup_q {E_q[r] - H(q|p)}，最优 q* ∝ e^r p
- 推广到一般 Φ-散度：用凸共轭 Φ* 替代 KL 散度，对应不同原始目标

**两种势函数**：
- Φ(r) = e^r（指数势）：精确对偶，原始目标为 L∞ 范数。q ∝ softmax(r/ε)，退火方案 ε^k = c·max(r)/(log(2+k))
- Φ(r) = r²+1（二次势）：非精确对偶，原始目标为标准差 √(E[r²]-E[r]²)。通过选择 ε^k 为归一化常数，q ∝ |r|/∑|r|

**平滑机制**：
- EMA 平滑：λ_i^{k+1} = γ λ_i^k + η* q^{k+1}(X_i)，γ∈[0,1)，η* = η/max q
- Uniform 插值：λ = γλ + η*(φ q + (1-φ) p_u)，φ∈[0,1] 控制自适应程度

**训练架构**：
- 一阶优化：Adam，lr=1e-3，decay rate 0.9/5000 step，Fourier feature embedding degree 10
- 二阶优化：5000 Adam 预热 + SSBroyden 主训练，每 100 iter 重采样 collocation points
- 6层 MLP (hidden dim 64) 用于一阶，3层 MLP (hidden dim 30) + 周期性编码用于二阶
- 全局权重自缩放 (self-scaling)：m_B^k = α m_B^{k-1} + (1-α)||∇L_E||/||∇L_B||

**Operator learning 混合策略**：
- DeepONet: branch 4层 + trunk 4层，各 hidden dim 100，GELU 激活，101K 参数
- FNO: 1.62M 参数，128×128 空间网格，80:10:10 数据分割
- TC-UNet: 2.43M 参数，FiLM 时间条件调制
- 空间权重矩阵 Q ∈ R^{N×N_func}，按函数归一化
- 函数级采样分布从聚合的 Λ 分数构建：s_j = Σ_i Λ_{i,j}，q̄_j = s_j/Σ s_j

**自实现 JAX SSBroyden**：
- 原版依赖 SciPy CPU 例程，无法 GPU 加速
- 自实现保留核心更新逻辑（动态计算 τ_k, φ_k）
- 自定义三阶段回退线搜索：c₂=0.9 → c₂=0.8 → c₂=0.5

### 核心结果

**Allen-Cahn 方程**：
| 配置 | 方法 | 参数量 | 优化器 | Rel. L² Error |
|------|------|--------|--------|---------------|
| 大网络+一阶 | Baseline | 21318 | Adam | 1.77×10⁻³ |
| 大网络+一阶 | vRBA Φ=r² | 21318 | Adam | **4.08×10⁻⁴** |
| 大网络+一阶 | vRBA Φ=e^r | 21318 | Adam | 4.14×10⁻⁴ |
| 小网络+二阶 | Baseline | 2011 | SSBroyden | 4.34×10⁻⁵ |
| 小网络+二阶 | vRBA Φ=r² | 2011 | SSBroyden | 2.42×10⁻⁶ |
| 小网络+二阶 | vRBA Φ=e^r | 2011 | SSBroyden | **2.14×10⁻⁶** |

**Burgers 方程 (SSBroyden, 2011 params)**：
- Baseline: 1.67×10⁻⁷ → vRBA Φ=e^r: **1.51×10⁻⁸** → SOTA

**Operator learning (all Adam)**：
| 问题 | 架构 | Baseline | vRBA best |
|------|------|----------|-----------|
| Bubble Growth | DeepONet | 7.41×10⁻² | **4.97×10⁻³** |
| Navier-Stokes | FNO | 5.13×10⁻² | **2.25×10⁻²** |
| Wave Equation | TC-UNet | 3.69×10⁻² | **1.00×10⁻²** |

**Surprising / 反直觉发现**：
1. 二阶优化器 + vRBA 的组合效果远超单独使用任一个 — vRBA 的收益不与优化器选择重叠
2. FNO 的 SNR 从头到尾都高，说明 Fourier features 提供的 spectral bias 让模型直接进入 productive diffusion 阶段
3. 指数势和二次势在多数实验中表现接近，但二次势在 TC-UNet 上更好（1.00e-2 vs 1.05e-2），说明 L∞ 最小化不是万能的
4. vRBA 最关键的效果不是提高绝对 SNR，而是加速从 transition 到 diffusion 的跃迁 — SNR 的 spike 比 sustained high SNR 更重要

**SNR 与学习三阶段分析**：
- Fitting: 高 SNR, 快速降低 training error, geometric complexity 上升
- Transition: 低 SNR, 几乎没有泛化提升, 子域梯度冲突
- Diffusion: SNR 突然升高, 泛化误差快速下降, geometric complexity 下降
- vRBA 通过降低梯度估计方差使模型更快进入 diffusion 阶段

### 明确局限

**作者明确指出的局限**：
1. 一般 Φ-散度的最优化子没有闭式解 — 当前只有指数势和二次势有实用方案
2. 对一般 Φ 的原问题目标函数 Λ_ε 缺乏直观解释（仅与 Orlicz 空间有推测性联系）
3. vRBA 与 simulated annealing 的形式联系未严格证明 — 作者明确说 "do not claim rigorous equivalences"
4. 论文未对一般 Φ 进行详细的数值研究，留给 future work
5. 从 annealing scheme 的 logarithmic decay 借鉴了 Geman & Geman (1986)，但缺乏在 SciML 中优化的严格 guarantee

**审读发现的局限**：
1. 所有实验都是标准 benchmark（Allen-Cahn, Burgers, NS, Wave），没有真机实验或真实物理数据
3. 没有与最新的 adversarial 自适应方法（如 gPINN）做对比
4. SSBroyden 的自实现 JAX 版本与原始 SciPy 版本的一致性未经过严格测试
5. 论文未系统分析 vRBA 对过拟合的影响 — TC-UNet 中指数势方差后期上升提示可能存在过拟合
6. 所有实验的计算开销增加很小（<10% per iteration），但未分析整体 wall-clock time 变化
7. vRBA 的超参数（γ, η, φ, λ_max0, λ_cap）较多，对超参数选择的指导不充分
8. 只在 operator learning 用了混合策略，没有在 PINNs 中尝试函数空间的采样
