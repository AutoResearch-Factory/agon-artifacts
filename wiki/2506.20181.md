# Causal Operator Discovery in Partial Differential Equations via Counterfactual Physics-Informed Neural Networks (2506.20181)

- **Year**: 2025.06
- **Authors**: Ronald Katende, Kabale University
- **Venue**: arXiv
- **Keywords**: causal PDE discovery, counterfactual PINNs, operator identification, sparse residual learning, causal sensitivity index, structural identifiability, PDE-FIND, scientific machine learning
- **One-liner**: 用反事实 PINN 做 PDE 算子级因果发现，但实验证据可信度偏弱。

## 精华提炼

### 管线/方法
- 论文把 PDE 的 governing operator `N[u; gamma] = 0` 解释为结构因果机制，而不是固定物理先验。
- 内生变量包括解场 `u` 和物理参数 `gamma_j`；外生变量 `U_j` 通过函数 `f_j(U_j)` 影响参数。
- 候选算子库写成 `Gamma = {T_1, ..., T_m}`，目标是从过完备库中恢复真实 causal support。
- 基本残差形式是 `R[u_theta] = sum_j alpha_j T_j[u_theta]`。
- 神经代理 `u_theta(x,t)` 用全连接网络表示，文中实验设置为 4 hidden layers、每层 128 neurons、sine activation。
- 所有候选微分算子通过 automatic differentiation 在训练点上求值。
- 训练目标由三部分组成：data fidelity、PDE residual、`L1` sparsity。
- 目标函数核心形式是 `L(theta, alpha) = data_loss + lambda_r ||R_theta||^2 + lambda_s ||alpha||_1`。
- 文中给出的正则权重是 `(lambda_r, lambda_s) = (1.0, 0.01)`。
- 优化使用 Adam，learning rate `1e-3`，并在 `theta` 和 `alpha` 之间交替更新。
- `theta` 的更新负责拟合解场，`alpha` 的更新负责选择 sparse operator support。
- `alpha` 更新描述为 proximal/soft-thresholding，用来压低非必要算子系数。
- 训练后先取 `supp(alpha)` 得到候选 active terms。
- 然后对每个算子执行结构干预：设 `alpha_j -> 0`。
- 对干预后的 PDE 重新计算或重训 counterfactual surrogate `u_theta^cf`。
- 算子重要性不只看 residual，而看删除该算子后解场是否发生实质偏移。
- 核心反事实偏移指标是 `delta_j = ||u_theta^cf - u_theta||_{L2(Omega x (0,T))}`。
- 若 `delta_j` 小于阈值，论文认为该算子不是 causally identifiable，即便它在残差里看起来显著。
- 论文还定义了 CSI：`CSI_j = ||R - R_(-j)|| / (||R|| + delta)`。
- CSI 是残差层面的快速 proxy，counterfactual deviation 是更接近结构必要性的验证。
- 论文明确指出 CSI 或 residual influence 可能误判正交/零空间项，因此必须配合 counterfactual validation。
- 方法输出包括 learned solution field、learned sparse operator set、CSI ranking 和 counterfactual solution。

### 关键技术细节
- 理论部分把 PDE 写成 deterministic SCM，并把 operator removal 对应到 `do(alpha_j = 0)`。
- 反事实 PDE 写作 `N[u^cf; gamma'] = 0`，允许修改参数、source term、constitutive law、初边值条件。
- 反事实 surrogate 的损失包含 `||R_cf[v_theta]||^2`，如有反事实观测则加 data fidelity。
- Causal derivative 定义为 `lim_delta (u(gamma + delta e_j) - u(gamma)) / delta`。
- 线性化后得到近似关系 `delta u / delta gamma_j = -(dR/du)^{-1} dR/dgamma_j`。
- 对 scalar observable `O[u]`，文中引入 adjoint sensitivity：`dO/dgamma_j = - <phi, dR/dgamma_j>`。
- 残差误差界要求 weak form continuous/coercive，给出 `||v_theta - u^cf||_{H1} <= alpha^{-1} ||N[v_theta; gamma']||_{H-1}`。
- Sparse recovery 保证要求 design matrix `A_ij = T_j[u](x_i,t_i)` 满足 RIP 或 mutual coherence。
- 文中使用条件包括 `delta_{2s} < sqrt(2)-1` 或 `mu < 1/(2s-1)`。
- 支持恢复 theorem 声称在足够小的 `lambda > 0` 下恢复 `supp(alpha*)`。
- 这些 sparse recovery 条件来自 compressed sensing，但在真实 PINN 联合训练场景中是否满足没有经验检验。
- 基准任务包括 1D reaction、2D advection-diffusion、3D incompressible Navier-Stokes、Maxwell、orthogonal perturbation。
- benchmark 表给出数据规模：1D reaction `N=500`，2D advection-diffusion `N=2500`，3D Navier-Stokes `N=10000`，Maxwell `N=3200`，orthogonal perturbation `N=300`。
- 3D Navier-Stokes 设定含 `1% Gaussian` noise，其他合成表格多为无噪声。
- 真实数据案例宣称使用 TCIA PET tumor concentration、ERA5 Uganda temperature、HYCOM Mozambique channel ocean current。
- Tumor 案例候选库是 `{Delta u, u, u^2}`，训练点 `512`，噪声 `5% Gaussian`。
- Tumor 表报告 MAE `0.012`、relative L2 error `0.028`、support `{Delta u, u^2}`、FPR `0%`。
- Tumor CSI：`Delta u = 0.91`，`u^2 = 0.84`，`u = 0.03`。
- Uganda climate 第一段候选算子包括 anisotropic diffusion、advection、external forcing、thermal memory、hyperdiffusion、topographic modulation。
- Uganda climate 后续又给出另一套候选库 `{partial_t u, partial_x u, partial_y u, Delta u, u, u^2, partial_xy u}`。
- Uganda 数据被描述为 2015-2016 El Nino，另处又说 30-day span、12 irregular locations。
- Uganda recovered PDE 写成 `partial_t u = 0.87 Delta u + 0.41 partial_y u - 0.23 u`。
- Mozambique ocean 候选库包括 `u, u^2, u^3, partial_x u, partial_y u, Delta u, v_x partial_x u, v_y partial_y u`。
- Mozambique recovered coefficients：`nu ~= 1.3e-4`，`v_x ~= 0.09`，`v_y ~= -0.06`，`alpha_1 ~= -0.01`。
- 汇总表声称 Causal PINN 在 tumor/climate/ocean 均达到 `100%` support accuracy，CSI-AUC 分别 `0.98/0.96/0.95`。

### 核心结果
- 论文最重要的 conceptual claim：残差拟合不等于因果相关，operator discovery 需要反事实删除验证。
- 合成任务里，Causal PINN 被报告能从过完备候选库恢复真实 sparse operators。
- 在 orthogonal perturbation counterexample 中，残差项非零但 solution deviation 为零，说明 residual influence 会产生 spurious causal attribution。
- 该 counterexample 是全文最有价值的论证：它直接指出 PINN/PDE-FIND 类方法的结构性盲点。
- 标准 PINN 的问题：需要预设 `N` 或不能识别 `alpha`，即便拟合好也不知道真实结构。
- DeepONet 的问题：学习输入输出 solution operator，不显式使用 PDE design matrix，因此没有 operator-level interpretability。
- Causal PINN 的增量：在 residual loss 里联合优化 `alpha`，再用 `alpha_j -> 0` 的 counterfactual deviation 做二次筛选。
- 论文声称相比 PINN/DeepONet，Causal PINN 同时具备 structural loss、sparsity prior、counterfactual validation。
- 真实案例结果非常强：三类应用全部 `100%` support recovery，这一点反而需要谨慎看待。
- Surprising/反直觉结果：Tumor 案例中线性增长 `u` 被判无关，而 `Delta u` 与 `u^2` 被判 active；这与常见 reaction-diffusion proliferation 写法不完全一致，需要更多数据说明。
- Surprising/反直觉结果：Climate 案例前后 active support 不一致，一处说 `{diffusion, advection, forcing}`，另一处说 `{Delta u, partial_y u, u}`，说明实验叙述存在混杂。
- Surprising/反直觉结果：汇总表中真实数据的 support accuracy 全为 `100%`，但真实系统的 ground-truth support 通常不可直接获得。
- 论文没有展示代码、数据 split、训练曲线、seed-level variance、统计置信区间或独立复现实验。

### 明确局限
- 作者承认 residual-based causal influence 与 CSI 可能被正交项、null-space components、operator aliasing 欺骗。
- 作者承认 3D Navier-Stokes 在噪声和有限观测下只是 partial recovery。
- 论文的理论保证依赖线性独立、RIP/mutual coherence、coercivity、dense surrogate 等强假设。
- 这些假设在 nonlinear PDE、真实气候/海洋数据和联合训练 PINN 中没有被验证。
- 真实数据部分缺少可复现细节：数据下载版本、预处理、网格、边界条件、训练/验证 split、单位、normalization 都不足。
- 真实数据的 "true operators" 和 "support accuracy" 如何定义不清楚，尤其是 climate 和 ocean。
- 论文没有给 GitHub/code/artifact 链接，无法快速复现实验。
- 公式和实验叙述有若干不一致，例如 Eq.~XX 占位、Uganda 两套候选库、Casual/Causal 拼写混用。
- 部分证明把 sparse recovery 的线性设计矩阵结论直接迁移到 PINN 联合非凸优化，论证跨度较大。
- 部分例子中 "solution remains valid" 或 orthogonality 推导看起来过于理想化，需要数学细查。
- 图像只在源码包中以 png 给出，tex 中没有生成脚本或数值表来源。
