# Risk-sensitive Reinforcement Learning Based on Convex Scoring Functions (2505.04553)

- **Year**: 2025.05
- **Authors**: Shanyu Han (Peking Univ.), Yang Liu (CUHK Shenzhen), Xiang Yu (HK PolyU)
- **Venue**: arXiv preprint
- **Keywords**: risk-sensitive RL, convex scoring function, Actor-Critic, Expected Shortfall, augmented state, time-inconsistency, two-stage optimization, statistical arbitrage
- **One-liner**: 提出基于凸评分函数的通用风险敏感 RL 框架，通过增广状态空间将时间不一致问题转化为两阶段优化，设计了 AC 算法并给出逼近理论保证，无需 MDP 连续性假设。

## 精华提炼

### 管线/方法

1. **问题建模**：目标是最小化累积成本的风险度量 ρ(Σ c_t)，其中 ρ 由凸评分函数 f(y,υ) 和变换函数 h 参数化：ρ(Y) = inf_{υ} h(E[f(Y,υ)], υ)
2. **时间不一致处理**：将原始问题 inf_{π} ρ(Σc_t) 重写为两阶段优化 inf_{υ} h(inf_{π} E[f(Σc_t, υ)], υ)
3. **增广状态**：引入 Y_t = -Σ_{τ=0}^{t-1} cost_τ 作为增广状态，从而内层动态优化在扩大的状态空间 (t, s, y) 上变得时间一致，可使用 Bellman 方程
4. **AC 算法**：
   - Critic 步：用 NN V^φ 逼近价值函数，通过式 (recursive) 的回归目标训练（MSE loss）
   - Actor 步：policy gradient 无需 simulation-upon-simulation（与 tamar2016sequential, CJ2024 等不同），梯度为 ∇_θ log π^θ · V^φ
   - 外层 υ 选择：SGD 求解 min_{υ} h(E[V^φ_{0,υ}], υ)
   - 高效采样：交替极小化 — υ 从 N(υ*_{n-1}, σ²_n) 采样，σ_n 和更新频率 L(n) 随 n 递减
5. **ES 特例**：对 ES 可进一步简化，υ* 直接用模拟总成本的 α-分位数更新

### 关键技术细节

- **价值函数定义**：V^π_{t,υ}(s,y) = E_π[f(Σ_{k=t}^{T-1} c_k - Y_t, υ) | S_t=s, Y_t=y]，终端 V^π_{T,υ}(s,y) = f(-y, υ)
- **Bellman 递归**：V^*_{T,υ}(s,y) = f(-y, υ)；V^*_{t,υ}(s,y) = inf_{a} ∫ p(s'|s,a) V^*_{t+1,υ}(s', y-c(s,a,s')) dν_S(s')
- **网络输入**：NN 同时以 (t, υ, s, y) 为输入，输出连续。价值网络 V^φ 和策略网络 π^θ 均为全连接多层前馈网络
- **训练参数**：Adam optimizer, batch size B, epochs K, 每 L 步更新一次 υ*
- **不要求 MDP 连续性**：Theorem 1 (Bellman) 和 Theorem 2 (t1, 价值函数逼近) 均不假设 p(s'|s,a) 或 c 的连续性——这是与 bauerle2011markov, bauerle2022markov, CJ2024 等工作的关键区别
- **Assumption 1**：仅要求 f 在 D×R 上连续，h 在 f(D,R)×R 上连续（当 MDP 不连续时）。若 MDP 满足连续性（Assumption C of bauerle2011），则可去除此假设并支持经典评分函数
- **Proposition 2**：在温和条件下可证明 υ 的范围可限制到紧集 [m, M]
- **Lemma (Lipschitz)**：存在 Δ>0 使得 |E[f(Y,υ₁)] − E[f(Y,υ₂)]| ≤ Δ|υ₁−υ₂|，关键用于 propmin 的连续性证明

### 核心结果

- **Theorem 1 (Bellman)**：不假设 MDP 连续性下，存在最优价值函数序列满足 Bellman 递归，存在策略序列使其价值函数逼近最优
- **Theorem 2 (t1, 价值函数逼近)**：固定策略 π^θ 下，存在 NN 参数 φ 使 V^φ 在 (s,y,υ) 上以任意精度逼近真实 V，逼近误差的测度可任意小
- **Theorem 3 (t2, υ 选择)**：若 θ 为 γ-最优参数且 φ 逼近充分好，则学习的 υ*(θ,φ) 到真实最优集 U* 的距离 d(U*, υ*) ≤ γ
- **Theorem 4 (thm:am_converge)**：交替极小化采样方案在某些正则条件下收敛到全局最优
- **Proposition (thm:limit, ES 特例)**：ES 目标下，交替用分位数更新 υ* 的收敛性成立
- **实验（统计套利）**：
  - RL-mean 达到最低 mean cost (-0.524)，但 ES₀.₈ (0.253) 和 ES₀.₆ (0.086) 并非最低
  - RL-ES₀.₈ 达到最低 ES₀.₈ (0.215)，RL-ES₀.₆ 达到最低 ES₀.₆ (0.079)
  - RL-Var 达到最低 Variance (0.030)，RL-Mean-Var 达到最低 Mean-Var utility (-0.150)
  - 相比 RL-OneStepES (CJ2024)，本方法 ES 值更低，尾部更薄
  - 策略可视化：风险厌恶越强买卖边界越陡，随 y 增大策略更保守，临近到期时策略趋于平仓

### 明确局限

- **不是真机实验**：所有实验均为仿真（OU 过程 + 统计套利 MDP），无真实金融市场数据
- **无标准 benchmark 对比**：未与广泛使用的 RL 风险敏感基准（如 RAINBOW、PPO with risk）做系统对比
- **小规模问题**：T=5, q_max=5, a_max=2，规模极小，未验证扩展性
- **无连续 MDP 下的最优策略存在性**：Theorem 1 仅保证 ε-逼近策略序列存在，不保证严格最优策略存在（除非附加连续性假设）
- **Assumption 1 强依赖**：当 MDP 不连续时，必须假设 f 和 h 的连续性——对某些实际风险度量可能不成立
- **交替极小化收敛条件强**：Theorem 4 (thm:am_converge) 需要 δ 函数的存在性，实际验证困难
- **仅考虑有限 horizon T**：未讨论无限 horizon 推广
- **无 regret bound**：仅提供逼近误差界，未给出 regret/样本复杂度分析
- **单维辅助变量**：框架限于 υ∈R，未扩展到多维风险参数

### 撞车评估
本文的核心贡献是凸评分函数 + 增广状态 + AC 算法的 risk-sensitive RL 统一框架。与 `0616-fex` 课题的潜在撞车点：
- **直接撞车场景**：如果我们的课题同样以 convex/consistent scoring function 为建模核心，且同样处理 time-inconsistency + augmented state，则 claim 层面近乎重叠。但是本文的重点在于「不要求 MDP 连续性的理论分析」和「辅助变量采样效率」，如果我们从不同角度切入（如不同的求解器、不同的 risk decomposition、更 real-world 的 benchmark），差异仍然成立
- **不撞车场景**：如果我们使用 scoring function 但以完全不同的方法论（如 distributional RL、variational inference、model-based planning）来求解，差异化显著

### 可借用的资产
1. **增广状态技巧**：Y_t = -累积成本，这个简单的技巧有效化解了静态风险度量的时间不一致难题
2. **不需要 MDP 连续性的证明策略**：在 propmin 中使用了换序技巧（inf 和 lim 交换），而非依赖连续性紧性论证——这种证明模式对其他非连续 MDP 场景也有参考价值
3. **交替极小化采样**：采样 υ ~ N(υ*, σ²_n) 比均匀采样更高效，收敛性在温和条件下成立
4. **Section 4.2 的 scoring function 分类**：提供了从 OCE 到 EVaR 到 MAD/Variance 的 scoring function 构造方法，可以作为研究扩展的起点

### 建议策略
- 关注本文未覆盖的 scoring function 类型（如 expectile、inter-quantile range 等 elicitable but non-convex 的度量）
