# A guided residual search for nonlinear state-space identification (2602.22964)

- **Year**: 2026.02
- **Authors**: Merijn Floren and Jan Swevers; KU Leuven / Flanders Make@KU Leuven
- **Venue**: arXiv; accepted for IEEE Control Systems Letters (L-CSS)
- **Keywords**: nonlinear state-space identification, NL-LFR, guided residual search, multiple shooting, distribution shift, Silverbox, F-16 ground vibration test, JAX/CasADi
- **One-liner**: 用 GRS 初始化 NL-LFR，再用多重射击稳健优化。

## 精华提炼

### 管线/方法

1. 输入是零均值、单位方差的输入输出序列 `D={(u(n), y(n))}_{n=0}^{N-1}`。
2. 目标是识别 nonlinear linear fractional representation (NL-LFR) 形式的非线性状态空间仿真模型。
3. 模型把主动态拆成 LTI 主体和静态神经网络非线性反馈。
4. 状态更新为 `x(n+1)=A x(n)+B_u u(n)+B_w w(n)`。
5. 输出为 `y_0(n)=C_y x(n)+D_yu u(n)+D_yw w(n)`。
6. 非线性输入为 `z(n)=C_z x(n)+D_zu u(n)`。
7. 非线性输出为 `w(n)=f_NN(z(n))`。
8. 静态非线性 `f_NN` 是 feedforward neural network，隐藏层数记为 `n_l`。
9. 方法假设先有一个稳定的线性基线模型 `M0=(A~, B_u~, C_y~, D_yu~)`。
10. 线性基线可由 standard linear identification 或 best linear approximation (BLA) 得到。
11. 第一步对线性基线做相似变换，用模拟状态轨迹的经验标准差构造对角矩阵 `T_x`。
12. 这个归一化让 latent state 近似单位方差，降低后续优化的尺度问题。
13. Stage (i) 是 bilevel guided residual search (GRS)。
14. GRS 的内层固定 `theta_wy=(B_w,D_yw)`，把 `w` 当成外生输入，滑窗估计 latent residual `w_*` 和状态 `x_*`。
15. 内层窗口长度为 `H+1`，对 `n=0,...,N-H-1` 解局部二次问题。
16. 局部目标是输出拟合误差加 `lambda ||S_w(n)||_{Q_w}^2 / 2` 正则项。
17. 正则矩阵 `Q_w=[B_w;D_yw]^T[B_w;D_yw]+epsilon/lambda I`，按 residual 对系统的实际影响而非原始幅值惩罚。
18. 内层问题是凸的，并给出闭式解 `S_{w_*}(n)=-G^{-1}T_w^T(...)`。
19. 每个窗口只保留第一段 `w_*(n)`，再用线性扩展模型递推 `x_*(n+1)`。
20. 初始 `x_*(0)=0` 会产生 transient，所以后续丢弃前 `N_0` 个样本。
21. Stage (i) 外层更新 `B_w,D_yw`，最小化 inferred 输出 `y_*(n)` 与测量输出 `y(n)` 的均方误差。
22. `theta_wy` 从 `U(-1,1)` 初始化；作者明确指出结果对随机初始化敏感。
23. Stage (ii) 用 inferred 数据集 `D_*={(w_*(n),x_*(n),y_*(n))}` 训练非线性残差模型。
24. 训练变量为 `theta_nl=(C_z,D_zu,theta_NN)`，目标是让 `f_NN(C_z x_*(n)+D_zu u(n))` 拟合 `w_*(n)`。
25. 这一阶段是 recursion-free 的 supervised learning，所以比直接长时域仿真优化稳定。
26. 但 recursion-free 训练会带来 deployment distribution shift。
27. 作者形式化了这个 shift：如果 learned transition `f_hat` 对状态的 Lipschitz 常数 `L>1`，且一步误差小于 `epsilon`，长时域仿真偏差最坏可达 `O(L^{N-1} epsilon)`。
28. 证明通过误差递推 `||e(n+1)|| <= L||e(n)|| + epsilon`，几何级数给出指数放大。
29. 作者还给出 tightness 构造：标量系统 `x_*(n+1)=Lx_*(n)` 与 `f_hat(x,u)=Lx+epsilon` 正好达到该量级。
30. Stage (iii) 用 multiple shooting 缓解这个问题。
31. 多重射击把完整轨迹切成长度为 `d` 的短区间。
32. 每个区间开头的 shooting state 作为决策变量，初值直接取 GRS 推断的 `x_*`。
33. 区间内部执行仿真约束，区间边界用 equality constraint 强制连续。
34. 极限情况下 `d=1` 可局部消除误差积累，但如果没有连续性约束就退化为一步预测误差最小化。
35. 最终优化同时调整完整参数 `theta=(theta_uy,theta_wy,theta_nl)` 和 shooting states。
36. 多重射击目标仍然是 simulation-error objective，而不是单纯 one-step prediction。
37. 论文实现使用 Python。
38. GRS 与神经网络训练用 JAX，优化器来自 Optax 和 Optimistix。
39. 最终 multiple-shooting NLP 用 CasADi 建模，用 IPOPT 求解。
40. 论文给出代码 URL：`https://github.com/merijnfloren/guided-residual-search`，但本次审读没有下载非 arXiv 仓库。

### 关键技术细节

1. NL-LFR 显式拆分 LTI dynamics 和静态非线性 NN，适合把 dominant linear behavior 与 localized nonlinearity 分开处理。
2. 假设测量输出有加性、零均值、平稳、有限方差噪声；噪声可 colored，但与输入不相关。
3. 假设输入 `u(n)` 精确已知，且 sufficiently persistently exciting。
4. 假设初始线性模型稳定，且捕捉 dominant linear input-output behavior。
5. 内层 GRS 的 stacked output 使用 extended observability matrix `O_x` 和 block Toeplitz matrices `T_u,T_w`。
6. 内层优化的 closed-form 求解依赖 `G=T_w^T T_w + lambda Q_w`。
7. `epsilon` 保障 `Q_w` 严格正定；实验均用 `epsilon=1e-8`。
8. Silverbox 使用 `n_x=2,n_w=1,n_z=1`。
9. Silverbox 的 GRS 参数为 `H=10,N_0=100,lambda=1,epsilon=1e-8`。
10. Silverbox 的 GRS 外层执行 25 次 Levenberg-Marquardt iterations。
11. Silverbox 的神经网络为单隐藏层、15 个 ReLU neurons。
12. Silverbox 的 NN 训练为 100 Adam iterations，learning rate `1e-3`。
13. Silverbox multiple shooting 使用 `d=1`，最终 IPOPT 200 iterations。
14. Silverbox single shooting 对照用 `d=N_tot`，最终 IPOPT 3000 iterations，使 CPU 时间约同为 20 minutes。
15. Silverbox 线性-only 对照将 `B_w,D_yw` 初始化为 `U(-1e-4,1e-4)`，避免零梯度同时保持接近 BLA loss。
16. Silverbox 每个 scenario 重复 25 个 random seeds，以 median 和 MAD 汇报。
17. Silverbox 数据包含 arrowhead 测试段 40000 samples，白高斯噪声且幅值线性变化。
18. Silverbox multisine 段包含 86750 samples，其中最后 21688 samples 留作测试。
19. Silverbox 采样率约 610 Hz。
20. F-16 ground vibration test 是 1 input、3 outputs。
21. F-16 激励来自安装在右翼下方的 electrodynamic shaker。
22. F-16 响应用 accelerometers 在 wing-tip payload interface 和 excitation point 附近测量。
23. F-16 训练数据为 8 个 steady-state periods，single random-phase multisine，频带 2-15 Hz。
24. F-16 每 period 8192 samples，训练 RMS 激励 73.6 N。
25. F-16 预处理对 periods 求平均，并用 15 Hz low-pass filter 降低测量噪声影响。
26. F-16 模型阶次经验选为 `n_x=15,n_w=2,n_z=6`。
27. F-16 GRS 参数为 `H=10,N_0=500,lambda=100,epsilon=1e-8`。
28. F-16 GRS 外层执行 50 次 Levenberg-Marquardt iterations。
29. F-16 神经网络为单隐藏层、32 个 tanh units。
30. F-16 NN 最多训练 1000 Adam iterations，learning rate `4e-3`。
31. F-16 按初始化敏感性建议，重复 GRS + NN 训练 50 个 random seeds。
32. F-16 GRS + NN 总 runtime 为 195 s +/- 1.71 s median +/- MAD，其中 GRS 约 30 s。
33. F-16 最终 multiple shooting 使用 `d=200`，1000 IPOPT iterations。
34. F-16 作者说明 `d=1` 预计更好，但会超过可用内存。
35. F-16 每个 IPOPT iteration 约 4.2 s，总最终优化时间约 70 min。

### 核心结果

1. 论文的核心经验结论是：GRS 初始化能把参数带到更有信息量的 region of attraction。
2. Silverbox 上，GRS 初始化相比 linear-only 初始化，最终优化初始 NRMSE 更低、前期下降更快。
3. Silverbox 上，无论是否使用 GRS，multiple shooting 都比 single shooting 收敛到更低的 final NRMSE。
4. Silverbox GRS + NN 初始化 overhead 平均只需 21 s。
5. Silverbox test RMSE 的 proposed NL-LFR 在 multisine 上为 `0.33 +/- 0.02 mV`。
6. Silverbox test RMSE 的 SUBNET 对照在 multisine 上为 `0.36 mV`。
7. Silverbox arrowhead full 上 proposed 为 `1.13 +/- 0.15 mV`，SUBNET 为 `1.40 mV`，PNLSS 为 `0.26 mV`。
8. Silverbox arrowhead no extrapolation 上 proposed 为 `0.31 +/- 0.03 mV`，SUBNET 为 `0.32 mV`。
9. 反直觉点：在 Silverbox full arrowhead，PNLSS 远好于 proposed，因为 cubic polynomial basis 与真实 cubic spring 非线性匹配。
10. proposed 与 SUBNET 的 extrapolation error 类似，说明纯黑盒 NN residual 在训练域外仍弱。
11. 作者报告 SUBNET 训练超过 1 天，而 proposed 方法约 20 min。
12. F-16 distribution-shift 实验显示 NN 静态训练 loss 单调下降，但完整 NL-LFR deployment simulation error 会快速恶化且高度波动。
13. 作者解释为：NN 训练降低一步误差 `epsilon` 的同时可能提高有效 Lipschitz 常数 `L`。
14. F-16 最终优化中，linear-only 初始化虽然起始 simulation error 较低，却陷入较差局部极小。
15. F-16 proposed 初始化起始误差因 shift 较高，但快速改善并收敛到明显更优解。
16. F-16 测试输出 1 的 BLA NRMSE 为 `11.59%`，NL-LFR 为 `9.20%`。
17. F-16 测试输出 2 的 BLA NRMSE 为 `19.40%`，NL-LFR 为 `12.55%`。
18. F-16 测试输出 3 的 BLA NRMSE 为 `20.59%`，NL-LFR 为 `13.27%`。
19. F-16 测试采用同一 random-phase multisine realization，但更高 RMS 激励 85.7 N。
20. 论文没有声称全面 SOTA，只说在两个 benchmark 上与黑盒方法 competitive，并改善 naive initialization 的收敛。

### 明确局限

1. 作者自己承认 GRS 对 `theta_wy` 随机初始化敏感，尤其高维系统需要 multiple initializations。
2. GRS 可产生多个 outer-loss effective minimizers，其中并非都对应有意义的 static relation between `x` and `w`。
3. Recursion-free residual training 会引入 distribution shift，这既是论文分析对象，也是方法短板。
4. Multiple shooting 引入 shooting-state decision variables，最终 NLP 的内存和计算成本成为主要瓶颈。
5. F-16 上作者无法使用 `d=1`，只能用 `d=200` 作为内存与性能折中。
6. F-16 的 `d` 需要 empirical tuning，附近超参数和随机种子并非都可靠收敛。
7. 当前实现没有利用 multiple-shooting intervals 的天然并行性。
8. Silverbox full arrowhead 上 PNLSS 因 cubic basis 明显优于 proposed，说明结构先验在可得时仍更强。
9. F-16 与 Silverbox 都是已有 benchmark，不是真机闭环控制验证。
10. F-16 benchmark 的 prior literature 稀疏，作者也说明直接横向比较受数据处理差异限制。
11. 论文实验没有做更大规模、多输入多输出工业系统的系统性 sweep。
12. 论文没有给出置信区间，只用 median/MAD 或 mean/std 汇报一部分结果。
13. 论文没有独立第三方复现实验。
14. 论文的理论 bound 是 worst-case Lipschitz 分析，解释力强，但不直接给出实际有效 `L` 的估计方法。
15. 论文使用 NN 作为 residual，训练域外 extrapolation 风险仍然存在。
