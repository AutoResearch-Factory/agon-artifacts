# Symbolic Discovery of Stochastic Differential Equations with Genetic Programming (2603.09597)

- **Year**: 2026.03
- **Authors**: Sigur de Vries, Sander W. Keemink, Marcel A. J. van Gerven (Radboud University, Donders Institute)
- **Venue**: arXiv preprint (not yet published in conference/journal)
- **Keywords**: symbolic regression, genetic programming, stochastic differential equations, SDE, automated scientific discovery, maximum likelihood estimation, Kozax, system identification
- **One-liner**: 首次用遗传编程(GP)从噪声时间序列数据中联合发现随机微分方程(SDE)的漂移项和扩散项的符号表达式, 在低维问题上与KM-SR竞争, 在高维和稀疏采样场景下显著优于基于Kramers-Moyal展开的稀疏回归方法.

## 精华提炼

### 管线/方法

**输入**: 观测到的离散时间序列数据 {x(t_k)}, 假设全可观性, 噪声服从正态分布, 漂移-扩散可分离.

**完整 Pipeline**:
1. **数据生成 (仿真)**: 使用 Diffrax 的随机 Runge-Kutta 方法 (Stratonovich SDEs) 积分 SDE 系统, 时间步长 Δt=0.001. 从积分中按采样率 τ 抽取离散观测. 每条轨迹使用不同的初始条件和 Wiener 过程.
2. **GP 种群初始化**: 随机生成 p 个个体, 每个个体包含一对 parse trees (漂移树 f̂_i + 扩散树 ĝ_i). 树节点来自预定义函数库: +, ×, 状态变量. 对于 SPDE 额外加入梯度和 Laplacian.
3. **适应度评估 (MLE)**: 对每个候选解 f̂_i, ĝ_i, 计算负对数似然: F(f̂_i, ĝ_i) = Σ_k [½log(2πσ²(t_k)) + (x_i(t_k) - μ_i(t_k))²/(2σ²(t_k))], 其中 μ_i(t_k) = x_i(t_{k-1}) + τ·f̂_i(x(t_{k-1})), σ(t_k) = √τ·ĝ_i(x(t_{k-1})). 逐变量独立优化(各变量各自有一对树).
4. **多步积分扩展 (GP-SDE-MS)**: 当采样率过低时, 在观测点之间用更小的时间步 L=5 进行数值积分, 均值 μ(t_k) 和方差 σ(t_k) 通过累积中间步的状态计算.
5. **NSGA-II 选择**: 同时优化适应度和复杂度(节点数). 使用 tournament selection.
6. **繁殖**: Crossover (交换同位置子树 + whole-tree crossover), Mutation (修改算子/叶子/插入/删除节点). 节点数受超参数 s 约束.
7. **梯度下降常数优化**: 对 fittest 子集 p* 的个体用梯度下降优化树中叶子的常数, 进行 g* 次迭代, 步长 η.
8. **多子种群**: 种群分入多个子种群, 各自有不同 crossover/mutation 概率. 每几代进行个体迁移保持多样性.
9. **模型选择**: 在 validation set 上选取 Pareto front 中 MSE 最低的个体作为最终解. 最终在 test set 上报告 MSE.

**输出**: 符号表达式 f̂_i(x) 和 ĝ_i(x) 构成的 SDE 系统, 可解释且支持生成采样.

### 关键技术细节

- **GP 框架**: Kozax (de Vries et al., GECCO 2025), 纯 JAX 实现. 代码开源: github.com/sdevries0/GP-SDE
- **SDE 积分器**: Diffrax 的 Stratonovich SDE Runge-Kutta (Foster et al. 2024), Δt=0.001
- **适应度函数**: MLE / 负高斯对数似然, 逐 transition pair 计算 (simulation-free, 类似 finite differences)
- **多步积分细节**: L=5 个子步. 均值: μ(t_k) = y(L), y(l) = y(l-1) + (τ/L)·f̂(y(l-1)). 方差: σ(t_k) = (√τ/L)·Σ_l ĝ(y(l))
- **硬件**: Intel Xeon Platinum 8360Y (36核) + 单 NVIDIA A100 40GB GPU (GP方法); AMD Genoa 9654 (192核) (KM-SR)
- **GP 超参数**: p=500-3000, g=50-200, m=5 (最大初始深度), s=15-20 (最大节点数), p*=100-500, g*=15-50, η=0.1
- **KM-SR 超参数搜索**: b ∈ [2,4,5,10,15,25,50,100], β ∈ [1,10,20], α ∈ [0.001,0.01,0.1], λ ∈ [0.05,0.1,0.2]
- **函数库**: +, ×, 变量名. 多项式阶数 d=2 或 3
- **独立噪声假设**: G(x) = diag(g_1,...,g_N), 每个变量由独立 Wiener 过程驱动
- **逐变量优化**: 每个变量独立进化一对 f̂_i, ĝ_i, 降低搜索空间
- **测试环境**: Double well (1D, 3种扩散), Van der Pol (2D), Rössler (3D, 混沌), Lorenz96 (5/10/20维), Lotka-Volterra (2D, 稀疏采样), Fisher-KPP (1D SPDE), 2D Heat Transfer (SPDE)
- **数据集规模**: 8条轨迹(Double well/Van der Pol/Rössler/Lorenz96/Lotka-Volterra), 4条(SPDEs). 每条轨迹 T=25-50

### 核心结果

| 实验 | GP-ODE | KM-SR | GP-SDE | 关键发现 |
|------|--------|-------|--------|----------|
| Double well (additive) | 稳定恢复漂移 | 漂移+扩散准确 | 竞争性但有方差 | GP-SDE 可行但一致性不如 KM-SR |
| Double well (non-linear mult.) | MSE 大幅上升 | 多数跑恢复正确 | 与KM-SR接近 | 复杂噪声下必须同时学扩散 |
| Van der Pol | 漂移最准确 | 结构正确 | 扩散结构未完全恢复 | 扩散项区分 stochastic/deterministic 变量有难度 |
| Rössler (3D混沌) | 部分跑恢复 | **MSE退化** | **GP-SDE全面超越KM-SR** | 混沌+高维使 binning 不准, GP 受益 |
| Lorenz96 (5维) | 正确恢复 | 竞争 | 匹配GP-ODE漂移精度 | GP 可扩展 |
| Lorenz96 (10维) | 正确恢复 | **严重退化** | **显著优于KM-SR** | KM-SR 4 bins 已超百万格, 计算爆炸 |
| Lorenz96 (20维) | 正确恢复 | 基本失败 | 虽不完全正确但准确 | GP 运行时间几乎不随维度增长 |
| Lotka-Volterra (τ=0.02) | 竞争 | 扩散更准 | 漂移更准 | 密集采样下三法接近 |
| Lotka-Volterra (τ=0.2) | 退化 | 轻微退化 | GP-SDE-MS **超越KM-SR** | 多步积分显著改善稀疏采样 |
| Lotka-Volterra (τ=0.5) | 很差 | 退化 | **GP-SDE-MS 最优** | 同时学扩散进一步提升漂移恢复 |
| Fisher-KPP SPDE | - | - | 结构正确, 常数略有偏差 | MLE 可推广到 SPDE |
| 2D Heat SPDE | - | - | **完全正确恢复** | SPDE 扩展可行 |

**Surprising result**: GP-SDE 学习扩散项不仅不加害漂移恢复, 在复杂场景下反而改善了漂移恢复. 论文认为这是因为建模 stochasticity 使搜索避免了将噪声错误归因于漂移.

**计算时间**: KM-SR 低维极快, 但升维爆炸 (10维 4 bins 已不可行). GP-SDE 运行时间随维度几乎不增长, 5维 ~20min/seed, 20维仅略增.

### 明确局限

1. **全可观性假设**: 假设所有变量可观测且无观测噪声. 实际系统常有隐藏变量和测量噪声.
2. **高斯噪声假设**: 仅支持正态分布噪声. 不支持 Lévy jumps、非高斯噪声. 作者在 Discussion 中承认需要推广.
3. **漂移-扩散可分离假设**: 噪声必须以加性方式进入系统 (additive/multiplicative 均可但需可分离). 不支持 noise-in-input 形式.
4. **无 latent SDE 能力**: 与 neural SDEs / latent SDEs 相比, 无法处理部分观测.
5. **Identifiability 问题**: 作者诚实指出 SDE 的漂移和扩散可能不唯一——不同方程可产生统计上不可区分的轨迹. 好的预测拟合不代表发现了正确的控制方程.
6. **低维一致性不如 KM-SR**: 在 1D 问题 (Double well) 上 GP-SDE 的方差大于 KM-SR, 扩散 MSE 略高.
7. **扩散结构恢复不完备**: Van der Pol 和 Rössler 实验中扩散项结构未在所有 run 中正确恢复 (虽然 MSE 低).
8. **无标准 benchmark 对比**: 未在 SRBench 或 PySR benchmark 上评估. 实验均为自建仿真环境.
9. **超参数未系统消融**: GP-SDE 的 GP 超参数 (p, g, m, s) 固定, 未做敏感性分析.
10. **未开源训练 log**: 代码开源但未见完整实验复现脚本和随机种子 log.

## Read by: 0616-fex (2026-06-17)

审读视角:

本论文处于符号回归 (symbolic regression) 和随机动力学系统 (stochastic dynamical systems) 的交叉点. 对于 0616-fex 课题, 核心价值不在于方法本身 (GP 发现 SDE), 而在于以下三个可迁移的 insight:

1. **"显式建模噪声改善确定性特征学习"** — 论文的核心发现: 同时学习漂移和扩散在复杂系统中反而改善了漂移恢复. 这是一个重要的设计原则: 在 fex 中, 如果目标系统存在显著的随机性, 不应忽视它, 而应显式建模. 隐式忽略噪声会导致学到的特征被噪声污染.

2. **工具层面** — Kozax 作为 JAX 原生 GP 库, 可以快速适配到 fex 中任何需要符号回归的子任务. Multi-step integration 技巧 (L=5 子步) 是处理稀疏数据的轻量级方案.

3. **Landscape 定位** — 本文填补了 "GP-based SDE symbolic discovery" 的空白. 在 fex 课题的文献调研中, 应注意到 SINDy 生态 (KM-SR) 在高维和稀疏数据上的固有局限 (binning curse + 两阶段). GP-based 方法在此类场景下具有结构性优势. 如果 fex 涉及高维动力学特征提取, 应优先考虑 GP-based 而非 SINDy-based 方法.

差异化明确: fex 关注 "从已建模系统中提取特征", 本文关注 "从数据中发现系统方程". 两者目标不同, 但方法论和工具可共享.
