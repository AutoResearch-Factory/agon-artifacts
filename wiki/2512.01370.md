# Beyond Loss Guidance: Using PDE Residuals as Spectral Attention in Diffusion Neural Operators (2512.01370)

- **Year**: 2025.12
- **Authors**: Medha Sawhney, Abhilash Neog, Mridul Khurana, A. Karpatne (Virginia Tech)
- **Venue**: arXiv preprint; ICLR 2026 conference format
- **Keywords**: PDE residual, spectral attention, diffusion neural operator, function-space diffusion, UNO, physics-guided generation, noisy observations, inverse problems
- **One-liner**: 把 PDE 残差做成扩散神经算子的谱注意力。

## 精华提炼

### 管线/方法

1. 论文提出 PRISMA: PDE Residual Informed Spectral Modulation with Attention。
2. 目标不是用 PDE residual 做外部 loss guidance, 而是把 residual field 变成 denoiser 内部的架构特征。
3. 状态变量是连续场拼接 `x=[a,u]`, 其中 `a` 是 PDE 参数/源项/初值, `u` 是解场。
4. 统一问题形式是学习 `p(a,u | a_obs, u_obs, M_a, M_u)`, 同一个模型覆盖 forward 和 inverse。
5. Forward full: `M_a=True, M_u=False`, 给定完整 `a`, 生成 `u`。
6. Inverse full: `M_a=False, M_u=True`, 给定完整 `u`, 生成 `a`。
7. Sparse setting 中 mask 不只是任务标识, 也记录观测位置。
8. Noisy setting 中方法不依赖一个已知的 corruption mask; 它通过 residual gate 动态降低不可信 residual 的影响。
9. Backbone 是 conditional Denoising Diffusion Operator, 噪声不是独立像素白噪声, 而是 Gaussian random field。
10. Denoiser 是 4-level U-shaped Neural Operator (UNO), 用 Fourier/spectral path 加 local residual block。
11. 标准 UNO 层可以写成 spectral path `F^{-1}(W^l * F(x^l))` 加 spatial path `psi^l(x^l)`。
12. PRISMA 的改动是每个 UNO layer 的 global spectral path 前插入 SRA block。
13. 每个 denoising step 先用观测值和当前模型状态构造 mixed fields。
14. `a_mix = M_a * a_obs + (1-M_a) * a_sigma`。
15. `u_mix = M_u * u_obs + (1-M_u) * u_sigma`。
16. 然后计算 observation-informed PDE residual `r = R(a_mix, u_mix)`。
17. residual 在模型 native resolution 计算一次, 再下采样给不同尺度的 UNO layer。
18. SRA block 把当前 feature map 和 PDE residual 都变换到 Fourier domain。
19. 令 `x_hat^l = F(x^l)`, `r_hat = F(r)`。
20. 它把 residual spectrum 当 key, feature spectrum 当 query。
21. compatibility score 是跨 channel 的 complex inner product magnitude:
22. `S^l(k)=1/sqrt(C) |sum_c x_hat_c^l(k) conj(r_hat_c^l(k))|`。
23. 每个 frequency mode 有可学习 spectral gain `w_gain^l(k)`。
24. attention mask 是 `A^l(k)=sigmoid(w_gain^l(k) * S^l(k))`。
25. 这个设计保留 phase alignment 信息; ablation 显示 phase-blind attention 在 inverse/noisy 任务上变差。
26. SRA 还学习一个 residual guidance strength `g_res^l in [0,1]`。
27. `g_res` 由两层 MLP 根据 residual spatial mean 和 diffusion noise embedding `c_sigma` 预测。
28. Appendix 给出 MLP 输入 shape `B x (E+1)`, 其中 timestep embedding 例子是 `E=256`, 输出是 `B x 1`。
29. SRA 输出是 weighted skip modulation:
30. `SRA(x^l,r^l)=F^{-1}(((1-g_res^l)+g_res^l A^l) * x_hat^l)`。
31. 高噪声阶段 residual 不可信, gate 倾向下调 residual correction。
32. denoising 后期预测更干净, gate 可以提高 residual-driven spectral modulation。
33. 训练算法: 采样 clean state `x0=[a,u]`, 采样 noise level `sigma`, 采样 task masks, 构造 `x_sigma=x0+eta`, 计算 guided residual, denoiser 预测 `x0_hat`。
34. 训练 loss 是 EDM objective: `lambda(sigma)||D_theta(x_sigma, sigma, y)-x0||^2`。
35. Inference 使用 2nd-order solver, 从 Gaussian random field 初始化。
36. 每个 predictor step 和 corrector step 都重新计算 guided residual。
37. residual 计算始终用 `(1-M)*x_i + M*x_obs`, 即已知区域由真实观测钉住, 未知区域用当前估计。
38. 这样 PRISMA 在 inference 时不需要对 PDE loss 做 gradient descent。
39. 与 DiffusionPDE / FunDPS 的核心差别: 它不是 unconditional model + test-time posterior sampling, 而是 conditional model + architecture-integrated residual guidance。
40. 论文把这个差别概括为 architectural guidance, point-wise guidance, spectral integration, gradient-descent-free inference。

### 关键技术细节

1. 模型规模: denoiser 约 64M parameters, 与 DiffusionPDE / FunDPS 量级相近。
2. 训练数据: 每个 PDE 50,000 training samples 和 1,000 test samples。
3. 分辨率: 数据包含 64x64 和 128x128; 主实验在 128x128 训练和评测, 部分 ablation 在 64x64。
4. 训练: 200 epochs, 2 张 NVIDIA A100, batch size 90 per GPU。
5. 实现基于 EDM-FS / Denoising Diffusion Operators 代码。
6. Hyperparameters: learning rate 1e-4, warmup 50 epochs, EMA half-life 5 epochs。
7. 其他 diffusion hyperparameters: dropout 0.13, rbf_scale 0.05, sigma_max 80, sigma_min 0.002, rho 7。
8. PDE benchmarks: Darcy Flow, Poisson, Helmholtz 是主三组; tables 还加入 Navier-Stokes noisy/full/sparse。
9. Appendix 还做 Burgers spatiotemporal field reconstruction 和 super-resolution。
10. Darcy: domain `(0,1)^2`, constant forcing `g=1`, zero Dirichlet boundary, coefficient GRF 经 piecewise map 得到。
11. Poisson: unit square, `laplace u = a`, homogeneous Dirichlet, source `a` 是 thresholded GRF binary field。
12. Helmholtz: `Delta u + k^2 u = a`, `k=1`, homogeneous Dirichlet, piecewise-constant heterogeneous coefficient。
13. Navier-Stokes non-bounded: vorticity-streamfunction form, periodic boundary, `nu=1e-3`, `Re=1000`。
14. 作者指出 DiffusionPDE 的 2D scalar `div omega` residual 不物理, 他们改成 one-step vorticity-transport residual, 用 `Delta psi=-omega` 恢复 velocity。
15. Noisy main setting: 100% pixels 加 unit Gaussian noise `N(0,1)`。
16. Sparse main setting: only 3% data known, 即 97% pixels missing。
17. Metrics: 大多数任务用 relative L2 error; Darcy inverse 用 classification error rate。
18. Baselines: PINN, FNO, DeepONet, PINO, DiffusionPDE, FunDPS。
19. Speed measurement: 单张 A100 上测 batch inference time, 10 runs 平均后除以 batch size。
20. Reproducibility note: 作者提供 supplementary code, 使用公开 dataset, 并说明 DiffusionPDE 重现实验有问题, 他们重跑了 full/sparse cases。

### 核心结果

1. Noisy main table 中 PRISMA 50 steps 在三大 PDE + Navier-Stokes 上平均 rank 1.13, time 0.8s/sample。
2. PRISMA 20 steps 已经很接近 50 steps, time 0.18s/sample, noisy avg rank 2.13。
3. DiffusionPDE 使用 2000 steps, time 213.0s/sample; FunDPS 500 steps 是 11.8s/sample。
4. 作者 claim 的速度优势是 15x 到 250x seconds/sample, 且 denoising steps 少 10x 到 100x。
5. Noisy Darcy forward: PRISMA 50 是 12.28%, DiffusionPDE 49.18%, FunDPS 500 是 55.09%。
6. Noisy Darcy inverse: PRISMA 50 是 23.14%, DiffusionPDE 70.08%, FunDPS 500 是 49.62%。
7. Noisy Poisson forward/inverse: PRISMA 50 是 18.35% / 41.14%, DiffusionPDE 是 44.44% / 130.01%。
8. Noisy Helmholtz forward/inverse: PRISMA 50 是 16.84% / 67.93%, DiffusionPDE 是 30.97% / 119.52%。
9. Noisy Navier-Stokes: PRISMA 50 forward/inverse 是 36.34% / 37.03%; FunDPS 200 forward 最好 31.55%, 但 inverse 38.44%。
10. Full observation table 中 PRISMA 50 avg rank 2.75, 与 PINO 同档, time 0.8s/sample。
11. Full Darcy forward: PRISMA 50 是 1.05%, 与最优接近。
12. Full Navier-Stokes inverse: PRISMA 50 是 1.25%, 是表中最好。
13. Sparse table 中 FunDPS 500 avg rank 1.75 最好, PRISMA 50 avg rank 2.25, 但 PRISMA 快得多。
14. Sparse Navier-Stokes forward: PRISMA 50 是 7.11%, 表中最好。
15. Sparse Darcy forward/inverse: PRISMA 50 是 3.11% / 6.80%, FunDPS 500 更好为 2.49% / 5.18%。
16. 20-step stress test 很强: full observation 下 FunDPS 20 steps 在 Helmholtz inverse 是 39.39%, DiffusionPDE 101.01%, PRISMA 11.03%。
17. Noisy 20-step stress test 中 FunDPS/DiffusionPDE 多数不收敛, PRISMA 明显低误差。
18. Step ablation: noisy setting 从 20 到 500 steps 收益很小; 例如 noisy Darcy forward 12.10 -> 12.06, Helmholtz inverse 68.43 -> 67.50。
19. Full setting也早收敛; Helmholtz forward 20 steps 8.12, 500 steps 6.80, 但大多数提升不值得 25x step 增加。
20. Residual strategy ablation: Helmholtz noisy inverse 上, no residual 是 111.87%, concat residual 是 97.09%, SRA 是 91.85%。
21. Sparse Helmholtz forward/inverse 上, SRA 30.33% / 60.96% 优于 no residual 36.2% / 66.75%, concat 最差 94.2% / 124.7%。
22. SRA gating ablation: full Helmholtz inverse Ours 12.58 最好; no gating 28.91。
23. Phase-aware attention有贡献: attention without phase 在 noisy Helmholtz inverse 102.19, Ours 93.2。
24. Super-resolution: 64x64 trained model 在 64/128/256/512 resolution inference 不重训。
25. Super-resolution time/memory: 64x64 是 0.242s 和 4.117GB, 512x512 是 4.849s 和 107.114GB。
26. Burgers spatiotemporal reconstruction: PRISMA test relative error 9.33%, DiffusionPDE 10.39%。

### 明确局限

1. 这不是 symbolic regression, 不输出 closed-form PDE solution 或 finite expression。
2. 方法需要已知 PDE operator 才能计算 residual, 不适合 PDE form 未知的 discovery setting。
3. 主证据来自 synthetic/grid PDE benchmark, 不是真实传感器或物理实验数据。
4. PRISMA 是 heavy amortized model: 每个 benchmark 需要 50,000 samples、200 epochs、2 A100, 与 FEX 的 per-instance symbolic search 计算结构不同。
5. Full/sparse setting 不是全线 SOTA: PINO、FunDPS 在若干 clean/sparse task 上更优。
6. Inverse noisy task 仍然很难: Helmholtz inverse 67.93%, Poisson inverse 41.14%, 不是“解决了” noisy inverse PDE。
7. SRA 对 forward clean/noisy 并非处处最好; 有些 ablation 中 no-residual 或 multiplicative gate 在单项 forward 上更低。
8. Residual concat 结果很差, 说明“把 residual 塞进模型”本身不稳定, 需要 phase-aware spectral gating 才有效。
9. Super-resolution 512x512 需要约 107GB reserved GPU memory, 实用性受限。
10. 作者说 supplementary code 可复现, 但 arXiv text 中没有独立复现报告或 artifact evaluation。
11. Baseline reproducibility 本身有问题; DiffusionPDE 的重跑依赖作者判断和本论文实现, 不等于第三方验证。
12. 复杂几何和 irregular mesh 是作者自己列出的 future work。
13. Spatiotemporal 只展示 Burgers, 还不是系统的 time-dependent PDE benchmark。
14. 对 FEX 来说, 其主要局限是没有回答 expression-tree search 如何利用 residual spectrum, 只证明 neural denoiser 可以利用它。

### 跟我们的关系

1. `0616-fex` 的题目是 FEX: RL for Symbolic Regression, 即用 RL 做符号回归/有限表达式搜索。
2. PRISMA 不直接撞车 FEX: 它是生成式神经 PDE solver, 不是 RL expression-tree search, 也不产生 symbolic expression。
3. 它直接撞的是“用 PDE residual 做 diffusion solver guidance”的方向, 不是“用 RL 搜索有限表达式求 PDE”的方向。
4. 对 `fex-synchro-prior` 特别相关: 它提供了强证据说明 PDE residual 的频谱结构比标量 residual loss 更有信息。
5. 我们的 Spectral-Prior Compiler claim 不能泛泛写成“首次利用 PDE residual spectrum”; PRISMA 已经把 residual spectrum 用作 neural architecture guidance。
6. 差异化应写成: PRISMA 把 residual spectrum 注入 neural denoiser; FEX 要把可观测谱信号编译成 RL controller 的离散候选、软采样 prior 或 frequency dictionary。
7. PRISMA 的 SRA 公式可作为设计参考: complex inner product + conjugate phase alignment + spectral gain + scalar trust gate。
8. 对 FEX 可迁移的组件包括: residual map 不聚合为 scalar、Fourier-mode compatibility、residual trust gate、mixed-field residual 计算。
9. 对 active-collocation idea 也有间接启示: residual 应该在 known observations 上钉住、在 unknown/predicted 区域评估, 否则 residual 信号会混入已知区域误差。
10. 该引。若 `0616-fex` 任何 paper 提到 spectral residual, residual-guided PDE solver, noisy PDE observation, function-space diffusion baseline, 都应引用 PRISMA。
11. 不应把它作为 FEX baseline, 除非实验目标是 neural PDE solver performance; 它更适合作为“residual spectrum 可被模型有效利用”的 prior work。
12. 若我们的 FEX 方案只做频谱先验但没有 symbolic search 缩减证据, PRISMA 会削弱 novelty; 必须展示 expression-search-specific gain, 如候选评估数、controller entropy、frequency hit rate。

## Read by: 0616-fex (2026-06-17)

本次从 `0616-fex` 视角看, PRISMA 是一个强相关但不直接撞车的 prior。它的贡献是在 diffusion neural operator 内部把 PDE residual field 做成 phase-aware spectral attention, 从而减少 test-time PDE loss guidance 和 gradient descent; `0616-fex` 则是 RL for Symbolic Regression, 目标是让 controller 搜索有限表达式/公式结构。两者的对象和输出不同: PRISMA 输出数值场样本, FEX 输出 symbolic expression 或表达式树。

真正需要警惕的是 novelty 表述。若 FEX 后续 idea 说“首次用 PDE residual spectrum 指导 PDE solver”, 这会被 PRISMA 和 spectral-prior PINN 系列直接冲掉。可保留的差异化是: 我们不是把 residual spectrum 当神经网络内部 feature, 而是把 RHS / early residual / synchrosqueezed signal 编译成 RL expression-tree search 的候选裁剪、软采样分布或 frequency prior, 直接减少离散符号搜索负担。换句话说, PRISMA 支持“residual spectrum 有信息”, 但没有解决“如何用这个信息缩小 symbolic search space”。

可用工具上, 最值得借的是 SRA 的四件套: complex inner product 的 phase-aware compatibility、per-frequency spectral gain、residual trust gate、mixed-field residual 计算。对 `fex-synchro-prior`, 这提供了一个成熟的神经架构先例; 对 `fex-active-collocation`, 它提醒 residual signal 要区分 observed 和 generated 区域, 不能只看 scalar loss。建议在后续搜索中跟进 `spectral residual attention`, `residual Fourier gating`, `function-space diffusion posterior sampling`, `physics-guided diffusion neural operator`, `gradient-free PDE generation`。
