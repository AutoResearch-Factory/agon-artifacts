# Policy Gradient Algorithms in Average-Reward Multichain MDPs (2602.18003)

- **Year**: 2026.02
- **Authors**: Jongmin Lee (Seoul National University), Ernest K. Ryu (UCLA)
- **Venue**: COLT 2026 (colt2026 document class)
- **Keywords**: average-reward multichain MDPs, visitation measure, recurrent and transient states, policy gradient theorem, policy mirror ascent, convergence analysis, sample complexity, generative model
- **One-liner**: First policy gradient theorem for average-reward multichain MDPs using recurrent/transient visitation measures, with sublinear and linear convergence of α-clipped policy mirror ascent in both tabular and generative model settings.

## 精华提炼

### 管线/方法

**问题设定**：无限时域平均奖励 MDP `(S, A, P, r)`，有限状态空间 S、有限动作空间 A、转移矩阵 P、有界奖励 r∈[-R,R]。目标是最大化 `J^π_μ = E_{s∼μ}[lim inf_{H→∞} (1/H) E[Σ r(s_h,a_h) | s_0=s]]`。

**整体流程**：
1. **状态分类** → 基于 Fact: 对于所有严格正策略 π∈Π+（每个动作概率>0），状态的 recurrent/transient 分类不变，仅取决于转移核 P
2. **定义 visitation measures** → recurrent visitation measure `d^π_μ`（Cesàro 极限，transient 状态上为零）和 transient visitation measure `δ^π_μ`（基于 Neumann 级数 (I-T̄^π)^{-1}，非概率测度）
3. **性能差异引理** → 将 J^π_μ - J^π'_μ 分解为 recurrent 项（用 Q^π'）和 transient 项（用 K^π'）
4. **策略梯度定理** → ∇_θ J^π_θ_μ = Σ_{s∈R} d^π_θ_μ(s) ∇π_θ Q^π_θ + Σ_{s∈T} δ^π_θ_μ(s) ∇π_θ K^π_θ
5. **策略镜像上升** → 使用加权 Bregman divergence D_{ρ^π_μ} 进行 mirror ascent，权重 ρ^π_μ(s) = d^π_μ(s)·1_R(s) + δ^π_μ(s)·1_T(s)
6. **α-clipping** → 限制策略空间到 Π_α = {π: π(a|s) ≥ α}，确保分布不匹配系数 C_α 和 B_α 有限
7. **采样设置** → 通过 Critic 算法（多轨迹 Monte Carlo）估计 G^π = Q^π·1_R + K^π·1_T，需要先通过采样识别状态分类

### 关键技术细节

- **参数化**：直接参数化（tabular），π_θ(a|s) = θ_{s,a}，不做神经网络函数逼近
- **Bregman divergence**：支持两种特例——Euclidean 距离（projected Q-descent）和 KL 散度（multiplicative weights update）
- **步长方案**：
  - 常数步长 η：sublinear rate O(1/k)
  - 自适应步长 (C_α-1)η_{k+1} ≥ C_α η_k：linear rate (1-1/C_α)^k
- **C_α**：分布不匹配系数，max_{s,π,π'∈Π_α} ‖ρ^π_μ(s) / ρ^π'_μ(s)‖_∞，在 Π_α 上有限
- **B_α**：max_{π∈Π_α} ‖ρ^π_μ‖_1，ρ^π_μ 的 L1 范数上界
- **α 范围**：(0, 1/|A|)，α→0 时 Π_α → Π_+
- **Critic 算法**：对每个 (s₀,a₀) 生成 N 条长度为 H 的轨迹估计 K̂^π，再生成 N' 条长度为 H' 的轨迹估计 Q̂^π
- **策略评估样本复杂度**：O(R³ t³_tar ‖Q^π‖⁴_∞ |S||A| / ε⁶ · log(2|S||A|/δ))
- **状态分类样本复杂度**：O((|T|+m)(t_{1/2,π} + t^π_cov) log((|T|+m)/δ))
- **期望目标时间 t^π_tar**：从 recurrent class 内任意状态到达目标状态的期望时间，在有限状态空间上有限
- **transient half-life t_{1/2,π}**：min{t≥1: ‖(T̄^π)^t‖_∞ ≤ 1/2}
- **cover time t^π_cov**：覆盖整个 recurrent class 的期望时间

### 核心结果

**Tabular 设定**：
| 步长类型 | 收敛率 | 迭代复杂度 |
|---------|--------|-----------|
| 常数步长 η | Sublinear O(1/k) | O((D_{ρ}(π_α,π₀)/η + C_α Δ₀)/ε) |
| 自适应步长 | Linear (1-1/C_α)^k | O(log(Δ₀/ε) / log(C_α/(C_α-1))) |

**生成模型采样设定**（含 Critic 估计误差）：
| 步长类型 | 样本复杂度（忽略 log 因子） |
|---------|--------------------------|
| 常数步长 | Õ(t³_tar ‖Q‖⁴ R⁹ C⁶_α B⁶_α |S||A| / ε¹²) |
| 自适应步长 | Õ(t³_tar ‖Q‖⁴ R³ C⁶_α B⁶_α |S||A| / ε⁶) |

**Weakly Communicating 特例**（选取 α = ε/(2(|A|+1)‖Q^π*‖_∞)）：
- 可获得真正的 ε-optimal 策略（不限于 positive policies）
- B_α = 1（简化为 d^π_μ 的 L1 范数恒为 1）

**Surprising/反直觉结果**：
- 在 multichain MDP 中，平均奖励 J^π 在策略单纯形边界上可以不连续，但在相对内部 Π_+ 上连续。这解释了为什么之前的 policy gradient 工作无法处理 multichain 设定
- 对于严格正策略，recurrent/transient 分类不变——意味着所有 softmax/exploratory 策略共享同一状态分类结构
- 性能差异引理中，recurrent 状态用 Q 函数、transient 状态用 K 函数（而非统一的 Q），这是 multichain 与 unichain 的本质区别

### 明确局限

**作者明确指出的局限**：
1. 样本复杂度可通过 variance reduction 技术进一步改进（引文 [5][2]）
2. 未扩展到函数逼近设定（restricted parametric policies）
3. 未在 Markovian noise / online 采样下分析
4. α 和 ε 之间的依赖关系未完全量化
5. 未开发 trust region 方法（如 TRPO/PPO）的 multichain 版本

**我们读出的额外局限**：
1. 仅 tabular/direct parameterization，无神经网络策略——与深度 RL 实践差距大
2. Critic 算法需要事先知道 recurrent/transient 分类，且分类过程本身需要采样（虽然样本复杂度低）
3. 自适应步长需要知道 C_α，而 C_α 依赖于 MDP 结构，实际中未知
4. 常数步长的 ε^{-12} 样本复杂度在实际中不可用——即使 ε=0.1 也需要天文数字的样本
5. 没有实验验证——纯理论论文，没有任何数值实验
6. α 需要满足 α < 1/|A|，当动作空间大时 Π_α 可能非常受限
7. t_tar, t_cov, t_{1/2} 等 MDP 特定参数在实际中难以估计
