# Remarks on the Polyak-Lojasiewicz inequality and the convergence of gradient systems (2503.23641)

- **Year**: 2025.03
- **Authors**: Arthur C. B. de Oliveira (Northeastern University), Leilei Cui (MIT), Eduardo D. Sontag (Northeastern University)
- **Venue**: arXiv
- **Keywords**: Polyak-Lojasiewicz inequality, generalized Lojasiewicz inequality, gradient flow, CT-LQR, policy optimization, class-K comparison functions, semi-global PL, linear-exponential convergence
- **One-liner**: 证明 CT-LQR 不满足全局 PL, 只能呈现混合收敛。

## 精华提炼

### 管线/方法
1. 论文从一般优化问题出发: 在开集 \(\mathcal X\) 上最小化实解析、下有界、proper/coercive 的代价函数 \(f(x)\)。
2. 算法对象不是离散梯度下降, 而是连续时间 gradient flow: \(\dot x=-\nabla f(x)\)。
3. 基础收敛链条是: proper \(f\) 使轨道留在紧 sublevel set 内; Lojasiewicz 定理给出收敛到 critical point; strict saddle 的吸引域测度为零。
4. 论文先复述强版本 PL: \(\|\nabla f(x)\|^2\ge \mu(f(x)-\underline f)\), 它与 cost 层面的 global exponential stability 等价。
5. 然后引入半全局 PL: 对每个 sublevel set 都存在 \(\mu_\epsilon>0\), 但 \(\epsilon\to\infty\) 时最优 rate 可以退化到 0。
6. 再引入 local PL: 只在某个固定最优值邻域内有 PL; 它只能保证进入该邻域后的指数型行为。
7. 论文的核心抽象是把 PL 写成 comparison function 下界: \(\|\nabla f(x)\|\ge \alpha(f(x)-f(x^*))\)。
8. \(\alpha\in\mathcal K_\infty\) 对应强鲁棒性/ISS; \(\alpha\in\mathcal K\) 对应 small-input ISS; \(\alpha\in\mathcal{PD}\) 对应 iISS。
9. 新增的中间类是 \(\mathcal K_{\mathrm{SAT}}\): \(\alpha(r)=\sqrt{ar/(b+r)}\), 它在大 \(r\) 时饱和。
10. 该饱和 comparison function 是全文连接理论和 LQR 的关键: 近最优时像 PL, 远离最优时下界不再随 cost gap 增长。
11. Lemma lpli-ksat 说明: \(\mathcal K_{\mathrm{SAT}}\) 下界等价于 local PL 加一个在无穷远不消失的 positive-definite 下界。
12. Lemma linexp-bnd 给出一个 sufficient condition: 若 \(\nabla f\) 全局有界且 \(f\) 有 \(\mathcal K_{\mathrm{SAT}}\) 下界, gradient flow 有 global linear-exponential stability。
13. linear-exponential 的意思是: 在进入某个 sublevel set 前只能用线性下降包住, 进入后再用指数下降包住。
14. 理论部分之后, 论文把框架套到 continuous-time LQR policy optimization。
15. LQR 系统为 \(\dot x=Ax+Bu\), feedback \(u=-Kx\), 稳定反馈集合 \(\mathcal G=\{K:A-BK \text{ Hurwitz}\}\)。
16. 优化目标是 \(J(K)=\mathrm{trace}(P_K)\), 其中 \(P_K\) 解 Lyapunov 方程。
17. 梯度公式为 \(\nabla J(K)=2(RK-B^\top P_K)Y_K\), 其中 \(Y_K\) 也由 Lyapunov 方程给出。
18. 文献已有结论是 CT-LQR 在 sublevel set 上有 semi-global PL, 但没有全局 PL 结论。
19. 本文构造 high-gain curve: \(K^*+\widetilde K(\rho)\) 的闭环特征值保持远离虚轴, 同时 \(J\to\infty\)。
20. Lemma hgd-exists 用 Ackermann pole placement 证明任意 controllable \((A,B)\) 都存在这样的 high-gain curve。
21. Lemma BndGradJ 证明沿 high-gain curve, 虽然 \(J(K)-J(K^*)\to\infty\), 但 \(\nabla J(K)\) 收敛到有限矩阵。
22. 因此 CT-LQR 不可能满足全局 PL: PL 右侧会随 \(\sqrt{J-J^*}\) 发散, 但左侧梯度范数保持有界。
23. 这就是 Corollary LQRnoPLI 的主要结论: continuous-time LQR cost can never satisfy global PL。
24. 论文同时证明另一侧: 若 \(K\) 接近稳定边界 \(\partial\mathcal G\), 则 \(\|\nabla J(K)\|_F\to\infty\)。
25. 因而 CT-LQR 既不是全局 PL, 也不是全局梯度有界; 它处在 GES 与 GLES 的 gap 中。
26. 为了恢复有界梯度, 作者限制到带 margin 的稳定集合 \(\mathcal G_\delta=\{K:A-BK+\delta I \text{ Hurwitz}\}\)。
27. 在 \(\mathcal G_\delta\) 上, Lemma LQRClassKPLIub 给出 \(\|\nabla J(K)\|<g\) 的全局上界。
28. scalar LQR 部分把机制写成一维公式, 便于看清两种区域。
29. scalar 设置取 \(A=a\), \(B=1\), \(Q=q\), \(R=r\), 稳定域要求 \(a-k<0\)。
30. 标量代价 \(J(k)=p\), 梯度 \(\partial J(k)=-2(p-rk)\ell\)。
31. 当 \(k\to+\infty\) 时, \(\partial J(k)\to r/2\): 高增益远端梯度饱和, 对应近似线性下降。
32. 当 \(k\to a^+\) 接近稳定边界时, gradient 与最优指数 rate 都发散, 对应快速下降。
33. 论文用 \(m(k)=\|\partial J(k)\|^2/(J(k)-J(k^*))\) 量化局部指数 rate。
34. proposition 给出 \(J(k^*+\epsilon)-J(k^*)=\ell r\epsilon^2\), \(\partial J(k^*+\epsilon)=\ell(-\delta(\epsilon)+r\epsilon)\)。
35. 因而 \(m(k^*)=r\ell^*>0\), 近最优区域有指数收敛。
36. 图 3 的关键经验图像是: \(k>k^*\) 一侧梯度上界明显, \(m(k)\) 快速趋近 0; \(k<k^*\) 接近边界时 \(m(k)\to\infty\)。
37. 图 4 用 \(a=q=r=1\) 显示同样初始 cost gap 下, \(k(0)>k^*\) 是 linear-exponential, \(k(0)<k^*\) 是更快的 exponential-like。
38. discrete-time LQR 对比部分解释为何 DT-LQR 可有 global PL: Euler 离散化后 stabilizing gain 区间变成有界区间。
39. 对任意固定步长 \(h>0\), scalar DT-LQR 的 \(m_d(k_d,h)\) 在稳定区间两端发散, 因此有正的全局最小 rate。
40. 但当 \(h\to0\) 时, 该最小 rate \(\underline m_d(h)\to0\), 与 continuous-time 无 global PL 一致。
41. 结尾讨论 \(L_1\) regularization 与 proximal gradient flow, 但这部分是 intuition 而不是完整定理。
42. 论文指出若远离最优时 \(|x|\gg1\) 或 \(|\nabla f(x)|\gg1\), proximal dynamics 可近似为普通 gradient flow 加减常数项。
43. 这个 comment 的价值是提示 generalized PL/PLI 也可扩展到非光滑 proximal dynamics, 但作者明确没有给出完整 local analysis。

### 关键技术细节
44. 假设层面: \(f\) real analytic, bounded below, proper; 这些条件保证 gradient flow 不逃逸且可用 Lojasiewicz 定理。
45. 收敛类型: GES 是 \(f(\phi(t,x_0))-\underline f\le (f(x_0)-\underline f)e^{-\mu t}\)。
46. semi-global PL 的关键退化量是 \(\overline\mu_\epsilon\); 若没有 global PL, 存在 \(\epsilon_i\) 使 \(\overline\mu_{\epsilon_i}\to0\)。
47. \(\mathcal K_{\mathrm{SAT}}\) 的饱和值是 \(\sqrt a\), 因此它不可能强迫梯度随着 cost gap 无界增长。
48. high-gain curve 构造用 pole placement: 单输入时令闭环 characteristic polynomial 为 \((s+\rho)^n\)。
49. 多输入情形借用 Sontag 控制理论中的 reduction: 找 \(F,v\) 使 \((A-BF,Bv)\) controllable, 再做单输入 pole placement。
50. Lemma BndGradJ 的证明把 \(B\widetilde K\) 分解成发散 eigenvalue block 和有界 block, 分别分析 \(Y_K\) 与 \(P_K\) 的 Lyapunov 方程极限。
51. 证明里显式假设 unbounded eigenvalues 以同一 rate \(\tilde\rho\) 发散; 多 rate 情形只给出 sketch, 这是可信度扣分点。
52. Lemma GradJUnbd 的证明用接近虚轴的最大实部 eigenvalue \(\lambda_i\), 得到 \(\mathrm{trace}(P_i)\ge -\lambda_Q/(2\mathrm{Re}(\lambda_i))\)。
53. scalar LQR 数值实验固定 \(a=q=r=1\), 图只用于解释机制, 不是大规模 benchmark。
54. 本文没有代码 release、没有第三方复现, 主要证据是数学推导与少量图示。

### 核心结果
55. 结果 1: global PL 与 gradient-flow cost gap 的 global exponential stability 等价。
56. 结果 2: semi-global PL 不等于 global PL; 远端 sublevel set 的最佳指数 rate 必须可能退化。
57. 结果 3: \(\mathcal K_{\mathrm{SAT}}\) 下界给出一种比 global PL 弱、比 pure semi-global PL 更有形状信息的收敛描述。
58. 结果 4: CT-LQR 沿 high-gain curve 的 cost gap 发散但梯度有界, 所以不可能有 global PL。
59. 结果 5: CT-LQR 接近稳定边界时梯度发散, 所以又不能简单归入全局梯度有界的 GLES sufficient condition。
60. 结果 6: 在带稳定裕度的 \(\mathcal G_\delta\) 中, 梯度可以恢复全局有界, 从而 linear-exponential 解释成立。
61. 结果 7: scalar LQR 显示同样 cost gap 可因初始化区域不同而呈现不同收敛 profile。
62. 结果 8: DT-LQR 的 global PL 与 CT-LQR 的 no-global-PL 并不矛盾; sampling step 使稳定集合有界, 但 \(h\to0\) 时 rate 退化。
63. 最反直觉结果是: continuous-time LQR 这种经典问题在 model-free policy optimization 下仍无法满足最强 PL, 即使它有很好的全局最优性结构。

### 明确局限
64. 作者自己的局限: proximal gradient / \(L_1\) 部分只是直觉性讨论, 没有完整理论。
65. 证明局限: high-gain gradient boundedness 的主证明对同 rate 发散 eigenvalues 给出细节, 多 rate 情形只说可类似扩展。
66. 实验局限: 只有 scalar LQR 图示, 没有多维随机系统、真实控制 benchmark 或数值稳定性统计。
67. 复现局限: arXiv source 有图和 tex, 但没有生成图的脚本、配置或 raw numerical logs。
68. 表述局限: 文中有少量 typo/不完整括号, 例如 proximal flow 公式和若干 proof notation, 需要读者自行校正。
69. 与 FEX 关系的局限: 本文研究连续优化里的 gradient flow/LQR, 不研究符号回归、表达式树、PDE residual 或 RL controller。
70. 因此它不能直接证明 FEX 搜索收敛, 只能提供 PL/gradient-dominance 语言和反例模板。

### 跟我们的关系
71. 对 `0616-fex` 不直接撞车: 本文没有 symbolic regression、没有 FEX grammar、没有 PDE solver、没有 expression-tree RL。
72. 直接可用的是 gradient dominance / PL taxonomy: FEX 若写 conditional PG theorem, 不应只说 "PL implies convergence", 而要区分 global、semi-global、local、saturated PL。
73. 本文给 `fex-search-complexity` 的 \(\kappa_{PG}\) 假设一个很好的反例语境: 即使经典 LQR 也可能只有饱和/半全局 rate, 全局指数收敛 claim 很危险。
74. high-gain vs stability-boundary 的二分可类比 FEX: 有些表达式参数区远离好解但 reward gradient 饱和, 有些接近 feasibility 边界时 gradient 爆炸。
75. 如果 FEX controller 使用 softmax policy gradient, 本文提示应记录 region-dependent gradient norm、cost gap 和 empirical PL ratio, 而不是只报告最终 recovery。
76. 如果 FEX 论文要讨论采样/离散化, CT vs DT LQR 的差异是一个有用警告: 离散化或 bounded action space 可能人为制造 global PL-like rate。
77. 该引应放在理论背景或 PG convergence 章节, 不是 FEX 直接 baseline。
78. 最值得引用的 claim 是 "saturated/semi-global PL leads to mixed linear-exponential/exponential convergence profiles"。
79. 差异化空间: 我们可以把本文的连续参数 gradient-flow taxonomy 移植到离散表达式树 PG, 但必须证明 FEX-specific reward geometry 或明确标为 conditional。
80. 对 novelty: 这篇不会预占 FEX discoverability factorization; 它加强的是我们对 \(\kappa_{PG}\) 和 saturated PL 的表述纪律。

## Read by: 0616-fex (2026-06-19)

从 `0616-fex` 视角看, 这篇不是 FEX/SR 直接竞品, 但它是一个很好的理论警告: "gradient dominance/PL" 不能被当成单一开关。FEX 当前的 `fex-search-complexity` idea 想把 discoverability 拆成 quotient cover、PG geometry 和 coefficient feasibility, 其中 PG geometry 常会被写成 \(\kappa_{PG}\) 或 PL-like 条件。本文说明即使在 LQR 这种结构很强的问题上, continuous-time policy optimization 也可能没有 global PL, 只能得到 semi-global/saturated 形式, 并且不同区域的收敛 profile 会改变。

撞车评估: 不撞车。本文没有表达式树、符号等价类、PDE residual grammar、FEX controller 或 coefficient feasibility hardness。它预占的是 "CT-LQR no global PL / saturated PL" 这类控制优化理论, 不是 FEX discoverability factorization。若我们在 FEX 里声称 "policy gradient 满足全局 PL 因而多项式收敛", 本文反而应作为反面 guardrail 引用, 迫使我们把 claim 降为 conditional 或 region-dependent。

可用工具: 记录 empirical PL ratio \(\|\nabla J\|^2/(J-J^*)\)、区分远端饱和区与边界爆炸区、用 \(\mathcal K_{\mathrm{SAT}}\) 语言描述全局慢/局部快的 mixed convergence, 以及把 gradient estimation noise 放进 ISS/siISS/iISS 框架。对 FEX 实验, 这转化为: 除 recovery rate 外, 应记录 reward gap、policy-gradient norm、good-class mass、entropy 和 empirical \(\kappa_{PG}\) 随搜索阶段的变化。
