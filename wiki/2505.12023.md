# Model-X Change-Point Detection of Conditional Distribution (2505.12023)

- **Year**: 2025.05
- **Authors**: Zhuofan Dong (Beijing Normal Univ.), Yiwen Huang (Peking Univ.), Yan Dong (USTC), Mengying Yan / Ziye Tian / Chuan Hong (Duke), Doudou Zhou (NUS, corresponding), Molei Liu (Peking Univ. Health Science Center / BICMR)
- **Venue**: arXiv preprint (PRIME/arxiv template; not yet a named venue — internal comments reference "neurips2025")
- **Keywords**: change-point detection; conditional distribution shift; model-X / conditional randomization test (CRT); distilled CRT (dCRT); neural distillation encoder; latent mixture model + EM; fused lasso; Invariant Risk Minimization (IRM); conditional independence testing
- **One-liner**: 用 model-X CRT + 神经网络蒸馏编码器检测条件分布 p(Y|X) 的变点，在高维/非线性下保持有效的 Type-I 控制，胜过 OLS-CUSUM / KCE / KCD / DP 等基线。

## 精华提炼

### 管线/方法
MEND (Model-X changE-point detectioN of conditional Distribution) 检测的是**条件分布 p(Y|X) 随时间的变化**（即 Y 与 X 的关系何时改变），而非经典变点检测关注的边际分布变化。设定：时间跨度 T，每个时间点 t 观测 n_t 个 i.i.d. 样本 {(Y_i, X_i)}，Y∈ℝ，X∈ℝ^p。零假设 H0: p_{Y|X,R}=p_{Y|X}（即 Y ⊥ R | X，条件独立于时间标签 R）；备择：存在变点 τ 使条件分布改变。本质是一个 **Y ⊥ R | X 的条件独立性检验**。

完整 pipeline：
1. **框架基座 = Conditional Randomization Test (CRT)**（Candès 2018 model-X knockoffs 家族）。CRT 重采样 R_i ~ p(R|X_i)，在**已知/可准确估计 p(R|X)** 前提下给出对检验统计量 S 选择不变的 Type-I 精确控制；p-value = (1+Σ 𝟙[S(D^(k))≥S(D)])/(K+1)。
2. **蒸馏函数 φ（核心创新）**：不同于 dCRT（Liu 2022）手工构造显式蒸馏函数，MEND 用一个**神经网络编码器**（四层全连接，前两层 ReLU）预训练后冻结作为固定映射。目的不只是降维/降计算（dCRT 的动机），而是把 X 投影到**时间不变的低维隐空间**，使变点前后的差异被放大 → 提升检测灵敏度。其它模态可换 CNN（图像）/ GNN（图结构）。
3. **训练 = Latent Mixture Model (LMM) + EM**：为每个样本引入隐段标签 E_i∈{1..K}，y_i = ω_k^⊤φ(x_i)+ε_k，共享编码器 φ + 段特定线性 decoder ω_k。EM：E-step 算责任度 p_ik；M-step 梯度更新 φ、ω，σ_k² 有闭式解。可选 **IRM (Invariant Risk Minimization)** 正则项 L_env=‖∇_ω̄ L_MSE(ω̄·φ)‖² 鼓励 φ 学到时间不变表征。
4. **检验统计量**：冻结 φ 后 Z_i=φ(X_i)，用 **fused-lasso** 损失估计时变系数 {β_t}（fused 惩罚 ‖β_t−β_{t−1}‖ 促分段常数 + ridge + l1）。D_t=‖β_t−β_{t−1}‖²，S(D)=Σ_{top (K-1)} D_t，最大跳变的时间索引即估计变点。
5. **回到 CRT** 用蒸馏后的统计量重采样得到有效 p-value。

**简化变体（单变点场景）**：
- **MEND-mean**：φ={m_0(X), m_1(X)}（变点前后条件均值），统计量 S=max_τ|â_{τ−}−â_{τ+}|，â 为两个 regime 均值的混合系数。计算极省。
- **MEND-repr (文中亦写作 MEND-LAD repr)**：φ={m_0, m_1, X_{B0∪B1}}，追加所选 top-k 坐标以增强线性特征。

### 关键技术细节
- 编码器 φ：4 层全连接 FC，前两层 ReLU；预训练+冻结。fused-lasso 超参（模拟）λ_fuse=1, λ_l1=1, λ_ridge=0.1；Beijing 数据 λ_fuse=1, λ_l1=0.5, λ_ridge=0.5。
- CRT 重采样次数 K=100；显著性 α=0.05（部分 0.10）。
- 模拟：T=100，X=[X_e, X_i] 各 ℝ^50（共 p=100），X_e−0.5~N(0,Σ), Σ_ij=0.5^|i−j|；X_i~N(0,I)。线性场景 Y=α_k^⊤X_e+G(X_e)+ε，G(X)=0.3X_0²+√|X_2|，k_s∈[0,1] 控制段间系数差异；非线性场景每段随机选 f_k∈{f_1..f_5}（含 |·|、√|·|、exp 项）。
- 简化模型模拟：T=10, τ=7, n=1000, 500 次重复；三个 Scenario（低维非线性×2 + 高维线性 p=100）。
- 理论：Proposition（model-X 可交换性）S(D*)|(Y,X) =d S(D)|(Y,X)，依赖生成模型 Y,X|R ~ p_{X|R}(X)·p_{Y|X}(Y) 与 R*|(Y,X)=d R|X=d R|(Y,X)；Theorem（严格有效性）lim_{n→∞} P(拒绝)≤α（**注意是渐近**，非有限样本，因为 p(R|X) 需估计）。
- 基线：线性 OLS-CUSUM、MOSUM（对残差跑单变量检测）；非线性 KCE / KCD（kernel，Nie & Nicolae 2022，500 bootstrap，SCAD 筛变量，带宽在 S_h 中调）；高维 DP（dynamic programming，Rinaldo 2021）。

### 核心结果
- **线性/非线性模拟**：MEND 在弱信号（k_s 小）下仍维持最低 Type-II error 与低 p-value；KCD 常 p-value>0.1 检不出，KCE 与 MEND 相当但 Type-I 差。
- **简化模型（δ_1=0.3，Scenario 1 Type-I）**：MEND-mean **0.046** vs KCD **0.48** / KCE **0.4** / OLS-CUSUM **0.1** —— 基线严重 Type-I 膨胀，MEND 控制良好。
- **Scenario 3（δ_3=3）**：MEND-mean power **0.959** / 定位准 **0.808**，MEND-repr **0.842** / **0.726**，远超 KCD/KCE（power ~0.74–0.76，定位仅 0.08–0.18）。Scenario 3 δ_3=0 Type-I：MEND-mean 0.054 / repr 0.057 vs KCD 0.28 / KCE 0.24。
- **Scenario 2（δ_2=0）**：MEND-mean Type-I 0.054，其余全 >0.45（covariate shift 无结构变化时基线全线失效）。
- **真实数据 1 — Beijing PM2.5 (2010–2014)**：41,757 样本 p=10，按天聚合。检出的四个主峰（2010.10.11 / 2012.01.20 / 2013.01.24 / 2014.02.27）与冬季供暖季起止吻合；另检出 2014.10.26 对应 APEC 峰会限污政策期。
- **真实数据 2 — Duke EHR 急诊 2022**：二元住院结局，n=1000 子样本，4–12 月。伪模拟 Type-I=0.029（验证 R|X 建模质量）。真实数据 MEND 估计变点=11 月（p=0.029，对应流感季），DP 同点估计；KCE(6, p=0.742)/KCD(8, p=0.6)/OLS-CUSUM(11, p=0.55) 均检不出显著变点。

**反直觉点**：kernel 方法（KCE/KCD）在存在**协变量漂移但无结构变化**时 Type-I 严重膨胀（>0.45–0.63），这正是条件变点检测比边际变点更难的核心陷阱；MEND 靠 model-X 重采样 R|X 直接规避。

### 明确局限
作者自述：(1) NN 超参选择敏感、非平凡；(2) 深度蒸馏 → 缺可解释性，无 X–Y 依赖的显式函数形式；未来想区分 structural vs numerical 变点。
我读出的额外局限：
- **无代码/artifact 发布**，图全为打包 .png，早期草稿里部分表格是占位符（注释掉的 "x" 值）；EHR 数据私有、n=1000 子样本很小、只检出单一变点。
- **有效性完全依赖正确建模 p(R|X)（model-X 假设）**——观测时序里这是命门，全文只有一个伪模拟(Type-I 0.029)做检查。
- 定理是**渐近** lim_{n→∞}，与正文暗示的 CRT "精确/非渐近"有张力（CRT 仅在 p(R|X) 精确已知时才 exact）。
- 手稿粗糙（typos；LMM 同时指 Latent Mixture Model 与 linear mixed model；MEND-LAD/MEND-repr 命名不一致；源码残留蓝色 TODO；重复 \usepackage{natbib}）——像未经严审的 preprint。
- 无标准 benchmark 数据集，两个真实案例都是"检出的点恰好对上已知事件"式的事后印证，非盲测/lead-time 评估。

### 跟我们的关系
对 **0710-causal-scs**（因果效应不稳定性作为强对流预警指标）课题：

- **不撞车（目标不同）**：MEND 做的是"何时/是否发生了条件分布变化"的**有效事后检验**（Type-I 精确控制），而本课题要的是"事件发生前，因果估计的脆弱性/不稳定性作为**领先预警**信号"。MEND 不做 lead-time、不用因果 DAG 结构、不测跨方法/跨窗口的 estimator fragility。核心 claim 无重叠。
- **是可用工具（而非竞争）**：
  1. MEND 可作为课题"**突变型指标**"里 conditional change-point 检测的一个原理化候选——相较 idea 现列的 PELT/BOCPD/CUSUM（多针对边际或均值），MEND 直接检测 p(Y|X) 的条件变化并给**有效 p-value**，且不需线性/低维假设。
  2. **正面回应本 idea 的 "strongest objection"**（"不稳定性可能只是估计噪声/有限样本伪影"）：MEND 的 model-X CRT 通过重采样 R|X 把"这是真变化还是估计伪影"变成一个 Type-I 可控的检验——这正是课题最缺的、区分真实 regime-change 与 estimator artifact 的模板/机制。可借鉴 CRT/knockoff 式重采样来给 fragility 信号做 surrogate/null 校准（与 idea 里的 IAAFT surrogate 互补，且更贴合条件依赖）。
- **该引**：作为 conditional change-point detection + valid inference for conditional shift 的相关方法引用；尤其它与课题已精读的 **[[2407.07290]]**（Gao 2025, Causal Discovery-Driven CPD）同属"causal / conditional CPD"谱系（MEND 正引用了 2407.07290），可一并纳入相关工作梳理。
- **差异化仍然成立**：课题的独特点（estimator fragility 作为事件特异的 latent-process-activation 诊断、跨方法分歧、快变中尺度对流迁移）均不被 MEND 覆盖；MEND 反而强化了"用 model-X 检验把不稳定性做成有效统计量"这条可执行路径。

**本次审读视角（因果效应不稳定性 → 强对流预警）下的结论：**

- **撞车评估：不撞车。** MEND 是条件分布 p(Y|X) 的**有效变点检验**（model-X CRT，Type-I 精确控制），关注"是否/何时已发生结构变化"；本课题关注"事件前因果估计的不稳定性作为**领先**预警"。二者目标、时序方向、贡献类型均不同，MEND 不涉及 lead-time、DAG 结构、跨方法 fragility。核心 claim 无重叠。
- **可用工具（3 点）：**
  1. 作为课题"突变型指标"里 conditional change-point 检测的原理化候选，优于 idea 现列的 PELT/BOCPD/CUSUM（后者多针对边际/均值），能处理条件、非线性、高维并给有效 p-value。
