# MODNO: Multi-Operator Learning With Distributed Neural Operators (2404.02892)

- **Year**: 2024.04
- **Authors**: Zecheng Zhang (Florida State University, Department of Mathematics)
- **Venue**: arXiv
- **Keywords**: Multi-Operator Learning, Deep Operator Network (DON), Distributed Training, Neural Operator, Scientific Machine Learning, PDE surrogate, Chen-Chen type operators
- **One-liner**: 提出分布式训练方法 MODNO，使参数极少的单个 DON 能同时学习多个算子，且在不增加平均成本的前提下，在 11/16 个 case 中超越独立训练的 SOL DON。

## 精华提炼

### 管线/方法
1. **输入**：多个算子 G_i: U_i → V_i，每个算子有自己的训练数据（输入函数 u_{i,p} 的传感器离散化 û_{i,p} 和对应输出函数在 x_i 点的值）
2. **网络结构**：基于 Chen-Chen 型神经算子（DON），包含 Branch Net（编码输入函数）和 Trunk Net（输出函数基）
3. **分布式策略**：
   - Trunk Net 的 β_i 参数：每个算子有专用的一组，只用该算子的数据训练（local loss L_i）
   - Branch Net 的 α 参数：所有算子共享，用全部数据训练（global loss L = Σ L_i）
4. **优化流程** (Algorithm 1)：每个 epoch 内，先用各算子的 local loss 梯度下降更新各自的 β_i，再用 global loss 更新共享的 α
5. **成本控制**：引入 q ∈ [0,1] 控制用于训练共享结构的数据比例，q < 1 时成本低于训练 N_op 个独立 DON
6. **输出**：对每个算子 G_i 产生预测 G_θ(û)(x) = Σ_k a_k(û; α) b_{k,i}(x; β_{k,i})

### 关键技术细节
- **网络基础**：Stacked DON（Lu et al. 2021），Branch Net 为单隐层 MLP（width M），Trunk Net 为单隐层 MLP（width K），激活函数为 Tauber-Wiener 函数 g
- **传感器**：输入函数在 N_s 个传感器点 {y_i} 处离散化
- **损失函数**：相对 L2 误差（文中记为 relative L2 error，百分比形式）
- **训练参数**：epoch = N_I，每个算子有独立学习率 η_i（trunk），共享学习率 η（branch）
- **对比基线**：训练 N_op 个结构相同的独立 DON，每个用完整数据
- **成本度量**：反向传播和正向传播总次数（而非参数量），考虑了每个算子数据量不同
- **实验规模**：5 个数值实验，涵盖 wave/Klein-Gordon/Sine-Gordon、porous media（不同阶数）、parabolic/Viscous Burgers/Burgers、KdV/Cahn-Hilliard/Advection、以及混合 4 方程
- **初始条件生成**：Fourier 级数型函数，参数均匀采样；部分实验用高斯混合型初始条件
- **测试特殊性**：测试空间网格与训练不同（测试 discretization invariance），实验五测试时间外推

### 核心结果
| 实验 | 算子数 | MODNO 超越 SOL 的 case | 数据减少至 70% 仍胜 SOL |
|------|--------|----------------------|----------------------|
| Exp 1: Wave/Klein/Sine | 3 | 3/3 (100% data) | Sine 持续胜出 |
| Exp 2: Porous Media m=2,3,4 | 3 | 3/3 (100% data) | m=2 在 80% 仍胜 |
| Exp 3: Parabolic/VB/Burgers | 3 | 2/3 (100% data, Burger 略差) | Parabolic 在 70% 仍 0.3% vs SOL 1.48% |
| Exp 4: KdV/CH/Advection | 3 | 1/3 (仅 KdV 胜出) | CH 和 Advection 变差 |
| Exp 5: 4 方程混合 | 4 | 2/4 (CH, Sine-Gordon) | 其余接近 SOL |

**关键数字**：
- Wave: MODNO 2.23% vs SOL 2.59%
- Parabolic: MODNO 0.22% vs SOL 1.48%（**提升 6.7 倍**）
- Porous Media m=2: MODNO 1.14% vs SOL 2.52%（**提升 2.2 倍**）
- KdV: MODNO 7.62% vs SOL 8.39%
- 反例：Burgers MODNO 5.34% vs SOL 4.82%，Advection MODNO 4.5% vs SOL 2.09%，CH MODNO 1.13% vs SOL 0.28%

**Surprising 发现**：
1. 减少训练共享结构的数据量（q 从 1 降到 0.7），很多 case 的精度几乎不降，作者猜想共享结构容易过拟合
2. MOL 可以帮助数据稀缺的算子——用其他算子的数据辅助学习，这是一个新的 MOL 优势
3. 某些实验（Exp 4）中 KdV 和 Advection 误差随数据减少反而下降，支持过拟合假说

### 明确局限
- **不能外推到未见算子**：与 PROSE 等 foundation model 不同，MODNO 只能学习训练集中已有的算子，不能 zero-shot 预测新算子（作者自己承认）
- **假设相同输入函数空间**：所有算子必须有相同的输入函数离散化方式（相同传感器点），这是"same input function space"假设
- **非 discretization-invariant**：虽然提到可以扩展到 BelNet 实现 discretization invariance，但当前版本没有
- **只有 1D PDE**：所有数值实验都是 1D 空间 + 时间，没有 2D/3D 空间问题
- **单作者论文**：只有一位作者，没有机构合作，评审强度存疑
- **没有真机实验**：纯数值仿真
- **没有标准 benchmark 对比**：只跟独立 DON 比，没跟其他 MOL 方法（PROSE, ICON, PDEformer）比精度，声称"参数少得多"但没有给出具体参数量
- **没有开源代码**：只说"upon publication"会放到 Google Colab
- **没有理论收敛保证**：只有直观 motivation，没有 convergence proof

### 撞车评估
不撞车。MODNO 做的是 neural operator 的多算子联合训练（学习 PDE 解算子），而 0616-fex 做的是 finite expression method（通过符号回归发现控制方程）。两者位于 SciML 的不同分支：一个是 surrogate modeling / operator learning，一个是 equation discovery / symbolic regression。

### 可借鉴的思路
1. **分布式共享-独立架构**：FEX 在多系统 expression discovery 中，可以共享符号结构的搜索过程（类似共享 branch），但对每个系统独立优化系数（类似独立 trunk）。这比直接用 FEX 跑每个系统更高效。
2. **成本分析框架**：用 back-propagation 次数而非参数量作为多任务学习成本度量，适用于 FEX 多方程发现的效率对比。
3. **MOL 提升数据效率的实验证据**：该文实验证明用其他算子的数据可以提升目标算子的预测精度——如果 FEX 做 multi-system discovery，引用此文可支持"联合学习比单独学习好"的论点。

### 不建议引用的原因
- 方法论正交（operator learning vs expression discovery）
- 无理论保证，实验局限在 1D 简单 PDE
- 单作者无独立验证，可靠性 B 级
