# DPOT: Auto-Regressive Denoising Operator Transformer for Large-Scale PDE Pre-Training (2403.03542)

- **Year**: 2024 (ICML 2024)
- **Authors**: Zhongkai Hao, Chang Su, Songming Liu (Tsinghua), Julius Berner, Anima Anandkumar (Caltech), Chengyang Ying, Hang Su, Jian Song, Jun Zhu (Tsinghua/RealAI)
- **Venue**: ICML 2024
- **Keywords**: Neural Operator, PDE Pre-training, Fourier Attention, Auto-Regressive Denoising, Foundation Model, Scientific Machine Learning, Operator Learning, Multi-PDE Transfer
- **One-liner**: 提出基于 Fourier attention 的自回归去噪预训练策略 DPOT，在 10+ PDE 数据集（100k+ trajectories）上预训练最大 1B 参数的 PDE 基础模型，零样本和微调性能均达到 SOTA，误差降低最高 52%。

## 精华提炼

### 管线/方法

完整 pipeline：多源 PDE 数据集 → 数据预处理（padding/masking/balanced sampling）→ Patchification + Temporal Aggregation → Fourier Attention Layers（核心）×L → 自回归去噪预训练 → 下游任务微调

1. **数据预处理**：统一分辨率到 H=128（低分辨率上采样插值，高分辨率随机采样下采样）；沿 channel 维度 padding（填 1）统一变量数；不规则几何添加额外 mask channel
2. **Balanced Data Sampling**：按 $p_k = w_k / (K|\mathcal{D}_k| \sum w_k)$ 概率采样，长轨迹数据集（DR, SWE）权重 w=3，其余 w=1
3. **Patchification + Temporal Aggregation**：卷积 patch embedding + 可学习位置编码 $W_p(x_i,y_j,t)$；Fourier features $\gamma$ 加权的时序聚合 $\mathbf{z}_{agg} = \sum_t W_t \cdot \mathbf{z}_p^t e^{-i\gamma t}$
4. **Fourier Attention Layer**：核心层，对 spatial features 做 FFT → 2-layer shared MLP（$W_1, W_2 \in \mathbb{R}^{d_z \times d_z}$）+ GELU → IFFT → Group Normalization → FFN（channel mixing）
5. **Multi-head 结构**：特征分 h 组，每组独立小 MLP 变换，计算复杂度 $O(d_z^2/h)$
6. **预训练目标**：自回归预测下一帧，输入注入高斯噪声 $\varepsilon \sim \mathcal{N}(0, \epsilon\| \mathbf{u}^{<t}\|I)$，MSE loss
7. **微调**：替换 FFT/IFFT 维度适配下游数据（如 2D→3D），复用 Fourier attention 层参数

### 关键技术细节

- **模型规格（5 个尺度）**：
  - Tiny: 512 dim, 4 layers, 4 heads, 7M params
  - Small: 1024 dim, 6 layers, 8 heads, 30M params
  - Medium: 1024 dim, 4096 MLP dim, 12 layers, 8 heads, 122M params
  - Large: 1536 dim, 6144 MLP dim, 24 layers, 16 heads, 509M params
  - Huge: 2048 dim, 8092 MLP dim, 27 layers, 8 heads, 1.03B params
- **训练配置**：AdamW optimizer, lr=$1\times10^{-3}$, One-cycle schedule, 1000 epochs, warmup 200 epochs, weight decay $1\times10^{-6}$, $\beta=(0.9,0.9)$, batch size 160, 8×A800(80GB) GPUs
- **预训练数据**：12 个数据集，4 个来源（FNO/PDEBench/PDEArena/CFDBench），100k+ trajectories
- **PDE 类型**：Navier-Stokes（多种粘度）、Compressible Navier-Stokes、Shallow-Water、Diffusion-Reaction
- **输入形式**：$T=10$ 帧历史预测下一帧，分辨率统一 128×128
- **噪声水平**：最优 $\epsilon=5\times10^{-5}$（消融实验验证）
- **Patch size**：最优 P=8（消融实验：P=2 时性能极差，L2RE=0.369 vs P=8 时 0.0174）
- **AFNO 差异**：不使用 soft-thresholding 稀疏约束（消融证明稀疏化对 PDE 有害：AFNO train loss 0.0664 vs DPOT 0.0127）

### 核心结果

**主实验（Table 1，L2RE，越低越好）**：

| 对比维度 | 关键结果 |
|----------|---------|
| Small models (≤30M) 零样本 | Ours-S 在 9/12 数据集上 SOTA，FNO-1e-4 从 0.116→0.0553（52% 提升） |
| Large models 零样本 | DPOT-H(1B) 在几乎所有数据集上最好，FNO-1e-5 从 0.055→0.0174（68% 提升） |
| Fine-tune (L-500, 500M) | 9/12 任务 SOTA，8/12 任务比零样本方法降低 50%+ 误差 |
| vs MPP-L(400M) | PDB-DR 上 MPP-L 更好（0.0098 vs 0.0232），其余数据集 DPOT 全面领先 |

**下游任务迁移**：
- 2D 高分辨率湍流 (PDB-turb)：DPOT-FT L2RE=0.135 vs FNO 0.193（30% 提升）
- 3D Navier-Stokes：L2RE 从 0.410→0.226（45% 提升），仅用 2D 预训练数据
- 稳态 PDE (CNO)：L1-Median 从 0.0357→0.0230（36% 提升）

**Scaling 行为**：7M→500M，零样本 L2RE 从 0.06→0.028，大致遵循 scaling law；DPOT 7M 性能优于 AViT 30M

**超分辨率**（SuperBench）：DPOT-pretrained RFNE=0.46% vs EDSR 0.57%

**长轨迹稳定性**：500 步时 DPOT-pretrained L2RE=0.0385 vs 从头训练 0.0912（57.8% 降低）

**Kolmogorov 湍流（混沌）**：DPOT-pretrained 全轨迹 L2RE=0.335 vs DPOT 0.822 vs FNO 1.04，说明预训练对混沌系统也有帮助

**推理速度**：DPOT-L 单步 5.87ms（A800），Tiny 0.408ms

### 明确局限

1. **作者自述局限**：神经网络预测缺乏可解释性，在安全敏感场景有风险；仅做 empirical/numerical 研究，理论分析不够深入
2. **非通用 PDE 覆盖**：只覆盖了时间演化类 PDE（Navier-Stokes、SWE、DR 等），缺少椭圆型/双曲型/多物理场耦合等广泛类别
3. **2D 为主**：预训练全是 2D 数据，3D 只在微调阶段验证，未做 3D 预训练
4. **固定输入长度 T=10**：时间聚合层设计假设固定历史帧数，对变长轨迹的灵活性有限
5. **PDB-DR 弱于 MPP**：在 diffusion-reaction 数据集上 MPP-L(400M) 零样本更好，说明 Fourier attention 并非对所有 PDE 类型都最优
6. **分辨率泛化需要特殊处理**：由于卷积 patchification 层存在，需要像 CNO 那样修改才能保持分辨率泛化（文中提到但未深入解决）
7. **未开源 1B 模型**：代码仓库只有较小模型，最大模型权重未公开
8. **跟我们的关系**：本工作是 PDE 预训练的 foundation model 范式，与 0616-fex 课题（如果聚焦 feature extraction / representation learning）在"从多样数据中学习通用表征"这一目标上有方法论重叠——去噪预训练 + Fourier attention 的组合可作为特征提取 backbone 的参考

### 跟我们的关系

对于 0616-fex 课题：

1. **不撞车**：DPOT 聚焦于 PDE 神经算子的预训练，目标是学物理系统的 solution operator。0616-fex 如果聚焦于通用特征提取（feature extraction），则方向不同。但如果 0616-fex 涉及从物理数据中学习表征，则存在方法论层面的参考价值。

2. **可用的工具/方法**：
   - Fourier Attention Layer 的设计（FFT→MLP→IFFT）可作为一种高效的 spatial feature extractor，适合处理网格化物理数据
   - Auto-regressive denoising 策略（输入加噪→预测下一帧）可作为一种自监督预训练范式
   - Balanced data sampling 公式 $p_k = w_k/(K|\mathcal{D}_k|\sum w_k)$ 可解决多源数据不平衡问题
   - Temporal aggregation with Fourier features 的时间特征融合方法

3. **差异化空间**：
   - DPOT 的 Fourier attention 限于规则网格（依赖 FFT），扩展到不规则几何/图结构是一个方向
   - 去噪预训练仅使用了 Gaussian 噪声，更复杂的 corruption strategy（masking、multi-scale noise）可能有提升
   - 时间聚合层设计较简单（加权求和），引入 recurrent/attention-based temporal modeling 是改进方向
   - 预训练任务单一（仅 next-step prediction），多任务预训练（forward + inverse + super-resolution）可能学到更好的表征

## Read by: 0616-fex (2026-06-24)

### 撞车评估
**不撞车。** DPOT 的核心贡献是 PDE 神经算子的预训练范式——用 Fourier attention + 自回归去噪在多个 PDE 数据集上预训练，然后微调到下游物理任务。0616-fex 如果关注通用特征提取（feature extraction），两者在目标函数和输出空间上完全不同。DPOT 的输出是物理场（velocity/pressure/density fields），0616-fex 的输出是特征表示。

### 可用工具/方法
1. **Fourier Attention Layer**：FFT→Shared MLP→IFFT 的 spatial mixing 设计可作为处理规则网格数据的通用特征提取器，计算效率高（$O(d_z^2/h)$），参数利用率好
2. **Auto-Regressive Denoising**：输入注入小噪声的自监督预训练策略是一种有效的正则化，等价于在输入空间做数据增强，可迁移到其他时序特征学习场景
3. **Balanced Data Sampling**：多源异构数据的平衡采样公式，简洁有效
4. **Temporal Aggregation with Fourier Features**：用可学习的 Fourier features 做时序信息融合，相比简单的 concat/sum 更能捕捉周期性动态

### 差异化方向
- DPOT 的 Fourier attention 依赖规则网格（FFT 要求），0616-fex 可以关注更通用的不规则几何/图结构上的特征提取
- 去噪策略可以扩展：multi-scale noise、structured dropout、masked reconstruction
- 时序聚合可以引入 attention 机制，而非固定的加权求和
- 如果 0616-fex 有物理约束需求，可以借鉴 DPOT 的思路做 physics-informed 的特征学习
