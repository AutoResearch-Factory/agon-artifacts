# Symbolic Graph Networks for Robust PDE Discovery from Noisy Sparse Data (2603.22380)

- **Year**: 2026.03
- **Authors**: Xingyu Chen, Junxiu An, Junjian Guo, Yuqian Zhou; Chengdu University of Information Technology
- **Venue**: arXiv
- **Keywords**: PDE discovery, graph neural networks, symbolic regression, PySR, noisy sparse data, message passing, weak-form alternative, PDE-Net 2.0
- **One-liner**: 用 GNN 消噪表征加 PySR，从稀疏噪声数据发现 PDE/更新式。

## 精华提炼

### 管线/方法
1. 输入是稀疏、带噪的时空观测场，形式化为 $\tilde U_{i,j}=u(x_i,t_j)+\epsilon_{i,j}$。
2. 目标是从观测中恢复稀疏、可解释的动力学表达式 $\hat{\mathcal F}$。
3. 作者把候选库写成 $\mathcal D=\{x,t\}\cup\{\tilde U\}\cup\Phi_\theta(\tilde U)$，也就是显式坐标、观测状态和 SGN 学出的 latent features。
4. 这使模型不只局限于 $u_t=F(u,u_x,u_{xx},...)$ 的强形式 PDE，也可以直接恢复 $u=f(x,t)$ 这种闭式解或离散更新式。
5. 核心框架分两块：Graph Neural Simulator 和 Symbolic Inverse Analyzer。
6. Graph Neural Simulator 把空间网格点映射为图节点 $\mathcal V=\{1,\dots,N_x\}$。
7. 边 $\mathcal E$ 由空间近邻关系建立，表达局部几何连接。
8. 每层消息传递采用 $h_i^{l+1}=\phi^{l}(h_i^l,\oplus_{j\in\mathcal N(i)}\psi^l(h_i^l,h_j^l))$。
9. $\psi$ 和 $\phi$ 都是小 MLP，聚合函数是 sum 或 mean 这类 permutation-invariant aggregation。
10. 作者把 message passing 解释为 mesh-free 的非局部积分算子，用邻域聚合替代显式数值微分。
11. 该设计针对的是传统 PDE discovery 中的两个痛点：强形式数值微分放大噪声，弱形式积分方法又需要密集网格做 quadrature。
12. 为了冷启动稳定，作者先对原始观测做 Savitzky-Golay polynomial filtering。
13. S-G filtering 的作用是压制高频随机噪声，同时保留几何趋势，给 GNN 一个 derivative-friendly initialization。
14. 为了避免 GNN 只记住特定噪声模式，训练时还加入动态 Gaussian noise injection。
15. noise injection 的强度按估计的观测噪声水平自适应设置，逼迫 message passing 保留扰动下稳定的 latent operator。
16. Symbolic Inverse Analyzer 使用 PySR，这是 genetic-algorithm-based symbolic regression 工具。
17. 符号发现分两步：先蒸馏 message function，再蒸馏 node update function。
18. Stage I 抽取成对节点状态 $(h_i,h_j)$ 和 latent message $e_{ij}$，用 PySR 拟合 $e_{ij}=f_{\text{msg}}(h_i,h_j)$。
19. 作者把 message latent dimension 限制到很低，通常是 1 或 2，借此控制下游符号搜索空间。
20. Stage II 在 message 已被符号化后，拟合 $u(t+\Delta t)\approx f_{\text{upd}}(u(t),\sum_j f_{\text{msg}}(h_i,h_j))$。
21. 最终解释式是 $f_{\text{upd}}\circ \text{Agg}\circ f_{\text{msg}}$，不是一个不可解释的黑盒 GNN。
22. 理论段落声称 message aggregation 在连续极限下近似 Fredholm-type integral operator。
23. 作者用 Riemann-Lebesgue lemma 说明高频噪声经邻域积分会衰减。
24. 这套论证更多是机制解释，不是严格的 finite-sample recovery theorem。

### 关键技术细节
25. 三个实验系统：2D wave equation、2D convection-diffusion equation、2D incompressible Navier-Stokes Taylor-Green vortex。
26. 数据统一在 $32\times32\times100$ 的时空网格上生成。
27. 噪声是 Gaussian white noise，标准差按 clean data 的噪声比例 $\eta$ 缩放。
28. Wave equation 使用 $u(x,y,t)=\sin(\pi x)\sin(\pi y)\cos(\pi t)$。
29. Wave speed 设为 $c^2=0.5$，所以 $v_t=c^2\Delta u=-\pi^2u\approx-9.87u$。
30. Wave 实验比较 PDE-Net 2.0 与 SGN；PDE-Net 在低噪声下可学到局部 ODE，但在 0.2% 噪声出现 NaN。
31. Convection-diffusion 使用 moving Gaussian packet。
32. 其参数是 $c_x=0.18$、$c_y=0.09$、$\nu=0.002$、$\sigma_0=0.1$。
33. 对应真方程是 $u_t=-0.18u_x-0.09u_y+0.002(u_{xx}+u_{yy})$。
34. PDE-Net 在 convection-diffusion 的 1.0% 噪声下发现负扩散项，MSE 爆到 $3.84\times10^4$。
35. Navier-Stokes 使用 Taylor-Green vortex，$u=e^{-2\nu t}\cos(x)\sin(y)$，$v=-e^{-2\nu t}\sin(x)\cos(y)$。
36. Navier-Stokes 的 viscosity 设为 $\nu=0.05$，时间步长 $\Delta t=0.1$。
37. 理论单步衰减系数是 $\lambda=e^{-2\nu\Delta t}=e^{-0.01}\approx0.9900498$。
38. 作者没有在 tex 中给出足够完整的训练超参数，例如 MLP 层数、hidden width、PySR population、operator set、训练步数和随机种子。
39. 论文声称代码公开在 `https://github.com/CXY0112/SGN`，但本文审读没有下载 GitHub，只基于 arXiv tex 和 arxiv_tool 结果判断。

### 核心结果
40. Wave 0.0% 噪声：PDE-Net MSE $5.91\times10^{-5}$，SGN MSE $2.54\times10^{-4}$。
41. Wave 0.2% 噪声：PDE-Net 产生 NaN，SGN 发现 $\sin(3.1416x)\sin(3.1416y)\cos(3.1416t)$，MSE $1.38\times10^{-10}$。
42. Wave 0.5% 噪声：SGN 仍恢复接近 $\pi$ 的闭式周期结构，MSE $2.97\times10^{-6}$。
43. Wave 10.0% 噪声：SGN 仍抓到部分周期骨架，但 MSE 升到 $7.84\times10^{-2}$，表达式已有明显补偿项。
44. Convection-diffusion 0.0% 噪声：PDE-Net MSE $5.38\times10^{-2}$，SGN MSE $5.56\times10^{-3}$。
45. Convection-diffusion 0.5% 噪声：PDE-Net MSE $4.03\times10^{-2}$，SGN MSE $1.43\times10^{-3}$。
46. Convection-diffusion 1.0% 噪声：PDE-Net 的负扩散使 rollout 崩溃，SGN 仍为 $3.06\times10^{-3}$。
47. Convection-diffusion 10.0% 噪声：SGN MSE $1.42\times10^{-2}$，高噪声下仍能给出稳定离散更新式。
48. Navier-Stokes 0.1% 噪声：SGN 学到 $u_{t+1}\approx0.9900558u_t$，相对理论 $\lambda=0.9900498$ 的误差小于 0.0006%。
49. Navier-Stokes 0.5% 和 1.0% 噪声：SGN 仍给出 $0.9900131$ 与 $0.9897041$ 一类接近理论的衰减系数。
50. Navier-Stokes 10.0% 噪声：MSE $3.93\times10^{-2}$，表达式退化但仍保留速度分量差分型 message。
51. 最有说服力的结果不是“恢复连续 PDE 原式”，而是在噪声下恢复稳定的闭式解或离散状态转移。
52. 这也意味着与传统 PDE discovery 的 claim 不完全等价：它有时绕开了微分方程本身，转而拟合 solution form 或 one-step map。

### 明确局限
53. 作者自己承认 message representation 维度太低会限制复杂耦合物理效应。
54. 作者自己承认 GNN 非局部聚合可能过度平滑，遇到 shock、discontinuity、sharp interface 时会损伤梯度保真度。
55. 实验全部是合成 PDE 数据，没有真实传感器数据或实验物理数据。
56. 数据系统较窄：standing wave、moving Gaussian packet、Taylor-Green vortex 都有很强的解析结构。
57. Wave 和 Navier-Stokes 结果中，SGN 经常直接恢复 closed-form solution 或 decay multiplier，而不是 recover governing PDE in strong form。
58. 这种能力对预测有价值，但对“发现真实局部物理 law”的解释要更谨慎。
59. 与 PDE-Net 的比较存在不完全公平：PDE-Net 输出连续强形式 PDE，SGN 输出离散更新式或闭式解，二者结构目标不同。
60. Navier-Stokes 部分直接省略 PDE-Net 定量比较，理由是 pressure projection 非局部，baseline 不适合；这削弱了横向比较力度。
61. 论文没有给出标准 SR benchmark 或 PDE discovery benchmark 上的广泛对比，例如 FEX、SINDy variants、weak-form methods、PINN-based discovery、Transformer/RL SR。
62. 没有报告随机种子方差、置信区间、失败率、表达式复杂度分布或 PySR 搜索预算。
63. 没有做真实 sparse irregular sensor placement 的系统评估；图构建细节也不够充分。
64. 理论分析是合理的 inductive-bias explanation，但没有证明在有限噪声、有限样本下能一致恢复真方程。

### 跟我们的关系
1. `0616-fex` 是 FEX / RL for symbolic regression 方向，核心是用 RL controller 在有限表达式空间中搜索解析表达式，尤其面向 PDE 求解和物理 law discovery。
2. 这篇 SGN 不使用 RL controller，也没有 FEX 的 $S_k$ approximation theory，因此不直接撞 FEX 的主方法 claim。
3. 但它与 FEX 的“从数据发现物理定律”和“噪声/稀疏数据鲁棒 SR”方向局部撞车。
4. 如果 `0616-fex` 要做 noisy sparse PDE law discovery，那么 SGN 是近邻 baseline：GNN 先做鲁棒特征/消息抽取，再用 PySR 做表达式蒸馏。
5. 如果 `0616-fex` 仍聚焦 PDE residual solver，即从 PDE/BC/IC 搜索解析解，SGN 更像可借鉴的工具，而不是直接竞品。
6. 最可用组件是“生成与检验/蒸馏分离”：GNN 先形成低维 latent message，PySR 再独立拟合 message/update 函数。
7. FEX 可以借鉴低维 message bottleneck，把复杂 PDE residual 分解成局部 interaction law 与 global update law，减少表达式搜索难度。
8. FEX 也可以借鉴 S-G warm start 和 adaptive noise injection，用于从实验数据发现 physical laws 的噪声鲁棒版本。
9. 与 `0616-fex` landscape 的 gap #7 高度相关：FEX 当前多数实验是干净 PDE 求解，SGN 正面处理 noisy sparse observations。
10. 与 gap #4 也相关：SGN 的 GNN simulator 可作为 FEX controller 的 warm-start 或 expression prior generator。
11. 近 6 个月相邻工作里，ANN-PYSR (2506.17908) 用 attention neural network + PySR 处理 sparse/highly noisy PDE discovery；Weak-PDE-Net (2603.22951) 用 weak formulation + differentiable symbolic networks；SymTorch (2602.21307) 做 neural symbolic distillation。
12. 因此 SGN 的 novelty 不在“神经网络 + PySR”本身，而在 graph message passing 被明确当作非局部积分/消噪的 PDE discovery 前端。
13. 该引不该引：应引。如果 FEX 论文涉及 noisy data、PDE discovery、symbolic distillation、GNN prior 或 physical-law extraction，应把 SGN 放在近邻方法中。
14. 差异化建议：FEX 可以强调 RL 搜索机制、表达式空间理论、PDE residual objective、可验证的搜索复杂度或 e-graph/LLM prior，而不要只声称“神经表征 + SR 更鲁棒”。

## Read by: 0616-fex (2026-06-16)

从 `0616-fex` 视角看，这篇是“GNN 表征 + PySR 蒸馏”的 PDE discovery 近邻，而不是 FEX/RL controller 的直接竞品。它和 FEX 的交集在 physical law discovery、表达式可解释性、noisy sparse observations；差异在于 SGN 的核心搜索不是 RL，也没有 FEX 的有限表达式空间理论，而是先学习低维 message/update 表征，再交给 PySR 拟合。

撞车风险是中低。若我们的 FEX idea 是做干净 PDE residual solver、搜索复杂度理论、LLM/预训练模型引导 controller、e-graph 剪枝或多尺度频率先验，SGN 不覆盖。若我们的 idea 是“FEX for noisy experimental PDE discovery”，SGN 就是必须比较的 baseline，因为它正面 claim 了噪声和稀疏采样鲁棒性。

可用工具主要有三类：第一，S-G filter + adaptive noise injection 可移植到 FEX 的 data-to-law pipeline；第二，message bottleneck 可作为表达式空间压缩方式，先学局部 interaction，再用 FEX/PySR 搜索符号形式；第三，closed-form solution 与 one-step update 的分流值得借鉴，FEX 评测时也应区分“恢复 PDE 本身”和“找到能预测的解析解/更新式”。

差异化上，FEX 应避免只复述“神经网络消噪后做符号回归”。更强的 claim 是：在同一 noisy sparse PDE discovery setting 下，用 RL/FEX 的表达式空间、复杂度惩罚和独立 verifier 给出比 PySR 蒸馏更稳定、更可审计的表达式；或者把 SGN 式 graph latent message 作为 FEX controller 的 prior/warm-start，并证明搜索预算或失败率下降。
