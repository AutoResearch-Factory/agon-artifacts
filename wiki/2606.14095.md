# Lyapunov-Based Sample Complexity Analysis for Weakly-Coupled MDPs (2606.14095)

- **Year**: 2026.06
- **Authors**: Tianhao Wu (UW-Madison ISyE), Matthew Zurek (UW-Madison CS), Weina Wang (CMU CS), Qiaomin Xie (UW-Madison ISyE)
- **Venue**: COLT 2026 (accepted for presentation)
- **Keywords**: Weakly-coupled MDPs, restless bandits, sample complexity, Lyapunov analysis, plug-in approach, LP perturbation analysis, average reward, generative model
- **One-liner**: 利用 Lyapunov drift + drift transfer 框架替代传统 bias/simulation lemma，首次为 WCMDP 和 RB 的 plug-in 学习建立了多项式样本复杂度上界。

## 精华提炼

### 管线/方法

整体 pipeline 是 plug-in 方法：

1. **数据采集**：对每个 arm 的每个 (state, action) pair，从 generative model（simulator）抽取 n 个 i.i.d. 样本，构造经验转移核 $\widehat{P}_i$
2. **构造经验 MDP**：将经验核代入 LP 松弛（WCMDP 用 per-arm LP, RB 用共享 LP）
3. **求解经验 LP** → 得到经验最优单臂策略 $\widehat{\bar\pi}^\star_i$ 和经验最优状态-动作频率 $\hat{y}^\star_i$
4. **构造 reference policy**：
   - WCMDP: ID policy（基于 ID 排序 + conforming number $N_t^\star$）
   - RB: Two-set policy（$D_t^{\mathrm{OL}}$ 用 Optimal Local Control, $D_t^{\bar\pi^\star}$ 用 Unconstrained Optimal Control）
5. **执行并评估**：在真实系统上运行学到的策略，分析 optimality gap

### 关键技术细节

**Lyapunov 框架核心（替代 simulation lemma）**：

传统 simulation lemma 依赖 bias function $h^\pi$，其范数难以控制。本文用显式构造的 Lyapunov function $V$ 替代：

- **Condition C1 (Drift)**：$[(\bp^\pi - I)V](z) \le -C_0 F(z) + G_1(N)$
- **Condition C2 (Gap Dominance)**：$\rho^{\mathrm{rel}} - r^\pi(z) \le C_0' F(z) + G_2(N)$
- **Drift Transfer**：经验系统漂移 = 真实系统漂移 + $[(\bphat^\pi - \bp^\pi)V](z)$，模型误差项通过 $\|\bphat^\pi - \bp^\pi\|_\infty \|V\|_\infty$ 控制

**WCMDP 的 Lyapunov 函数**：
- 基于 Zhang et al. 2025 的 projection-based Lyapunov function
- $V(\bmx) = h(\bmx, m(\bmx)) + L_h(1 - m(\bmx))$，其中 $L_h = 2\max\{c_{\max}, r_{\max}\} \cdot 16\tau$
- $\tau = (3 + \log_2 S)\tau_{\mathrm{unif}}$
- Drift 参数：$\rho_V = \frac{1-\gamma}{1 + L_h/\eta_c}$, $K_V$ 由 mixing time 和 cost/reward 界决定
- $\|V\|_\infty \le L_h$

**RB 的 Lyapunov 函数**：
- 基于 Hong et al. 2024 的 two-set Lyapunov function
- 使用 $\mu^\star$-weighted 范数：$\|v\|_{\mu^\star}^2 = \sum_s \mu^\star(s) v(s)^2$
- 矩阵 $W, U$ 通过 Neumann 级数定义，量化分布收敛速度
- $V(\sigma) = (\tilde{V} - \bar\eta/2)^+ + \frac{\gamma}{4(\beta + r_{\max})}(\mu^\star - \bmx([N]))Qb$
- $\tilde{V}$ 使用 weighted $\ell_2$ 子集 Lyapunov 函数 $h_U, h_W$

**LP 扰动分析（Theorem 4，独立贡献）**：
- 对 RB 的 LP 松弛做 primal-dual 扰动分析
- 构造候选解 $\hat{\mu} = \mu^\star(I - (\hat{U} - U)H_U)^{-1}$, $\hat{y} = \hat{\mu} W$
- 证明：在非退化 + strict gap 条件下，经验 LP 保持与真实 LP 相同的支撑集
- 扰动界：$\|\hat{y}^\star - y^\star\|_\infty \le 3\|H_U\|\delta$, $\|\hat{h}^\star - h^\star\|_{\mathrm{span}} \le 6\delta \|H_U\| \|h^\star\|_{\mathrm{span}}$
- Markov chain 结论：$\widehat{\bar\pi}^\star$ 在 $\widehat{P}$ 下保持 ergodic，mixing time $\le (3+\log_2 S)\tau$

**假设条件**：
- WCMDP: Assumption 1 (Unichain + Uniform Mixing)，各 arm 的 MDP 在所有策略下 mixing time 一致有界
- RB: Assumption 2 (Ergodicity: $\bar\pi^\star$ 诱导的链 aperiodic + irreducible; Non-degeneracy: 存在唯一 neutral state $\tilde{s}$; Local stability: $\|\Phi\|_{\mu^\star\to\mu^\star} < 1$)
- RB: Assumption 3 (Strict action-value gaps: 对 $y^\star(s,a)=0$ 的 inactive 动作有正的 dual slack)

**样本复杂度（生成模型设定）**：

- WCMDP: $n = \Omega\left(\frac{(S + \log(SAN/\eta))N^2}{\epsilon^2}\right)$，总样本量 $NSAn = \mathrm{poly}(N)$
- RB: $n = \Omega\left(\frac{(S + \log(SA/\eta))N^2}{\epsilon^2}\right)$，但 $\epsilon$ 可小至 $\Omega(\exp(-CN))$，此时总样本量仍为 poly(N)
- Blocking scheme: 当 n 不足时，将 N arms 分为 B blocks，每 block 独立运行 two-set policy，$B = \sqrt{N}$ 可得 $\tilde{O}(1/\sqrt{N})$ gap

### 核心结果

| Setting | 参考策略 | 最优性 Gap | 样本复杂度 |
|---------|---------|-----------|-----------|
| 异构 WCMDP | ID policy ($\piid$) | $O(N/\sqrt{n}) + O(1/\sqrt{N})$ | $\tilde{O}((S+\log(SAN))N^2/\epsilon^2)$ |
| 同构 RB | Two-set policy ($\pits$) | $O(N/\sqrt{n}) + O(\exp(-CN))$ | $\tilde{O}((S+\log(SA))N^2/\epsilon^2)$，$\epsilon$ 可至 $\exp(-CN)$ |

**关键发现**：
- 利用 weakly coupled 结构，避免了对 $S^N \times A^N$ 的指数依赖
- LP 扰动定理说明：在合适的结构性假设下，基于 LP 的策略对转移核扰动具有内在鲁棒性——仅 neutral state 处的 action 概率需要调整以应对扰动
- Drift transfer 技术使得经验系统的分析可以"继承"真实系统的 drift 性质，只需额外控制模型误差项

### 明确局限

1. **仅限生成模型（generative model/simulator）**：未考虑 online RL 或 offline 设定，所有 (s,a) pair 都能被采样到
2. **已知 reward 和 cost**：只学习 transition，实际中 reward/cost 也可能未知
3. **Heterogeneous WCMDP 的 $O(1/\sqrt{N})$ gap**：继承了 ID policy 本身的局限，无法通过增加样本改进
4. **RB 严格的结构假设**：Assumptions 2, 3（non-degeneracy, local stability, strict dual gaps）在某些实际问题中可能不满足
5. **单臂 MDP 状态/动作空间小**：分析假设 S, A 是常数（与 N 无关），扩展到 large S/A 需要额外工作
6. **无下界**：没有证明多项式样本复杂度的下界，optimality of the rates 未知
7. **仅处理 average reward**：discounted 或 finite-horizon 版本未讨论
8. **均匀 mixing 假设（WCMDP）**：要求所有策略下各 arm 的 mixing time 一致有界，是较强的假设。RB 部分放松了这一要求，仅对 $\bar\pi^\star$ 要求 ergodicity
9. **无实验验证**：纯理论论文，没有数值实验或仿真结果
