# Random Feature Models for Learning Interacting Dynamical Systems (2212.05591)

- **Year**: 2022.12
- **Authors**: Yuxuan Liu, Scott G. McCalla, Hayden Schaeffer (Carnegie Mellon / UCLA)
- **Venue**: arXiv preprint
- **Keywords**: random features, interaction kernel, multi-agent dynamics, sparse regression, hard thresholding pursuit, trajectory-based regression, radial basis functions, dynamical systems
- **One-liner**: 用随机高斯径向基函数逼近多体交互核，通过 HTP 稀疏化减少过拟合并加速预报，在一阶/二阶/异质系统上验证。

## 精华提炼

### 管线/方法

完整 pipeline：
1. **数据生成**：MATLAB ODE15s 求解器（relTol 1e-5, absTol 1e-6），从 L 个随机初始条件采样 L 条轨迹，每条 J 个时间戳，时间区间 [0, T]
2. **速度估计**：中心差分 `v ≈ (x(t_{j+1}) - x(t_{j-1})) / (t_{j+1} - t_{j-1})`，不需要等距时间采样
3. **随机特征构造**：从 N(0, σ²) 采样 N 个 ω_k，构建特征 φ_k(r₁,...,rₙ) = (α(ω_k)/n) · Σᵢ exp(-||rᵢ||² · ω_k²) · rᵢ
4. **特征矩阵组装**：A ∈ R^{n_tot × N}，其中 a_{ĩ,k} = φ_k(r^ℓ_{:,i}(t_j))，n_tot = d·n·J·L
5. **训练**（二选一）：
   - RRE：min_c ||V - A c||²（普通最小二乘）
   - SRRE：min_{c, ||c||₀ ≤ s} ||V - A c||²（HTP 算法迭代求解）
6. **预报**：用学到的 g_N 代入 ODE 系统，从新初始条件数值积分，预报时间 T̃ > T

### 关键技术细节

**数学基础**：
- Schoenberg 定理 (1938)：连续函数 g: R₊ → R 在所有维度上是径向正定的 ⇔ g(r) = ∫₀^∞ e^{-r²ω²} dν(ω)，ν 为有限非负 Borel 测度
- 由此导出加权积分表示 g(r) = ∫₀^∞ α(ω) e^{-r²ω²} dν(ω)，速度场化为 F = ∫₀^∞ φ(·, ω) dν(ω)
- 蒙特卡洛近似：从提议分布 θ(ω) 采样 ω_k，F_N = (1/N) Σₖ c_k φ_k

**归一化因子**：α(ω) = 2^n ω^{n+1} / π^{(n-1)/2}，用于避免大 ω 时高斯核接近零导致的条件数恶化。实践中 ω^p (p≥1) 缩放在经验上也有效。

**特征空间**（三种系统变体）：
- 一阶同质：φ_k = (α(ω_k)/n) Σᵢⁿ exp(-||rᵢ||² ω_k²) rᵢ
- 一阶异质（2 类型）：φ_k^{heterog} = Σᵢ (β(ω_k)/n_{kᵢ}) exp(-||rᵢ||² ω_k²) rᵢ，β(ω) = √π/(2ω)
- 二阶同质：φ_k^{sec} = (β(ω_k)/n) Σᵢ exp(-||rᵢ||² ω_k²) (d/dt)rᵢ

**HTP 算法（Foucart 2011）**：
- h=0 初始化 c⁰
- Step 1：S^{h+1} = s 个最大绝对值索引，按 |c^h + A*(V - Ac^h)| 排序
- Step 2：c^{h+1} = argmin_{supp(c) ⊆ S^{h+1}} ||V - Ac||²（在支持集上最小二乘）
- 迭代至收敛或 H 步

**超参数选择**（粗网格搜索）：
- LJ：N=1000, θ=N(0,35), s=150, L=100, T=0.01, T̃=0.5, σ_noise=0.001
- CS：N=1000, θ=N(0,1), s=500, L=50, T=0.25, T̃=0.5, σ_noise=0.01
- Predator-Prey：N=[500,500; 500,50]（每 kernel 不同），θ=N(0,30), s=400
- Sheep-Food：N=[500,500; 50,50], θ=N(0,10), s=600, T=100, T̃=400

**误差度量**：
- Kernel generalization error：||G' - G'_N||_{L²(ρ)}，其中 ρ 是 pairwise distance 的经验分布
- Path-wise error：E(c) = E_{x(0)}[sup_{t∈[0,T̃], i∈[n]} ||x_i(t) - x̃_i(t)||]

### 核心结果

| 系统 | 方法 | Kernel Error | Path Error | 加速比 |
|------|------|-------------|-----------|--------|
| Lennard-Jones | RRE | 8.64e-4 | 2.22e-2 ± 6.48e-2 | — |
| Lennard-Jones | SRRE | 3.00e-4 | 8.20e-3 ± 2.44e-2 | 6.67× |
| Cucker-Smale | RRE | 1.30e-3 | 6.39e-1 ± 5.40e-2 | — |
| Cucker-Smale | SRRE | 1.74e-1 | 6.36e-1 ± 5.58e-2 | 2× |
| Predator-Prey | RRE | ~1e-2 | 3.70e-2 ± 2.49e-1 | — |
| Predator-Prey | SRRE | ~2e-2 | 8.19e-3 ± 4.63e-2 | — |
| Sheep-Food | RRE | ~1e-2 | 8.32e-1 ± 9.53e-1 | — |
| Sheep-Food | SRRE | ~2e-2 | 1.72 ± 9.49e-1 | — |

**Surprising 发现**：
- SRRE 并不总是优于 RRE：CS 系统中 RRE kernel error (1.30e-3) 远低于 SRRE (1.74e-1)，但 path error 几乎相同。这说明 kernel 层面的局部误差（如 SRRE 在 CS 中的小振荡）不一定会显著传导到轨迹层面。
- 非负约束 vs 无约束：对于正定核（如 CS），非负约束提供更光滑的逼近；对于非正定核（如 LJ 在原点附近发散），非负约束完全失效（系数全零或退化）。因此无约束版本更泛用。
- 与 Lu et al. (2019) 的 piecewise polynomial 对比：SRRE 用 10× 更少的训练轨迹（100 vs 1000）实现了更低误差（kernel 3e-4 vs 2.26e-2, path 8.2e-3 vs 7.31e-2），归因于全局光滑基函数 vs 局部分段多项式。
- 稀疏度饱和：LJ 系统中 s≥80 后误差趋于平稳，因此 s=150 << N=1000 在维持精度同时带来 6.67× 查询加速。

**与 RFF（随机傅里叶特征）对比**：
- RFF 产生全局振荡逼近，与交互核的局部衰减/增长特性不匹配
- 在 r∈[40,60] 区域，RFF 呈现非单调"噪声"拟合
- 本文的径向高斯特征因匹配结构性假设（正定径向函数）而显著优于 RFF

### 明确局限

1. **作者自述**：高噪声和异常值情况下性能下降；无法处理超出交互核假设的额外动力学项（如外力场、边界条件）
2. **无真机实验**：全部为合成数据，ODE 由已知核生成后加噪，无真实物理/生物实验数据验证
3. **仅 d=2**：所有实验都在二维空间，作者声称是"可视化目的"，但未验证高维扩展性
4. **理论缺口**：RFM 文献中关于泛化误差的理论保证（如 Rudi & Rosasco 2017, Mei 2021）依赖于 i.i.d. 样本假设，而轨迹数据高度相关，理论不直接适用
5. **无标准 benchmark**：系统参数（n, T, σ_noise 等）为手工选择，未与其他学习方法（如 neural ODE, GNN-based）在固定 benchmark 上系统比较
6. **HTP 收敛性不保证**：HTP 的理论保证依赖于 RIP 等压缩感知条件，交互系统的特征矩阵 A 不满足这些条件
7. **σ（特征宽度）敏感**：θ = N(0,σ²) 中 σ 是核心超参数，通过粗网格搜索确定——σ 过小则特征过宽欠拟合，过大则特征过窄过拟合
8. **预报长度有限**：虽然声称 50× T 预报（LJ），但 Cucker-Smale 仅预报 2× T(T=0.25 → T̃=0.5)
