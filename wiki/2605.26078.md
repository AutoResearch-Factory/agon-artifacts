# Global Convergence of Wasserstein Policy Gradient for Entropy-Regularized Reinforcement Learning (2605.26078)

- **Year**: 2026.05
- **Authors**: Zhaoyu Zhu (Shanghai Jiao Tong University), Rui Gao (The University of Texas at Austin), Shuang Li (The Chinese University of Hong Kong, Shenzhen)
- **Venue**: arXiv
- **Keywords**: Wasserstein Policy Gradient, entropy-regularized RL, continuous-action MDP, Langevin dynamics, Bellman residual, log-Sobolev inequality, distributional PL condition, optimal transport
- **One-liner**: 证明熵正则WPG全局收敛到离散偏差内。

## 精华提炼

### 管线/方法
- 论文研究对象是无限时域 discounted MDP，状态空间一般，动作空间固定为 `R^d`，策略是每个状态条件下的动作密度。
- 目标函数是 entropy-regularized return：每步 reward 使用 `r(s,a) - beta/2 ||a||^2 - tau log pi(a|s)`。
- 二次动作惩罚 `beta/2 ||a||^2` 是关键建模条件；没有它，在无界动作空间上 entropy bonus 会让目标可无界增大。
- 作者把二次惩罚等价写成 Gaussian reference `rho_beta(a) ∝ exp(-beta ||a||^2/(2 tau))`。
- admissible policy class 要求每个状态的 `KL(pi(.|s) || rho_beta)` 一致有界，从而保证二阶矩和 Bellman 评估有限。
- 软 Q 函数定义为 `Q^pi(s,a)=tilde r(s,a)+gamma E[V^pi(s')|s,a]`。
- 软 Bellman optimality operator 是 action density 上的 log-partition：`T*V(s)=tau log int exp(Q_V(s,a)/tau) da`。
- 给定当前 policy `pi`，定义当前 soft Q 诱导的 Gibbs policy `p_s^pi(a) ∝ exp(Q^pi(s,a)/tau)`。
- 核心恒等式 1：Bellman residual 等于状态级 KL，`T*V^pi(s)-V^pi(s)=tau KL(pi(.|s)||p_s^pi)`。
- WPG flow 是 statewise Wasserstein gradient flow：`partial_t pi_t = -div(pi_t grad_a(Q^pi_t - tau log pi_t))`。
- 该 flow 可重写为相对于移动 Gibbs `p_s^pi_t` 的 Fokker-Planck 方程。
- 粒子视角下，每个状态的 actor 更新是 nonlinear Langevin diffusion：`dA_t^s = grad_a Q^pi_t(s,A_t^s) dt + sqrt(2 tau) dB_t^s`。
- 离散算法 WPGD 是 Euler-Maruyama：`A_{k+1}^s=A_k^s+eta grad_a Q^pi_k(s,A_k^s)+sqrt(2 tau eta) xi_{k+1}`。
- 论文先做 continuous-time formal analysis，用来展示 Bellman geometry；再给 rigorous discrete-time theorem。
- 连续时间证明链不是传统 mean-field Langevin 的 convex functional argument。
- 替代机制是：Bellman residual -> KL，KL + uniform LSI -> Fisher information，Fisher information 通过 Bellman resolvent -> value improvement。
- 核心恒等式 2：沿 WPG flow，`(I-gamma P^pi_t) dV^pi_t/dt = tau^2 I(pi_t(.|s)||p_s^pi_t)`。
- 因为 resolvent `(I-gamma P)^-1=sum gamma^n P^n` 保正，得到每个状态的 value 单调提升。
- 价值函数单调和 bounded reward 给出 `V^pi_t` 与 `Q^pi_t` 的 uniform bounds。
- uniform Q bound 说明每个移动 Gibbs `p_s^pi_t` 是 Gaussian reference 的 bounded perturbation。
- 用 Holley-Stroock perturbation lemma，把 Gaussian LSI 常数转移为 trajectory-uniform LSI。
- Bellman contraction 把 `V* - T*V^pi_t` 控制在 `gamma ||V* - V^pi_t||_inf`。
- 于是 residual 在近 worst state 上至少控制 `(1-gamma)` 倍的 sup-norm value gap。
- 连续时间主结论：`||V* - V^pi_t||_inf` 指数衰减，rate 为 `2 alpha tau (1-gamma)`。
- 作者明确说明 continuous-time 部分依赖 smooth-flow regularity convention，不声称完整 PDE well-posedness。
- 离散时间部分的额外难点是 fixed-step Langevin 的 discretization error 和 policy-dependent drift 的 uniform regularity。
- Assumption 4.1 要求 bounded reward、初始 `KL(pi_0||rho_beta)` 一致有界，并记录初始二阶矩 `M_0`。
- Assumption 4.2 要求 reward action-gradient 有界且 Lipschitz，transition density 的 action-gradient 在 `L1(lambda)` 下有界且 Lipschitz。
- 作者给出 additive-noise transition example：`S'=F(s,a)+sigma zeta`，非退化噪声且密度一阶/二阶导可积即可满足假设。
- Bellman drift 写作 `b_k(s,a)=grad_a Q^pi_k(s,a)=-beta a + h_k(s,a)`。
- 其中 `h_k` 由 reward gradient 和 transition derivative 组成，若 `||V^pi_k||_inf <= Vmax` 则一致有界。
- drift Lipschitz 常数为 `L_b(Vmax)=beta+L_r+gamma L_p Vmax`。
- discrete Bellman resolvent：`(I-gamma P^pi_{k+1})(V^pi_{k+1}-V^pi_k)=g_k`。
- `g_k(s)` 精确等于 frozen Gibbs 下 KL 的下降量乘以 `tau`。
- fixed-drift one-step interpolation 证明 `KL(pi_{k+1}||p_s^pi_k) <= exp(-alpha tau eta) KL(pi_k||p_s^pi_k)+delta_eta`。
- 离散误差 `delta_eta = L_b^2 d eta^2/2 + L_b^2 B^2 eta^3/(6 tau)`。
- 该 one-step KL contraction 转化为 Bellman improvement：`g_k(s) >= c_eta R_k(s) - tau delta_eta`。
- 再经过 resolvent 和 Bellman residual lower bound，得到 scalar gap recursion。
- 递推形式：`E_{k+1} <= (1-(1-gamma)c_eta) E_k + tau delta_eta/(1-gamma)`。
- uniform stability proposition 是论文最长的技术部分：闭合 value bound、moment bound、drift second moment、Gaussian KL、frozen KL、LSI 常数。
- 关键闭合方式是先假设到第 k 步 `||V||_inf <= Vbar`，推出 drift dissipative，再推出 action moment bound，再推出 Gaussian KL 和 admissibility，再回到 value lower bound。
- Gaussian smoothing lemma 用于证明更新后的策略对 `rho_beta` 的 KL 有界，虽然更新是 push-forward 后加 Gaussian noise。
- 主定理给出固定步长 `eta <= eta_0` 下的全局界：
- `J(pi*)-J(pi_k) <= ||V*-V^pi_k||_inf`。
- `||V*-V^pi_k||_inf <= exp(-0.5 alpha_bar tau (1-gamma) eta k) ||V*-V^pi_0||_inf + 2 C_delta eta/(alpha_bar (1-gamma)^2)`。
- 因此 discrete WPGD 是 geometric contraction up to `O(eta)` bias。
- 论文强调这不是 KL/NPG/mirror descent 的三点恒等式证明，而是 Wasserstein/Fisher information + Bellman residual 证明。
- 与 Siska and Zhang 2605.22622 的 note 相比，本文声称自己的 continuous-time bound 是 state-uniform，不需要 density-ratio/coverage 常数 `kappa`。
- 本文还声称自己的主要 rigorous guarantee 是 discrete-time Langevin 分析，而 Siska note 更偏 formal gradient-flow argument。

### 关键技术细节
- `gamma in (0,1)`，`tau > 0`，`beta > 0`。
- action space 是 `R^d`，不是 compact action set。
- admissible class `Pi` 要求 `sup_s KL(pi(.|s)||rho_beta)<infty`。
- `rho_beta(a)=Z_beta^{-1} exp(-beta ||a||^2/(2 tau))`，`Z_beta=(2 pi tau/beta)^{d/2}`。
- continuous-time boundedness assumption：`|r(s,a)| <= Rmax`，`sup_s KL(pi_0(.|s)||rho_beta) <= K0`。
- continuous-time uniform LSI constant：`alpha = beta/tau * exp(-2 Umax/tau)`，其中 `Umax=Rmax+gamma Vmax` 型常数。
- discrete-time reward/transition regularity constants：`L_r, L_p, G_r, G_p`。
- drift decomposition：`b_k=-beta a+h_k`，`||h_k|| <= G_r+gamma G_p Vmax`。
- drift Lipschitz：`L_b(Vmax)=beta+L_r+gamma L_p Vmax`。
- fixed-step KL error包含维度项 `L_b^2 d eta^2/2` 和 drift second moment 项 `L_b^2 B^2 eta^3/(6 tau)`。
- stability step size要求包括 `eta <= 1/(4 beta)`、`alpha_bar tau eta <= 1`、以及累计离散偏差不超过闭合余量。
- 论文不使用 sample complexity、critic approximation 的具体 estimator；actor-critic only appears as future/practical extension context, not theorem body.
- 没有 MuJoCo/DM Control 数值实验；这是一篇理论证明论文。

### 核心结果
- Continuous WPG flow：在 smooth-flow convention 下，sup-norm optimality gap 指数收敛。
- Discrete WPGD：固定步长 Langevin actor update 全局几何收敛到 `O(eta)` 邻域。
- 最核心 novelty 是把 RL Bellman recursion 转换成类似 PL 的 geometry，而不是假设目标在 policy distribution 上 flat convex。
- Bellman residual 的 statewise KL identity 是全篇 proof 的支点。
- Bellman resolvent identity 把 Fisher information dissipation 传回 value improvement。
- uniform LSI 来自 Gaussian quadratic penalty 和 bounded perturbation，不来自目标凸性。
- 反直觉点：即使 entropy-regularized RL 不是普通意义的 convex objective，Bellman contraction 仍能给出足够强的 residual-to-gap 控制。
- 另一个重要点：state-uniform sup-norm 分析避免了 occupancy distribution mismatch 和 density-ratio coverage 常数。
- 论文和 2605.22622 是近乎同时期的 convergence theory；本文在文中明确把自己定位为更 rigorous 的 discrete-time result。

### 明确局限
- continuous-time 部分是 formal，不包含完整 nonlinear Fokker-Planck well-posedness theorem。
- discrete theorem 依赖强假设：bounded reward、无界动作上的 quadratic penalty、初始 Gaussian KL 有界、transition density 对 action 有 `L1` 可微和 Lipschitz 控制。
- deterministic dynamics 或 degenerate-noise dynamics 不被作者给出的 additive-noise example 覆盖。
- theorem 是 exact policy evaluation / exact Q-gradient 层面的 policy-space guarantee，不包含 finite-sample critic estimation。
- 没有 parametric neural policy projection 的误差分析；WPGD 的 statewise policy density 更新在实践中无法直接枚举所有 state。
- 没有 benchmark 实验；无法证明该 update 在实际 continuous-control benchmark 上比 SAC/PPO/WPO 更好。
- 离散结果有固定步长 `O(eta)` bias，不是固定 `eta` 下精确收敛到 optimum。
- 常数可能极差：LSI 常数含 `exp(-2(Rmax+gamma Vbar)/tau)`，小温度或大 reward bound 时收敛界会非常松。
- 论文不处理 entropy temperature annealing、finite particle approximation、function approximation、off-policy replay 等工程关键问题。
- 与 2603.02576 WPPG 存在作者和主题重叠；本文更偏 explicit Langevin WPGD 的理论闭环，WPPG 更偏 operator-splitting/implicit policy 可实现性。
