# MVNN: A Measure-Valued Neural Network for Learning McKean-Vlasov Dynamics from Particle Data (2604.00333)

- **Year**: 2026.04
- **Authors**: Liyao Lyu, Xinyue Yu, Hayden Schaeffer (Department of Mathematics, UCLA)
- **Venue**: arXiv preprint
- **Keywords**: mean field approximation, interacting particle systems, data-driven modeling, McKean-Vlasov SDE, Wasserstein space, approximation theory, measure-valued neural network, cylindrical functional
- **One-liner**: 提出 MVNN——一种排列不变的测度值神经网络，从粒子轨迹数据直接学习 McKean-Vlasov 平均场动力学中的测度依赖漂移项，O(N) 复杂度且有理论保证。

## 精华提炼

### 管线/方法

**输入**：M 条独立轨迹的离散时间观测 {X^i_{t_l,m}}，i=1..N, l=0..L, m=1..M。
**预处理**：一阶有限差分计算速度 V^i_{t_l,m} = (X_{t_{l+1}} - X_{t_l})/Δt。
**核心模型（MVNN）**：b_θ(x, μ) = φ_int(x, ⟨φ_emb, μ⟩; θ_int)，其中：

1. **Embedding Network φ_emb: ℝᵈ → ℝᵏ** — 学习 k 个测试函数，从测度中提取特征。将每个粒子的状态映射到 k 维特征向量。
2. **测度积分 ⟨φ_emb, μ⟩ = ∫ φ_emb(y) μ(dy)** — 对经验测度 μᵗ_N = (1/N)Σδ_{Xᵗⱼ} 退化为 (1/N)Σφ_emb(Xᵗⱼ)，排列不变，O(N) 复杂度。
3. **Interaction Network φ_int: ℝᵈ × ℝᵏ → ℝᵈ** — 将局部状态 x 和全局测度特征 ⟨φ_emb, μ⟩ 映射为漂移向量。

**损失函数**：从 Girsanov 定理推导的对数似然 → 离散化 → MSE 回归损失（假设 σ 为常数）：
θ* = argmin (1/MLN) Σ Σ Σ ||V^i_{t_l,m} - b̂_θ(X^i_{t_l,m}, μ̂_{t_l,m})||²

**优化**：Adam optimizer，JAX 自动微分，mini-batch 训练。
**输出**：训练好的 MVNN 漂移 b̂_θ，可直接代入 McKean-Vlasov SDE 进行前向模拟。

**多群体扩展（MG-MVNN）**：K 个群体各有独立的 φ_{emb,k} 和 φ_{int,k}，所有群体的测度特征拼接后输入各自的 interaction network。

**二阶系统扩展**：输入空间扩展为 (位置, 速度) 对，embedding 和 interaction 网络相应调整维度。

### 关键技术细节

- **网络架构**：φ_emb 和 φ_int 均为标准 ReLU 全连接网络。论文未给出具体层数/宽度数值，仅通过理论分析给出深度 O(log(ε⁻¹))、宽度 O(ε^{-(d+r)}) 的量级估计。
- **训练数据规模**：
  - 1D Motsch-Tadmor: N=16000 agents, M=100 trajectories (97 train / 3 test), L=200 timesteps, Δt=0.01, T=2
  - 2D aggregation: N=16000, M=100, L=200, Δt=0.01
  - 多群体: N₁=16000, N₂=4000, N₃=200, M=100
  - GP 对比：N=16 agents, M=9 trajectories, L=20, Δt=0.05 — MVNN 可处理更大规模
- **初始分布**：随机多模态高斯混合（2-8 个分量，均值 U[0,3]，方差 0.25，Dirichlet 混合权重）
- **评估方式**：比较 KDE 密度演化（非逐粒子轨迹），L² 误差在 KDE 平滑密度上计算
- **OOD 测试**：双环、均匀圆盘、二值非对称分布 — 训练时仅见单环初始分布
- **扩散系数**：σ 视为已知常数（确定性 σ=0，随机 σ=0.1），不学习
- **交互核（ground truth）**：
  - Motsch-Tadmor: φ(r) = exp(-(r/ℓ)²), ℓ=0.5, 带归一化分母
  - 2D aggregation: φ(r) = c_rep·exp(-(r/ℓ_rep)²) - c_att·exp(-(r/ℓ_att)²), c_rep=1.0, ℓ_rep=0.5, c_att=0.7, ℓ_att=2.0
  - Cucker-Smale: 仅速度对齐，无位置力

| 实验 | 关键发现 |
|------|---------|
| 1D Motsch-Tadmor | MVNN 准确复现聚类合并和密度峰保持；OOD 初始化下 L² 误差小 |

**Surprising/反直觉结果**：
- MVNN 在 N=16 小规模数据上训练仍优于 GP（通常神经网络需要大数据），说明圆柱泛函结构有强 inductive bias
- 仅用单环训练即可泛化到双环/圆盘/二值分布，说明 MVNN 学到了内在交互算子而非表面几何

### 明确局限

1. **只学漂移，扩散系数假设已知且为常数** — 作者明确说扩展到状态/测度依赖的扩散项是 future work
2. **无真机实验，全为合成数据** — 所有实验在已知 ground truth 交互核上生成
3. **无标准 benchmark 对比** — 只和 Gaussian Process 对比，未与 WSINDy (Messenger 2021)、DeepONet、FNO 等 operator learning 方法做数值对比
4. **平均场假设可能不足** — 作者在 Discussion 承认，稠密粒子系统中高阶关联（BBGKY 层级）重要，MVNN 只能学到平均场
5. **维度诅咒** — Theorem 3.4 显示无结构假设时网络复杂度随 d 指数增长；Theorem 3.5 依赖低维假设（Assumption 1），但该假设在真实系统中无法验证
6. **仅测试 N=16000** — 未探索不同 N 下的 propagation of chaos 收敛行为
7. **无代码/数据开源声明** — 论文未提及代码是否公开
8. **与 PDE foundation model 的关系仅口述未实验** — 讨论中称可视为 weak-form operator learning，但无实验支撑

## Read by: 0616-fex (2026-06-24)

### 撞车评估
不撞车。MVNN 在 **measure space**（Wasserstein space, 概率测度）上做 operator learning，我们的 0616-fex 在 **function space** 上做 PDE foundation model / operator learning。输入输出空间不同：MVNN 输入是点云/经验测度，输出是向量场（drift）；我们输入是函数（PDE 解/参数），输出是函数（PDE 解）。

### 可用工具/方法
1. **圆柱泛函（cylindrical functional）架构** — 用 learned test functions + 积分 + interaction network 的三段式设计，可以迁移到 function space operator learning：用 learned basis functions 替代 FNO 的 Fourier modes 或 DeepONet 的 fixed sensors
2. **Partition-of-unity 逼近框架**（Theorem 3.4 proof）— 对测度空间的有限维逼近提供了一个构造性方法，可以用于分析 function space 上 operator learning 的 approximation rate
3. **Low-dimensional assumption（Assumption 1）** — "有效交互仅依赖少数宏观统计量" 这一洞察可类比到 PDE 场景：很多 PDE 解算子实际上只依赖少数隐变量，不必在 full function space 中学习
4. **Girsanov → MSE 推导链** — 从随机过程的路径测度似然推导到 MSE 回归损失的技术路线，可以用于设计 stochastic PDE 场景下的训练目标
5. **Multi-group 架构** — K 个独立 embedding + 共享/独立 interaction 的设计模式，可用于多物理场耦合的 PDE foundation model

### 差异化要点
- 我们做 function-to-function mapping，MVNN 做 measure-to-vector-field mapping
- 我们目标是跨 PDE 族的 foundation model（multi-task/ multi-physics），MVNN 是 per-system 训练
- 我们的输入有网格结构（regular grid / mesh），MVNN 的输入是无序点云
- MVNN 的 O(N) 效率和排列不变性对我们不一定关键（我们的输入有序）
- 可以用 MVNN 的 weak-form 视角来框架化我们的方法，在 related work 中作为"operator learning 的第三种范式"（pointwise / spectral / weak-form）
