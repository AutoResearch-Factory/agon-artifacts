# Polyak--Łojasiewicz Inequality for Quadratically Regularized Optimal Transport (2605.27175)

- **Year**: 2026.05
- **Authors**: Alberto González-Sanz, Marcel Nutz, Andrés Riveros Valdevenito (Department of Statistics / Mathematics, Columbia University)
- **Venue**: arXiv
- **Keywords**: Optimal Transport, Quadratic Regularization, PL Inequality, Linear Convergence, Error Bound, Gradient Ascent, Coordinate Ascent, Sparse Couplings
- **One-liner**: 证明 QOT 对偶目标函数满足局部 Polyak--Łojasiewicz 不等式，并推导出三种对偶算法的显式线性收敛速度。

## 精华提炼

### 核心问题
Quadratically regularized optimal transport (QOT) 是 entropic OT (EOT/Sinkhorn) 的主要替代方案。QOT 产生稀疏耦合（EOT 产生全支撑耦合），且避免了 EOT 中指数缩放导致的数值不稳定性。但从优化角度看，QOT 对偶目标函数包含 `(t)_+ = max(t,0)` 项，破坏了强凹性（在 `f⊕g - c ≤ 0` 的区域甚至是线性的），optimizers 也只有有限光滑性。因此 EOT 中 standard 的强凹性 → 线性收敛论证完全不适用。

### 核心贡献
本文证明了 QOT 对偶目标函数满足 **局部 Polyak--Łojasiewicz (PL) 不等式**，即梯度范数与 suboptimality gap 之间的定量关系，尽管没有强凹性。这是对 QOT 优化性质的第一个结构性理解。

**主定理 (Theorem 2.1 / th:PL-Gamma)**：对所有 `(f,g) ∈ L^∞(Ω) × L^∞(Ω')`，
```
‖f⊕g - f_*⊕g_*‖_{L^2(P⊗Q)} ≤ γ_ε · max(‖f⊕g - f_*⊕g_*‖_∞, ε) · ‖DΓ(f,g)‖_{L^2(P)×L^2(Q)}
```
```
‖DΓ(f,g)‖^2 ≥ [1 / (γ_ε · max(‖f⊕g - f_*⊕g_*‖_∞, ε))] · (QOT_ε - Γ(f,g))
```
其中 γ_ε 是显式常数（见下文）。

### 关键技术细节

#### 假设条件
1. **P（第一边缘分布）**：支撑集 Ω 为紧凸集（可放松到紧连通 Lipschitz 域），密度 ρ 有上下界 `0 < λ_P ≤ ρ(x) ≤ Λ_P < ∞`
2. **Q（第二边缘分布）**：紧支撑 Ω'（**可以是离散的**，因此同时覆盖连续和 semi-discrete transport）
3. **代价函数 c**：L-Lipschitz（可放松到一致连续）

#### QOT 对偶目标
```
Γ(f,g) = ∫[f(x) + g(y) - (1/2ε)(f(x) + g(y) - c(x,y))_+²] d(P⊗Q)(x,y)
```
梯度 `D₁Γ(f,g) = 1 - (1/ε)∫(f(·) + g(y) - c(·,y))_+ dQ(y)`，`D₂` 对称。

#### 证明路线（核心洞察）
1. 定义 **Φ(f⊕g) = Γ(f,g)** 在 Hilbert 空间 `ℋ = {u⊕v : (u,v) ∈ L²(P)×L²(Q)}` 上操作，消除 potentials 的平移非唯一性
2. 沿插值路径 `(f_t, g_t) = t(f,g) + (1-t)(f_*,g_*)` 考虑 `φ(t) = -Φ(f_t⊕g_t)`
3. **关键步骤**：对二阶导数 `φ''(t) = (1/ε)∫_{f_t⊕g_t ≥ c} ((f_*-f)⊕(g_*-g))² d(P⊗Q)` 建立均匀下界
4. 下界对着一个积分算子 `𝕄` 的最小特征值估计，该算子核为 `1_{f_r⊕g_r ≥ c}`
5. 利用该算子在 eigenvector 上的显式结构做谱分析（算子非紧，但通过 compact embedding 技术证明 infimum 可达且是特征值）
6. 结合 Bourgain--Brezis--Mironescu 型 Sobolev 能量的分块估计和几何测度下界

#### γ_ε 的显式形式
```
γ_ε = 16 · [δ_P^{-1} · max(8L/ε, 1)^d] · [Λ_P²/λ_P²] · [(⌈8L·diam(Ω)/ε⌉)^{d+2}] / [inf_{y∈Ω'} Q(B_{ε/8L}(y))]
```
其中 δ_P 来自锥条件 `P(B_r(x)) ≥ δ_P · min(r^d, 1)`。

**对 ε 的依赖**：γ_ε 以 `ε^{-(2d+2)}` 量级增长（d 为空间维数），因为分子有 `(1/ε)^d × (1/ε)^{d+2}` 结构。不过数值实验中实际收缩率常优于该 theoretical bound。

#### PL 不等式的局部性
常数通过 `max(‖f⊕g - f_*⊕g_*‖_∞, ε)` 依赖于到最优解的距离。这是无法避免的：在远离最优解的区域，对偶目标是线性的（梯度为常数），强凹性完全消失。

### 算法应用（三个推论）

| 算法 | 步长约束 | 收缩因子 q | 特点 |
|------|---------|-----------|------|
| **梯度上升 (GD)** | η < ε | `η(1-η/ε) / (γ_ε · M)` | 显式，需步长 |
| **坐标上升 (CA)** | 无（隐式） | `ε / (2γ_ε · M)` | Sinkhorn 的 QOT 对应，每步需求解隐式方程 |
| **坐标梯度上升 (CGA)** | η < ε/√2 | `η(1-η/2ε) / (2γ_ε · M)` | 显式，使用最新 f 更新 g |

其中 `M = max(2‖f₀⊕g₀ - f_*⊕g_*‖_∞, ε)`。

**L² 迭代收敛**：三个算法均有 `‖f_n⊕g_n - f_*⊕g_*‖²_{L²(P⊗Q)} ≤ Const · Δ₀ · (1-q)^n`。

### 核心结果
- QOT 对偶虽然没有强凹性，但在最优解附近有 PL 不等式 → **局部曲率**
- 三种自然对偶算法均以显式收缩率线性收敛
- 梯度上升的线性收敛之前已在 González-Sanz, Nutz, Riveros.25 中证明，但本文通过 PL 不等式给出 **显式收缩常数**，且统一覆盖三类算法
- 先前的 gradient ascent 证明依赖对称性，不易推广到坐标类算法

### 明确局限
1. **PL 不等式是局部的**：常数随远离最优解而退化。在全局范围内没有 uniform PL 常数
2. **纯理论论文**：没有数值实验（引用 Lorenz.2019.preprint 的数值结果但没有自己的实验）
3. **无真机/无 benchmark**：理论性质，不在 ML 任务上评估
4. **γ_ε 保守**：显式常数对 ε 有悲观的依赖（小 ε 时极大），作者明确指出实践中收缩率通常远优于 bound
5. **步长约束保守**：理论要求 η < ε（GD）或 η < ε/√2（CGA），数值中最佳步长通常远大于这些约束
6. **第一边缘的凸性假设**：主结果需要 Ω 凸，附录扩展到连通 Lipschitz 域但增加额外常数 C_Ω
7. **cost 函数的连续性假设**：Lipschitz 条件可放松到一致连续，但常数表达式更复杂
8. **对 f_*, g_* 的 L-Lipschitz 性质的依赖**：证明中大量使用 potentials 的 Lipschitz 性质做几何下界
9. **没有讨论离散化误差**：算法在实际中需要在离散网格上运行，但理论分析是连续形式的
