# Pretraining Codomain Attention Neural Operators for Solving Multiphysics PDEs (2403.12553)

- **Year**: 2024.03
- **Authors**: Md Ashiqur Rahman (Purdue), Robert Joseph George (Caltech), Mogab Elleithy (Caltech), Daniel Leibovici (Caltech), Zongyi Li (Caltech), Boris Bonev (NVIDIA), Colin White (Caltech), Julius Berner (Caltech), Raymond A. Yeh (Purdue), Jean Kossaifi (NVIDIA), Kamyar Azizzadenesheli (NVIDIA), Anima Anandkumar (Caltech)
- **Venue**: NeurIPS 2024
- **Keywords**: neural operator, codomain attention, multiphysics PDEs, self-supervised pretraining, few-shot learning, transformer, function space, Fourier neural operator, graph neural operator, fluid-structure interaction
- **One-liner**: CoDA-NO 将 transformer 自注意力从有限维向量空间推广到无限维函数空间，通过对 codomain（通道空间）而非空间域做 tokenization 和 attention，实现多物理场 PDE 系统的预训练和跨变量组合的少样本迁移。

## 精华提炼

### 管线/方法

**整体架构 — 三步 pipeline：**

1. **Variable Specific Positional Encoding (VSPE)**：为每个物理变量 $a^i: \mathcal{D} \rightarrow \mathbb{R}$ 学习一个位置编码 $e^i: \mathcal{D} \rightarrow \mathbb{R}^{d_{en}}$（通过可学习的 Fourier 系数 $\kappa^i$ + iFFT 实现），将 $[a^i, e^i]$ 沿 codomain 拼接得到 $\bar a^i$。

2. **Lifting**：通过共享的 pointwise MLP 将 $\bar a^i$ 从 $\mathbb{R}^{d_{en}+1}$ 提升到 $\mathbb{R}^D$（$D > d_{en}+1$）。对不规则网格，先用 GNO$_{per}$（permutation-equivariant GNO）将非均匀网格上的函数映射到均匀 latent grid。

3. **Codomain Attention Block（堆叠 L 层）**：每一层包含：
   - **Key/Query/Value 算子**：用 FNO（或 GNO）将每个 token function $w^j$ 映射为 $k^j = \mathcal{K}[w^j]$、$q^j = \mathcal{Q}[w^j]$、$v^j = \mathcal{V}[w^j]$
   - **函数空间注意力**：计算 $L^2$ 内积 $\langle q^j, k^m \rangle = \int_\mathcal{D} q^j(x) k^m(x) dx$ 作为 attention logits，除以温度 $\tau$ 后 Softmax，加权求和得到 $o^j$
   - **多头注意力**：H 个 head 并行，各自用独立的 $\mathcal{K}^h, \mathcal{Q}^h, \mathcal{V}^h$，输出 concatenation 后再经 $\mathcal{M}$ 算子融合
   - **Function Space Normalization**：将 instance norm 推广到函数空间——计算每个 token 在空间域上的均值 $\mu^j$ 和标准差 $\sigma^j$，norm 后加残差连接
   - **Permutation-equivariant Integral Operator** $\mathcal{I}_{per}$：共享权重的 FNO/GNO 对 attention 输出做进一步积分变换

4. **Projection**：pointwise MLP 将 latent 函数从 $\mathbb{R}^D$ 投影回 $\mathbb{R}^1$，得到输出 $u^i$。

**训练流程 — 两阶段：**

1. **预训练（Self-supervised）**：
   - 目标：Masked Reconstruction — 从被 mask 的输入重建完整函数
   - Mask 策略：以等概率选择 (a) 50% 网格点 mask 掉 60% 的变量，或 (b) 30% 的变量完全 mask
   - 不规则网格上的 mask：随机选种子点，mask 其邻域内的点，迭代至覆盖预定比例
   - 架构：Encoder（L 层 CoDA-NO）+ Reconstructor（L 层 CoDA-NO），不涉及 Predictor

2. **微调（Supervised few-shot）**：
   - 目标：给定当前时刻状态预测下一时刻状态
   - 架构：Encoder（预训练权重冻结或微调）+ Predictor（随机初始化，1 层 CoDA-NO）
   - 新变量适配：仅为新变量训练额外 VSPE，其余参数复用预训练权重
   - 最小可训练参数模式：冻结 Encoder 全部参数，仅训练 Predictor 和新 VSPE

### 关键技术细节

- **离散化收敛性（Discretization Convergence）**：CoDA-NO 作为 neural operator 的子类，保证在网格细化极限下收敛到唯一算子，因此对不同分辨率的输入/输出具有分辨率无关性
- **FNO 作为 K/Q/V 算子**：使用 FNO 而非 MLP 实现 $\mathcal{K}, \mathcal{Q}, \mathcal{V}$，确保这些算子本身也是 resolution-agnostic 的
- **L² 内积注意力**：$\langle q^j, k^m \rangle = \int_\mathcal{D} q^j(x) k^m(x) dx$，通过数值积分（quadrature）在离散网格上计算，保持函数空间的连续性
- **Permutation Equivariance**：所有操作对物理变量的排列顺序不变——共享权重的 $\text{FNO}_{per}$ 和 $\text{GNO}_{per}$ 确保这一点
- **数据集规模**：
  - 流体-结构相互作用：8000 snapshots 预训练（Re ∈ {200, 2000}），TurtleFSI (FEniCS) 生成，1000 时间步
  - RB 对流：40000 snapshots 预训练（Navier-Stokes 单独），Ra = 12×10³ 和 20×10³
  - PDEBench：SWE + DIFF + NS 单物理场，35 epochs
- **参数量对比**：CoDA-NO 43M（FSI 实验用）/ 11M（PDEBench 实验用），FNO 1.9B，DPOT 7M~500M
- **计算开销**：Training 1.25 s/sample（CoDA-NO）vs 0.136 s/sample（GNN），Inference 0.44 s vs 0.012 s（GNN）
- **超参数**：Encoder/Reconstructor 各 3 层 CoDA-NO，Predictor 1 层；d' = D（每个 token function 的 codomain 维度等于 latent 维度）；temperature τ 可调
- **不规则网格处理**：GNO 的 radius 参数控制邻域范围；biharmonic extrapolation 将固体域变形场扩展到流体域
- **FSI 边界条件**：4 阶多项式 inlet velocity profile，参数 $(c_1, c_2) \in \{-6, -4, -2, 0, 2, 4, 6\}^2$ 控制不对称度
- **代码**：已开源，集成到 neuraloperator 库（github.com/neuraloperator/neuraloperator）

### 核心结果

**流体-结构相互作用（FSI, Re=400）：**

| # Few-shot | Model (Pretrain) | NS (L² error) | NS+EW (L² error) |
|-----------|-------------------|--------------|------------------|
| 5 | CoDA-NO (NS+EW) | **0.024** | **0.040** |
| 5 | GNN (scratch) | 0.038 | 0.045 |
| 5 | GINO (scratch) | 0.200 | 0.122 |
| 25 | CoDA-NO (NS+EW) | **0.006** | **0.005** |
| 25 | GNN (scratch) | 0.008 | 0.009 |
| 100 | CoDA-NO (NS+EW) | 0.005 | **0.003** |

- 预训练 CoDA-NO 在 5-shot 下平均比最佳 baseline（GNN）低 36.8% 误差
- 仅在 NS 上预训练的 CoDA-NO 可无缝适应 NS+EW（增加变形场变量 d_x, d_y）
- Re=4000 更湍流场景下同样超越所有 baseline

**Rayleigh-Bénard 对流（Ra=12×10³）：**

| # Few-shot | Model | L² error |
|-----------|-------|----------|
| 5 | CoDA-NO (NS pretrain) | **0.016** |
| 5 | Unet (scratch) | 0.049 |
| 5 | FNO (scratch) | 0.119 |

- 仅基于 NS 预训练的 CoDA-NO 轻松适应新温度变量 T，误差降低约 2 倍

**PDEBench（单物理场）：**

| PDE | CoDA-NO (11M) | FNO (1.9B) | 提升 |
|-----|--------------|-----------|------|
| SWE | 0.0407 | 0.0463 | 12% |
| DIFF | 0.0081 | 0.0141 | 43% |
| NS+DIFF+SWE (联合) | 0.00302 | 0.00118 | FNO 更优 |

- 仅 2% 参数量，在 DIFF 上降低 43% 测试误差
- 联合训练（无共享物理的 PDE 混合）时 FNO 略优，说明 CoDA-NO 在有共享物理时优势更明显

**Ablation 关键发现：**
- 去掉 Normalization → 模型**不收敛**（5-shot 场景直接失败）
- 去掉 VSPE → 性能显著下降（0.024→0.045 on NS, 5-shot）
- 去掉 Codomain Attention → 等价于 patch-based attention，性能退化为 baseline 水平（0.024→0.271 on NS, 5-shot）
- 冻结 Encoder 仅训练 Predictor → 仍显著优于所有 baseline（参数效率极高）

### 明确局限

1. **作者自述**：性能受训练样本数影响大，建议整合物理信息方法（physics-informed）进一步改进
2. **仅 2D 实验**：所有 multiphysics 实验均为 2D（FSI 是 2D channel flow + 2D 弹性 strap；RB 是 2D 矩形域），未展示 3D 能力
3. **仅两个 multiphysics 系统**：FSI 和 RB，代表性有限。没有涉及 3 个以上耦合 PDE 的系统
4. **计算开销偏大**：训练和推理时间显著高于 GNN/GINO（~10x），对实时应用不友好
5. **PDEBench 联合训练表现不突出**：当预训练和微调 PDE 无共享物理变量时（NS+DIFF+SWE 混合），CoDA-NO 在预测任务上不如 FNO，说明 codomain attention 的前提是有共享的物理变量语义
6. **温度参数 τ 的选择未深入讨论**：仅提到是一个 hyperparameter，没有系统性研究其影响
7. **没有与物理信息方法对比**：如 PINNs、physics-informed neural operators，虽然 Conclusion 中提到这是未来方向
8. **VSPE 的学习方式**：可学习的 Fourier 系数仅在预训练阶段的变量上学习，对新变量的 VSPE 需要微调时从头学——对于与预训练变量语义差距大的新变量，这种迁移可能有限
9. **不规则网格的 mask 策略依赖邻域半径**：半径选择可能影响预训练质量，未做消融
10. **无理论收敛保证**：虽然声称神经网络算子的离散化收敛性，但未给出 CoDA-NO 特有的理论保证或误差界

- **可用工具/方法**：
  - CoDA-NO 的开源代码（neuraloperator 库）可直接作为 baseline 或起点
  - FNO/GNO 作为 K/Q/V 算子的设计模式可以复用
  - Function Space Normalization 是一个通用组件，可用于其他 neural operator 架构
  - Masked reconstruction 预训练框架可用于不同 PDE 系统
  - TurtleFSI 数据生成 pipeline 可直接复用于 FSI 实验

- **该引不该引**：如果我们的工作涉及 neural operator、multiphysics PDEs、或 PDE foundation model，**必须引用**。CoDA-NO (NeurIPS 2024) 是 multiphysics neural operator 的 SOTA baseline

- **差异化方向**：
  1. CoDA-NO 没做 physics-informed pretraining → 可以将物理约束（PDE residual）加入预训练目标
  2. CoDA-NO 仅测试了 2 个 multiphysics 系统 → 可以在更多样的耦合 PDE 系统上验证（如 3+ 耦合方程）
  3. CoDA-NO 用固定 temperature τ → 可以研究 learnable temperature 或 attention sparsity
  4. CoDA-NO 的 VSPE 对所有新变量从零学 → 可以研究 VSPE 的 zero-shot generalization（如通过物理变量语义编码）
  5. 3D multiphysics 实验是明显的 gap
  6. CoDA-NO 在无共享物理变量时表现不佳 → 可以研究更通用的 cross-PDE 迁移机制

## Read by: 0616-fex (2026-06-24)

### 审读视角

CoDA-NO 是我在 multiphysics PDE foundation model 方向上看到的最系统性的工作之一。它的核心创新——**codomain attention**（将物理变量而非空间位置视为 token）——是本质上不同于 ViT-style patching 的范式转换。

**撞车评估：**
如果 0616-fex 的方向也涉及 "multiphysics PDE 的预训练 + 微调框架"，CoDA-NO 是直接 prior art。需要明确我们的方法在以下至少一个维度上有本质差异：(1) 预训练目标不只是 masked reconstruction；(2) 处理 3D 或 ≥3 耦合方程的场景；(3) 引入 physics-informed 约束；(4) 变量之间的 attention 机制有根本不同（如 learnable sparsity、variable-semantic embedding）。

**可迁移工具：**
- `neuraloperator` 库（也是 CoDA-NO 的代码所在）可以作为我们的基础框架
- Function Space Normalization 是通用组件
- GNO 编码器/解码器 + 均匀 latent grid 的 pipeline 设计可以直接复用
- TurtleFSI 数据生成流程可用于我们的 FSI 实验

**差异化空间（推荐利用）：**
1. CoDA-NO 在无共享物理的 PDE 混合训练上表现不佳 → 我们的方法如果能在更异构的 PDE 集合上实现正迁移，是一个强 claim
2. 3D multiphysics 实验完全空白
3. Physics-informed pretraining（在预训练目标中加入 PDE residual）未被探索
4. VSPE 的 zero-shot generalization（通过物理变量的语义描述编码而非从零学习）是一个有趣的方向
