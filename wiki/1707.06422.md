# Domain Adaptation by Using Causal Inference to Predict Invariant Conditional Distributions (1707.06422)

- **Year**: 2017.07 (NeurIPS 2018)
- **Authors**: Sara Magliacane (IBM Research / U. Amsterdam), Thijs van Ommen, Tom Claassen, Stephan Bongers, Philip Versteeg, Joris M. Mooij (U. Amsterdam / Radboud University)
- **Venue**: NeurIPS 2018
- **Keywords**: causal domain adaptation, invariant conditional distribution, separating set, Joint Causal Inference (JCI), transfer bias, incomplete information bias, covariate shift, Markov blanket vs. causal feature set, answer-set-programming causal discovery
- **One-liner**: 用因果分离集(separating set)做特征选择以保证 zero transfer bias，证明并用受控 γ(扰动强度)×N(样本量) 扫描实验实证：因果条件化只在扰动足够大时才跑赢忽略分布变化的标准特征选择基线。

## 精华提炼

### 管线/方法

输入 → 输出的完整流程：

1. **问题设定（Task 1，causal domain adaptation task）**：给定一个或多个 source domain（记 $C_1=0$）的完整数据，以及一个或多个 target domain（记 $C_1=1$）的数据——除了目标变量 $Y=X_j$ 的值缺失。系统变量 $\{X_j\}$ 与 context 变量 $\{C_i\}$（domain 指示器，视为外生变量）共同构成一个 SCM。任务：仅用 source 数据估计的规律，预测 target domain 里缺失的 $Y$。
2. **核心假设（3 条，比标准 Markov+faithfulness 更弱）**：
   - (i) $\Prb(\B V)$ 对完整因果图 $\C G$（含 context 变量）Markov 且忠实；
   - (ii) 涉及 $Y$ 的条件独立性从 source 的混合分布可传递到 target 的混合分布（比"source 和 target 分别对同一个子图 Markov+忠实"更弱，附录证明了这一蕴含关系）；
   - (iii) $C_1$（目标 domain 的干预指示器）对 $Y$ 没有直接因果效应——这是不可测试但被认为是使问题 well-posed 必要的强假设。
3. **核心方法：分离集(separating set)**。寻找一个特征子集 $\B A \subseteq \B V\setminus\{C_1,Y\}$，使得 $C_1 \dsep Y \mid \B A$ 在因果图 $\C G$ 中成立（d-separation）。由 Markov 假设，这蕴含 $C_1 \indep Y \mid \B A$ 在联合分布中成立，从而 $\Prb(Y\mid \B A, C_1=0)=\Prb(Y\mid \B A, C_1=1)$：条件分布在 source/target 间不变，可安全迁移。
4. **偏差分解（本文最核心的诊断工具）**：用 $\hat Y^0_{\B A}$（在 source 上训练、忽略 domain 差异）预测 target 的总偏差 = **transfer bias**（$\hat Y^1_{\B A}-\hat Y^0_{\B A}$，因为分布真的变了）+ **incomplete information bias**（$\hat Y^1_{\B V\setminus\{Y,C_1\}}-\hat Y^1_{\B A}$，因为只用了子集特征而非全部信息）。用全部特征忽略分布变化：transfer bias 可以无穷大（即使无穷样本下）；用因果分离集：transfer bias 恒为 0，但可能因特征减少而 incomplete information bias 增大。**这正是"causal conditioning vs. non-causal 全特征/marginal 预测"权衡的最干净数学刻画**。
5. **可实现算法（不需要已知因果图/干预目标）**：
   - Step 1：用标准特征选择方法（Random Forest 的 out-of-bag 分数，scikit-learn 默认参数）对 source 数据上所有候选特征子集按经验风险升序排序；
   - Step 2：从最优候选子集开始，把该子集"是否为分离集"的问题转成一个逻辑查询：将 JCI assumption（Assumption 1）+ "$C_1\to Y\notin\C G$"（Assumption 2(iii)）编码为约束，喂给自动定理证明器；可测试的条件独立性 p 值（用 partial correlation test, $\alpha=0.05$）作为输入，通过 Ancestral Causal Inference (ACI, Magliacane et al. 2016) 的置信度加权方案 + ASP solver `clingo 4.5.4` 求解，判断是否能证明 $C_1\dsep Y\mid \B A$；
   - Step 3：能证明则用该子集预测（用协变量偏移稳健的方法处理 $\B A$ 分布本身的 shift）并停止；不能证明则换下一个候选；全部候选都不行则**弃权（abstain）**，不做预测。
   - 计算瓶颈：ASP 求解器 + 遍历所有特征子集，2018 年时"笔记本电脑上大约只能处理 7 个变量"。

### 关键技术细节

- **动机例子（Example 1, Fig. 1）**：3 变量 $X_1\to X_2\to X_3$，$C_1$ 同时影响 $X_1,X_3$。标准特征选择在 source 上会选 $\{X_3\}$或$\{X_1,X_3\}$（因为 $X_3$ 预测 $X_2$ 方差更小），但在 target domain 里 $\Prb(Y\mid X_3)$ 剧烈变化，导致任意大偏差；而 $\{X_1\}$ 虽在 source 上预测方差更大，但 $\Prb(Y\mid X_1)$ 跨 domain 不变，可安全迁移。
- **反例 (Example 2, `ex:identifiable_but_fs_fails`)**：构造了一个可从数据条件独立性识别出分离集 $\{X_1\}$ 的具体 ADMG 族，并证明标准 feature selection 会渐进选出 Markov blanket $\{X_1,X_3\}$（因为其在 source 上确实是最优预测集），在 target domain 上可导致任意大误差。文中脚注明确指出 **Rojas-Carulla et al. (2015/2018) 的 Invariant Causal Prediction 风格方法在这个反例上会给出错误答案**：他们假设"若不变性在所有 source domain 间成立，则在 target 也成立"，但该假设在此反例中被违反。
- **合成数据生成器**：改编自 Hyttinen 等 (`antti`) 的模拟器。200 个随机线性无环 SCM，3 个系统变量 + 2 个 context 变量 + 0-2 个随机 latent 变量；结构方程系数从 $\mathcal N(\pm0.2, 0.64)$ 采样，噪声 $\mathcal N(0,0.0064)$；context 变量模拟 soft intervention（在其对应 domain 里取 1，否则取 0）；随机选 $C_1$、$Y$，强制 $C_1$ 不直接影响 $Y$、且无 context 变量同时影响所有系统变量。
- **核心受控扫描实验（附录，Fig. `extra_results_ifactor` / `extra_results_samples`）——本轮任务重点关注的可迁移设计**：
  - **扰动强度扫描**：引入缩放因子 $\gamma$，把 context 变量的系数乘以 $\gamma$，扫描 $\gamma\in\{0.1, 1, 10, 100\}$（$N=1000$ 固定）。
  - **样本量扫描**：固定 $\gamma=10$（"large perturbation"），扫描 $N\in\{100,1000,5000\}$。
  - 两个扫描维度构成一个干净的 2×2-ish factorial 网格，用箱线图展示 baseline（标准 feature selection，忽略 shift）vs. 因果方法（分离集）的 $L_2$ 预测损失分布。
- **真实数据实验**：IMPC（International Mouse Phenotyping Consortium）血液学表型数据，441 只野生型 + 13 个基因敲除各约 10 只突变体老鼠，取 16 个非确定性关系的表型。随机采 1000 个 3-变量+2-knockout 子问题，每个子问题随机选 $Y,C_1$ 并挖去 target domain 里 $Y$ 的值。

### 核心结果

- **$\gamma=0.1$（极小扰动，source/target 分布几乎相同）**：因果方法（分离集）**完全没有优势，甚至比 baseline 表现更差**——这是本文最重要的诚实负结果，直接对应"non-causal marginal 基线通过存在性门槛，causal conditioning 落 chance"这一现象的最早受控复现之一。
- **$\gamma=1,10,100$（扰动递增）**：因果方法优势开始显现，且**优势随 $\gamma$ 增大而单调增大**（Fig. `synthresults_vsmall`→`synthresults_large`）。
- **样本量效应**：$N=100$ 时两种方法准确率都低；$N=1000$ 时大幅改善，且**因果方法改善幅度更大**（因为条件独立性检验的功效随 $N$ 提升）——与 round1/2 已建立的"CI 检验功效随条件集增大/样本不足而下降"([[1911.07147]]) 直接呼应，是同一枚硬币的另一面（这里是"CI 检验功效不足 → 因果方法在小样本下无法体现其理论优势"）。
- **真实小鼠数据**：在 1000 个子问题中，因果方法在 **170 个（17%）上弃权**；在剩余 830 个给出答案的子问题上，与 baseline **表现相近、无明显优势**——这是"大规模/真实数据审计下因果特征全面弱于/持平全特征预测"现象（对应 [[2402.09891]]）在一个独立、更早（2017）数据集和方法上的复现。

### 明确局限

- Assumption 2(iii)（$C_1$ 对 $Y$ 无直接因果效应）不可测试，是让问题 well-posed 的代价；作者承认这一点，只给出脚注级别的部分放松方案。
- 可扩展性差：ASP 求解器 + brute-force 遍历特征子集，2018 年时约 7 个变量是笔记本电脑上的实际上限，远小于我们讨论的 fragility regime 场景里可能涉及的变量规模。
- 全文实验都是**静态/横截面因果图上的 domain adaptation**（用 JCI + ACI + ASP 定理证明器），不涉及时序动态系统或 PCMCI 类算法；与我们 PCMCI+ fragility regime 的时序设定有方法论层面的距离，但"causal-constrained 预测 vs. 忽略结构的全特征/marginal 预测"这一权衡的性质是通用的。
- 真实数据实验样本极小（每个子问题仅约 20-30 个数据点：source ~441 wild-type 均分 + target ~10 只突变体），本身就处于"CI 检验功效不足"区间，故该实验更多是佐证"小样本使因果方法优势消失"而非提供强 positive evidence。
- 没有报告 AUC/ROC 类判别指标（本文任务是回归预测，用 $L_2$ loss），若要迁移到我们的 fragility regime 分类场景，需要自行把"预测损失差"重新表述为"分类判别力差"。

### 跟我们的关系

- **不撞车，是同一现象更早、更干净的独立源头**。我们的 causal-scs-indicator 假设——"PCMCI+ causal conditioning 在 fragility regime 判别力落 chance，而 non-causal marginal 基线通过存在性门槛，但两者在真实图变化正对照下都好"——与本文"因果分离集在小扰动($\gamma=0.1$)下没有优势甚至更差、在大扰动($\gamma\geq1$)下优势随强度单调增大"的发现是**同一机制在不同任务包装(回归预测 vs. regime 分类)下的表现**。本文比 round1/2 已读的 [[2002.09414]]（2020，Markov blanket 最优性）和 [[1802.05844]]（2018，anchor regression minimax 边界）**更早（2017）**，且是这条谱系里第一个把"扰动强度"做成显式连续/可扫描参数、而非仅做二元 in-distribution/out-of-distribution 对比的实证工作。
- **直接可迁移的实验设计模板（回应本轮问题 b）**：本文的 $\gamma$（intervention/perturbation 缩放因子，扫描 0.1→1→10→100）× $N$（样本量，100→1000→5000）2D factorial 扫描，正是把我们"两个混淆 regime"退化为"连续强度扫描"的现成骨架。迁移方案：固定一个小型因果图族（类比本文的 3 系统变量+2 context 变量随机 SCM），引入一个显式的扰动/机制变化幅度参数 $\gamma$（对应我们 fragility regime 里"图结构变化"的等效强度），在 $\gamma$ 的 log-scale 网格上分别跑 PCMCI+ conditional 判别器和 marginal 判别器，直接画出 $\Delta=\mathrm{AUC}_{\text{conditional}}-\mathrm{AUC}_{\text{marginal}}$ 关于 $\gamma$（以及样本量 $N$）的曲线，观察 $\Delta$ 何时穿越 0。这比本文的 $L_2$ loss 更直接对应我们的判别力指标，只需将回归任务替换为分类/判别任务即可复用其"用缩放因子控制扰动幅度"的核心手法。
- **偏差分解框架可直接借用**：本文的 transfer bias / incomplete information bias 分解，为我们提供了一个可以镜像到 AUC 差值上的解释性框架——$\Delta$ 为负（causal 输给 marginal）对应"incomplete information 项占主导"（条件化后可用判别信息变少，且 CI 检验在有限样本下功效不足，参见 [[1911.07147]]），$\Delta$ 为正对应"transfer/鲁棒性项占主导"（marginal 基线因未做因果条件化而在真实图变化下判别力失真）。
- **竞争方法关系澄清**：本文明确指出 Rojas-Carulla et al. 的 Invariant Causal Prediction 风格方法（不同于我们此前读到的 [[1812.08233]] 的 ICP，是同一大家族的另一分支，2015 年 arXiv:1507.05333／2018 JMLR 版）在一个具体反例上会给出错误的分离集。下一轮若继续深挖这条 invariance-testing 谱系，应把 arXiv:1507.05333（Rojas-Carulla et al., "Causal Transfer in Machine Learning"）列为高优先候选，以确认它与 [[1812.08233]] 的关系及其是否报告了类似的"小扰动下无优势"现象。

## Read by: 0710-causal-scs (2026-07-11)

本文是 causal-scs-indicator round1/round2 已建立现象学（[[2002.09414]] Markov blanket 最优性、[[1802.05844]] anchor regression minimax 边界、[[1812.08233]] ICP 假阳性率保证不含功效保证、[[1706.08576]] CI 检验二阶矩盲区、[[1911.07147]] CI 检验功效随条件集增大下降、[[2402.09891]] 大规模真实数据审计因果特征弱于全特征）这条谱系里**时间上最早（2017）且实验设计上最直接可迁移**的一篇。

**不撞车判断**：本文的方法（JCI + ACI + ASP 定理证明器做 domain-adaptation 特征选择）与我们的 PCMCI+ conditional fragility-regime 判别在具体技术路线上完全不同，任务类型也不同（回归预测 vs. regime 分类）。但其核心实证发现——"因果约束方法在扰动幅度=0.1 时没有优势甚至更差，在扰动幅度≥1 时优势随幅度单调增大，且这一优势的显现速度依赖样本量（CI 检验功效）"——与我们的假设在机制层面完全吻合，是可以在 idea/proposal 文档中引用的独立佐证，不构成 novelty 冲突。

**可用工具/方法**：
1. Transfer bias / incomplete information bias 的偏差分解，可直接借用作为解释 $\Delta_{\text{AUC}}$ 符号的理论语言。
2. $\gamma \times N$ 2D 扫描的实验设计骨架，是本轮问题 (b) 要求的"可直接迁移的受控 factorial/连续强度扫描实验设计"里目前读到的最接近现成模板的一篇——比 anchor regression 论文（[[1801.06229]]）的理论 minimax 边界更偏工程可操作，比我们自己当前"两个混淆 regime"的设计更干净。
