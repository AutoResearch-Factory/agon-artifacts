# Physics-informed Temporal Alignment for Auto-regressive PDE Foundation Models (2505.10930)

- **Year**: 2025.05
- **Authors**: Congcong Zhu, Xiaoyan Xu (equal contribution), Jiayue Han, Jingrun Chen — University of Science and Technology of China (USTC) + University of Kansas Medical Center
- **Venue**: ICML 2025 (accepted)
- **Keywords**: PDE foundation model, auto-regressive prediction, shortcut learning, temporal alignment, sparse regression, PDE discovery, physics-informed finetuning, inverse problem
- **One-liner**: 提出 PITA 框架，用数据驱动的 PDE 发现作为物理约束来对齐自回归预测的时间一致性，解决 PDE 基础模型的 shortcut 误差累积问题。

## 精华提炼

### 管线/方法

完整 pipeline（图 pipeline_final.png）：
1. **自回归预测阶段**：预训练 PDE 模型（DPOT / MPP / FNO）取初始 $T_{in}=10$ 帧作为输入，自回归预测未来 $T_{ar}$ 帧（1 或 10 步）。每预测一帧即将其拼接至输入序列。
2. **数据压缩**：将预测序列在空间域随机下采样至 1/4 网格点数（$l=4$），时间域保留最后 $T_C=3$ 帧。
3. **构建候选项库**：基于压缩数据构造 $\bm{\varPhi}(\bm{\theta}) \in \mathbb{R}^{(n\times m)\times S}$，包含多项式项（$u$, $u^2$, ...）、空间导数（$\partial_x u$, $\partial_{xx} u$, ...）及混合项（$u\partial_x u$）。导数用多项式插值有限差分计算。
4. **稀疏回归 PDE 发现**：用 STRidge（Algorithm 1）求解 $\partial_t \bm{U} = \bm{\varPhi} \bm{\varLambda}$，得到稀疏系数矩阵 $\bm{\varLambda}$。
5. **三项损失函数**：
   - $\mathcal{L}_{Data}$：归一化相对误差（nRMSE 形式）
   - $\mathcal{L}_{Phy}$：$\|\partial_t\bm{U} - \bm{\varPhi}\bm{\varLambda}\|_2^2 + \alpha\|\bm{\varLambda}\|_0$
   - $\mathcal{L}_{Con}$：$\|\bm{\varLambda}^* - \bm{\varLambda}_i\|_2^2$，对齐预测数据的物理定律与真实数据的物理定律
6. **不确定性加权 + 交替方向优化（ADO）**：$\frac{1}{2\delta_i^2}\mathcal{L}_i + \log\delta_i$ 自适应调权；交替更新 $\bm{\varLambda}$（稀疏回归）和 $\bm{\theta}, \bm{\delta}$（梯度下降）。

### 关键技术细节

- **模型**：DPOT (Ti/S/M/L: 7M/30M/122M/509M)、MPP (Ti/S: 7M/30M)、FNO-M (170M)
- **数据集**（12个）：FNO-NS-$\nu$ (3种粘度: 1e-5/1e-4/1e-3), PDEBench-CNS (6种参数组合), PDEBench-DR (扩散反应, 101步), PDEBench-SWE (浅水方程, 101步), PDEArena-NS/NS-Force, CFDBench-NS, Burgers (泛化测试)
- **优化器**：Adam/AdamW，初始学习率 1e-3，Cycle 学习率调度，warmup 50 epochs
- **训练 epochs**：finetune 500-2500 epochs（不同实验设置不同）
- **$\ell_0$ 正则化系数** $\alpha$：用于 STRidge 的稀疏性控制
- **下采样策略**：空间 $w=\lceil n/4\rceil$，时间保留 3 帧（为主实验）；消融显示 $T_C=7$ 最优但 $T_C=3$ 接近且计算更高效
- **GPU**：单卡 A800 80GB
- **PDE 发现工具**：STRidge（Sequential Threshold Ridge Regression），带 hard threshold $\beta$ 和最大迭代次数 $K$
- **候选项库**：包含常数项、多项式项（1-2 阶）、空间一阶/二阶导数、混合项（共 S 个 term）
- **STRidge 时间开销**：0.08-0.11 秒/batch，不随模型规模增长
- **额外计算开销**：Ti 模型约 42% 时间增加，L 模型约 22%（因大模型基础训练时间更长）

### 核心结果

| 指标 | 结果 |
|------|------|
| 短轨迹（10步）小模型 (Ti/S) | 75% 任务最优，平均 nRMSE 降 11.21%，最大 63.81% (DPOT-S on FNO-NS-1e-4) |
| 短轨迹中大模型 (M/L) | 77.78% 任务最优，平均降 13.53%，最大 53.36% (DPOT-L on FNO-NS-1e-3) |
| 长轨迹预测 | 平均累计误差降 30.22%，每步平均降 2.00% |
| Burgers 泛化 | 平均 nRMSE 降 37.8% vs baseline |
| CNS 泛化 | 平均 nRMSE 降 5.93% |
| 消融 Task 1：$\mathcal{L}_{Phy}$ 贡献 | 单独加 Physics Loss 使准确率提升 44.97% |
| 消融 Task 1：$\mathcal{L}_{Con}$ 贡献 | 单独加 Consistency Loss 使准确率提升 44.10% |
| 噪声鲁棒性 (noise=0.05) | nRMSE 从 0.45795→0.3214（降 29.8%）|
| 数据 masking 25% | nRMSE 从 0.1264→0.0801 (91步) |
| Kolmogorov 湍流 (150步) | nRMSE 从 0.4459→0.3152（降 29.3%）|
| 数据效率 | PITA 用 500 样本达到 auto-regressive 1000 样本的精度 |
| 参数量效率 | PITA 30M 参数量可超越 auto-regressive 100M 参数量的表现 |

**Surprising 发现**：
- 预训练模型的 out-of-distribution 泛化很差，有时甚至不如从头训练（Table scratch）：PDE 基础模型的预训练主要学到训练数据的物理规律，不同 PDE family 之间的物理规律相似性很低
- 空间下采样越大（网格越少）反而效果越好（$l=4$ 最优于 $l=1$），因为稀疏采样迫使模型关注关键空间特征
- PITA 在物理 loss 完全失效时仍能通过 data loss + consistency loss 获得提升

### 明确局限

1. **额外计算开销**：PITA 比纯自回归多 22%-42% 训练时间（Ti 模型从 0.29s/batch → 0.41s/batch；L 模型从 2.52s → 3.04s），但该开销不随模型规模增长
2. **短期预测收益较小**：短轨迹（10步）的 nRMSE 降幅（约 15.8%）远小于长轨迹（30.22%）
3. **依赖候选项库完整性**：library of terms 必须覆盖实际的 PDE 结构，缺失关键项会影响效果；但实验显示 80% 保留也能接近全库性能
4. **依赖低噪声数据**：噪声数据会影响 PDE 发现的精度
5. **预训练模型的 OOD 局限**：论文本身发现 finetuning pretrained model 在 OOD 上不如 train from scratch
6. **稀疏回归方法依赖**：伪逆法和最小二乘法的残差比稀疏回归大 2-3 个数量级，可能导致梯度爆炸
7. **与 fex 的关系**：PITA 解决的是自回归预测的 shortcut/误差累积问题，本质是一个 finetuning 框架；它没有解决搜索问题（search），也不是新的 PDE 求解架构。它表明物理一致性约束对自回归预测很有价值；这个 insight 对 fex 的 auto-regressive 管线有参考意义

## Read by: 0616-fex (2025-06-25)

### 撞车评估
PITA 与 0616-fex 不撞车。PITA 聚焦于 PDE 基础模型的自回归预测 shortcut 问题，核心贡献是将 PDE 发现（inverse problem）转化为物理一致性约束（physics loss + consistency loss），用于 finetuning 已有预训练模型。0616-fex 的核心是 PDE 搜索/探索机制（search over PDE function space），两者的 problem setting 正交。

### 可复用工具
1. **STRidge + 候选项库**：PDE 发现的完整 pipeline（library construction → sparse regression）可以直接作为 feX 搜索管线中的验证/约束模块
2. **不确定性加权多任务损失**：$\frac{1}{2\delta^2}\mathcal{L}+\log\delta$ 权重自适应机制可用于 feX 在多目标搜索中的损失平衡
3. **ADO 交替优化**：解耦稀疏回归和梯度优化的框架，对 feX 中物理约束和搜索策略的交替迭代有参考价值
4. **Shortcut 诊断方法**：Temporal SSIM vs Frame SSIM 的量化方法可用于检测 feX 的 auto-regressive 组件

### 差异化方向
feX 的独特贡献在于"搜索"——在 PDE function space 中探索和选择。PITA 假设已知 training data（特定 PDE 的轨迹），专注于如何更好地拟合。feX 可以引用 PITA 作为监督信号的设计参考，但核心创新（搜索策略、探索机制、function space 表征）与 PITA 完全不同。

### 关键风险提示
论文的 empirical finding 值得注意：大规模 PDE 预训练的 OOD 泛化能力可能被高估——finetuning pretrained model 有时不如 train from scratch。如果 feX 计划依赖预训练模型作为 backbone，这个发现需要严肃评估。
