# Are you sure? A Comprehensive and Comprehensible Survey of Uncertainty Quantification in Symbolic Regression (2606.06567)

- **Year**: 2026.06
- **Authors**: Julia Reuter and Fabricio Olivetti de Franca (Center for Mathematics, Computation and Cognition, Federal University of ABC, Brazil)
- **Venue**: arXiv
- **Keywords**: symbolic regression, uncertainty quantification, Bayesian symbolic regression, Fisher information, confidence interval, conformal prediction, model selection, minimum description length
- **One-liner**: 系统梳理 SR 中 UQ 的统计基础、方法谱系和研究缺口。

## 精华提炼

### 管线/方法

这篇论文不是提出一个新的 SR 算法，而是一篇面向 symbolic regression (SR) 的 uncertainty quantification (UQ) survey。

它先把 SR 写成在假设空间 `H` 中搜索函数 `f(x; theta)` 的问题，区分最优可达模型 `h*`、搜索得到的模型 `hat h`、参数 `theta` 和数据集 `D={(x_i,y_i)}`。

作者强调 SR 的不确定性至少有三层：aleatoric uncertainty 来自测量噪声或数据生成随机性；model uncertainty 来自真实函数不在当前搜索空间中；approximation / parameter uncertainty 来自已有搜索空间内多个模型或参数都能解释数据。

全文的技术管线是：先建立 likelihood / Fisher information / confidence interval / prediction interval 的统计基础，再介绍 frequentist、Bayesian 和 model selection 三类 UQ 思路，最后系统整理 SR 文献中已经出现的 UQ 相关方法。

文献检索来源是 DBLP、The Genetic Programming Bibliography 和 Google Scholar。

检索关键词都与 `"symbolic regression"` 组合，包括 uncertainty、uncertainty quantification、Fisher information、Hessian、epistemic、aleatoric、conformal prediction、Bayesian、Bayes、MDL、AIC、BIC、model selection、regularization、noisy data、noise、bias-variance、statistical machine learning。

纳入标准是：论文在 SR 场景中提出或改造 UQ 方法；使用依赖 UQ 的模型选择方法；或建立 UQ 前提，例如 Bayesian posterior。

排除标准是：只是直接套用 cross-validation、AIC、BIC 的论文；主要讨论 generalization 但没有明确 UQ 机制的论文；dynamic / uncertain environment 但不是统计 UQ 的论文；以及方法细节不完整且联系作者后仍不能澄清的论文。

最终作者整理出 18 篇 UQ-related SR 论文：11 篇 Bayesian SR，4 篇 non-Bayesian 方法，3 篇 model selection 方法。

这个数量本身就是结果之一：SR 社区对 UQ 的系统性处理还很少，且 18 篇中只有 3 篇发表于 2020 年前。

论文把基础概念和 survey 分开：基础概念部分服务于读者理解 UQ；survey 部分逐篇讨论 SR 中这些概念怎么被真正用上。

Frequentist 线索包括 Fisher information、Hessian、parameter standard error、confidence interval、prediction interval、profile likelihood、conformal prediction。

Bayesian 线索包括 posterior distribution、MCMC、credible interval、posterior predictive interval、Bayes factor、fractional Bayes factor、SMC、model-structure posterior、structural prior、variational inference。

Model selection 线索包括 Bayes factor、fractional Bayes factor、MDL、AIC、BIC 和 cross-validation，其中 AIC/BIC/CV 的细节放在 appendix 里。

作者对 Bayesian SR with SMC 和 Bayesian forest / soft symbolic tree 两条线写得最细，因为这些工作试图近似模型结构上的 posterior，而不是只给固定表达式的参数做误差条。

### 关键技术细节

Likelihood 设定：回归模型写作 `y_i = f(x_i; theta) + epsilon_i`，常见假设是 i.i.d. Gaussian noise `epsilon_i ~ N(0, sigma_i^2)`。

Homoscedastic 情形下 log-likelihood 是平方误差项加常数项；若 `sigma^2` 未知，UQ 需要先估计噪声方差。

Heteroskedastic 情形下每个数据点有不同 `sigma_i^2`，可用 weighted least squares、已知传感器方差、重复测量，或代理函数 `g(x_i; gamma) ~= sigma_i^2` 来估计。

MLE 通常通过最小化 negative log-likelihood 实现；Gaussian 情形下梯度为 `J(theta)^T (y - f(x;theta)) / sigma^2`。

Hessian 在局部最优附近可用 Gauss-Newton 近似为 `-J^T J / sigma^2`。

Observed Fisher information 是 negative Hessian；其逆矩阵给出参数方差估计，平坦或 ill-conditioned likelihood surface 表示更高参数不确定性。

作者特别指出：参数 UQ 只量化某个已收敛 optimum 附近的不确定性，不能解决多峰 likelihood 中“选哪个峰”的模型选择问题。

Frequentist confidence interval 用 `theta_hat +- t * s_theta`；prediction interval 额外包含新观测的 irreducible noise，所以一定宽于 confidence interval。

Conformal prediction 的优势是 finite-sample coverage、model-agnostic、distribution-free；代价是需要 calibration set，小样本时 interval 可能很宽。

Bayesian credible interval 直接覆盖 posterior probability mass；SR 中 posterior 往往无闭式解，通常用 MCMC samples 排序后用 shortest sliding window 近似 highest posterior density interval。

Posterior predictive interval 还要对 `sigma^2` 和 posterior samples 采样，因此比 posterior mean 的 credible interval 更宽。

Bayes factor 比较两个 hypothesis 的 marginal likelihood；若 prior improper，fractional Bayes factor (FBF) 用一小部分 likelihood 生成 proper prior。

线性模型中常用 `gamma=d/N`，Laplace approximation 下 FBF 只依赖 likelihood、参数数 `d` 和 fraction `gamma`。

MDL 把总 codelength 写成 `L(D)=L(h)+L(D|h)`，其中 `L(D|h)` 常是 negative log-likelihood，`L(h)` 编码表达式复杂度和参数精度。

Bayesian SR with SMC：Bomarito / Leser 线工作把 SR 的参数优化和模型选择 Bayesian 化，用 sequential Monte Carlo 估计 normalized marginal log likelihood (NMLL)。

SMC 从 proposal distribution 出发，逐步提高 likelihood exponent `phi_t`，通过 reweight、resample、MCMC mutate 避免 particle degeneracy。

SMC 的优势是能处理多峰 posterior；代价是比 Laplace approximation 更慢，且窄峰或相距很远的 mode 仍会造成数值困难。

Bomarito 2026 进一步把 posterior 目标从连续参数扩展到离散 symbolic model structures，用 SMC 近似 `p(h|D)`。

该 model-space SMC 中，粒子是表达式，权重由 marginal likelihood / NMLL 给出，resampling 后用 genetic operators 产生 offspring，再按 MH acceptance probability 接受或拒绝。

Guimera / Sales-Pardo 线工作使用 structural prior：从 Wikipedia 数学表达式语料估计算子频率，配合 BIC / marginal likelihood 做模型选择。

Bartlett / Desmond / Ferreira 的 priors for SR 进一步引入上下文，用 n-gram / Katz back-off 模型估计树节点在祖先上下文下出现的概率。

Bartlett / Desmond 的 x-error likelihood 处理输入变量 `x` 本身有测量误差的场景，把 latent true `x_t` marginalize 掉；这对天体物理、实验测量型 SR 很关键，但计算更贵。

Agapitos et al. 的 Bayesian model averaging 用多样化 SR 模型 ensemble 处理 approximation uncertainty，给每个模型学习权重和输出方差。

Roy et al. 的 HierBOSSS 把模型写成多个 symbolic tree 的线性组合，借助 Normal-Inverse-Gamma conjugacy 对线性系数和噪声方差做闭式 marginalization。

Roy et al. 的 VaSST 用 soft symbolic trees 和 Gumbel-Softmax relaxation 做 variational inference，优化 ELBO 后从节点级 operator / feature distributions 中采样硬树。

作者对 Roy 线工作评价较谨慎：概念新，但线性树组合限制了非线性参数表达力；mean-field 假设忽略节点依赖；作者还检查源码并指出 reported experiments 与代码实现有不一致。

Non-Bayesian 方法包括 Hoekstra 的 symbolic quantile regression、de Franca / Kronberger 的 profile likelihood interval、Manzi / Vasile 的 bootstrap structural UQ、Schmidt / Lipson 的 random terminal `R` noise modeling、Thuong et al. 的 conformal prediction + GP。

Profile likelihood 是本文认为很有价值的 SR-specific frequentist 方法：固定某个参数，重新优化其他参数，得到非对称、连续、可揭示 unbounded interval 的 CI。

Bootstrapping 方法能可视化不同数据重采样下表达式结构和参数分布，但如果只画 histogram 而不估计分布，统计解释力有限。

Schmidt / Lipson 的 random terminal `R ~ U[-1,1]` 把噪声关系直接纳入表达式搜索，可免费产生 prediction range，但没有传统 PI 或 conformal prediction 的 formal guarantee。

### 核心结果

这是一篇 survey，因此核心结果不是 benchmark 数字，而是对 SR-UQ 文献结构的归纳。

最重要的数字是：作者只找到 18 篇真正 UQ-related 的 SR 论文，其中 11 篇 Bayesian、4 篇 non-Bayesian、3 篇 model selection。

只有 3 篇在 2020 年前发表，说明 SR-UQ 是很新的研究方向。

真正计算 confidence / credible / prediction interval 的论文很少，作者点名 Zhao 2025、Bomarito 2023、Thuong 2017、de Franca 2022。

Bayesian 方法是目前最活跃的一支，因为 MCMC / SMC 天然产生 posterior samples，理论上 UQ “comes almost for free”。

但 Bayesian SR 的现实瓶颈是 posterior over model structures 难以近似：搜索空间离散、组合爆炸、表达式等价类很多、参数 likelihood 多峰。

SMC-SR 与 GP variants 的比较中，作者转述 Bomarito 2026 的发现：在 noisy datasets 上 SMC-SR 训练和测试 error 都更好，并且更不容易 overfit。

SMC 的改进来源不只是 NMLL，还包括 likelihood tempering、probabilistic selection、resampling 带来的 exploration / exploitation 平衡。

Laplace approximation 更快，但假设 posterior 是 single MVN；遇到多峰 posterior 会漏掉 mode。

SMC 能更好捕捉多峰，但 runtime 高，并且 narrow modes / far-separated modes 仍困难。

结构先验方面，Guimera 的 operator frequency prior 简单但忽略上下文；Bartlett 的 n-gram prior 捕捉祖先依赖，但会受语料偏置影响。

VaSST / HierBOSSS 给出“posterior over tree-generating process”的新方向，但作者明确质疑其 mean-field 节点独立假设是否能近似真实 model posterior。

MDL 在 ESR 中不能保证第一名就是 ground truth，但 ground truth 往往在 top choices 中，因此适合作为候选筛选和后续人工审查工具。

Conformal prediction 在 SR 中能给出 coverage guarantee，但要留 calibration data；对于小数据科学发现任务，interval width 可能牺牲实用性。

Profile likelihood 比 delta method 更准确，能给出非对称 confidence regions，但计算成本更高。

作者的总体结论是：SR 社区系统性低估了 UQ 的重要性；未来 SR 方法应更认真分析数据噪声、模型选择不确定性、参数置信度和结构 posterior。

### 明确局限

作者自己承认：UQ in SR 仍然 underexplored，现有论文数量很少，很多方法只是初步探索。

本文是 survey，不提供新的算法 benchmark，也没有自己跑统一实验比较所有 UQ 方法。

文献检索主要依赖 DBLP、GP Bibliography 和 Google Scholar，没有给出 PRISMA-style 流程、完整 query log、去重规则或可复现检索脚本。

作者排除了“直接使用 AIC/BIC/CV”的论文，这有助于聚焦，但也可能漏掉工业或应用论文里实际可用的 UQ practice。

本文重点是 SR，不是 PDE solver；对 FEX、RL-based PDE symbolic solvers、高维 PIDE、FEX 的 S_k 近似理论没有专门讨论。

本文讨论噪声和 UQ，但没有给出一个可直接落地到 FEX residual loss / PDE collocation loss 的完整 protocol。

本文对 Bayesian methods 的评价很细，但对 RL controller 的 policy-gradient uncertainty、reward noise、search stochasticity、sample complexity 没有展开。

Survey 的“first to clearly address this issue” claim 可能成立于 UQ-in-SR 这个精确定义，但它不是 general SR robustness / generalization / noisy SR 的第一篇综述。

对 Roy 系列论文的源码审查很有价值，但 survey 中没有提供可复现实验脚本或详细日志；第三方很难直接复核作者的 replication attempt。

作者没有将 e-graph / expression equivalence 纳入 UQ 主线，尽管等价表达式对 model posterior 和 posterior mass 合并非常关键。

对 LLM-driven SR、Transformer SR、recent 2026 SR benchmark 的覆盖有限；这可能是 arXiv 时间点导致的，也说明这篇 survey 本身需要快速更新。

对真实实验数据、传感器噪声、PDE inverse problem、scientific law discovery 中的 noise model 讨论还偏概念，没有形成 benchmark-level 标准。

### 跟我们的关系

对 `0616-fex`，这篇论文不直接撞车：它没有提出 FEX/PDE solver，没有做 RL controller，没有给出高维 PDE 闭式解搜索，也没有触碰 FEX 的 S_k approximation guarantee。

它应该被视为 FEX 的“可靠性与评价层”背景文献，而不是竞品。

最直接相关的是 topic landscape 中的 gap #7：FEX 对噪声和不完整数据的鲁棒性。本文把 SR 中的 aleatoric / epistemic / model / approximation / parameter uncertainty 分清楚，能帮助把“FEX 在 noisy physical law discovery 中是否可靠”写成更严格的问题。

如果我们做 FEX 从实验数据发现物理定律，必须处理输入输出测量误差、heteroskedastic noise、重复测量、prediction interval 和 parameter CI；本文正好提供术语和方法清单。

如果我们做 FEX 的搜索效率或 controller 改进，这篇不提供 RL 收敛理论，但它提醒：只报 final RMSE 不够，还应报告 model posterior / candidate ambiguity / parameter uncertainty。

如果我们做 FEX macro lift 或低维到高维 skeleton transfer，这篇的 model structure uncertainty 很有用：lift selector 不应只给一个表达式，而应该给候选结构集、权重和拒绝条件。

如果我们做 LLM+FEX skeleton generation，Bayesian structural priors、MDL、FBF、NMLL 可作为 ranking / veto layer，用来避免 LLM 生成的复杂表达式过拟合 residual。

如果我们做 Multi-Scale FEX 或 synchrosqueezed prior，本文不是频率先验论文，但 profile likelihood / Fisher information 可帮助判断频率参数是否 identifiable。

可引用的位置：related work 中引用它作为 SR-UQ survey；method / evaluation 中引用它解释为什么需要 prediction intervals、confidence intervals、Bayesian posterior 或 MDL；discussion 中引用它说明 SR 社区对 UQ 的缺口。

不该把它作为“FEX 搜索复杂度”或“RL policy-gradient 收敛”的 prior art；它没有解决这些问题。

最可用工具是：profile likelihood、conformal prediction、Bayesian model averaging、SMC posterior over model structures、FBF/NMLL ranking、MDL with Fisher information。

差异化机会：FEX 社区可以做第一套 PDE-symbolic-solver-specific UQ protocol，把 residual error、coefficient uncertainty、operator-structure posterior、collocation noise、input measurement error 统一到一个 evaluation standard 里。

## Read by: 0616-fex (2026-06-16)

从 `0616-fex` 视角看，本文不构成直接竞争：它不是 FEX、不是 PDE solver、不是 RL 表达式树搜索论文。它的价值在于给 FEX 后续工作补上可靠性评价语言。FEX 目前 landscape 中有多个机会点：噪声鲁棒性、LLM / prior 引导、macro lift、e-graph 剪枝、搜索复杂度理论。本文最能服务的是“噪声鲁棒性”和“结构不确定性”两块：FEX 论文若只报告 residual / L2 error，无法回答表达式结构是否唯一、参数是否 identifiable、同等误差候选是否很多、预测区间是否覆盖真实解。可移植方法包括 profile likelihood 做系数 CI、conformal prediction 做分布无关 PI、SMC / NMLL / FBF 做候选表达式 posterior ranking、MDL 做复杂度惩罚。建议该引，尤其是在讨论 FEX 从干净 PDE 求解扩展到 noisy physical law discovery 时。差异化方向是提出 PDE-symbolic-solver-specific UQ protocol：把 PDE residual、采样点噪声、输入测量误差、表达式 posterior、参数 Fisher information 和结构拒绝机制合成一个 evaluation layer。
