# Evaluating the skill of a geometric early warning for tipping in a rapidly forced nonlinear system (2605.16128)

- **Year**: 2026.05
- **Authors**: Paul Ritchie, Sneha Kachhara, Peter Ashwin (Department of Mathematics and Statistics, University of Exeter, UK)
- **Venue**: arXiv preprint (funded by ClimTip / AdvanTip; CC BY)
- **Keywords**: rate-induced tipping (R-tipping), geometric early warning signal, R-tipping threshold, edge state, signed distance indicator, critical slowing down failure, ROC/AUC skill, AMOC box model
- **One-liner**: 在快速强迫下经典 CSD 预警失效, 提出"到 R-tipping 阈值的带符号距离"(R-tipping 指标)作为几何预警, 在 3 箱 AMOC 模型上比经典回复率强、比北大西洋盐度阈值稳。

## 精华提炼

### 管线/方法
本文是动力系统/气候临界点方向的 EWS 方法论文, 不是因果推断论文。核心管线:

1. **问题设定**: 非自治随机系统 `dx = f(x, P_θ(t)) dt + σ dW_t`, 其中 `P_θ(t)` 是外部强迫(forcing), 渐近常值(`t→±∞` 时 `P_θ→p_±`)。区分两类 tipping:
   - **B-tipping**(分岔诱导): 慢强迫准静态穿过分岔点, attractor 消失 → 突变。经典 CSD 预警(lag-1 自相关↑、方差↑、回复率↓)适用。
   - **R-tipping**(速率诱导): 快强迫即使**不穿过任何分岔**也能把系统推过盆边界(basin boundary)到另一个 attractor。经典 CSD 在这里**失效**, 因为快速强迫的系统会跑到远离 frozen 系统任何 attractor 的状态, 无法检测 attractor 韧性下降。
2. **构造 R-tipping 阈值**: 压缩时间 `s = tanh(αt)`, 把非自治系统扩展成 `s∈[-1,1]` 上的**自治**扩展系统(compactified extended system)。未来极限 `s=+1` 的盆边界上有余维一不变集 = **R-tipping edge state η**; η 在扩展系统里的**稳定流形** = **R-tipping 阈值**。这是相空间里一条**随强迫动态移动**的边界, 依据**未来**强迫剖面把 tipping 和 not-tipping 轨迹分开。
3. **几何预警指标 = 带符号距离**: 系统当前状态到 R-tipping 阈值的带符号距离, 称 **R-tipping indicator**。在阈值内(正距离)→ 不 tip; 在阈值外(负距离)→ tip。关键突破: 前作 [chapman2025quantifying] 只在**强迫结束后**算这个 EWS; 本文把它推广到**强迫仍在进行中**也能算。
4. **数值实现(附录 A)**: 强迫结束时 R-tipping 阈值 = ON/OFF 盆边界 = edge state 稳定流形。沿这条曲线均匀插值取点, **向后积分**(backward integration), 每隔小时间步重新插值保持点均匀分布(因为很多点会快速跑出感兴趣区域), 重复到仿真起点。判定"在阈值内/外": 从 tipped state 到当前系统位置连一条线, 与阈值交点数为奇 → 内部(正距离), 偶 → 外部(负距离)。

### 关键技术细节
- **测试系统**: 3 箱 AMOC 概念模型 (Wood 2019 五箱 → Alkhayuon 2019 三箱 → Chapman 2025)。二维状态: `S_N`(北大西洋盐度)、`S_T`(热带盐度); AMOC 强度 `q ∝ S_N`; `S_S,S_B` 设平衡消去, `S_IP` 由总盐守恒消去。参数用 doubled-CO₂ FAMOUS_B 标定(附录表 1)。
- **强迫 = 淡水 hosing `H(t)`**: 分段线性——ramp up `T_rise=100yr` → plateau `T_plat=300 或 400yr` → ramp down `T_fall=200yr` → 回到 `H_0=0`; `H_max=0.38`。
- **关键实验设计**: 选择的强迫**不穿过任何临界阈值**(no B-tipping), 但依 plateau 时长仍可 R-tip: `T_plat=300` 不 tip, `T_plat=400` tip。**把 forcing reversal 推迟 100 年就足以让 AMOC collapse** —— 纯速率效应。
- **噪声**: 随机版用 2×2 噪声矩阵 A (`A11=0.1263, A12=-0.0869, A21=0, A22=0.1008`), `σ=0.01`, 双独立 Wiener 过程; 噪声结构标定自 HadGEM3MM。
- **评测**: 每种强迫跑 100 条 tip + 100 条 not-tip 随机轨迹(ground truth 由**前向积分到最终命运**客观确定); 用 **ROC 曲线 + AUC** 时间序列评 4 个指标: (1) R-tipping 指标, (2) `S_N` 偏离临界盐度, (3) `S_T` 偏离临界盐度, (4) 回复率平方 `α²`(经典 CSD 基线)。统计量: sensitivity(TPR)、specificity(1-FPR)、false omission rate、informedness(=TPR+TNR-1); 对比 optimal threshold(左列) vs fixed threshold(右列)。
- **代码公开**: https://github.com/PaulRitchie89/R-tipping_threshold (可复现)。

### 核心结果
- **经典回复率(CSD)完全没用**: AUC 全程 ~0.5, 强迫结束后甚至**掉到 0.5 以下(比抛硬币还差)**; 无法分离 tip/not-tip 轨迹。这独立坐实了"快速强迫下 CSD 预警失效"。
- **R-tipping 指标 + `S_N` 盐度都早早获得 skill**: AUC → 1, 在淡水开始撤除**前约 100 年**就分离清楚。R-tipping 阈值比 `S_N` **略早、略好**。`S_T` 延后约 100 年才起 skill。
- **R-tipping 指标最大优势 = fixed-threshold 鲁棒性**: 用**固定阈值(距离=0, 天然选择)**时性能≈optimal threshold; 而 `S_N` 用固定阈值比 optimal threshold **差很多**。R-tipping 指标的 optimal threshold 随时间变化远小于盐度指标。实操中只能用固定阈值 → R-tipping 指标更实用。
- **AUC 会在淡水撤除中段冲高后回落**(tip 轨迹短暂盐度升高再回落), 是 skill 的非单调现象, 作者如实报告。
- 作者定性: R-tipping 指标可看作**状态变量的最优(非线性)组合**。

### 明确局限
- **必须知道: 系统动力学 `f`(governing ODE) + 当前相空间状态 + 未来强迫剖面**。这是相对经典 EWS 的核心代价——经典 EWS 是 model-free / 数据驱动, 本方法是 **white-box, model-based, forward-looking**。不需要过去历史/过去强迫。
- 未来强迫有不确定性 → tipping 不确定; 需要**概率化框架**(future work, 未做)。
- 高维复杂模型算 edge state / R-tipping 指标**计算上很难**; 作者建议用数据驱动模型发现 / 深度学习辅助。
- **只在 2 维概念箱模型上验证**, 无真实观测数据, 无高维模型。相对课题目标(真实 SCS 临近预报)证据基础很薄。
- 方法学上是**增量**: 直接扩展 chapman2025quantifying(唯一新意 = 强迫进行中也能算), 且在 AMOC 上 `S_N` 简单盐度阈值已几乎一样好。
