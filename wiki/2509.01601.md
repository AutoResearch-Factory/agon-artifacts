# Disorder-aided Early Warning Signals: Predicting Catastrophic Shifts in Athermal Systems (2509.01601)

- **Year**: 2025.09
- **Authors**: Tapas Bar (ICN2 巴塞罗那, 一作+通讯), Anurag Banerjee (Paris-Saclay / Ben-Gurion Univ), Blai Casals (Universitat de Barcelona), Gustau Catalan (ICN2), Javier Rodríguez-Viejo (ICN2 / UAB, 通讯)
- **Venue**: arXiv preprint（revtex4-2 / prx 格式，标注 "To be submitted"，尚未正式投稿/发表）
- **Keywords**: early warning signal, critical slowing down, tipping point, zero-temperature random field Ising model (ZTRFIM), avalanche, disorder-induced resilience, spinodal instability, MOKE microscopy
- **One-liner**: 数值模拟(ZTRFIM)+实验(钴薄膜 MOKE)证明无热雪崩式突变前也有临界慢化/方差增强/空间关联增长的预警信号，且无序度越高预警提前量越大。

## 精华提炼

### 管线/方法

数值模拟轨:
1. 建模：3D cubic lattice，边长 $L$（周期性边界），零温随机场 Ising 模型 (ZTRFIM)，Hamiltonian $\mathcal{H} = -J\sum_{i,j}s_is_j+\sum_i[H(t)+h_i]s_i$，$J=1$。
2. 无序：位点随机场 $h_i \sim \text{Gaussian}(0,\sigma^2)$（$\sigma$ 即无序强度），另外交叉验证 Beta、Bimodal（双峰高斯）、Uniform 分布。
3. 驱动：外场 $H(t)$ 从 $+\infty$ 步进扫到 $-\infty$（athermal-step-driven dynamics），场步长 $\delta H$ 取 0.005/0.001/0.0005 对比不同驱动速率。
4. 零温 spin-flip 规则：仅由局域场 $E_i=J\sum_j s_j+h_i+H$ 的符号翻转决定（无热涨落），每个场值都让雪崩传播到系统达到稳态（无更多位点翻转）为止。
5. 对每个外场值记录三个宏观量：(a) 弛豫时间常数 $\tau$（从上一稳态到本次稳态所需的雪崩步数）；(b) 空间自旋-自旋关联函数 $C(r)$ 及由其导出的关联长度 $\xi$；(c) 序参量（磁化强度）方差 $\text{Var}(\phi)$。
6. 对 100–200 个独立无序实现 (disorder realization) 取平均以降低噪声。
7. 另设一支 "phase fluctuation" 对照实验：quench 到某个 $H$ 后叠加均匀白噪声 $\eta(t)\sim\text{Uniform}[-\eta_0/2,\eta_0/2]$（$\eta_0=0.01$），观察噪声功率谱是否相关——用来论证零温体系中噪声谱方法（power spectral density / lag-1 autocorrelation）失效，因为相位涨落本身不相关。

实验轨（Cobalt 薄膜 + MOKE）：
1. 材料：100 nm 钴薄膜，热蒸镀在玻璃基底上。
2. 测量：Magneto-Optical Kerr Effect (MOKE) 显微镜，偏振光以约 45° 入射，经分析器后由 CCD 成像，反射光偏振变化对应局域磁化方向。
3. 外场：电磁铁线性扫场，速率 1.5 μT/s；为提高分辨率把整幅图像切成 4 个子区域分别统计雪崩，避免空间平均掩盖小尺度雪崩事件。
4. 统计：单样品重复扫场 40 个 cycle 得到 avalanche size distribution；另在一个更薄的 Co 样品上重复 35 次测量专门提取方差和空间关联曲线。
5. 空间关联提取算法：先做背景差分（减去首帧），再用 Wiener–Khinchin 定理对差分图像 $X$ 做傅里叶自相关 $C(X)=F^{-1}[F(X)\tilde F(X)]$，关联长度定义为自相关衰减到 $1/e$ 处的距离。
6. 角度依赖对照实验：外场方向偏离样品易磁化轴角度 $\theta-\theta_c$（$\theta_c\approx21^\circ$，由矫顽场-角度曲线拟合得到），用偏角大小来"唯象地"(phenomenologically) 代理无序强度——偏角越大，转变越平滑，效果类比无序增强。

### 关键技术细节

- Hamiltonian/参数设置与同一团队既往论文(Bar_prb21, Bar_prb23, Bar_prl18)一脉相承，本文是该研究计划在"预警信号"角度上的新延伸（此前工作侧重 spinodal instability 本身，本文侧重其预警/预测价值）。
- 数值正确性交叉检验：mean-field 3D-ZTRFIM 的滞后回线宽度在 $\sigma>4.75$ 消失，数值给出的阈值 $\sigma_{Th}=4.7873$ 与既往解析结果 (PerezReche_Prb04) 吻合；有限维（非 mean-field）计算中滞后消失点更高，因为局域无序涨落把 spinodal 的尖锐 signature "抹平" (smeared out)。
- 核心趋势：$\tau$ 和 $\xi$ 的峰值（强度）随 $\sigma$ 增大而降低，但峰宽（FWHM）随 $\sigma$ 增大而变宽——即预警信号"强度变弱、但提前量变长"，两者此消彼长，整体上无序增大对预警更有利。
- Disorder distribution 消融实验：Gaussian / Beta ($a=b=2$) / Bimodal ($\mu=2,\sigma=0.8$) / Uniform ($W=10$) 四种分布交叉验证。前三种（"smooth"分布）都能观察到渐进式 critical slowing down；Uniform（线性/box）分布反而在起始场处出现 kink 式突变（sudden jump）——即预警方法对无序分布形状敏感，这是作者明确指出并用图示证实的方法局限（而非回避）。
- 实验角度扫描与数值模拟的一致性：$\theta-\theta_c$ 越大，方差/关联长度峰越矮越宽，与数值模拟中 $\sigma$ 增大的效果定性一致，构成实验-数值的交叉验证（尽管无法定量对应）。
- 驱动速率消融：三种 field step (0.005/0.001/0.0005) 对比显示，avalanche size distribution 中的多峰 ("bump") 数量随驱动速率降低而减少，绝热极限 ($R=0$) 下只剩单个雪崩——说明多峰特征是有限驱动速率的产物而非普适现象；相比之下，$\tau$、方差、$\xi$ 这三个宏观指标在绝热极限下依然成立，因此作者将其作为比 avalanche size distribution 形状更稳健的预警量。

### 核心结果

- 三个经典 EWS 指标（critical slowing down 对应的弛豫时间 $\tau$、方差、空间关联长度 $\xi$）在**无热** (athermal, $T=0$)、由雪崩实现的突变（saddle-node bifurcation，无噪声驱动动力学）中依然成立——这是对经典 EWS 理论（通常假设有限噪声驱动下的临界慢化，如 lag-1 autocorrelation、noise spectroscopy）的非平凡扩展：本文体系中噪声本身不相关，传统 noise-spectroscopy 类方法失效，但宏观响应函数增强依然可用。
- 核心发现（"disorder 的保护作用"）：无序强度 $\sigma$ 增大 → $\tau$ 和 $\xi$ 的峰高降低但峰宽变宽 → 预警信号出现得更早、跨度更大，即无序越大，可供干预/缓解的预警窗口越宽。
- 实验证实：钴薄膜 MOKE 测量在矫顽场 $H_c=-2.44\ mT$ 附近观察到方差和空间关联长度的增长，定性复现数值模拟结果。
- 角度扫描（$\theta-\theta_c$ 代理无序度）实验显示同样的"峰变矮变宽"趋势，说明"等效无序增加预警窗口"这一结论不依赖于无序的具体来源（真实位点无序 vs. 几何/各向异性诱导的等效无序）。
- 给出一张"tipping point prediction possibility diagram" (Fig. 3c)，把无序轴分为三段：低无序区（无限雪崩/spanning avalanche，突发不可预测）、中间 critical-like 区（本文方法可预测，且 $\sigma$ 越大越好预测）、$\sigma>\sigma_{Th}$ 后两支滞后曲线汇合（转变不再是真正意义上的 tipping point）。
- 明确报告反例：Uniform/线性无序分布下，方差和 $\tau$ 表现为 sudden jump 而非渐进增长——预警方法在此失效，这一负结果被显式报告而非隐藏或淡化。

### 明确局限

- 方法仅对"smooth"无序分布（Gaussian/Beta/Bimodal）有效；线性/均匀分布下出现 kink 式突变，预警方法失效——作者自己指出并给出对照图（Fig. All_dist d1–d4）。
- 作者承认"abrupt disorder distribution"在自然界中并不常见，某种程度上是在为方法适用范围做辩护性限定，而非彻底解决该失效模式。
- ZTRFIM 模型未显式包含角度依赖场或各向异性；用角度偏移唯象代理无序强度是一个类比，并非严格对应，作者自己承认这是简化，且相关的详细建模留作未来工作。
- 实验上无法精确控制/表征跨样品的无序强度，角度实验只能做定性验证，不能定量对应数值模拟中的 $\sigma$ 值。
- 只研究了 saddle-node bifurcation / 雪崩型突变这一类 catastrophic shift，未涉及其他分岔类型（Hopf、pitchfork 等），也未系统研究从零温极限到有限温度（有热噪声）体系的过渡区间。
