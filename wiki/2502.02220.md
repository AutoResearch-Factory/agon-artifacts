# On the Existential Theory of the Reals Enriched with Integer Powers of a Computable Number (2502.02220)

- **Year**: 2025.02
- **Authors**: Jorge Gallego-Hernandez (IMDEA Software Institute and Universidad Politecnica de Madrid), Alessio Mansutti (IMDEA Software Institute)
- **Venue**: arXiv; extended version of a conference paper
- **Keywords**: existential theory of the reals, integer powers, computable numbers, root barriers, quantifier elimination, Schanuel-free decidability, transcendence measures, entropic risk games
- **One-liner**: 判定带固定可计算数整数幂谓词的存在实理论。

## 精华提炼

### 管线/方法
- 输入对象是结构 $(\mathbb{R};0,1,\xi,+,\cdot,\xi^\mathbb{Z},<,=)$ 的存在片段公式。
- 其中 $\xi>0$ 固定，可计算，谓词 $\xi^\mathbb{Z}(x)$ 表示 $x\in\{\xi^i:i\in\mathbb{Z}\}$。
- 论文先把任意存在公式视作量词前束后的 quantifier-free 公式 $\phi(x_1,\ldots,x_n)$。
- Step I: 对每个实变量 $x_i$ 引入 $u_i,v_i$，把 $x_i$ 写成 $u_i\cdot v_i$。
- 约束是 $u_i\in\xi^\mathbb{Z}$，且 $v_i=0$ 或 $1\le |v_i|<\xi$。
- 这样 $\xi^\mathbb{Z}(x_i)$ 可以被替换成 $v_i=1$。
- 随后用 Basu-Pollack-Roy 的 real quantifier elimination 消去所有 $v_i$。
- 输出是只在 $u_1,\ldots,u_n\in\xi^\mathbb{Z}$ 上求解的 quantifier-free 公式 $\psi$。
- Step II: 证明 $\exists\xi^\mathbb{Z}$ 有 small witness property。
- 也就是说，若 $\psi$ 可满足，则存在有限整数集合 $P_\psi$，只需尝试 $u_i=\xi^{g_i}$ 且 $g_i\in P_\psi$。
- 核心技术是 $\lambda(a)$ 函数：对 $a>0$ 返回不超过 $a$ 的最大整数幂 $\xi^j$。
- 论文证明如果 $p(\xi,\vec{x})>0$，则 $\lambda(p(\xi,\vec{x}))$ 必落在有限多个 $\xi^g\vec{x}^{\vec d_i}$ 形式附近。
- 对有 polynomial root barrier 的 $\xi$，这个有限集合可显式取 $G=[-L,L]$。
- 其中 $L=(2^{3c}D\lceil\ln H\rceil)^{6nk^{3n}}$，$D,H$ 来自系数多项式的 degree/height。
- 接着分析一元公式 $\phi(u,\vec y)$ 在实数上的解区间。
- 若区间含有某个整数幂，则存在靠近区间端点的整数幂作为 witness。
- 端点来自某个多项式 $q(u,\vec y)=0$，于是问题转成刻画 $\lambda$ 在这些根附近的可能值。
- Lemma LastLambdaLemma 给出候选形状 $x^j=\xi^g\vec y^{\vec\ell}$。
- Lemma RelativiseQuantifiers 把 $\exists u\,\phi$ 变成只枚举有限候选的等价公式。
- Lemma RemoveU 用模 $j$ 的余数分解消去 $u^j=\xi^k\vec y^{\vec\ell}$ 中的根式。
- 多次消去变量后，把所有替换反向传播回原始变量，得到每个 $u_i$ 的指数界。
- 对 root barrier $\sigma(d,h)=c(d+\lceil\ln h\rceil)^k$，最终可取
  $P_\psi=[-U,U]$。
- 这里 $U=(2^c\lceil\ln H\rceil)^{D^{2^5n^2}k^{D^{8n}}}$。
- Step III: 对候选赋值 $u_i=\xi^{g_i}$ 后，只剩若干 $p(\xi)\sim0$ 的符号判断。
- 若 $\xi$ 是可计算超越数，非零多项式不可能在 $\xi$ 上为零，可用近似 $T_n$ 搜到稳定符号。
- 若 $\xi$ 有 root barrier，则用 Algorithm sign-evaluation。
- 给定一元整数多项式 $p$，degree 为 $d$，height 为 $h$，算法取
  $n=1+2\sigma(d,h)+3d\lceil\log(h+4)\rceil$。
- 若 $|p(T_n)|\le 2^{-2\sigma(d,h)-1}$ 且 $|T_n|<h+2$，返回 $p(\xi)=0$。
- 否则返回 $p(T_n)$ 的符号；证明依赖 Cauchy root bound 和近似误差界。
- 若 $T$ 多项式时间计算 $\xi$ 且 $\sigma$ 是 polynomial root barrier，则符号判断多项式时间。
- 小基底也被处理：$\xi=1$ 直接退化到 ETR；$0<\xi<1$ 改写到 $1/\xi>1$。

### 关键技术细节
- 公式中的整数通常二进制编码，输入多项式指数默认一元编码。
- 论文对 $\exists\mathbb{R}(\xi^\mathbb{Z})$ 的核心算法是理论判定过程，不是实现系统。
- RealQE 使用 Basu-Pollack-Roy 1996 定理，复杂度随变量块和 degree/height 指数增长。
- Polynomial root barrier 定义：若 $p(\xi)\ne0$，则 $\ln|p(\xi)|\ge-\sigma(\deg p,\mathrm{height}(p))$。
- 代数数的 root barrier 来自 Bugeaud；其 exponent $k=1$。
- 对自然数基底，作者用 fewnomial/sparse polynomial 表示来避免巨大展开。
- Fewnomial sign 算法处理 $p(n)$，单项式 degree 和 coefficient 都二进制存储。
- 对经典超越数，root barrier 使用 Waldschmidt 的 transcendence measures。
- 覆盖的超越基底包括 $\pi,e^\pi,e^\eta,\alpha^\eta,\ln\alpha,\ln\alpha/\ln\beta$。
- $\alpha,\beta,\eta$ 为固定代数数，且需要相应表示。
- 对 $\alpha^\eta$，若 $\eta$ 有理则 $\alpha^\eta$ 仍代数；若无理则用 Waldschmidt 表。
- 有理性检查通过 LLL 分解代数数最小多项式的线性因子。
- 对 $\ln\alpha/\ln\beta$，有理性等价于 $\alpha,\beta$ 的 multiplicative dependence。
- 这一步引用 Masser 和 Cai-Lipton-Zalcstein 的格基构造结果。
- 多项式时间 Turing machine 的构造覆盖代数数、$\pi$、指数和对数。
- $e^r$ 近似用 Maclaurin series，截断长度满足 $M\ge L+8k^2$。
- $\ln r$ 近似用 $2\sum_j \frac{1}{2j+1}((r-1)/(r+1))^{2j+1}$。
- 论文没有训练数据、loss、RL、仿真器、真机平台或 benchmark；这是纯理论复杂性论文。

### 核心结果
- 若只给一台计算 $\xi$ 的 Turing machine，甚至无法判定 $\xi=2$，所以一般不可判定。
- 若知道 $\xi$ 是可计算超越数，则 $\exists\mathbb{R}(\xi^\mathbb{Z})$ 可判定，但一般无复杂度上界。
- 若 $\xi$ 有 polynomial root barrier 且多项式时间可计算，则一般算法给出双指数或三指数时间上界。
- 若 root barrier exponent $k=1$，general result 是 2EXPTIME。
- 若 $k>1$，general result 是 3EXP。
- 若 $\xi$ 是自然数，satisfiability 属于 NEXPTIME。
- 若 $\xi$ 是代数数，satisfiability 属于 EXPSPACE。
- 若 $\xi$ 属于列出的经典超越族，包括 $\pi$ 和 $e$ 相关数，satisfiability 属于 3EXP。
- 代数数本可由 $k=1$ 推出 2EXPTIME，但作者用二进制猜测指数和 ETR PSPACE 得到 EXPSPACE。
- 自然数基底进一步利用 fewnomial sign evaluation 得到 NEXPTIME。
- 作为应用，Baier et al. 2023 的 entropic risk threshold problem 可去掉 Schanuel conjecture。
- 对固定代数 $\alpha,\eta$，$\mathrm{ERisk}[e^{-\eta}]$ 与 $\mathrm{ERisk}[\alpha^{-\eta}]$ 属于 EXPTIME。
- 这点是本论文最直接的“去条件化”应用：从依赖 Schanuel 到无条件复杂度上界。

### 明确局限
- 复杂度上界很大：NEXPTIME/EXPSPACE/3EXP 主要是可判定性与上界结果，不是实用求解器。
- 没有给出公开代码、实验评测或 benchmark。
- 唯一下界基本继承自 ETR/PSPACE，精确复杂度仍远未确定。
- 对任意 computable $\xi$，必须知道其代数/超越身份；只给近似 oracle 不够。
- 对有复杂度上界的超越数，必须有可用的 polynomial root barrier 或 transcendence measure。
- 结果没有覆盖完整实指数域 $\mathbb{R}(e^x)$，只覆盖整数幂谓词这一离散片段。
- 多个整数幂谓词的存在片段仍有开放空间；全文 FO 多基底已知会因量词交替导致 undecidability。
- 应用到 stochastic games 只处理 entropic risk threshold 的符号/判定侧，不提供数值算法优化。
- 论文符号和证明链很长，复现需要逐条检查 lemma 依赖，而不是运行 artifact。
