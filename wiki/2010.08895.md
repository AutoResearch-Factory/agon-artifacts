# Fourier Neural Operator for Parametric Partial Differential Equations (2010.08895)

- **Year**: 2020.10
- **Authors**: Zongyi Li, Nikola Kovachki, Kamyar Azizzadenesheli (Caltech / Purdue), Burigede Liu, Kaushik Bhattacharya, Andrew Stuart, Anima Anandkumar (Caltech)
- **Venue**: ICLR 2021
- **Keywords**: neural operator, Fourier transform, PDE solver, super-resolution, integral kernel, function space, spectral method, Darcy flow, Navier-Stokes, turbulence
- **One-liner**: 提出 Fourier Neural Operator (FNO)，直接在 Fourier 空间参数化积分核，用 FFT 高效计算，实现分辨率不变性、零样本超分辨率和三个数量级的加速。

## 精华提炼

### 管线/方法
1. **输入**: 参数函数 \(a(x) \in \mathcal{A}(D; \mathbb{R}^{d_a})\)（如 PDE 初始条件、扩散系数等）
2. **Lifting (P)**: 浅层全连接网络将输入从 \(d_a\) 维提升到高维表示 \(d_v = 32\)(2D) 或 \(64\)(1D) 通道
3. **Fourier 层 ×4**: 每层包含两个并行的路径:
   - **(上分支) Fourier 路径**: FFT → 截断高频只保留 \(k_{\max}=12\)(2D) 或 \(16\)(1D) 个低频模式 → 复数权重矩阵 \(R \in \mathbb{C}^{k_{\max} \times d_v \times d_v}\) 逐模式线性变换 → 逆 FFT
   - **(下分支) 局部路径**: 线性变换 \(W: \mathbb{R}^{d_v} \to \mathbb{R}^{d_v}\)（bias term，负责非周期性边界条件）
   - 两分支求和后过 ReLU + BatchNorm
4. **Projection (Q)**: 浅层全连接网络将 \(d_v\) 通道投影回目标维度 \(d_u\)
5. **输出**: 解函数 \(u(x) \in \mathcal{U}(D; \mathbb{R}^{d_u})\)

### 关键技术细节
- **Fourier 截断策略**: 不按 \(\ell_1\)-norm 选低频，而是取每个维度 \(k_j \leq k_{\max,j}\) 或 \(s_j - k_j \leq k_{\max,j}\) 的 "corners" 区域，便于并行矩阵-向量乘法实现
- **\(R\) 参数化**: 实验了 constant（直接可学习）、linear（依赖 \(\mathcal{F}a\)）和 NN 三种。constant 和 linear 性能相近，NN 反而更差（因为频率空间离散）。论文主要用 constant 版本
- **超参数**:
  - 1D: \(k_{\max,j} = 16, d_v = 64\)
  - 2D: \(k_{\max,j} = 12, d_v = 32\)
  - 训练集 \(N = 1000\)（Navier-Stokes 高精度需要 10000），测试集 200
  - Adam, lr = 0.001, 每 100 epoch 减半, 共 500 epoch
  - 单卡 Nvidia V100 16GB
- **Burgers 数据生成**: \(u_0 \sim \mathcal{N}(0, 625(-\Delta + 25I)^{-2})\)，viscosity \(\nu = 0.1\)，split-step 方法在 8192 分辨率生成后降采样
- **Darcy Flow 数据生成**: \(a \sim \psi_\#\mathcal{N}(0, (-\Delta + 9I)^{-2})\)，其中 \(\psi\) 将正值映射到 12、负值到 3；\(f(x) = 1\)；421 × 421 二阶有限差分
- **Navier-Stokes 数据生成**: \(w_0 \sim \mathcal{N}(0, 7^{3/2}(-\Delta + 49I)^{-2.5})\)，pseudospectral + Crank-Nicolson，256 × 256 → 降采样 64 × 64，时间步长 \(10^{-4}\)（MCMC 时放宽到 \(2 \times 10^{-2}\)）
- **Navier-Stokes 学习任务**: 输入为前 \([0,10]\) 时间步的涡度，输出为 \((10, T]\) 时间步的涡度（\(T = 50, 30, 20\) 取决于粘度）
- **复杂度**: FFT 实现 \(O(n \log n)\)，内部乘法仅 \(O(k_{\max}) = O(12^d)\)，远小于 \(n\)

### 核心结果

| 方程 | 最佳 Benchmark 误差 | FNO 误差 | FNO 优势 |
|------|-------------------|---------|---------|
| Burgers (1D) | PCANN ~0.039 | **0.0139–0.0160** | ~60% 更低 |
| Darcy Flow (2D) | FCN 0.0253 (s=85) | **0.0098–0.0109** | ~60% 更低 |
| Navier-Stokes \(\nu=10^{-3}\) | U-Net 0.0245 | **0.0086** (FNO-3D) | ~65% 更低 |
| Navier-Stokes \(\nu=10^{-4}\) (N=1000) | U-Net 0.2051 | **0.1559** (FNO-2D) | ~24% 更低 |
| Navier-Stokes \(\nu=10^{-4}\) (N=10000) | TF-Net 0.1168 | **0.0820** (FNO-3D) | ~30% 更低 |

- **Surprising 结果**:
  1. FNO 首次用 ML 方法成功建模湍流 Navier-Stokes（\(\nu = 10^{-4}, 10^{-5}\)），之前的图神经算子 (GNO) 在湍流下不收敛
  2. 零样本超分辨率：64×64×20 训练 → 256×256×80 推理，没有任何高分辨率训练数据，**不仅空间超分，时间维度也可超分**
  3. FNO 仅用 \(k_{\max}=12\) 个傅里叶模式就实现了比 20 模频谱分解更低的误差（1% vs 2%），因为非线性激活函数和最终 decoder Q 恢复了高频
  4. FNO 推理速度 0.005s vs 传统 pseudospectral solver 2.2s（440 倍加速），在 MCMC 贝叶斯逆问题中 2.5 分钟 vs solver 18+ 小时
  5. 即使加上离线数据生成和训练 12 小时，FNO 仍比传统 solver 做一次 MCMC 更快
  6. FCN/U-Net 等 CNN 方法在提高分辨率时误差**增长**（因为 mesh-dependent），而 FNO 误差恒定

- **FNO-2D vs FNO-3D**: FNO-3D 在充足数据下更优（6.5M 参数），FNO-2D 在少数据下更鲁棒（414K 参数）

### 明确局限
- **FFT 要求均匀网格**: 虽然在数学上 FNO 不限于均匀网格（任意离散化均可学习，因为参数在 Fourier 空间），但 FFT 实现要求均匀网格且分辨率最好是 2 的幂
- **周期性边界假设**: Fourier 核内在地假设周期性。虽然 bias term W 能补偿非周期边界（Darcy Flow 和 Navier-Stokes 时间域的实验证明了这一点），但极端的非周期边界条件可能仍有困难
- **训练数据依赖**: 湍流 (\(\nu=10^{-4}\)) 需要 N=10000 训练样本，对于更复杂的 PDE，生成训练数据本身就很昂贵
- **没有与传统 solver (FEM/FDM) 直接对比精度**: 只对比了其他 ML 方法和经典降阶方法 (RBM)
- **全监督学习**: 需要 (a, u) 对，未利用 PDE 方程的物理知识（与 PINN 等无监督方法不同）
- **仅测试了规则域**: 所有实验在 unit box/torus 上，未测试不规则几何（传统 FEM 的优势领域）
- **没有开源代码**: 2020 年发表时未开源（后续有社区实现如 neuraloperator 库）
- **Fourier 层数少**: 仅 4 层，未探索更深架构及其饱和效应
- **图像分类实验被放弃**: 论文提到 Fourier 层在 MNIST/CIFAR10 上浅层还行但堆叠多层不兼容 pooling，性能不及 VGG/ResNet

### 跟我们的关系
对 **0616-fex** 课题：

**撞车评估**: 不撞车。这是 FNO 的创始论文（2020），属于奠基性 prior work 而非竞争者。0616-fex 的任何 neural operator / PDE 学习工作都应该引用这篇论文。

**可用工具/方法**:
1. **Fourier 层架构**: 直接可用的 building block——FFT → 低频截断 → 复数线性变换 → iFFT + 局部线性 skip connection。是 neural operator 的标准 backbone
2. **分辨率不变性**: 训练和推理可用不同分辨率，支持 low-res 训练 + high-res 推理
3. **零样本超分辨率**: FNO 是目前唯一能做的 ML PDE solver，可做人无我有的实验亮点
4. **贝叶斯逆问题范式**: 用 FNO 做 surrogate model 加速 MCMC，从 18 小时 → 2.5 分钟，0616-fex 如果涉及逆问题或设计优化可直接复用此范式
5. **频谱分析工具**: 对比 Fourier 截断误差 vs FNO 误差（FNO 用更少模式达到更低误差），这说明非线性激活恢复了高频——这种分析框架可用于分析其他 neural operator 架构
6. **FFT 加速**: quasi-linear \(O(n \log n)\) 的积分算子计算，替代传统 Nyström 采样的 \(O(n^2)\) 复杂度

**差异化方向**:
- FNO 假设核函数是卷积（平移不变），可探索非卷积 Fourier 核（如直接在 Fourier 空间的 attention）
- FNO 的 R 不依赖输入 a（constant 版本），可探索 Fourier 域的 cross-attention 或 conditioning
- FNO 在非规则几何上有理论可能但无实验验证，可与 GNO 融合做混合架构
- FNO 的截断策略固定（每个维度对称截断），自适应频率选择是 open direction

## Read by: 0616-fex (2026-06-24)

### 撞车评估
不撞车。FNO (2010.08895) 是 2020 年的奠基性工作，定义了 Fourier Neural Operator 架构。任何在 0616-fex 中做 neural operator / PDE 学习 / 函数空间映射的工作都应该将其作为 baseline 引用。

### 核心可复用组件
1. **Fourier 层**: FFT → 截断低频 → 逐模式复数线性变换 → iFFT，加上 spatial skip connection W。这是可直接复用的 building block，目前在 neuraloperator、Modulus 等库中已有实现
2. **分辨率不变性范式**: 训练和推理可用不同分辨率——在参数空间中学习 Fourier 模式数固定，物理空间的基函数处处良定义
3. **频谱分析框架**: 对比 "截断误差 vs 模型误差" 证明非线性激活恢复高频。此方法可推广到任何 spectral neural operator
4. **贝叶斯逆问题 surrogate 范式**: FNO 加速 MCMC 三个数量级 (0.005s vs 2.2s/forward)，适用于任何需要大量前向评估的场景

### 差异化空间
- FNO 核是卷积（平移不变），可换为非卷积 Fourier kernel（cross-attention in Fourier domain）
- FNO 截断策略固定且各向同性，自适应或可学习截断是 open problem
- 非规则几何上 FNO 理论上可行但无实验验证——FNO+GNO 混合可能填补此 gap
- FNO 未利用 PDE 物理信息（纯数据驱动），physically-informed Fourier operator 是自然延伸
