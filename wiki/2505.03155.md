# Rethinking the Global Convergence of Softmax Policy Gradient with Linear Function Approximation (2505.03155)

- **Year**: 2025.05
- **Authors**: Max Qiushi Lin (Simon Fraser University), Jincheng Mei (Google DeepMind), Matin Aghaei (Simon Fraser University), Michael Lu (Simon Fraser University), Bo Dai (Google DeepMind), Alekh Agarwal (Google Research), Dale Schuurmans (Google DeepMind / University of Alberta), Csaba Szepesvari (Google DeepMind / University of Alberta), Sharan Vaswani (Simon Fraser University)
- **Venue**: arXiv
- **Keywords**: softmax policy gradient, linear function approximation, stochastic bandits, global convergence, feature conditions, reward ordering, arbitrary learning rate, approximation error
- **One-liner**: 证明Lin-SPG收敛看特征排序而非逼近误差。

## 精华提炼

### 管线/方法
- 论文研究对象是 K 臂随机 bandit, 真实均值奖励向量为 \(r \in \mathbb{R}^K\)。
- 策略参数化为 log-linear softmax: \(\pi_\theta=\mathrm{softmax}(X\theta)\), 其中 \(X \in \mathbb{R}^{K \times d}\), 且 \(d<K\)。
- 每个 action \(a\) 有一个特征 \(x_a\), logit 为 \([X\theta](a)=\langle x_a,\theta\rangle\)。
- 目标是最大化期望奖励 \(\langle \pi_\theta,r\rangle\), 最优策略是把全部概率放到唯一最优臂 \(a^\star\)。
- 第一阶段先分析 exact setting, 即算法知道真实奖励 \(r\), 更新为确定性梯度上升。
- exact Lin-SPG 更新为 \(\theta_{t+1}=\theta_t+\eta X^\top(\mathrm{diag}(\pi_{\theta_t})-\pi_{\theta_t}\pi_{\theta_t}^\top)r\)。
- 第二阶段分析 stochastic setting, 每轮采样 \(a_t\sim\pi_{\theta_t}\), 观察 noisy reward \(R_t(a_t)\)。
- stochastic Lin-SPG 使用 on-policy importance sampling reward estimate: \(\hat r_t(a)=1\{a=a_t\}R_t(a_t)/\pi_{\theta_t}(a)\)。
- stochastic 更新为 \(\theta_{t+1}=\theta_t+\eta X^\top(\mathrm{diag}(\pi_{\theta_t})-\pi_{\theta_t}\pi_{\theta_t}^\top)\hat r_t\)。
- 作者先构造两个 4 臂 2 维特征例子, 说明 approximation error 与全局收敛没有决定关系。
- 第一个例子 \(r=(9,8,7,6)\), approximation error \(\sqrt{202.6}\approx14.2338\), 但 Lin-SPG 可以收敛到最优臂。
- 第二个例子只交换了特征矩阵 \(X^\top\) 的第 2 和第 3 列, approximation error \(\sqrt{205}\approx14.3178\), 但 Lin-SPG 收敛到次优臂 2。
- 这两个例子说明, 即使 approximation error 大于任何 suboptimality gap, 收敛仍可能成功。
- 也说明 approximation error 相近时, 收敛行为仍可能完全不同。
- 核心替代视角是 feature representation 是否保留 reward ordering。
- Reward Ordering Preservation 条件要求存在 \(w\), 使得 \(Xw\) 与真实奖励 \(r\) 的 action 排序完全一致。
- 对 3 臂情形, 还需要 feature condition: \(\langle x_2-x_3,x_1-x_3\rangle>0\)。
- 对一般 \(K\ge3\), 作者提出 general feature conditions, 约束任意 \(i,j,k\) 的内积 \(\langle x_i-x_j,x_{a^\star}-x_k\rangle\) 的符号。
- 条件的直觉是: 从任一次优 plateau 出发, 与最优臂相关的 logit ratio 必须有单调增长方向。
- exact setting 的证明先用 smoothness 给出单调改进, 再证明有限区域内不存在 stationary point。
- 关键协方差恒等式是 \(\langle x,(\mathrm{diag}(\pi)-\pi\pi^\top)y\rangle\) 可分解为所有 action pair 的差分乘积和。
- 在 reward ordering 条件下, 该协方差结构排除有限 stationary point。
- 于是参数范数趋向无穷, softmax 策略趋向某个 one-hot policy。
- 然后用特征内积条件排除收敛到任一次优 one-hot policy。
- stochastic setting 的证明把 logit 动态分成 progress term 和 noise term。
- progress term 由 deterministic gradient 的结构控制, 在 feature condition 下向最优臂累积正增量。
- noise term 用 martingale concentration 控制, 证明其量级为 \(\Theta(\sqrt{S_t\log(S_t/\delta)})\)。
- progress term 量级为 \(\Theta(\eta S_t)\), 因此当 \(S_t\to\infty\) 时 progress 支配 noise。
- 小学习率 stochastic 结果依赖 non-uniform smoothness 和 strong growth condition。
- strong growth condition 表示随机梯度方差随接近 stationary point 而下降, 因此不用 diminishing step size。
- arbitrary learning rate 结果改用不同证明, 不再依赖期望奖励单调上升。
- arbitrary learning rate 证明引入 \(N_t(a)\) 和 \(\mathcal{A}_\infty\), 即被无限采样的 action 集合。
- 先证至少两个 action 被无限采样, 且最优 action 必然在 \(\mathcal{A}_\infty\) 中。
- 对每个无限采样的次优 action, 如果期望奖励最终超过它的奖励, 则 \(\sup_t \pi_t(a^\star)/\pi_t(a)=\infty\)。
- 最后按 \(\mathcal{A}_\infty\) 中 action 的 reward order 做强归纳, 排除所有次优臂。
- 实验只用于 sanity check: exact setting 随机生成满足条件的环境, stochastic setting 在 5 个环境上跑不同 learning rates。
- 实验规模为 exact: 每个环境 50 次随机生成, \(10^6\) iterations; stochastic: 5 个环境, 每个 5 次, 共 25 runs, \(10^6\) iterations。

### 关键技术细节
- 基本假设 1: 所有 action 的 true mean reward 互不相同, 因此最优 action 唯一。
- 基本假设 2: 无损地按 \(r(1)>r(2)>\cdots>r(K)\) 排序。
- reward gap 定义为 \(\Delta=\min_{i,j}|r(i)-r(j)|>0\)。
- exact setting 的学习率要求为 \(0<\eta<4/(9R_{\max}\lambda_{\max}(X^\top X))\)。
- exact smoothness 常数为 \(L=9R_{\max}\lambda_{\max}(X^\top X)/2\)。
- exact theorem: 在 no-identical-arm, reward-ordering-preservation, general-feature-conditions 下, Lin-SPG 收敛到最优策略。
- 3 臂 theorem: 在 \(\langle x_2-x_3,x_1-x_3\rangle>0\) 加 ordering preservation 下, Lin-SPG 收敛。
- 3 臂 necessity: 如果缺少 feature condition, 可构造初始化 \(\theta_1=C(x_3-x_1)\) 让算法不收敛到最优。
- 另一个 necessity: 即使 feature condition 成立, 若 ordering preservation 不成立, 也可能没有任何参数能实现最优臂占优。
- stochastic 小学习率定义 \(\rho=8R_{\max}^3K^{3/2}/\Delta^2\)。
- stochastic 小学习率定义 condition number \(\kappa=\lambda_{\max}(X^\top X)/\lambda_{\min}(X^\top X)\)。
- stochastic 小学习率要求 \(\eta \le \min\{1/[6(\lambda_{\max}(X^\top X))^{3/2}\sqrt{2R_{\max}}], \lambda_{\min}(X^\top X)/[6\rho(\lambda_{\max}(X^\top X))^2]\}\)。
- stochastic monotonicity lemma 给出 \(\mathbb{E}_t[\langle\pi_{t+1},r\rangle]-\langle\pi_t,r\rangle \ge \|\nabla_z\langle\pi_t,r\rangle\|^2/(6\rho\kappa^2)\)。
- stochastic 小学习率 theorem: 在 no-identical-arm 和 general-feature-conditions 下, almost surely 收敛到最优策略。
- stochastic rate: \(\mathbb{E}[\langle\pi^\star,r\rangle-\langle\pi_{\theta_{T+1}},r\rangle] \le 6\rho\kappa^2/(\mu T)\)。
- 其中 \(\mu=[\mathbb{E}[\inf_t \pi_t(a^\star)^{-2}]]^{-1}\)。
- arbitrary constant learning rate theorem: 只要 \(\eta>0\) 为常数, 在同样 feature condition 下 almost surely 收敛。
- arbitrary learning rate 的 asymptotic average suboptimality 为 \(O(\log T/T)\)。
- 证明里使用 Extended Borel-Cantelli, Doob supermartingale convergence, martingale concentration, non-uniform PŁ 条件。
- 论文没有提出可直接部署的新 RL 算法, 主要是收敛判别条件和证明技术。
- 论文没有把 feature condition 转换为自动学习 feature 的方法。

### 核心结果
- 结论 1: approximation error 不是 Lin-SPG 全局收敛的可靠指标。
- 结论 2: 非零甚至很大的 approximation error 不阻止 Lin-SPG 收敛。
- 结论 3: approximation error 相近的两个问题, 可以一个全局收敛, 一个收敛到次优。
- 结论 4: 对 exact linear bandit, reward ordering preservation 加 general feature conditions 足以保证全局收敛。
- 结论 5: 对 3 臂 case, feature condition 的必要性通过反例展示得很清楚。
- 结论 6: 对 stochastic linear bandit, 同样 feature conditions 下可用常数学习率 almost-sure 收敛。
- 结论 7: 小学习率 stochastic Lin-SPG 可达 \(O(1/T)\) expected suboptimality rate。
- 结论 8: 任意常数学习率也 almost surely 收敛, 但平均 suboptimality 只给 asymptotic \(O(\log T/T)\)。
- 结论 9: tabular setting \(X=I_K\) 是 general feature conditions 的特例, 因而复现既有 tabular Softmax PG 结果。
- 结论 10: 该文对 Mei et al. ordering-based 条件做了修正和推广, 明确指出 non-domination condition 不足以从任意初始化保证全局最优。
- 比较反直觉的点是: approximation error 可大于任何策略 suboptimality gap, 但仍可全局收敛。
- 另一个反直觉点是: stochastic noise 反而通过持续采样帮助避免 action 概率衰减过快。

### 明确局限
- 作者明确承认分析只覆盖 stochastic bandits, 不是一般 MDP。
- 论文假设固定 feature matrix, 没有研究深度网络 feature 在训练中变化的情况。
- 主要结果依赖人工可检验但较强的 feature inner-product 条件。
- 这些条件在真实高维 RL 表征中未必容易满足或验证。
- 小学习率结果依赖未知的 problem-dependent quantities, 包括 reward gap 和 \(X^\top X\) 的谱量。
- arbitrary learning rate 结果解决了学习率问题, 但仍需要 feature conditions。
- 实验是合成 bandit 环境, 没有 MuJoCo, Atari, 真实机器人或标准深度 RL benchmark。
- 实验没有提供开源代码或独立复现实验痕迹, arXiv source 只包含图和 TeX。
- 论文的可靠性主要来自数学证明, 不来自外部实证覆盖。
- 对一般 function approximation 的 approximation error 观点做了反例, 但没有给出替代的可学习 representation design recipe。
- 对 0616-fex 若关注 feature extraction 或 representation learning, 它更像理论判据, 不是现成工程方法。

### 跟我们的关系
- 如果 0616-fex 的 claim 是「feature/representation 条件决定策略梯度收敛」, 这篇高度撞车。
- 具体撞车点是「approximation error 不是充分解释变量」和「reward/order-preserving feature 是关键」。
- 如果 0616-fex 想做 feature extraction for RL, 这篇可作为理论约束: 学到的 feature 不只要低 reconstruction error, 还要保留 reward ordering。
- 可用工具是它的 feature-condition checklist: 检查 \(Xw\) 是否保序, 检查 \(\langle x_i-x_j,x_{a^\star}-x_k\rangle\) 符号。
- 可用证明技术是 logit-ratio analysis, progress-noise decomposition, strong growth condition, stochastic action-count induction。
- 可借鉴的实验是构造「approx error 相近但收敛不同」的 synthetic feature pairs。
- 差异化方向 1: 从固定 feature 判别推进到学习 feature 的 regularizer 或 diagnostic。
- 差异化方向 2: 从 bandit 推进到 tabular MDP 或 entropy-regularized MDP。
- 差异化方向 3: 从 sufficient feature condition 推进到可计算、可优化的 surrogate metric。
- 差异化方向 4: 用实证深度 RL 或 GRPO/PPO style softmax policies 检验 ordering condition 是否解释训练现象。
- 该引: 必须引, 尤其是讨论 linear FA 下 Softmax PG global convergence 和 approximation error 局限时。
- 不该过度声称: 它没有证明一般神经网络策略梯度会全局收敛。

## Read by: 0616-fex (2026-06-17)

从 0616-fex 视角看, 这篇最重要的信号是: 只用 approximation/reconstruction error 评价 feature 可能会误判 Softmax PG 的可优化性。真正相关的是 feature 是否保留 reward ordering, 以及 action 差分内积是否让从次优 action 到最优 action 的 logit ratio 有正向累积 progress。

撞车评估: 如果我们的课题声称「linear/softmax policy gradient 的全局收敛由 feature ordering 决定」, 这篇已经非常接近, 需要改 framing。若我们的目标是「学习或诊断 feature extractor」, 它是强相关 prior, 但没有解决如何学习满足条件的 feature。

可用工具: 可以把 reward-ordering preservation 和 general feature conditions 做成 diagnostic, 在 synthetic bandits 或 small tabular MDP 上测试不同 representation。也可复用它的两个反例模式, 构造 approximation error 相同但优化行为不同的数据集。

差异化: 0616-fex 应避免再证明固定 \(X\) 的 bandit 收敛。更好的方向是把这些条件变成可优化 regularizer, 或研究 learned representation 在训练过程中是否逐步满足/破坏这些 ordering constraints。
