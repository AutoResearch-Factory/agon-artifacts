# Time-Series Forecasting, Knowledge Distillation, and Refinement within a Multimodal PDE Foundation Model (2409.11609)

- **Year**: 2024.09
- **Authors**: Derek Jollie (Montana State), Jingmin Sun (CMU), Zecheng Zhang (FSU), Hayden Schaeffer (UCLA)
- **Venue**: arXiv
- **Keywords**: PDE foundation model, multi-operator learning, symbolic encoding, SymPy, particle filter, knowledge distillation, time-series forecasting, operator learning
- **One-liner**: 用 SymPy 自动标准化 PDE 符号编码替代手工预处理，并引入粒子滤波精炼学习到的方程系数，提升多模态 PDE 基础模型的预测精度。

## 精华提炼

### 管线/方法

完整 pipeline：**输入时间序列 + PDE 符号表达式 → SymPy 自动标准化 → 符号编码器（token embedding）→ PROSE 多模态 Transformer → 双输出（时间序列预测 + 符号方程解码）→ SMC 粒子滤波精炼方程系数 → 最终预测**

具体步骤：
1. **输入**：时间序列数据 `{u(t_i,·)}` (16 个时间步，128 空间点) + 对应 PDE 的符号表达式
2. **SymPy 标准化**：equation → SymPy 解析 → 简化树 → 序列（Polish notation） → SymPy tokens → token 压缩（如 `u(x,t)` 的 6 个 token 压缩为 1 个）
3. **符号编码**：压缩后的 token 序列通过可学习的 token embedding 表映射到特征空间
4. **PROSE Transformer**：多模态架构，同时处理时间序列模态和符号模态，学习 operator 到 solution 的映射
5. **双解码器输出**：(a) 时间序列预测（forecasting）；(b) 符号方程解码（PDE term identification + coefficient estimation）
6. **SMC 粒子滤波**：对符号解码器输出的系数做后处理精炼——初始化 M=500 粒子，10 步迭代，输出精炼后的系数均值
7. **最终预测**：用精炼后的 PDE + 初始条件做数值求解，得到长时间序列预测

### 关键技术细节

- **基础架构**：PROSE（Predicting Operators and Symbolic Expressions），来自 Liu et al. 2023, Sun et al. 2024
- **PROSE 树编码**：PDE → 树结构（节点=操作符，叶=变量/系数）→ Polish notation 序列 → 可学习 token embedding
- **SymPy 树改进**：使用 SymPy 的 `sympy.simplify()` 做自动标准化，处理数学等价但表示不同的表达式（如 `x-1+1+y` vs `y+x`）
- **Token 压缩**：SymPy 原生 tokenization 产生大量冗余 token（如 `u(x,t)` 被拆成 6 个 token），通过规则合并简化为单个 token
- **粒子滤波参数**：M=500 粒子，10 步精炼，过程噪声方差=1e-5，观测噪声方差=0.05 × ||u(t=0)||_2
- **先验分布**：uniform centered at α_PROSE，范围 [0.9α, 1.1α]
- **更新规则**：α_k = α_{k-1} + ν，ν ~ N(0, ε²)
- **重要性权重**：w_i ∝ f_σ(σ)，基于观测噪声的似然
- **数据集**：6 族守恒律（Burgers'、inviscid Burgers'、cubic flux CL、inviscid cubic flux CL、sine flux CL、inviscid sine flux CL），参数从 ±10% 随机采样，153.6K 训练方程 + 30.72K 测试方程
- **训练配置**：AdamW，batch=512，30 epochs × 2K steps，lr=1e-4，weight decay=1e-4，cosine scheduler + 10% warmup，RTX 4090 24GB
- **训练时间**：PROSE tree ≈3.0h，SymPy tree ≈11.5h（慢 3.8×）
- **Loss**：标准 MSE（在归一化空间）

### 核心结果

| 实验 | 编码方式 | L² 误差 | R² | 符号误差 | 有效比例 |
|------|---------|--------|-----|---------|---------|
| 无噪声 | PROSE Tree*（手工） | 2.18% | 0.995 | 1.24% | 99.90% |
| 无噪声 | Swapping PROSE Tree | 3.26% | 0.983 | 1.43% | 85.94% |
| 无噪声 | **SymPy Tree** | **1.42%** | **0.996** | 1.40% | **99.95%** |
| 有噪声 | Noisy Swapping PROSE | 4.53% | 0.968 | 2.06% | 76.01% |
| 有噪声 | **Noisy SymPy Tree** | **3.81%** | **0.973** | 3.21% | **83.23%** |

粒子滤波精炼效果（符号误差 / 时间序列误差，Without Filtering → With Filtering）：
- Burgers': 1.02→0.88% / 1.50→1.38%
- Inviscid Burgers': 1.11→**0.65%** / 3.47→**2.12%**
- CL w. cubic flux: 3.07→2.79% / 3.94→3.62%
- ICL w. cubic flux: 2.31→1.73% / 3.61→2.97%
- ICL w. sine flux: **0.50→0.29%** / 3.94→**2.22%**

**Surprising findings**：
- 自动 SymPy 标准化不仅效率更高，预测精度也**超过**手工标准化的 PROSE tree（1.42% vs 2.18%），因为人工标准化可能引入不一致
- 粒子滤波对 inviscid 方程的改进尤其显著（符号误差降 41%），因为无粘性项的方程对系数更敏感
- Noisy SymPy tree 的符号误差（3.21%）比 Noisy PROSE（2.06%）高，但有效比例更高（83.23% vs 76.01%），说明 SymPy tree 更"诚实"地暴露了不确定性

### 明确局限

1. **仅 1D 守恒律**：只在 6 族一维标量守恒律上验证，未测试二维或三维 PDE、Navier-Stokes 等更复杂系统
2. **训练慢**：SymPy tree 训练时间是 PROSE tree 的 3.8 倍（11.5h vs 3.0h），因为 token 库更大
3. **粒子滤波后处理**：不与训练联合优化，是 post-hoc 步骤；理论上可做 end-to-end 训练
4. **仅精炼系数**：粒子滤波只精炼方程系数，不修正方程结构（term identification）；若 PROSE 识别错 term，滤波无法挽救
5. **简单噪声模型**：过程和观测噪声均为高斯，实际 PDE 系统的噪声可能更复杂（heteroscedastic、correlated）
6. **无真机/实验数据**：纯数值实验，没有物理实验或真实观测数据验证
7. **无标准 benchmark 对比**：没有与 FNO、DeepONet、PINN 等经典方法在标准 benchmark 上的对比
8. **只测了 forecasting**：未展示对 operator extrapolation（新 PDE 族的 zero-shot 预测）的效果
9. **粒子滤波参数未消融**：M=500、10 步的选择没有 sensitivity analysis
