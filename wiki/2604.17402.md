## 精华提炼

### 管线/方法
1. 论文研究对象是 GP-based symbolic regression: 输入是监督回归样本 `S={(x_i,y_i)}_{i=1}^m`，输出是一个表达式树函数 `f(x)`。
2. 表达式树的内部节点是算子，叶节点是输入变量、固定常数或可学习常数符号 `c`。
3. 每个树结构 `T` 有 `p(T)` 个可学习常数叶，按确定顺序组成参数向量 `theta in R^{p(T)}`。
4. 固定结构后，函数类是 `F_T(R)={f_{T,theta}: ||theta||_2 <= R}`。
5. 整体假设类是 `F_{s,D}(R)= union_{T in T_{s,D}} F_T(R)`，其中树大小不超过 `s`，深度不超过 `D`。
6. 分析先把 GP-SR 的复杂度拆成两部分: 选树结构的离散复杂度，以及固定结构后拟合常数的连续复杂度。
7. 第一步用 finite-union Rademacher bound 控制 `union_T F_T(R)`，得到额外项 `B sqrt(2 log |T_{s,D}| / m)`。
8. 第二步固定树结构，用参数诱导的经验伪度量 `d_S(theta,theta')` 分析可学习常数。
9. 系数敏感性假设给出 `d_S(theta,theta') <= G ||theta-theta'||_2`。
10. 于是欧氏球覆盖数转化为预测伪度量覆盖数: `N(eps,Theta_p(R),d_S) <= (1+2RG/eps)^p`。
11. 再用 Dudley entropy integral 得到固定结构的 Rademacher 复杂度 `O(RG sqrt(p/m))`。
12. 因为 `p(T) <= size(T) <= s`，固定结构项统一上界为 `O(RG sqrt(s/m))`。
13. 第三步数结构: unary/binary operator regime 下，树形数量由 bounded-depth rooted ordered trees 控制。
14. 使用 de Bruijn-Knuth-Rice planted plane tree 高度渐近式，得到深度相关指数基 `rho_D=4 cos^2(pi/(D+2)) < 4`。
15. 加上算子标签和叶标签，得到 `log |T_{s,D}| = O(log s + s log(rho_D (M1+M2)(N0+1)))`，忽略低阶项。
16. 最后把标准 bounded Lipschitz loss 的 Rademacher 泛化界、finite union、固定结构覆盖数、结构计数合并。
17. 论文不是提出新的 GP 搜索算法，而是给现有 GP-SR 启发式一个 learning-theoretic 解释。
18. 对实践的映射是: tree size/depth/parsimony 控制结构项，stable operators/interval arithmetic/constant constraints 控制 `B` 和 `G`。

### 关键技术细节
19. 数据设定: `x in X subset R^d`，`y in Y subset R`，样本 i.i.d. 来自分布 `D`。
20. 风险定义: population risk `L(f)=E[ell(f(x),y)]`，empirical risk `Lhat_S(f)=1/m sum_i ell(f(x_i),y_i)`。
21. 损失假设: `ell: R x Y -> [0,1]`，且对第一参数 1-Lipschitz。
22. 论文指出 MSE 需要通过归一化或 bounded-domain restrictions 才能纳入这个假设。
23. 算子词表按 arity 分块: `O = union_k O_k`，`M_k=|O_k|`。
24. 示例 unary 算子是 `sin, cos, exp`，binary 算子是 `+, -, *, /`。
25. 叶集合 `L_0={x_1,...,x_d} union C_fix`，固定常数示例包括 `1, pi`。
26. 特殊叶符号 `c` 表示可学习常数，一个树内多个 `c` 位置对应向量参数。
27. 树预算: `size(T)` 是节点数，`depth(T)` 是 root-to-leaf 最大边数。
28. 结构类: `T_{s,D}={T: size(T)<=s, depth(T)<=D}`。
29. 参数半径: `Theta_p(R)={theta in R^p: ||theta||_2<=R}`。
30. 预测有界假设: 存在 `B>0`，对所有预算内树、参数、输入，`|f_{T,theta}(x)|<=B`。
31. 系数 Lipschitz 敏感性假设: 存在 `G>0`，使 `|f_{T,theta}(x)-f_{T,theta'}(x)|<=G||theta-theta'||_2`。
32. 该 `G` 可以由参数梯度范数统一有界推出: `||nabla_theta f_{T,theta}(x)||_2 <= G`。
33. finite union lemma: 对 `M` 个 bounded 类，union 复杂度不超过最大单类复杂度加 `B sqrt(2 log M / m)`。
34. 固定结构覆盖数基于欧氏球覆盖数 `(1+2R/eta)^p`，将 `eta=eps/G` 代入。
35. Dudley integral 截断到直径 `2RG`，得到常数项 `C_par R G sqrt(p/m)`。
36. 结构计数只在 unary/binary operator regime 给出显式 `rho_D` 形式。
37. `rho_D` 随 `D` 增大单调趋近 4，因此深度越大，结构空间指数增长越接近无深度限制的树形增长。
38. 标签计数里内部节点最多有 `M1+M2` 种算子，叶节点最多有 `N0+1` 种终端。
39. 最终泛化界是 uniform over all `f in F_{s,D}(R)`，不是只对 GP 算法实际返回的模型成立。
40. 证明没有依赖 mutation、crossover、selection 的具体动态，因此它解释的是候选类容量，而不是进化搜索过程本身。

### 核心结果
41. 主定理: 以至少 `1-delta` 概率，对所有 `f in F_{s,D}(R)` 同时成立:
42. `L(f) <= Lhat_S(f) + C1 R G sqrt(s/m) + C2 B sqrt(log|T_{s,D}|/m) + C3 sqrt(log(1/delta)/m)`。
43. 第一项 `R G sqrt(s/m)` 是 constant-fitting term: 常数参数越多、半径越大、表达式对参数越敏感，泛化代价越大。
44. 第二项 `B sqrt(log|T_{s,D}|/m)` 是 structure-selection term: 可选树结构越多，模型选择代价越大。
45. 第三项是置信项，与常规统计学习界一致。
46. 深度限制进入 `log|T_{s,D}|`，不是直接进入常数拟合项。
47. parsimony pressure 在该框架下等价于偏向更小 `s`，降低结构选择项。
48. bloat 被解释为扩大有效假设类，从而可能提高过拟合风险。
49. interval arithmetic 的作用被解释为剔除 undefined/unbounded 表达式，主要控制 `B`，同时帮助控制敏感度 `G`。
50. analytic quotient / smooth operator definitions 被解释为限制敏感性放大。
51. constant optimization 如 nonlinear least squares、CMA-ES、linear scaling 不是无条件有利: 如果产生高 `G` 表达式，仍可能增加过拟合风险。
52. 论文给出的一个高敏感表达式例子是 `exp(exp(c1 x))`。
53. 较稳定例子是 `sin(c2 x)+c3 x`。
54. 反直觉点: 固定 tree size 时，depth limit 仍有价值，因为它减少 elongated tree shapes 的指数增长率。
55. 另一个重要点: 论文没有给 benchmark 数值结果，核心贡献完全是定理和解释框架。

### 明确局限
56. 作者明确承认这是 worst-case complexity analysis，界是 data-independent，因此量化上可能保守。
57. `B` 和 `G` 是强统一假设，真实 GP 搜索中很多表达式会违反或只有通过 clipping/protected operators 才能满足。
58. MSE 不是天然 `[0,1]` bounded 1-Lipschitz，需要归一化、截断或有界域处理。
59. bound 对所有预算内表达式 uniform 成立，因此没有利用 GP 实际搜索轨迹、selection bias 或 population dynamics。
60. 结构计数的显式形式限定在 unary/binary operator regime，更多 arity 的 GP function set 需要额外处理。
61. 论文没有实验验证 bound 的 tightness，也没有展示 `G` 或 `log|T|` surrogate 能否预测真实 test error。
62. 论文没有给出可直接运行的 generalization-aware GP 算法，只把它作为 future work。
63. 证明使用多处大 O 和 absolute constants，适合 qualitative explanation，不适合直接当模型选择准则。
64. 对 `p(T)<=s` 的处理较粗，会忽略不同结构中常数叶比例的差异。
65. 和 MDL/Bayesian SR 的关系是概念相似但推导路径不同: 本文从泛化界出发，MDL/Bayes 从编码长度或 evidence 出发。
66. 对 `0616-fex` 如果目标涉及实证 SR 系统，这篇不会提供算法性能证据，只能提供理论动机和正则化项设计线索。
