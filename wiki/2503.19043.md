# Enhancing Symbolic Regression with Quality-Diversity and Physics-Inspired Constraints (2503.19043)

- **Year**: 2025.03
- **Authors**: Jean-Philippe Bruneton, CNRS LIED / Université Paris Cité
- **Venue**: arXiv (preprint, formatted for JMLR)
- **Keywords**: Symbolic Regression, Genetic Programming, Quality-Diversity, MAP-Elites, Dimensional Analysis, Feynman-AI Benchmark, Exact Symbolic Recovery, Evolutionary Algorithms
- **One-liner**: QDSR combines GP with MAP-Elites quality-diversity grid and physics-informed dimensional analysis to achieve 91.6% exact symbolic recovery on the Feynman-AI benchmark, surpassing prior SOTA by 20+ percentage points.

## 精华提炼

### 管线/方法
QDSR 的完整 pipeline：
1. **输入**：dataset (x_i, y)，包含变量维度信息（mass, length, time 等），target dimension
2. **初始化**：
   - 随机生成 10,000 个维度一致（dimensionality-consistent）的树形表达式，去重后约 8,500 个
   - 构建空的 4D QD grid（维度：方程长度、自由标量数、变量出现次数、一元函数数）
   - bin size 前两个维度=1，后两个维度=2
3. **主循环**（最多到 1e6 个个体被评估或 6h timeout）：
   - 从训练数据中 subsample 250 个数据点（Feynman-AI 有 10^6 点，实际训练用 <10%）
   - 对种群中每个方程的 free scalar parameters 做 least-squares 拟合（SciPy least_squares）
   - 计算 R² 作为 fitness score
   - 若 R² >= 1-1e-14 → early stop，用 SymPy 做 exact equivalence check（含 physics-like rounding）
   - 提取每个方程的 4 个 phenotypic features
   - 将个体投影到 QD grid（同 cell 只保留 fitness 最高的）
   - Parent selection：从 flattened QD grid 中按 1/(rank+k) 概率选择，偏向高 fitness 个体
   - Genetic operations（各约占 1/3）：
     - Crossover：仅在同维度节点间进行
     - Mutation：保持维度一致性的节点替换
     - Unary removal：随机删除冗余一元函数节点
   - 新种群大小 = 当前 QD grid 大小（~1000）
4. **超时扩展**：若 1h 内未找到目标方程 → 提升 max length 35→45，增加第5维（是否有嵌套函数），grid 扩展到 ~4500
5. **输出**：QD grid 中所有 cell 的最佳方程，按 complexity-fitness 构成 Pareto front

### 关键技术细节
- **树表示**：每个节点携带符号和物理维度（如 [M, L]），维度通过树结构传播
- **词表**（Leaves 扩展是核心竞争力）：
  - Binary ops: +, -, *, /, ** (幂要求 f,g dimensionless 或指数为整数)
  - Unary: exp, log, sqrt, 全部三角函数和反三角函数，全部双曲函数和反双曲函数（共 14 个一元函数）
  - 基础 leaves: 变量 x_i、自由标量 A、整数（幂次）
  - **扩展 leaves**（本文核心技巧）：
    - 无维组合：所有 x_i/x_j 及其 reciprocal，可选它们之间的乘积
    - Norms：同维度所有变量的平方和 Σ_i x_i^2
    - Scalar products：变量 pair 的平方差组合（如 (x0-x1)^2+(x2-x3)^2），共 n!/(2^{n/2}(n/2)!) 种
  - 这引入了大量辅助变量（如 eq I.9.18 从 9 个变量扩展到 50+），但显著简化了目标方程的复杂度
- **维度引擎**：SymPy 实现，检测 equation 是否 dimensionally trivial（26/117 equations = 22% baseline），识别同维变量 group，计算无维组合
- **嵌套函数限制**：hard limit K=2（但允许通过 sin(arccos(x)) 等替代写法绕过）
- **自由标量优化**：SciPy least_squares（非 BFGS），subsample 250 点做拟合，效率和精度平衡
- **噪声处理**：在 noisy data 上演化 QD grid，但每代提取 ~50-70 个 Pareto front 方程在 noiseless data 上 refit 后才做 early stop 判断
- **Complexity score**（token-wise）：变量=1, 基本算数=1-3, 自由标量=2, exp/log/sqrt=3, trig=4, hyperbolic=5, inverse trig=5, inv hyperbolic=6
- **Random tree generator**：预计算维度可能 monomial dictionary，try 3 种策略（direct / numerator-denom split / prefactorize），resample hyperparameters 提高多样性
- **实验配置**：40 threads, PySR 用 15 populations×100 individuals×500 cycles, 30min timeout
- **运行次数**：Main run = 10 independent runs per target；Ablation = 5 runs

### 核心结果
| Method | Exact Recovery Rate (noiseless Feynman-AI, 117 eqs) |
|--------|-------------------------------------------------------|
| **QDSR Main run** | **91.6%** |
| QDSR Ab_0 (QD only, no DA, no extra vars) | 74.2% (already beats all prior SOTA!) |
| QDSR Ab_1 (QD+DA, no extra vars) | 85.1% |
| QDSR Main w/ 0.1 noise | 79.4% |
| uDSR (Landajuela 2022) | ~70% |
| PySR (Cranmer 2023) | 65.8% |
| PhySO (Tenachi 2023) | 58.5% |
| ParFam (Scholl 2024) | ~58% |
| AI-Feynman 2.0 (Udrescu 2020) | ~53% |

**Surprising 发现**：
- QD alone（Abl_0，无 DA 无扩展词表）= 74.2%，已超所有 prior SOTA。意味着仅加 QD grid 就能大幅提升性能，无需大改现有系统
- 扩展 vocabulary 引入大量辅助变量反而**降低**了方程复杂度，使更难的目标变得更容易——反直觉的 trade-off
- 26/117 方程 dimensionally trivial（22% baseline），QDSR/PySR 都 100% 能解
- PySR 能解但 QDSR 也能解的有 22 个简单方程
- 35 个 "Easy" target（通常 <10,000 个体评估，几分钟内解出）
- 20 个 "Medium" target (20k-250k 个体)
- 14 个 "Difficult" target，其中 5 个完全无法解（B10, B11, B20, I.29.16, III.9.52）
- **Noise 0.1 时仍达 79.4%**，说明方法有较强的鲁棒性
- PySR 在 easy targets 上仅 57.7% hit rate，medium 39%，difficult 0%

### 明确局限
- **作者自己说的**：
  - 未优化计算性能（"not optimized for high performance; left for future work"），PySR 评估的个体数远超 QDSR
  - 未在 SRSD 数据集上测试（"Due to computational limitations"）
  - 随机树生成器偏向产生较短的树
  - 14 个困难目标中有 5 个完全失败（B10, B11, B20, I.29.16, III.9.52）
  - False positive 问题：部分方程（尤其 difficult table 前 5 行）易提前收敛到高精度近似解（如高阶 Taylor series）
- **我读出的局限**：
  - **仅有一个作者**，无外部验证或独立复现
  - **无真机实验**，纯 benchmark 测评
  - 仅在 Feynman-AI 一个 benchmark 上评测，未在 SRBench 或 PMLB 等其他标准 benchmark 上测试
  - 依赖物理维度信息——不适用于无物理维度的通用 SR 问题
  - 扩展词表依赖人工设计的预计算（dimensionless combos, norms, scalar products），不是自动学习
  - 超参数（max length 35→45, timeout 1h→5h, bin sizes, K=2）缺乏系统调优说明
  - 计算成本高：40 threads, 6h timeout per target，117 target x 10 runs 的总成本极大
  - 无与其他 multi-objective GP 方法（如 NSGA-II）的直接比较
  - 开源代码？提及了 GitHub 链接但未见实际 repo 状态

## Read by: 0616-fex (2026-06-16)

QDSR 是一篇用进化算法+物理约束做符号回归的工程性论文，核心 insight 简洁有力：QD grid 维持种群多样性 + 维度分析约束搜索空间 + 预计算辅助变量降低方程复杂度。91.6% 的 Feynman-AI 精确恢复率是一个 impressive 的数字，但方法存在明显边界：依赖物理维度先验、仅在一个 benchmark 上评测、计算成本高、单一作者无外部验证。

差异化空间：QDSR 无法处理的 5 个困难方程、SRSD 数据集缺失、通用 benchmark 缺失、以及不依赖物理维度的纯数据驱动场景——这些都是 FEX 可以发力的方向。
