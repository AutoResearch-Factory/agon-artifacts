# DAS-PINNs for high-dimensional partial differential equations: extending deep adaptive sampling to spacetime domains (2606.06314)

- **Year**: 2026.06
- **Authors**: Anshima Singh, David J. Silvester, Department of Mathematics, University of Manchester
- **Venue**: arXiv
- **Keywords**: Physics-informed neural networks, DAS-PINNs, KRnet, normalising flows, adaptive collocation, high-dimensional PDEs, spatiotemporal sampling, residual distribution
- **One-liner**: 把 DAS-PINNs 扩展到时空统一采样以求解高维时变 PDE。

## 精华提炼

### 管线/方法
- 目标问题是一般半线性、抛物或双曲、二阶时变 PDE，定义在 $\Omega \times (0,T]$。
- PINN 解网络 $u_{\theta_p}(x,t)$ 直接在整个闭包时空域上近似解，而不是按时间步推进。
- 总损失由 PDE 残差、边界条件、初始条件三项组成：$\mathcal{J}_r + \gamma_{bd}\mathcal{J}_{bd} + \gamma_{ic}\mathcal{J}_{ic}$。
- 三类点集分别维护：内部残差点 $\mathcal{S}_r$、边界点 $\mathcal{S}_{bd}$、初始点 $\mathcal{S}_{ic}$。
- 初始阶段三类点都从各自域均匀采样。
- 解网络先在当前累计点集上训练，得到当前近似 $u_{\theta_p^*}$。
- 计算当前 PDE residual $\mathcal{R}(x,t;\theta_p)=\mathcal{L}u_{\theta_p}(x,t)-f(x,t)$。
- 把 residual square 视为目标采样密度，即采样概率正比于 $\mathcal{R}^2$。
- 使用 KRnet 作为 normalising flow，学习这个 residual-induced distribution。
- KRnet 建立 $f_{\theta_k}: \mathbb{R}^{d+1}\to\mathbb{R}^{d+1}$ 的可逆映射，并用 change-of-variables 得到显式密度。
- 时空输入统一为 $d+1$ 维变量，时间 $t$ 被缩放到 $[-1,1]$，与空间坐标同等处理。
- 空间域先假设可仿射映射到 $[-1,1]^d$。
- 为处理 bounded support，作者把 $[-1,1]^{d+1}$ 扩展到 $\mathcal{D}_n=(-(1+\delta),1+\delta)^{d+1}$。
- 实验中使用 $\delta=0.01$。
- 每个坐标用 logit 型映射 $\Phi(\xi_i)=\log\frac{\xi_i+(1+\delta)}{(1+\delta)-\xi_i}$ 映射到实线。
- residual 目标密度还乘以 cutoff function $h(\xi)$，使其在扩展域外为零，在 $[-1,1]^{d+1}$ 内为一。
- KRnet 训练目标是最小化 modified residual distribution 与 flow density 的 KL divergence。
- 实际优化用 importance sampling 近似 cross entropy。
- 每个 adaptive stage 后，KRnet 生成新的候选 collocation points。
- 候选点若落在扩展域但不在标准域内，先逐坐标 clipping 投影到 $[-1,1]^{d+1}$。
- 投影后按位置分类：归一化时间坐标为 -1 的点进入初始条件集。
- 若任一空间坐标在边界且时间不是初始面，则进入边界点集。
- 其余点进入内部 residual 点集。
- 如果投影生成的边界点或初始点不足，则用新的均匀采样补齐。
- 每一 stage 累加新点集，不丢弃旧点集。
- 整个方法的核心 claim 是：不显式 time marching、不建 moving mesh，也能由 residual density 自动追踪时空中的高误差区域。

### 关键技术细节
- 解网络在不同实验中使用 6 个 fully connected hidden layers。
- 激活函数为 $\tanh$。
- 低维 transient Gaussian peak 中，解网络每层 32 hidden neurons。
- rotating peak 中，解网络每层 64 hidden neurons。
- 高维 parabolic problem 中，解网络每层 64 hidden neurons。
- KRnet 在所有报告配置里使用 6 个 affine coupling layers。
- transient Gaussian peak 中，KRnet 每个 coupling layer 的子网络为 2 层、每层 24 neurons。
- 高维 parabolic problem 中，KRnet 每个 coupling layer 的子网络为 2 层、每层 64 neurons。
- 优化器为 Adam。
- 所有实验固定 learning rate 为 $10^{-4}$。
- 误差指标包括 relative $L_2$ error $\mathcal{E}_2$、$L_\infty$ error $\mathcal{E}_\infty$、MSE $\mathcal{E}_{mse}$。
- 低维 transient Gaussian peak：$\Omega=[-1,1]^2$，$T=1$。
- transient Gaussian exact solution 是随时间从 $(-0.5,0)$ 移到 $(0,0)$ 的 Gaussian，sharpness $\alpha=100$。
- transient Gaussian 使用 $n_{train}=2000$、batch size 500。
- transient Gaussian 每 stage 解网络和 flow 都训练 3000 epochs。
- transient Gaussian 有 1 个 initial stage 加 4 个 adaptive stages，总训练迭代数 180000。
- transient Gaussian 的 penalty 参数 $\gamma_{bd}=1.0$、$\gamma_{ic}=1.0$。
- transient Gaussian 验证网格为 $100\times100$ 空间点、5 个时间层，总 50000 点。
- rotating peak 是纯 transport problem，无扩散、无 source term。
- rotating peak 的 peak center 走 $(\cos t,\sin t)$，sharpness 同为 100。
- rotating peak 使用 1 个 initial stage 加 3 个 adaptive stages，总 120000 iterations。
- rotating peak 验证网格为 $200\times200$ 空间点、5 个时间层，总 200000 点。
- Burgers problem 是二维 viscous Burgers，扩散系数 0.001。
- Burgers exact solution 为 logistic front，前沿沿 $x_1+x_2=t$ 移动，指数系数 500。
- Burgers 使用和 rotating peak 相同的验证设置。
- 高维 parabolic problem 使用 exact solution $\exp(-10\|x\|^2)(1-e^{-5t})$。
- 高维 parabolic 分别测试 spatial dimension $d=6$ 和 $d=8$。
- 高维 parabolic 使用 $n_{train}=10000$、batch size 5000。
- 高维 parabolic 有 1 个 initial stage 加 4 个 adaptive stages，总 90000 iterations。
- 高维 parabolic 到最终 stage 累计 50000 collocation points。
- uniform baseline 也用 50000 points 且同样 total iteration budget。
- 高维 parabolic 的验证只在 near-origin grid $[-0.1,0.1]^d$ 上做。
- $d=6$ 验证点为 $5^6\times5=78125$。
- $d=8$ 验证点为 $3^8\times5=32805$。
- 高维 parabolic 用 hard constraint $u_\theta=t\cdot y_\theta$ 精确满足零初始条件。
- 高维 hyperbolic/advection problem 使用 $\Omega=(-0.2,1.2)^6$、$T=1$。
- advection exact solution 是沿主对角线 $(t,t,\ldots,t)$ 移动的 Gaussian。
- advection 使用 $n_{train}=5000$、batch size 1000。
- advection 验证网格为 $5^6\times5=78125$。

### 核心结果
- transient Gaussian peak 中，最终最小 $\mathcal{E}_{mse}=2.11\times10^{-5}$。
- transient Gaussian peak 中，最终最小 $\mathcal{E}_2=9.14\times10^{-2}$。
- transient Gaussian peak 中，最终最小 $\mathcal{E}_\infty=1.32\times10^{-2}$。
- transient Gaussian 的 temporal histogram 显示采样自然偏向 $t=0$ 附近。
- 这个早期时间偏置不是手工设定，而是由快速增长因子 $(1-e^{-5t})$ 造成的 residual 自动触发。
- rotating peak 中，initial stage 的 $\mathcal{E}_2$ 为 $1.71\times10^{-1}$。
- rotating peak 到 adaptive stage 3，$\mathcal{E}_2$ 降到 $8.65\times10^{-3}$。
- rotating peak 到 adaptive stage 3，$\mathcal{E}_\infty$ 降到 $6.59\times10^{-3}$。
- rotating peak 的新点沿 circular trajectory 聚集，说明 KRnet 能追踪移动高 residual 区。
- Burgers problem 中，initial stage 的 $\mathcal{E}_2=1.20\times10^{-1}$。
- Burgers 到 adaptive stage 4，$\mathcal{E}_2=1.70\times10^{-3}$。
- Burgers 到 adaptive stage 4，$\mathcal{E}_\infty=3.81\times10^{-2}$。
- Burgers 到 adaptive stage 4，$\mathcal{E}_{mse}=2.03\times10^{-6}$。
- Burgers 的误差集中在薄的 diagonal front 附近，这符合问题本身难点。
- 高维 parabolic $d=6$ 中，DAS-PINNs 的 MSE 为 $3.83\times10^{-4}$，uniform 为 $2.12\times10^{-2}$。
- 高维 parabolic $d=6$ 中，DAS-PINNs 的 $\mathcal{E}_2=3.22\times10^{-2}$，uniform 为 $2.40\times10^{-1}$。
- 高维 parabolic $d=8$ 中，DAS-PINNs 的 MSE 为 $1.81\times10^{-3}$，uniform 为 $2.23\times10^{-1}$。
- 高维 parabolic $d=8$ 中，DAS-PINNs 的 $\mathcal{E}_2=8.77\times10^{-2}$，uniform 为 $9.74\times10^{-1}$。
- 最反直觉也最重要的结果是：$d=8$ 时 uniform baseline 几乎预测为零，而 DAS-PINNs 仍能捕捉 near-origin Gaussian。
- 高维 advection initial stage 的 $\mathcal{E}_2=9.51\times10^{-1}$，几乎失败。
- 高维 advection 到 adaptive stage 4，$\mathcal{E}_2$ 降到 $4.57\times10^{-2}$。
- 高维 advection 到 adaptive stage 4，$\mathcal{E}_\infty=9.12\times10^{-2}$。
- 高维 advection 到 adaptive stage 4，MSE 为 $1.17\times10^{-3}$。
- advection loss curves 最终 residual、PDE、boundary、initial losses 都到 $10^{-6}$ 到 $10^{-7}$ 量级。
- 总体结论是：当解的支撑在高维时空域中很局部时，uniform collocation 的失效非常快，residual-driven flow sampling 的优势随维度增长而放大。

### 明确局限
- 论文主要是 arXiv preprint，没有同行评审信息。
- 论文没有报告公开代码、随机种子、训练日志或可复现实验脚本。
- 实验都是有解析解的构造 benchmark，没有复杂真实物理系统。
- 高维 parabolic 的验证集中在 near-origin grid，不能完全说明全域误差。
- uniform baseline 是主要比较对象，但缺少与 moving sample method 的逐项数值对比表。
- 与 SA-PINNs、RAR、RAR-G、residual-based adaptive refinement 等 PINN adaptive baselines 没有系统横向比较。
- 缺少 ablation：例如去掉 bounded-support mapping、改变 $\delta$、换 flow 架构、换 residual power。
- 没有说明多随机种子统计，表格只有单次最优误差。
- 报告的是每 stage 的 minimum error，不是最终 checkpoint error，可能乐观。
- 没有 wall-clock、GPU 型号、显存、采样和训练开销分解。
- KRnet 本身训练成本不低，但论文没有量化相比 uniform PINN 的总计算效率。
- 高维 advection 仍然是 6 个空间维度，不是更高维极限；高维 parabolic 到 8 个空间维度。
- 方法默认空间域可仿射映射到 box，对复杂几何的适配未展示。
- 对 shock、非光滑边界、随机系数、非局部或 fractional PDE 只是 future direction，没有实验。
- 论文没有展示失败案例，例如 residual density 多峰、边界奇异、initial layer 极强时的采样退化。

### 跟我们的关系
- 如果 `0616-fex` 的目标是 flow-based adaptive collocation for time-dependent/high-dimensional PDEs，这篇是直接撞车。
- 最核心的撞车 claim 是：把时间当作额外维度，用 KRnet 学 residual-induced distribution，并在时空统一域中采样。
- 第二个撞车 claim 是：不做 explicit time marching 或 moving mesh，也能追踪 moving/localised features。
- 第三个撞车 claim 是：在 6 到 8 个空间维度的局部 Gaussian/high-dimensional advection benchmark 中显著胜过 uniform PINN。
- 如果我们的课题只是在 PDE solver 中使用一般 adaptive sampling，这篇属于必须引用的强相关背景。
- 如果我们的课题也使用 normalising flow/KRnet 做采样，需要明确差异化，例如不同 flow、不同 verifier、不同 sampling objective、不同 benchmark 或真实物理任务。
- 可直接复用的工具是 residual-square-as-density 的表述、bounded-support logit mapping、cutoff function、projection/classification of candidate points。
- 可直接复用的实验 setting 是 moving Gaussian、rotating peak、2D Burgers sharp front、高维 parabolic near-origin Gaussian、6D advection moving Gaussian。
- 对 novelty-check，最近 6 个月 arXiv 检索中最接近的是 2601.18575 moving sample method；本文与它的差异是把动态和采样耦合在统一时空域，而不是用移动点策略逐步推进。

- 差异化建议：不要只复现 DAS-PINNs；应加入独立 verifier、uncertainty-aware residual、multi-fidelity sampling、复杂几何、非局部 PDE，或把 evaluation 从 analytic toy benchmark 扩展到真实 simulation trace。

## Read by: 0616-fex (2026-06-17)

从 `0616-fex` 视角看，这篇论文的风险点很清楚：如果我们的计划是“用 normalising flow 或 KRnet 根据 PDE residual 自适应生成时空 collocation points”，它已经覆盖了最自然的版本。它的 claim 分解为三层：第一，时间作为额外坐标进入 DAS-PINNs；第二，KRnet 学习 $\mathcal{R}^2$ 诱导的时空密度；第三，不做 time marching 也能在 moving Gaussian、rotating peak、Burgers front、6D/8D Gaussian 和 6D advection 上击败 uniform sampling。

它不是完全封死后续方向。论文的检验仍然偏自证：没有代码、没有多 seed、没有 wall-clock、没有真正独立 verifier，也缺少与 moving sample method 和其他 adaptive PINN baseline 的完整横向比较。对我们的差异化，更稳的方向是把“生成采样点”和“验证采样点价值”分离：例如用不同模型或数值残差 estimator 做 veto，报告每个新增 batch 的 marginal utility，或者在复杂几何、非局部 PDE、fractional PDE、真实仿真轨迹上做不依赖解析解的评估。该文必须引用，尤其是 2112.14038、2601.18575、2402.11283 和 2305.18702 这条相关链。
