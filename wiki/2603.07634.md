# Dissecting Spectral Granger Causality through Partial Information Decomposition (2603.07634)

- **Year**: 2026.03
- **Authors**: Luca Faes (University of Palermo / Univ. Novi Sad, corresponding), Gorana Mijatovic, Riccardo Pernice, Daniele Marinazzo, Sebastiano Stramaglia, Yuri Antonacci
- **Venue**: arXiv preprint (revtex4-1 journal-style manuscript, filename `PDGC_arxiv_revision.tex` — appears to be a revision round of a journal submission, journal not stated in source)
- **Keywords**: Granger causality, partial information decomposition, high-order interactions, redundancy, synergy, spectral/frequency-domain analysis, state-space models, network physiology, cardiovascular/cerebrovascular variability
- **One-liner**: Decomposes spectral multivariate Granger causality into unique/redundant/synergistic components via partial information decomposition (PID), applied to cardiovascular/cerebrovascular syncope data.

## 精华提炼

### 管线/方法

End-to-end pipeline (PDGC = Partial Decomposition of Granger Causality):

1. **Input**: an $M$-dimensional discrete-time random process $S_n = \{X_n, Y_n\}$, split into $N=M-1$ driver processes $X$ and one scalar target $Y$.
2. **Time-domain GC** (Eq. 1-2): standard Geweke-style GC $F_{X\to Y} = \ln(\sigma^2_{Y|Y}/\sigma^2_{Y|S})$, computable for any driver subset $X_{\alpha_j}$.
3. **VAR modeling** (Eq. 3-4): full and reduced VAR models fit via ordinary least squares; model order via BIC (range 3-12 for real data).
4. **Frequency-domain GC** (Eq. 5-9): Cholesky decomposition removes zero-lag (instantaneous) correlation between innovations, then Fourier transform of the VAR gives the transfer-function matrix $\tilde H(\omega)$ and the spectral GC $f_{X_{\alpha_j}\to Y}(\omega) = \ln[P_Y(\omega) / (\sigma_2^2 |\tilde H_{22}(\omega)|^2)]$. Spectral integration recovers the time-domain GC exactly.
5. **State-space (SS) reformulation** (Sect. II.C, Eq. 15-19): VAR is re-expressed as an innovations-form SS model (observation eq. + state transition eq. + Kalman gain $K$). This is done specifically because the reduced VAR model for a subset of drivers is theoretically infinite-order — SS models are closed under marginalization/subsetting, so GC can be computed for *any* driver subset without truncation-order bias. This SS-based spectral GC (Eq. 19) is the actual computational engine used everywhere downstream.
6. **PID decomposition of GC** (Sect. II.B, the core contribution): the full GC $F_{X\to Y}$ is expanded over the PID redundancy lattice $\mathcal{A}$ (antichain of driver-subset collections) as $F_{X\to Y} = \sum_{\alpha\in\mathcal{A}} F^\delta_{X_\alpha\to Y}$ (Eq. 10). Solving PID requires a redundancy function; the paper defines a **frequency-specific redundant GC**: $f^\cap_{X_\alpha\to Y}(\omega) := \min_{j=1,\ldots,J} f_{X_{\alpha_j}\to Y}(\omega)$ (the minimum-mutual-information/Williams-Beer principle, applied pointwise per frequency rather than in the time domain). Integrating this over frequency gives the time-domain redundant GC (Eq. 11-12).
7. **Möbius inversion** (Eq. 14): given the redundant GC over the whole lattice, standard Möbius inversion recovers the atomic (non-redundant) GC contribution of every lattice node, both in the spectral domain (Eq. 14) and, by whole-band integration, in the time domain (Eq. 10). Citing `jansma2025fast` for algorithmic efficiency of this step.
8. **Coarse-graining** (Eq. 16, following `rosas2020reconciling`'s aggregation rules): the (super-exponentially many, $|\mathcal{A}|=18$ for $N=3$) lattice atoms are summed into exactly $N+2$ interpretable quantities per frequency: one **unique** GC per driver $f^{\mathcal U}_{X_i\to Y}(\omega)$, one aggregate **redundant** GC $f^{\mathcal R}_{X\to Y}(\omega)$, one aggregate **synergistic** GC $f^{\mathcal S}_{X\to Y}(\omega)$, satisfying $f_{X\to Y}(\omega) = f^{\mathcal R} + f^{\mathcal S} + \sum_i f^{\mathcal U}_{X_i}$ (Eq. 16) and its time-domain analogue via band integration (Eq. 18) — can be restricted to a physiological frequency band $[\omega_1,\omega_2]$.
9. **Significance testing**: IAAFT surrogates (`schreiber1996improved`) per time series (preserve amplitude distribution + PSD, destroy cross-PSD), 100 surrogate sets, percentile test at 5% (empirical GC must exceed 95th percentile of the surrogate-derived null).
10. **Output**: for each subject/condition, a full spectral GC curve, an 18-atom (for $N=3$) spectral decomposition, and $N+2$ coarse-grained spectral/time-domain curves (whole-band, LF band [0.03-0.15 Hz], HF band [0.15-0.4 Hz]), each flagged significant/non-significant against surrogates, compared across REST vs. head-up-tilt (HUT) via paired Wilcoxon signed-rank test.

Toolbox: released as part of the open-source **HOP Matlab toolbox** (`www.lucafaes.net/hop.html`, `github.com/YuriAntonacci/HOP`).

### 关键技术细节

- **Model fitting**: ordinary least-squares VAR identification; order selection via BIC (`faes2012measuring`), searched in range 3-12 for the physiological application.
- **Simulation validation setup**: Gaussian VAR system, $M=4$ processes, 3 drivers $X_1,X_2,X_3$ oscillating at $\bar f_1=0.1$, $\bar f_2=0.3$, $\bar f_3=0.4$ Hz (sampling $\bar f_s=1$ Hz); ground-truth wiring: $X_2,X_3\to Y$ directly, $X_1\to Y$ only indirectly through mediator $X_2$. This lets them show the atomic GCs recover exactly the expected structure: unique atoms peak at $\bar f_2,\bar f_3$ (direct effects), the $\{1\}\{2\}$ redundant atom peaks at $\bar f_1$ (indirect cascade $X_1\to X_2\to Y$ shows up as redundancy, not as a spurious unique effect), and the $\{23\}$ synergistic atom peaks between $\bar f_2,\bar f_3$ (collider structure $X_2\to Y\leftarrow X_3$).
- **Finite-sample behavior** (important for anyone using GC as an instability/reliability signal): 100 realizations at lengths 100–2000 samples; estimation variance shrinks with $N_s$ as expected (5th–95th percentile band narrows), but redundant and unique components carry an **asymptotic bias** that does not vanish — consistent with prior finding (`antonacci2024measuring`) that parametric GC estimation accuracy depends critically on the ratio of data points to model parameters. Also tested robustness to VAR order misspecification (Fig. S2b) — described as "relatively robust."
- **Real data**: 13 SYNC (pre-syncope, tilt-induced) + 13 age/gender-matched nonSYNC controls; REST (10 min supine) then head-up-tilt (HUT, up to 40 min, 60°). Signals: ECG, finger arterial blood pressure (photoplethysmographic volume-clamp), cerebral blood velocity (transcranial Doppler, middle cerebral artery), respiration (piezoelectric belt), all at 1 kHz, beat-to-beat series (HP, SAP, MAP, MCBV, RESP) of 250 stationary points after trend removal (zero-phase AR low-pass, cutoff 0.0156 cycles/beat).
- **Two analysis settings**: CV interactions ($Y=$HP, $X=\{$RESP, SAP$\}$, i.e. $N=2$) and CB interactions ($Y=$MCBV, $X=\{$RESP, MAP, HP$\}$, i.e. $N=3$).
- **Relation to the PID redundancy literature**: this redundancy definition (pointwise minimum, spectral) is contrasted explicitly against the time-domain minimum-information redundancy from `sparacino2025decomposing` — spectral-then-integrated redundancy is *provably smaller* than time-domain minimum-information redundancy, because minimum-information redundancy computed directly in the time domain is not additive across frequencies; the authors argue this makes their version "less conservative" (mitigates known redundancy overestimation, citing `liardi2026mathematical`'s review of PID redundancy measures).
- **Relation to PIRD** (`faes2025partial`, direct predecessor by the same group): PIRD decomposes multivariate mutual information rate (symmetric, bidirectional); PDGC decomposes GC (directed, causal). PIRD reduces to PDGC exactly when all causal edges point toward the single chosen target (verified in a second $M=3$ simulation, Supplementary Sect. III / Fig. S3) — PDGC additionally captures HOIs from edges that originate at the target.
- **Relation to competing HOI-causal frameworks** (Discussion, explicit positioning): (a) **SURD** (`martinez2024decomposing`) — cross-prediction-based, scales better with $N$ (fewer atoms) but can infer spurious reverse-direction causality under self-dependencies and conflates some source contributions; (b) **MACE** (`jansma2026decomposing`) — also PID-lattice + Möbius inversion, but built on *interventional* (Pearl-style) causality on random variables, not observational time-series GC — harder to apply when only observational data exist.

### 核心结果

- **Simulation**: atomic GCs match theoretical/ground-truth structure exactly (peak locations at the correct frequencies for unique/redundant/synergistic atoms); estimation bias-variance tradeoff behaves as expected with sample size; robust to VAR order misspecification.
- **CV interactions (RESP, SAP → HP)**: nonSYNC controls show a marked, statistically significant REST→HUT increase in unique GC(SAP→HP) across all bands, redundant GC in full+LF bands, synergistic GC in LF band. SYNC (pre-syncope) patients show **no significant change in any GC component** REST→HUT — i.e., the coarse-grained decomposition reveals an blunted/absent baroreflex-linked response that a plain full-GC comparison would show as merely "less pronounced," but PDGC pins the deficit to specific unique+redundant components.
- **CB interactions (RESP, MAP, HP → MCBV)**: nonSYNC controls show no significant REST→HUT change in any component. SYNC patients show a significant **increase** during HUT in redundant GC (whole-band) and in unique GC(HP→MCBV), redundant GC, synergistic GC (LF band); unique GC(MAP→MCBV) near-significant ($p=0.057$). The paper argues the previously-reported bivariate GC(MAP→MCBV) increase in syncope patients (`pernice2022spectral`) is only partly a direct MAP effect — it's "determined mainly by the HOIs between MAP and HP," a claim only visible after PID decomposition.
- Both physiological findings are framed as refining/explaining prior bivariate/conditional-GC literature, not overturning it — explicitly "these contrasting responses could already be inferred from the multivariate GC computed at rest and during tilt... high-order causal analysis revealed them more clearly and linked them to specific information components and frequency bands."

### 明确局限

- **Sample size**: only 13 participants/group — paper explicitly flags this as limiting statistical power of the group-level comparisons, calling the physiological findings "preliminary."
- **No ground-truth on real data**: the physiological claims rely on plausibility/consistency with prior literature, not an independently verified causal graph (unlike the simulation, where ground truth is known by construction).
- **Redundancy-definition dependency, acknowledged as an open problem**: the entire PID output is contingent on the chosen redundancy function; the paper is explicit that "the choice of an appropriate redundancy function remains one of the main open issues in the PID literature," and different reasonable choices yield materially different decompositions. This is a foundational, not incidental, limitation.
- **Estimator bias**: redundant and unique component estimates carry asymptotic bias even as variance shrinks with sample size (Fig. S2a) — the paper does not fully characterize or correct this bias, just documents it.
- **Linear/Gaussian scope**: built on VAR/state-space models — implicitly linear-Gaussian; no discussion of nonlinear extensions or robustness to non-Gaussian innovations.
- **No non-stationarity / regime-change treatment**: all analysis assumes each REST or HUT segment is internally stationary (a single VAR model per condition/subject); the framework is not designed for, and is not tested on, transitions, tipping points, or time-varying causal structure within a segment — REST and HUT are treated as two separate stationary regimes compared post hoc, not as a continuous non-stationary trajectory.
- **No public code/data release confirmed for the specific analysis scripts** beyond the general HOP toolbox link (methods are toolbox-generic, not a reproducibility package tied to this exact paper's figures/data).

### 跟我们的关系

Topic: **causal-scs-indicator** — testing whether *instability of rolling causal-effect estimates* (dispersion + jump channels) is a pre-event diagnostic for severe convective storm (SCS) onset, using PCMCI+/LKIF/VAR-proxy causal estimators on paired synthetic saddle-node trajectories (current v3 pilot result: PCMCI+-conditioned instability underperforms a same-complexity non-causal marginal-correlation baseline; raw-state EWS still dominates both).

**Does not collide**: PDGC operates entirely within a *stationary-segment* paradigm — it fits one VAR/SS model per condition (REST or HUT) and compares decomposed GC *between* two separately-stationary regimes post hoc via group statistics. It never treats within-segment non-stationarity, rolling/windowed re-estimation, or estimator instability as a signal — the opposite of the causal-scs premise, where the instability of the estimate itself (not its point value) is the hypothesized diagnostic. No claim overlap at the "instability-as-signal" level.

**Reusable pieces**:
1. **The finite-sample bias/variance characterization of parametric (VAR-based) GC estimation** (Sect. III validation, and the cited `antonacci2024measuring`) is directly relevant background for the causal-scs project's "estimator noise" confound (a `README.md`-listed non-goal to conflate with true instability) — this paper is a citable, quantitative source for "how much does the *estimate itself* wobble with sample size alone, absent any real regime change," useful for calibrating what magnitude of dispersion/jump should be attributed to estimator noise vs. a genuine precursor.
2. **Surrogate/percentile null calibration methodology** (IAAFT surrogates → 100 sets → 95th-percentile significance test) is methodologically parallel to causal-scs's own "method-specific null percentiles, learned only from calibration controls" — not novel to either, but confirms this is a standard, defensible pattern for causal-signal significance in physiological/dynamical time series, and IAAFT specifically (destroys cross-PSD, preserves each series' own PSD/amplitude distribution) could be a stronger null-generation choice than the calibration-control approach currently used if a within-condition (not just paired-control) null is ever needed.
3. **State-space reformulation to avoid infinite-order-VAR bias under conditioning** (Sect. II.C) is a reusable numerical-implementation trick if causal-scs ever needs conditional (not just pairwise/full) GC estimates on short windows — directly addresses a bias source that would otherwise contaminate exactly the kind of small-window, high-frequency-of-refitting regime causal-scs operates in.
4. Not directly reusable but conceptually adjacent: the **unique/redundant/synergistic decomposition** itself could, in principle, be repurposed as a *finer-grained instability feature* (e.g., "does the redundant component specifically destabilize pre-event while the unique component stays flat?") — but this is a nontrivial extension beyond what's in the paper, not something to cite as prior art if attempted, since PDGC decomposes structure within a stationary fit, not instability across refits.

**Should cite**: yes, as supporting/background citation for GC estimator finite-sample behavior and as a state-of-the-art marker for what "sophisticated causal-network analysis in physiological time series" currently looks like — not as competing or overlapping work.

## Read by: 0710-causal-scs (2026-07-11)

不撞车。PDGC 在**单一平稳区段内**分解 Granger causality 的结构（谁贡献了独有/冗余/协同信息），比较的是两个分别拟合的平稳区段（REST vs. HUT）之间的组间差异；它从未把"因果估计本身的不稳定性"当作信号，这与 causal-scs-indicator 的核心前提（rolling causal-effect estimate 的 dispersion/jump 作为转变前兆）方向相反，没有 claim 级别的重叠。

可用之处：(1) 论文对参数化 VAR-based GC 有限样本偏差/方差特性的仿真刻画（含 `antonacci2024measuring` 引用）可作为背景引用，量化"仅因样本量导致的估计抖动"，帮助 causal-scs 项目把 estimator noise 这个已声明的 non-goal/混淆源与真实前兆信号分开；(2) IAAFT surrogate + 百分位显著性检验的方法论，与 causal-scs 目前"仅从校准对照学习的 method-specific null percentiles"方法路数一致，可作为该类方法在生理/动力学时间序列中站得住脚的旁证，也提示了一种可能更强的 null 生成方式（保留各序列自身 PSD、破坏互 PSD）；(3) 状态空间重构以避免条件化 GC 的无穷阶 VAR 截断偏差（Sect. II.C），若 causal-scs 未来需要在短窗口上做条件化（而非仅成对/全局）GC 估计，这是可直接复用的数值实现技巧。
