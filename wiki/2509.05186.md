# Probabilistic operator learning: generative modeling and uncertainty quantification for foundation models of differential equations (2509.05186)

- **Year**: 2025.09
- **Authors**: Benjamin J. Zhang (Brown), Siting Liu (UC Riverside), Stanley J. Osher (UCLA), Markos A. Katsoulakis (UMass Amherst)
- **Venue**: arXiv
- **Keywords**: probabilistic operator learning, generative modeling, uncertainty quantification, in-context learning, Bayesian inference, random differential equations, GenICON
- **One-liner**: 用随机微分方程(RDE)形式化给出 ICON 的概率解释——ICON 隐式执行贝叶斯推断近似后验预测均值，并提出 GenICON 从后验预测分布采样实现不确定性量化。

## 精华提炼

### 管线/方法

**整体框架**：随机微分方程 (RDE) → 概率测度建模 → ICON 训练数据生成 → 条件期望近似（后验预测均值）→ 生成式扩展 (GenICON)

1. **数据生成过程**：参数 $\alpha \sim \mathbb{P}_\alpha$，条件 $y \sim \mathbb{P}_y$，QoI $z \sim \mathbb{P}_{z|y,\alpha}$。对每个参数 $\alpha_m$ 生成 $J$ 组 condition-QoI pairs
2. **ICON 映射**：$\mathcal{T}_\theta: Y \times (Y\times Z)^{J-1} \to Z$，输入 example pairs + 新 condition，输出预测 QoI
3. **训练目标**：最小化 MSE $\min_\theta \frac{1}{M} \sum_{m=1}^M \|z_m^J - \mathcal{T}_\theta(y_m^J; \{y_m^j, z_m^j\}_{j=1}^{J-1})\|_Z^2$
4. **理论保证**：Theorem 1 证明最优解是条件期望 $\mathbb{E}[z^J | y^J, \{y^j,z^j\}_{j=1}^{J-1}]$，即后验预测均值
5. **GenICON 训练**：GAN 框架，最小化 $(f,\Gamma)$-divergence（Lipschitz-regularized KL divergence），generator $\mathcal{G}_\theta(\eta, \mathbf{c})$ 从噪声 $\eta$ + context $\mathbf{c}$ 生成样本

### 关键技术细节

**模型架构（GenICON）**：
- Generator 和 Discriminator 均为 decoder-only Transformer
- 层数：6，Heads：8，嵌入维度：128，FFN hidden：512
- 每模型参数量：~3.9M
- Generator 额外在 token 序列前追加 latent noise vector $z \in \mathbb{R}^{8\times 32}$
- Discriminator 输入 context $\mathbf{c}$ + predicted $z^J$，输出标量分数

**训练配置**：
- 优化器：AdamW（Generator $\beta_1=0.5$，Discriminator $\beta_1=0.0$，$\beta_2=0.9$）
- Peak LR：$2\times 10^{-5}$，Warmup + Cosine Decay（warmup 1%）
- Gradient norm clipping：10.0
- Disc:Gen 训练比：5:1
- 总训练步数：100,000，Batch size：64
- Gradient penalty weight $\lambda = 0.1$

**$(f,\Gamma)$-divergence**：选择 $f(x) = x\log x$（前向 KL），$\Gamma$ 为 $L$-Lipschitz 算子空间。通过 one-sided penalty 实现 Lipschitz 约束：$\max(0, \|\nabla_{z^J} \mathcal{D}\|^2/L^2 - 1)$。

**实验设置**：
1. ODE 正/逆问题：线性 ODE $u' = \gamma_1 c(t) u + \gamma_2$，2-parameter（可识别）vs 3-parameter（不可识别，$\eta_1\eta_3 = \gamma_1$）
2. 反应扩散方程 BVP：含噪声 + 部分观测，参数场 $k(x,\omega)$ 为 softmax 变换的高斯过程，噪声 $\sigma=0.1$，观测点 $M \in \{40,50\}$（全集 100 点）
3. Poisson 方程 + 标量噪声：$\sigma \in \{0.1, 0.05, 0.025\}$，100 source terms × 10 BC pairs = 1000 problems
4. Poisson 方程 + 随机右边界：$u_1|u_r \sim \mathcal{U}[u_r-0.1, u_r+0.1]$

### 核心结果

| 实验 | 关键数字 | 含义 |
|------|---------|------|
| ODE 正问题 | Forward error < $2\times10^{-3}$ (all train$\to$test) | ICON 对不可识别参数鲁棒 |
| ODE 逆问题 | Inverse error < $1.55\times10^{-3}$ | 正反问题同框架处理 |
| 反应扩散 + 噪声 | Relative error < 2%，随 demo 数增加递减 | ICON 从噪声和 partial obs 中学习去噪解 |
| ODE 1-demo | 仅 3 个点的 demo 即可准确预测 | Bayesian 视角正确预测 $J=1$ 足够 |
| GenICON Poisson + noise | $\hat{\sigma} = 0.119, 0.0617, 0.029$ vs true $\sigma = 0.1, 0.05, 0.025$ | 方差估计略偏高（forward KL 的 moment-capturing 特性） |
| GenICON random BC | 左边界误差低方差小，右边界保持高方差 | 正确捕获 ill-posed inverse problem 的 output 分布 |
| Physical consistency error | 随 demo 数增加单调下降 | 更多 context 改善物理一致性 |
| QoI prediction error | 饱和不收敛到 0 | irreducible output variability（一对多映射） |

**反直觉发现**：
- 前向 KL 训练导致方差高估而非低估——与通常认知"GAN mode collapse"相反，这里 forward KL 是 moment-capturing 而非 mode-capturing
- ICON 仅需 $J=1$（单个 example pair）即可准确预测——Bayesian posterior predictive 在确定性似然下是 Dirac delta

### 明确局限

**作者自述**：
1. 假设 GenICON 精确刻画后验预测，但实际存在近似误差和有限样本误差——量化这些误差是未来工作
2. GenICON 比 ICON 需要显著更多的数据和计算资源，sample complexity 需要研究
3. 后验预测均值在 PPD 多模态时可能不是最有信息量的统计量（如多个微分方程可解释同一 context）
4. 仅考虑了 1D 简单 ODE/PDE 例子，未扩展到高维和真实应用
5. 未探索 structure-preserving 架构（FNO、Galerkin-based、polynomial chaos-based）

**审读者发现的额外局限**：
1. 无真机实验，全在 toy numerical examples 上验证（1D ODE + 1D PDE）
2. 没有代码开源声明（虽然有详细架构参数）
3. 未与 baseline UQ 方法（如 ensemble、MC dropout、Gaussian Process）对比
4. 理论仅保证存在性（Theorem 2），未给出收敛率或 sample complexity bound
5. 未讨论 GenICON 在真实 high-dimensional PDE（如 Navier-Stokes）上的 scaling 行为
6. 对 variable $J$（context size 可变）仅一笔带过说留给 future work
7. 未提供 calibration 评估（如 PIT histogram、coverage probability）
