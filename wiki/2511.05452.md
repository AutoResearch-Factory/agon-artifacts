# Self-adaptive weighting and sampling for physics-informed neural networks (2511.05452)

- **Year**: 2025.11
- **Authors**: Wenqian Chen, Amanda A. Howard, Panos Stinis; Pacific Northwest National Laboratory
- **Venue**: arXiv
- **Keywords**: PINNs, adaptive sampling, adaptive weighting, BRDR, residual-based sampling, inverse distance weighting, PDE solvers, mFCN
- **One-liner**: 把 BRDR 点权重与残差自适应采样合并用于 PINNs。

## 精华提炼

### 管线/方法
- 论文目标是改进 PINN 在复杂 PDE、尖锐梯度、边界层和 shock-like 区域上的训练稳定性与采样效率。
- 输入是一组 PDE/BC/IC 约束、初始 residual collocation points、候选采样点，以及 PINN 网络参数。
- 基础 PINN loss 由 PDE residual loss 和 boundary/initial condition loss 组成。
- 方法第一条线是作者前作的 point-wise adaptive weighting：BRDR, balanced residual decay rate。
- BRDR 给每个 PDE residual 点和 BC/IC 点分配正权重，而不是只给 loss component 一个全局系数。
- 加权 loss 形式为 residual/BC squared error 的加权平均，再乘一个全局 scale factor `s`。
- 权重约束是所有点权重的 mean 等于 1，避免权重整体漂移。
- BRDR 的核心观测是：全局收敛速度往往被 residual decay 最慢的点拖住。
- 每一步先计算当前训练点 residual `R_t(x)`。
- 再维护 residual 四次方的指数滑动平均 `overline{R^4}`。
- IRDR 定义为 `R^2(t) / sqrt(overline{R^4}(t) + epsilon)`。
- IRDR 越大，表示该点 residual decay 越慢，下一步应该获得更大权重。
- reference weight 是当前点 IRDR 除以全体 IRDR 均值。
- 实际权重用 EMA 更新：`w_t = beta_w w_{t-1} + (1-beta_w) w_ref`。
- 方法第二条线是 residual-based self-adaptive sampling。
- 作者先在 candidate points 上计算 residual。
- 新训练点不是取最大 residual 点，而是按 clipped residual squared 形成概率分布后随机采样。
- 基础采样概率是 `p(x) = R^2(x) / sum_x R^2(x)`。
- 为了避免极端 residual 点支配采样，先做 residual clipping。
- clipping 公式把 `R^2(x)` 限制在 `[median(R^2), gamma * median(R^2)]` 内。
- 默认 `gamma = 100`。
- 采样不是每一步都替换，而是每隔 `N_s` optimization steps 才更新。
- 默认 `N_s = 100`。
- 每次只替换一部分旧点，默认更新比例 `p_u = 0.2`。
- 旧点中被替换的点是随机选取，新点来自 residual-clipped proposal distribution。
- 组合 adaptive weighting 和 sampling 的难点是：新点没有历史权重，也没有历史 `overline{R^4}`。
- 作者用 inverse distance weighting (IDW) 从旧点插值新点的权重和 `overline{R^4}`。
- IDW 公式用距离倒数加权，将旧点的 `w_i` 与 `overline{R^4}_i` 传播到新采样点。
- 完整训练循环是：计算 residual -> 更新 BRDR 统计量和权重 -> 到达 resampling step 时生成候选点 -> residual clipping -> 采样新点 -> IDW 初始化新点统计量 -> 组装加权 loss -> Adam 更新网络。
- 输出是训练好的 PINN solution，以及自适应后的 residual point distribution 和 point-wise weight distribution。

### 关键技术细节
- 网络结构使用 modified fully-connected network (mFCN)。
- 大多数实验使用 6 个 hidden layers，每层 128 neurons。
- activation function 是 `tanh`。
- Allen-Cahn 的输入先做 Fourier feature transform。
- Allen-Cahn 使用 10 Fourier modes，把 `(x,t)` 映射成 21 维输入特征。
- 其他标量 PDE 常用 `[2]+[128]x6+[1]` 的 mFCN。
- lid-driven cavity 输出是三维 `[u,v,p]`，网络为 `[2]+[128]x6+[3]`。
- 参数初始化使用 Kaiming Uniform。
- 实现框架是 PyTorch。
- 计算使用 32-bit single precision。
- 硬件是 NVIDIA Tesla P100 GPU cluster。
- BRDR 默认 `beta_c = 0.999`，用于 `overline{R^4}` 的 EMA。
- BRDR 默认 `beta_w = 0.999`，用于权重 EMA。
- adaptive sampling 默认 `p_u = 0.2`、`gamma = 100`、`N_s = 100`。
- 评价指标是 test set 上的 `L2 relative error = ||u-u_E||_2 / ||u_E||_2`。
- 训练点生成方式因问题而异：Allen-Cahn 和 Burgers PDE 点用 Latin Hypercube，perturbation 用 uniform，lid-driven 用 random。
- Adam steps：Allen-Cahn `3e5`，perturbation `1e5`，Burgers `4e4`，lid-driven `8e4`。
- Adam learning rate 分别为 `0.001*0.99^(t/750)`、`0.005*0.99^(t/250)`、`0.001*0.99^(t/100)`、`0.001*0.99^(t/400)`。
- uncertainty 展示用 5 independent runs 的 mean ± 2 standard deviations。
- 分布可视化使用 SciPy `gaussian_kde`，bandwidth 默认 Scott's rule，并把 adaptive weights 作为 kernel weights。

### 核心结果
- 仅 adaptive sampling 或仅 adaptive weighting 都不是稳定万能解。
- 两者作用不同：sampling 找大 residual / 大梯度区域，weighting 找 residual decay 慢的训练项。
- 组合方法在四个 benchmark 上都给出最佳或最稳定的 prediction accuracy。
- computational cost 方面，在相同 batch size 和最大迭代数下，组合方法相对 plain PINN 的训练时间增加小于 20%。
- 但达到同一 target error 时，组合方法所需训练时间显著少于其他策略。
- perturbation 问题中，`epsilon = 1e-4`，真实解有两个厚度约为 `epsilon` 的薄边界层。
- plain/adaptive-weight-only 在少量点时会出现 residual overfitting：128 uniform training points 上 residual 可低于 `1e-10`，但 unseen points residual 仍大，解偏离真值。
- perturbation 上 adaptive weighting alone 帮助有限，adaptive sampling 明显改善。
- perturbation 上组合方法用 2048 residual points 达到 `L2` 相对误差 `0.01%`。
- 同一 perturbation setting 下，文中引用 Fang et al. 的 ensemble PINN 误差为 `0.43%`，standard PINN 约 `12.56%`，且二者 batch size 为 10000。
- perturbation 上组合方法的 pointwise absolute error 在 thin boundary layers 内低于 `1e-3`。
- Allen-Cahn 方程中，viscosity/diffusion 参数 `D = 1e-4`。
- Allen-Cahn 上 adaptive sampling alone 随 batch size 增加不明显；adaptive weighting 随 batch size 增加误差下降更清楚。
- Allen-Cahn 上组合方法同时关注 initial condition 与 large-gradient regions。
- Burgers 方程中，viscosity `v = 0.01/pi`。
- Burgers 上 sampling alone 会稳定聚焦 shock region，但 batch size 增加带来的收益有限。
- Burgers 上 weighting alone 的关注区域会随 batch size 从 initial condition 逐渐转向 shock region。
- Burgers 上组合方法继承 weighting 对 initial condition 的强调，同时保留 sampling 对 shock region 的捕获。
- lid-driven cavity 使用不可压 Navier-Stokes，顶盖速度为 `u_w(x)=16x^2(1-x)^2`，用于避免上角奇异速度。
- lid-driven 上 sampling alone 聚焦 top-right corner，weighting 强调 two bottom corners，组合方法覆盖所有关键区域。
- 一个反直觉发现是：在若干问题中，单独 adaptive sampling 即使加大 batch size 也未必继续降低误差。
- 另一个关键发现是：weights 最终可以近似均匀，但 sampling distribution 已经把点移动到难区域，二者并不必然表现为同一种空间分布。

### 明确局限
- 代码和数据在 tex 中写的是 "will be made available upon publication"，当前源码未给出可直接复现实验仓库。
- 论文主要是数值 benchmark；没有真实物理实验或工业数据闭环验证。
- 四个 benchmark 覆盖 perturbation、Allen-Cahn、Burgers、lid-driven cavity，但仍属于 PINN 常见测试集，外推到更复杂几何/多物理/高维问题需要验证。
- adaptive sampling 的候选点生成规模与 candidate distribution 细节在正文中交代不够细，复现时可能影响结果。
- 关键结果大量依赖 Fig. ErrorCost，可读 tex 中没有完整数值表，精确比较需要图数据或代码。
- IDW 初始化新点权重在高维空间可能有距离集中问题，论文没有系统分析高维退化。
- sampling probability 仍由 residual 主导，若 residual 本身被 PINN 错误低估，候选点策略可能漏掉困难区域。
- 默认 `gamma=100`、`p_u=0.2`、`N_s=100` 来自 Burgers hyperparameter study，跨任务最优性没有理论保证。
- 没有与所有 2025-2026 最新 adaptive sampling / dynamic weighting 方法做同场复现比较。
- 检验者不是独立团队，且没有 artifact evaluation 或第三方复现报告。

### 跟我们的关系
- `0616-fex` 的完整课题描述未给出；以下按其关注 physics-informed / PDE solver / adaptive training 的方向判断。
- 若我们的 claim 是 "用 residual 指导采样、用动态权重平衡 PINN loss"，本文会构成直接相关 prior work。
- 若我们的 claim 是 "把难点区域采样和点级权重同时做"，本文有部分撞车，因为它已经把 BRDR point-wise weighting 与 residual adaptive sampling 合并。
- 本文没有 agentic search、自动实验设计、跨问题策略选择、数据驱动候选点生成策略学习等内容；这些仍可作为差异化空间。
- 本文 sampling 仍是 residual-proportional stochastic resampling，不是 uncertainty-aware、goal-oriented、active-learning acquisition 或 learned sampler。
- 本文 weighting 依赖 residual decay rate，不是基于 verifier、conservation audit、operator error estimator 或 multi-fidelity discrepancy。
- 可以引用本文作为 "PINN adaptive weighting + adaptive sampling 组合基线"。
- 可复用的工具/方法是 BRDR IRDR 公式、residual clipping、partial point replacement、IDW 初始化新点历史统计量。
- 我们若做 FEX/自动发现方向，可把本文作为一个 candidate primitive：在实验平台中自动切换 sampling、weighting、二者组合，并统计何时谁有效。
- 差异化重点应落在更强的验证分离、更完整 ablation、最近 2026 方法对比，以及对 non-standard PDE/几何/高维问题的评测。
