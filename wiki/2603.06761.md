# Diversity-Aware Adaptive Collocation for Physics-Informed Neural Networks via Sparse QUBO Optimization and Hybrid Coresets (2603.06761)

- **Year**: 2026.03
- **Authors**: Hadi Salloum 等；Innopolis University, Q-Deep, Center for Engineering Systems and Sciences, Central University, National Medical Research Center of Oncology
- **Venue**: arXiv preprint (ICLR 2026 template; acceptance not established)
- **Keywords**: Physics-Informed Neural Networks, adaptive collocation, QUBO, sparse BQM, coreset, kNN graph, Burgers equation, hybrid anchors
- **One-liner**: 用稀疏 QUBO/BQM 选 PINN 配点，兼顾高残差与空间多样性。

## 精华提炼

### 管线/方法
- 输入是一个 PINN 求解任务：1D time-dependent viscous Burgers equation，空间 $x\in[-1,1]$，时间 $t\in[0,T]$。
- 目标不是改 PINN 主网络，而是改 interior collocation point 的选择策略。
- 候选池 $\mathcal{C}=\{(x_i,t_i)\}_{i=1}^N$ 先从空间-时间域均匀或分层采样得到。
- 训练先做 warm-start：用少量 uniform interior set 加固定 IC/BC set 训练 $S_{\mathrm{warm}}$ 步。
- warm-start 后的参数 $\theta_0$ 用来给每个候选点打分。
- 点的重要性分数定义为 $s_i=r_{\theta_0}(x_i,t_i)^2$，即 PDE residual 的平方。
- residual 是 Burgers 方程残差：$u_t + u u_x - \nu u_{xx}$。
- 分数可做 min-max 或 robust quantile normalization，并可在 99th percentile clipping，避免 shock 附近极端值支配选择。
- 为了降低规模，先 prefilter 到 $M\ll N$ 的工作集。
- prefilter 是混合策略：保留 top $\beta M$ 高残差点，再加入 $(1-\beta)M$ 个 uniform/background 点。
- 论文实验中写明 $\beta=0.7$，即 70% 偏向高残差，30% 保留背景覆盖。
- 多样性由 space-time 相似度建模：$w_{ij}=\exp(-((x_i-x_j)/\ell_x)^2-((t_i-t_j)/\ell_t)^2)$。
- dense 版本会在所有点对上建 quadratic redundancy penalty，复杂度接近 $O(M^2)$。
- 提出版本把相似图稀疏化：在 scaled space-time 中建 kNN graph，只保留边集 $E$ 上的 $w_{ij}$。
- 实验设定中的 kNN degree 固定为 $k=12$。
- dense k-hot QUBO baseline 的目标是：线性项奖励高 residual score，二次项惩罚相似点同时入选，另加 $\lambda(\sum_i z_i-K)^2$ 强制 cardinality。
- 作者指出 dense cardinality penalty 会产生 all-to-all couplers，即使 redundancy 图本身稀疏也会变 dense。
- proposed sparse BQM 删除 dense cardinality penalty，改用线性 bias $\mu$ 控制期望选择数量。
- sparse BQM 目标是 $\sum_i(-\alpha s_i+\mu)z_i+\sum_{(i,j)\in E}\gamma w_{ij}z_iz_j$。
- $\alpha$ 控制 residual informativeness，$\gamma$ 控制 diversity/repulsion，$\mu$ 控制 soft selection size。
- $\mu$ 的启发式是用 score mean 和 $K/M$ 估计，或者跑少量 calibration search。
- sparse BQM 不保证正好选 $K$ 个点，所以后面接 exact-K repair。
- repair 如果选多了，就反复删除 marginal utility 最小的点。
- repair 如果选少了，就反复加入 marginal gain 最大的未选点。
- marginal utility/gain 都用 $\alpha s_i$ 减去与当前已选集合的稀疏图相似度惩罚。
- repair 只在稀疏图上算 redundancy，所以复杂度约 $O(Mk)$。
- 另一个提出组件是 hybrid anchors。
- hybrid anchors 把预算 $K$ 分成 $K_{\mathrm{anchor}}=\lfloor\rho K\rfloor$ 和 $K_{\mathrm{select}}$。
- anchors 用 stratified sampling、Latin hypercube sampling 或 Sobol/grid-jitter 做全局覆盖。
- 剩余预算再用 sparse BQM 选择高 residual 且不冗余的点。
- 实验使用 anchor fraction $\rho=0.2$。
- 论文还支持 adaptive refresh：训练中每隔 $M_{\mathrm{refresh}}$ iterations 重新计算 residual score 并重选 collocation set。
- adaptive refresh 复用 candidate pool，只重算当前参数下的 scores 和受影响边。
- 整体输出是一个大小正好为 $K$ 的 interior collocation set $\mathcal{S}$，用于 PDE residual loss。

### 关键技术细节
- PINN loss 是 IC loss、BC loss、PDE residual loss 的加权和。
- 论文没有在 TeX 中给出完整神经网络宽度、深度、optimizer、learning rate、训练迭代数等可复现细节。
- 基准 PDE 是 1D viscous Burgers equation，viscosity 写为 $\nu=0.01/\pi$，$T=1.0$。
- 参考解由 stable finite-volume scheme 生成，并插值到 evaluation points。
- 参考解脚注称实现中使用 Rusanov flux with explicit diffusion。
- 评测指标包括 relative $L^2$ error、$L^\infty$ error、held-out residual mean、held-out residual 95th percentile。
- 速度指标拆成 warm、score、prefilter、graph、qubo-build、qubo-solve、repair、train。
- 另有 selection overhead ratio：选择相关时间除以训练时间。
- 还报告 time-to-accuracy $T_{\rightarrow\varepsilon}$，即首次达到目标误差阈值的 elapsed time。
- 求解器后端是 D-Wave Ocean 的 classical simulated annealing sampler `neal`，不是量子硬件实验。
- dense QUBO 作为 reference baseline，主要用于显示 dense coupler 带来的 solve overhead。
- sparse BQM + repair 是核心效率贡献。
- hybrid anchors + sparse BQM 是核心鲁棒性贡献。
- optional baseline 包括 uniform、random、residual top-K、residual top-K + anchors、greedy k-center。
- 论文的 novelty claim 可分解为三条：QUBO 化 collocation coreset、稀疏 soft-K BQM + repair、hybrid anchors 保全局覆盖。
- 真正较有差异的是把 residual informativeness 与 diversity repulsion 放进同一个 BQM，并明确报告 selection overhead。
- 与普通 residual adaptive refinement 的差异：不是 sequentially add high residual points，而是 fixed-budget subset selection。
- 与 DPP/k-center/submodular selection 的差异：使用 QUBO/BQM 能接 classical annealing 或 quantum-ready solver。

### 核心结果
- 所有结果声称平均 5 个 random seeds。
- 在 $K=1000$ 时，Uniform relative $L^2$ error 为 $4.8\times10^{-3}$。
- Residual Top-K 在 $K=1000$ 时为 $3.1\times10^{-3}$。
- Sparse BQM 在 $K=1000$ 时为 $2.4\times10^{-3}$。
- Hybrid Anchors + BQM 在 $K=1000$ 时最好，为 $1.9\times10^{-3}$。
- 作者声称 Hybrid BQM 用少 35% 的 collocation points 达到与 Uniform 相同误差。
- time-to-reach $2\times10^{-3}$ error：Uniform 412 s。
- Residual Top-K 达到该阈值用 365 s。
- Sparse BQM 用 298 s。
- Hybrid BQM 用 254 s，是表中最快。
- 作者据此声称 Hybrid BQM 相比 uniform 减少 38% total wall-clock。
- timing breakdown 表中 Uniform total 410 s，train 410 s。
- Residual Top-K score 12 s，train 353 s，total 365 s。
- Dense QUBO score 14 s，solve 121 s，train 342 s，total 477 s。
- Sparse BQM score 14 s，solve 41 s，repair 6 s，train 237 s，total 298 s。
- Hybrid BQM score 14 s，solve 38 s，repair 6 s，train 196 s，total 254 s。
- sparse solve time 相比 dense solve time 从 121 s 降到 41 s，约 3x reduction。
- $\gamma=0$ 的 ablation 使 error 增加 22%，说明 diversity penalty 有贡献。
- anchor fraction 最优点报告为 $\rho=0.2$。
- anchors 太少会 global drift，太多会削弱 shock focus。
- kNN degree 在 $k\ge12$ 后性能稳定，低于 $k=6$ diversity modeling 退化。
- adaptive refresh 每 2000 iterations 使 final error 降低 15%，overhead 增加 9%，net time-to-accuracy 仍改善 11%。
- surprising 点：Dense QUBO 尽管训练时间较低，总时间反而比 Uniform 更慢，瓶颈完全在 solve。
- 另一个关键点：Hybrid anchors 不只是精度更好，还把训练时间从 Sparse BQM 的 237 s 降到 196 s。

### 明确局限
- 实验只覆盖 1D Burgers equation，一个经典但较窄的 PINN benchmark。
- 没有看到 2D/3D PDE、多物理场、inverse problem 或复杂边界条件实验。
- 没有真机平台，也没有真实物理实验；所有评测是数值 PDE benchmark。
- 没有量子硬件实验；QUBO/BQM 只用 classical simulated annealing `neal`。
- TeX 中没有提供代码仓库、artifact、完整超参数表或可直接复现实验脚本。
- 神经网络架构、优化器、batching、IC/BC 点数、candidate pool size $N$、working set size $M$ 等关键信息缺失。
- timing 结果没有说明硬件、CPU/GPU 型号、并行设置，因此 wall-clock 可比性有限。
- finite-volume reference solver 只在脚注中说明为 Rusanov flux + explicit diffusion，缺少网格分辨率和 CFL 设置。
- paper 使用 arXiv/ICLR 模板，但当前只能视作 preprint，不能当作 peer-reviewed 结论引用。
- 所有关键结果都来自作者自己的实现，没有 independent verification。
- 结果表没有置信区间或标准差，虽然说 average over 5 seeds。
- 没有与 PINNACLE、RAR-D、RAD 等强 adaptive collocation 方法做完整同配置复现实验，只是概念性列入相关工作和 baseline 名称。
- 没有证明 sparse BQM + repair 对 PDE loss 有理论近似保证。
- repair 可能改变 annealing solution 的能量结构，论文没有系统分析 repair 后解的 optimality gap。
- 对 $\alpha,\gamma,\mu,\rho,k,\beta$ 的敏感性只给了简短结论，缺少完整 ablation table。
- 如果目标课题强调高维 PDE 或真实工程仿真，这篇只能作为 collocation selection idea，不足以证明 scale-out。
