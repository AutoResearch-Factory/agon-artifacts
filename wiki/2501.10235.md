# SpaceTime: Causal Discovery from Non-Stationary Time Series (2501.10235)

- **Year**: 2025.01
- **Authors**: Sarah Mameche, Lénaïg Cornanguer, Urmi Ninad, Jilles Vreeken (CISPA Helmholtz Center; TU Berlin; German Aerospace Center DLR)
- **Venue**: AAAI 2025 (39th AAAI Conference on Artificial Intelligence)
- **Keywords**: causal discovery, non-stationary time series, regime detection, changepoint detection, context partitioning, Minimum Description Length (MDL), Gaussian processes, kernelized conditional independence testing, temporal/window causal graph
- **One-liner**: 用 MDL 分数统一"非平稳时序因果图发现 + 变点/regime 重建 + 数据集(context)/时间段(regime)划分"三任务的 score-based 方法，靠 GP 建模 + 核判别测试实现，应用于河流径流与生态-大气通量。

## 精华提炼

### 管线/方法
SpaceTime 的输入是**一组**多变量时序数据集 $\mathbf X_\mathcal{T}^\mathcal{D}$（$d\in\mathcal D$ 个不同地点/数据集，各跨越 $\mathcal T$ 时间索引），最大时滞 $\tau_{max}$、变点最小间隔 $d_{min}$ 为超参。输出：一个 window causal graph (WCG) $\mathcal G_{\tau_{max}}$、一组 regime 变点 $\mathcal L$、以及每个变量的 context 划分 $\mathcal C$（空间维，跨数据集）和 regime 划分 $\mathcal R$（时间维，跨时间段）。

核心是一个**迭代坐标下降**（Alg. 1，通常 ~3 轮收敛）：
1. **初始化**：空图下先按边缘分布漂移把 $\mathcal T$ 切成初始变点。
2. **ChangepointDetection**（Line 3）：在当前图 $\mathcal G$ 下，对每个变量/context 在"假定 regime 恒定"的窗口（从上个变点起、长度 $d_{min}$）拟合 GP，用该 GP **预测整段数据**，再对**所有变量与 context 的预测残差同时**跑统计变点检测（推荐 PELT，无需预设变点个数）来定位下一个变点。关键思想：regime 差异体现为"至少一个变量-父节点函数关系变化"，用 GP 预测误差而非原始数据找变点更灵敏。
3. **Partitioning**（Line 4）：用当前图 + 更新后的变点，做成对核判别测试得到 $\mathcal R,\mathcal C$。
4. **EdgeGreedySearch**（Line 5）：固定 $\mathcal L,\mathcal C,\mathcal R$，用贪心加边发现 WCG。
迭代直至收敛。

### 关键技术细节
- **打分基础 = 两部 MDL / 算法因果模型 (AMC)**：postulate 是"最短 Kolmogorov 复杂度的联合分布分解 = 真因果模型"，用两部 MDL 逼近（Marx & Vreeken 2021 形式化连接了两部 MDL 与 AMC）。总描述长度 $L(\mathbf X;\mathcal G)=\sum_i L(X_i\mid \mathrm{pa}(X_i);\mathcal M)$。
- **模型类 = 高斯过程 (GP)**：非参数、且有 refined MDL 分数。单变量条件描述长度 = 负对数似然（拟合）+ $\|f\|_\kappa^2=\alpha^\top K\alpha$（函数平滑度/复杂度惩罚，$\alpha=(K+\sigma^2 I)^{-1}y$）+ $\tfrac12\log\det(\sigma^{-2}K_S+I)$（refined 项）。
- **非平稳目标函数**（Eq. objective）：$\arg\min_{\mathcal G,\mathcal L,(\mathcal C,\mathcal R)}\sum_{X\in\mathcal G}\sum_{(r\in\mathcal R,k\in\mathcal C)} L(X^{(k,r)}\mid \mathrm{pa}(X^{(k,r)});\mathcal M_\kappa)$ —— 对每个"子样本"$(k,r)$（context×regime）分别编码各变量的因果机制。
- **Edge-Greedy Search**：沿用 Mian/GLOBE 思路，边强度 = 加该边后的**压缩增益** $\delta E:=L(\cdot;\mathcal M)-L(\cdot;\mathcal M')$；前向按边强度降序加边，后向剪枝精修父集与时滞。$\delta E$ 本身就是一个**连续因果强度度量**。
- **Regime/Context 划分 = 核化假设检验**：对两个子样本 $(d,w),(d',w')$，引入指示变量 $S\in\{0,1\}$，在合并数据上检验条件独立 $X\indep S\mid \mathrm{pa}(X)$（用 HSIC/KCI 核检验，源自 CD-NOD、Perry et al.）。Context 在**空间**维（跨 $\mathcal D$、聚合全 $\mathcal T$）成对测；Regime 在**时间**维（跨 $\mathcal L$ 诱导的时间窗、合并全部数据集）成对测；再把"无机制变化"的成对关系聚成划分。
- **因果模型**：Temporal Causal Graph (TCG) → 在 "Repeating Edge Property"（边方向随时间/数据集不变）下压缩为 Window Causal Graph (WCG)。假设 Causal Markov + Faithfulness + Sufficiency（无隐混杂/选择）。用 **latent-variable / mixture 形式**表达"独立机制变化"：每个变量附一个**regime-context 混杂隐变量 $Z_j$**（离散，含混合权 $\pi_j$）+ 参数 $\theta_j$ + 外生噪声 $N_j$，$Z$ 即把非平稳/异质当作隐混杂建模（见 Fig. augmented model）。
- **一致性定理 (Thm)**：在 Assumptions 1–6（Markov+Faithfulness+Sufficiency、SCM with contexts/regimes、Sufficient Capacity & Persistence、Shift Faithfulness、Independent Changes）下，目标函数在 $\mathcal D,\mathcal T\to\infty$ 极限被真模型 $(\mathcal G^*,\mathcal L^*,\mathcal C^*,\mathcal R^*)$ 最小化。证明按情况分解：变点/划分已知→DAG 发现（Markov 等价，$\hat G\supsetneq G^*$ 靠 Markov+Faithfulness，$\hat G\subsetneq G^*$ 靠 $D_{KL}$ 项）；方向靠 Independent Change（$Z_i\indep Z_j$）；变点/机制识别靠 Shift Faithfulness + $D_{KL}$ 项。
- **实验设置**：合成数据用 Tigramite (Runge) 生成随机 WCG（$\tau_{max}=2$），随机采样变点（$d_{min}=30$），每个 context/regime 重采样一部分边权；线性/非线性 × 高斯/均匀噪声。对比方法：PCMCI+、J-PCMCI+、R-PCMCI、DYNOTEARS、VARLiNGAM、CD-NOD。指标：变点用 F1（over $\mathcal L$）、因果发现用有向边 F1（over $\mathcal G$ 与 summary DAG）、regime 划分用 ARI 与 NMI。真实数据：GRDC 河流径流（$|\mathcal D|=307$ 欧洲流域，T/P/Q 三变量）、FLUXNET 生态-大气通量（$|\mathcal D|=63/64$ 站点，T/$R_g$/NEE/H/LE/VPD 六变量），日分辨率、$T$ 取一年、固定 $d_{min}=30,\tau_{max}=2$。

### 核心结果
- **合成**：SpaceTime 在非线性高斯设定下全面稳健。变点检测/划分的完整结果(solid)接近 oracle(hatched，已知图)，说明它对因果图的少量错误鲁棒，且迭代通常 3 轮内收敛。
- **对比洞见**：DYNOTEARS/VARLiNGAM 因线性/非高斯假设在此非线性场景弱；PCMCI+/CD-NOD 等单序列法用多数投票聚合；已知(hatched) vs 未知(solid)变点差距不大（合并 regime 增大样本量、且边强度变化不如图结构变化剧烈）。J-PCMCI+ 虽可处理多 context 但优势不显著（机制漂移太细微，或 context 数不够多而 CI 检验因 one-hot 高维变贵）。优于 R-PCMCI 的原因：SpaceTime 显式施加 regime 持久性，而 R-PCMCI 只能把 regime 切换次数当背景知识给。
- **河流径流**：跨 307 流域发现共同 WCG，含滞后边 $P_{(t-1)}\to Q_{(t)}$（降水→径流，合理）；该边的因果强度 $\delta E$ 在各流域间有区域相似性也有地理差异（地形/气候/流域面积/海拔/植被）。对选定流域展示了变点 + regime 划分。
- **FLUXNET**：发现 7 条边的共同 DAG；把各(地点,月,年)的 $\delta E$ 用 t-SNE 降到 2 维，**分组不强绑定地理/生态系统，而与水分可用性和全球辐射相关**——强边连通对应高辐射高降水（最优生长期），弱边连通对应冬季低能量态。**反直觉发现**：不同生态系统会穿越非常相似的气象状态（与 Krich 2021、Kraemer 2019 一致）。Regime 划分揭示异常年份：Hainich/Tharandt 站在 **2003 欧洲干旱/热浪**、2006 年被识别为独立 regime（对应月降水 P 异常）——即因果机制变化检测到了真实极端气象事件。

### 明确局限
- 真实数据**无 ground-truth 网络**，只能"复现前人定性结论"式验证（作者自陈）；t-SNE 空间非度量、无法直接做距离比较（作者自陈，并说 SpaceTime 的核检验可补此缺）。
- 强假设：Repeating Edge Property（边方向不随时间/空间变，只机制变）、无隐混杂/选择、regime 变点集 $\mathcal L$ 对所有数据集**共享**（假定所有 dataset 被非平稳同样影响，如季节性）、每个子样本样本量足够大 + regime 持久 $d_{min}$。
- 时间尺度：全部实验是**慢变、日/月/年**尺度（水文/气候），无小时级快变过程；变点是"regime 切换"而非快速非平衡启动。
- 一致性只在 $\mathcal D,\mathcal T\to\infty$ 极限；有限样本下核检验/GP 拟合成本与统计功效未系统刻画。
- 无真机/无实时；PELT 等变点技术可替换但未做敏感性消融（部分消融推到 Appendix）。
- 跟我们的关系：它把非平稳/异质当作要**正确建模**的对象（隐混杂 $Z$、regime 划分），而非我们要利用的"估计不稳定性信号"——哲学相反，见下。

### 跟我们的关系
对 `0710-causal-scs`（因果效应**不稳定性**作为**强对流 SCS 早期预警**指标）课题：

- **撞不撞车：不撞**。SpaceTime 的目标与我们**方向相反**：它把非平稳/context 异质当作要"建模正确"的东西（隐混杂 $Z$、regime 划分、假设 shift-faithfulness），追求更准的因果图/机制；我们恰恰**不满足这些假设**，把假设违背导致的**估计不稳定性**当正向信号。因此它不占我们核心 novelty（核心最近前作仍是 CNM 2412.16235 与 NPG 2026），只是强 related-work / 候选方法。
- **能用的工具/方法**：
  1. **score-based 候选方法**：topic 明说"视资源纳入 score-based 方法"。SpaceTime 正是非平稳时序的 score-based(MDL) 法，其边强度 $\delta E$（压缩增益）是连续因果强度度量，可作为**第 4 种估计器**加入 PCMCI+/J-PCMCI+/LKIF 的**跨方法分歧**基线，检验"不稳定性是否跨 estimator paradigm（约束型 vs 信息流 vs 打分型）稳健"。
  2. **突变型指标的现成机制**：SpaceTime 的"GP 残差→PELT 变点"正对应 idea 的"变点检测 PELT/BOCPD/CUSUM"；其"核检验判两窗机制是否不同(HSIC/KCI on $X\indep S\mid pa$)"可直接实现 idea 的"DAG 结构相邻窗口 edge 变化 / 机制突变"指标。
  3. **隐混杂视角**：其 augmented model 把 regime-context 当离散隐混杂 $Z_j$——与我们"confounder 假设违背"框架同构，可借来形式化"我们保留而非消除的那个 $Z$"。
  4. **气象领域先例**：FLUXNET 案例用因果机制变化检测出 **2003 热浪**这一真实极端气象事件——这是"因果机制变化可标记极端气象事件"的慢变尺度先例，可作为我们快变 SCS 迁移的动机锚点（同时也提醒尺度错配风险，正是 idea-reviewer 指出的漏洞）。
- **该引不该引：该引**。作为(a)非平稳时序 score-based 因果发现代表、(b)在气象/水文真实数据上用因果机制变化做 regime 检测的先例。J-PCMCI+ (2306.12896) 与 PELT (1101.1438) 经它引出，都是我们已在用/待用的方法。
- **差异化**：SpaceTime 慢变、追求正确建模、AAAI 主流因果发现；我们快变(小时级)中尺度对流、把"估计脆弱性"反转为信号、跨方法分歧检验。两者可在同一 pipeline 里作为"正确建模 baseline vs 不稳定性信号"的对照。

- **撞车评估：无核心撞车，方向相反可互为对照**。SpaceTime 是"把非平稳建模正确"的正面工程（隐混杂 $Z$、regime/context 划分、shift-faithfulness、一致性证明），我们是"利用估计不稳定性做信号"。它不碰 SCS、不碰"不稳定性即信号"、不碰快变尺度，故不占我们 novelty；核心撞车对象仍是 CNM(2412.16235)/NPG(2026)。它的价值是"正确建模 baseline"——正好是我们要对比反转的对象。
- **可用工具**：(1) 边强度 $\delta E$（MDL 压缩增益）= 第 4 种 score-based 因果强度估计器，加入跨方法分歧分析，检验不稳定性是否跨 estimator paradigm 稳健；(2) "GP 残差 → PELT 变点"与"HSIC/KCI 核检验判两窗机制差异 $X\indep S\mid pa$"直接实现 idea 的突变型指标（变点 + DAG edge 变化）；(3) augmented latent-confounder $Z_j$ 视角，形式化我们"保留而非消除的混杂"；(4) 公开代码 eda.group/spacetime + FLUXNET/GRDC 公开气象水文数据。
- **差异化 & 风险提示**：SpaceTime 的 FLUXNET 案例已在**慢变**尺度用因果机制变化标记 2003 热浪——既是我们迁移到"快变中尺度对流"的动机锚，也再次暴露 idea-reviewer 指出的**尺度错配**软肋（CSD/regime 理论多为慢变近平衡）。建议在 proposal 里显式把 SpaceTime 归为"正确建模非平稳"阵营，与我们"不稳定性信号"阵营对置，并借其 $\delta E$/核检验作为我们指标的其中一个具体算子。
