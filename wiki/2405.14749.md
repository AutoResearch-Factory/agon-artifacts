# Policy Gradient Methods for Risk-Sensitive Distributional Reinforcement Learning with Provable Convergence (2405.14749)

- **Year**: 2024.05
- **Authors**: Minheng Xiao, Xian Yu (corresponding), Lei Ying — Department of Integrated Systems Engineering, Ohio State University; EECS, University of Michigan
- **Venue**: arXiv (formatted for ICML 2025)
- **Keywords**: distributional reinforcement learning, risk-sensitive RL, policy gradient, coherent risk measures, CVaR, categorical distribution, finite-time convergence, distributional Bellman operator
- **One-liner**: 提出首个分布策略梯度定理（probability measure gradient 的解析形式），构造 CDPG 算法在 categorical 近似下给出有限支持最优性保证和 inexact policy evaluation 下的有限时间局部收敛保证。

## 精华提炼

### 管线/方法

完整 pipeline：**给定 policy π_θ → 分布式策略评估（Distributional Policy Evaluation）→ 得到概率测度 η_θ^(s,a) → 计算概率测度梯度 ∇_θ η_θ^s → 代入风险测度梯度公式（Theorem 2.2）→ 梯度下降更新 θ**。

具体两阶段循环：
1. **分布策略评估（Distributional Policy Evaluation）**：利用 projected distributional Bellman operator Π_C ∘ T^π 的 √γ-收缩性质（在 supremum-Cramér 距离 l̄₂ 下），反复应用该算子直到收敛，得到 categorical 分布 η_{N,∞} ∈ P_N^{S×A}
2. **分布策略改进（Distributional Policy Improvement）**：通过 Categorical Policy Gradient Theorem（Theorem 4.5）计算 ∇_θ p_i^θ，代入 Theorem 2.2（risk measure gradient 对偶表示）得到 ∇_θ ρ(Z_N^s) 的闭式表达，梯度下降更新 θ

CDPG 算法（Algorithm 2）的具体步骤：
- 采样一条 trajectory τ_{θ_t} 跟随 π_{θ_t}
- 初始化 η_{N,0} ∈ P_N^{S×A}，运行 k 次 Π_C T^π 得到 η_{N,k}
- 计算 g(s_h) = Σ_a ∇_θ π_{θ_t}(a|s_h) · η_{N,k}^{(s_h,a)}
- 沿 trajectory 累积 projected pushforward：∇_θ η_{N,k}^s ← g(s₀) + Σ_{t=1}^{|τ|} B̃^{τ(s₀,s_t)} g(s_t)
- 从 ∇_θ η_{N,k}^s 提取 ∇_θ p_i^θ，代入 CVaR 梯度公式（Example 4.1）：∇_θ ρ_CVaR = (1/α) Σ_i ∇_θ p_i^θ (z_i - q_α) 1_{z_i > q_α}
- θ_{t+1} ← θ_t - δ · ∇_θ ρ(Z_N^s)

### 关键技术细节

- **Risk measure**: 静态 coherent risk measure，通过 risk envelope U 的对偶表示 ρ(Z) = max_{ξ∈U} E_ξ[Z] 来统一处理。重点分析 CVaR（α∈[0,1]），其中 risk envelope U = {ξ: ξ(ω)∈[0,α⁻¹], ∫ξ f_Z dω = 1}
- **Categorical 近似**：用 N 个固定支撑点 z_min=z_1<...<z_N=z_max（等距分割）上的 categorical 分布 P_N^θ = {Σ p_i^θ δ_{z_i}} 近似任意连续分布
- **Projection operator Π_C**：将任意 Dirac 测度 δ_y 投影到最近的两个网格点（线性插值），保证投影后仍在 P_N 内
- **Distributional Bellman operator T^π**：定义在概率测度空间上，(T^π η)^(s,a) = Σ_{s'} P(s'|s,a) Σ_{a'} π(a'|s') (b_{C(s,a),γ})_# η^(s',a')，其中 b_{c,γ}(z)=c+γz 是 bootstrap 函数
- **Contraction**: T^π 在 Wasserstein 度量 d̄_p 下是 γ-contraction；Π_C T^π 在 supremum-Cramér 距离 l̄₂ 下是 √γ-contraction
- **Convergence guarantee (Theorem 4.7)**：对 CVaR，设 stepsize δ=1/β，policy evaluation oracle 调用次数 k(N,|τ|) = κ N |τ+1|。则 min_{t=1..T} ||∇_θ ρ_t||² ≤ ε 当 T ≥ 4β(ρ_1-ρ^*)/ε 且 κ ≥ max{O(log(N^{1.5} ε^{-0.5})/N), O(log(N ε_α^{-2})/N)}。关键：κ 只随 N 对数增长（logarithmic dependence）
- **有限支持最优性 gap (Lemma 4.3)**：|min_θ ρ(Z^s) - min_θ ρ(Z_N^s)| ≤ ε_opt 当 N ≥ L_1² (z_max-z_min)² / ((1-γ) ε_opt²) = O(ε_opt^{-2})
- **误差放大系数 δ_Π**：投影算子引入的误差放大，δ_Π = 2(γ+1)(β+γμ)(N-1) / (γ(z_N-z_1))，其中 β=max{c_max-z_min, z_max-c_min}, μ=max{|z_max|,|z_min|}
- **实验环境**：随机 3×3 Cliffwalk（滑落概率 p=0.2）和 CartPole（连续状态空间 → NN 策略）
- **网络结构**：2-layer MLP with ReLU activation；Actor_lr=0.01, Critic_lr=0.01；Adam optimizer
- **加速技巧**：Warm-start（下一轮 policy evaluation 用上一轮估计的分布初始化）+ Early stopping（分布差异不再下降时提前终止）。Cliffwalk 上训练加速 ~5x

### 核心结果

- **Theorem 4.1 (Distributional Policy Gradient Theorem)**：∇_θ η_θ^s = E_τ [g(s₀) + Σ_{t=1}^{|τ|} B^{τ(s₀,s_t)} g(s_t)]，其中 g(s)=Σ_a ∇_θ π(a|s) η_θ^(s,a)，B 为 t-step pushforward operator。**这是首个给出概率测度梯度的解析形式的定理**，将经典 PG theorem 从标量推广到测度空间。
- **Theorem 4.5 (Categorical Policy Gradient Theorem)**：将 Theorem 4.1 适配到 categorical 分布，pushforward 替换为 projected pushforward B̃ = Π_C(b_{c₀,γ})_# Π_C(b_{c₁,γ})_# ...
- **CVaR Gradient 闭式 (Example 4.1)**：∇_θ ρ_CVaR(Z_N^s) = (1/α) Σ_{i=1}^N ∇_θ p_i^θ (z_i - q_α) 1_{z_i > q_α}，直接可计算
- **对比 SPG (Tamar 2015)**：Cliffwalk 上 CDPG 样本效率 ~4x SPG；CartPole 上 ~10x（含 early stopping）
- **风险中性 vs 风险厌恶**：风险中性策略在 Cliffwalk 上的 cost distribution 有长尾高方差；风险厌恶（α=0.1）策略显著降低尾部风险

### 明确局限

- **仅在小规模 toy 环境验证**：3×3 Cliffwalk 和 CartPole，没有在更复杂的连续控制任务（MuJoCo 等）或真实高风险应用（自动驾驶、金融）上测试
- **仅 CVaR 被实验验证**：框架声称支持所有 coherent risk measures，但实验只做了 CVaR。其他风险测度（entropic risk、mean-semideviation、distortion risk）仅给出梯度公式示例，未实现
- **收敛保证是局部的**：Theorem 4.7 只能保证找到 ε-stationary point（梯度范数小），不是全局最优。需要 gradient dominance 才能提升到全局收敛（论文自己也指出 Huang 2021 证明了 Markov coherent risk 不是 gradient dominated）
- **仅考虑静态风险测度**：动态（嵌套）风险测度的 case 未被覆盖
- **有限状态/动作假设**：理论分析假设 S 和 A 是有限集。CartPole 实验用 NN 扩展到连续状态，但没有对应的理论收敛保证
- **categorical 近似精度与 N 的关系**：Lemma 4.3 要求 N = O(ε_opt^{-2})，在高精度下 N 需求快速增长
- **误差放大 δ_Π 随 N 线性增长**：δ_Π ∝ N，当 N 很大时（高精度近似），每个 projection step 的误差放大可能抵消 contraction 效果
- **只与 SPG (Tamar 2015) 对比**：未与更近期的 risk-sensitive DRL 方法（DSAC、WCPG 等）对比
- **与 FEX 课题的关系（详见下文）**：不撞车。此文的 risk-sensitive DRL PG 框架与 FEX 的表达式树搜索在问题设定上完全不同，但分布策略梯度定理的分析方法学（inexact policy evaluation 下的收敛分析、smoothness 证明）可作为 idea claim (iii) conditional PG theorem 的技术参考模板。

**撞车评估：不撞车。** 本文研究的是在标准 MDP 设定下（Cliffwalk、CartPole），使用 DRL 进行 risk-sensitive policy optimization，目标是优化 CVaR 等 coherent risk measures 下的策略。FEX 课题研究的是表达式树搜索的 discoverability 复杂度，目标完全不同。本文不涉及：(a) 表达式树 grammar 的等价类 quotient counting；(b) 离散结构搜索的 sampler lower bound；(c) FEX controller 的 PG convergence 与 good-class mass 的关系；(d) 系数可行性 ∃R-hardness。

**可用的工具/方法：**
1. **Theorem 4.1/4.5（Distributional/Categorical Policy Gradient Theorem）**：提供了概率测度梯度的解析求法——如果 FEX controller 需要计算分布性 reward（如 PDE residual 的分布而非点估计），这个框架可以直接参考。当前 FEX 使用点估计 reward（PDE residual squared），但若扩展到 uncertainty-aware search，可能需要分布性建模。
2. **Inexact policy evaluation 下的收敛分析框架（Theorem 4.7 + Lemma 4.10）**：这个方法学对 idea claim (iii) conditional PG theorem 有参考价值。CDPG 的收敛分析将 gradient error 分解为 policy evaluation 的分布估计误差（由 k 次 Π_C T^π 迭代的 contraction 控制）→ probability measure gradient error（Lemma 4.10）→ CVaR gradient error（Lemma 4.12）→ 最终 convergence（Theorem 4.7）。这个逐层误差传播的分析 template 可以适配到 FEX 的 controller convergence analysis。
3. **β-smoothness 证明（Lemma 4.8）**：通过 bound 住 risk measure 对各参数（p_i^θ, ξ^*, λ^*）的一阶和二阶导数来证明 smoothness。如果 FEX controller 的 expected reward 需要证明 smoothness，这个逐项 bound 的方法可参考。
4. **Projection operator + contraction 分析**：Π_C T^π 在 l̄₂ 下 √γ-contraction 的证明结构（projection 不破坏 contraction）可用于分析 FEX 搜索中如果引入类别投影/离散化后的收敛行为。

**可迁移的技术要素（对 idea 的次要贡献）：**
- **逐层误差传播的收敛分析模板（Lemma 4.10 → Lemma 4.12 → Theorem 4.7）**：distribution estimation error → probability measure gradient error → risk measure gradient error → convergence。这个三层误差传播结构可作为 idea claim (iii) conditional PG theorem 的技术写作参考——将 FEX controller gradient error 分解为 critic error（PDE residual 评估）→ policy gradient estimate error → convergence。
- **δ_Π 误差放大系数的分析**：投影算子每步放大误差 δ_Π(N)，与 support 点数 N 成正比。类比到 FEX：语法投影（expression tree → canonical form）每次 rewrite 也可能放大搜索误差——这个 "per-step error amplification" 的概念结构可迁移。
- **β-smoothness 的逐项 bound 证明（Lemma 4.8）**：对风险测度的 Hessian 逐项求界（ξ^* term + equality constraint term + inequality constraint term）。FEX 的奖励函数 smoothness 分析也可采用类似的分项策略。
