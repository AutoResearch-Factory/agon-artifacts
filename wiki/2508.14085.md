# Parameter-Aware Ensemble SINDy for Interpretable Symbolic SGS Closure (2508.14085)

- **Year**: 2025.08
- **Authors**: Hanseul Kang, Ville Vuorinen, Shervin Karimkashi (Aalto University, Mechanical Engineering)
- **Venue**: arXiv
- **Keywords**: SINDy, SGS closure, parameter-aware discovery, Dimensional Similarity Filter, symbolic regression, ensemble methods, Gram-matrix accumulation, turbulence modeling
- **One-liner**: 扩展 SINDy 框架，通过参数感知库、量纲相似性滤波器、内存高效 Gram 矩阵累积和集成共识四大增强，从多参数仿真数据自动发现 PDE 和亚格子应力闭合模型的符号表达式。

## 精华提炼

### 管线/方法

完整 pipeline：**多参数仿真数据生成 → 参数感知候选库构建 → 量纲相似性滤波 (DSF) → Gram 矩阵增量累积 → 集成稀疏回归 (ElasticNet/SR3) → 系数稳定性分析与共识筛选 → 符号模型输出**。

1. **数据生成**：用有限差分法对 Heat/Burgers/KdV-Burgers 三个 1D PDE 数值积分。初始条件用 Perlin 噪声（octaves ∈ [1,3], frequency ∈ [0.5,1.5]），网格分辨率从 (100,3000) 到 (150,500) 不等。
2. **参数感知库构建**：基础项包括 field (u)、parameter (p₁, p₁⁻¹, p₂, p₂⁻¹, ...)、derivative (u_x|cd1, u_x|uw2, u_xx|cd2, ...)。对 degree d，生成所有单项式组合，随后执行逆消除、交换排序和纯参数消除三道规约。
3. **DSF**：为每个 primitive 分配 [L, T] 维度向量，复合项的维度为分量之和。Hard filter 要求精确匹配目标项维度；soft filter 用归一化 taxicab/euclidean 距离加容差 δ_tol。计算量 O(|Θ|)，典型缩减 1-2 个数量级。
4. **Gram 矩阵累积**：将 Θ 分 batch 计算 G = Σ Θ_i^T Θ_i 和 b = Σ Θ_i^T u_{t,i}，内存从 O(np) 降至 O(p²)。兼容 LASSO/ElasticNet/SR3。
5. **集成共识**：Bagging ensemble (10 estimators, 80% subsampling)。双阈值筛选：consensus frequency f_j ≥ 0.8 且 CV_j ≤ 0.15。CV_j = σ_j/(|μ_j| + ε)，只对超过 λ_noise=10⁻⁵ 的系数计算。
6. **SGS 闭合发现**：用 box-filtered Burgers' 数据，500 parameter realizations，50 ensemble batches，adaptive CV threshold（CV_init=0.5, decay=0.5），DSF tolerance=0.25，degree-3 多项式库含 Δ 和 Δ²。

### 关键技术细节

- **模型架构**：非神经网络方法，纯稀疏回归。ElasticNet（α=0.01, l1_ratio=0.5）和 SR3（threshold=0.1, nu=1.0）两种回归器并行使用以确保收敛一致性。
- **数据规模**：Burgers: (150,500) 网格，ν ∈ [0.001/π, 0.01/π]；KdV-Burgers: (150,500) 网格，C₁ ∈ [1,15], C₂ ∈ [0.002,0.042], ν ∈ [0.045,0.45]。
- **过滤参数**：SGS discovery 使用 Δ ∈ [1,11]δx，box filter 宽度扫描。
- **后处理**：SINDy 发现的 τ_SGS = C·Δ²·ū_x² 被转换为符号修正形式 τ_SGS = -C·Δ²·ū_x·|ū_x| 以匹配物理 parity。
- **计算平台**：Python 3.11.13, numpy 2.0.2, scipy 1.16.0, scikit-learn 1.6.1, Google v2-8 TPU。Seed=1234。
- **Loss 函数**：稀疏回归用 ℓ₁ 正则化（LASSO/ElasticNet）或 SR3 的 relaxed ℓ₁。
- **数值方法**：有限差分 — cd1 (1st-order central), cd2 (2nd-order central), uw2 (2nd-order upwind)。边界用 2-cell wall buffer 排除边界误差。

### 核心结果

| 指标 | 结果 |
|------|------|
| PDE 恢复（3 个方程） | 100% 结构准确，系数标准差 10⁻⁹~10⁻⁸ 量级 |
| DSF 库缩减 | KdV-Burgers: 346→124 term (64% reduction)；Burgers: 22→17 |
| Gram 内存节省 | 最高 95% (Burgers A2: 3.74→0.19 GB) |
| 烧蚀最优配置 | DSF+Gram: Burgers 14.5s/0.04GB，Exact recovery |
| **关键发现**：Ensemble 无 DSF 失败 | KdV-Burgers B3: 2134.7s/34.7GB，5/346 term，仅部分识别 |
| Ensemble+DSF | B4: 123.4s/15.7GB，3/124 term，Exact |
| **SGS 发现** | τ_SGS = 0.1604·Δ²·ū_x²，C_s^SINDy ≈ 0.4005 |
| SGS R² | 0.885 (avg across filter widths)，Taylor/Leonard 为负 R² |
| 双回归器一致性 | ElasticNet: 0.1600, SR3: 0.1604，7 轮迭代内收敛至单一 term |

**反直觉发现**：SINDy 发现的 Smagorinsky 常数 C_s ≈ 0.4005 远超经典 3D 湍流值 0.1-0.24。作者解释为"模型不足"(model inadequacy)：1D Burgers' 激波主导的能量耗散机制不同于 3D 湍流级串，SINDy 自动适配了更大的常数来补偿这一结构性差异。网格细化分析证实该升高常数在所有分辨率下持续存在。

### 明确局限

1. **仅 1D**：所有验证限于 1D 方程。作者承认扩展到 2D/3D 需耦合 autoencoder 等降维方法，但会牺牲符号可解释性——这是本文维护的核心价值。
2. **无 a-posteriori LES 测试**：SGS 闭合仅在 a-priori（即直接比较预测 τ_SGS 与真实 τ_SGS）上评估，未在真实 LES 求解器中闭环测试。
3. **无真实湍流数据**：仅用 Burgers' 方程（含激波的 1D 非线性方程），非真实 3D 湍流 DNS 数据。
4. **单 term 闭合**：发现的 SGS 模型是单项式 Δ²ū_x²，虽然能恢复 Smagorinsky 结构，但未发现更复杂的多尺度或多 term 闭合形式。
5. **无代码开源**：论文描述方法详细但未提供代码仓库。
6. **小规模计算**：v2-8 TPU 上的实验规模较小，未展示在大规模 HPC 场景下的扩展性。
7. **与 DNN 方法无定量比较**：仅与 Taylor/Leonard/Smagorinsky 解析闭合比较，未与 Kurz2023、Meng2023 等 DNN-based SGS 方法做定量对比（作者认为与黑箱模型比预测精度 "不能服务于可解释性目标"，但这一立场可能被 reviewer 质疑）。
8. **Perlin 噪声初始条件**：仅测试了光滑随机场初始条件，未展示对更真实湍流初始条件的泛化。
9. **CV/consensus 阈值需手动设置**：f_threshold=0.8, CV_threshold=0.15 等超参数缺乏自动选择机制。

### 可借用的工具与方法
1. **DSF（量纲相似性滤波）**：轻量、通用的候选 term 筛选策略——给每个 term 分配物理维度向量，按与目标项的距离过滤。可用于任何需要预筛选符号回归候选库的场景。
2. **集成共识机制**：双阈值（consensus frequency + coefficient of variation）比单一稀疏性更鲁棒，可泛化到其他回归/特征选择任务。
3. **Gram 矩阵增量累积**：将大数据回归的内存复杂度从 O(np) 降到 O(p²) 的标准技巧，本文给出了 SINDy 语境下的具体实现。
4. **参数感知符号库构造**：将物理参数作为符号变量嵌入候选库，使单一模型覆盖多参数区间——如果 0616-fex 涉及多条件下的规律发现，可直接借鉴。

### 差异化方向
- 该文仅做 1D + a-priori 验证，2D/3D + a-posteriori LES 闭环测试是明显的 extension gap。
- 该文发现的 SGS 模型是单项式，多 term/多尺度闭合的自动发现仍是开放问题。
- DSF 当前仅支持 [L,T] 二维量纲空间，扩展到更多基本量纲（质量、温度等）是自然方向。
