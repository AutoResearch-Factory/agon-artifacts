# Delightful Gradients Accelerate Corner Escape (2605.11908)

- **Year**: 2026.05
- **Authors**: Jincheng Mei, Ian Osband (Google DeepMind)
- **Venue**: NeurIPS 2026 (preprint)
- **Keywords**: policy gradient, softmax, corner escape, delightful gradient, bandit theory, convergence rate, self-trapping, surprisal gating, tabular MDP, function approximation counterexample
- **One-liner**: DG 用 advantage × surprisal 的 sigmoid 门控替代标准 policy gradient 的等权更新，消除 softmax 策略在次优角落的自陷机制，将逃逸时间从指数级降为对数级，并保持 O(1/t) 全局收敛。

## 精华提炼

### 管线/方法

**问题设定**：K-armed bandit + tabular MDP，softmax 参数化策略。

**核心管线**：
1. **输入**：当前策略 π（softmax 输出）、reward vector r（bandit）或 Q/V 值（MDP）
2. **计算 advantage**：U(a) = r(a) - π^T r（bandit）；U(s,a) = Q^π(s,a) - V^π(s)（MDP）
3. **计算 surprisal**：ℓ(a) = -log π(a)
4. **计算 delight gate**：w(a) = σ(U(a)·ℓ(a) / η)，η 为温度参数
5. **门控梯度更新**：Δθ(a) = Σ_i w(i)·π(i)·U(i)·(𝟙{a=i} - π(a))
6. **极限情况**（η→0）：w(a) → 𝟙{U(a) > 0}，称为 Enlightened Gradient (EG)，硬性删除所有负 advantage 动作

**对比 PG**：标准 PG 的 w(i) ≡ 1，所有动作等权参与更新。

### 关键技术细节

**核心发现——ally/enemy 分类**：
- 对角落 arm j，将其他 arm 分三类：
  - **Allies** S⁺ = {i: r(i) > r(j)}：advantage > 0，pull mean reward up
  - **Harmful** S⁻ = {i: r(i) < r(j)}：advantage < 0，pull mean reward down，给角落 arm 正 advantage，是自陷的根源
  - **Ambiguous** {j}：角落 arm 自身，advantage = O(ε)

**Logit-Gap 恒等式（核心分析工具）**：
- Δθ(a) - Δθ(b) = Σ_i w_i·π(i)·U(i)(𝟙{a=i} - 𝟙{b=i}) + (π(b)-π(a))·S
- 其中 S = Σ_i w_i·π(i)·U(i) 为加权 population drift
- PG 下 S = 0（advantage centered），EG/DG 下 S > 0

**Sector bound（定理 1-2）**：
- EG：在 sector 条件 π(1) ≥ ε/K 下，Δθ(1) - Δθ(j) ≥ Δ_{1j}·ε/(4K)
- DG（有限温度）：Δθ(1) - Δθ(j) ≥ Δ_{1j}·ε/(8K)（w₁ ≥ 1/2 补偿 sigmoid 衰减）
- 核心机制：arm-1 贡献 Ω(ε/K)，allies 贡献 ≥ 0，harmful 被多项式压制（w(k)·π(k) ≤ π(k)^{1 + Δ_{jk}/(2η)} = o(ε)）

**逃逸时间（定理 3）**：
- τ ≤ (4K/(Δ_{1j}·ε₀)) · log(π₀(j)/π₀(1))
- 对数级 vs PG 的指数级，问题难度 1/Δ 只线性影响常数因子

**全局收敛（定理 4-5）**：
- Bandit：V_t monotonically non-decreasing → V_t 收敛 → limit point 是 one-hot → 所有次优 corner 是 repeller → 收敛到 π*
- MDP：per-state 解耦 + performance difference lemma + Bellman optimality 刻画
- 渐进速率 O(1/t)，与 PG 相同但无 corner transient 指数惩罚
- Remark 5 关键：EG/DG 下 bad region measure zero → 证明比 PG 简单得多（无需间接逃逸论证）

**有限温度扩展**：
- DG 在任意 η ∈ (0,∞) 下保持 sign-preserving 结构：w(a) > 0 且 w(a)·U(a) 与 U(a) 同号
- 单调性恒等式：V' - V = (1/Z) Σ_i π(i)·U(i)·(e^{α·w(i)·U(i)} - 1) ≥ 0
- 全局收敛和 O(1/t) 速率对所有有限温度成立，但 η 越大 bound 越松

**Surprisal 的结构必要性**（Remark 3）：
- 如果只用 advantage gating（w_k = σ(-Δ_{jk}/η)），harmful arm 的 gating weight 是非零常数
- S⁻ 贡献是 Θ(ε)，与 beneficial term 同阶，sector bound 破裂
- product U·ℓ 是必须的：rarity-sensitive suppression

**函数近似反例（Section 7）**：
- 双状态 MDP：s₁ 中 a₁ 有益（+10），s₂ 中 a₁ 灾难性（-100），共享参数 θ 控制 π(a₁|s) = σ(θ)
- PG：F_PG(θ) = -45p(1-p) < 0，正确驱动 p→0
- EG：F_EG(θ) = 5p(1-p)(1-11p)，在 p*=1/11 有稳定不动点
- DG(η=1)：数值不动点 p*≈0.116
- 失败三条件：(1) 共享参数，(2) 同动作在不同状态有冲突 advantage，(3) gate 压制了更大 magnitude 的修正（因为负 delight）
- 类比 TD(λ) + function approximation 的经典反例（Tsitsiklis & Van Roy, Baird）

**实验设置**：
- Tabular escape-time：3-armed bandit, r=(1, 1-Δ, 0), θ₀=(-1,5,1) → π₀≈(0.01,0.94,0.05)，Δ sweep from 0.5 to 0.01，forward Euler dt=1.0，max 10⁷ time units
- MNIST contextual bandit：2-hidden-layer MLP [100,100], ReLU, Adam lr=10⁻³, batch 100, corner pretraining toward wrong digit (1~10⁴ steps), recovery training T=10k steps, 100 runs per config

### 核心结果

| 结果 | 数值/陈述 |
|------|----------|
| EG sector gap | ≥ Δ_{1j}·ε/(4K) |
| DG sector gap | ≥ Δ_{1j}·ε/(8K) |
| EG escape time | O((K/Δ)·log(π₀(j)/π₀(1))) — 对数级 |
| PG escape time | O(exp(...)) — 指数级（Li et al. 2021） |
| EG bandit convergence | global → π*, O(1/t) rate |
| EG MDP convergence | global → π*, O(1/t) rate |
| DG (finite η) convergence | 同 EG，但 bounds 随 η 退化 |
| Tabular escape scaling | DG near-linear in 1/Δ, PG superlinear |
| MNIST corner recovery | DG error flat across corner depth, PG degrades |
| Counterexample EG fixed point | p* = 1/11 (stable interior) |
| Counterexample DG fixed point | p* ≈ 0.116 |

**Surprising/反直觉发现**：
- All better-than-corner arms are allies（Remark 1）：它们对逃逸的贡献是非负的，不会互相竞争。arm 数量增加只通过 sector 条件中的 1/K 因子影响常数。
- EG/DG 的全局收敛证明比 PG 简单（Remark 5）：因为 bad region measure zero，子最优 corner 是 repeller，不需要 PG 的间接矛盾论证。
- DG 的函数近似反例不是实用禁令，而是理论边界标记：MNIST 实验表明 DG 在共享参数神经网络上仍然优于 PG。

### 明确局限

1. **Tabular only 的理论保证**：所有收敛定理和 sector bound 都假设 tabular 参数化（per-state independent logits），不适用于函数近似。
2. **共享参数反例**：Section 7 明确展示了 DG/EG 在共享参数下可以收敛到次优不动点。
3. **限于 discrete action spaces**：整个分析基于 softmax 参数化和有限动作空间。
4. **唯一最优策略假设**：MDP 收敛定理假设唯一 π*（distinct rewards in bandit case）。
5. **全可达性假设**：MDP 证明需要 d_ρ^π(s) ≥ d_min > 0。
6. **温度退化**：η→∞ 时 δ_η→0，bad region 不再 measure zero，机制崩溃。
7. **无真机实验**：仅有 tabular simulation 和 MNIST contextual bandit，无更复杂环境。
8. **与我们的关系**：该方法本身不是 PDE/foundation model，但其"门控梯度更新"的思想（用 advantage × surprisal 做 rarity-sensitive suppression）可能在 FEX 的 policy optimization 组件中有用——特别是在处理 exploration-exploitation 权衡时。
