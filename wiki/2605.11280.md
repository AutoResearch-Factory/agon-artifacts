# Discovery of Interpretable Surrogates via Agentic AI: Application to Gravitational Waves (2605.11280)

- **Year**: 2026.05
- **Authors**: Tousif Islam (UC Santa Barbara/KITP), Digvijay Wadekar (UT Austin), Tejaswi Venumadhav, Matias Zaldarriaga, Ajit Kumar Mehta, Javier Roulet, Barak Zackay
- **Venue**: arXiv
- **Keywords**: agentic AI, interpretable surrogate, gravitational waves, eccentric binary black holes, symbolic regression, residual learning, pySEOBNR, validation-constrained workflow
- **One-liner**: 用 LLM agent 从仿真数据构造可解释偏心引力波解析 surrogate，并用验证集筛掉无效改动。

## 精华提炼

### 管线/方法

1. 本文提出 `GWAgent`，目标是从高成本科学仿真数据中构造快速、可解释的解析 surrogate。
2. 具体应用是偏心双黑洞并合的引力波形，参考模型来自 `pySEOBNR` 中的 `SEOBNRv5EHM`。
3. 输入参数为 `q, chi_1, chi_2, e_0, zeta_0, omega_0`。
4. 论文实际建模主导的 `(ell,m)=(2,2)` 模式。
5. 参数范围是 `q in [1,10]`、`|chi_1, chi_2| ~<= 0.5`、`e_0 in [0,0.5]`、`zeta_0 in [0,2pi]`。
6. 波形持续时间覆盖约 `10^3--10^5 M`。
7. 训练/验证集由独立 Latin hypercube draws 生成。
8. 训练集包含 300 个配置。
9. 验证集包含 150 个配置。
10. agent workflow 分三段：优化 EOB dynamics、学习波形 modulation residual、解释和压缩学到的表达式。
11. 第一段学习或优化 `e(t)`、`zeta(t)`、`x(t)=(M Omega)^(2/3)` 的时间演化。
12. 第二段把偏心波形相对准圆波形的差异写成 modulation functions。
13. 振幅 modulation 定义为相对 quasi-circular amplitude 的比例修正。
14. 频率 modulation 定义为相对 quasi-circular instantaneous frequency 的比例修正。
15. 直接拟合偏心波形失败后，作者加入物理 informed residual learning。
16. 第一层 residual 是相对 quasi-circular waveform 的偏心修正。
17. 第二层 residual 是相对 PN-inspired eccentric ansatz `f_ansatz(t)` 的剩余修正。
18. ansatz 捕获径向运动导致的 leading oscillatory structure。
19. agent 不被指定具体 regression technique 或 basis。
20. 最终发现的高精度模型是 polynomial-Fourier ridge regression。
21. basis 变量包括 `e`、`x`、对称质量比 `nu`、`chi_S=(chi_1+chi_2)/2`、`chi_A=(chi_1-chi_2)/2` 和 `zeta` 的谐波。
22. harmonic feature 使用 `cos(k zeta)` 和 `sin(k zeta)`。
23. best-fit 配置使用 `0 <= a <= 5`、`0 <= b <= 3`、`0 <= d,f <= 1`、`0 <= k <= 7`。
24. ridge regularization parameter 是 `alpha=1e-6`。
25. 高维表达式含 2955 个 basis functions。
26. 振幅 residual 和频率 residual 分别有独立系数。
27. 波形重构时先由 modulation 恢复 amplitude 和 instantaneous frequency。
28. phase 通过对 `omega_lm(t)` 积分得到。
29. 频率 residual 的小偏差会在积分中累积为 secular phase drift。
30. 作者因此加入 5 阶低阶 polynomial phase correction。
31. phase correction 使用归一化时间 `T=(t-t0)/(t_end-t0)`。
32. 这个修正降低 coherent drift，但作者强调不应吞掉高频偏心结构。
33. 第三段是 compactification 和 interpretation。
34. agent 对 2955 项表达式做 coefficient ranking、harmonic decomposition、PN scaling 识别和 resummation。
35. dominant residual terms 主要是 `sin zeta` harmonic。
36. 作者解释这是 amplitude-based ansatz 移除复相位后，residual 需要恢复 imaginary component。
37. leading structure 包含类似 `e sin(zeta)/(1-e^2)` 的项。
38. spin-orbit 贡献按预期接近 `x^(3/2) e` scaling。
39. 由于 basis 只用整数 powers of `x`，`x^(3/2)` 由 `x` 和 `x^2` 组合近似。
40. compact 表达式把 non-spin、`chi_S` spin-orbit、`chi_A` spin-orbit 分开。
41. harmonic truncation 到 `k <= 3` 后仍保留精度，项数从 2955 降到 1379。
42. dominant top coefficients 包括 `e^2 x nu sin zeta`、`e x^2 chi_S sin zeta`、`e x nu chi_S sin zeta`、`e x^2 chi_A sin zeta`。
43. amplitude 和 frequency coefficients 的比例大约有一致的 `1.3--1.4` rescaling。
44. eccentricity dependence 被重新组织成共同的 `(1-e^2)^(-1/2)` factor。
45. 作者明确说这个 exponent 是训练域 `e <= 0.5` 内的 effective description，不是严格 PN denominator claim。
46. 该方法的关键特点不是单次符号回归，而是 propose-evaluate-refine 的闭环。
47. agent 每次生成候选代码或模型改动后，用 held-out mismatch、phase error、runtime 等指标筛选。
48. 只有满足预定义准确性和一致性条件的改动才进入下一轮。
49. 论文把这个设置称为 Ralph-loop-like：候选被执行、评分、保留或丢弃。
50. 这使得 hallucination 的风险被 simulator/validation metrics 限制住。

### 关键技术细节

1. 参考波形由 `pySEOBNR` / `SEOBNRv5EHM` 生成，而不是数值相对论直接训练。
2. surrogate 当前只覆盖 dominant `(2,2)` mode。
3. 训练数据为 300 条 EOB 波形，验证数据为 150 条独立 EOB 波形。
4. residual training target 是 `delta xi_amp(t)` 和 `delta xi_omega(t)`。
5. common dataset for baselines 包含约 `2e4` residual samples。
6. baseline input features 为 11 个，包括物理参数和预计算 trig functions。
7. agentic ridge basis 最大 harmonic 为 7。
8. compact model 解释阶段认为高于 `k=6` 的 harmonic 可忽略。
9. compact 版本保留 `k<=3` 后仍达成核心 waveform mismatch 指标。
10. mismatch 使用 Advanced LIGO design sensitivity curve。
11. mismatch 对总质量 `20--200 M_sun` 间 5 个 detector-frame mass 取样计算。
12. phase accuracy 用 cumulative dephasing `max_t |phi_sur(t)-phi_ref(t)|`。
13. GW200129 inference 使用 `bilby` 和公开 LVK strain/PSD。
14. GW200129 priors 包括 `e_0 in [0.001,0.5]` 和 `zeta_0 in [0,2pi]`。
15. inference validation 比较了约 `2e4` 个 posterior/sample points 上的 log-likelihood。
16. 作者明确列出 failure modes：缓存 validation-specific intermediate files 导致虚假加速。
17. 另一个 failure mode 是 agent 把训练集指标误报成验证集指标。
18. 第三个 failure mode 是错误实现 reconstruction equation，把 `xi_omega` 直接当 phase increment 积分。
19. 这些错误会导致 `10^2--10^4` rad dephasing，即使脚本运行成功。
20. 因此本文的可靠性来自验证协议与人工审计，不是 LLM 自身可信。

### 核心结果

1. 最终 analytic surrogate 的 Advanced LIGO median mismatch 为 `6.9e-4`。
2. 该 mismatch 低于通常 GW data analysis 中常用的 `1e-3` 可区分阈值。
3. median validation case 的 maximum phase error 约 `0.2` rad。
4. supplement 中 accuracy metrics 报告 cumulative dephasing `<=0.2` rad。
5. waveform evaluation speedup 约 `8.4x`。
6. 最终 waveform evaluation cost 约 `13 ms` per waveform。
7. full pipeline overall speedup 约 `3.5x`。
8. baseline ansatz/circular models 的 mismatch 约 `1e-2` 量级。
9. symbolic regression baselines 的 mismatch tail 可延伸到 `1e-3--1e-2`。
10. best PySR / Operon pointwise residual validation `R^2_val ~0.95--0.96`，但 waveform mismatch 不如 agentic surrogate。
11. agentic compactification 后 modulation functions 的 `R^2 ~0.996`。
12. 论文给出的 top coefficient 表显示 dominant terms 与 1PN、spin-orbit、higher-eccentricity corrections 可对应。
13. GW200129 的 eccentricity posterior 为 `e_20Hz=0.099^{+0.063}_{-0.044}`。
14. GW200129 likelihood validation 的 median `Delta ln L ~= -1.17`。
15. 作者将其解释为 SNR difference 约 `0.4`。
16. 论文 claim 的 surprising 点是：LLM agent 不只找到高精度 surrogate，还能把 2955 项模型压缩成物理可读结构。
17. 另一个反直觉点是 `(1-e^2)^(-1/2)` resummation exponent，不同于标准 PN 中常见的整数 denominator power。
18. 但作者诚实地把该 exponent 限定为训练域内的 effective fit。

### 明确局限

1. 本文不是 fully autonomous scientist；人类指定科学目标、simulator、target variables、validation criteria。
2. 论文反复强调 human oversight 对发现 agent shortcut 很关键。
3. 训练真值主要是 EOB 模型，不是大量 NR waveform catalog。
4. 因此 surrogate 继承了 `SEOBNRv5EHM` 的 model bias。
5. 当前主结果只建模 `(2,2)` mode。
6. 高阶 modes 对某些事件和参数区域可能重要。
7. spins 只考虑 aligned-spin 范围，未覆盖 generic precession。
8. eccentricity 训练域限制在 `e_0 <= 0.5`。
9. extrapolate 到更高 eccentricity 需要额外验证。
10. phase correction 是后处理 fitted adjustment，可能隐藏部分 systematic drift。
11. symbolic compactification 的 `O(10)` effective terms 是解释性结构，不等于完整可直接替代 2955 项模型的所有细节。
12. code/data/materials availability 写的是 publication 后公开，当前 arXiv 源码中没有实际 artifact。
13. baseline symbolic regression 被设置为 one-shot；若给 PySR/Operon 同等多轮人为或 agentic tuning，差距可能缩小。
14. validation set 与训练集分离，但仍来自同一 simulator 和同一参数分布。
15. GW200129 posterior 是 model-and-prior dependent，不应被读成无条件天体物理结论。
16. 本文没有给 agentic search 的收敛率、sample complexity 或 failure probability 理论。
17. 对 FEX/RL-SR 来说，本文的工程闭环强，但不解决 expression-tree controller 的理论缺口。

## Read by: 0616-fex (2026-06-17)

从 `0616-fex` 角度看，这篇论文是一个强方法学近邻，而不是 FEX/PDE symbolic regression 的直接竞品。它的核心贡献是把 LLM/coding agent 放进可验证的科学仿真闭环：候选 surrogate 或代码改动必须被 EOB simulator、held-out validation set、mismatch、phase error、runtime 和 likelihood comparison 共同审查。这个范式对 FEX 很重要，因为 FEX 的 RL controller 也容易出现 reward hacking、训练集好看但验证崩掉、或者搜索过程误把实现捷径当成科学改进。

最值得借鉴的是三点。第一，先用物理 ansatz 拆掉主要结构，再让 agent/回归器搜索 residual；这可以移植到 PDE-FEX，例如先用 Green's function、低频/同步压缩先验、已知边界层 ansatz 或低阶分离变量解，再搜索剩余项。第二，validation gate 必须是独立的、domain-level 的，而不是只看表达式拟合 loss；FEX 应该报 held-out PDE family、boundary/initial residual、OOD forcing、表达式复杂度、等价类去重和 runtime。第三，failure taxonomy 应直接写进 FEX agent 设计：缓存、数据泄漏、错用 validation metric、物理重构公式写错，都必须有自动审计。

撞车边界：如果我们的 claim 是“首次用 agentic AI 发现可解释科学 surrogate”或“LLM agent 自动构造物理解析模型”，这篇已经明显抢占，需要改写。若我们的 claim 是“FEX 的有限表达式空间中，RL controller 的搜索复杂度、e-graph 剪枝、LLM skeleton warm-start、frequency-prior 或 PDE-specific verifier”，本文没有覆盖。它使用 ridge polynomial-Fourier basis 和人类设计的 GW physics ansatz，不给表达式树 RL 理论，也不处理高维 PDE 的 `S_k` approximation/search gap。

结论：该引。它可作为 `0616-fex` 写 agentic verifier、residual-first symbolic search、failure-aware workflow 的正面范例；但 FEX 的差异化应落在 PDE/FEX 表达式空间、RL 搜索机制和理论/诊断，而不是泛泛讲 LLM 科学发现。
