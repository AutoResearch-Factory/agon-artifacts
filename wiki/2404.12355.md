# Towards a Foundation Model for Partial Differential Equations: Multi-Operator Learning and Extrapolation (2404.12355)

- **Year**: 2024.04
- **Authors**: Jingmin Sun (CMU), Yuxuan Liu (UCLA), Zecheng Zhang (FSU), Hayden Schaeffer (UCLA)
- **Venue**: arXiv preprint (未标注会议)
- **Keywords**: PDE foundation model, multi-operator learning (MOL), multi-modal machine learning, symbolic encoding, Polish notation, extrapolation of physical features (EPF), transformer, neural operator
- **One-liner**: PROSE-PDE 是一个双模态（数值数据 + 符号方程）transformer 架构的 PDE 基础模型，在 20 种 1D PDE 上同时训练多个算子，无需微调即可外推到未见方程和未见物理现象。

## 精华提炼

### 管线/方法

完整 pipeline：**双模态输入 → 编码 → 特征融合 → 双模态解码 → 输出**

1. **数值数据输入**：短时观测，16 个时间步（t ∈ [0, t_f/2]），128 点空间网格（x ∈ [0, 2]）。每个空间网格点 u(x, t_i) 连同时间变量 t_i 通过线性层映射为高维特征向量。
2. **符号方程输入**：用 Polish Notation（波兰表示法）将 PDE 方程编码为树结构，再转为 token 序列。操作符、符号、尾数等对应可训练的 token embedding。
3. **Data Encoder**：2 层 self-attention transformer，以时间变量 t_i 作为位置编码，在序列元素间交换信息（可定位极值、检测模式）。
4. **Symbol Encoder**：4 层 self-attention transformer，标准正弦位置编码，处理和解释符号输入的数学结构。
5. **Feature Fusion**：将处理后的 data 特征和 symbol 特征拼接，加入可学习的模态类型嵌入（modality-type embedding），再经 8 层 self-attention 实现跨模态信息交互——data 获得方程结构信息，symbol 获得数据参数范围。
6. **Data Decoder**：8 层 cross-attention，融合特征作为"基函数"来源，独立 query time points 作为评估点。评估点之间相互独立，时空复杂度与 query 点数线性关系。类似 DeepONet 的 branch-trunk 哲学。
7. **Symbol Decoder**：8 层标准 encoder-decoder transformer，自回归生成 Polish notation 方程。从 start-of-sentence token 开始，逐 token 生成直到 end-of-sentence。推理时用 greedy search（已足够获得有效数学表达式）。

三种模态配置：
- **2-to-2**：全模型，五组件完整（Data Enc + Symbol Enc + Fusion + Data Dec + Symbol Dec）
- **2-to-1**：去掉 Symbol Decoder，用于测试对未见算子的泛化
- **1-to-1**：仅有 Data Encoder 和 Data Decoder，用作 ablation baseline

### 关键技术细节

- **模型架构**：Hidden dim = 512，FFN hidden dim = 2048，8 个 attention heads。Encoder-Decoder 均为 transformer 变体。
- **PDE 方程编码**：将方程转为树结构（节点=操作，叶=变量/常数），再转为 Polish Notation 序列。支持两种等价编码（直接偏导符号 u_x 或微分算子符号 ∂_x），本文采用前者。
- **训练数据**：20 种 1D 时间相关非线性 PDE（扩散、多孔介质、Klein-Gordon、Sine-Gordon、Cahn-Hilliard、KdV、对流、波动、扩散反应 4 种、粘性守恒律 4 种、无粘守恒律 3 种、Fokker-Planck）。每个 PDE 参数在 [0.9q_c, 1.1q_c] 均匀采样生成 512 个变体，每个变体配 50 种初始条件，总计 **512K 个系统**。
- **初始条件生成**：5 种：正弦波叠加（PDEBench 风格）、高斯随机场、高斯分布、均匀分布、二次函数。训练主要用周期正弦波，测试可切换为 GRF 或其他。
- **求解器**：方法线（method of lines）、PDEBench、Fourier 谱方法、矩阵数值方法、精确解——取决于方程类型。
- **Loss 函数**：数据分支用归一化空间的 MSE（L_d），符号分支用标准交叉熵（L_s），总 loss L = αL_d + βL_s，默认 α=5.0, β=1.0。
- **优化器**：AdamW，lr=1e-4，weight decay=1e-4，cosine scheduler，10% warmup steps，batch size=512/GPU，gradient clip=1.0。单张 RTX 4090 训练约 4.5 小时，30 epochs × 2000 steps。
- **评测指标**：数据端用相对 L² 误差和 R² 分数；符号端先解码 Polish notation 为表达式树，验证其是否有效数学表达式，再用四阶空间多项式 × 二阶时间多项式作为测试函数评估符号误差。

### 核心结果

| 设置 | 数据相对误差 | R² | 符号相对误差 | 有效符号比例 |
|------|-------------|-----|------------|------------|
| Known (已知方程) | 0.92% | 0.998 | 0.01% | 100.00% |
| Skeleton (未知系数) | 1.06% | 0.998 | 0.768% | 99.94% |

**外推 Study 1 — 五种外推场景（2-to-1 模型，无需微调）**：

1. **Temporal Grid**：训练和测试用不同的 query time points → 标准设定，作为所有实验的基础验证
2. **Time Marching**：训练时预测 t∈[1,2]，测试时 rollout：t_end=2.25 误差 ~3%，t_end=2.5 误差 7.09%，t_end=3 误差 10.09%。误差随 rollout 步数累积但可控
3. **Out-of-Distribution (OoD)**：训练系数范围 [0.9q_c, 1.1q_c]，测试 [0.8q_c, 0.9q_c] ∪ [1.1q_c, 1.2q_c] → 仍能预测
4. **Input Function Class**：训练用周期正弦波 IC，测试用 GRF（更不规则）→ 仍有效
5. **Unseen Operators**：训练 5 种算子族（128K 系统），测试 viscous Burgers'（102.4K 系统）→ 无需微调即可预测

**外推 Study 2 — Transferring Physical Features（转移物理特征）**：
- 目标：在 viscous Burgers' 上预测 rarefaction wave，但训练中该方程仅见过 shock
- 通过混合其他方程的 shock/rarefaction 比例（5 组实验），PROSE-PDE 能从其他守恒律的 rarefaction 中学到机制并迁移
- 最低误差（Exp. 5）2.34%，优于直接用 cosine flux 作为预测的 3.59%
- 相似度检验：训练方程与目标方程 rarefaction 的 L² 相似度均高于预测误差，排除记忆复制

**外推 Study 3 — Generalizing to Multiple Shocks（冲击波交互）**：
- 训练仅含单冲击解，测试要求预测双冲击交互
- 目标 viscous cubic flux 在训练中仅有单冲击，测试准确预测双冲击行为
- Exp. 3 预测误差 2.35%，远低于直接用 Burgers' 预测的 16.06%

**Ablation — 符号模态的必要性**：
- 仅给初始时间步数据（ill-posed）时：2-to-1 (1.03% 误差) >> 1-to-1 >> FNO/DeepONet → 证明符号编码解决 well-posedness
- 输入时间步数变化：2-to-2 Skeleton 模型误差稳定，说明符号模态提供额外信息稳定预测
- Loss 权重 α/β 从 5 降到 0.2：数据误差仅轻微上升，符号输出促进 Fusion 层的编码质量

**Gradient Flow 分析**：加大符号 loss 权重增强网络下半部分（Symbol Encoder/Decoder）学习，同时影响 Data Decoder。但 Fusion 结构被更好地训练，因此数据预测仅轻微下降。

### 明确局限

1. **仅 1D PDE**：所有实验限于一维空间+时间，作者明确说未来方向是扩展到多维非线性 PDE
2. **仅常系数**：训练数据中 PDE 系数为常数（虽有 ±10% 随机化），没有变系数 PDE
3. **仅时间依赖 PDE**：不含稳态/椭圆型 PDE
4. **规模有限**：512K 系统 vs LLM 数万亿 token；hidden dim 512 vs 数千；单卡 4090 训练 4.5 小时。作者承认更大训练集会带来更强能力
5. **依赖符号输入**：1-to-1（纯数据）模式在 MOL 中因 ill-posed 而失败；对外推任务需要完整方程信息作为输入
6. **外推范围有限**：Unseen operator 测试仅从一个训练算子族泛化到一个测试算子；物理特征外推仅在守恒律族内
7. **无真机实验数据**：所有数据来自数值模拟，未验证对真实实验数据的泛化
8. **无不确定性量化**：纯确定性预测，没有置信区间
9. **符号生成用 greedy search**：虽声称足够有效，但与 beam search 相比可能有性能损失
10. **没有与 LLM-based 方法直接对比**：仅在相关工作部分提及 UPS（用 LLM 生成方程标签），但没有直接实验比较
