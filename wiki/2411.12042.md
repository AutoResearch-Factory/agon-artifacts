# Fast Convergence of Softmax Policy Mirror Ascent (2411.12042)

- **Year**: 2024.11
- **Authors**: Reza Asad (Simon Fraser University), Reza Babanezhad (Samsung AI), Issam Laradji (ServiceNow), Nicolas Le Roux (Mila), Sharan Vaswani (Simon Fraser University)
- **Venue**: AISTATS 2025
- **Keywords**: policy gradient, mirror ascent, softmax parameterization, NPG, function approximation, linear convergence, reinforcement learning
- **One-liner**: 提出 SPMA（Softmax Policy Mirror Ascent），通过在 logits 对偶空间做 mirror ascent 实现线性收敛，无需归一化步骤，在 tabular / linear FA / deep RL 三档均 match 或超越 NPG/MDPO/PPO/TRPO。

## 精华提炼

### 管线/方法

论文提出的 SPMA 算法管线如下：

1. **策略参数化**：使用 softmax 函数将 logits z 映射为动作概率 π(a|s) = exp(z(s,a)) / Σ_a' exp(z(s,a'))
2. **Mirror map 选择**：使用 log-sum-exp mirror map Φ(z) = Σ_s w(s) ln(Σ_a exp(z(s,a)))，权重 w(s) = d^{π_t}(s)（当前策略的状态占用度量）
3. **Mirror ascent 更新**（等价形式）：
   - 梯度形式：∇Φ(z_{t+1}) = ∇Φ(z_t) + η ∇_z J(z_t)
   - 优化形式：z_{t+1} = argmax_z [⟨z - z_t, ∇_z J(z_t)⟩ - (1/η) D_Φ(z, z_t)]
4. **Tabular 更新**：π_{t+1}(a|s) = π_t(a|s) · (1 + η A^{π_t}(s, a))，线性更新，无需归一化
5. **函数近似扩展**（Algorithm 1 - SPMA with FA）：
   - 将 log-linear 参数化解释为对 logits 空间的约束 z ∈ Z = {Xθ}
   - 先做无投影 mirror ascent 得到 z_{t+1/2}（即 tabular 更新）
   - 再做 Bregman 投影：θ_{t+1} = argmin_θ D_Φ(z_θ, z_{t+1/2})
   - 该投影等价于加权 softmax 多分类问题（凸优化！）
   - 实践中用 m 步 SGD 近似求解，内环 step-size 用 Armijo line search
6. **实际实现**：actor-critic 架构，GAE 估计 advantage，CNN（Atari）/ MLP（MuJoCo）

### 关键技术细节

- **Mirror map 的深刻含义**：log-sum-exp mirror map 的梯度 ∇Φ(z) 就是 softmax 函数本身，这意味着 logits 空间和概率空间互为对偶空间。SPMA 在对偶空间做 mirror ascent，这是其与 NPG（在概率空间做 mirror ascent）和 SPG（用 Euclidean mirror map）的本质区别。
- **Tabular MDP 收敛**：
  - Bandit：δ_T ≤ (1 - 1/K) exp(-η Δ_min T / K)，其中 Δ_min 是 optimal arm 与次优 arm 的最小 gap
  - MDP：||V* - V^{π_T}||_∞ ≤ (Π α_t) ||V* - V^{π_0}||_∞，其中 α_t = 1 - η C_t (1-γ)
  - C_t = min_s {π_t(ã_t(s)|s) · Δ^t(s)} 是关键常数，经验验证其 non-zero
  - 步长条件：η < min{1-γ, 1/(C_t(1-γ))}
- **Bandit 超线性收敛**：使用 gap-dependent step-size η_{a,a'} = 1/|Δ(a,a')| 时，δ_T ≤ [(1 - 1/K)]^{2^T}，这是 PG 方法在 bandit 上的首个全局超线性收敛结果。
- **函数近似收敛**：
  - J(π*) - J(π_T) ≤ (Π α_t) (J(π*) - J(π_0)) + β Σ_t Π_{i=t+1}^{T-1} α_i
  - β = √2 / ((1-γ)² ρ_min) · √(ε_stat + ε_bias) + ε_approx/(1-γ)
  - ε_stat：采样误差（有限样本，O(1/n + 1/m)）
  - ε_bias：表示偏差（FA 表达能力不足，tabular 时 ε_bias=0）
  - ε_approx：critic 估计 advantage 的误差
  - ρ_min：初始分布下界（exploration assumption）
- **与 MDPO 的关键差异**：MDPO 的 surrogate 即使在 tabular softmax 参数化下也是非凸的，而 SPMA 的 surrogate 在 log-linear 参数化下是凸的（加权 softmax 分类），因此有理论保证。
- **实验超参数**：
  - η 网格搜索：{0.3, 0.5, 0.7, 0.9, 1.0}
  - 内环步数 m=5（默认），Ablation 测试 m=10, 15
  - Atari: CNN actor-critic, framestack=4, horizon=128, 8 env copies
  - MuJoCo: 2-layer MLP, horizon=2048, full batch + Armijo line search
  - 总 timesteps: 10^7，5 seeds，95% CI

### 核心结果

| 实验场景 | SPMA vs 基线 | 关键发现 |
|----------|-------------|---------|
| Bandit (理论) | SPG sublinear, NPG linear | SPMA linear，gap-dependent η 可达超线性 |
| Tabular MDP | vs SPG, NPG | SPMA ≈ NPG >> SPG，对 η 更不敏感 |
| CliffWorld (linear FA) | vs MDPO, SPG | SPMA > MDPO (小 d 时)，m↑ 差距缩小 |
| FrozenLake (linear FA) | vs MDPO, SPG | SPMA ≈ MDPO >> SPG |
| Atari (5 games, CNN) | vs MDPO, PPO, TRPO-C, TRPO-R | SPMA ≥ MDPO >> PPO, TRPO |
| MuJoCo (4 tasks, MLP) | vs MDPO, PPO, TRPO-C, TRPO-R | TRPO-C > SPMA ≥ MDPO > PPO; m↑ SPMA→TRPO-C |
| MuJoCo ablation m=10,15 | - | SPMA 随 m 增大持续改进，趋近 TRPO-C |

**Surprising 结果**：
- TRPO-C（constrained，用 conjugate gradient）在 MuJoCo 上最优但在 Atari 上表现很差——作者推测 CNN 下 conjugate gradient 失效
- TRPO-R（regularized，与 SPMA 目标函数相似）在所有环境上都远差于 SPMA——说明把 importance sampling ratio 换成 log-ratio 能带来巨大经验增益
- SPMA 在完全没有 code-level optimization 的情况下就能与高度优化的 baseline 竞争

### 明确局限

**作者自述局限**：
1. 线性收敛理论依赖 C_t non-zero，这在理论上无法证明，只能经验验证
2. FrozenLake 的终端状态导致 C_t=0，需要排除这些状态
3. η 需要 grid search，没有自适应调参机制
4. 增加 m 不一定提升 Atari 性能（可能因为内环固定 step-size 未针对 m 重调）
5. 没有 off-policy 版本

**审读者发现的局限**：
1. **无真机/无 real-world 实验**：只在仿真环境（Gym MuJoCo, Atari）测试，没有机器人真机实验
2. **没有标准 benchmark 比较**：未与 Rainbow DQN、SAC 等 value-based 方法比较，仅与 on-policy PG 家族比较
3. **没有大规模实验**：Atari 只测了 BeamRider, DemonAttack, Alien, Amidar 4 个游戏，MuJoCo 只测了 Hopper, Walker2d, HalfCheetah, Ant 4 个任务
4. **没有消融日志**：未展示 GAE λ、horizon、折扣因子等超参数的消融
5. **计算成本未报告**：未报告 SPMA vs MDPO 的 wall-clock time 或 FLOPs
6. **Critic 结构不详**：critic 网络的具体结构和训练方式未详细说明
7. **与 fex-search-complexity 的关系**：本文是纯 PG 优化方法，不直接涉及 search/planning 复杂度。但 SPMA 的"不需要归一化"和"凸 surrogate"特性可能降低搜索空间中的策略优化计算成本——如果 fex 涉及在搜索空间中使用 RL-based policy，SPMA 可以作为一个更简单、更有理论保证的替代方案。

### 撞车评估
不撞车。本文做的是 on-policy 策略梯度优化算法的收敛性分析和实验验证，核心贡献在优化理论（mirror map 选择 → 线性收敛）和算法设计（凸 surrogate）。与 fex-search-complexity 关注的搜索复杂度分析是正交维度。如果 fex 涉及在搜索空间中使用 RL policy，SPMA 可以作为一个工具方法被引用。

### 可用方法
- **SPMA 更新公式**：π_{t+1}(a|s) = π_t(a|s)(1 + η A^{π_t}(s,a)) 是极简的线性更新，可以嵌入到任何需要策略改进的系统中
- **凸 surrogate 特性**：对于 log-linear 参数化，每轮优化等价于加权 softmax 分类（凸优化），这是 SPMA 相对于 MDPO 的最大实践优势——收敛更稳定、不需要调 KL 约束
- **Bandit 超线性结果**：如果 fex 涉及 bandit-based 搜索方向选择，这个结果可以引用来论证某些 bandit 策略的快速收敛

### 可引用的理论结果
- Tabular linear convergence without distribution mismatch ratio dependence
- FA setting 的 neighborhood convergence 分解（ε_stat + ε_bias + ε_approx）
- log-sum-exp mirror map 的 duality 视角（logits ↔ probabilities）
