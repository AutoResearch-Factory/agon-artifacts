# VaSST: Variational Inference for Symbolic Regression using Soft Symbolic Trees (2602.23561)

- **Year**: 2026.02
- **Authors**: Somjit Roy, Pritam Dey, Bani K. Mallick (Department of Statistics, Texas A&M University)
- **Venue**: arXiv
- **Keywords**: symbolic regression, variational inference, soft symbolic trees, Bayesian symbolic regression, uncertainty quantification, Gumbel-Softmax, SRBench, Feynman equations
- **One-liner**: 用变分软树做符号回归并给结构不确定性。

## 精华提炼

### 管线/方法
- 任务输入是监督数据集 `D_n={(x_i,y_i)}`，目标是恢复可解释的闭式符号表达式。
- 方法把一个表达式表示为 symbolic tree：内部节点是 unary/binary operators，叶节点是输入特征。
- 允许的操作符集合在实验中固定为 `{+, *, -, /, exp, log, sin, cos, ^2}`。
- VaSST 不直接搜索单棵树，而是使用由 `K` 棵 symbolic trees 组成的 symbolic ensemble。
- 主模型是线性回归形式：`y_i = beta_0 + sum_j g_j(x_i) beta_j + epsilon_i`。
- 每个 `g_j` 是一棵符号树在输入 `x_i` 上的求值结果。
- 回归系数和噪声方差使用 Normal-Inverse-Gamma 共轭先验。
- 树结构使用最大深度为 `D` 的 full binary skeleton 表示。
- skeleton 中每个节点有三个潜变量：expand indicator `e`, operator assignment `o`, feature assignment `h`。
- 真实 symbolic tree 通过 deterministic pruning 从 skeleton 得到。
- 如果 `e=0`，该节点成为叶子，所有后代被剪掉。
- 如果 `e=1` 且 operator 是 unary，右子树被剪掉。
- split prior 使用深度相关概率 `p_zeta = alpha (1+d_zeta)^(-delta)`，用 Occam 约束惩罚深层复杂结构。
- operator 权重和 feature 权重分别使用 Dirichlet prior。
- 后验中的回归参数 `beta, sigma^2` 被解析边缘化，只留下树结构相关参数 `Theta`。
- 边缘似然 `log p(y|T)` 可用 NIG 共轭形式计算，依赖 expression design matrix `T`。
- 直接对离散树结构做 VI 会遇到组合爆炸，作者给出的规模为 `O((2p|O|)^(N*K))`。
- 核心 trick 是把离散树变量替换为 continuous relaxations，形成 soft symbolic trees。
- expansion indicator 使用 Binary Concrete relaxation。
- operator assignment 和 feature assignment 使用 Gumbel-Softmax relaxation。
- 每个 soft node 的 leaf contribution 是 soft feature mixture `h_tilde^T x_i`。
- 每个 soft node 的 unary contribution 是所有 unary operators 对左子树求值的加权和。
- 每个 soft node 的 binary contribution 是所有 binary operators 对左右子树求值的加权和。
- 完整 soft node output 由 `e_tilde` 在 terminal contribution 和 nonterminal contribution 间做 gating。
- 对所有样本和 `K` 棵 soft trees 求值后得到 soft design matrix `T_soft`。
- ELBO 由 `E_q[log p(y|T)] - KL(q||prior)` 构成。
- `E_q[log p(y|T)]` 不可解析，使用 Monte Carlo samples from soft tree distribution 估计。
- KL 项对 Bernoulli、Categorical、Dirichlet 各部分有解析表达式。
- 训练使用 automatic differentiation 和 AdamW 最大化 stochastic ELBO。
- temperature 从 `tau_start=1.0` 线性 anneal 到 `tau_end=0.5`，推动软结构逐步变尖。
- 训练后用优化出的 variational parameters 抽样 `H` 个 hard symbolic tree ensembles。
- 每个 hard sample 都重新构造 expression design matrix，并用解析后验均值估计 `beta` 和 `sigma^2`。
- 最终候选表达式按 minimum in-sample RMSE 排序。
- 论文的输出不是单个确定表达式，而是一组 ranked symbolic expressions，作者将其解释为结构不确定性量化。

### 关键技术细节
- 实验中 `K=3` 棵树，每棵最大深度 `D=3`。
- 深度为 `D` 的 skeleton 节点数是 `N=2^(D+1)-1`。
- 实验 prior 设定为 `(alpha, delta)=(0.95, 2.00)`。
- 回归参数先验为 `mu_0=0`, `Sigma_0=10I`, `a_0=2`, `b_0=2`。
- operator 和 feature assignment 使用 uniform Dirichlet priors。
- VaSST 优化器是 AdamW，learning rate `5e-5`。
- AdamW betas 设置为 `(0.90, 0.99)`，weight decay 为 `0`。
- 训练步数为 `2000` steps。
- 每步 ELBO 的 Monte Carlo sample 数为 `S=8`。
- gradient clipping threshold 为 `c=1.0`。
- temperature annealing 在 `T_tau=1500` steps 内完成。
- 训练后 hard symbolic ensembles 抽样数为 `H=2000`。
- 所有方法使用 `90/10` train-test split。
- simulation 输入样本量为 `n=2000`。
- simulation 自变量为 `x_j ~ Unif(2j, 2j+1)`，`j=0,1,2`。
- simulation 测试两个真函数：`x0^2 - x1 + 0.5 x2^2` 和 `6 sin(x0) cos(x1)`。
- 噪声设置包括 noiseless、`sigma^2=0.1^2`、`sigma^2=0.2^2`。
- Feynman/SRBench 任务从每个方程的 `10^5` observations 中 subsample `n=2000`。
- Feynman 方程包括 CL, CPE, FCE, FTC 四个代表问题。
- baselines 包括 QLattice/Feyn、gplearn、DEAP、Bayesian Machine Scientist、Bayesian Symbolic Regression。
- gplearn 配置为 population size `2000`, generations `20`, tournament size `20`, parsimony coefficient `1e-4`。
- DEAP 配置为 `(mu+lambda)`，`mu=200`, `lambda=200`, generations `20`, crossover `0.5`, mutation `0.2`。
- BMS 使用 AutoRA 实现，训练 `2000` epochs。
- BSR 使用 AutoRA 实现，MCMC iterations 为 `2000`，树数与 VaSST 对齐。
- 作者限制 baseline 的 operator set 与 VaSST 一致，以减少 grammar 差异造成的不公平。

### 核心结果
- 在 simulation 1 噪声 `0.1^2` 下，VaSST 恢复 `0.0246 + 0.99 x0^2 - 1.01 x1 + 0.61 x2^2`。
- 同一设置下 BMS、BSR、QLattice、gplearn、DEAP 多数输出明显更复杂的表达式。
- simulation 1 out-of-sample RMSE noiseless: VaSST `0.002925 +/- 0.000661`，QLattice `0.005195 +/- 0.002953`。
- simulation 1 noise `0.1^2`: VaSST `0.100255 +/- 0.001524`，QLattice `0.101894 +/- 0.002699`。
- simulation 1 noise `0.2^2`: VaSST `0.200428 +/- 0.003076`，QLattice `0.202036 +/- 0.004099`。
- simulation 2 noiseless: VaSST `0.001679 +/- 0.000378`，BMS `0.002887 +/- 0.002660`。
- simulation 2 noise `0.1^2`: VaSST `0.099378 +/- 0.000920`。
- simulation 2 noise `0.2^2`: VaSST `0.198707 +/- 0.001831`。
- Feynman exact symbolic recovery 表中，VaSST 在 CL/CPE/FCE/FTC 的三种噪声设置全部打勾。
- BMS 在 CL/CPE/FCE 全部成功，但 FTC 全部失败返回无效表达式。
- gplearn 在 CL/CPE/FTC 成功，但 FCE 失败。
- QLattice 只在 CL 全部成功，FTC 仅 noiseless 成功，CPE/FCE 失败。
- BSR 和 DEAP 在四个 Feynman 任务全部 structural recovery 失败。
- CL noiseless 中 VaSST RMSE `4.67e-5`，BMS `0.0`，QLattice `4.27e-6`。
- CPE noiseless 中 VaSST RMSE `3.66e-3`，BMS `0.0`，gplearn `8.61e-16`。
- FCE noiseless 中 VaSST RMSE `1.30759e-3`，BMS `0.0`，其他 baseline 在十几到二十量级。
- FTC noiseless 中 VaSST RMSE `1.26016e-2`，QLattice `1.21977e-4`，gplearn `1.73076e-16`。
- 在 FTC `sigma^2=0.1^2` 下，VaSST 输出 `0.002 -0.943*kappa*A*T1/d +1.006*kappa*A*T2/d`，RMSE `1.00325e-1`。
- FTC 同一设置下 gplearn 输出精确结构 `kappa*A*(T2-T1)/d`，RMSE `1.00224e-1`。
- Bayesian runtime 表显示 VaSST 在四个 Feynman 方程和三种噪声设置下都比 BMS/BSR 快。
- 例如 FTC `sigma^2=0.1^2` runtime: VaSST `121.30s`, BMS `201.51s`, BSR `214.95s`。
- CL noiseless runtime: VaSST `85.19s`, BMS `107.33s`, BSR `186.44s`。
- 反直觉点：VaSST 并不总是最低 RMSE；在一些 Feynman 任务上 gplearn 或 QLattice 能给更低 RMSE。
- 作者的强 claim 更准确地说是 structural recovery 稳定性、Bayesian/UQ 框架和相对 Bayesian baseline 的速度，而非所有场景绝对 RMSE 第一。
- 另一个反直觉点：FCE 表中的 VaSST top expression 带有多余小项和系数 `1.068`，不是严格简洁的物理式，但仍被作者判为 exact recovery。

### 明确局限
- 论文没有真实物理实验或真实机器人/实验室闭环，只在 synthetic simulation 和 SRBench/Feynman 数据上验证。
- Feynman 数据是已知解析式 benchmark，不能完全代表真实科学发现中的 confounding、measurement bias 和 variable misspecification。
- evaluation 主要是作者自跑，没有独立复现实验或第三方 artifact audit。
- 代码链接是 anonymous 4open science 链接，不是长期稳定的正式 repository citation。
- `K=3, D=3` 限制了表达能力，适合本文 benchmark，但不清楚更深/更多树时的 scaling。
- hard tree sample 选取按 in-sample RMSE 排序，可能把 UQ 变成 posterior samples 加训练误差筛选的混合流程。
- mean-field VI 可能低估 posterior uncertainty，这一点论文没有系统校准。
- temperature annealing 到 `0.5` 后仍是软近似，不等价于完整离散后验推断。
- 作者比较了 BMS/BSR，但现代 symbolic regression 系统如 PySR 在主表中没有作为 baseline。
- Feynman 方程只取四个代表任务，不是完整 SRBench 全套 benchmark。
- 对 baseline 的调参空间披露有限，不能排除某些方法在更强 tuning 下缩小差距。
- 论文强调 UQ，但缺少 coverage、calibration、posterior predictive interval 等直接 UQ 指标。
- FCE 中 top expression 有明显附加项，说明 symbolic simplification 和 exactness 判定需要更严格规则。
- 对复杂表达式等价变换的 canonicalization 没有成为核心组件。
- 跟 0616-fex 的关系取决于该课题是否做 equation/formula extraction 或 symbolic discovery；若是文本/图像公式抽取，则本文只间接相关。

### 跟我们的关系
- 对 `0616-fex`，如果目标是从数据中发现公式、Feynman equation extraction、或可解释 scientific law discovery，VaSST 是直接相关工作。
- 撞车点 1：如果我们也主张“用 continuous relaxation 把 symbolic tree search 变成 gradient optimization”，本文已经覆盖核心 claim。
- 撞车点 2：如果我们也主张“Bayesian symbolic regression with uncertainty over structures”，本文也有明显重合。
- 撞车点 3：如果我们只做 deterministic symbolic search 或 LLM formula proposal，本文不是完全撞车，但会成为强 prior work。
- 可用工具：Binary Concrete/Gumbel-Softmax relaxation、depth-dependent split prior、NIG marginal likelihood、hard tree posterior sampling。
- 可用评测：simulation 的两个表达式、SRBench/Feynman 的 CL/CPE/FCE/FTC，以及 structural recovery + RMSE + runtime 三类指标。
- 差异化方向 1：把 UQ 从 ranked hard samples 扩展到 calibrated credible sets 和 coverage metrics。
- 差异化方向 2：加入 symbolic simplification/canonicalization，避免 FCE 这类“近似正确但带小项”的判定模糊。
- 差异化方向 4：从 benchmark equations 扩到真实 noisy scientific datasets，并加入 variable misspecification 和 unit constraints。
- 差异化方向 5：如果 `fex` 是 formula extraction from text/images，则本文可作为下游 formula fitting/discovery 模块，而不是前端 extraction 模块。
