# Physically interpretable interatomic potentials via symbolic regression and reinforcement learning (2511.20506)

- **Year**: 2025.11
- **Authors**: Bilvin Varughese 等, University of Illinois Chicago / Argonne National Laboratory / Oak Ridge National Laboratory
- **Venue**: arXiv
- **Keywords**: symbolic regression, interatomic potential, EqNN, reinforcement learning, continuous-action MCTS, EAM, DFT training data, copper
- **One-liner**: 用 EqNN+MCTS 学出可解释 Cu 原子势并优于 Sutton-Chen EAM。

## 精华提炼

### 管线/方法
- 论文目标是把传统预设函数形式的经验势，替换为从数据中学习出来的显式解析势函数。
- 任务对象是铜 Cu，作者把它作为代表性过渡金属来验证方法。
- 物理框架仍然保留 EAM 分解：总能量由 pair potential、electron density、embedding energy 三部分组成。
- 输入数据是原子构型的距离矩阵，以及每个结构的原子数。
- 距离矩阵只保留 cutoff 内的 pair distance，超出 cutoff 的 pair 被置零。
- 模型输出是每个结构归一化到每原子的总能量。
- 第一个阶段是验证 EqNN 是否能恢复已知 Sutton-Chen EAM 函数形式。
- 初始训练集约 21 个接近平衡 FCC 的结构。
- 之后用 nested ensemble Monte Carlo 在能量景观上采样新结构。
- 如果当前 EqNN 和 Sutton-Chen 参考模型在某个结构上的误差超过当前训练误差的 110%，该结构加入训练池。
- 这个 active learning 循环最终得到 341 个结构，足以恢复接近 Sutton-Chen 的解析式。
- 第二个阶段把这 341 个结构的能量重新用 DFT/VASP 计算，作为学习新 Cu 势函数的数据。
- 作者训练了两个符号回归势：SR1 和 SR2。
- SR1 固定 embedding 函数为 Sutton-Chen 风格的 sqrt(rho)，只让 pair 与 density 函数自由搜索。
- SR2 在 SR1 基础上扩展 embedding 函数搜索空间，允许 sqrt(rho)、rho、rho^2、rho^3、rho^4 等项。
- EqNN 由三个并行 symbolic neural networks 组成：Pair Potential Network、Density Network、Embedding Energy Network。
- Pair basis 包括 exp(-r)、r^-1、r^-6、r^-7、r^-8、r^-9、r^-10、r^-12、r^-14。
- Density basis 包括 exp(-r)、r^-1、r^-5、r^-6、r^-7、r^-8。
- SR1 embedding basis 只有 rho^0.5。
- SR2 embedding basis 包括 rho^0.5、rho、rho^2、rho^3、rho^4。
- 训练采用两阶段优化：先全局搜索，再局部梯度下降。
- 全局搜索用 continuous-action Monte Carlo Tree Search，即 c-MCTS。
- c-MCTS 的节点表示连续参数空间中的区域，而不是离散动作。
- 作者对 c-MCTS 做了三处改动：唯一性函数避免分支退化、树深度关联搜索半径、playout 对父节点附近区域自适应偏置。
- Cu 实验中参数搜索上下界设为 +/-5.0。
- 深度缩放率设为 0.5，深度上限 25，最大搜索半径 5.0。
- playout value 使用 10。
- 当全局搜索把 loss 降到约 150 meV/atom 后，切到局部优化。
- 另一个触发条件是 MCTS 到约 100 meV/atom 且 5000 步没有改进。
- 局部优化用 Adam，学习率 1e-4。
- SR2 报告在 55000 个 MCTS steps 后达到约 80 meV/atom loss，再用 10000 个 Adam steps 得到最终模型。
- 为了解决 basis 函数难以真正关掉的问题，作者引入 pseudo-weight 和 null zone。
- null zone 宽度固定为 +/-0.125，用于把小权重精确压成 0。
- loss 使用加权 MAE，并叠加 pair 负值、density 负值、NaN/Inf domain error 的大惩罚项。
- 近基态结构被赋予更高权重，因为它们影响 EOS、弹性常数等关键性质。
- 验证管线覆盖能量/力 parity、EOS、弹性常数、声子色散、缺陷形成能、表面能、Wulff 构型、晶界能、Bain path、热膨胀、多晶型 EOS、两相熔点、LAMMPS 性能。
- 熔点测试采用两相固液共存法，而不是简单升温熔化。
- 两相体系沿 z 方向堆叠固体和液体 slab，周期边界，NPT，1 bar。
- 液体半区先加热到约 2500 K，固体半区通过置零力保持晶体性。
- 共存模拟时间步长 2 fs，每个温度 1,000,000 steps，即 2 ns。
- 温度扫描范围是 700 K 到 1420 K。
- 界面漂移通过 FCC ordered atoms 与 amorphous atoms 比例判断。

### 关键技术细节
- Sutton-Chen 参考 pair term 是 1289.3050 / r^9。
- Sutton-Chen 参考 density term 是 527.6214 / r^6。
- Sutton-Chen 参考 embedding term 是 -sqrt(rho)。
- EqNN 恢复出的 Sutton-Chen-like pair term 是 1292.5470 / r^9。
- EqNN 恢复出的 density term 是 859.0663 / r^6。
- EqNN 恢复出的 embedding term 是 -0.7852 sqrt(rho)。
- 这个恢复实验说明 EqNN 能在 sparse active-learning 数据上找回已知函数族，但 density 与 embedding 系数存在尺度等价性。
- SR1 pair function 是指数包裹的多项反幂与指数项组合，形式比 Sutton-Chen 更复杂。
- SR1 density function 包含 r^-7、r^-6、r^-5 与 exp(-1.3632 r)。
- SR1 embedding function 是 -1.5598 sqrt(rho)。
- SR2 pair 与 density 基本延续 SR1，但参数微调。
- SR2 embedding function 是 -1.5788 sqrt(rho) - 0.0009 rho^2 + 0.0175 rho。
- SR2 的关键新增能力来自 embedding 函数在高 rho 区间的更强 many-body stabilization。
- 作者认为 SR2 在缺陷、表面、高配位或局部电子密度偏离 bulk 的区域更合理。
- DFT reference force 用 VASP 计算。
- potential-based force 用 LAMMPS 计算。
- 声子色散用 Phonopy finite-displacement 方法。
- 性能基准在 AMD EPYC 7763, 2.7 GHz 单 CPU core 上测量。
- SR1 运行速度是 4.50 microsecond/atom/step。
- SR2 运行速度是 4.41 microsecond/atom/step。
- Sutton-Chen EAM 是 3.60 microsecond/atom/step。
- 对比 MLIP：MTP 20.7、SNAP 40.9、qSNAP 41.3、MACE-Small 42、GAP 1512 microsecond/atom/step。
- 因此 SR 势只比经典 EAM 慢约 20-25%，但比常见 MLIP 快一个到三个数量级。

### 核心结果
- Sutton-Chen 恢复实验中，SR 模型 energy MAE 为 train 4.6 meV/atom、test 2.9 meV/atom。
- 同一实验中 force MAE 为 train 10.0 meV/Angstrom、test 14.0 meV/Angstrom。
- SC-EAM 在 DFT 数据上远离平衡时显著失效。
- SC-EAM 近基态 energy MAE 是 38.8 meV/atom。
- SC-EAM 远离基态 energy MAE 最高达到 1526.5 meV/atom。
- SC-EAM 近基态 force MAE 是 1200.5 meV/Angstrom。
- SC-EAM 远离基态 force MAE 最高达到 16539.0 meV/Angstrom。
- SR1 近基态 energy MAE 降到 4.8 meV/atom。
- SR1 近基态 force MAE 降到 147.8 meV/Angstrom。
- SR2 在非平衡结构上的 force error 从 SR1 的 1114.6 meV/Angstrom 降到 684.3 meV/Angstrom。
- SR1 uniaxial EOS MAE 为 5.26 meV/atom。
- SR1 volumetric EOS MAE 为 2.89 meV/atom。
- SR1 shear/tetragonal EOS MAE 为 5.4 meV/atom。
- SR1 对 C11、C12 的实验误差约 5% 和 10%，C44 误差约 20.9%。
- SR2 相对 DFT 的弹性常数误差为 C11 3%、C12 6.2%、C44 16%。
- 表中 DFT/SC/SR1/SR2/experiment 的 C11 分别是 180/170/166.94/186.21/176 GPa。
- C12 分别是 127/130/112.27/119.11/125 GPa。
- C44 分别是 79/58/64.84/66.40/82 GPa。
- SR1/SR2 的 vacancy formation energy 是 1.32/1.37 eV，接近实验 1.29 eV。
- Sutton-Chen vacancy formation energy 是 0.875 eV，低估约 32%。
- SR1/SR2 的 interstitial formation energy 是 3.08/3.02 eV，接近 DFT 2.902 eV。
- 表面能测试中 SR1/SR2 在低指数和高指数 facets 上都明显优于 SC-EAM。
- Wulff construction 中 SR1/SR2 复现了更接近 DFT 的 facet 类型和相对面积。
- 晶界能覆盖 24 个 Cu GB 几何，包括不同 Sigma、tilt/twist、rotation axis 与 GB plane。
- 多数 GB 上 SR1/SR2 相对 DFT 约在 5-10% 误差内。
- 一个重要 outlier 是高角 Sigma 3 tilt，DFT 为 0.85 J/m^2，而 SR1/SR2 约为 2.981/3.026 J/m^2。
- Bain transformation path 中 fcc 点 SR1/SR2 与 DFT 数值一致，SC 有约 5 meV/atom 偏移。
- Bain path 的 bcc barrier 中，DFT 约 0.029 eV/atom；SR1 偏高 3%，SR2 偏高 7%，SC 偏高 55%。
- 线性热膨胀系数 CTE 在 200-750 K 区间，SR1/SR2 距 SRM 736 实验参考约 5-10%。
- SC 的 CTE 高出实验约 40-70%。
- BCC/HCP polymorph EOS 中，SR1/SR2 在能量最小值附近与 DFT 差距小于约 1 meV/atom。
- SC 在这些 polymorph EOS 上约高出 3-4 meV/atom，误差约 15-20%。
- 两相熔点模拟中，SR1/SR2 预测 Cu 熔点约 1300 K。
- 实验 Cu 熔点是 1358 K，因此 SR1/SR2 偏差约 4.5%。
- SC-EAM 预测熔点约 1154 K，明显偏低。
- 反直觉/值得注意的点：SR2 比 SR1 更复杂，但单核 LAMMPS 速度反而略快，可能是实现和表达式代价的细节导致，不能简单解读为复杂度更低。
- 另一个值得注意的点：SR1/SR2 仍是显式 EAM-like 解析势，而不是黑箱神经势，却能在很多非平衡测试上接近 DFT。

### 明确局限
- 论文只在单元素 Cu 上做了完整展示，不能直接证明多组元合金、离子体系、共价体系或反应体系的泛化。
- 虽然标题强调 physically interpretable，但最终表达式含嵌套指数和多项反幂；它是解析透明，不一定每个系数都有清晰化学含义。
- 该方法仍然保留 EAM 分解，因此对显式电荷转移、强方向性键、化学反应的表达能力有限。
- SR1/SR2 的训练数据来自 341 个结构的 DFT 重新标定，规模相对小；这有利于效率，但也限制了覆盖面。
- 表面结构被纳入训练，所以表面能与 Wulff 测试不是完全独立的外推验证。
- 作者报告了大量验证任务，但没有看到第三方独立复现或跨代码 artifact 验证。
- Data availability 写的是代码、脚本、框架和数据可向作者索取；训练结构数据有 GitHub 链接，但不是完整开源 artifact。
- DFT 设置、cutoff 半径、具体结构划分等复现实操细节在 tex 中不够完整，需要仓库或作者数据补足。
- GB 表中 Sigma 3 tilt outlier 很大，说明局部环境或结构弛豫路径仍可能存在失配。
- 文中 stacking fault energy 的表格与正文叙述看起来有不一致：正文说 SR1/SR2 约 43.5/43.3 mJ/m^2，表中相关行显示 49.84/66.40，需要核查原始数据。
- SR2 的改进主要来自扩展 embedding basis；但消融比较有限，不能完全分离 basis choice、优化预算、初始化继承的贡献。
- MCTS 搜索成本没有给出完整 wall-clock 训练代价，实际开发一个新体系势函数的成本仍不清楚。
- 论文没有把 SR 势放到统一开放 MLIP benchmark 上做 blind test。
- 由于文章是 2025-11 arXiv 预印本，S2 反引文截至本次工具查询只有 2 篇，社区验证还很早。

### 跟我们的关系
- 对 `0616-fex`，如果课题关注 force-field expression extraction、formula extraction 或可解释函数学习，这篇是强相关且可能局部撞车的 prior。
- 直接撞车的 claim 是：用 symbolic regression / equation learner network 从 DFT 数据学习显式 interatomic potential。
- 另一个撞车点是：把强化学习式搜索，即 continuous-action MCTS，与 EqNN/符号表达式参数搜索结合。
- 若我们的 claim 只是“用 LLM 或 agent 做材料势函数自动审阅/生成”，它不完全撞车，但必须引用这篇作为可解释势函数学习的近邻工作。
- 若我们的 claim 是“从少量 DFT 数据自动发现可部署的解析 EAM 势”，这篇已经做得很接近，需要明确差异化。
- 可复用的工具链是：nested ensemble sampling 生成覆盖能量景观的数据、EAM-like EqNN 分解、pseudo-weight null zone、c-MCTS+Adam 混合优化。
- 可复用的评测清单是：EOS、elastic constants、phonon、defects、surfaces/Wulff、GB、Bain path、CTE、polymorph EOS、two-phase melting、LAMMPS speed。
- 差异化方向一：从单元素 Cu 扩展到多组元、反应性或电荷转移体系，并诚实评估 EAM 分解是否不够。
- 差异化方向二：引入独立 verifier 或自动 proof/checker，把表达式生成和验证严格分离，提高 R2。
- 差异化方向三：公开完整 artifact、训练日志、held-out blind benchmark，主打 reproducibility。
- 差异化方向四：不仅输出解析势，还输出 claim-level provenance，说明每个方程项由哪些结构族和验证任务支撑。
- 该引。它是 2025 年很近的相关工作，尤其覆盖 symbolic regression、RL search、interpretable interatomic potentials 三个关键词。
