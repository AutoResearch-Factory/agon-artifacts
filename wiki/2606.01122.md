# A Per-Component Diagnostic Protocol for Neural HJB-PIDE Solvers under Control-Dependent L\'evy Jumps (2606.01122)

- **Year**: 2026.06
- **Authors**: R. Drissi
- **Venue**: arXiv
- **Keywords**: HJB-PIDE, neural PDE solver, L\'evy jumps, Variance Gamma, CRRA portfolio, per-component diagnostic, finite-difference reference, importance sampling
- **One-liner**: 用逐项诊断抓出神经 HJB-PIDE 的半积分 bug。

## 精华提炼

### 管线/方法
- 论文目标不是提出新的神经网络架构，而是提出一个用于审计残差训练神经 HJB-PIDE 求解器的五步诊断协议。
- 核心对象是带控制依赖 L\'evy 跳的财富控制问题：跳位移为 `x -> x(1 + u z)`，所以非局部积分和控制变量直接耦合。
- 输入问题为 CRRA Merton-Variance-Gamma 投资组合 benchmark，状态是财富 `X_t`，控制是风险资产比例 `u_t`。
- 动态为 `dX_t/X_t = (r + u_t(mu-r))dt + u_t sigma dW_t + u_t dJ_t`，其中 `J_t` 是补偿纯跳 L\'evy 过程。
- HJB-PIDE 的 Hamiltonian 被拆成漂移项、扩散项、补偿项、非局部 L\'evy 积分项。
- 诊断协议第一步要求所有求解器使用同一方程、同一控制集、同一 L\'evy 截断域。
- 第二步要求至少一个从零写的独立参考实现，且坐标、时间推进、积分规则、argmax、边界处理尽量不同。
- 第三步要求在 `u` 网格上逐项比较 Hamiltonian，而不是只看最终最优控制。
- 第四步要求先在 `(t,x)` 网格上比较 `V`、`V_x`、`V_xx`，再比较 argmax 控制。
- 第五步要求最后才信任 headline 控制；如果某一项跨 `u` 恒定偏一倍，优先怀疑 proposal density 或归一化尺度错误。
- 神经求解器使用两个网络：`V_theta(t,x)` 近似值函数，`u_phi(t,x)` 近似控制策略。
- 输入包含时间编码和正财富状态，隐藏层为 128 hidden units 的 residual blocks，激活函数 GELU。
- 训练损失含三项：HJB residual、terminal condition、argmax-target policy loss。
- policy loss 不是用无约束一阶条件，而是在 `K_u=21` 个控制候选上离散求 Hamiltonian argmax，再让策略网络拟合该 argmax。
- 这避免了最优控制在边界时 `partial_u H = 0` 不成立的问题。
- 非局部 L\'evy 积分用截断域 `|z| in [0.01,0.99]`，上界小于 1 是为了保证 long-only `u in [0,1]` 下 `1+uz>0`。
- Variance Gamma 使用 Madan 参数 `sigma_VG=0.2, theta=-0.1, nu=0.3` 映射到 `(C,M,G)` L\'evy 密度。
- 积分估计用 50/50 两侧 shifted exponential proposal，并用 `w(z)=nu(z)/q(z)` 做 importance weighting。
- 论文发现原实现把每侧条件密度 `q_pm` 当成总体 proposal density，漏掉 50/50 mixture 的 `1/2` 因子。
- 漏掉该因子会让 importance weights 正好变成真实值的一半，从而把非局部积分估计缩小为 `0.5 * integral`。
- 这个 bug 不能靠更长训练、更细网格、更宽网络、截断 sweep 或 loss 曲线发现。
- bug 是通过逐项 Hamiltonian 比较发现的：漂移和扩散匹配 FD，补偿项和 L\'evy 积分项恒定为 FD 的 `0.502 +/- 0.001`。
- 参考实现包括 FD v1、FD v2、半解析 scalar baseline。
- FD v1 是 log-wealth 坐标、implicit Euler、trapezoidal quadrature、linspace argmax。
- FD v2 是独立重写：linear-x 坐标、Crank-Nicolson、composite Simpson、Brent argmax、显式边界检查。
- scalar baseline 来自常系数 CRRA-L\'evy 问题的 homogeneity collapse：PIDE 降为一维 `F(u)` 最大化。
- 因此该 benchmark 本身并不需要神经求解器；最有效方法是 scalar quadrature + bounded scalar maximization。
- 作者很明确地把贡献限定为诊断协议，而不是高维 HJB 理论、神经架构、强 empirical policy claim 或 viscosity convergence proof。

### 关键技术细节
- 基准参数：`gamma=2`, `mu=0.08`, `sigma=0.2`, `r=0.02`, `T=1`。
- 控制集：`U=[0,1]`，long-only、无杠杆。
- VG 参数：`sigma_VG=0.2`, `theta=-0.1`, `nu=0.3`，负偏跳风险。
- 截断：`z_min=0.01`, `z_max=0.99`，同时服务于小跳截断和破产保护。
- 神经训练：`N_z=64` L\'evy MC samples per gradient step。
- batch size 为 256，训练 500 epochs。
- warm-up 为 100 epochs，初期关闭 L\'evy term 并加强 terminal condition。
- optimizer 为 Adam，learning rate `1e-3`，cosine schedule。
- 梯度裁剪 norm 1.0。
- 报告主数值通常为 5 seeds：`{42,123,999,7,2024}`。
- 2D 例子用 3 seeds：`{42,123,999}`。
- diffusion-only Merton ratio 为 `0.75`，神经结果 `0.762 +/- 0.011`，平均相对误差 `1.6%`。
- VG post-fix 神经最优控制为 `0.344 +/- 0.006`，相对 Merton 降低约 `54.1%`。
- scalar baseline 的 VG 最优控制为 `0.3521`。
- FD v1 refined 结果为 `0.347`。
- FD v2 independent reimplementation 结果为 `0.344`。
- post-fix 四方参考在最优控制上约 `2.3%` 内一致。
- pre-fix 神经 VG 控制约 `0.475 +/- 0.008`，与 FD 约有 `26%` 差距。
- 修复后 MC 估计 `int z nu(dz)` 约 `-0.098`，与 FD 的 `-0.097` 一致；修复前约 `-0.049`。
- surface check 在 `8 x 21` 网格上评估 `t in [0,0.95]`, `x in [0.5,1.5]`，共 168 点。
- 神经 `V` 相对 sup error 为 `0.60%`。
- 神经 `V_x` 相对 sup error 为 `3.88%`。
- 神经 `V_xx` 相对 sup error 为 `26.4%`。
- 神经 policy field 相对 scalar constant policy 的 sup error 为 `14.9%`。
- FD 在同类 surface metrics 上均低于 `0.05%`。
- 时间成本：1D FD-PIDE VG legacy grid 约 `11.3s`，audit-finest FD grid 约 `800s`。
- 神经 VG 每 seed 约 `170-220s`。
- scalar baseline 在 1D 用 4001 Simpson nodes per side 时约 `0.12s`。
- 2D coupled example 中资产 1/2 的神经 post-fix 控制为 `0.348 +/- 0.005` 和 `0.258 +/- 0.005`，现金约 `0.394 +/- 0.005`。

### 核心结果
- 最重要结果不是神经求解器达到 SOTA，而是一个诊断协议成功抓到了真实实现 bug。
- 该 bug 是 importance proposal density 的常数尺度错误，导致 L\'evy 积分被精确减半。
- 单一 headline 指标会误导：diffusion-only Merton 诊断通过，但 jump operator 仍然错。
- 单点控制也会误导：`u(0,1)` 接近参考值不代表 `V_xx` 或 policy surface 正确。
- post-fix 后，神经、两个 FD、scalar baseline 对 VG 最优控制给出同一数量级结论：负偏跳使最优风险暴露从约 `0.75` 降到约 `0.34-0.35`。
- 表面精度显示神经法在该常系数 1D 问题上明显劣于 FD，尤其是二阶导数。
- 作者没有隐瞒这一点：论文明确说此 benchmark 中 FD 和 scalar baseline 更合适。
- interior-optimum ablation 中，full neural 为 `0.2774 +/- 0.0038`，FD reference 为 `0.2764`，差距约 `0.4%`。
- interior ablation 去掉 compensation 后控制坍缩到 `0.0056 +/- 0.0020`，说明补偿项不仅是数学形式，还实际决定策略。
- no importance weighting 在 interior benchmark 中把控制从 `0.277` 移到 `0.243`，约 `12%` 变化。
- no weight clipping 几乎不动，说明 clipping 是数值稳定器，不是主要机制。
- S&P 500 MLE calibration 只是 sanity check，不是策略收益声明。
- S&P long-only post-fix 中 diffusion policy 为 `0.997`，VG policy 为 `0.949`，尾部指标改善约 1 个百分点，效果很小。
- 2D coupled portfolio 只是轻量示范，不支持高维 scaling claim。

### 明确局限
- 只在 Variance Gamma 一个主要 L\'evy specification 上完整展示，没有覆盖 NIG、CGMY、compound-Poisson 之外的广泛分布。
- 主要 CRRA-Merton-VG benchmark 由 homogeneity collapse 降为一维 scalar maximization，因此并不是神经 HJB-PIDE 真正必要的场景。
- 没有 viscosity-solution convergence proof。
- 没有新的 `z_min -> 0` 或 `N_z -> infinity` 收敛 sweep。
- 神经 solver surface accuracy 明显弱于 FD，`V_xx` 误差高达约 `26%`。
- S&P calibration 使用用户自备数据，原始指数数据不随 supplement 分发，完全复现 empirical calibration 需要外部合法数据源。
- 2D 例子仍是 3 seeds、500 epochs 的小规模示范，不能证明高维可扩展。
- 当前跳过程主要是 1D 或 independent jumps，多资产相关跳还没有处理。
- 所有独立参考都来自同一作者/同一代码发布上下文，独立性主要是实现层面的，不是不同实验室复现。
- 对 0616-fex 这类后续课题，最可用的是审计协议和 failure-mode 叙事，而不是把该神经求解器作为强 baseline。
