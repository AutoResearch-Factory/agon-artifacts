# Automated Discovery of Conservation Laws via Hybrid Neural ODE-Transformers (2511.00102)

- **Year**: 2025.11
- **Authors**: Vivan Doshi, Independent Researcher, San Jose, CA
- **Venue**: NeurIPS 2025 MATH-AI Workshop (5th Workshop on Mathematical Reasoning and AI)
- **Keywords**: conservation laws, Neural ODE, Transformer, symbolic regression, PPO, verification, hybrid architecture, dynamics learning
- **One-liner**: 用 Neural ODE 学连续动力学 → Transformer (PPO fine-tune) 生成符号候选守恒量 → 数值验证器过滤，显著优于直接在噪声轨迹上做符号回归的 baseline。

## 精华提炼

### 管线/方法

完整的 3 模块 pipeline：

1. **Module 1: Neural ODE 动力学学习**
   - 输入：含噪声的轨迹数据 {z(t_i)}（带 2% Gaussian noise）
   - 用 MLP 网络 f_θ 逼近未知向量场
   - 使用 adaptive-step ODE solver (Dopri5) 做 forward simulation
   - 用 adjoint sensitivity method 做反向传播（常数内存）
   - 输出：连续 denoised 向量场模型 f_θ(z)

2. **Module 2: Transformer 符号候选生成**
   - 输入：Module 1 产生的 state-derivative pairs (z, f_θ(z))
   - 两阶段训练：
     - Pre-training：大规模数学表达式语料上学 syntax prior（变量：x,y,v_x,v_y；算子：+,-,*,/,sin,cos,pow）
     - Fine-tuning：用 PPO（RL）生成候选 Ĉ(z)
   - Reward function：R(Ĉ) = exp(-λ₁ · err) + λ₂ · ||∇Ĉ||₂
     - err = mean squared invariance error over batch
     - λ₂ ||∇Ĉ||₂ = 非退化惩罚项，防平凡解（如 Ĉ ≡ 0）
   - 输出：符号候选不变量 Ĉ(z)

3. **Module 3: 符号-数值验证器**
   - 用 SymPy 精确计算 ∇_z Ĉ(z)
   - 在训练数据 convex hull 内均匀采样 10,000 个 grid points
   - 数值评估 |∇_z Ĉ(z) · f_θ(z)|
   - 若 max < 10⁻⁶ → 认证为 learned model 的不变量
   - 高度并行化（100 points 并行）

### 关键技术细节

- **Neural ODE**: 4-layer MLP, 128 hidden units, Swish activations, PyTorch + torchdiffeq
- **训练**: Adam, lr=10⁻³, batch size 64, 200 epochs, 目标 validation MSE < 10⁻⁵
- **Transformer**: 6-layer, 50 epochs PPO fine-tuning
- **Baselines**: (1) PySR (genetic symbolic regression), (2) End-to-End Transformer (直接在原始轨迹上操作)
- **测试系统**: 3 个 canonical system —— harmonic oscillator (2D), pendulum (2D), Kepler two-body (4D)
- **成功率判定**: 表达式与 ground truth 功能等价 + 非平凡 + RMSE 阈值达标
- **硬件**: 单张 NVIDIA RTX 3090 (24GB), 每个实验 3-6 小时 wall-clock
- **Reward 超参**: λ₁ 和 λ₂ 未给出具体数值（仅描述了公式形式）
- **Noise sweep**: 0-10%，harmonic oscillator 上 10% noise 仍保持 >70% 发现率
- **失败分析**: Neural ODE 欠拟合时（val MSE > 10⁻³），Transformer 产生虚假不变量 —— 对错误的 f_θ 成立但对真实系统偏差大（如 pendulum 上 Ĉ = 0.8(p²+q²) + 0.3 sin(q) 偏 15%）

### 核心结果

| System | PySR | End-to-End Transformer | Ours (Hybrid) |
|--------|------|----------------------|---------------|
| Harmonic Oscillator (Energy) | 75% [51,91] | 60% [36,81] | **95% [75,100]** |
| Pendulum (Energy) | 60% [36,81] | 55% [32,77] | **90% [68,99]** |
| Kepler Problem (Energy) | 15% [3,40] | 5% [0,25] | **70% [46,88]** |
| Kepler Problem (Ang. Mom.) | 20% [6,44] | 10% [1,32] | **80% [56,94]** |

- 一致大幅超越 baseline，尤其在复杂系统 (Kepler) 上差距最明显（15%→70%, 5%→70%）
- Ablation: 移除 Neural ODE 模块后性能急剧下降，确认了 denoising 的关键性
- Ablation: 移除 Transformer pre-training 也降性能 —— 语法先验很重要
- Lorenz 混沌系统初步实验：45% 发现耗散关系，但未能发现两个二次不变量

### 明确局限

**作者明确说的**：
1. 依赖学出准确的 ODE 模型，stiff/混沌系统上困难
2. 发现的不变量是 learned model f_θ 的不变量，不是真实系统的不变量 —— 量化此 gap 是关键挑战
3. 仅在低维、well-behaved 系统上实验（harmonic oscillator, pendulum, Kepler）
4. 验证模块计算量大（10,000 grid points），虽然可并行化
5. 代码未公开（"available upon reasonable request"）
6. 未讨论 broader impact

**我读出来的额外局限**：
7. 单作者、独立研究者，无机构背书 —— peer review 强度未知（workshop paper）
8. 无 formal guarantee —— 验证器只给 numerical certificate（< 10⁻⁶），不是 formal proof
9. 只测了 synthetic data，无真机实验/真实测量数据
10. Transformer 的 reward 超参 λ₁, λ₂ 未披露具体数值
11. 只测了 3 个 toy 系统 + 1 个混沌系统初步实验 —— 覆盖面窄
12. 与 HNN 等 physics-informed 方法的对比不充分（只讨论，未实验对比）

### 撞车评估
**不撞。** 这篇是守恒律发现的特定应用（Neural ODE + Transformer + Verifier），不是通用符号回归方法。FEX 是通用 SR 框架（RL-based），定位和范围不同。唯一的交叉点在于都用 RL (PPO) 生成符号表达式，但在具体机制、适用范围、理论基础上完全不同。

### 可用的工具/方法
1. **"Learn-then-Search" 范式**：先用 NN 学一个 denoised proxy model，再在其上做符号搜索 —— 这对 FEX 应用于含噪声数据时有直接参考价值。FEX 可以加一个 Neural ODE 或简单的 NN regressor 作为前置 denoiser。
2. **PPO reward 结构**：R = exp(-λ₁·err) + λ₂·||∇Ĉ||₂ 中的非退化项值得借鉴到 FEX 的 reward 设计。
3. **数值验证器设计**：10,000 点 grid sampling + gradient dot-product check 作为快速候选筛选 —— FEX 可以加类似模块减少假阳性。
4. **Expression pre-training**：在大表达式语料上预训练 Transformer 学语法先验 —— FEX 的 tree-LSTM/RL 组件也可受益。

### 差异化
- FEX 定位更通用，不限于物理系统/守恒律
- FEX 有 Haizhao Yang 组的理论和实验积累
- 这篇是单作者 workshop paper，权威性有限
- 可以作为 "neural-symbolic hybrid SR" 相关工作中一个低优先级引用
