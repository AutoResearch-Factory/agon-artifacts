# On Global Convergence Rates for Federated Softmax Policy Gradient under Heterogeneous Environments (2505.23459)

- **Year**: 2025.05
- **Authors**: Safwan Labbi (CMAP, CNRS, École Polytechnique), Paul Mangold (CMAP, CNRS, École Polytechnique), Daniil Tiapkin (CMAP & LMO, Paris-Saclay), Eric Moulines (MBZUAI & LRE EPITA)
- **Venue**: AISTATS 2026 (accepted)
- **Keywords**: federated reinforcement learning, policy gradient, softmax parameterization, global convergence, Łojasiewicz inequality, heterogeneity, entropy regularization, FedAVG
- **One-liner**: 首次证明异构环境下 Federated Softmax Policy Gradient 的全局收敛性，发现联邦目标不满足单智能体的 Łojasiewicz 条件，并通过局部正则性分析给出显式收敛速率。

## 精华提炼

### 管线/方法

整体管线分为三层抽象：

1. **通用 FedAVG 分析框架 (proj-FedAVG)**：定义在一般非凸优化问题上，每个 client c 通过有偏随机梯度 oracle `g_c^z(θ)` 获取局部函数的梯度估计。算法流程：server 广播 `θ^r` → 每个 agent 执行 H 步局部 SGD → server 聚合 `θ̄^{r+1} = (1/M) Σ θ_c^{r,H}` → 应用 improvement operator `T` 得到 `θ^{r+1} = T(θ̄^{r+1})`。

2. **S-FedPG (无正则化)**：将 proj-FedAVG 实例化到 FRL，`T = Id`，局部目标是 agent 的 value function `J_c(θ)`，stochastic gradient 用 REINFORCE 估计器（batch B 条长度为 T 的截断轨迹），梯度公式如 (eq 37)。

3. **RS-FedPG (熵正则化)**：局部目标替换为 `J̃_{c,λ}(θ) = J_c(θ) + λ·H_c(θ)`（熵正则化），引入 projection operator `T_τ` 在策略空间阻止策略过于 deterministic，保证 `π_θ(a|s) ≥ τ`。`T_τ = L ∘ U_τ`，其中 `U_τ` 在 policy space 操作，`L` 将 policy 映射回 logit space。

输入：各 agent 独自的 MDP `(S, A, γ, P_c, r_c, ρ)` → 局部 REINFORCE 梯度 → 局部 SGD 步 → 联邦平均 → projection → 全局 softmax policy → 重复。

### 关键技术细节

- **模型假设**：tabular MDP，softmax parameterization `π_θ(a|s) = exp(θ(s,a)) / Σ_{a'} exp(θ(s,a'))`，agent 间 dynamics 和 reward 可不同
- **异构度量**：
  - `ε_P = max ‖P_c(·|s,a) - P_c'(·|s,a)‖_1`（transition kernel 异构）
  - `ε_r = max ‖r_c - r_c'‖_∞`（reward 异构）
- **核心正则性常数（无正则化）**：
  - Gradient bound: `L_1 ≈ (1-γ)^{-2}`
  - Smoothness: `L_2 ≈ 8/(1-γ)^3`
  - Third derivative bound: `L_3 ≈ 480/(1-γ)^{-4}`
  - Heterogeneity: `ζ² ≈ 56ε_P²/(1-γ)^6 + 36ε_r²/(1-γ)^4`
  - Bias (截断): `β ≈ 2γ^T T/(1-γ) + 2γ^T/(1-γ)^2`
  - Variance: `σ₂² ≈ 12/(B(1-γ)^4)`, `σ₄⁴ ≈ 1120/(B²(1-γ)^8)`
- **核心正则性常数（熵正则化）**：所有常数乘以 `(1 + λ log|A|)` factor
- **单智能体 Łojasiewicz**：
  - 无正则化：`‖∇J_c(θ)‖² ≥ 2μ_c(θ)[J_c^* - J_c(θ)]²`（quadratic PL）
  - 正则化：`‖∇J̃_c(θ)‖² ≥ 2μ̃_c(θ)[J̃_c^* - J̃_c(θ)]`（linear PL）
- **关键发现**：这些局部 PL 条件 **不对全局联邦目标成立**（Lemma: counterexample），因此传统 FedAVG 在 PL 条件下的分析（demidovich2025, haddadpour2019）不适用
- **突破方案**：从局部 QL/PL 导出全局 quasi-QL/PL：`ζ² + ‖∇F(θ)‖² ≥ μ_min(F* - F(θ))²`（QL 情况）或 `ζ² + ‖∇F(θ)‖² ≥ 2μ̃_min(F* - F(θ))`（PL 情况）
- **Descent Lemma**（Lemma, general）：对任意 `η H L_2 ≤ 1/6`，有
  ```
  F(θ^r) - E[F(θ̄^{r+1})|F^r] ≲ -ηH/4·‖∇F(θ^r)‖² + η²L_2Hσ₂²/M + ηHβ² + η³L_2²H³ζ² + η⁵L_3²H³σ₄⁴
  ```
- **S-FedPG 收敛**（Theorem）：`E[J* - J(θ^R)] ≲ O(1/R) + O(ε_P/((1-γ)³μ_min^{1/2})) + O(ε_r/((1-γ)²μ_min^{1/2})) + ...`，其中残差 floor 由异构 + bias + variance 组成
  - 通信轮数：`R ∼ O(1/ε)`（sublinear）
  - 每 agent 样本数：`R H B ∼ O(1/(M ε³))`（需要 ε > ε_min 的 floor）
  - 线性加速：`M ≲ min(1/(μ_min^{1/2}ε(1-γ)²), 1/(μ_min B (1-γ)⁴ε²))`
- **RS-FedPG 收敛**（Theorem）：`E[J̃* - J̃(θ^R)] ≲ (1-ηHμ̃_min/2)^R + residual`
  - 通信轮数：`R ∼ O(log(1/ε))`（linear convergence in communication!）
  - 每 agent 样本数：`R H B ∼ O(log(1/ε)/(M ε))`
  - 线性加速：`M ≲ min(1/(μ̃_min^{1/2}ε^{1/2}(1-γ)^{3/2}), 1/(μ̃_min ε B (1-γ)⁴))`
- **Projection operator `U_τ`**（仅 RS-FedPG）：
  - 阈值 `τ_λ = min(⅓·exp(-(16+8γλlog|A|)/(λ(1-γ)²ρ_min)), 1/(3⁸|A|⁴))`
  - 保证 `π(a|s) ≥ τ_λ/2` 且 `J̃(T_τ(θ)) ≥ J̃(θ)`
  - 这使得 PL 常数有下界：`μ̃_min = λ(1-γ)ρ_min²τ_λ²/|S|`
- **Sufficient exploration 假设**：`ρ_min = min_s ρ(s) > 0`（初始分布全覆盖）

### 核心结果

| 指标 | S-FedPG (无正则化) | RS-FedPG (熵正则化) | Prior work (Fed-Q/FedPG) |
|------|---------------------|----------------------|--------------------------|
| 全局收敛 | ✅ (首次) | ✅ (首次) | ❌ (仅 stationary point) |
| Last-iterate | ✅ | ✅ | ❌ |
| 通信轮数 | O(1/ε) | O(log(1/ε)) | O(1/ε) |
| 样本复杂度 | O(1/(Mε³)) | O(log(1/ε)/(Mε)) | O(1/(Mε^{3/2})) |
| 异构 floor | 有 (ε_min > 0) | 有 (ε_min > 0) | N/A |
| 线性加速 | ✅ (有限范围) | ✅ (有限范围) | ✅ |

**Surprising/反直觉发现**：
1. 单智能体 RL 中始终存在最优 deterministic 策略，但在异构 FRL 中 **不成立**：最优策略可能是 stochastic 甚至是 non-stationary/history-dependent 的
2. 具体构造了反例证明：`max_{π∈Π_det} J(π) < max_{π∈Π_sta} J(π) < max_{π∈Π_ℓ} J(π)`
3. 这一发现意味着 Fed-Q-learning 等基于 deterministic policy 的方法在异构 FRL 中 **provably suboptimal**
4. 只有 transition kernel 异构才导致此现象 —— 仅 reward 异构不改变最优策略的 deterministic 性质

**实验验证**（GridWorld 3×3 + Synthetic 5-state）：
- 验证了 linear speedup：2/10/50 agents 都表现出一致的收敛加速
- RS-FedPG 在异构环境下显著优于 S-FedPG 和 Fed-Q-learning
- 高异构场景（ε_P ≫ 0.3）下 Fed-Q-learning 的劣化与理论预测一致

### 明确局限

1. **Tabular MDP 限制**：全部分析基于 tabular setting，未涉及 function approximation 或 deep RL
2. **Sufficient exploration 假设强**：`ρ_min > 0` 确保了 PL 常数的下界，但实践中初始分布可能不覆盖全部状态
3. **无正则化时的 uniform minorization 条件**：S-FedPG 需要 `inf_r min_s π_{θ^r}(a*(s)|s) ≥ μ_min > 0` 几乎必然成立，这是一个强假设，作者承认"restrictive but unavoidable in our setting"
4. **残差 floor 未消除**：即使 ε → 0，异构 bias floor 始终存在（`∼ ε_P/(1-γ)^³ + ε_r/(1-γ)²`），没有提出消除异构 bias 的方法
5. **REINFORCE 估计器方差大**：使用 vanilla REINFORCE，未使用 baseline/actor-critic 等方差缩减技术
6. **实验规模小**：tabular 环境仅 5-8 states，4 actions，未在更大规模环境中验证
7. **无真机实验**：纯模拟，无可扩展性验证
8. **未来工作明确提到需要 Scaffold-like bias correction**：解决异构 bias 的方向 → 可能与 Karimireddy et al. (2020) 的 SCAFFOLD 结合
9. **与我们的关系**：没有涉及 PDE/foundation model/operator learning，纯 RL theory
