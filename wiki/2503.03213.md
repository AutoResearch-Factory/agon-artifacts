# Convergence Rates for Softmax Gating Mixture of Experts (2503.03213)

- **Year**: 2025.03
- **Authors**: Huy Nguyen, Nhat Ho$^{\star}$, Alessandro Rinaldo$^{\star}$ (The University of Texas at Austin; $\star$ co-last authors)
- **Venue**: arXiv preprint (ICML 2025 format)
- **Keywords**: Mixture of Experts, softmax gating, convergence rates, strong identifiability, Voronoi loss, dense-to-sparse gating, hierarchical MoE, parameter estimation, expert estimation, PDE interaction
- **One-liner**: Establishes convergence rates for MoE expert estimation under softmax/dense-to-sparse/hierarchical gating, revealing that strongly identifiable experts (e.g. FFN) need polynomially many samples while linear experts require exponentially many due to PDE-expressed parameter interactions.

## 精华提炼

### 管线/方法

**整体框架**：将 MoE 建模为回归问题 Y = f_G*(X) + ε，然后用最小二乘法估计 mixing measure G*，通过分析 mixing measure 的收敛速率来确定 expert estimation 的样本效率。

1. **输入**：i.i.d. 样本 (X_i, Y_i)，X ∈ R^d, Y ∈ R，噪声 ε_i|X_i ~ N(0, ν)
2. **回归函数**：f_G*(x) = Σ softmax(ω_i^T x + β_i) · E(x, η_i)，其中 E(x, η) 是 expert 函数
3. **过参数化**：用 k > k_* 个 expert 的最小二乘估计器，Ĝ_n = argmin Σ(Y_i - f_G(X_i))²
4. **核心分析工具**：
   - Taylor 展开分解回归差异 f_G(x) - f_G*(x)
   - **Voronoi loss** 函数：基于 Voronoi cells 度量参数估计的差异，能区分 exactly-specified 和 over-specified 参数的估计速率
   - Fatou 引理用于反证法
   - Le Cam 引理用于 minimax lower bound
   - Bracketing entropy 用于建立回归函数估计的收敛速率

### 关键技术细节

**模型假设 (A.1-A.4)**：
- (A.1) 参数空间 Θ 紧致，输入空间 X 有界
- (A.2) ground-truth expert 参数 η* 互不相同，expert 函数有界且 Lipschitz
- (A.3) 最后一对 gating 参数为零 (β*_k = 0, ω*_k = 0d)，防止 softmax 平移不变性
- (A.4) 至少一个 ω* 非零，保证 gating 输入依赖

**Strong Identifiability 条件 (Definition 3.1)**：
- Expert 函数 E(x, η) 需对 η 二阶可微，且集合 {x^ν · ∂^|ρ|E/∂η^ρ(x, η_j) : 0 ≤ |ν|+|ρ| ≤ 2} 线性独立
- **满足的专家**：FFN with GELU/sigmoid/tanh 激活 + 归一化输入 E(x,(a,b,c)) = cσ(a·x/‖x‖ + b)
- **违反的专家**：多项式激活 σ(z) = z^p，以及线性专家 a^Tx+b

**Algebraic Independence 条件 (Definition 4.2，用于 dense-to-sparse gating)**：
- Router π(x,ω) 和 expert E(x,η) 的集合 {E, πE, ∂π/∂ω·E, (∂π/∂ω)(∂π/∂ω)E, ∂²π/∂ω²E, π·∂π/∂ω·E, ∂E/∂η, π·∂E/∂η, (∂π/∂ω)(∂E/∂η), ∂²E/∂η²} 线性独立
- **满足的配对**：π(x,ω) = σ₁(ω^Tx) 和 E(x,η) = cσ₂(a·x/‖x‖+b)，其中 σ₁,σ₂ 是不同的非线性激活函数 (GELU/sigmoid/tanh)
- **违反的配对**：σ₁ 和 σ₂ 都是多项式形式

**Voronoi Loss 函数**：
- L1(G,G*) 用于 standard MoE + strongly identifiable experts
- L2,r(G,G*) 用于 standard MoE + linear experts（含任意 r≥1 的 r-范数）
- L3,r(G,G*) 用于 dense-to-sparse MoE + linear router
- L4(G,G*) 用于 dense-to-sparse MoE + algebraically independent router/expert
- L5(G,G*) 用于 hierarchical MoE + strongly identifiable experts
- L6,r(G,G*) 用于 hierarchical MoE + linear experts

**参数交互 PDE**：
- Standard MoE + linear expert：∂²F/∂ω∂b = ∂F/∂a （gating ω 和 expert a,b 交互）
- Dense-to-sparse MoE + linear router：∂F/∂τ = -(1/τ) ω^T ∂F/∂ω（温度 τ 和 gating ω 交互）
- Hierarchical MoE + linear expert：∂²F̃/∂ω∂b = ∂²F̃/∂κ∂b = ∂F̃/∂a（两层参数同时交互）

### 核心结果

| MoE 变体 | 强可辨识 Expert | 线性 Expert | 对应定理 |
|---------|--------------|-----------|---------|
| Softmax Gating MoE | O_P([log(n)/n]^{1/4}) 最快 | O_P(1/log^λ(n)) 极慢 | Thm 3.2, Thm 3.4 |
| Dense-to-sparse + Linear Router | O_P(1/log^λ(n)) 极慢 | O_P(1/log^λ(n)) 极慢 | Thm 4.1 |
| Dense-to-sparse + General Router | O_P([log(n)/n]^{1/4}) 最快 | O_P(1/log^λ(n)) 极慢 | Thm 4.3 |
| Hierarchical MoE | O_P([log(n)/n]^{1/4}) 最快 | O_P(1/log^λ(n)) 极慢 | Thm 5.1, Thm 5.2 |

**详细发现**：
1. **回归函数估计**：对所有变体，回归函数 f 本身的估计速率为参数速率 O_P([log(n)/n]^{1/2})
2. **Exactly-specified vs Over-specified**：Voronoi cell cardinality 为 1 的参数估计速率为 O_P([log(n)/n]^{1/2})，cardinality > 1 时为 O_P([log(n)/n]^{1/4})
3. **样本需求**：强可辨识 expert 只需多项式样本量 O(ε^{-2}) 或 O(ε^{-4}) 达到误差 ε；线性 expert 需要指数级样本量 O(exp(ε^{-1/λ}))
4. **反直觉发现**：Dense-to-sparse gating 中的 temperature 参数虽然平滑了 expert 选择过程，但引入了与 gating 参数的 PDE 交互，无论 expert 结构如何都会减慢收敛速率
5. **补救方案**：将 linear router 替换为满足 algebraic independence 的 general router（如 MLP + 非线性激活），可以消除 temperature 的负面影响，恢复快速收敛

### 明确局限

1. **作者承认的局限**：
   - 假设数据服从 MoE-based 回归框架，实际场景中回归函数可能任意，需要处理函数空间非凸性
   - 仅考虑单层 MoE，实际应用（DeepSeek, Mixtral）使用多层 MoE
   - k*_1（expert group 数量）在 HMoE 中假设已知

2. **我们读出的局限**：
   - 没有真机实验验证，纯理论分析
   - 所有结果基于高斯噪声假设（虽然作者说只是为了证明方便）
   - 参数空间紧致、输入空间有界等假设在实际中可能不成立
   - 没有涉及 Top-K sparse gating 的收敛分析（仅在 Remark 中提及可扩展）
   - 没有与现有 MoE 训练的 benchmark 结果对比
   - 分析的 expert 类型有限（仅 FFN 和线性），未涉及 attention-based expert 等现代结构
