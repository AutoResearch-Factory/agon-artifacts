# Operator Learning at Machine Precision (2511.19980)

- **Year**: 2025.11
- **Authors**: Aras Bacho, Aleksei G. Sorokin, Xianjin Yang, Theo Bourdais, Edoardo Calvello, Matthieu Darcy, Alexander Hsu, Bamdad Hosseini, Houman Owhadi (Caltech, IIT, U. Washington)
- **Venue**: arXiv preprint
- **Keywords**: operator learning, Newton-Kantorovich, Cholesky factorization, machine precision, neural operators, PDE solver emulation, foundation model, inverse problems, Gauss-Newton, Tikhonov regularization
- **One-liner**: 不直接学解算子, 而是学 Tikhonov 正则化 Gauss-Newton Hessian 逆的 Cholesky 因子, 嵌入 Newton-Kantorovich 迭代中, 首次在算子学习中达到机器精度 (10^{-16}).

## 精华提炼

### 管线/方法

本文提出 CHONKNORIS (Cholesky Newton-Kantorovich Neural Operator Residual Iterative System), 其核心管线分离线训练和在线推理两阶段:

**离线阶段**:
1. 从测度 μ 采样输入 u^(1),...,u^(M) (如 PDE 右端项或初始条件)
2. 对每个 u^(m), 运行传统 NK 求解器 n_warm 步, 生成 flow data v_k^(m), k=0,...,n_warm
3. 对每个 (u^(m), v_k^(m)), 计算 Tikhonov 正则化 Gauss-Newton Hessian 的逆: Q(u,v) = (J(u,v)*J(u,v) + λI)^{-1}, 其中 J = δF/δv 是 Fréchet 导数
4. 对 Q 做 Cholesky 分解: Q = R R^T, R 下三角
5. 用监督学习训练代理模型 R̂_θ(u,v,λ) 预测 Cholesky 因子 R

**在线推理**:
1. 给定输入 u, 初始猜测 v_0
2. 迭代 n=0,...,N-1:
   a. 计算残差 r = F(u, v_n)
   b. 计算 Jacobian J_n = δF/δv(u, v_n)
   c. 预测 Cholesky 因子 R = R̂_θ(u, v_n, λ_n)
   d. 计算增量 δv = -α_n R R^T J_n* r
   e. 更新 v_{n+1} = v_n + δv
   f. 通过 line search 调整 α_n, λ_n (在 9 种组合中选使残差下降最多的)
3. 返回 v̂_N

**FONKNORIS 变体**: 不学 R̂(u,v), 而是学 R̂(a,b,c), 其中 a,b,c 是 Fréchet 导数的系数函数 (对二阶 PDE, J = a ∂_xx + b ∂_x + c). 这使模型可在不同 PDE 间泛化.

### 关键技术细节

**代理模型选择**:
- 非线性椭圆方程和 Darcy 流: 向量值 GP 回归, 核为 Matérn ν=5/2 和平方指数核混合, 长度尺度调优
- Burgers 方程: MLP, 结构为 (127, 500, 1000, 8128), 激活函数 Tanh
- 逆问题: 全部使用 GP-Gaussian 回归

**数据规模** (按问题):
- 非线性椭圆: 训练 896, 验证 128, 网格 N_x=63
- Burgers: 训练 448, 验证 64, 网格 (N_t, N_x)=(151,127)
- Darcy: 训练 896, 验证 128, 网格 (20,20)
- Calderon: 训练 7500, 验证 2500, 网格 (9,9)
- 逆波散射: 训练 9750, 验证 250, 网格 (7,7)
- 地震成像 FWI: 训练 800, 验证 200, 网格 (5,5)/(7,7)/(10,10)

**NK warmup 参数**: n_warm=5-6, λ_flow 在 0 到 10^{-10} 之间变化

**正则化**: λ_n 自适应调整, 随迭代减小以实现从梯度下降到 Gauss-Newton 的平滑过渡. 每个迭代从 {β_λ λ_n, λ_n, κ_λ λ_n} × {β_α α_n, α_n, κ_α α_n} 共 9 种组合中选择最佳者.

**离散化**: 有限差分, 规则网格. 方法声称 discretization-invariant — 可通过 optimal recovery 或 FNO 框架适配任意离散化.

**Cholesky 因子的关键性质** (附录):
- max-min ordering 使 Cholesky 因子具有 rank-revealing 和稀疏性
- 稀疏 Cholesky: 仅需 O(N log^d(1/ε)) 非零元即可达精度 ε
- 随机 ordering 也保持 rank-revealing 性质

**FONKNORIS 训练详情**:
- 非线性椭圆: M_ell=5000 realizations, 1000 个外力的 NK flow
- Burgers: M_Burg=5000, 1000 个初值条件
- Darcy: M_Darcy=10000, 2000 对 (conductivity, forcing)
- 聚合方法: nested Kriging (带约束 Σα_i=1 的 BLUP), 使用算术平均核长度尺度
- 测试: Klein-Gordon 和 Sine-Gordon (未训练)

### 核心结果

| 问题 | GP baseline | FNO | TNO | CHONKNORIS |
|------|-------------|-----|-----|------------|
| 非线性椭圆 | 5.1e-6 | 1.3e-3 | 5.7e-3 | **8.9e-16** |
| Burgers | 1.1e-1 | 8.8e-3 | 2.2e-2 | **5.1e-16** |
| 非线性 Darcy | 1.8e-3 | 4.8e-3 | 3.9e-3 | **9.6e-16** |
| Calderon | 1.6e-2 | -- | -- | **3.2e-15** |
| 逆波散射 | 1.5e-2 | -- | -- | **9.2e-13** |
| 地震 5x5 | 2.3e-2 | -- | -- | **2.0e-14** |
| 地震 7x7 | 4.4e-2 | -- | -- | **3.0e-12** |
| 地震 10x10 | 5.7e-2 | -- | -- | **1.2e-3** |
| 地震 14x14 | -- | -- | -- | 不收敛 |

**关键发现**:
1. CHONKNORIS 在所有 forward 问题上一致达到机器精度 (~10^{-16}), 传统算子学习方法 (GP/FNO/TNO) 卡在 10^{-2}~10^{-6}
2. 非线性椭圆: 从 v_0=0 出发, CHONKNORIS 10 次迭代收敛, 而精确 NK 只需 4 次 (差别在于预测的 Hessian 不是精确的)
3. Darcy 流最困难: 需要 100-1000 次迭代. 10 次达 10^{-3}, 100 次达 10^{-6}, 1000 次 95% case 达机器精度
4. FONKNORIS 泛化到未见 PDE: 在训练 3 个 PDE 后, 对 Klein-Gordon 和 Sine-Gordon 达到接近机器精度 (<10^{-14})
5. 地震成像 FWI: 随分辨率和条件数增加, 性能下降. 14x14 分辨率的条件数达 ~10^8, CHONKNORIS 无法精确预测 Cholesky 因子
6. CHONKNORIS 的残差 RMSE ||F(u,v_n)|| 下降趋势与 L^2 相对误差趋势一致, 可作为无需真解的收敛指标

**反直觉结果**: 
- 学 Cholesky 因子 (一个线性椭圆算子) 比直接学非线性解算子简单得多, 却能达到高得多的精度
- 稀疏 Cholesky 策略 (附录) 可能实现加速但会牺牲精度 — 存在精度-速度权衡

### 明确局限

**作者自述**:
1. 高维 Cholesky 因子的计算和存储成本高
2. 显式计算和存储 Fréchet 导数不可行时需要近似 (如地震成像)
3. 直接学习 NK 增量而非 Cholesky 因子的实验 "仅获得有限收益"
4. FONKNORIS 训练数据需大量 PDE 族, 系数函数容易 OOD

**审读发现**:
1. 所有实验均在规则网格 + 有限差分上进行, 未在有限元或非结构网格上测试
2. 代理模型仅用 GP 和 MLP, 未测试 FNO/DeepONet 作为 Cholesky 因子预测器 (尽管声称 agnostic)
3. 地震成像 14x14 失败表明条件数 ~10^8 时方法崩溃 — 实际工程问题常远大于此
4. 无真机实验 — 纯数值仿真
5. 未与 physics-informed operator learning (如 PI-DeepONet) 系统性对比, baselines 仅限纯数据驱动
6. 逆问题实验需要 10^3 次迭代 (Calderon) 才能收敛 — 在线推理开销大
7. FONKNORIS 仅测试 1D 二阶 PDE 泛化, 2D/3D 或高阶方程未经测试
8. 方法要求已知 F 及其 Fréchet 导数的显式形式 — 实际中可能不满足
9. 虽然声称 discretization-invariant, 但未给出不同离散化下的实验
10. 未提供任何置信区间或统计显著性检验

### 跟我们的关系

**对 0616-fex 课题的评估**:

1. **不撞车**: CHONKNORIS 的核心贡献是 "学 Cholesky 因子 + NK 迭代" 范式, 目标是算子学习达到机器精度. 它不解决 fex 课题可能关注的 extreme-scale 或 functional 计算问题. 方法和目标是正交的.

   - **Cholesky 因子预测范式**: 将 SPD 矩阵学习转化为 Cholesky 因子学习, 自动保证正定性. 这对需要学习 SPD 算子的 fex 场景有直接参考价值.

   - **Nested Kriging**: 附录中的分层 GP 聚合方法对大尺度 GP 的扩展有价值.

3. **该引**: 如果 fex 课题涉及算子学习、PDE solver emulation 或需要高精度数值求解, 应该引用 CHONKNORIS 作为精度上限的标杆.

4. **差异化**: fex 课题应聚焦于 CHONKNORIS 未解决的方面: 高维 scalable 计算、任意域/非结构网格、非显式 F 的情形、实时推理约束等.

### 撞车评估
不撞车. CHONKNORIS 的目标是算子学习中达到机器精度, 核心贡献是 "学 Cholesky 因子 + NK 迭代" 范式. 0616-fex 课题关注的方向与 CHONKNORIS 正交 — CHONKNORIS 不解决 extreme-scale 计算、非规则域或实时推理问题.

### 可迁移的工具
1. **Cholesky 因子学习范式**: 将 SPD 矩阵学习问题转化为下三角因子学习, 自动保证正定性. 如果 fex 需要学习任何 SPD 算子 (如协方差矩阵、刚度矩阵、预条件子), 这个范式直接可用.
2. **稀疏 Cholesky + max-min ordering**: 附录中的 rank-revealing 性质和 O(N log^d(1/ε)) 稀疏性理论对高维算子压缩有指导意义. 随机 ordering 保持 rank-revealing 性质的结论也可能有用.
3. **Inexact NK 收敛理论** (Theorem 1): 量化学习误差 ε_λ 对收敛率的影响, 提供了 "近似算子 + 迭代精化" 的通用分析框架.
4. **Nested Kriging**: 分层 GP 聚合, 训练成本从 O(N^3) 降到 Σ O(n_i^3), 对大尺度 GP 有实际价值.

### 差异化方向
CHONKNORIS 留下的空白 (fex 可占据): 高维可扩展性 (>10^4 自由度)、非结构网格/任意几何、F 未知或仅有黑盒访问的情形、实时/低延迟推理需求. CHONKNORIS 可作为精度上限标杆引用, 但方法本身不会与 fex 的核心贡献竞争.
