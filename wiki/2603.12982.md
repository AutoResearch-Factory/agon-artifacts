# RUNNs: Ritz-Uzawa Neural Networks for Solving Variational Problems (2603.12982)

- **Year**: 2026.03
- **Authors**: Pablo Herrera (BCAM / PUCV), Jamie M. Taylor (CUNEF Universidad), Carlos Uriarte (BCAM / Curtin), Ignacio Muga (PUCV), David Pardo (UPV/EHU / BCAM / Ikerbasque), Kristoffer G. van der Zee (University of Nottingham)
- **Venue**: arXiv
- **Keywords**: Ritz-Uzawa neural networks, variational PDEs, weak formulation, ultra-weak formulation, spectral bias, Fourier features, stochastic quadrature, LS/Adam
- **One-liner**: 用Ritz-Uzawa迭代和谱调频稳健求解低正则PDE。

## 精华提炼

### 管线/方法

1. 论文目标是把神经网络 PDE 求解从单次 PINN/Deep Ritz 优化，改造成可迭代修正的 Ritz-Uzawa 框架。
2. 输入是一类线性变分问题：找 \(u^* \in \mathbb{U}\)，使 \(b(u^*,v)=\ell(v)\) 对所有 \(v\in\mathbb{V}\) 成立。
3. 作者先假设双线性形式满足 continuity、injectivity、surjectivity，由此得到 inf-sup 型稳定关系。
4. 对当前近似 \(u^k\)，定义残差 \(r^k \in \mathbb{V}\)：\((r^k,v)_\mathbb{V}=\ell(v)-b(u^k,v)\)。
5. 再定义 trial-space correction \(\delta^k \in \mathbb{U}\)：\((\delta^k,w)_\mathbb{U}=b(w,r^k)\)。
6. 更新规则是 Uzawa 梯度下降视角：\(u^{k+1}=u^k+\rho\delta^k\)，等价于对 \(\frac12\|r(u)\|_\mathbb{V}^2\) 做下降。
7. 理论部分把误差写成 \(e^{k+1}=(I-\rho B'B)e^k\)，并给出收敛步长 \(\rho<2\|B\|^{-2}\)。
8. 最优步长写成 \(\rho^*=2/(\|B\|^2+\|B^{-1}\|^{-2})\)，最优收缩率由 \(\|B\|\) 和 \(\|B^{-1}\|\) 控制。
9. 如果给 \(\mathbb{U}\) 使用 graph inner product \((u,w)_\mathbb{U}=(Bu,Bw)_\mathbb{V}\)，则 \(B'=B^{-1}\)，\(\rho=1\) 可一步收敛于理想连续解。
10. Approach 1 显式保留残差 Ritz minimization、correction Ritz minimization、再累加所有 \(\delta^j\)。
11. Approach 2 为避免网络函数空间不闭合于加法，把 \(u^{k+1}\) 也写成一个 \(\mathbb{U}\)-norm projection minimization。
12. Approach 3 用 \(u^{k+1}\) 直接替代 correction，形成两步 Ritz-Uzawa minimization，并连接到 deep double Ritz / adjoint Ritz。
13. 强形式下残差可显式写成 \(r^k=f-Bu^k\)，correction loss 是 \(\|B\delta+Bu^k-f\|^2_{L^2}\)。
14. 弱形式下若 \(b\) 对称正定且 \(\mathbb{U}=\mathbb{V}\)，则 \(r^k=\delta^k=u^*-u^k\)，更新可直接用 learned residual 修正。
15. 超弱形式下 \(\mathbb{U}=L^2\)，测试空间使用 adjoint graph norm，correction 可写为 \(B'r^k\)，适合低正则解。
16. 实际算法使用 inexact Ritz-Uzawa，允许每个 inner minimization 只是相对误差 \(\varepsilon\) 内的 quasi-minimizer。
17. Approach 1 的 inexact 收敛条件是 \(\varepsilon^2+2\varepsilon < (1-\|I-\rho B'B\|)/(\rho\|B\|^2)\)。
18. Approach 2 还要控制 projection step 的相对误差，条件变为 \(\varepsilon^3+3\varepsilon^2+3\varepsilon < (1-\|I-\rho B'B\|)/(\rho\|B\|^2)\)。
19. 神经网络 ansatz 的核心是 first-layer sinusoidal Fourier feature mapping：\(\sin(\kappa_j(w_j^\top x+b_j))\)。
20. 若需要强制 Dirichlet 边界条件，网络输出乘以 smooth cut-off \(\xi(x)\)，一维实验中使用 \(\xi(x)=1-x^2\)。
21. 输出层是线性组合 \(v_\theta(x)=\sum_j w_{out,j}\phi_j(x)\)，hidden 参数与线性输出权重分开优化。
22. 谱调频模块用离散 FFT 计算 normalized cumulative power spectral density，简称 NCPSD。
23. NCPSD 使用 Sobolev 权重 \((1+|\omega_m|^2)^s\)，其中 \(s\) 根据对象正则性选择。
24. 强形式 residual 用 \(s=0\)，trial correction proxy 用 \(s=2\)。
25. 弱形式 residual 当作 \(H^{-1}\) 对象，用 \(s=-1\)，weak correction 用 \(s=1\)。
26. 超弱形式 residual 当作 \(H^{-2}\) 对象，用 \(s=-2\)，primal correction proxy 用 \(s=0\)。
27. 频率带宽通过能量分位数确定：选择 \(\omega_{\min}\)、\(\omega_{\max}\)，使 NCPSD 达到 \(\alpha\) 和 \(1-\alpha\)。
28. 论文示例中 \(\alpha\) 可取 0.05，即丢掉两端各 5% 的谱能量尾部。
29. \(\kappa_j\) 从 \([\omega_{\min},\omega_{\max}]\) 的 log-uniform 分布采样，用来初始化 Fourier feature 频率。
30. 空间权重 \(w_j\) 按 SIREN 风格从 \(\mathcal{U}(-1,1)\) 初始化，bias 从 \((-\pi,\pi)\) 初始化。
31. 后续 hidden layer 若用 \(\tanh\) 则 Glorot uniform 初始化，若用 \(\mathrm{ReLU}^3\) 则 He uniform 初始化。
32. 优化器是 hybrid LS/Adam：固定 hidden 参数时，输出层线性权重通过最小二乘闭式/线性系统更新。
33. loss 对输出层权重是二次型：\(\frac12 w^\top H w-f^\top w+q\)，因此可解 \(Hw=f\)。
34. 为处理 \(H\) 奇异或病态，作者用 \(S=\mathrm{diag}(\sqrt{H_{jj}})\) 做 diagonal scaling。
35. 实际解的是 \((S^{-1}HS^{-1}+\lambda I)\tilde{w}=S^{-1}f\)，再恢复 \(w_{out}=S^{-1}\tilde{w}\)。
36. 每个 epoch 先 LS 更新 output layer，再 Adam 更新 hidden parameters，训练结束后再做一次 final LS。
37. 积分离散不用固定 deterministic quadrature，而使用 unbiased stochastic quadrature，避免 Deep Ritz 过拟合 quadrature points。
38. 具体采用三阶无偏分层随机 quadrature \(P_3\)，每个区间随机采一个 \(x_1\sim 3x^2\) on \((0,1)\)。
39. \(P_3\) 的规则是 \([L(x_1)-2L(0)+L(-x_1)]/(3x_1^2)+2L(0)\)，对三次多项式精确。
40. 当 integrand 足够光滑时，作者引用该规则方差可达 \(\mathcal{O}(N_K^{-9})\)。
41. variance analysis 的关键结论：强形式 correction loss 的梯度方差会随 \(\|B\delta-r^k\|_{L^2}\) 下降而被动减小。
42. 弱形式中即使整体 weak residual 为零，梯度 integrand \(\nabla u\cdot\nabla v-fv\) 逐点仍可非零，所以方差不消失。
43. 超弱形式中同理，\(-u\Delta v-fv\) 的 integrand 在 exact solution 处也不必逐点为零，方差仍持久存在。
44. 第一个数值实验是一维 Poisson，\(\Omega=(-1,1)\)，\(H^1_0\) 弱形式，exact solution 为 \(\sin(\pi x)\)。
45. Adam baseline 使用 shallow Fourier network \(L=1\)、30 个 Fourier feature neurons、9000 collocation points。
46. Adam baseline 的 schedule 是 \(u^0\) 1000 epochs at \(9\cdot10^{-3}\)，\(r^0\) 2000 epochs at \(10^{-4}\)，\(r^1\) 3000 epochs at \(10^{-5}\)。
47. LS/Adam 版本仍用 shallow configuration 和 9000 points，但三阶段各 1000 epochs，learning rates 为 \(10^{-2},10^{-3},10^{-3}\)。
48. 作者声称 LS/Adam 用 3000 total epochs 达到比 Adam 6000 epochs 更好的 \(H^1_0\) relative error。
49. 高频实验 exact solution 是 \(u^*(x)=\sin(40\pi x)\)，测试 spectral matching 对高频误差的捕捉能力。
50. 高频实验使用 deep configuration：30 个 Fourier feature generators 后接一层 \(\tanh\) hidden layer。
51. 高频实验 schedule 是 \(u^0\) 300 epochs、\(r^0\) 1000 epochs、\(r^1\) 1000 epochs，均 9000 points，learning rate \(10^{-3}\)。
52. 超弱实验求解 \(-u''=\delta'\)，Dirichlet 边界，exact solution 为 \(u^*(x)=\frac12(x+1)-H(x)\)，有跳跃不连续。
53. 这个例子中 \(f=\delta'\in H^{-2}\)，标准 \(H^1\) 能量形式不兼容，但超弱 \(L^2\) trial formulation 可处理。
54. 初始 adjoint test function \(v\) 通过 \(\int \frac12(v''_\theta)^2dx+v'_\theta(0)\) 最小化得到，随后 \(u^0=-v''\)。
55. 后续 residual correction \(r^k\) 最小化 \(\int [\frac12(r''_\theta)^2+u^k r''_\theta]dx+r'_\theta(0)\)。
56. 超弱更新是 \(u^{k+1}=u^k-(r^k)''\)，也就是用 learned adjoint test function 的二阶导作为 primal correction。
57. 超弱初始网络用 shallow \(\mathrm{ReLU}^3\) 保证 \(H^2\) 连续；后续 correction 用 Fourier feature layer + \(\mathrm{ReLU}^3\) layer，30 neurons。
58. 超弱 schedule 是 \(u^0\) 3000 points/100 epochs，\(r^0\) 4500 points/2500 epochs，\(r^1\) 6000 points/2500 epochs，learning rate \(8\cdot10^{-5}\)。
59. 实验图显示 learned correction \(-(r^k)''\) 能定位 \(x=0\) 处 jump discontinuity，并降低 \(L^2\) relative error。
60. 论文结论把 RUNNs 定位为 mesh-free solver，而不是符号发现或可解释公式提取方法。

### 关键技术细节

- 理论 backbone 是 inexact Uzawa for saddle point / variational problems，不是单纯启发式 residual fitting。
- 三种 formulation 同时覆盖 strong、weak、ultra-weak，这是它相对普通 PINN/Deep Ritz 的主要结构优势。
- 频率自适应不是训练中连续学习 \(\kappa\)，而是按上一轮 residual/proxy 的 NCPSD 初始化下一轮 Fourier feature bandwidth。
- 训练把 output-layer least squares 与 hidden-layer Adam 分开，降低 purely first-order optimization 的慢收敛问题。
- 数值积分明确使用 unbiased stochastic quadrature，而不是固定网格 quadrature；这点直接针对 Deep Ritz 的 quadrature overfitting。
- 被动 variance reduction 只在 strong formulation 中成立；作者明确承认 weak/ultra-weak 的 stochastic gradient 方差会持续存在。
- 实验都在一维 \((-1,1)\) 上，重点是正则性/频率/奇异性，而不是高维几何复杂度。
- 网络规模较小：主要是 30 个 Fourier generators，个别实验加一层 \(\tanh\) 或 \(\mathrm{ReLU}^3\)。
- 论文给了严格连续层面的收敛命题，但实际神经训练只靠相对误差假设，未提供可验证 optimizer 达标证据。

### 核心结果

- LS/Adam 在 smooth Poisson 上用 3000 total epochs 优于 Adam 的 6000 total epochs，说明 output-layer LS 对收敛很关键。
- \(u^*(x)=\sin(40\pi x)\) 高频例子中，NCPSD-guided Fourier initialization 可让网络捕捉高频 error/correction。
- 超弱 \(-u''=\delta'\) 例子中，RUNNs 能恢复不属于 \(H^1_0\) 的 discontinuous \(L^2\) solution，这是普通 strong PINN/energy method 不适配的场景。
- 理论上 strong formulation 的 correction gradient variance 随 loss 收敛而下降，是一个有价值但受 formulation 限制的结论。
- 反直觉点是：weak/ultra-weak formulation 更适合低正则问题，但 stochastic gradient 方差不会因全局残差为零而自然消失。

### 明确局限

- 没有真机、真实 PDE 数据、真实工程几何或复杂边界实验。
- 所有主要数值实验都是一维 manufactured solution，不能证明高维 PDE 上的可扩展性。
- 没有公开代码或 artifact 信息，第三方复现需要自己实现 RUNNs、NCPSD、LS/Adam 和 \(P_3\) quadrature。
- 结果主要通过图展示，正文没有给出完整数值表、置信区间、多随机种子统计或 wall-clock 时间。
- 与 PINN/Deep Ritz/RVPINN/Deep Uzawa 等 baseline 的公平定量对比不足，尤其缺少相同参数预算和相同 collocation budget 下的系统比较。
- LS/Adam 中 Hessian assemble/solve 的计算成本没有详细报告；在高维和大宽度下可能成为瓶颈。
- inexact convergence 条件依赖 inner minimization 的相对误差，但神经网络优化很难在线认证该误差。
- NCPSD 需要 residual/proxy 的 dense grid/FFT 估计，一维可行，高维规则域之外会复杂很多。
- 超弱实验处理的是高度结构化的一维 Dirac derivative；更一般的 measure source、多维 singularity、复杂 domain 未验证。
- 声称 mesh-free，但 \(P_3\) stratified quadrature 仍依赖分区/采样设计，不是完全免离散。

## Read by: 0616-fex (2026-06-16)

本次从 `0616-fex` 视角看，RUNNs 不是 finite expression search 或 symbolic regression 论文，而是一个 variational neural PDE solver。它和 FEX 的主要交集在“高频 PDE + 频率自适应 + 低正则解”这组三个 claim 上。最值得复用的是 Sobolev-weighted NCPSD：先用 residual 或 correction proxy 做 FFT，再按能量分位数确定频率带宽，这可以转化为 FEX operator set 的频率候选生成器。另一个可复用点是 ultra-weak formulation，把不属于 \(H^1\) 的解放到 \(L^2\) trial space，用 adjoint test function 的导数做 correction，这对 discontinuity/contact/measure-source 类型问题有启发。撞车风险中等偏低：RUNNs 输出神经网络近似，不输出可解释有限表达式；但如果 FEX 写作中声称首次用谱自适应解决高频 PDE 或首次覆盖低正则 ultra-weak case，需要把 RUNNs 作为相邻 baseline 并明确差异。建议引用。
