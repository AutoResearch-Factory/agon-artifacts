# SpatioTemporal Causal Network Diagnostics for Geographic Tipping Point Early Warning (2606.17553)

- **Year**: 2026.06 (arXiv)
- **Authors**: Zhaoyuan Yu\* (通讯, 南京师范大学 / 江苏地理信息资源开发与利用协同创新中心), Zhangyong Liang (天津大学 国家应用数学中心); 二人 equal contribution
- **Venue**: arXiv preprint, 用 Science China Information Sciences (SCIS2026) 模板投稿; 基金 NSFC 42230406
- **Keywords**: tipping points, causal graph dynamics, dynamic network diagnostics, transfer entropy, dynamic mode decomposition (DMD), Moran's I, AMOC, sea surface temperature, geographic resilience, early warning signal
- **One-liner**: 把地理场表示为随时间演化的**有向因果网络**(转移熵推断), 在每个因果子网内用 Graph-DMD 估恢复率、用 DNSD 三信号打分选出"最脆弱子网", 融合成可解释的 tipping 概率——在 AMOC/SST 上做**局部化**空间早期预警, AUROC 0.783、子网 IoU 0.378。

## 精华提炼

### 管线/方法
ST-CND 在每个滚动时间窗内跑一条四步流水线 (Algorithm 1, 对应 Figure 1 schematic)：

1. **输入与去趋势**：时空场 $\mathbf{X}\in\mathbb{R}^{N\times T}$ ($N$ 个空间格点)。对窗口快照 $\mathbf{X}^{(t)}=[\mathbf{x}_{t-w+1},\dots,\mathbf{x}_t]$ 沿时间轴做高斯平滑 $\mathcal{G}_\sigma$ 并相减, 得残差 $\tilde{\mathbf{X}}^{(t)}=\mathbf{X}^{(t)}-\mathcal{G}_\sigma*\mathbf{X}^{(t)}$。所有后续推断都在残差上做, 目的是压掉共享的低频强迫 (slow drift), 避免其主导预测依赖。
2. **因果拓扑重建 (transfer entropy → 有向图)**：对每个有向对 $j\to i$ 算转移熵 $T_{j\to i}^{(t)}$ (问"源节点 $j$ 的历史能否改善对目标 $i$ 下一步的预测, 超出 $i$ 自身历史所能解释")。默认估计器是 **KSG1 $k$-NN 互信息估计** ($k=4$, Kraskov2004 + JIDT/Lizier2014), history embedding $h=1$。用 **effective TE**(减去 IAAFT 相位随机化 surrogate 的经验基线, Marschinski-Kantz2002 + Schreiber1996) 消除短窗正偏, surrogate 分位数转 edge-level $p$ 值, 再用 **BH-FDR** ($\alpha=0.05$) 控制多重检验, 得到稀疏有向邻接 $\mathbf{A}^{(t)}$。每个节点的因果邻域 $M_i^{(t)}=\{j: A_{ij}>0 \text{ 或 } A_{ji}>0\}\cup\{i\}$ (对称化仅用于成团, 有向权重保留作归因)。
3. **子图约束的物理衰减提取 (Graph-DMD)**：对每个候选子网 $M_i$, 只在该子网节点上做 DMD (避免全局 DMD 的空间稀释)。拟合局部线性演化算子 $\mathbf{Z}_{2,M_i}\approx\mathbf{K}_{M_i}\mathbf{Z}_{1,M_i}$, 取特征值 $\lambda_{i,k}$ 映射到连续时间 $\omega_{i,k}=\ln(\lambda_{i,k})/\Delta t$, 领头实部 $\gamma_{\mathrm{lead}}^{(t)}(M_i)=\max_k\mathrm{Re}(\omega_{i,k})$ 作为**衰减代理**。临界慢化下 $\gamma_{\mathrm{lead}}\to 0^-$ (越接近 0 恢复越慢、韧性越弱)。
4. **临界子网识别 (DNSD 三信号打分)**：对每个 $M_i$ 及其补集 $\overline{M}_i$ 算三个量——内部平均标准差 $\mathrm{SD}_{\mathrm{in}}$、内部平均绝对相关 $\mathrm{PCC}_{\mathrm{in}}$、内外平均绝对耦合 $\mathrm{PCC}_{\mathrm{out}}$。合成 ST-CND 分数
   $$I_{\mathrm{ST\text{-}CND}}^{(t)}(M_i)=\frac{\mathrm{SD}_{\mathrm{in}}\cdot\mathrm{PCC}_{\mathrm{in}}}{\mathrm{PCC}_{\mathrm{out}}+\epsilon}$$
   分母的"外部耦合"起 **common-mode rejection** 作用: 空间均匀红噪声会同时抬高内、外相关, 比值把这种非特异抬升抵消掉——这是全文对抗红噪声假警报的核心设计。再用衰减门控 $R_{\mathrm{ST\text{-}CND}}=I/( -\gamma_{\mathrm{lead}}+\eta)$ (只对 $\gamma_{\mathrm{lead}}<0$ 的稳定侧估计), 选 $M^*(t)=\arg\max R$ 为临界子网。
5. **概率融合与可解释读出**：把 $M^*(t)$ 的五个特征 $\mathbf{f}_t=[\gamma_{\mathrm{lead}}, I_{\mathrm{ST\text{-}CND}}, \mathrm{SD}_{\mathrm{in}}, \mathrm{PCC}_{\mathrm{in}}, \mathrm{PCC}_{\mathrm{out}}]$ 喂给**正则化 logistic 分类器**, 输出 tipping 概率 $\mathcal{P}_{\mathrm{tip}}(t)\in[0,1]$。标准化后 $\boldsymbol\beta$ 的符号/大小提供特征级归因 (可解释性卖点)。

### 关键技术细节
- **TE 估计**：KSG1 ($k=4$), $h=1$; $S=200$ IAAFT surrogates/对; BH-FDR $\alpha=0.05$。直方图估计器仅作 robustness 备选。
- **大网格 "soft-weight" 协议 (关键工程折衷)**：真实大网格上 all-pairs surrogate-FDR 不可行 (原生 $1°$ 有 ~21,600 节点 → $4.7\times10^8$ 有向对/窗)。故实盘用: (a) $2°$–$4°$ 空间粗化; (b) 每窗只保留 TE 最大的 top-$m_{\max}=500$ 正有向对 (确定性 top-$m$ 筛选, 非随机采样); (c) **不做 surrogate gating**。观测数据行 (AMOC/OISST) 全部用此协议, 作者明确声明**不是** native-grid all-pairs surrogate-FDR 结果。
- **DMD 数值**：rank $r$ 用 Gavish-Donoho 最优硬阈值; $\Delta t=1$; 强振荡领头模排除在门控外; 短窗 ($\min(|M_i|,w-1)<10$) 用 de-biased total-DMD (Hemati2017)。
- **logistic 训练协议**：系数 $(\boldsymbol\beta,\beta_0)$ **只在 1,000 条随机 saddle-node 轨迹上训练一次** (pre-tipping 窗 $|\mu-\mu_c|<0.1$ 为正, post-tipping 排除), 其余所有实验用这套**固定**权重。elastic-net $\rho_{\mathrm{EN}}=0.5$, 10-fold blocked CV, 95% bootstrap CI ($B=1000$), VIF$\le$10。
- **报警规则**：阈值 $\tau_{\mathrm{alarm}}$ 取 no-tipping 红噪声场上 $\mathcal{P}_{\mathrm{tip}}$ 的 $1-\alpha_{\mathrm{alarm}}$ 分位 ($\alpha_{\mathrm{alarm}}=0.05$); 需 $\kappa=3$ 次连续越界才起报, 孤立越界忽略。
- **实验平台**：纯 CPU, 每窗运行 0.16 s (OISST); 复杂度 TE $O(N^2)$ + Graph-DMD $O(N\cdot r^2\cdot w)$。
- **四类合成实验**：(1) Allen-Cahn 反应扩散 (局部成核, 测空间稀释); (2) teleconnected 系统 (测非局部驱动恢复); (3) 空间相关随机场无分岔 (红噪声 no-tipping 对照); (4) 随机 saddle-node + 多元恢复率系统 (测 DMD/RQA/ML 读出)。
- **两个观测基准**：HadISST AMOC SST fingerprint (1870–2022, $2°$ 网格, subpolar gyre $46°$–$66°$N); NOAA OISST v2 Indo-Pacific (1982–2022, $4°$ 网格)。AMOC 用 post-1980 二分类 regime 标签 (跟 Boers2021/Ditlevsen2023 一致)。

### 核心结果
- **AMOC / HadISST (Table 5/benchmark)**：ST-CND **AUROC 0.783** (0.760–0.807) vs 经典 EWS 0.681、$\lambda$-AR1 0.596、recurrence-net 0.500; **IoU 0.378** (对 Caesar2018 SST-fingerprint mask) vs $\lambda$-AR1 0.206。**但 AUPRC 0.533 反被经典 EWS 0.546 略胜**——作者诚实说"改进集中在 specificity 而非 recall"。
- **OISST Indo-Pacific**：ST-CND **AUROC 0.720** (best), AUPRC 0.330 (best), IoU 0.240 (best), 均在 bootstrap CI 内略超 climate-net 0.693。
- **红噪声假警报 (Table 3)**：Moran's $I$ (queen) 随高斯平滑 $\sigma$ 从 $-0.0078$ ($\sigma$=0) 单调涨到 **0.9532** ($\sigma$=4), 纯粹是强加的空间自相关而非韧性损失; ST-CND 比值反而随 $\sigma$ 下降, 假警报率维持在 nominal 5% 附近。**这是全文最干净、最有说服力的对照**。
- **Graph-DMD 恢复率验证**：在已知真实恢复率的受控多元系统上, local Graph-DMD 与真值相关性 > global DMD > 1D Hankel-DMD——支持"限制在脆弱子网内估计避免稀释"。
- **反直觉点**：recurrence-net 基线 AUROC 恰好 = 0.500 (=随机), 因为被强行套进 60 月窗二分类 regime setup, 而非其原设计的 trend-detection——**这是被人为削弱的基线**, 作者虽注明但仍以"outperform"叙述。

### 明确局限
- **不是结构因果**：$\mathbf{A}^{(t)}$ 是 Wiener-Granger 预测因果 (information-flow graph), **不是** Pearl 干预因果; 双变量 TE 会被隐藏共同驱动混淆。作者把 PCMCI (Runge2019) 明列为 future work (line 258, 725)——**正是本课题采用的方法**。
- **$O(N^2)$ 不可扩展**：native $1°$ 分辨率不可行, 只能靠 $2°$–$4°$ 粗化 + top-500 soft-weight。观测结果并非全分辨率全对 surrogate-FDR 跑出来的。
- **异质红噪声开放**：DNSD 比值只对**空间均匀**红噪声 robust; 空间相关强度随场变化的异质红噪声仍是 open problem。
- **AMOC lead time 1,168 月 = 回溯性检测能力, 不是运营化的百年预报** (作者明确澄清)。
- **代码/数据可用性段落在 arXiv 版被整段注释掉** (tex line 728–733), 只留"匿名 review handle, 接收后换永久 DOI"; 无实际公开仓库链接。
- **tex 内部有大量"注释掉的旧版 + 改写后的新版"并存** (几乎每个 subsection 都有一份被 `%` 注释的措辞略异的前身) → 重度改写/AI 辅助润色痕迹明显。
- **单组自评**：2 作者, 全部基线自行重实现, 无独立/跨 lab 验证。

## Read by: 0710-causal-scs (2026-07-10)

**审读视角**：本课题 (因果效应不稳定性作为强对流预警指标) 把这篇作为"因果网络 marker 预警" cluster 的第二篇前作精读 (第一篇是 CNM [[2412.16235]])。

**血缘定位 (关键新发现)**：ST-CND 本质是 **CNM/DNB 谱系的"空间 + 转移熵 + DMD"表亲**。三者共祖 Chen2012 DNB。ST-CND 的 DNSD 打分 $\frac{\mathrm{SD_{in}\cdot PCC_{in}}}{\mathrm{PCC_{out}}}$ 与 CNM wiki 里记录的 DNB 公式 $\frac{\mathrm{SD_d\cdot PCC_d}}{\mathrm{PCC_o}}$ **是同一个比值形式**——只是 ST-CND 把它 (a) 搬到 TE 推断的因果子网上, (b) 加了 Graph-DMD 衰减门控, (c) 加了 logistic 读出, (d) 主打**空间定位** (哪块子区最脆弱) 而非 CNM 的"marker 发散"。所以 CNM 与 ST-CND 覆盖的是同一机制家族的两个投影, 本课题必须把这两篇当作**同一撞车面**处理。

**撞车评估 (细化到 claim)**：
- ST-CND 的核心 claim = "把地理场表示成 TE 有向因果网络, 用 DNSD 三信号 + Graph-DMD 在**空间上定位**最脆弱子网并给 tipping 概率, 比 Moran's I / recurrence-net / $\lambda$-AR1 更抗红噪声、更局部化"。这是**空间定位 + 抗红噪声**轴上的贡献。
- 本课题核心 claim = "因果效应**估计**在假设违背/跨方法/滑窗下的**统计不稳定性** (方差/突变/DAG 结构变化) 本身是独立于经典 EWS 的预警信号"。这是**估计过程脆弱性**轴上的贡献。
- **两轴正交**: ST-CND 想要一个**好的**因果图并在其中测慢化, 把 TE 估计噪声当**要压制的东西** (surrogate + BH-FDR + soft-weight 全是为了净化图); 本课题把估计不稳定性当**信号本身**。这是可立住的机制差异 (与 CNM 的"确定性因果强度变化 vs 本课题统计脆弱性"差异同构、且更强——因为 ST-CND 的整套 pipeline 就是在**主动消除**本课题想利用的那个量)。

**领域错配 (机会 + 风险, 与 CNM 同调)**：ST-CND 全部案例是**慢变气候 tipping** (AMOC 百年际、SST 十年际, 缓慢参数漂移近平衡分岔)——又一篇落在慢变域的前作。**本课题的快变中尺度强对流 (小时级、非平衡) 域仍无人占**。风险: Stage 1 的 Lorenz 慢变 pilot 与 ST-CND 的 saddle-node 合成实验高度重叠, 信息增量低 (与 idea-reviewer 对 NPG 2026 的判断一致)。机会: ST-CND 的 DNSD/DMD 全靠"稳定侧 $\gamma_{\mathrm{lead}}<0$ + 缓慢逼近分岔"假设, 快变非平衡下大概率失效——正是本课题差异化地带。

**可用工具/方法 (直接搬)**：
1. **IAAFT surrogate + BH-FDR ($\alpha=0.05$) 的 TE 显著性协议** → 本课题 strongest objection ("不稳定性是否只是统计伪影") 的**现成证伪工具**: 用 surrogate 检验区分真信号 vs 有限样本噪声, 这正是 idea 里点名要用的 IAAFT。
2. **DNSD 比值 (内部涨落×内部同步/外部耦合) 的 common-mode rejection 设计** → 作为红噪声鲁棒的**确定性机制基线**, 与本课题"不稳定性 marker"并列对照 (和 CNM 基线一起构成机制区分实验)。
3. **Graph-DMD 领头率 + Gavish-Donoho rank + de-biased total-DMD (Hemati2017)** → 若本课题要在 3km 空间场上做子网级恢复率估计的数值配方。
4. **soft-weight top-$m$ TE 协议 + $O(N^2)$ 粗化策略** → 本课题真实 3km 网格 (格点数巨大) 上跑滑窗因果的**可扩展性教训**: ST-CND 明说 native 分辨率不可行, 本课题 Stage 2 需正视同一瓶颈。
