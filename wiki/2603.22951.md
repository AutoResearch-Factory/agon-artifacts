# Weak-PDE-Net: Discovering Open-Form PDEs via Differentiable Symbolic Networks and Weak Formulation (2603.22951)

- **Year**: 2026.03
- **Authors**: Xinxin Li et al.; East China Normal University, Institute of Applied Physics and Computational Mathematics, Beihang University, Shanghai Zhangjiang Institute of Mathematics, Northeast Normal University
- **Venue**: arXiv
- **Keywords**: PDE discovery, weak formulation, open-form PDE, symbolic network, differentiable architecture search, learnable Gaussian embedding, Galilean invariance, U(1) equivariance
- **One-liner**: 用弱形式和可微符号网络从稀疏噪声数据发现开放式 PDE。

## 精华提炼

### 管线/方法

1. 输入是稀疏、可带噪的时空观测点 `(t_i, X_i, U_i)`，不要求规则网格。
2. 论文假设真实系统响应函数为 `U: [0,T] x Omega -> R^h`，`h=1` 是标量场，`h>=2` 是多变量/向量场。
3. 目标 PDE 写成开放式结构 `D^alpha0 F0(U) = sum_s xi_s D^alphas Fs(U)`。
4. 这里的 `Fs` 不是固定候选库里的项，而是由符号网络从基本算子空间组合出来。
5. 基本算子集合在实验超参中给为 `{id, square, cube, sin, multiply}`。
6. 第一模块是 forward response learner，用 learnable Gaussian feature embedding 加轻量 MLP 拟合连续场 `U_hat`。
7. Gaussian embedding 对原始时空坐标 `x=(t,X)` 做局部特征展开。
8. 第 `j` 个特征是 1000 个可学习高斯核的加权和，参数包括中心 `mu`、协方差 `Sigma` 和权重 `f`。
9. 论文动机是缓解普通 MLP 的 spectral bias，使网络能在高梯度或高频区域放置更强表达能力。
10. 轻量 MLP 深度为 2，隐藏维度 32；NS 实验用 `sin` 激活，其余实验用 `tanh`。
11. 第二模块是 weak-form PDE generator，由 symbolic network 和 integration module 组成。
12. 强形式 PDE 先乘 compact-support test function `Phi`，再在时空域积分。
13. 通过 Green's lemma / integration by parts，把对 `U` 的高阶导数转移到已知平滑测试函数 `Phi` 上。
14. 因此弱形式避免直接对稀疏噪声观测做数值微分。
15. 每个测试函数生成一条积分约束，最终形成代数系统 `b = G xi`。
16. 测试函数使用 polynomial bump 或 exponential bump；KdV 用 poly-bump，其余实验用 exp-bump。
17. bump 函数通过中心和半径仿射变换放到时空域中的局部支持区域。
18. 支持半径从 `(eta_min, eta_max)=(0.2,0.4)` 乘最短域边长的均匀分布中采样。
19. 多维测试函数用一维 bump 的 tensor product 构造。
20. symbolic network 用可解释算子替代普通激活函数。
21. 每层先线性变换，再把节点送入一元或二元符号算子。
22. 一元算子消耗 1 个输入节点，二元算子消耗 2 个输入节点。
23. 网络输出 `F(U_hat)` 被视为候选函数项，再配合导数 multi-index 进入弱形式积分。
24. architecture search 阶段先构造最大深度 `L_max=3` 的完整符号网络。
25. 每个算子类型的候选保留数量来自集合 `{0,1,2,3}`。
26. 参数 `alpha` 控制每层每类算子的活跃数量，使用 softmax 连续化候选 mask。
27. 参数 `beta` 控制网络有效深度，同样用 softmax 对各层输出加权。
28. 搜索结束后用 `argmax alpha` 和 `argmax beta` 离散化出最终符号网络结构。
29. 训练分三阶段：Searching、Pruning、Tuning。
30. Searching 阶段联合训练 Gaussian embedding、MLP、symbolic network、integration module 和 DNAS 参数。
31. Searching loss 包含数据拟合 `L_data` 和弱形式残差 `lambda_w L_weak`。
32. Pruning 阶段固定架构，加入 `L1` 正则到 `xi`、`W`、`b`，剪掉弱连接和冗余项。
33. Pruning 结束后，从 symbolic network 参数和 integration coefficients 中得到候选库 `G`。
34. Tuning 阶段只保留 response learner，用它重建规则网格上的 `U_hat`。
35. Tuning 把已识别的候选库转成线性系统，使用 least-squares regression 精修系数 `xi`。
36. 多变量系统在 tuning 阶段触发 physics-informed refinement。
37. Galilean Invariance 用作硬约束，主要面向流体和输运系统。
38. GI 规则保留能抵消移动坐标系时间导数漂移的通量项，如 `div(U tensor U)` 或 `div(omega tensor U)`。
39. GI 规则删除依赖绝对速度的项，如 `U`、`|U|^2`、`U^3`、`U x omega`。
40. NS 实验中该约束用于 2D vorticity equation，保留 `partial_x(u omega)` 和 `partial_y(v omega)` 这样的平流通量。
41. U(1) symmetry equivariance 用作结构假设，面向 complex-valued systems。
42. 对 `psi=u+iv`，若动力学算子与全局相位旋转可交换，则满足 `N_im(u,v) = -N_re(-v,u)`。
43. 论文把这个关系用于 NLS 的 real/imag candidate library augmentation。
44. 该机制不是硬编码最终方程，而是把成对候选提供给 regression，由数据决定是否保留。
45. 评测指标包括 TPR、`E_infty(xi_hat)`、`E_2(xi_hat)` 和重建 MSE。
46. TPR 定义为 `TP/(TP+FN+FP)`，对结构错误很敏感。
47. baseline 只有 Weak-PDE-LEARN，主要比较弱形式加神经 surrogate 的 fixed-library 方法。
48. 数据集均为 PDE benchmark 或数值模拟数据，没有真实实验传感器数据。
49. 论文没有报告公开代码或公开数据包，只写了 data/code available on request。
50. 因此方法完整度较高，但复现实证仍依赖作者提供 artifact。

### 关键技术细节

1. 弱形式核心等式是 `b = G xi`，每一行对应一个测试函数积分约束。
2. `G_{k,s}` 由 `D^alphas Phi_k(x_j) * Fs(U_hat(x_j)) * Delta x_j` 的离散求和近似。
3. `b_k` 由左端时间/空间导数项对应的测试函数导数和 `U_hat` 积分得到。
4. 总损失为 `L = L_data + lambda_w L_weak + lambda_r L_reg`。
5. `lambda_r` 只在 pruning 阶段非零。
6. `L_reg = sum_s |xi_s| + sum |W| + sum |b|`。
7. 实验硬件是 Intel Xeon Platinum 8468H CPU、1.5T RAM、2 张 NVIDIA H800 80GB。
8. Gaussian kernel 数量 `N=1000`，feature dimension `d'=4`。
9. MLP depth `L=2`，hidden dimension 32。
10. 最大符号网络深度 `L_max=3`。
11. 可选 operator count set 为 `{0,1,2,3}`。
12. Burgers 数据域 `[0,10] x [-8,8]`，网格 `101 x 256`，真实形式 `u_t=-0.5 partial_x u^2 + 0.1 u_xx`。
13. KdV 数据域 `[0,20] x [-30,30]`，网格 `201 x 512`，真实形式 `u_t=-3 partial_x u^2 - u_xxx`。
14. KS 数据域 `[0,50] x [-10,10]`，网格 `251 x 256`，初值 `-sin(pi x/10)`，周期边界。
15. CI 数据域 `[0,0.5] x [0,3]`，网格 `200 x 301`，Dirichlet 边界，初值 `sin(x)`。
16. NLS 数据域 `[0,pi] x [-5,5]`，网格 `251 x 256`，拆为 `u,v` 两个实变量。
17. Wave 数据域 `[0,10] x [-5,5] x [-5,5]`，网格 `100 x 64 x 64`。
18. SG 数据空间域 `[-pi,pi] x [-1,1]`，空间网格 `403 x 129`，`Delta t=0.025`，截取前 70 步。
19. NS 用 IBPM 生成 cylinder wake，`Re=100`，原始 `Delta x = Delta t = 0.02`。
20. NS 记录 2000 步后做 10% temporal subsampling，有效 `Delta t_eff=0.2`，最终 `101 x 300 x 150`。

### 核心结果

1. Burgers 在 `r=2.5%` 无噪声时 TPR=1，系数约 `-0.4722` 和 `0.1059`。
2. Burgers 在 `r=10%` 且 100% 噪声时仍 TPR=1，但非线性系数退到 `-0.4078`，误差明显变大。
3. KdV 在 `r=2.5%` 无噪声时 TPR=1，恢复 `-3.0394 partial_x u^2 -0.9281 u_xxx`。
4. KdV 在 `r=25%` 且 100% 噪声时 TPR=1，`E_infty=0.1240`。
5. KS 在 `r=2.5%` 无噪声时 TPR=1，三项结构完整恢复。
6. KS 在 `r=25%` 且 100% 噪声时 TPR=1，但重建 MSE 达到 `1.18e-1`。
7. CI 在无噪声稀疏设置下 TPR=1，但对噪声极敏感。
8. CI 在 1% 噪声时 diffusion coefficient 被压到约 `0.0084` 或 `0.0050`，几乎丢失。
9. CI 在 10% 噪声、`r=25%` 时 TPR=0.50，只剩 `u^3` 和 `u` 项。
10. CI 在 10% 噪声、全采样时更差，TPR=0.20，并出现 `partial_x u^3`、`partial_x u` 等伪项。
11. NLS 在 `r=10%` 到 `50%` 的无噪声设置下两条方程 TPR 都为 1。
12. NLS 在 `r=25%` 且 100% 噪声时结构仍对，但系数误差较大，`E_infty` 分别到 `0.2414` 和 `0.3755`。
13. 论文特别指出没有 U(1) 结构假设时，低采样和高噪声下 real/imag 两条方程会不同步收敛。
14. Wave 在低到 `r=1%` 无噪声时仍 TPR=1。
15. Wave 在 `r=5%` 且 80% 噪声时仍 TPR=1，但系数约 `0.7055,0.6527`。
16. Wave 在 100% 噪声时失败，TPR=0.67，并引入伪 `-0.1528u` 项。
17. SG 对 transcendental `sin(u)` 的发现是亮点，所有设置 TPR=1。
18. SG 在 100% 噪声下仍找到 `sin(u)`，但 `sin` 系数约 `-0.4294`，偏差很大。
19. NS 在 `Re=100`、`r=5%/10%/100%` 无噪声下均 TPR=1。
20. NS 在 `r=10%` 且 50% 噪声下仍 TPR=1，发现 diffusion coefficient 约 `0.0071` 和 `0.0099`。
21. 对 Weak-PDE-LEARN 的比较中，Burgers/KdV/KS 高噪声下多数误差更低。
22. KS 在 100% 噪声下 Weak-PDE-LEARN 未正确识别结构，而 Weak-PDE-Net TPR=1，`E_infty=0.1340`。
23. 消融显示去掉 Gaussian embedding 后，Burgers、KdV、KS、NLS、SG、NS 的结构或系数明显恶化。
24. 消融显示去掉 NAS 后，固定完整符号网络容易保留高阶/冗余项，导致方程复杂化和伪项。
25. Wave 是例外之一，without NAS 在干净 `r=5%` 设置下仍表现接近 full model。

### 明确局限

1. 作者明确承认标准表示不能表达 nested differential operators。
2. 例子是 porous medium equation `u_t = partial_x(u partial_x u)`。
3. 在现有弱形式里，integration by parts 后还剩 `u partial_x u`，它不是只依赖 `F(U)` 的无导数组合。
4. 作者建议未来加入 differential operator layer，再接 symbolic network，表达 `G(D^alpha F)` 这样的组合。
5. 这个扩展会牺牲部分抗噪优势，因为又需要对 response learner 做较低阶导数。
6. 另一个明确局限是弱形式积分的 low-pass filter 效应。
7. 该效应能压噪，但也会抹掉局部小尺度物理信号。
8. CI 实验就是负面证据：只加 1% 噪声就几乎洗掉 diffusion term。
9. 论文主要是 synthetic/numerical benchmark，没有真实传感器、真实实验、跨设备数据。
10. baseline 覆盖偏窄，只系统比较 Weak-PDE-LEARN；没有与 SGA-PDE、DISCOVER、ICNet 做完整同设置横向比较。
11. 代码和数据未公开，只声明 available on request，复现门槛较高。
12. DNAS 搜索和三阶段训练的稳定性、随机种子方差、训练耗时没有详细报告。
13. 物理先验目前是手工选择的 GI 或 U(1)，并非自动从全部候选对称性中发现。
14. 多变量系统成功依赖正确先验；如果先验选错或系统不满足假设，可能引入错误候选偏置。
15. Open-form 的范围实际受 operator set 限制，例如实验集合只有 `id, square, cube, sin, multiply`。
16. 因此“open-form”不是无限表达，而是可微搜索的受限符号组合空间。
17. 表格中有若干 LaTeX/记号小错误，如 KdV 比较表一处写成 `partial_xx`，说明稿件仍偏 preprint 状态。

## Read by: 0616-fex (2026-06-17)

从 `0616-fex` 视角看，这篇的核心价值是把 weak-form PDE discovery、可微符号网络和局部 Gaussian 表征绑成一个端到端系统。若我们的课题也声称从稀疏噪声数据中自动发现开放式 PDE，那么本文已经覆盖了相当多的基础 claim：不用数值微分、候选项不是固定库、用 DNAS 搜符号结构、用 GI/U(1) 等物理结构减少多变量伪项。差异化应避开这些表述的正面重复。

可以利用的部分很明确：弱形式积分适合作为噪声鲁棒 residual；Gaussian embedding 适合替代普通 MLP 做连续场重建；search-prune-tune 是一个可复用训练框架；TPR/系数误差/MSE 加噪声和采样率扫描可以直接成为评测模板。最值得攻击的缺口是 nested differential operators、真实数据复现、artifact 公开、自动物理先验选择和更宽 operator space。本文应该列为相关工作中的强相关论文，但其可信度还不到“已被社区确认”的程度，因为代码数据未公开、baseline 少、全是作者自评 benchmark。
