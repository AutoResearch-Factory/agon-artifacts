# Global linear convergence of entropy-regularized softmax policy gradient beyond tabular MDPs (2605.24939)

- **Year**: 2026.05
- **Authors**: Ziyue Chen, David Siska, Lukasz Szpruch (University of Edinburgh)
- **Venue**: arXiv
- **Keywords**: entropy-regularized RL, softmax policy gradient, log-linear policy, continuous state/action MDP, Polyak-Lojasiewicz inequality, Fisher information, KL Lyapunov, linear convergence
- **One-liner**: 证明连续状态/动作下 log-linear softmax policy gradient 在线性可实现和特征条件下全局线性收敛。

## 精华提炼

### 管线/方法

1. 论文研究 discounted infinite-horizon entropy-regularized MDP，状态空间 `S` 为 Polish space，动作空间 `A` compact，cost `c` 有界且归一到 `|c| <= 1`。
2. 目标是最小化 `V_tau^pi(rho)`，即累计 cost 加 `tau log(d pi / d mu)` 的 KL/entropy 正则项，其中 `gamma in [0,1)`，`tau > 0`。
3. 策略类不是 tabular direct softmax，而是 log-linear softmax：
   `pi_theta(da|s) propto exp(<theta, g(s,a)>) mu(da)`。
4. 特征 `g:S x A -> R^p` 有界，论文假设 `sum_i |g_i|_infty^2 <= 1`，从而梯度和 log density 可控。
5. 核心可实现性假设是 `Q_tau^pi`-realizability：对任意 `pi in Pi_mu`，存在唯一 `theta(pi)` 使 `<theta(pi), g(s,a)> = - Q_tau^pi(s,a) / tau`。
6. 该假设保证最优策略也落在参数化策略类里：存在 `theta*` 表示 optimal softmax policy。
7. 第一层技术是 regularized performance difference lemma，把两个策略的 value gap 写成 `Q_tau` 项和 KL 项。
8. 第二层技术是 sandwich inequality，把 suboptimality 上界为当前策略与 proximal policy `bar pi_theta` 的 KL。
9. 第三层技术是 KL-logit inequality：当两个 policy density 有统一上界时，`KL(pi_theta || pi_theta') <= (1/2W)||theta-theta'||_2^2`。
10. 对 simplex features，KL-logit inequality 只控制去掉 `1` 方向后的参数差，即 `theta-theta'-c*1`。
11. 第四层技术是 policy gradient theorem，得到
    `nabla V_tau^{pi_theta}(rho) = 1/(1-gamma) E[(Q_tau^{pi_theta}+tau log pi_theta) nabla log pi_theta]`。
12. 在 `Q_tau^pi`-realizability 下，梯度可改写成 Fisher information matrix 乘参数误差：
    `nabla V = tau/(1-gamma) int_S G^{pi_theta}(s) d_rho^{pi_theta}(ds) (theta-theta(pi_theta))`。
13. Fisher information matrix 定义为 action 条件分布下 `nabla log pi` 的二阶矩/协方差。
14. 因此，如果 integrated FIM 的最小特征值不退化，梯度范数可以下界参数误差。
15. sandwich inequality 给 value gap 上界参数误差平方，FIM 下界给参数误差上界梯度平方，二者合成非均匀 PŁ inequality。
16. 非均匀 PŁ 常数依赖 `theta`，具体依赖 log-density bound `R`、occupancy density ratio、以及 integrated `lambda_min(FIM)` 的倒数平方。
17. 对 simplex features，FIM 在全 `1` 方向上奇异，论文改用 uncentered covariance `int g g^T dpi` 来控制正交子空间。
18. 线性收敛的核心问题变成：沿 gradient flow 如何保证 PŁ 常数不退化。
19. 论文用 Lyapunov 思路：`d/dt V_tau^{pi_theta_t}(rho) = -||nabla V||^2 <= 0`，所以 value sublevel set 正向不变。
20. 由于 cost 有界，sublevel set 的径向有界性必须来自 KL 项；这就是论文最重要的新技术点。
21. 对 full affine span features，假设每个 state 下 `a -> g(s,a)` 连续，且任意单位向量 `u` 的最大值集合 `argmax_a u^T g(s,a)` 的 `mu` 测度为 0。
22. 在这个假设下，若 `theta = r u` 且 `r -> infinity`，Gibbs policy 会集中到 `u^T g` 的近最大集合。
23. 因为真正最大集合测度为 0，用二元 partition 的 KL lower bound 可推出 `KL(pi_theta(.|s) || mu) -> infinity`。
24. 这给出 full affine feature 情况下的 radial unboundedness，从而把 gradient flow 限制在有界参数集内。
25. 论文给出的 full affine example 是 trigonometric features：`cos(k^T a + l_k^T s)` 和 `sin(k^T a + l_k^T s)`，要求没有 action-constant Fourier mode，也没有冗余 action mode。
26. 对 simplex features，假设 `g(s,a) in Delta_{p-1}`，且对所有 `u perp 1, u != 0`，`argmax_a u^T g(s,a)` 的测度为 0。
27. simplex 情况下只需要证明 `||theta_perp|| -> infinity` 时 KL 径向无界，因为 `1^T theta_t` 沿 gradient flow 保持不变。
28. 论文给出的 simplex 正例是 Bernstein polynomial features：先把 affine scalar `q(s)+u^T a` 归一到 `[0,1]`，再用 degree `k` 的 Bernstein basis。
29. 论文还给了一个重要反例：1D finite-element hat functions 在中间区间有 plateau，最大值集合测度为 `1/3`。
30. 对该反例，沿 `theta=(-1,1,1,-1)` 放大时，KL 不发散而收敛到 `log 3`，说明零测度 maximizer 假设不是装饰条件。
31. 有了 radial unboundedness，作者证明 gradient flow 全局存在，并且 `sup_t |log d pi_theta_t/dmu|_infty < infinity`。
32. 再用 density lower bound 和 variance concavity，把初始化处 FIM 的正定性传递为沿轨迹的 integrated eigenvalue 正下界。
33. full affine 情况要求初始化满足每个 `s` 的 `G^{pi_theta0}(s)` positive definite。
34. simplex 情况要求初始化处 `int g(s,a)g(s,a)^T pi_theta0(da|s)` positive definite。
35. 最后由非均匀 PŁ 的常数上界化得到 global linear convergence：
    `V_tau^{pi_theta_t}(rho)-V_tau^*(rho) <= exp(-t/C_theta0) [V_tau^{pi_theta0}(rho)-V_tau^*(rho)]`。

### 关键技术细节

1. `S` 是 Polish space，`A` compact；`mu` 对 Lebesgue measure 绝对连续且 full support。
2. 折扣因子 `gamma in [0,1)`，entropy/KL 强度 `tau > 0`。
3. cost 假设为 bounded measurable，且无损设为 `|c|_{B_b(S x A)} <= 1`。
4. 策略是 continuous action density：`d pi_theta / d mu = exp(<theta,g>)/Z_theta(s)`。
5. `nabla log pi_theta(a|s) = g(s,a) - E_{a'~pi_theta}[g(s,a')]`。
6. 梯度 log policy 有统一 bound：`||nabla log pi_theta||_2 <= 2`。
7. 占用 kernel 定义为 `(1-gamma) sum_{t>=0} gamma^t P_pi^t`。
8. 占用 kernel 有 Lipschitz bound：`|d^{pi'}-d^pi| <= gamma/(1-gamma) |pi'-pi|`。
9. 参数到 occupancy 的 Lipschitz bound 为 `2gamma/(1-gamma)||theta'-theta||_2`。
10. local Lipschitz 常数显式依赖 `gamma,tau,R`，论文给出完整表达式：
    `C_{gamma,tau,R} = 1/(1-gamma) * (gamma(5+tau R)/(1-gamma)+6) * ((1+gamma tau R)/(1-gamma)+tau R) + 2tau/(1-gamma)`。
11. 非均匀 PŁ 常数里出现 `hat R=max(2(1+gamma tau R)/((1-gamma)tau), R)`。
12. full affine PŁ 常数含 `lambda^theta=[int_S lambda_min(G^{pi_theta}(s)) d_rho^{pi*}(ds)]^{-2}`。
13. simplex PŁ 常数把 `G` 替换为 `int_A g g^T pi_theta(da|s)` 的最小特征值。
14. eigenvalue lower bound 的关键因子形如 `e^{-R-2||theta0||_2}`。
15. 证明完全是连续时间 gradient flow，未处理离散步长、采样误差、actor-critic critic error 或 stochastic approximation。
16. 证明依赖 exact policy gradient / exact value evaluation，没有给可执行算法或数值实验。

### 核心结果

1. Result 1: 在 `Q_tau^pi`-realizability 下，建立 general state/action log-linear softmax policy 的非均匀 PŁ inequality。
2. Result 2: 对 full affine span features，证明 KL regularizer 对 `||theta||` 径向无界。
3. Result 3: 对 simplex-valued features，证明 KL regularizer 对 `||theta_perp||` 径向无界。
4. Result 4: full affine + 初始化 FIM 正定时，entropy-regularized policy gradient flow 全局线性收敛。
5. Result 5: simplex + 初始化 uncentered covariance 正定时，同样得到全局线性收敛。
6. Result 6: 三角特征和 Bernstein 特征分别作为两个正例；hat functions 作为反例。
7. surprising 点：tabular softmax 里 KL 不会对参数径向无界，但 continuous/log-linear 特征下 KL 反而可作为 Lyapunov coercive term。
8. surprising 点：simplex features 的 FIM 必然在 `1` 方向奇异，但这不阻止收敛，因为策略对加常数方向不敏感。
9. negative result：常见 finite-element hat basis 可能违反 zero-measure maximizer 条件，导致 KL 沿某方向不发散。
10. 与 tabular Mei et al. 2020 的差异：本文不能用 `sum_a grad=0` 推出 action probability 下界，改用 KL 径向无界控制轨迹。

### 明确局限

1. 这是理论论文，没有 benchmark、没有模拟实验、没有真机或实际 RL 任务验证。
2. 核心 `Q_tau^pi`-realizability 很强：要求任意 policy 的 regularized Q 函数都能被同一特征线性表示。
3. 收敛结果是 continuous-time gradient flow，不等价于实际训练中的 finite step-size policy gradient。
4. 没有处理采样噪声、off-policy 数据、critic approximation error、function approximation misspecification。
5. action space compact、reference measure regular、cost bounded，这些条件排除了不少常见连续控制设定。
6. PŁ 常数显式依赖 `gamma`、`tau`、log-density bound、occupancy density ratio 和 eigenvalue lower bound；实际数值可能非常差。
7. full affine 和 simplex 的 maximizer zero-measure 条件需要逐个 feature family 验证，不能自动套用。
8. 论文中的证明有若干记号/排版瑕疵，例如 simplex proof 中引用了 affine assumption、个别积分域写成 `A`/`S` 混用、`K`/`p` 混用。
9. radial unboundedness 从固定 `u` 推到 uniform coercivity 的部分写得较简略，严格性需要二次审查。
10. 对 `0616-fex` 如果关注实际 agent/data factory 或 empirical feature extraction，这篇只提供理论边界，不提供可直接落地的训练组件。
