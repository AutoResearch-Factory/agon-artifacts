# Confidence interval for the AUC of SROC curve and some related methods using bootstrap for meta-analysis of diagnostic accuracy studies (2004.04339)

- **Year**: 2020.04
- **Authors**: Hisashi Noma (The Institute of Statistical Mathematics, Tokyo), Yuki Matsushima (Graduate University for Advanced Studies / Otsuka Pharmaceutical), Ryota Ishii (Keio University Hospital)
- **Venue**: arXiv preprint (未检索到确认的正式发表期刊信息，元数据仅给出 arXiv 摘要页；不臆测期刊)
- **Keywords**: diagnostic test accuracy (DTA) meta-analysis, SROC curve, AUC confidence interval, parametric bootstrap, bivariate random-effects model, influence diagnostics, leave-one-study-out, R package dmetatools
- **One-liner**: 给 DTA meta-analysis 的 SROC-AUC 提出一个参数化两级 bootstrap 算法构造 95% CI/比较/影响诊断，无小样本覆盖率模拟验证。

## 精华提炼

### 管线/方法

完整管线（三个算法共享同一套模型拟合 + resample 骨架）：

1. **底层模型**：对 N 个诊断研究（每个研究提供一个 2x2 表 TP/FP/FN/TN），用 Reitsma et al. (2005) 双变量随机效应模型。先做 logit 变换：`y_Ai = logit(TP_i/n_Ai)`（敏感度）、`y_Bi = logit(FP_i/n_Bi)`（FPR，即 1-特异度）；研究内层 `y_i ~ N(θ_i, S_i)`，`S_i` 由观测计数直接估计（如 `(TP_i x FN_i / n_Ai)^-1`）视为已知不重抽；研究间层 `θ_i ~ N(μ, Σ)`。
2. **REML 拟合**：标准多元 meta-analysis 框架求 `{μ̂, Σ̂}` 的 REML 估计（Reitsma/Jackson-Riley-White/Mavridis-Salanti 方法），Wald 型 CI 对 μ 有闭式解。
3. **SROC/AUC 推导**：通过 Rutter-Gatsonis (2001) 层级回归框架，`{μ̂, Σ̂}` 唯一确定一条 SROC 曲线，AUC 是该曲线下面积——是 `(μ̂, Σ̂)` 的确定性（非线性、无闭式）函数，不是对原始预测值算出的经验 ROC/PR 曲线积分。
4. **Algorithm 1（AUC 的 bootstrap CI）**：(a) 从 `N(μ̂, Σ̂)` 参数化重抽 N 个新的研究间"真值" `θ_i^(b)`；(b) 从 `N(θ_i^(b), S_i)` 参数化重抽新的研究层 `y_i^(b)`（`S_i` 全程固定不变）；(c) 用重抽出的 N 个 `y_i^(b)` 重新拟合模型、重新推导 AUC，得 `AUC^(b)`；(d) 重复 B 次（建议 >=1000，示例用 2000）；(e) 95% CI = 经验分布的 2.5%/97.5% 分位数——纯 **percentile bootstrap**，无 BCa、无 studentization、无二次 bootstrap。
5. **Algorithm 1'（二项重抽变体）**：给只接受原始计数（TP/FP/FN/TN）而非预汇总 `y` 的软件包（如 R 的 `mada`）用，把步骤 (b)(c) 换成直接从 `Bin(n_Ai, p_Ai^(b))`、`Bin(n_Bi, p_Bi^(b))` 重抽计数，论文声称两版本"结果几乎一致"，但**未在本文内做任何数值对比验证这个声称**。
6. **Algorithm 2（两组 dAUC 检验）**：对两个诊断方法各自独立跑 Algorithm 1 的两级重抽（组 1 用 N1 研究，组 2 用 N2 研究），每次重抽同时算 `AUC1^(b)`、`AUC2^(b)`，取差 `dAUC^(b) = AUC1^(b) - AUC2^(b)`；95% CI 仍是分位数法；bootstrap p 值由 `dAUC^(b)` 经验分布相对 0 的位置得到（论文未给出 p 值的精确计算公式，只描述"用 bootstrap 分布作为参照分布"）。
7. **Algorithm 3（leave-one-study-out 影响诊断）**：定义 `ΔAUC(i) = AUC(全部N个研究) - AUC(剔除第i个研究后的N-1个研究)`。关键细节（从 Table 3 数值反推确认）：每个研究 i 有**自己独立的** bootstrap 参照分布——同一次 bootstrap 重抽样本内同时算全量 AUC^(b) 与剔除 i 后的 AUC^(b)，取差得 `ΔAUC(i)^(b)`，重复 B 次得到该研究专属的分位数区间，再拿真实（非 bootstrap）的 `ΔAUC(i)` 去对照这个专属区间是否越界——Table 3 里不同研究的 bootstrap 2.5%/97.5% 边界确实互不相同（如剔除第5个研究是 `[-0.026, 0.024]`，剔除第12个研究是 `[-0.034, 0.026]`），证实这不是共享同一条全局 CI 的简化做法。

### 关键技术细节

- 全文只字未提任何超出"标准多元 meta-analysis + REML"之外的估计器；模型自由度极简（一个 2x2 相关系数 `ρ` + 两个异质性方差 `σ_A^2, σ_B^2` + 两个总体均值 `μ_A, μ_B`）。
- bootstrap 重抽次数 B：理论建议 `>1000`（引用 Efron & Tibshirani 1994），两个示例统一用 `B=2000`；论文未讨论 B 增大后 CI 是否稳定收敛（无 Monte Carlo error 诊断图或表）。
- 例 1（cervical cancer, Scheidler et al. 1997）：N=44 篇研究，按诊断技术拆成 3 个子集——LAG N=17、CT N=17、MRI N=10；每个子集独立拟合双变量模型、独立算 AUC 及其 bootstrap CI。
- 例 2（asthma, Korevaar et al. 2015）：单一诊断指标（呼出气一氧化氮）N=12 篇研究。
- 两个例子的研究数量级（N=10~44）与本任务 `>=50 storm systems` 同一数量级、同属"个位数到几十/大几十"的小样本 regime，这正是该论文被选中精读的原因。
- 影响诊断（Algorithm 3）不是本文原创——是把 Matsushima et al. (2020) 提出的 `ΔAUC` 影响统计量从贝叶斯框架搬到频率派框架，用 bootstrap 给出可操作的显著性阈值（贝叶斯版本无法给出可量化的判定阈值，这是本文这部分的唯一增量）。
- 软件实现：R 包 `dmetatools`（`github.com/nomahi/dmetatools`），三个算法各对应 `AUC_boot()`、`AUC_comparison()`、`AUC_IF()` 三个函数，Appendix 给出完整可运行示例代码。

### 明确局限

- **全文没有任何模拟（simulation）验证**：两个例子都是真实已发表数据集的一次性应用展示，不是覆盖率（coverage）的统计验证——没有"生成已知真值的数据、跑 K 次 bootstrap、统计 95% CI 实际覆盖真值的比例"这类实验，因此**无法从本文直接读出任何小样本覆盖率数字**。
- Discussion 段（原文 518-521 行）唯一一句涉及覆盖率的话——"its large sample approximation can have better coverage performances compared with Wald-type confidence intervals"——引用的是 Noma et al. (2018)（网络 meta-analysis 的 Bartlett 校正）与 Ukyo et al. (2019)（不完整纵向重复测量混合效应模型的小样本改进推断），**两篇引用文献针对的都是不同统计量（不是 AUC/SROC）**，是间接类比，不是本文方法自身的覆盖率证据。
- Algorithm 1 与 1' 是否给出"几乎一致"的结果只是断言，本文未提供任何数值对比（如两版本在同一数据集上的 CI 是否重合）来支撑这个说法。
- 影响诊断阈值本质上是"该研究专属的 bootstrap 抽样噪声区间"，不是传统假设检验意义上校正过多重比较的阈值——N 越小，个别研究越容易被标记为"influential"，本文未讨论这个假阳性率如何随 N 缩小而上升。
- 重抽机制是**参数化**的（从拟合出的双变量正态分布重抽"新研究"），前提是双变量随机效应模型设定正确；如果真实异质性分布明显偏离正态或研究间存在未建模结构，bootstrap CI 的名义覆盖率可能失真——论文未讨论模型误设下的稳健性。
- 样本单位是"研究"（每个研究已经是对大量患者的高度汇总统计量），与本任务"storm-level"重抽单位（每个 storm 是相对原始的预测/标签对）不是同一层级的可比数据结构。

- **不能直接借鉴的（结构性错配）**：本文的 resample 机制是**参数化两级重抽**——从拟合好的双变量正态随机效应分布重抽"新研究"，AUC 是重新拟合模型后由 `(μ,Σ)` 确定性推导出的模型曲线下面积；而本任务需要的是**非参数化 storm-cluster（case resampling）bootstrap**——直接对 storm 单位做有放回重抽，AUPRC 是对重抽出的 storm 集合汇总原始预测/标签算出的经验 PR 曲线积分。两者的 resample 对象、estimand 定义方式均不同，无法把 Algorithm 1 的具体重抽公式照搬到 storm-cluster bootstrap 上。
- **对 gap (2) 的直接回答**：**不能提供小样本覆盖率的可引用参考基准**——全文没有任何覆盖率模拟数字，Discussion 里唯一相关的一句话是转引他文关于不同统计量的间接论断，不构成本方法自身的覆盖率证据。能提供的只是**间接、定性的佐证**：在 N=10~44 研究（与 `>=50 storm` 同量级的小样本 regime）的真实应用中，95% percentile bootstrap AUC CI 的宽度普遍落在 0.09~0.29 区间（多数在 0.19~0.29），且宽度主要由数据异质性而非 N 决定——这与 idea v10 review 记录的 `n=8` seed-bootstrap CI 宽度 `±0.08-0.09`（对应全宽约 0.16-0.18）量级吻合，进一步印证 proposal 现有 Validation Focus 弱项（storm-cluster bootstrap CI 能否被压到 0.01 实用阈值内需要功效分析）不是过虑，而是这一类小样本 percentile bootstrap 方法的普遍现象。**结论：gap (2) 未被这篇论文实质性关闭**，proposal 仍需自己做 storm-cluster bootstrap 在预期 N 下的功效/覆盖率模拟，或另找专门研究"小 cluster 数下 cluster bootstrap 覆盖率"的文献（更对口的方向是计量经济学的 wild cluster bootstrap 文献，如 Cameron-Gelbach-Miller 一脉，专门研究 G=10-50 量级 cluster 数下的 bootstrap 覆盖率，这与本任务 storm-cluster bootstrap 的重抽结构更接近，比 DTA meta-analysis 文献更值得作为下一轮搜索方向）。

## Read by: 0710-causal-scs (2026-07-13)

精读结论：**本文不能直接关闭 gap (2)**。原因有二，且都是结构性的，不是"读得不够细"能解决的：

1. **本文没有覆盖率模拟**。它是方法+软件展示论文（两个真实数据集的一次性应用），不是验证论文。全文唯一涉及"bootstrap 覆盖率优于 Wald"的句子转引自另外两篇统计不同量的论文（网络 meta-analysis 的 Bartlett 校正、纵向重复测量混合效应模型），不构成 AUC-bootstrap 本身的覆盖率证据。因此无法从本文摘取任何"N=X 时期望覆盖率 Y%"这类可直接引用的数字。
2. **resample 机制结构错配**。本文是参数化两级 bootstrap——从拟合好的双变量正态随机效应分布重抽"新研究"，AUC 由重新拟合的模型参数确定性推导；本任务需要的是非参数化 storm-cluster（case resampling）bootstrap——直接对 storm 单位重抽，AUPRC 是对重抽出的原始预测/标签算经验 PR 曲线积分。两者的重抽对象、estimand 生成方式都不同，Algorithm 1 的具体步骤无法照搬。

能提供的价值是间接、定性的：(a) 确认 percentile bootstrap（非 BCa）+ B>=1000-2000 是这类问题的领域标准做法，proposal 现有设计方向没有方法论风险；(b) 两个真实示例在 N=10~44 时 95% CI 宽度普遍 0.09~0.29（多数 0.19~0.29），量级上与 idea v10 review 记录的 `n=8` CI 宽度吻合，为"`>=50` storm 未必能把 CI 压到 0.01 阈值内"这一功效顾虑提供了跨领域的间接佐证，但不是正式基准；(c) Algorithm 3 的逐个体专属 bootstrap 参照区间做影响诊断，是未来 P3 归因阶段"哪些 storm 主导了因果特征增量"分析的一个可迁移模板（本轮不需要用）。

**给下一轮搜索的建议**：这个方向的更优候选不在 DTA meta-analysis 文献里，而在计量经济学的 **wild cluster bootstrap** 文献（Cameron-Gelbach-Miller 一脉）——那支文献专门研究"cluster 数量 G 很小（10~50 量级）时 cluster-robust bootstrap 的实际覆盖率"，重抽结构（对 cluster 做 case resampling）与本任务 storm-cluster bootstrap 高度一致，很可能有本文缺失的、直接可用的小样本覆盖率模拟表。此外 Ukyo et al. (2019)（标题直指"small sample inference"）也值得快速查一遍摘要，确认是否有可迁移的覆盖率数字，即便其目标统计量（重复测量混合效应模型）与 AUC 不同。
