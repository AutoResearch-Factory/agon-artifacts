- **Year**: 2026.04
- **Authors**: Xiaoxiao Liang, Juyuan Zhang, Liming Pan, Linyuan Lü (USTC, School of Cyber Science and Technology)
- **Venue**: ICML 2026 (submitted)
- **Keywords**: Relational Inference, Neuro-symbolic AI, Symbolic Regression, Graph Learning, LLM-Guided Search, Dynamical Systems, Interpretability
- **One-liner**: COSINE 框架联合优化隐式交互图和稀疏符号动力学，用 LLM 外循环反馈剪枝/扩充基函数库，在合成和真实疫情数据上实现结构恢复+机理发现双 SOTA。

## 精华提炼

### 管线/方法

COSINE 是一个两层闭环系统：

**内层（Joint Optimization）**：
1. **图生成模块**：用 Gumbel-Softmax 参数化每条有向边 $A_{ij}$ 的二元分布（有/无边），latent variables $\Psi_{ij}^{0/1}$ → soft adjacency $A_{ij}^{\text{soft}}$
2. **稀疏符号回归**：将动力学分解为 Message（边特征：$x_i, x_j$ → $\mathbf{e}_{ij}$）和 Update（节点特征：$x_i, \mathbf{h}_i, k_i$ → $\Delta \mathbf{x}_i$）两个阶段
3. **Message 阶段**：线性组合基函数 $\Theta_{\text{msg}} = \{\phi_1, \dots, \phi_M\}$，$\mathbf{e}_{ij} = \sum_{m} \mathbf{W}_{\text{msg}}[m] \odot \phi_m^{\text{msg}}(x_i, x_j)$
4. **聚合**：$\mathbf{h}_i = \sum_j A_{ij}^{\text{soft}} \mathbf{e}_{ij}$
5. **Update 阶段**：线性组合基函数 $\Theta_{\text{upd}}$，$\Delta \mathbf{x}_i = \sum_n \mathbf{W}_{\text{upd}}[n] \odot \phi_n^{\text{upd}}(x_i, \mathbf{h}_i, k_i)$
6. **一步预测**：$\hat{\mathbf{x}}_i^{t+1} = \mathbf{x}_i^t + \Delta \mathbf{x}_i^t$（连续时间看作 Euler 离散化）
7. **损失**：$\mathcal{L}_{\text{total}} = \mathcal{L}_{\text{nll}} + \beta_{\text{KL}} \mathcal{L}_A + \lambda_W \mathcal{L}_W$（NLL + KL 图先验 + L1 系数稀疏）

**外层层（LLM-Guided Library Evolution）**：
1. 内层收敛后提取验证 Loss、系数重要性（mean |W|）、残差模式
2. LLM 分析反馈 → 剪枝低权项 + 扩充新项 → 生成候选库 $\widetilde{\Theta}$
3. Best-so-far 策略：仅当 $\mathcal{L}_{\text{val}}(\widetilde{\Theta}) < \mathcal{L}_{\text{val}}(\Theta^\star)$ 时接受
4. 最多 10 轮 outer loop，每轮 1000 epoch 内层优化

### 关键技术细节

- **LLM 模型**：默认 GPT-OSS (20B)，本地 Ollama 部署在 RTX 4090 上；疫情数据用 MiMo-V2-Flash (OpenRouter API)
- **LLM 规模实验**：测试了 DeepSeek-R1 (8B/14B/32B)、Qwen3 (8B/32B)、Qwen2.5 (72B)、GPT-OSS (20B/120B)
- **优化器**：Adam，结构学习率 lr=0.1，系数学习率 lr=0.005
- **温度参数**：Gumbel-Softmax τ=0.3
- **正则化**：β_KL=0.1，λ_W=0.1
- **训练模式**：teacher forcing 单步监督（predict x^{t+1} from x^t），评估时可多步 rollout
- **初始基库**：LLM 零样本生成，无需系统特定模板
- **LLM Prompt 设计**：三个独立 prompt（System/Init/Refine），严格变量作用域（Message 只能用 xi/xj/diff，Update 只能用 x/h/deg），禁止 reshape/view/permute/cat/stack 操作，要求 JSON 输出
- **图类型**：ER (p=0.1)、BA (m=2)、WS (k=2, p=0.1)，各 50 节点
- **六种动力学系统**：MM（生化反应）、Diffusion（扩散）、Spring（弹簧）、Kuramoto（同步）、FJ（观点动力学）、CMN（耦合混沌映射）
- **数据规模**：MM/Diff 50×10，Spr 15×10，Kura 30×30，FJ/CMN 20×10（轨迹×采样步）
- **评估指标**：结构恢复用 AUC（edge-level），机理发现用 Term Accuracy（top-3 系数覆盖 ground-truth 原语比例）

### 核心结果

**关系推断 AUC**（Table 1，N=50）：
- COSINE 在 18 个 setting 中的 17 个取得最优或并列最优
- 最难的 Kuramoto+BA：COSINE 99.85 vs NRI 69.70 / GDP 90.13
- Spring 系统所有图类型均达到 100% AUC
- CMN+BA 是最弱项（93.23），但仍然优于 NRI (83.35) 和 GDP (88.83)

**机理发现**（Table 2）：
- Update 模块 Term Accuracy 全 1.00（完美恢复聚合/归一化项）
- Message 模块：Diff/Spr/Kura/FJ 全 1.00，MM 为 0（ground-truth 是 x_j/(1+x_j)，恢复的是 (x_j-x_i)^2 等代理非线性），CMN 为 0.50（缺少 logistic map 原语）
- Kuramoto 成功发现 sin(x_j-x_i)（系数 0.53，最高权）

**LLM 外循环消融**（Figure 2）：
- 固定库（w/o update）在 MM/CMN 等复杂系统上显著衰退
- 阈值剪枝（λ）不稳定且超参敏感
- COSINE 全系统接近 AUC≈1.0

**LLM 规模影响**（Table 3）：
- Spring/Kuramoto：8B 即可达到 AUC≈1.0
- MM/CMN：需要 14B+，8B 模型仅略优于随机

**效率对比**（Figure 3）：
- COSINE 训练时间和 GPU 显存均显著优于 NRI，可扩展至 N=200 节点
- 优势来源：稀疏线性回归取代重型神经网络

**低数据鲁棒性**（Table 4，10×10 样本）：
- COSINE 在所有 setting 上保持最高 AUC
- 特别是在 Diffusion+BA (91.98 vs 次优 68.07) 和 Kura+BA (96.06 vs 92.25) 上优势明显

**疫情真实数据**：
- 推断的图结构反映区域异质性（Arizona/Connecticut 密集，Illinois/Michigan 稀疏）
- Message 普遍发现 x_i·x_j（质量作用传播），Update 区分外部压力（x·h）vs 局部 logistic/负反馈

### 明确局限

1. **复杂动力学对 LLM 要求更高**：MM/CMN 等需要丰富组合原语的系统，8B 模型失效
2. **稀疏符号回归的表达力有上限**：可能无法表示超出基函数组合的更复杂机制
3. **有限噪声数据下的不可辨识性**：多个近似等价项可能模糊机理结论
4. **Message 模块的代理非线性问题**：在 MM 系统中，虽然结构恢复极好，但发现的原语（(x_j-x_i)^2）与 ground-truth（x_j/(1+x_j)）不完全一致——这说明当多种非线性可近似相同效应时，符号发现并非唯一
5. **仅限 pairwise 交互**：message-update 分解假设 pairwise 交互，不支持三体及以上的高阶作用
6. **单步监督训练**：可能遗漏长期依赖
7. **LLM 外循环开销**：每轮需完整训练，10 轮即 10× 训练代价
8. **无真机/物理实验验证**：仅在数值仿真和观测数据上验证，未在受控物理系统上测试

- **Tier B**：主要扣分在 Reproducibility（匿名代码 + prompt 不完整）和 Independence（无第三方验证）
