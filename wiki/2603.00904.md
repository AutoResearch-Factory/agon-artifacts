# From Frequency Bias to Spectral Balance: Operator-Aware Preconditioners for PINNs (2603.00904)

- **Year**: 2026.03
- **Authors**: Roy Y. He (City University of Hong Kong), Ying Liang (Duke University), Hongkai Zhao (Duke University), Yimin Zhong (Auburn University)
- **Venue**: arXiv
- **Keywords**: PINN, spectral bias, operator preconditioning, Green's function, elliptic PDE, multiscale PDE, MMNN, frequency dynamics
- **One-liner**: 用 Green 积分预条件器抵消 PINN 中微分算子的高频偏置。

## 精华提炼

### 管线/方法
- 论文研究对象是二阶椭圆 PDE 的 PINN 求解, 重点不是提出新网络, 而是解释并修正训练目标的频谱偏置。
- PDE 写成 `Lu=f` in `Omega`, `u=g` on boundary, 其中 `L` 是二阶椭圆算子。
- 标准 PINN 损失为 `L_PINN = L_residual + lambda_b L_boundary`, 残差项直接惩罚 `Lu_theta - f`。
- 直接函数近似的对照损失为 `0.5 int |u_theta-u|^2 dx`, 对应 `L2` 解误差。
- 核心观察: PINN 残差项近似惩罚 `||L(u_theta-u)||_L2`, 对 Poisson 类问题相当于更接近高阶 Sobolev 误差, 会放大高频误差。
- 这会和神经网络本身的低频偏置竞争, 最终频谱偏置取决于网络表达偏置和微分算子偏置的平衡。
- 对两层网络, 作者引用其前作说明低频偏置仍可能占主导, 高频分量难学。
- 对设计良好的多层网络, 如 Sine 激活加首层缩放、FMMNN/MMNN, 网络自身低频偏置较弱, 于是微分算子的高频放大主导训练。
- 主张: 对弱频偏网络, 标准 PINN 会优先压低高频残差, 反而迟迟学不会低频/光滑解分量。
- 预条件器做法是把残差先过一个固定积分算子 `G`, 再计算平方损失。
- 新残差项为 `L_G,r = 0.5 int | int G(x,y)(Lu_theta(y)-f(y)) dy |^2 dx`。
- `G` 是参考椭圆算子 `L0` 的 Green 函数, 理想情况 `G=L0^{-1}` 且 `L0` 与 `L` 谱等价。
- 在这种情况下, `G L` 近似抵消微分算子的频率放大, 使 `L_G` 更接近 `||u_theta-u||_L2`。
- 直观上, 预条件 PINN 把 residual learning 从高阶残差度量拉回到解误差度量。
- 1D 区间 `(-1,1)` 中, 他们使用齐次 Dirichlet Laplacian 的解析 Green 函数。
- 2D/3D 中, 对复杂域和变系数问题使用自由空间 Green 函数作为近似预条件器。
- 2D 自由空间核为 `-(1/(2*pi)) log(|x-y|)`, 3D 核为 `-1/(4*pi|x-y|)`。
- 实现上还使用截断积分域: 对每个 `x`, 只在以 `x` 为中心、半径等于到边界距离的球内对 `y` 求和。
- 截断的两个目的: 避免近边界积分域不对称破坏核近似, 并降低求和成本。
- 截断还保留 Green 函数奇异性, 作者认为高频渐近主要由这个局部奇异性决定。
- 离散实现里 `x_i` 和 `y_ij` 在测试中取网格点, 以保留可解析频率模式的谱性质。
- 作者指出 NN 可在任意点评价和自动微分, 因此可设计更自适应的 meshless 采样。
- 边界项方面, 预条件后残差项近似 `L2` 解误差, 因而和 Dirichlet 边界误差的量纲更匹配。
- 所有预条件实验中边界权重 `lambda_b` 设为 1, 作者称性能对该参数不敏感。
- 计算成本方面, 额外 Green 核求和可用 FMM 或 HSS 加速, 但实验中主要成本仍是 NN 导数和反传。
- 论文的理论段证明截断 Green 损失弱于 `H^{-1+eps}` 型残差范数, 即至少回收约 `1-eps` 阶导数。
- 在带 Fourier 系数衰减假设时, 理论上可能回收约 `3/2` 阶导数。
- 这个理论不等同于完整收敛证明, 更像是解释预条件损失的频谱弱化效应。

### 关键技术细节
- 实验统一使用全 batch 训练。
- 激活函数主要使用 Sine, 首层权重按 `n_1^(1/d)` 缩放, 其中 `n_1` 是首层宽度, `d` 是输入维度。
- 早期函数近似实验用 Adam 加 cyclic learning rate, 下界 `1e-5`, 上界随网络/实验设置不同并带 `gamma=0.999` 衰减。
- Example 1: FCNN 宽 50、深 3、5201 参数, 目标 `sin(2*pi*x)+sin(5*pi*x)+sin(9*pi*x)`, 2000 采样点。
- Example 1 结论: 未缩放的 Tanh/Sine/ReLU 都呈低频偏置; 首层缩放后的 Sine 最接近无偏, 最终误差约 `6.09e-03`。
- Example 2: 宽 100 的 2/4/6 层 FCNN, 参数量分别为 301、20501、40701。
- Example 2 结论: 增加深度会减弱频率偏置; 缩放 Sine 的深网各模式可到 `1e-4` 或更好。
- Example 3: 4 层、宽 200 FCNN, 4000 采样点, 81001 参数, 比较 `ReLU^3` 与缩放 Sine 在 approximation/PINN 中的频谱行为。
- Example 3 关键数字: approximation `ReLU^3` 误差 `6.00e-01`, approximation Sine 误差 `6.91e-02`。
- Example 3 关键数字: PINN `ReLU^3` 误差 `1.85e-01`, PINN Sine 误差 `1.83e+00`。
- 这个反直觉结果是全文动机: 表达能力更强、低频偏置更弱的 Sine 网络, 在标准 PINN 中反而被微分算子高频偏置拖垮。
- Example 4: 1D Poisson, 解为 `u1`, FCNN 宽 50 深 3, 5251 参数, 2000 collocation 点。
- Example 4 训练时间: direct approximation 16s, standard PINN 24s, preconditioned PINN 26s。
- Example 4 结论: 预条件 PINN 的频率模式动态接近函数近似, `L2` 误差下降明显优于标准 PINN。
- Example 5: 1D Poisson, 局部高频解 `u2`, FCNN 宽 200 深 3。
- Example 5 训练时间: direct approximation 32s, standard PINN 38s, preconditioned PINN 41s。
- Example 5 结论: 标准 PINN 对高频残差过度偏置, 低频模式收敛很慢; 预条件后几乎恢复函数近似型频谱动态。
- Example 6(a): 1D 变系数椭圆, `A=1.8`, `epsilon=0.1`, FCNN 宽 100 深 3, 20501 参数。
- Example 6(a) 关键数字: standard PINN `L2` 相对误差 `9.61e-01`, preconditioned PINN `1.80e-02`, approximation `1.88e-03`。
- Example 6(b): 同一变系数问题用 FMMNN, 宽 50、rank 20、深 3, 2091 参数。
- Example 6(b) 关键数字: standard PINN `9.78e-01`, preconditioned PINN `3.07e-02`, approximation `4.16e-03`。
- Example 6 显示预条件和 MMNN 是互补的: 架构降参数, 预条件修 loss 的谱偏置。
- Example 7: 2D 方形域, 四个不同尺度 bump, MMNN 宽 200、rank 20、深 3, 8241 参数, 10000 点。
- Example 7 关键数字: 2000 epochs 后 approximation `4.38e-02`, standard PINN `4.30e-01`, preconditioned PINN `6.15e-02`。
- Example 7 训练时间: approximation 217s, standard PINN 256s, preconditioned PINN 259s。
- Example 7 误差形态: standard PINN 误差集中在最大低频 bump, 预条件和直接近似误差更偏高频/随机。
- Example 8: 2D unit disk, 非齐次 Dirichlet, exact solution 为两个 sinusoidal 项组合, MMNN 宽 200、rank 40、深 3, 16281 参数, 7854 点。
- Example 8 最终正文给出训练时间: approximation 231s, standard PINN 250s, preconditioned PINN 253s, 现象与 Example 7 类似。
- Example 9: 2D 变系数多尺度问题, `A=1.8`, `epsilon=0.3`, FMMNN 宽 200、rank 40、深 3, 16281 参数, 10000 点。
- Example 9 关键数字: 5000 epochs 后 standard PINN `1.39`, preconditioned PINN `1.02e-01`, approximation `5.75e-02`。
- Example 9 说明即使自由空间 Laplacian Green 核不是变系数算子的精确逆, 仍可部分抵消微分算子的频谱偏置。
- Example 10: 3D unit ball Poisson, 三个 compact bump, MMNN 宽 200、rank 40、深 3, 16281 参数, 65450 点。
- Example 10 用球 Bessel 函数和球谐函数做模式分解, 定性结果与 2D 类似: 预条件明显减弱 operator-induced frequency bias。

### 核心结果
- 最重要的科学结论是: PINN 训练失败不只来自网络表达的低频偏置, 还可能来自 PDE 残差 loss 对高频的过度放大。
- 当网络本身很弱或偏低频时, operator high-frequency bias 可能与 network low-frequency bias 部分抵消。
- 当网络本身很强、频谱较平衡时, 标准 PINN 的 operator bias 反而成为主导瓶颈。
- 预条件 PINN 的目标是把训练动态从 `H2`/residual 取向拉回接近 `L2` 解误差取向。
- 1D 简单 Poisson 中, 预条件器几乎恢复函数近似的频谱学习曲线, 代价只比标准 PINN 多约 2-3 秒。
- 1D 高局部频率解中, 预条件 PINN 明显解决标准 PINN 对低频模式长期不收敛的问题。
- 1D 变系数问题中, 标准 PINN 接近失败, 误差约 `0.96-0.98`, 预条件可降到 `0.018-0.031`。
- 2D 方形多尺度 bump 中, 预条件 PINN `0.0615` 接近直接近似 `0.0438`, 明显优于标准 PINN `0.430`。
- 2D 变系数多尺度问题中, 标准 PINN 误差高达 `1.39`, 预条件降到 `0.102`, 接近直接近似 `0.0575`。
- 2D/3D 结果支持一个工程结论: 即使用近似 Green 核, 只要抓住椭圆算子的高频渐近, 也有可观收益。
- 论文最反直觉处在 Example 3: 更会表示高频的 Sine-scaled 网络在标准 PINN 中比 `ReLU^3` PINN 更差, 因为微分算子主导后导致强高频偏置。

### 明确局限
- 论文没有看到公开代码或可直接复现实验脚本, 只有 TeX、图和较多超参数/误差数字。
- 实验主要是合成椭圆 PDE, 没有真实工程 PDE 数据集或社区标准 benchmark。
- 预条件器目前聚焦 elliptic/PINN residual loss, 对 parabolic/hyperbolic、Deep Ritz、operator learning 只在结论中展望。
- 2D/3D 使用自由空间 Green 函数和截断局部积分, 对复杂边界/强变系数的误差控制还不完整。
- 理论分析说明了截断核的 Sobolev 弱化效应, 但没有给出完整的非凸 NN 训练收敛保证。
- 实验大多 full batch, 对 mini-batch、随机采样误差、adaptive sampling 的稳定性没有系统报告。
- 计算成本声明合理, 但没有 FMM/HSS 实装对比, 大规模 3D 问题下 `O(N^2)` 核求和仍可能成为问题。
- 对 `lambda_b=1` 的稳健性主要是经验结论, 没有系统 sweep 表格。
- 反引文为 0, 说明社区独立复核尚未出现; 这是 2026.03 新论文的时间因素, 但仍降低可靠性。
- 与最近 2026 的 Green-integral/Helmholtz 类工作存在邻近关系, 需要在 novelty framing 中避免把“积分形式神经 PDE 求解”泛化为唯一贡献。
