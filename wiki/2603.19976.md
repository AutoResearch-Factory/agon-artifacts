# Constrained Nonnegative Gram Feasibility is \exists R-Complete (2603.19976)

- **Year**: 2026.03
- **Authors**: Angshul Majumdar, IIIT Delhi
- **Venue**: arXiv
- **Keywords**: constrained nonnegative Gram feasibility, rank-2 Gram factorization, existential theory of the reals, ETR-AMI, completely positive matrices, affine Gram constraints, nonnegative matrix factorization, fixed-rank hardness
- **One-liner**: 证明带仿射约束的非负 Gram 可行性在 rank 2 已经是 \exists R-complete。

## 精华提炼

### 管线/方法

1. 论文研究的问题是：给定部分指定的对称矩阵条目和若干关于 Gram 条目的仿射关系，是否存在非负矩阵 \(H \in R_+^{n\times r}\)，使得 \(W=HH^\top\) 满足这些条目和约束。
2. 目标不是求解一个数值优化问题，而是判定 exact feasibility；因此复杂度类自然落在 existential theory of the reals，即 \(\exists R\)。
3. 论文先证明 membership：把 \(H\) 的每个元素 \(h_{ik}\) 当作实变量，Gram 条目 \(W_{ij}\) 展开为 \(\sum_k h_{ik}h_{jk}\)。
4. 指定条目变成二次多项式等式，非负性变成 \(h_{ik}\ge 0\)，仿射 Gram 约束在代入后也变成多项式等式。
5. 因此 constrained rank-\(r\) nonnegative Gram feasibility 属于 \(\exists R\)。
6. hardness 的源问题是 ETR-AMI：变量 \(x_i\ge 0\)，约束只允许三类：\(x_i=1\)、\(x_i=x_j+x_k\)、\(x_i=x_jx_k\)。
7. 这是一种算术 normal form，用常数、加法、乘法来表达一般实闭域存在可行性。
8. reduction 的核心是把每个算术变量 \(x_i\) 编码成 rank-2 非负向量 \(h_{u_i}=(x_i,1)\)。
9. 为了固定坐标系，构造两个 anchor row \(e,f\)，指定 \(W_{ee}=1\)、\(W_{ff}=1\)、\(W_{ef}=0\)。
10. 在 \(R_+^2\) 中，两个非负单位向量内积为 0 时支撑必须不相交，所以它们只能是两个坐标轴方向。
11. 因此可在坐标交换意义下固定 \(h_e=(1,0)\)、\(h_f=(0,1)\)。
12. 对每个变量 row \(u_i\)，指定 \(W_{fu_i}=1\)。
13. 因为 \(h_f=(0,1)\)，该约束强制 \(h_{u_i}\) 的第二坐标等于 1。
14. 非负性保留第一坐标 \(x_i\ge 0\)，从而得到唯一形态 \(h_{u_i}=(x_i,1)\)。
15. 变量值通过 \(W_{eu_i}\) 线性读出，因为 \(\langle (1,0),(x_i,1)\rangle=x_i\)。
16. 常数约束 \(x_i=1\) 被翻译成仿射约束 \(W_{eu_i}=1\)。
17. 加法约束 \(x_i=x_j+x_k\) 被翻译成 \(W_{eu_j}+W_{eu_k}-W_{eu_i}=0\)。
18. 乘法约束 \(x_i=x_jx_k\) 利用 variable rows 的内积：\(W_{u_ju_k}=x_jx_k+1\)。
19. 因此乘法被翻译成 \(W_{u_ju_k}-W_{eu_i}-1=0\)。
20. 整个 target instance 只需要 \(n+2\) 个 rows：两个 anchors 加每个源变量一个 row。
21. 指定 Gram 条目数量为 \(O(n)\)：三个 anchor 条目加每个变量一个 \(W_{fu_i}=1\)。
22. 每个 ETR-AMI 约束只生成一个仿射 side constraint。
23. 所有系数都在 \(\{-1,0,1\}\)，常数在 \(\{0,1\}\)，所以编码长度线性增长。
24. soundness：若 ETR-AMI 有非负解，就显式构造 \(h_e,h_f,h_{u_i}\)，所有 Gram 条目和 side constraints 自动满足。
25. completeness：若目标 Gram instance 可行，anchor rigidity 先固定坐标，再由 \(W_{fu_i}=1\) 抽出 \(x_i\)，仿射约束逐条还原原始算术约束。
26. 由此得到 constrained rank-2 nonnegative Gram feasibility 是 \(\exists R\)-hard。
27. 结合 membership，得到 rank 2 情况的 \(\exists R\)-complete。
28. 论文还给出 fixed rank \(r\ge 2\) 的推广。
29. 一种简单传播论证是把 rank-2 向量嵌入到 \(R_+^r\) 的前两维，其余维度置零，Gram entries 不变。
30. 论文也写了更显式的 higher-rank anchor frame：引入 \(r\) 个 anchor rows \(e_1,\dots,e_r\)，指定 \(W_{e_ae_b}=\delta_{ab}\)。
31. 非负性加正交单位约束会强制这些 anchors 成为标准基向量，至多差一个全局坐标置换。
32. higher-rank 变量 row 通过 \(W_{e_2u_i}=1\) 和 \(W_{e_tu_i}=0\) for \(t=3,\dots,r\) 被限制为 \((x_i,1,0,\dots,0)\)。
33. 然后 \(W_{e_1u_i}=x_i\)，\(W_{u_ju_k}=x_jx_k+1\)，常数、加法、乘法约束完全同 rank-2。
34. 因为 \(r\) 是 fixed，额外 anchor 和条目只带来常数阶或 \(O(r^2)\) 开销，所以仍为多项式 reduction。

### 关键技术细节

1. 输入模型：部分指定 symmetric matrix entries，若干 affine constraints over selected \(W_{ij}\)，rank parameter \(r\)。
2. 变量模型：目标 instance 的 existential variables 只有 \(H\) 的 entries；\(W\) 只是 \(HH^\top\) 的符号记法。
3. rank-2 anchors：\(W_{ee}=1\)、\(W_{ff}=1\)、\(W_{ef}=0\)。
4. anchor rigidity 依赖非负性；若没有 \(H\ge 0\)，两个单位正交向量不一定是坐标轴方向。
5. variable constraint：每个变量只需指定 \(W_{fu_i}=1\)，不需要单独约束 \(u_i\) 的 norm。
6. coordinate extraction：\(W_{eu_i}\) 就是变量值，这是加法和常数约束的线性接口。
7. multiplication gadget：内积 \(\langle(x,1),(y,1)\rangle=xy+1\)，再用 affine constraint 减去 1 和目标变量。
8. reduction source：ETR-AMI，变量域为 \(R_{\ge 0}\)，约束类型为 constant/addition/multiplication。
9. 目标规模：\(n+2\) rows，\(O(n)\) specified entries，\(m\) affine constraints，数字 bit-length 为 \(O(1)\)。
10. 证明风格：纯 theorem/proof，无算法实现、无实验、无 benchmark、无 empirical evaluation。
11. 高阶 rank 推广：用 standard-basis anchor frame 或直接把 rank-2 feasible solution padded with zeros。
12. 复杂度结论：每个 fixed \(r\ge 2\)，constrained rank-\(r\) nonnegative Gram feasibility 是 \(\exists R\)-complete。

### 核心结果

| 结论 | 条件 | 证明依据 | 重要性 |
|------|------|----------|--------|
| Membership in \(\exists R\) | 任意 rank \(r\) | Gram 条目代入为多项式等式/不等式 | 给出上界 |
| Rank-2 hardness | 有部分指定条目和仿射 side constraints | 从 ETR-AMI 多项式归约 | 最小非平凡维度已经足够难 |
| Rank-2 completeness | rank \(r=2\) | membership + hardness | 主定理 |
| Fixed-rank completeness | 每个 fixed \(r\ge 2\) | anchor frame 或 zero padding | 排除“高维才难”的解释 |
| Open problem | unconstrained \(W=HH^\top\) | 本文 reduction 依赖 side constraints | 明确边界 |

1. 反直觉点：rank 2 看起来几何上非常弱，但在非负 orthant + affine Gram constraints 下已经能模拟完整实数存在算术。
2. 另一个关键点：hardness 并不是来自复杂的矩阵尺寸或高 rank，而是来自能对 Gram entries 加仿射关系。
3. 论文没有给出 numerical solver，也没有讨论近似算法性能；它只建立 exact decision complexity。
4. 结果对完全正矩阵、copositive optimization、NMF 相关问题提供复杂度参照，但没有直接证明普通 NMF 或 unconstrained symmetric factorization 的 \(\exists R\)-hardness。

### 明确局限

1. 最大局限是 side constraints：hardness proof 必须使用 affine relations among selected Gram entries。
2. 作者明确说 unconstrained symmetric nonnegative factorization feasibility 的复杂度仍然 open。
3. 论文也没有解决 approximate factorization，即给定 tolerance \(\tau\) 判定 \(\|W-HH^\top\|_F^2\le \tau\) 的复杂度。
4. 没有研究 restricted constraint families 是否可多项式求解，例如只约束 diagonal entries、只约束 bounded number of entries、或把等式换成不等式。
5. 没有 artifact、代码、数据或独立复现材料；这是理论论文，复现主要是逐行检查 proof。
6. 依赖 ETR-AMI normal form，但文中把 normal-form 结果作为标准引用，没有重新证明从一般 ETR 到 ETR-AMI 的细节。
7. 对 \(\exists R\)-hardness 的表达是可信的，但 reduction 本身非常模板化；需要审查是否存在坐标置换处理和 variable-row 约束遗漏，本文对这两点有明示说明。
8. 和实验型或系统型工作关系有限；它更适合作为“可行性问题边界/负面复杂度证据”的引用。

2. 如果 `0616-fex` 研究的是低秩/非负/Gram/PSD/NMF 类可行性，那么它在 claim 层面有明显接近：同样利用低秩非负结构表达代数约束。
3. 如果 `0616-fex` 是经验型建模或系统工具，本文不撞车；它证明的是 constrained exact feasibility 的 worst-case hardness。
4. 可复用的技术是 anchor gadget：用非负性和正交单位 Gram 条目固定坐标轴。
5. 第二个可复用工具是 affine Gram constraint gadget：通过 \(W_{eu}\) 读变量，通过 \(W_{uv}-1\) 得到乘法项。
6. 这套 gadget 可以作为负面结果模板：一旦我们的 setting 允许类似 anchors、variable rows 和 affine constraints，就要警惕 \(\exists R\)-hardness。
7. 差异化空间在于：去掉 affine side constraints、限制可观测 Gram entries、研究 approximate/noisy feasibility、或给出实际可解子类。
8. 如果我们的工作声称某类非负 Gram/NMF 可行性 tractable，必须明确排除本文允许的 side constraints，否则会与主定理冲突。
9. 该文应该引用，尤其用于说明 fixed rank 2 并不自动带来 tractability。
10. 最值得继续追的关键词是 unconstrained symmetric nonnegative factorization、restricted Gram constraints、approximate nonnegative Gram feasibility、\(\exists R\)-complete factorization。
