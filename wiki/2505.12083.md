# Discovering Symbolic Differential Equations with Symmetry Invariants (2505.12083)

- **Year**: 2026.03 (TMLR accepted)
- **Authors**: Jianke Yang, Manu Bhat, Bryan Hu, Yadi Cao (UCSD), Nima Dehmamy (IBM Research), Robin Walters (Northeastern), Rose Yu (UCSD)
- **Venue**: TMLR (Transactions on Machine Learning Research)
- **Keywords**: symbolic regression, PDE discovery, Lie symmetry, differential invariants, sparse regression, genetic programming, symmetry-invariant, equivariant learning
- **One-liner**: 将微分不变量作为符号回归的原子变量以强制物理对称性, 使发现的 PDE 自动满足给定对称群, 兼容 SINDy/GP/Transformer 多种方法.

## 精华提炼

### 管线/方法

1. **输入**: 已知的对称群 (Lie 点对称, 由 infinitesimal generator 向量场指定) + PDE 观测数据 (spatiotemporal grid 上的 (x, u) pairs) + 一个 base SR 算法 (SINDy / GP / Transformer).
2. **计算微分不变量**: 用 prolongation formula 计算给定对称群的所有 n 阶微分不变量 `η^1,...,η^K`。具体步骤: (a) 先解低阶不变量的 PDE `v^(n)(η)=0`, 获得 p 个水平 Jacobian 非退化的不变量; (b) 用 Prop. 4 (基于 Olver 1995 Thm 5.48) 的递归公式 `det(D_i η̃^j)/det(D_i η^j)` 从低阶构造高阶不变量; (c) 实践中将代数复杂的不变量转换为物理可解释的形式 (如 Laplacian for rotation)。
3. **SR 适配**:
   - **General explicit SR (GP/Transformer)**: Algorithm 1 -- 对每个不变量 `η^k` 作为 LHS, 其他不变量作为 RHS features, 跑 base SR; 选相对误差最小的方程。
   - **Sparse regression (SINDy)**: (a) 选一个含原始 LHS 项 (如 `u_tt`) 的不变量作为新 LHS; (b) 直接用不变量作为 SINDy library 的输入 (SI-raw); 或 (c) 将对不变量做函数组合映射回原 SINDy library, 把对称条件转换为对参数 W 的线性约束 (Prop. 1, 等效于 EquivSINDy 推广到 PDE)。
   - **弱形式 SINDy (WSINDy)**: 通过线性约束方法 (`W=Q^ijk β^i`) 直接嵌入 WSINDy 的 integration-by-parts 目标函数。
   - **D-CIPHER**: 用不变微分算子替换手工定义的 extended derivatives, 自动生成微分函数字典。
4. **松弛对称 (Relaxed symmetry)**: 对不完美对称系统, 用 RPP (Residual Pathway Prior) 方法: `W = A + B`, 其中 `A` 在对称子空间 (spanned by Q), `B` 在正交补空间, 对 B 加更强的正则化。
5. **输出**: 满足指定对称的符号 PDE, 可选地展开回原始 jet 变量。

### 关键技术细节

- **核心定理**: Olver (1993) Prop. 2.56: PDE F(x, u^(n)) = 0 承认对称群 G 当且仅当它等价于 G 的不变量函数 F̃(η^1,...,η^k) = 0。
- **Prolongation formula**: `pr^(n) v = ξ^i ∂_xi + φ_J^α ∂_uα_J`, 其中 `φ_J^α = D_J Q^α + ξ^i u^α_{J,i}`。
- **递归构造高阶不变量** (Prop. 4): 给定 p 个水平 Jacobian 非退化的不变量 `η^1,...,η^p` 和一组严格 n 阶不变量 `ζ^1,...,ζ^{q_n}`, 则 `det(D_i η̃^j_(k,k'))/det(D_i η^j)` 构成完备的严格 (n+1) 阶不变量集。这是 paper 的数学贡献之一 (自行证明, 源自 Olver 1995 Thm 5.48)。
- **SINDy 对称约束的线性子空间** (Prop. 1): 如果 invariant terms `η_0^i = T^ijk θ^k ℓ^j` 和 `(θ')^i = S^ij θ^j` 都编码在 SINDy library 中, 则满足对称的 W 落在 `R^{q×m}` 的线性子空间中。通过 SVD + Sparse PCA 获得稀疏基 Q, 参数化 `W^jk = Q^ijk β^i`。
- **实验数据**:
  - Boussinesq equation (4th-order, scaling+translation symmetry): 256 格点, Fourier pseudospectral + RK4, 2% subsample.
  - Darcy flow (2nd-order, SO(2) rotation): 128x128 格点, PDEBench 生成, 10k subsample.
  - Reaction-diffusion (2 coupled equations, SO(2) in phase space): 128x128, FFT spectral, 10% subsample, 0.05% noise.
  - 3D reaction-diffusion (SO(3) spatial symmetry): 32^3 grid.
  - Channel flow (JHTDB, roto-translation symmetry): 4D spatiotemporal grid of 3D velocity + scalar pressure.
- **Evaluation**: 100 次随机运行, 报告 Success Probability (SP, 方程项完全匹配) 和 Prediction Error (PE, median RMSE)。
- **SINDy 超参**: Boussinesq: threshold 0.25, L2=0.05; RD: threshold 0.05, L2=0.1 (SI), L2=0 (baseline).
- **GP 超参**: Boussinesq: 127 pop x 27 size; Darcy: 127 pop x 64 size, exp operator, no nested exp; RD: 127 pop x 64 size.

### 核心结果

| 场景 | Baseline 最好 SP | SI (Ours) SP | 关键发现 |
|------|-----------------|-------------|---------|
| Boussinesq (SINDy) | 0.00 (PySINDy library 缺 u_x^2) | **1.00** | 对称自动选择正确的 function library |
| Boussinesq (GP) | 0.90 (15 its) | **1.00** (5 its) | SI 效率高 3 倍 |
| Boussinesq (Transformer) | 0.53 | **0.85** | 对称约束显著优于无约束 |
| Darcy flow (GP) | 0.00 | **0.79** | Baseline GP 完全失败, SI 可发现 |
| Darcy flow (Transformer) | 0.00 | 0.00 | Transformer 均失败 (分布偏移) |
| Reaction-diffusion (SINDy) | 0.53 | **0.54** (SI) / **0.56** (SI-aligned) | 极小但稳定的提升; 复杂度从 38 降到 28/14 |
| Reaction-diffusion (GP) | 0.00 | **0.81** | 巨大的提升 |
| RD + noise (WSINDy) | baseline (auto-tuned) | SI 在所有 noise level 持续优于 baseline | auto-tuned 下基线更好, 但 SI 仍胜出 |
| RD + unequal diffusivities | WSINDy 随 ε 增大改善 | SI-relaxed > WSINDy > SI (strict) | 松弛对称在 symmetry breaking 下最优 |
| RD + external forcing | 同上 | SI-relaxed ≈ 1.0 @ ε=0.5, WSINDy ≈ 0.5 | 松弛对称优势明显 |
| 3D RD (SINDy) | 0.51 (C=286) | **1.00** (C=84) | 高维度下优势更显著 |
| Channel flow vs SPIDER | SPIDER vector: 精确 | SI 发现 1 个 spurious term (∇·u) | 与 SPIDER 可比, 但 SI 更通用 |

**Surprising 结果**:
- 预训练 Transformer (E2E) 在两个数据集上完全失败, 仅在 Boussinesq 上工作 -- 说明 PDE 数据分布与其预训练分布差距大。
- WSINDy auto-tuned 参数选择对 baseline 有利但对不同 noise regime 的 SI 不利; default 参数 (更少的 test functions, 更低的 polynomial degree) 反而在高 noise 下对 SI 更好。
- 对 Darcy flow, SINDy 完全不适用 (需要 exp 函数), 但 SI+GP 能发现 -- 说明 SI 不只改善 SINDy, 对 GP 帮助更大。

### 明确局限

1. **需要已知对称群**: 作者明确指出方法假设对称群已知。如果对称群未知或指定错误, 方法失效。提出未来与自动对称发现 (yang2024symmetry, ko2024learning) 结合。
2. **不变量计算的复杂性**: 从 `v^(n)(η)=0` 求解和递归公式应用虽然符号计算可行, 但结果可能代数复杂, 需要 ad-hoc 调整以获得可解释且数值稳定的不变量。这是一次性工作量。
3. **数值稳定性**: 不变量可能包含有理函数 (如分母有 `u_x`), 需要过滤 `|u_x| < 0.1` 的数据点。proposal 的表格显示阈值选择影响 SP (0.0001 时 SP=0.36, 0.001-0.1 时 SP=1.00)。
4. **弱形式下的局限**: Appendix D 详细讨论了将微分不变量直接用于 WSINDy 的三种情况: Case 1 (不变算子为线性微分算子, 不变量为 ordinary function) 可直接积分; Case 2 (不变算子线性但不变量含导数) 只能降低导数阶数不能消除; Case 3 (不变算子含导数) 反可能涉及更高阶导数, 弱形式优势丧失。
5. **只处理标量不变量的 SR**: 对 Channel flow 与 SPIDER 比较时, SI 只能用标量不变量 library, 而 SPIDER 用向量 library 效果更好。SI 发现的方程含一个 spurious term。
6. **SINDy library 局限性**: SI-aligned 方法要求不变量能映射回 SINDy library 函数, 需要 `T` 和 `S` 张量存在, 并非总是可行。
7. **没有真机实验**: 所有实验都是合成数据 + 数值模拟。没有真实物理实验数据验证。
8. **仅考虑 Lie 点对称**: 推广到更高阶对称 (如 Lie-Bäcklund, 广义对称) 或离散对称需要额外工作。
