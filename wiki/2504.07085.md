# Identifying Unknown Stochastic Dynamics via Finite expression methods (2504.07085)

- **Year**: 2025.04
- **Authors**: Senwei Liang (Lawrence Berkeley National Lab), Chunmei Wang (U Florida), Xingjian Xu (U Florida)
- **Venue**: arXiv (SIAM template)
- **Keywords**: FEX, finite expression method, symbolic regression, SDE identification, two-stage framework, stochastic dynamics, interpretable ML, reinforcement learning for equation discovery
- **One-liner**: 用 RL 驱动的符号回归 (FEX) 学 SDE 确定性漂移项 + 免训练扩散模型学随机项，两阶段框架外推泛化远超纯 NN 方法。

## 精华提炼

### 管线/方法

论文提出 **FEX-DM** 两阶段框架，从 SDE 轨迹数据中识别未知随机动力学：

**输入**: 多条 SDE 轨迹 {x_t, x_{t+Δt}}，漂移 μ 和扩散 σ 均未知。

**Stage 1 — 确定性漂移学习 (FEX)**:
- 将漂移项 D(x_t) 建模为有限运算符组合的符号表达式
- 表达为 k-节点二叉树 T，每个节点分配一元或二元运算符
- 一元算子集 U = {sin, cos, exp, 0, Id, (·)², (·)³, (·)⁴}
- 二元算子集 B = {+, -, ×}
- 对每个输出维度 i，构造组合优化问题: min L^(i)(u(·, T, e, θ₁))
- 损失函数 L 是状态差与 u(x_t) 的 MSE
- RL 控制器 (单层 ReLU NN) 采样算子序列 e，策略梯度优化
- 使用 ε-greedy 探索 + top-v 分位数过滤 (仅高分序列用于梯度更新)
- 候选池 N=30，后处理用 Adam + cosine decay (80k iterations)
- Score = (1 + √L(e))⁻¹，两步优化: SGD 初估 + LBFGS 精调

**Stage 2 — 随机扩散学习 (TF-CDM 变体)**:
- 从 Stage 1 得到的 \hat{D} 计算残差: x_{t+Δt} - x_t - Δt \hat{D}(x_t)
- 用 forward-reverse SDE 技术: forward SDE 将残差分布推向标准高斯 (τ: 0→1)
- Forward: dZ_τ = b(τ)Z_τ dτ + σ(τ) dW_τ, 其中 α_τ = 1-τ, β²_τ = τ
- Reverse ODE: dZ_τ = [b(τ)Z_τ - ½σ²(τ)V(Z_τ,τ)] dτ, Z_1 ~ N(0,I)
- 用 Monte Carlo 解 V(Z_τ, τ) (非 NN 方式)
- 构造训练对 (z, y): z~N(0,I) 作输入, ODE 数值解 y 作目标 (Euler, K=10000 步)
- 单隐层 NN (50 neurons, tanh) 学习 z→残差映射
- Adam lr=1e-2, weight decay=1e-6, 2000 iterations

**输出**: 符号漂移表达式 \hat{D}(x) + 生成模型 \hat{S}(z)，完整 SDE 表示为 x_{t+Δt} = x_t + Δt \hat{D}(x_t) + \hat{S}(z), z~N(0,I)

### 关键技术细节

- **树结构**: depth-3 二叉树，3 unary + 1 binary 节点 (L=3)
- **控制器**: 单层 NN + ReLU，输出维度 = |B| + 3|U|
- **RL**: batch size=2，policy gradient，Adam lr=2e-3
- **Score 优化**: Adam lr=8e-3，10000 iterations，后接 LBFGS
- **候选精调**: Adam lr=5e-3 initial → cosine decay, 80000 iterations
- **数据**: Euler-Maruyama 生成，Δt=0.01，T=1.0 (100 时间点/轨迹)，ODE 求解 K=10000
- **评测指标**: 有效漂移 \hat{μ}(x) = E[(x̂_{t+Δt} - x_t)/Δt]，有效扩散 \hat{σ}(x) = Std(x̂_{t+Δt})/√Δt
- **对比基线**: TF-CDM (Liu et al. 2024, 单阶段纯 NN 方法)

### 核心结果

**1D OU 过程** (θ=1.0, μ=1.2, σ=0.3):
- 训练域 [0, 2.5], 测试外推至 x_0 = -6, 1.5, 6
- FEX 仅 90 轮即恢复真漂移: 1.1989 - 0.9953x_t (真: 1.2 - x_t)
- 外推域 [-6,6] 中 \hat{μ} 和 \hat{σ} 始终准，TF-CDM 在外推处明显偏离
- 条件分布在 out-of-domain 输入下仍与 ground truth 一致

**三角函数漂移 SDE** (sin(2πx_t), σ=0.8):
- 训练域 [0,1], 测试外推至 x_0 = -3, 0.6, 3, T=5
- 120 轮找到等价形式: -1.1989 cos(6.2476x_t - 4.6837) - 0.0104
- 外推域 [-5,5] 中漂移和扩散估计准确
- TF-CDM 在外推域标准差和分布形状严重偏离

**双势阱 SDE** (x_t - x_t³, σ=0.5):
- 训练域 [-2,2], 测试 x_0 = -5, 1.5, 5
- 500 轮恢复: -0.9922x_t³ + 0.9709x_t + 0.0019
- 成功捕捉 t=5,10,30,100 时的双模分布演化
- 外推域 [-5,5] 中漂移/扩散估计保持准确

**2D OL 过程** (V = 2.5(x₁²-1)² + 5x₂²):
- 200 轮学习，虽有多余高阶项但系数极小 (如 0.1625x₂³)
- 训练至 T=1.0，外推至 T=5 仍准确
- 展示 FEX 在多维 + 非线性 + 外推场景的优势

**非高斯噪声 (指数噪声 SDE)**:
- μ=-2.0, σ=0.1, η_t ~ exp(1)
- 需减去噪声均值确保零均值驱动
- 200 轮恢复: -1.9751x_t (真: -2x_t)
- 外推域 [-5,7] 准确，TF-CDM 严重偏离

**Surprising 发现**: FEX 在三角函数案例中未恢复精确 sin(2πx_t) 形式，而是找到等价 cos 表示（相位偏移 + 幅度缩放），仍保持预测准确性——说明符号回归的目标是函数等价而非语法等价。

### 明确局限

1. **扩散项假设为状态无关**: Stage 2 假设 S(z) 独立于 x，对 multiplicative noise (σ(x_t)dW_t) 不适用。作者明确提到此为未来工作。
2. **仅处理时齐 SDE**: 不支持非自治系统 (μ(x_t, t), σ(x_t, t) 含显式时间依赖)。
3. **无跳跃过程支持**: 不支持 Lévy 噪声、跳跃扩散等。
4. **FEX 算子集需人工预定义**: 若真漂移包含不在预设集中的算子 (如 tanh, sigmoid)，FEX 无法恢复精确形式。
5. **Stage 2 采用非条件扩散模型**: 只学习残差的全局分布，丢失了条件依赖信息。
6. **无理论保证**: 作者承认缺乏 score-based sampling 的收敛性和误差界分析。
7. **规模有限**: 实验仅到 2D，无高维 (d>10) 验证。FEX 在高维下的搜索复杂性未讨论。
8. **无标准 benchmark 比较**: 未与 SINDy 变体、Neural ODE、或其他符号回归方法做系统比较。仅在单篇 TF-CDM 基线上比较。
9. **Stage 2 依赖特定方法**: 用的是 Liu et al. 2024 的 training-free CDM，未探索其他生成模型 (GAN, VAE, normalizing flow) 作为 Stage 2 的效果差异。
10. **无真机/真实数据**: 全部为仿真数据，无真实物理/生物/金融数据验证。
