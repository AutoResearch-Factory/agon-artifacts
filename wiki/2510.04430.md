# Achieve Performatively Optimal Policy for Performative Reinforcement Learning (2510.04430)

- **Year**: 2025.10
- **Authors**: Ziyi Chen, Heng Huang (University of Maryland, College Park)
- **Venue**: arXiv preprint
- **Keywords**: performative reinforcement learning, performatively optimal policy, performatively stable policy, zeroth-order Frank-Wolfe, entropy regularization, policy-dependent MDP, gradient dominance, derivative-free policy optimization
- **One-liner**: 用 0-FW 首次证明 performative RL 可多项式收敛到 PO 策略。

## 精华提炼

### 管线/方法
- 论文研究的对象是 performative reinforcement learning: MDP 的 transition kernel `p_pi` 和 reward `r_pi` 都依赖部署策略 `pi`。
- 输入问题是有限状态、有限动作的 tabular MDP，给定初始分布 `rho`、折扣因子 `gamma`、策略空间 `Pi`、策略相关环境映射 `p_pi, r_pi`。
- 目标不是传统 RL 的固定环境最优策略，也不是已有 performative RL 的 performatively stable (PS) policy。
- 目标是 performatively optimal (PO) policy: 最大化 `V_{lambda,pi}^{pi}`，即策略 `pi` 被部署到由自身诱导的环境 `(p_pi, r_pi)` 后的熵正则化价值。
- 论文先回顾 PS 策略定义 `pi_PS in argmax_pi V_{lambda,pi_PS}^{pi}`，即在固定为当前策略诱导环境时做最优响应。
- 作者强调 PS 不是 PO 的必要或充分条件，已有 Mandal et al. 2022 已证明 PS 与 PO 可有正的常数差距。
- 新方法把搜索目标改为 `V_{lambda,pi}^{pi}` 自身的 stationary point，而不是 repeated retraining 的 fixed point。
- 论文采用负熵正则 `E[sum gamma^t log pi(a_t|s_t)]`，并说明它可视作 occupancy measure 上的强凸正则。
- 第一段理论管线证明 gradient dominance: 在 regularizer dominance 条件下，任意近似 stationary policy 都是近似 PO。
- 第二段理论管线证明 policy lower bound: 足够 stationary 的策略不会把任何动作概率压到 0。
- 这个 lower bound 允许把搜索限制在紧凸子空间 `Pi_Delta = {pi: pi(a|s) >= Delta}`。
- 在 `Pi_Delta` 上，作者证明 performative value function 具有 Lipschitz continuity 和 Lipschitz smoothness，常数按 `1/Delta` 放大。
- 第三段算法管线构造 zeroth-order performative policy gradient estimator。
- 估计器只要求能评估 `V_{lambda,pi}^{pi}`，不要求知道 `nabla_pi p_pi` 或 `nabla_pi r_pi`。
- 梯度估计使用 two-point estimator: 对随机方向 `u_i` 评估 `pi + delta u_i` 和 `pi - delta u_i` 的价值差。
- 随机方向不能从整个欧式空间采样，因为策略必须满足每个 state 上动作概率和为 1。
- 作者把扰动限制到线性子空间 `L_0 = {u: sum_a u(a|s)=0 for all s}`。
- 实现采样时先从单位球面取 `v_i`，再按 state 去均值投影到 `L_0`，最后归一化得到 `u_i`。
- 为了证明估计误差，作者构造正交变换 `T`，把策略流形映射到 `R^{|S|(|A|-1)}` 上做标准 zeroth-order 分析。
- 第四段算法管线使用 Frank-Wolfe: 每轮用估计梯度在 `Pi_Delta` 上解线性 oracle。
- Frank-Wolfe oracle 有闭式解: 每个 state 选择估计梯度最大的动作给概率 `1 - Delta(|A|-1)`，其他动作给 `Delta`。
- 更新为 `pi_{t+1} = pi_t + beta(tilde_pi_t - pi_t)`。
- 输出规则选择估计 Frank-Wolfe gap 最小的一轮。
- 最终理论保证: 选择合适 `Delta, beta, delta, epsilon_V, N, T` 后，以至少 `1-eta` 概率输出 `D epsilon`-stationary 策略。
- 如果 regularizer dominance 让 `mu >= 0`，该策略同时是 `epsilon`-PO。

### 关键技术细节
- 状态空间 `S` 和动作空间 `A` 均假设有限，策略是 tabular policy。
- 价值函数为 `V_{lambda,pi'}^pi = E_{pi,p_{pi'},rho}[sum gamma^t (r_{pi'}(s_t,a_t) - lambda log pi(a_t|s_t))]`。
- PO 定义: `max_{pi'} V_{lambda,pi'}^{pi'} - V_{lambda,pi}^{pi} <= epsilon`。
- Stationary 定义: `max_{tilde_pi in Pi} <nabla_pi V_{lambda,pi}^{pi}, tilde_pi - pi> <= epsilon`。
- Assumption 1 是环境敏感性: `||p_pi' - p_pi|| <= epsilon_p ||pi' - pi||`，`||r_pi' - r_pi|| <= epsilon_r ||pi' - pi||`。
- Assumption 2 是 `p_pi` 和 `r_pi` 对策略的 Lipschitz smoothness，常数为 `S_p, S_r`。
- Assumption 3 是全策略全环境的 state occupancy lower bound: `inf d_{pi,p}(s) >= D > 0`。
- Theorem 1 的核心常数 `mu` 是正则强度项减去环境 shift 项。
- `mu` 的正项是 `D lambda / (1-gamma)`。
- `mu` 的负项包含 transition sensitivity `epsilon_p`、reward sensitivity `epsilon_r`、transition smoothness `S_p`、reward smoothness `S_r`。
- 当 `lambda` 足够大且 `epsilon_p, epsilon_r, S_p, S_r` 足够小，`mu >= 0`。
- Corollary 1: 任意 `D epsilon`-stationary policy 是 `(epsilon + |mu||S|)`-PO；若 `mu >= 0`，就是 `epsilon`-PO。
- Theorem 2 给出任意动作概率 lower bound: `pi(a|s) >= pi_min * exp[-C <nabla V, pi' - pi>]`。
- 其中 `pi'` 是把每个 state 下最大概率动作和最小概率动作交换后的策略。
- `pi_min` 显式依赖 `|A|, gamma, lambda, epsilon_p, epsilon_r, |S|`。
- Theorem 3: 在 `Pi_Delta` 上 `|V(pi')-V(pi)| <= L_lambda/Delta * ||pi'-pi||`。
- 同时 `||nabla V(pi')-nabla V(pi)|| <= ell_lambda/Delta * ||pi'-pi||`。
- Zeroth-order estimator 的维度因子是 `|S|(|A|-1)`，不是 `|S||A|`。
- 梯度估计误差为 `O(epsilon_V/delta + log(N/eta)/sqrt(N) + delta)`，对应 policy evaluation error、采样方差、finite difference bias。
- Algorithm 1 输入包括 `T, N, Delta > delta > 0, epsilon_V, beta`。
- Theorem 4 的建议超参: `Delta = pi_min/3`，`beta = D*pi_min*epsilon/(36 ell_lambda |S|)`。
- Theorem 4 还要求 `delta = O(epsilon)`，`epsilon_V = O(epsilon^2)`，`N = O(epsilon^{-2} log(eta^{-1} epsilon^{-1}))`，`T = O(epsilon^{-2})`。
- 总 policy evaluations 数量为 `2NT = O(epsilon^{-4} log(eta^{-1} epsilon^{-1}))`。
- 附录还说明二次 occupancy regularizer `1/2 ||d_{pi,p_pi}||^2` 下可按同样思路调整到 PO 收敛。

### 核心结果
- 主 claim 是首次给出 performative RL 中收敛到 PO policy 的多项式时间算法，而不是只到 PS policy。
- 理论突破点 1: 对非凸 `V_{lambda,pi}^{pi}` 证明了 gradient dominance。
- 理论突破点 2: 证明 stationary 附近策略自动远离 simplex 边界，从而处理负熵导致的无界梯度。
- 理论突破点 3: 在策略流形的线性子空间中做 zeroth-order gradient estimation，并给出 high-probability error bound。
- 理论突破点 4: 用 Frank-Wolfe 避免投影到带 lower bound 的策略空间，同时每个 state 的线性 oracle 有闭式解。
- 实验是一个小型模拟环境: 5 states，4 actions，`gamma=0.95`，`lambda=0.5`。
- 模拟 transition 为 `p_pi(s'|s,a) = [pi(a|s)+pi(a|s')+1] / sum_{s''}[pi(a|s)+pi(a|s'')+1]`。
- 模拟 reward 为 `r_pi(s,a)=pi(a|s)`。
- 0-FW 实验跑 401 iterations，`N=1000`，`beta=0.01`，`Delta=1e-3`，`delta=1e-4`。
- 价值函数用 value iteration 评估。
- baseline 是 repeated retraining，每轮用 entropy-regularized natural policy gradient，401 iterations，stepsize 0.01。
- 两个算法都从 uniform policy `pi_0(a|s)=1/4` 开始。
- 作者报告实验总耗时约 110 minutes，硬件是 Apple M1 Pro 8 cores / 16 GB memory，Python 3.9。
- Figure 1 显示 repeated retraining 卡在初始 uniform policy，因为 uniform policy 是 PS。
- Figure 1 同时显示 0-FW 在 regularized objective `lambda=0.5` 和 unregularized objective `lambda=0` 上都能提高价值。
- 反直觉点: PS 可能完全停在初始策略，而直接优化 performative objective 的 0-FW 可以继续提升。
- 反直觉点: 负熵带来的无界梯度不是障碍，反而可用于证明 stationary policy 自动有概率下界。

### 明确局限
- 理论仍强依赖 regularizer dominance；`lambda` 不足以压过环境 shift 时，只能得到带 `|mu||S|` gap 的近似 PO。
- 论文默认 tabular finite state/action setting，没有处理 large-scale function approximation。
- Assumption `d_{pi,p}(s) >= D` 对所有策略和所有环境成立，实际系统里很强。
- `p_pi` 和 `r_pi` 的 Lipschitz sensitivity / smoothness 常数在真实 performative 系统中通常不可知。
- 算法需要多次 policy evaluation of deployed self-induced environments，对真实在线系统可能昂贵。
- 实验只有一个合成小环境，没有标准 benchmark，没有真实 recommender/autonomous driving/LLM alignment 环境。
- 实验没有报告误差条、随机种子方差、ablation 或 wall-clock 对比细节。
- 没有公开代码或 artifact 信息，复现实验需要按论文公式自行实现。
- 证明中存在若干排版或符号问题，例如 `quadratice` 拼写、部分公式中的 `pi_{pi_T}`、以及若干维度记号容易误读。
- Appendix 的 quadratic regularizer 推广只是 sketch，不是完整定理级证明。
- 论文声称 first polynomial-time convergence to PO，但该 claim 限于其假设族，不代表无正则或一般 performative RL。
- 对 `0616-fex` 如果关注真实环境反馈或 foundation-model feedback loop，本论文偏理论，不直接给工程系统结果。
