# Detecting Mutual Excitations in Non-Stationary Hawkes Processes (2601.11717)

- **Year**: 2026.01
- **Authors**: Elchanan Mossel (MIT, Mathematics), Anirudh Sridhar (NJIT, ECE)
- **Venue**: arXiv preprint (LaTeX template header says "ISIT 2025/2026"; likely an ISIT-style conference submission, exact venue/acceptance not stated in metadata)
- **Keywords**: multivariate Hawkes process, non-stationary point process, dependency graph recovery, higher-order event-count statistics, mutual excitation, sparse structure learning, finite-sample guarantee, martingale concentration
- **One-liner**: 用二阶(triple)事件计数统计量, 在时变基线率+非平移不变延迟核的非平稳多元 Hawkes 过程中, 以 T=polylog(n) 时间可证明恢复稀疏依赖图。

## 精华提炼

### 管线/方法
- Input: n 个点过程 (event streams) $N_1,\ldots,N_n$，观测于 $[0,T]$。
- 模型: 非平稳多元 Hawkes 过程 $\lambda_i(t) = \mu_i(t) + \sum_j w_{ij}\int_0^t \phi_{ij}(t,s)dN_j(s)$，其中 $\mu_i(t)$ 时变 (Assumption 1)，$\phi_{ij}(t,s)$ 非平移不变，只要求 $\phi_{ij}(s,s)=1$ 且关于第二个参数递减 (Assumption 2)。
- 第一步 (pairwise 分析): 定义一阶差分统计量 $\Delta_{ij}^{(1)}(t,\epsilon) = X_{ij}(t,\epsilon) - X_{ji}(t,\epsilon)$（$ij$-事件计数减 $ji$-事件计数）。证明其条件期望 $\approx \epsilon^2(w_{ji}\lambda_i(t) - w_{ij}\lambda_j(t))$，并指出关键失败模式: 当 $w_{ij}, w_{ji}$ 同时为正 (双向耦合) 时会发生对消，纯 pairwise 统计量不足以判定依赖是否存在。
- 第二步 (triple 分析): 引入二阶统计量 $\Delta_{ij}^{(2)} = X_{iij} - 2X_{iji} + X_{jii}$（计数两个 $i$-事件夹住/紧邻一个 $j$-事件的三种事件排列）。证明 $(\Delta^{(1)}, \Delta^{(2)})$ 的期望联合构成一个 2×2 非退化线性系统 $M$，且 $\det(M) = w_{ij}w_{ji}(w_{ii}-w_{ij}) > 0$（由 Assumption 6 保证），从而消除了 pairwise 分析遗留的对消歧义。
- 算法: 在 $[0,T]$ 上以步长 $3\epsilon$ 累加得到 $D_{ij}^{(1)}, D_{ij}^{(2)}$，统计量绝对值加权和 $\ge$ 阈值 $h$ 则判边 $(i,j)$ 存在 (Algorithm 1)。
- Output: 无向依赖图 $\widehat{G}$（作者称为 "Markov blanket"），顶点集 $[n]$，$(i,j)\in\widehat{G} \iff w_{ij}>0$ 或 $w_{ji}>0$。注意: 只恢复"是否存在耦合"的无向图，不恢复方向，不估计 $w_{ij}$ 具体数值。

### 关键技术细节
- 主定理: 取 $T=\log^{100}(n)$，$\epsilon=\log^{-17}(n)$，$h = w_{\min}\cdot w_{\mathrm{sep}}\cdot\mu_{\min}/8$，算法输出 $\widehat{G}=G^*$ 的概率为 $1-o(1)$。
- 五条核心假设: (1) baseline rate 下界 $\mu_i(t)\ge\mu_{\min}>0$（不要求上界为常数，只要求下界）；(2) delay kernel 非负、$\phi_{ij}(s,s)=1$、关于第二参数递减（允许非平移不变的一般形式）；(3) stability，$\sum_j w_{ij}\int\phi_{ij}dx \le 1-\eta$，$\eta>0$ 为固定常数（保证过程不爆炸）；(4) smoothness，$|\mu_i'|/\mu_i \le L$ 且 $|\partial_t\phi_{ij}|/\phi_{ij}\le L$（排除重尾核，如地震学中常用的核，作者在 Conclusion 中明确列为待去除的限制）；(5) mutual excitation + sparsity，$w_{ij}\in\{0\}\cup[w_{\min},w_{\max}]$，自激发-互激发分离 $w_{ii}-w_{ij}\ge w_{\mathrm{sep}}>0$（对所有 $i\ne j$ 固定常数），每节点最多 $d$ 个非零耦合。
- 证明技术: 连续时间 Freedman 不等式（引自 Shorack & Wellner 附录，控制鞅 $M_{ij}(t)$ 尾部）、Azuma 不等式（控制 $D^{(1)}, D^{(2)}$ 的集中）、对 $k$ 阶事件计数矩做归纳证明（Lemma EX，给出 $\epsilon^k$ 阶主项 + $O((d\Lambda\epsilon)^{k+1})$ 误差项的一般表达式）、鞅法证明全局最大速率 $\sup_{t\le T}\lambda_{\max}(t) = O(d^2\log^4(nT))$ 的高概率上界。
- 计算效率: 算法只需一次遍历事件流做计数统计，不需要估计 $\mu_i(t)$ 或 $\phi_{ij}(t,s)$ 本身，这是相对此前工作 (cai2022network 等需要完整参数估计) 的关键简化。
- 主定理里 $T=\log^{100}(n)$、$\epsilon=\log^{-17}(n)$ 中的多项式指数 (100、17) 未经优化，属理论 CS 常见的"存在性证明"风格常数——这意味着 $T=\mathrm{polylog}(n)$ 渐进结论虽然理论上远小于 $n$，但对小到中等的 $n$（例如 severe-convection 场景中典型的几十到几百个格点/站点变量）而言，实际所需观测时长可能远超实用范围，论文未讨论有限 $n$ 下的具体样本需求。
- 鲁棒性声称 (定性讨论，未做超出主定理外的独立形式化分析): (a) 部分观测——边 $(i,j)$ 判定只依赖 $N_i,N_j$ 的数据，故子集观测下仍能恢复正确子图；(b) 事件时间量化——算法本就在离散 $\epsilon$-bin 上计数，对事件时刻的小扰动天然鲁棒。

### 核心结果
- 唯一实质性结果是一个渐进正确性定理 ($T=\mathrm{polylog}(n)$ 时高概率精确恢复)；全文 6 页 (ISIT 格式) + 附录完整证明，**无仿真实验、无真实数据实验、无与其他方法的数值对比**——纯理论论文。
- 与最相关前作 cai2022network (即 Cai, Zhang, Guan, "Latent Network Structure Learning From High-Dimensional Multivariate Point Processes", arXiv:2004.03569) 的关键差异: 该文是作者认定的唯一处理非平稳 Hawkes 过程的前作，但要求 delay kernel 平移不变，且需要估计完整参数集合；本文放宽平移不变性要求，但只恢复依赖图（更弱目标换取更一般的非平稳设定）。
- 反直觉发现 (Section III-A): 简单的一阶 (pairwise) 事件计数差分统计量在双向耦合时会系统性对消，使真实存在相互激发的一对过程表现得像独立——这是论文强调"仅靠 pairwise 分析不够"的核心动机，通过二阶 (triple) 统计量解决。

### 明确局限
- **无真实数据/无仿真**: 纯理论 + 渐进复杂度分析，未验证有限样本下图恢复的实际表现，未与 PCMCI+/LKIF 等因果发现基线做横向比较。
- **"非平稳"范围严格受限于外生时变性**: 仅指 $\mu_i(t)$ 随时间变化 + $\phi_{ij}$ 非平移不变这两处显式设定；耦合权重矩阵 $w_{ij}$ 本身（决定图结构和整个系统稳定性的对象）在推导和算法运行期间始终假设为**固定常数**，不随时间漂移。
- **Assumption 3 (stability, $\eta>0$) 与 Assumption 6 (mutual excitation separation, $w_{\mathrm{sep}}>0$) 都是被要求不随 $n$ 退化的固定正常数** ——全文没有讨论 $w_{\mathrm{sep}}\to 0$（自激发与互激发强度趋同，图论上"临界稀疏阈值"的模拟）或 $\eta\to 1$（Hawkes 过程分支比趋于 1，逼近爆炸/稳定性边界，即动力系统意义上最接近"分岔"的对象）时算法或理论保证如何退化。这两个量恰好是把"非平稳"推广到"趋近分岔/参数漂移"场景时最先需要处理的对象，但本文将其作为标准正则性假设排除在研究范围之外，而非作为研究对象。
- Assumption 4 (smoothness，排除重尾核) 被作者自己列为 future work 要去除的限制，但即使去掉它也不等于处理了 $w_{\mathrm{sep}}\to 0$ 或 $\eta\to 1$ 的退化情形——这是两个独立的开放方向，论文只提到了前者。
- **全文（含附录）未出现 "faithfulness" 措辞或概念**。Assumption 6 的 $w_{ii}-w_{ij}\ge w_{\mathrm{sep}}$ 条件在功能上等价于因果发现文献中的 strong-faithfulness / beta-min 条件（阈值化以避免任意小信号无法被样本复杂度区分），但作者未将其与因果发现或 local independence 文献建立联系，未引用 Didelez / Mogensen-Hansen / Thams-Hansen 的 local independence 框架论文（仅在"unobserved nodes"鲁棒性脚注中提及 Thams & Hansen 2021，见下方引文表）。
- 未考虑图结构本身随时间变化（$w_{ij}$ 从 0 跳变为正，或反之）的情形——这与我们 idea 中"已知真实图变化正对照 regime"(硬系数跳变 $0\to0.6$) 场景更接近，但本文完全未涉及，只处理固定图 + 外生非平稳噪声。
