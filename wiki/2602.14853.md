# BEACONS: Bounded-Error, Algebraically-Composable Neural Solvers for Partial Differential Equations (2602.14853)

- **Year**: 2026.02
- **Authors**: Jonathan Gorard (Princeton University), Ammar Hakim, James Juno (Princeton Plasma Physics Laboratory)
- **Venue**: arXiv
- **Keywords**: formally-verified neural networks, hyperbolic PDEs, method of characteristics, compositional deep learning, L∞ error bounds, automated theorem proving, flux limiters, PINN critique
- **One-liner**: 用特征线法预测 PDE 解光滑性 + Mhaskar 逼近理论得外推误差界 + 代数可组合性构造深层架构 = 具有机器可验证正确性证书的神经 PDE 求解器。

## 精华提炼

### 管线/方法

完整 pipeline：**输入**（PDE 系统 + 初始条件 + BEACONS 超参数）→ **Racket DSL 表示** → **自动代码生成器** 合成 C 代码（生成训练数据 + 构建网络 + 训练 + 验证 + 推理）→ **自动定理证明器** 生成 machine-checkable 的 L∞ 误差界证明 → **输出**（训练好的 BEACONS 架构 + 形式验证证书）。

具体步骤：
1. **用形式验证数值求解器生成训练数据**：Lax-Friedrichs 或 Roe 型有限体积法 + 二阶通量外推（选 minmod/monotonized-centered/van Leer/superbee limiter），仅用前 1/3 时间帧作为训练数据
2. **浅层网络逼近 PDE 解**：MLP with single hidden layer（tanh 激活，符合 C∞ smooth 条件），PDE residual loss + boundary/initial condition loss，full-batch gradient descent
3. **特征线法预测光滑性**：对一阶双曲型守恒律 PDE，利用特征线方程 $dx/dt = f'(u)$ 和初始数据光滑性，在任意时空点预测 shocks 出现时间 $t_\infty = (\sup_{x_0} \{-f''(u_0(x_0)) u_0'(x_0)\})^{-1}$
4. **分解为函数组合**：将间断解 $u(x)$ 分解为 $u = f \circ g$，其中 $f$ 是光滑慢变函数（小 Lipschitz 常数 L），$g$ 是包含所有间断的函数
5. **逐层训练**：每层单独训练一个组成函数 $f$，用其专用 loss function，标准 backpropagation
6. **误差界推导**：$\|f \circ g - \widetilde{f} \circ \widetilde{g}\|_\infty \leq e_f + L e_g$，用 $f$ 的小 L 压制 $g$ 的大误差 $e_g$

### 关键技术细节

- **网络类型**：多层感知器（MLP），每个组成函数用单隐藏层浅层网络逼近，然后组合成深层架构
- **激活函数**：$\tanh(x)$（C∞-smooth，满足 Mhaskar 定理条件，且在任意子区间上不是多项式）
- **学习率**：固定 $10^{-4}$
- **训练方式**：Full-batch gradient descent（非 mini-batch，为保证确定性可复现）
- **最大 epoch**：50（最小 10）
- **隐藏层宽度**：64（6 层架构）或 128（8 层架构）
- **损失函数**：PDE residual 的加权 L2 范数 + boundary condition residual 的加权 L2 范数，$J(\theta) = \lambda_{PDE} J_{PDE}(\theta) + \lambda_{BC} J_{BC}(\theta)$
- **向量 PDE 系统**：残差用加权欧氏范数 $\|\mathbf{r}\|_W = \sqrt{\sum_{\alpha,\beta} r_\alpha W_{\alpha\beta} r_\beta}$，W 为对称半正定矩阵
- **定理证明器架构**：全局合流 + 强规范化的符号重写系统（Racket 实现），仅允许尊重 IEEE-754 浮点标准的 correctness-preserving 代数变换
- **候选分解函数**：自动试验 arcsinh/sinh 等三角/双曲函数组合，找到最小化 $e_g$ 增长率（作为 $1/L$ 函数）的候选
- **CFL 系数**：1.0（线性对流、Burgers）或 0.95（Euler）
- **空间分辨率**：1024 cells（1D）、256×256 cells（2D）

### 核心结果

全线 BEACONS 架构在 L∞ 误差、L2 误差和守恒误差上均大幅优于同规模全连接网络。

**1D 线性对流 Riemann 问题**（训练帧：t=0 到 0.33，预测至 t=1.0）：
| 架构 | L∞ (Final) | L2 (Final) | 守恒误差 (Final) | 守恒误差 (Total) |
|------|------------|------------|------------------|------------------|
| 6-layer NN | 1.034 | 2.537 | -21.17 | 498.71 |
| 8-layer NN | 0.977 | 9.031 | -130.42 | -4391.64 |
| 6-layer BEACONS | 0.612 | 1.132 | -3.77 | -155.32 |
| 8-layer BEACONS | 0.605 | 1.212 | 1.88 | 49.57 |

BEACONS 定理证明界：6-layer 0.904, 8-layer 0.707；实际最大 L∞ 0.782/0.633，均在界内。

**1D Burgers "top-hat" 问题**：
| 架构 | L∞ (Final) | L2 (Final) | 守恒误差 (Final) |
|------|------------|------------|------------------|
| 6-layer NN | 1.310 | 15.059 | 69.35 |
| 8-layer NN | 1.209 | 12.128 | 30.71 |
| 6-layer BEACONS | 0.987 | 2.165 | 4.46 |
| 8-layer BEACONS | 1.028 | 2.404 | 12.79 |

**反直觉发现**：对于 Burgers 方程，8-layer BEACONS 反而比 6-layer 差（L∞ 更高、守恒误差更大），作者认为 6-layer 可能对该类非线性问题已接近最优。

**2D 线性对流 disk**：8-layer BEACONS 几乎完美保持 disk 形状，8-layer NN 导致 disk 变形为"蛋形"并低估对流速度。BEACONS L∞(All) 0.938 vs NN 1.031。

**2D Burgers disk**：BEACONS 正确预测"彗星"形状和传播速度，NN 使解看起来"尖"且传播过快，提前撞到边界。

**1D Euler Sod 激波管**：8-layer BEACONS 成功捕获全部三个波（稀疏波、接触间断、右行激波），而 6-layer BEACONS 仍会扩散接触间断。两种 NN 都完全未能捕获激波和接触间断。

**2D Euler quadrants**：NN 完全扩散解的定性结构（右上角"射线"和激波前沿完全消失、左下角波前被误判为凹而非凸）。BEACONS 正确预测所有波的形状和速度，仅在细微结构（晚期"指状"不稳定性）上有额外数值扩散。

### 明确局限

1. **仅适用于双曲型 PDE**：依赖特征线法预测光滑性。椭圆/抛物型 PDE 需等未来工作（作者提议用椭圆正则性理论替代）
2. **仅做正向问题**：作者明确说逆向问题（PDE 约束优化、降维）是未来方向
3. **需要光滑激活函数**：非光滑激活（ReLU 等）不满足 Mhaskar 定理条件
4. **仅用 tanh 激活**：未系统比较其他 C∞ 光滑激活函数
5. **大规模训练的可行性未验证**：实验规模很小（1024/256×256 resolution，最多 8×128 网络），未展示对大规模 3D 仿真的 scaling
6. **Burgers 8-layer < 6-layer**：更深的 BEACONS 不总是更好，最优深度/宽度选择需要 trial
7. **Euler 2D 丢失细微结构**：数值扩散增加导致晚期"指状"不稳定性消失
8. **没有跟其他神经 PDE 方法对比**：仅对比了全连接 NN，没有对比 DeepONet、FNO、CNO 等 operator learning 方法
9. **训练数据依赖形式验证求解器**：对没有形式验证求解器的 PDE（如 Euler），仅能做条件性证明
10. **定理证明步骤数指数增长**：1D 线性对流 99 步，2D Euler 需 142,104 步；scalability 存疑
11. **无真机实验**：纯数值验证，没有任何物理实验或工程应用
12. **单作者机构集中**：仅 Princeton + PPPL，无独立第三方验证
