# The ballistic limit of the log-Sobolev constant equals the Polyak–Łojasiewicz constant (2411.11415)

- **Year**: 2024.11
- **Authors**: Sinho Chewi (Yale Statistics & Data Science), Austin J. Stromme (ENSAE/CREST)
- **Venue**: arXiv preprint
- **Keywords**: log-Sobolev inequality, Polyak–Łojasiewicz condition, Langevin dynamics, low-temperature limit, functional inequalities, ballistic limit, Poincaré inequality, sampling-optimization connection
- **One-liner**: 证明在低温极限下，log-Sobolev 常数的 ballistic limit 精确等于 PL 常数，建立了优化理论（梯度流收敛率）与采样理论（Langevin 动力学 KL 散度收敛率）之间的严格对应。

## 精华提炼

### 管线/方法

本文是纯理论论文，不涉及实验。其证明管线如下：

1. **设定**：给定 C² 函数 f: ℝᵈ → ℝ，定义 μ_t ∝ exp(-f/t)，研究 t → 0⁺ 时 log-Sobolev 常数 C_LS(μ_t) 的渐近行为。
2. **Ballistic 常数定义**：C_bLS(f) := lim_{t→0⁺} C_LS(μ_t)/t。类似定义 ballistic Poincaré 常数 C_bP(f)。
3. **下界证明（Section 3）**：通过构造局部光滑分布 ν → δ_x，利用 log-Sobolev 不等式展开和取极限，证明 C_PL(f) ≤ liminf C_LS(μ_t)/t。论证绕过了对 t=0 时 log-Sobolev 常数的直接依赖。
4. **上界证明（Section 4）**：这是主要技术贡献。分五步：
   - **Step 1 - Gaussian LSI**：将 KL(ν||μ_t) 分解为 KL(ν||N(0,Σ_t)) + 残差，利用 Gaussian LSI 和分部积分。
   - **Step 2 - 尺度分离**：将空间分为小球 B(0,r) 和大球补集 Bᶜ。小球内用泰勒展开，大球外用 PL 不等式直接控制。
   - **Step 3 - 内积项处理**：通过散度定理将内积项转化为球面积分和 Hessian 迹项。
   - **Step 4 - 大尺度项控制**：用二次增长不等式和分部积分将大尺度项转化为 Fisher 信息项。
   - **Step 5 - Tightening**：利用改进的 tightening 引理（Wang 2024），从 defective LSI 得到精确 LSI 常数，取 r₀ = A√t，令 t→0⁺ 再令 A→∞。
5. **Poincaré 常数证明（Section 2.2）**：用 Lyapunov 准则（Bakry-Émery 理论）+ 局部强凸性，证明 C_P(μ_t) = O(t) 且 ballistic limit = 1/λ_min(∇²f(x*))。

### 关键技术细节

- **核心工具链**：
  - Rothaus 引理的改进版（Wang 2024, Prop. 5）：C_LS(μ) ≤ C + (D/2)·C_P(μ)，去掉了传统 tightening 中的 +1 项
  - Bakry-Émery Lyapunov 准则（Theorem 4.6.2）
  - Otto 的 optimal transport Riemannian 几何视角
  - 二次增长不等式（PL ⇒ quadratic growth）：(1/(2C_PL))·d(x,S)² ≤ f(x) - f*
  
- **关键假设**：
  - f ∈ C²(ℝᵈ)
  - 唯一全局最小值点（necessary，反例见正文讨论）
  - Laplacian 增长条件：Δf ≤ L₀ + L₁·‖∇f‖²（技术性假设，排除极端行为）
  - 对下界只需 Lipschitz 连续性（更弱）
  
- **非渐近界**（Remark 中的量化）：
  - 当 ∇²f ⪯ βI 且 ∇³f 是 γ-Lipschitz：C_LS(μ_t) ≤ C_PL·t + O((β·C_PL^{7/3}·γ^{1/3}·d·t)^{6/5})
  - Poincaré 常数：C_P(μ_t) ≤ 5·C_PL·t（光滑情况下的简单界）
  
- **Defective LSI 结果**（非渐近）：Δf ≤ L₀ 时，μ ∝ e^{-f} 满足 (C_PL, C_PL·L₀ - d)-defective LSI

### 核心结果

| 结果 | 公式 | 条件 | 
|------|------|------|
| **主定理** | C_bLS(f) = C_PL(f) | f∈C²，唯一最小值点，Δf bounded by 1+‖∇f‖² |
| **Ballistic Poincaré** | C_bP(f) = 1/λ_min(∇²f(x*)) | 同上 + PL < ∞ |
| **下界** | C_PL(f) ≤ liminf C_LS(μ_t)/t | 只需 Lipschitz 连续性（极弱） |
| **上界** | limsup C_LS(μ_t)/t ≤ C_PL(f) | 全部假设 |
| **Defective LSI** | (C_PL, C_PL L₀ - d)-defective | 不依赖于 t→0 |

**反直觉发现**：
- Ballistic log-Sobolev 常数取决于 f 的**全局**优化景观（PL 条件），而 ballistic Poincaré 常数只取决于最小值点处的**局部** Hessian。
- PL 条件比强凸性严格弱，但在唯一最小值点的低湍极限下，LSI 的行为和强凸情况一致（C_LS ~ C_PL·t）。
- 改善的 tightening 引理（去掉 +1 项）对获得精确常数至关重要，否则上界会差 2 倍。

### 明确局限

1. **唯一最小值点假设是关键限制**。f 有多个全局最小值时，定理不成立（作者给出反例：f(x) = (α/2)·d²(x,K) 对凸集 K）。
2. **C² 正则性要求**在部分论证中是刚性的。下界证明可放松到 C¹ + Lipschitz，但上界需要 C²。
3. **Laplacian 增长条件**（Δf ≤ L₀ + L₁‖∇f‖²）排除了一些病态但可能有趣的 f。
4. **没有算法设计或实验**。纯粹的理论分析。从理论到实际采样算法的转化需要额外工作。
5. **多最小值点情形的猜想**（公式 1.13）：lim(C_LS(μ_t) - C_LS(μ₀))/t = C_PL(f)，作者承认其证明技术"out of reach"。
6. **高维常数**：非渐近界中的常数对维度 d 的依赖未优化（尤其是 d^{6/5} 的 exponent）。
7. **与我们的关系**：如果课题涉及 Langevin 采样、优化-采样 duality 或 non-convex optimization 的理论分析，本文的连接视角有用。但如果我们的课题纯粹是 empirical/algorithmic，本文是理论工具。
