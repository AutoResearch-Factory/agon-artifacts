# Scientific Machine Learning of Chaotic Systems Discovers Governing Equations for Neural Populations (2507.03631)

- **Year**: 2025.07
- **Authors**: Anthony G. Chesebro 等; Stony Brook, MIT CSAIL, Dartmouth, Santa Fe Institute
- **Venue**: arXiv
- **Keywords**: PEM-UDE, universal differential equations, chaotic system identification, symbolic regression, SINDy/STLSQ, neural mass model, sparse connectivity, equation discovery
- **One-liner**: 用 PEM 稳定混沌 UDE 训练并恢复神经群体方程。

## 精华提炼

### 管线/方法

1. 输入是一组来自混沌物理系统或神经系统的时间序列观测。
2. 作者先写出一个不完整但有物理含义的初始动力学模型。
3. 初始模型承担已知物理项, 未知项交给 Universal Differential Equation (UDE) 中的神经网络学习。
4. UDE 形式为 `dx/dt = f_p(x,u(t)) + NN_theta(x)`, 观测由 `y_theta(t)=h_q(x_{p,theta}(t),t)` 给出。
5. 传统 UDE 直接最小化轨迹 MSE, 在混沌系统中会因为 Lyapunov 时间后的轨迹发散而失败。
6. 本文核心改动是把 prediction-error method (PEM) 注入 UDE 训练过程。
7. PEM 在每个 ODE 求解步加入 `K * epsilon_theta(t)`, 其中 `epsilon_theta(t)=observed y - predicted y`。
8. 合并后的动力学为 `d x_tilde/dt = f_p(x_tilde,u) + K epsilon_theta(t) + NN_theta(x_tilde)`。
9. `K` 是关键超参数: 太小不能消除敏感依赖, 太大又会让其他参数方向出现平坦平台。
10. 训练完成后, 作者不直接把神经网络当最终模型。
11. 他们从训练好的 UDE 网络输出中采样, 再用符号回归恢复显式方程。
12. 符号回归主要使用 STLSQ, 即 SINDy 风格的顺序阈值最小二乘。
13. 对更复杂约束也支持 genetic algorithm / PySR 风格符号回归, 但本文主结果默认 STLSQ。
14. 整体路线是: 观测数据 -> 已知动力学骨架 -> PEM-UDE 学未知项 -> STLSQ 提取符号式 -> 用显式方程外推和分析。
15. 这不是纯 neural ODE, 因为已知物理项和最终符号表达式都保留下来。
16. 这也不是纯 SINDy, 因为 SINDy 不直接吃原始带噪混沌导数, 而是吃 UDE 学出的去噪动力学。
17. 方法的直觉是: PEM 先把混沌轨迹拟合变成较平滑的优化问题, 然后符号回归把黑箱项压回可解释方程。
18. 作者强调该方法适合有部分已知结构、但缺少若干动力学项的科学建模问题。

### 关键技术细节

19. Rössler 系统实验中, 已知 `x` 和 `z` 方程, 完全学习 `y` 状态的动力学。
20. Rössler 参数为 `a=0.2`, `b=0.22`, `c=14`。
21. Rössler 主实验使用 `K=0.25`。
22. Rössler UDE 训练使用 AdamW, 训练 `100,000` epochs。
23. 作者报告该参数区间的 Lyapunov time 约为 `lambda^{-1} ~= 10`。
24. PEM 对损失景观的作用在 Fig. 2b 和 supplemental landscape 图中展开: rough landscape 被平滑, global minimum 位置保持。
25. 附图指出 `K ~= 0.18` 足以消除非常接近参数扰动下的敏感依赖示例。
26. Petrzela-Polak 电路实验使用一个物理可实现的三维混沌电路。
27. 其无量纲系统为 `dot x=z`, `dot y=a x^2 - z - c`, `dot z=y-x-bz`。
28. 电路参数包括 `a=1.25`, `b=0.5`, `c=1.0`; 物理元件包括 `C1=C2=1.0 uF`, `L=1.0 H`, `I0=1.0 mA`。
29. 电路任务把 `x` 状态观测污染到噪声幅度为真实信号的 `5x`。
30. 在同样数据上, 直接 STLSQ/SINDy 从导数估计恢复符号式会失败。
31. PEM-UDE 先学习一个平滑/去噪动力学, STLSQ 再从 UDE 输出恢复正确函数形式。
32. 神经群体实验使用 1,000 个 Izhikevich neurons 的网络仿真。
33. 连接概率 `p(c)` 覆盖 `[0.05, 0.1, 0.2, ..., 1.0]`。
34. 为保持平衡活动, synaptic conductance 按 `s_jump = s_jump,0 / sqrt(p(c))` 重标定。
35. 初始 NGNMM 变量为 firing rate `r`, mean voltage `v`, recovery current `w`, synaptic activity `s`。
36. 可观测训练量只使用 `r` 和 `v`, 因为这是更接近实际实验可获得的数据。
37. UDE 同时学习 `r` 和 `v` 两个方程的未知 correction terms。
38. 神经 UDE 网络在方法部分描述为 4 层 RBF 网络, 输入维 `N1`, 输出维 `N2`, hidden 为 `10 x 10`。
39. 插值使用 DataInterpolations.jl 的线性插值。
40. ODE solver 默认非刚性高阶 solver, 通常为 Vern7; 若刚性明显可切到 FBDF。
41. 神经网络训练默认 AdamW; supplemental 中 `p_c=0.5` 的 NGNMM UDE 使用 Adam 训练 `40,000` iterations。
42. `p_c=0.5` supplemental 训练先用 `K=0.3`, 再短程降低到 `K=0.1`; 低于 `0.1` 后作者称优化不再改善。
43. STLSQ 后得到的稀疏连接 NGNMM correction terms 为 `f1` 和 `f2`。
44. `f1(u)=phi1*s + phi2*r*s + phi3*v*s + c1`。
45. `f2(u)=phi4*r + phi5*r*v + phi6*r*s + phi7*s^2 + c2`。
46. 关键参数为 `phi1=-0.50`, `phi2=-0.12`, `phi3=0.88`, `phi4=-5.01`, `phi5=9.80`, `phi6=-0.05`, `phi7=-0.27`, `c1=0.12`, `c2=0.06`。
47. 原 NGNMM 参数包括 `alpha=0.6215`, `g_syn=1.2308`, `a=0.0077`, `b=-0.0062`, `tau_s=2.6`, `Delta=0.02`。
48. 同步度用 Kuramoto order parameter 计算, 其中 `W=pi*r+i*v`, `Z=(1-W*)/(1+W*)`。
49. 实验验证同步度使用三套 intracranial electrode datasets: rat MEC `N=6`, rat prefrontal cortex `N=10`, human OFC `N=10`。
50. 同步指标为 Hilbert transform 相位差的 phase-locking value (PLV)。

### 核心结果

51. Rössler 任务中, 标准 UDE 快速掉入局部均值解, PEM-UDE 能继续收敛。
52. Rössler 的 PEM-UDE mean error 比 unassisted UDE 小 `2-3` 个数量级。
53. STLSQ 从 PEM-UDE 输出恢复缺失项形式 `y = p1*x + p2*y`。
54. 初始 STLSQ 参数约为 `p1=1.01`, `p2=0.17`; 真值为 `p1=1`, `p2=0.2`。
55. 追加参数优化后, Rössler 参数可到三位小数精度, 并能外推到训练时间之外。
56. 电路任务中, 即使 `x` 观测噪声为真实信号 `5x`, PEM-UDE + STLSQ 仍恢复正确函数形式。
57. 直接 STLSQ/SINDy 对同一噪声数据在多个 `lambda` 下失败, 高稀疏时甚至会把 noisy state `x` 整体丢掉。
58. 神经群体任务中, PEM-UDE 能高精度拟合不同稀疏度下的 mean-field dynamics。
59. 模型自然预测: 连接越稀疏, dominant oscillation 从 theta band 转向 alpha band。
60. 这个 theta-to-alpha 转移被作者解释为与深部 hippocampal/entorhinal theta 和 cortical alpha 的经验现象一致。
61. 模型还预测: 更高连接密度带来更高 resting baseline synchrony。
62. 三套 intracranial 数据的 PLV 分布支持该趋势: rat MEC 同步度高于 rat PFC, human OFC resting synchrony 更低。
63. Supplemental 指出仅用 `p_c=0.3` 到 `0.9` 训练的 PEM-UDE 也能外推到 `p_c=0.05`, 但有轻微频率偏移。
64. 对 NGNMM, 最有生物解释的新项是 `phi7*s^2`: voltage 对 synaptic dynamics 的二次依赖。
65. 作者把 `s^2` 项联系到树突整合/ NMDA receptor 类的超线性突触整合现象。
66. 反直觉点: 论文不是把混沌系统的长时轨迹当可精确预测对象, 而是用 PEM 让训练阶段局部贴住观测, 最终恢复可外推的方程。
67. 另一个关键点: 该方法对 noisy chaos 的成功主要来自两阶段分解, 不是 STLSQ 本身变强。

### 明确局限

68. 方法依赖已有的较好物理骨架; 完全无先验的 latent dynamics 不在本文范围。
69. PEM 可能在已知物理参数不准时制造 close-but-inaccurate minima。
70. `K` 的选择需要任务调参; 论文给直觉和示例, 但没有通用选择理论。
71. 极端噪声系统仍可能难学, 尤其 chaotic systems 的跨 trial averaging 本身复杂。
72. 作者承认 shadowing 可能允许非物理解 shadow 物理解, 需要符号形式和物理检验缓解。
73. 神经 NGNMM 的符号式主要在 `p_c=0.5` 展示并调参, across sparsity 的统一闭式还不够干净。
74. 稀疏连接对实验脑区的映射是区域级粗粒度假设, 不是直接测量每个数据集的真实局部 connectome。
75. 三套实验数据跨物种、脑区、任务状态不同; synchrony 趋势支持模型, 但不是强因果验证。
76. 代码和 simulated data 可用, 但 intracranial datasets 需去 CRCNS 按各自条件获取。
77. 论文没有给 FEX/RL-style expression search 这类方法的直接比较。
78. 论文没有系统比较 PySR、DSO、LLM-SR 等现代 SR 生成器; SR 主要作为 UDE 后处理。
79. 对真实 resting-state 无已知 perturbation 的适配仍是 future work。
80. 对 PDE/field model 的扩展被列为 future work, 本文实际实验还是 ODE/mean-field ODE。

### 跟我们的关系

81. 对 `0616-fex` 来说, 这篇是 equation discovery / symbolic regression 的强相关近邻, 但不是 FEX 的直接竞品。
82. 它不做 RL controller, 不做 FEX 的 finite expression space `S_k`, 也不以 PDE residual solver 为主任务。
83. 直接撞车风险: 低到中。若我们的 claim 是“用 RL 搜索有限表达式求 PDE”, 本文只是背景; 若 claim 是“从 noisy chaotic data 中发现 governing equations”, 它就是必须比较的强 prior。
84. 它与 FEX 的交叉点在“两阶段结构”: 先用一个可训练连续模型稳定拟合, 再抽取符号表达式。
85. 对 FEX 最可借鉴的工具是 PEM: 当 FEX 用于从实验轨迹发现动力学时, 可用 prediction-error correction 缓解 chaotic rollout loss 的不稳定。
86. 对 FEX 另一个启发是“先拟合去噪动力学, 再做符号搜索/稀疏回归”, 这可作为 noisy-data FEX 的 pipeline baseline。
87. 该文也提示 FEX 的干净 PDE benchmark 不足: 真实 scientific discovery 会遇到 partial observability、strong noise、parameter misspecification 和 chaotic shadowing。
88. 如果 `0616-fex` 做 LLM/FEX skeleton generation, 本文不是 skeleton 竞品, 但可以作为下游 validation domain: 让 FEX/LLM 生成 PEM-UDE 后处理可恢复的方程项。
89. 如果 `0616-fex` 做 failure taxonomy, 本文给出一个很具体的 failure mode: 直接 trajectory MSE 在 chaotic systems 中 ill-defined, direct SINDy 在 noisy derivative estimates 上失败。
90. 差异化建议: FEX 可以强调有限表达式搜索和 PDE residual theory; PEM-UDE 则强调 known-physics UDE + observer correction + STLSQ 后处理。
91. 该引不该引: 若写 equation discovery、UDE、SINDy、noisy/chaotic dynamics、neural mass model discovery, 应引用。
92. 若只写 FEX 的 approximation theory 或 RL search complexity, 可作为应用侧背景, 不是核心 prior。

## Read by: 0616-fex (2026-06-17)

本次按 FEX/RL symbolic regression 视角审读。结论是: 这篇不直接撞 FEX 的核心 claim, 因为它不做 RL 表达式树搜索、不处理 FEX 的 `S_k` 有限表达式空间, 也没有 PDE residual solver 的理论保证。它的强相关点在 noisy/chaotic equation discovery: 先用 PEM 稳定 UDE 训练, 再用 STLSQ 抽取符号式。若 `0616-fex` 后续从干净 PDE 求解扩展到实验轨迹、动力系统识别或 noisy symbolic discovery, 这篇必须作为强 baseline 引用。

可借鉴模块有三类。第一是 PEM correction, 可用于 FEX 在 chaotic rollout loss 下的训练/评估稳定化。第二是“连续去噪模型 -> 符号提取”的两阶段分解, 可作为 noisy FEX 的 pipeline: FEX 不一定直接拟合带噪导数, 可以先拟合受约束动力学再搜索有限表达式。第三是 failure taxonomy: standard UDE 轨迹 MSE 在 Lyapunov 时间后失效, direct SINDy/STLSQ 在 5x 噪声电路观测上失败, 这给 FEX 的鲁棒性论文提供了清晰反例设置。

差异化很明确: FEX 可强调有限表达式搜索、RL controller、PDE residual objective 和 approximation theory; PEM-UDE 强调 known physics + observer feedback + UDE + sparse regression。下一轮若围绕 FEX noisy/chaotic discovery 展开, 应搜索 `prediction error method symbolic regression`, `chaotic system identification UDE`, `SINDy noisy partial observations`, `neural mass equation discovery`, `observer-based equation discovery`。
