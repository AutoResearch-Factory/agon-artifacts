# Hierarchical Bayesian Operator-induced Symbolic Regression Trees for Structural Learning of Scientific Expressions (2509.19710)

- **Year**: 2025.09
- **Authors**: Somjit Roy, Pritam Dey, Debdeep Pati, Bani K. Mallick; Texas A&M University and University of Wisconsin-Madison
- **Venue**: arXiv
- **Keywords**: Bayesian symbolic regression, symbolic regression trees, Scientific Machine Learning, uncertainty quantification, Occam's window, posterior concentration, Feynman equations, single-atom catalysis
- **One-liner**: 用全贝叶斯树集成做科学表达式符号回归。

## 精华提炼

### 管线/方法
- **任务定义**：输入观测数据 `D_n={(x_i,y_i)}`、候选原始特征 `x_1,...,x_p`、候选算子集合 `O`，输出一组可解释的符号表达式树及其线性组合。
- **符号树表示**：终端节点是原始特征；非终端节点是算子；一元算子包括 `exp`, `inv`, `neg`, `sin`, `cos`, `^2`, `^3`，二元算子包括 `+`, `*`。
- **表达式类限制**：作者定义 `S_vartheta` 为深度不超过 `vartheta` 的符号树函数集合，用深度和非终端节点数控制复杂度。
- **核心模型**：HierBOSSS 建模为 `y = beta_0 + sum_{j=1}^K beta_j t_j(x) + epsilon`，其中每个 `t_j` 是一棵符号树。
- **似然**：`epsilon ~ N(0, sigma^2 I_n)`，表达式设计矩阵为 `[1, t_1(X), ..., t_K(X)]`。
- **参数先验**：`(beta, sigma^2)` 使用 Normal-Inverse-Gamma 共轭先验。
- **树结构先验**：每个节点在深度 `m` 处分裂为非终端节点的概率为 `p_m = alpha(1+m)^(-delta_0)`。
- **实验中树先验参数**：主实验设置使用 `(alpha, delta_0)=(0.95,1.20)`，用于抑制过深树但仍允许表达式增长。
- **算子/特征权重**：每棵树有 operator weight `w_op,j` 和 feature weight `w_ft,j`，均使用 Dirichlet 先验。
- **后验采样**：使用 Metropolis-within-partially-collapsed Gibbs sampler。
- **树更新**：对符号树结构用 Metropolis-Hastings，目标分布是边缘化后的 joint marginal posterior of tree ensemble, 即 JMP-ensemble。
- **GROW move**：随机选一个终端特征节点，用按树先验生成的子树替换。
- **PRUNE move**：随机选一个非终端算子节点，把它剪成一个按特征权重采样的终端节点。
- **权重更新**：给定树结构后，`w_op,j` 和 `w_ft,j` 从 Dirichlet full conditional 采样。
- **模型参数更新**：给定树结构后，`beta` 和 `sigma^2` 从 NIG full conditional 采样。
- **模型选择**：不是按 visit frequency，也不是单纯 marginal likelihood；作者按 JMP-ensemble 排序保留 top `r` 个 symbolic forests。
- **Occam's window**：JMP 中包含树先验和 Dirichlet 边缘化项，因此偏好解释力足够且结构更简单的表达式。
- **结构评估**：作者用 minimum graph edit distance (mGED) 比较学到的树与真值表达式树，捕捉语法和语义接近程度。
- **理论对象**：定义 `F={beta_0+sum beta_j t_j(x)}` 与 Gaussian model class `M`，用 empirical `d_n` 距离度量函数误差。
- **理论条件**：需要 expression design matrix 条件良好，以及分裂概率满足 sub-critical condition `p_m <= q < 1/2`。
- **理论结果**：后验以 `epsilon_n^2 ~= max{ n^-1(K^dagger log p + S^dagger log |O| + log n), a_n^2 }` 收缩。
- **well-specified 速率**：若真函数可由 `K*` 棵树和 `S*` 个非终端节点精确表示，速率近似 `sqrt((K*log p+S*log|O|+log n)/n)`。
- **misspecified 速率**：若真函数是 Holder smooth 且可被符号树近似，作者声称可达到近 minimax 速率到 polylog 因子。
- **代码可用性**：论文称 R 实现、代码和数据发布在 `github.com/Roy-SR-007/HierBOSSS`。

### 关键技术细节
- **主算法输入**：`D_n=(y,X)`, 树数量 `K`, 算子集 `O`, 分裂参数 `(alpha,delta_0)`, `p_grow`, Dirichlet concentration, NIG hyperparameters, proposal operator/feature weights, `niter`。
- **模拟例子设置**：`y = 5(x_1+x_2)x_3 + epsilon`，`x_j ~ N((2j+2)1_n,I_n)`，`(n,p)=(1000,3)`，`sigma^2=1.5`。
- **模拟例子采样**：HierBOSSS 对 `K in {2,3,4,5}` 分别跑 `niter=10000`。
- **模拟算子集**：`O=(exp, inv, neg, ^2, ^3, +, *, sin, cos)`。
- **Feynman 设置**：从 Feynman Symbolic Regression Database 取 `10^5` 观测中的 `n=1000` 子样本，并加噪声。
- **Feynman 方程**：GPE `Delta U=m1*m2*(1/r2-1/r1)`，Coulomb `F=q1*q2/r_*^2`，Lorentz `F=q(E_f+Bv sin theta)`。
- **Feynman 采样**：GPE 用 `K=5`，Coulomb 用 `K=2`，Lorentz 用 `K in {2,3}`，每个 `K` 跑 `niter=1000`。
- **Feynman 噪声**：测试 no noise 和 `sigma^2 in {0.25,1.50,2.00,4.00}` 等情形。
- **单原子催化数据**：13 种 transition metals 与 7 种 oxide supports，合计 `n=91` metal-support pairs。
- **催化特征**：`Delta H_sub`, `Delta H_f,ox,bulk`, `Delta E_vac`。
- **催化算子集**：使用 `exp`, `inv`, `neg`, `^2`, `^3`, `+`, `*`，排除 `sin` 和 `cos`，因为物理上不相关。
- **催化采样**：对 `K in {2,3,4,5}` 跑 `niter=2000`，每个 `K` 从 top `r=10` JMP forests 中选 RMSE 最低的一组描述符。
- **对比方法**：BSR, iBART+`l0`, QLattice/Feyn, LASSO* 描述符方法。
- **重要实现差异**：BSR 是 pseudo-Bayesian/plug-in 风格，作者批评其不完整传播参数不确定性；HierBOSSS 用共轭边缘化给 tree ensemble 后验。
- **可复现性细节**：补充材料列出 competing methods 的代码来源、迭代次数、默认参数、operator application order 和 out-of-memory 失败情况。

### 核心结果
- **模拟例子 JMP top-3**：top 1 学到 `{x3*x2, x1*x3}`，`beta=(4.99,5.00)`，`log JMP=-770.98`，RMSE `1.2699`。
- **模拟例子 top-2**：等价地学到 `{x3*x1, x2*x3}`，`beta=(5.00,4.99)`，RMSE `1.2708`。
- **模拟例子 top-3 单树形式**：学到 `{(x2+x1)x3}`，`beta=4.99`，RMSE `1.2720`。
- **JMP vs marginal likelihood**：marginal likelihood 选出重复项和更复杂表达式，例如两个近重复 `(x1+x2)x3`，RMSE 相近但结构冗余。
- **模拟对比**：BSR 和 QLattice 也可恢复真式，RMSE 接近；iBART+`l0` 只找到部分结构，RMSE `2.7134`。
- **25 次模拟重复**：HierBOSSS 一直达到 `mGED=0`；BSR 和 QLattice 有失败；iBART+`l0` 结构识别差。
- **runtime**：HierBOSSS 明显低于 BSR 和 iBART+`l0`，与 QLattice 接近。
- **GPE, `sigma^2=0.25`**：HierBOSSS top rank 学到 `(m1*m2)/r2` 与 `(m1*m2)/r1` 两个真组件，RMSE `0.0783`，mGED `0`。
- **GPE 多噪声对比**：no noise RMSE `0.0277`，`sigma^2=0.25` RMSE `0.0783`，`sigma^2=1.50` RMSE `1.2264`，三者 mGED 都为 `0`。
- **GPE 竞争方法**：BSR 在三档噪声中 mGED 分别为 `6,6,3`；QLattice mGED 为 `>>6,10,10`，且 RMSE 显著更高。
- **Coulomb no noise**：HierBOSSS RMSE `0.0002`, mGED `0`；QLattice 也能恢复真式，RMSE `0.00017`。
- **Coulomb `sigma^2=0.25`**：HierBOSSS RMSE `0.4885`, mGED `0`；BSR RMSE `0.4667` 但 mGED `29`，说明预测接近但结构严重过复杂。
- **Coulomb `sigma^2=4.00`**：HierBOSSS RMSE `2.0030`, mGED `0`；BSR RMSE `1.8668` 但 mGED `4`；QLattice RMSE `2.11`, mGED `4`。
- **Lorentz no noise**：HierBOSSS `K=3` top model包含 `q B v sin theta + q E_f`，RMSE `0.7816`, mGED `0`。
- **Lorentz `sigma^2=0.25`**：HierBOSSS `K=2` top model包含 `q B v sin theta` 和 `q E_f + q E_f`，RMSE `2.2027`, mGED `0`。
- **Lorentz `sigma^2=2.00`**：HierBOSSS RMSE `1.1079`, mGED `0`；BSR 和 QLattice RMSE 分别 `10.3110` 和 `6.72`，结构距离非零。
- **单原子催化 K=2**：HierBOSSS RMSE `0.9185`，核心描述符含 `-(Delta H_sub-Delta H_f,ox,bulk)` 与比值项。
- **单原子催化 K=3**：RMSE `0.8294`。
- **单原子催化 K=4**：RMSE `0.7979`，作者认为泛化表现从 K=4 后趋于平台。
- **单原子催化 K=5**：RMSE `0.7947`。
- **单原子催化 LASSO***：K=5 RMSE `0.9120`，高于 HierBOSSS。
- **单原子催化 BSR**：K=5 RMSE `0.9019`，高于 HierBOSSS。
- **单原子催化 iBART+`l0`**：K=5 RMSE `0.9235`，高于 HierBOSSS。
- **单原子催化 QLattice**：单一描述符 RMSE `0.7650`，预测接近/略优，但表达式含 Gaussian operator，作者认为更复杂且物理解释较弱。
- **MCMC diagnostics**：补充材料给出 log-JMP trace plots、Geweke Z-scores、Gelman-Rubin PSRF，用于说明链收敛。
- **反直觉点**：预测 RMSE 更低并不等于结构更正确；多处 BSR 在 RMSE 接近甚至更低时 mGED 很高。

### 明确局限
- **理论/实验参数不完全一致**：理论假设要求 `p_m <= q < 1/2`，但实验使用 `alpha=0.95`，根节点分裂概率显然超过 `1/2`；理论保证和主实验设置之间存在缺口。
- **检验不独立**：所有实验由作者完成，没有独立第三方复现实验或外部 benchmark 报告。
- **benchmark 覆盖有限**：Feynman 只重点展示 3 个方程，而 FSReD 有 100 个方程；结论不能直接外推到全库。
- **竞争方法调参风险**：BSR、QLattice、iBART 的设置虽有说明，但是否为各自最优调参未被系统搜索。
- **iBART 失败解释偏工程**：Feynman 上 iBART out-of-memory 和 penalized regression failure 被作为失败依据，但缺少更严格的资源等价对比。
- **真实数据很小**：single-atom catalysis 只有 `n=91`，out-of-sample 90/10 split 的测试集约 9 个点，泛化方差可能很大。
- **没有单位约束**：作者自己在 discussion 提到未来要加入 physical unit constraints；这对科学表达式发现很关键。
- **operator set 依赖强**：如果真式所需算子不在 `O` 中，方法只能做 misspecified 近似；operator set 的 expressibility 仍是开放问题。
- **表达式等价性困难**：mGED 捕捉图结构距离，但代数等价、尺度等价、冗余项抵消仍可能让结构评估变复杂。
- **全贝叶斯不等于全自动科学发现**：方法仍依赖人为选择特征、算子、K、先验参数、迭代次数和 top-r 策略。
- **反引很少**：截至本次 S2 查询，只有 1 条 2026 survey 引用；影响力和后续采用尚未形成。
