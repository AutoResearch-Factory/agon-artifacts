# Overcoming Spectral Bias via Cross-Attention (2512.18586)

- **Year**: 2025.12
- **Authors**: Xiaodong Feng (Beijing Normal-Hong Kong Baptist University), Tao Tang (Guangzhou Nanfang College / BNBU), Xiaoliang Wan (Louisiana State University), Tao Zhou (Chinese Academy of Sciences)
- **Venue**: arXiv
- **Keywords**: spectral bias, cross attention, random Fourier features, high-frequency approximation, PINN, Deep Ritz, adaptive frequency enhancement, multiscale PDE
- **One-liner**: 用 cross-attention 动态重权 multiscale RFF，缓解高频函数和 PDE 学习中的频谱偏置。

## 精华提炼

### 管线/方法

1. 论文的核心对象是高频/多尺度函数近似和 PDE 求解中的 spectral bias。
2. 作者把输入 `x` 先投影到 multiscale random Fourier feature bank。
3. base frequencies `omega_m` 从高斯分布采样，形成 `Omega_base`。
4. 对每个 base frequency 加 dyadic scale：`omega_{m,k}=2^k omega_m`。
5. 所有 scale 叠加后得到 `M = M_base * (K+1)` 个频率。
6. 每个频率配一个固定 random phase `b_{m,k}`。
7. 频率幅度用可学习 envelope `a_{m,k}=exp(-beta ||omega_{m,k}||_2)` 缩放。
8. `beta` 约束为非负，tex 中建议用 softplus 参数化。
9. RFF feature 是 `sqrt(1/M) * a_{m,k} cos(omega_{m,k}^T x + b_{m,k})`。
10. 这个向量再 reshape 成 token bank `H(x)`，维度为 `N_tok x d_q`。
11. 初始 latent `Q^(0)(x)` 可以来自 `psi(x)=phi(x)` 或 `psi(x)=x`。
12. `psi(x)=phi(x)` 时对应 RFF baseline 及 RFF-CA。
13. `psi(x)=x` 时对应 plain NN baseline 及 NN-CA。
14. 每一层用当前 latent query 去 cross-attend 到固定的 RFF token bank。
15. Query/key/value projection 都是 `d_q x d_q` 的可学习矩阵。
16. Cross attention logits 是 `Q_l K_l^T / sqrt(d_q)`。
17. Attention 输出以 residual 形式加回 `Q^(l)`。
18. 然后接一个 feed-forward residual block，得到 `Q^(l+1)`。
19. 经过 `L` 个 block 后用线性层输出 `u_theta(x)`。
20. 方法的关键 claim 不是“使用 Fourier features”，而是“用 input-dependent cross-attention 对 Fourier tokens 做动态选择/重权”。
21. 论文还提出 AFE: adaptive frequency enhancement。
22. AFE 的第一阶段先训练 baseline RFF-CA 得到 `u_theta^(0)`。
23. 在均匀网格上评估中间解，然后做 DFT/FFT。
24. 用阈值 `|u_hat_k| > lambda * max_j |u_hat_j|` 选 posterior modes。
25. Posterior frequency 设为 `omega_k^post = 2 pi k`。
26. Posterior Fourier features reshape 成额外 tokens。
27. 原始 tokens 和 posterior tokens 直接沿 token 维拼接为 `H_aug(x)`。
28. 为避免突然依赖 injected modes，作者在 attention logits 上加 mask。
29. Base token mask 为 0，posterior token mask 为 `eta_l <= 0`。
30. 训练后期让 `eta_l` 逐步上升到 0，使 posterior tokens 逐渐可用。
31. PDE 部分不是简单把同一网络套进 PINN。
32. 作者先指出 physics-based objective 会通过微分算子放大高频。
33. 1D Poisson 例子中，regression loss 表现为低频先收敛。
34. Deep Ritz 和 PINN loss 因一阶/二阶导数项对高频有 `k^2`/`k^4` 加权，反而高频误差下降更快。
35. 因此 PDE 中的问题从“高频学得慢”转成“高频信号在 residual 中过强，低频可能被压制”。
36. 为此作者提出两网络分解：`u = u_h + alpha u_l`。
37. `u_h` 由 RFF-CA 或 NN-CA 捕获高频。
38. `u_l` 由简单 fully connected network 捕获低频。
39. 边界条件处理上，`u_h` 满足原始边界，`u_l` 满足齐次边界。
40. `alpha` 有两种策略：作为 trainable scalar，或每个 epoch 解析求近似最优值。
41. 对线性 PDE，`alpha_opt` 由 residual inner product 和 `N[u_l]` 的二范数比值给出。
42. 实践中作者还把 boundary term 中的 `alpha_opt` 省略，以稳定估计。

### 关键技术细节

1. 默认全局容量：`m_base=128`，`K=3`，`d_q=64`。
2. Attention heads：`n_heads=4`。
3. Attention/residual blocks：默认 `L=4`，AFE 1D 例子里 `L=3`。
4. Learnable envelope 初始化：`beta_0=0.1`。
5. 训练精度：double precision。
6. Optimizer：Adam；PDE 2D Poisson 中使用 AdamW。
7. Gradient clipping：`1.0`。
8. 统一误差指标：relative `L^2` error。
9. 图像回归额外指标：PSNR 和 HFEN。
10. HFEN 用 Laplacian-of-Gaussian high-pass filter。
11. HFEN kernel：`15 x 15`，`sigma approx 1.5` pixels。
12. Synthetic 2D regression domain：`[-1,1]^2`。
13. Synthetic targets 包含 sectorwise anisotropy、narrow high-frequency ring、Gabor-like spiral packet、star-shaped discontinuity、checkerboard discontinuity。
14. Synthetic ground truth 和 test 都在 `500 x 500` grid 上评估。
15. Synthetic training mini-batch size：`4000`。
16. Synthetic base learning rate：`2e-3`。
17. Synthetic LR schedule：stepwise decay，`gamma=0.5`。
18. Synthetic epochs：`f1/f2` 为 1000，`f3` 为 500。
19. DIV2K 图像回归选 4 张 validation images。
20. 图像尺寸分别为 `2040x1356`、`2040x1536`、`2040x1344`、`1356x2040`。
21. 图像坐标映射到 `[-1,1]^2`，RGB 当作坐标函数值。
22. 图像数据 downsample factor：4。
23. 图像训练：full-batch Adam，5000 epochs。
24. AFE 1D target：`sin(2*pi*2x) + 0.5 sin(2*pi*20x+0.3) + 0.5 cos(2*pi*40x-0.2)`。
25. AFE training grid：`N_train=2048`。
26. AFE test grid：`N_test=4096`。
27. AFE FFT grid：`N_fft=4096`。
28. AFE threshold：`lambda=0.02`。
29. AFE extracted dominant modes：`k=2,20,40`。
30. AFE Stage 1：5000 epochs baseline training。
31. AFE Stage 2：5000 epochs augmented-token training。
32. AFE posterior-token mask：前 70% Stage 2 固定 `eta=-6`，后 30% cosine release 到 0。
33. 1D Poisson high-frequency parameter：`nu=100`。
34. 1D Poisson solution 混合 `0.1*pi`、`pi`、`nu/3*pi`、`2nu/3*pi`、`nu*pi` 五个尺度。
35. 2D Poisson exact solution：`sin(mu x_1^2)+sin(mu x_2^2)`。
36. 2D Poisson 测试 `mu=50` 和 `mu=100`。
37. 2D Poisson collocation：`N_r=1e4` interior points per iteration。
38. 2D Poisson boundary：每边 `N_b=1000`，四边。
39. 2D Poisson training：`2e4` AdamW steps，double precision。
40. 2D Poisson StepLR：step size 2000，decay `0.5`。
41. 2D Poisson boundary penalty：`lambda=1e4`。
42. Poisson-Boltzmann exact solution：`exp(sin(mu x1)+sin(mu x2)+sin(mu x3))/(|x|^2+1) * (|x|^2-1)`。
43. Poisson-Boltzmann `mu=15`。
44. Poisson-Boltzmann coefficients：`epsilon=1` everywhere，`kappa=1` in `Omega_1`，`kappa=5` in `Omega_2`。
45. Singular domain：centered ball radius 0.5 plus 20 small balls sampled on surface。
46. Small ball radii：uniform in `[0.1,0.2]`。
47. Whole domain truncated by ball radius 1 with zero boundary condition。
48. Poisson-Boltzmann training：10000 epochs。
49. Poisson-Boltzmann Adam learning rate：`1e-3`。
50. Poisson-Boltzmann StepLR：step size 1000，decay `0.6`。
51. Poisson-Boltzmann samples per epoch：5000 interior，4000 boundary。
52. Poisson-Boltzmann boundary penalty：`gamma=1e4`。

### 核心结果

1. Synthetic function regression 中，RFF-CA 在 `f1/f2/f3` 上相对 RFF-NN 都更低 training loss 和 relative `L^2` error。
2. 作者没有在 tex 中给出最终误差表；结论主要来自曲线和图像对比。
3. NN-CA 与 RFF-CA 谁更优随函数变化，说明 cross-attention 的收益依赖 target 结构。
4. 对 RFF family，加入 CA 的改进是更稳定的观察。
5. 在 pinwheel 和 checkerboard 可视化中，RFF-CA 比 RFF-NN 更快恢复局部振荡和 discontinuity 边界。
6. DIV2K image-as-function 中，NN-CA 相对 RFF-NN 取得更低 HFEN、更低 relative `L^2`、更高 PSNR。
7. 图像 zoom 对比显示 NN-CA 对 texture 和 edge contrast 保留更好。
8. AFE 1D 稀疏频谱例子中，Stage 1 从预测频谱中恢复 `k=2,20,40`。
9. Posterior tokens 释放后，AFE-enhanced RFF-CA 的 relative `L^2` 下降明显加速。
10. AFE 结果支持“attention module 可以无改 backbone 地接收新 spectral tokens”。
11. PDE 预实验显示 regression loss 下高频模式最慢。
12. 同一 Poisson 目标下，Deep Ritz 和 PINN objectives 明显加快高频模式误差下降。
13. 这个现象说明 PDE residual 的微分算子本身就是一种 spectral weighting。
14. 1D Poisson 混合策略中，loss 最小的是 learnable `alpha`。
15. 但 relative `L^2` error 最小的是 per-epoch optimal `alpha`。
16. 这是一条重要发现：training loss 最优不一定等于 solution error 最优。
17. `alpha=0` 会产生大的 structural error，说明只有高频子网不够。
18. `alpha=1` 比 `alpha=0` 明显好，说明低频分量是必要 correction。
19. Optimal scaling 初期取较大负值，用强低频 correction 快速降低 relative `L^2`。
20. 1D Poisson 结构比较中，RFF-CA 和 NN-CA 都比 RFF-NN 下降更快且最终误差更低。
21. 该实验里 NN-CA 达到最佳整体 accuracy。
22. Learnable `beta` amplitude scaling 在 `sigma=0.02` 的过高 prior frequency 设置下比 `beta=0` 更稳。
23. 对 unscaled RFF-CA，增大 boundary penalty 会改善精度，作者解释为更多低频边界信息注入。
24. 2D Poisson 中 `mu=50` 时 RFF-CA 误差地板略低且更稳定。
25. 2D Poisson 中 `mu=100` 时差距放大：RFF-NN 很快饱和，RFF-CA 继续下降。
26. 2D Poisson component visualization 显示 `u_h` 捕获 oscillatory structures，`alpha u_l` 提供 smooth correction。
27. Poisson-Boltzmann singular domain 中，RFF-CA 比 RFF-NN 更快降低 Deep Ritz loss 和 relative `L^2` error。
28. 最终预测图显示 RFF-CA 更接近 ground truth，RFF-NN 距离较远。
29. 全文最强 evidence 是多个任务类型上的一致方向：synthetic regression、image regression、Poisson、Poisson-Boltzmann 都显示 CA 有益。
30. 最弱 evidence 是缺少数值表、置信区间、跨随机种子统计和公开代码链接。

### 明确局限

1. 作者承认缺少 cross-attention multiscale Fourier representations 的严格 approximation/optimization dynamics 分析。
2. AFE 目前最自然适用于 periodic 或 grid-friendly settings。
3. AFE 扩展到 non-periodic geometry、复杂 domain、高维 full DFT 仍是开放问题。
4. 作者只在 remark 中建议复杂域可 embed 到 hypercube 并 masking，高维可做 1D component modes；没有系统实验。
5. PDE experiments 多为 manufactured solution，source term 由 exact solution 生成。
6. 这验证了方法的表达/优化能力，但离真实工程 PDE 的 unknown-solution regime 还有距离。
7. 论文没有报告 wall-clock、GPU、参数量、memory、attention token 数增加后的复杂度。
8. RFF-CA 相比 RFF-NN 的额外 attention 成本没有量化。
9. 缺少 ablation：token grouping size、head 数、`K`、`m_base`、`beta` initialization、mask release schedule。
10. 缺少随机种子统计；曲线是否稳定不能完全判断。
11. `RFF-NN/RFF-CA/NN-CA` 的公平性依赖“same global capacity parameters”，但参数量是否完全匹配没有逐项列出。
12. 论文结果大量依赖图，tex 没有 tabulated final errors。
13. 图像 regression 只用了 4 张 DIV2K validation images，样本规模偏小。
14. Poisson-Boltzmann 几何奇异例子有应用价值，但仍是构造解和构造几何。
15. `alpha_opt` 被称为最优 scaling，但在实验解读里有时描述为直接 minimizing prediction error；实际公式来自 residual objective 的线性最优条件，需注意表述边界。
16. 对 nonlinear PDE，`alpha_opt` 的解析形式不直接成立；论文主要写明 linear differential operator 的公式。
17. 没有和 2026 年出现的 one-shot/least-squares Fourier-feature PDE solver 比较。
18. 没有把 cross-attention spectral control 接入 neural operator pipelines，只在 conclusion 中列为未来方向。
19. 对 `0616-fex`，如果目标是“频率显式建模 + PDE/函数近似”，这是强相关 prior；如果目标是“agentic research/workflow 自动化”，则方法层面只有 representation/diagnostic 可借鉴，不构成系统级撞车。

### 跟我们的关系

1. 对 `0616-fex` 的撞车点取决于我们自己的核心 claim。
2. 如果我们的 claim 是“用 attention 在 Fourier/RFF tokens 中做 input-dependent frequency selection”，这篇高度撞车。
3. 如果我们的 claim 是“针对 PDE 高频/多尺度问题做频谱诊断后动态增强 feature bank”，这篇也高度相关，尤其 AFE 部分。
4. 如果我们的 claim 是“构建一个可解释/可审计的研究 agent 或实验流水线”，这篇不撞系统层面，但可以作为频谱诊断模块的技术来源。
5. 可直接借鉴的工具一：multiscale RFF bank + learnable exponential amplitude envelope。
6. 可直接借鉴的工具二：把 RFF features reshape 成 tokens，让 latent query cross-attend 到 frequency bank。
7. 可直接借鉴的工具三：中途 DFT 诊断 dominant modes，并通过 posterior tokens 增强 bank。
8. 可直接借鉴的工具四：attention mask schedule，先抑制 posterior tokens，后期逐步释放。
9. 可直接借鉴的工具五：PDE 中 `u_h + alpha u_l` 的高/低频两网络分解。
10. 可直接借鉴的 diagnostic：分别画 Fourier mode error，确认任务到底是低频先收敛还是 PDE residual 诱发高频放大。
11. 差异化机会一：把结果表格化，补充 seeds、置信区间、wall-clock、参数量和显存。
12. 差异化机会二：做非周期/非网格复杂域上的 AFE 替代方案，例如 graph spectral modes、local Fourier/wavelet packets、learned spectral probes。
13. 差异化机会三：把 spectral token selection 与 operator learning 或 time-dependent rollout 结合。
14. 差异化机会四：独立验证 `alpha_opt` 在 nonlinear PDE 或 non-manufactured setting 中是否仍优于 learnable `alpha`。
15. 近半年 arXiv 查重中，`2602.10541 FastLSQ` 是同一 Fourier-feature PDE solver setting 的邻近工作：它主打 analytical derivatives 和 one-shot least squares，不是 cross-attention RFF token routing。
16. 因此当前最接近的直接 prior 仍是本文引用的 frequency-adaptive multiscale DNN、FMMNN、RFF-PINN/eigenvector bias、MscaleDNN，而不是 2026 FastLSQ。
