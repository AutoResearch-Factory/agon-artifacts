# Mirror descent actor-critic methods for entropy regularised MDPs in general spaces: stability and convergence (2602.10838)

- **Year**: 2026.02
- **Authors**: Denis Zorba, David Šiška, Lukasz Szpruch (School of Mathematics, University of Edinburgh)
- **Venue**: arXiv
- **Keywords**: Actor-Critic, Mirror Descent, Temporal Difference Learning, Entropy Regularisation, Convergence Guarantees, Linear Function Approximation, Polish Spaces
- **One-liner**: Proves stability and (sub)linear convergence for discrete-time policy mirror descent with inexact TD-learned advantage functions in entropy-regularised MDPs over general Polish state/action spaces.

## 精华提炼

### 管线/方法

**问题设置**: 无限时域熵正则化 MDP (S, A, P, c, γ), S 和 A 均为 Polish 空间。目标是最小化正则化累积成本 V^π_τ。策略限制在 Π_μ（允许的策略类，密度关于参考测度 μ 的 exp 形式）。

**方法管线**:
1. **输入**: 初始策略 π^0 ∈ Π_μ, 初始 critic 参数 θ^0 ∈ R^N, 特征映射 φ: S×A → R^N
2. **Critic 步骤**: 线性函数近似 Q(s,a;θ) = ⟨θ, φ(s,a)⟩, 使用 TD 学习的半梯度 g(θ,π) 更新, 定义为 MSBE 对 θ 的偏梯度
3. **Actor 步骤**: Policy mirror descent — π^{n+1} = argmin_π ∫A(s,a;θ,π^n)π(da|s) + (1/λ)KL(π|π^n), 点态可解，有闭式解
4. **单循环** (Algorithm 1): 每轮 1 步 TD + 1 步 mirror descent
5. **双循环** (Algorithm 2): 每轮 M(n) 步 TD + 1 步 mirror descent, M(n) 随 n 增长
6. **输出**: 收敛的策略序列 {π^n} 使 V^π_τ(ρ) → V^*_τ(ρ)

### 关键技术细节

- **函数近似架构**: 线性 Q(s,a;θ) = ⟨θ, φ(s,a)⟩, φ 满足 ||φ(s,a)||₂ ≤ 1
- **核心假设**:
  - Assumption 1 (λ_β > 0): 特征协方差矩阵 ∫φφᵀ dβ 的最小特征值 > 0, 与策略无关
  - Assumption 2 (Q^π_τ-realisability): ∀π ∈ Π_μ, ∃θ_π 使 Q^π(s,a) = ⟨θ_π, φ(s,a)⟩
  - Assumption 3: ||φ(s,a)||₂ ≤ 1 (不失一般性)
- **关键参数**: Γ = λ_β(1-γ)(1-√γ), critic 步长 h, actor 步长 λ, 熵正则系数 τ > 0
- **Critic 步长约束** (单循环): h ≤ (1/2) min{Γ/(3(1+γ)²), (Γ² - 32γ²)/(Γ(48γ² + 3(1+γ)²))}
- **Critic 步长约束** (双循环): h < min{Γ/(2(1+γ)), 1/Γ}
- **Actor 步长**: 0 < τλ < 1
- **单循环稳定性条件**: 32γ²/Γ² < 1 (对小 γ 的限制)
- **双循环 TD 步数下界**: M(n) ≥ (4/(hΓ)) log(c(n+1))
- **线性收敛额外条件**: 集中度 (concentrability) 假设 |d(d^π*_ρ)/dρ|_{B_b(S)} ≤ ξ < ∞, 且 τλ ≤ (ξ-1)/ξ; M(n) ≥ (4c/(hΓ))(n+1)
- **Loss 函数**: Mean Squared Bellman Error (MSBE) — 关于 β ∈ P(S×A) 定义, β 与策略无关

### 核心结果

| 结果 | 设置 | 收敛速度 | 条件 |
|------|------|----------|------|
| Theorem 1 (稳定性) | 单循环, Polish 空间 | — | 32γ²/Γ² < 1, 小 h |
| Theorem 2 (收敛) | 单循环 | min_{r≤n-1} (V^r - V^*) ≤ C(1/√(nh) + λ/h) | 稳定性 + finite action 或 Thm 1 |
| Theorem 3 (稳定性) | 双循环, Polish 空间 | — | M(n) ≥ (4/(hΓ)) log(c(n+1)) |
| Theorem 4 (次线性收敛) | 双循环 | O(1/n) | M(n) 对数增长 |
| Theorem 5 (线性收敛) | 双循环 | O(e^{-min{1/ξ, c}n}) | 集中度假设 + M(n) 线性增长 |
| Corollary 1 (有限动作稳定性) | 单循环, |A| < ∞ | — | μ 全支撑自动满足 |

**Surprising/反直觉发现**:
1. 单步 TD 不足以保证任意 γ ∈ (0,1) 在 Polish 动作空间中的稳定性 — 需要 γ 足够小
2. 多步 TD 只需对数增长的步数（而非线性）即可获得稳定性
3. 双循环的总 TD 步数为 O(n log n)，但收敛速率按策略更新次数计算仍为 O(1/n)
4. 稳定性分析不依赖于 Q^π-realisability，仅依赖于特征矩阵正定性和有界性

### 明确局限

**作者明确指出的局限**:
1. 分析限于 population setting（所有积分精确计算），未扩展到 sample-based 设置（future work）
2. 使用线性函数近似而非神经网络 — 因为深度网络的监督学习收敛性尚未被证明
3. 未在真机或仿真环境中做实验验证
4. 稳定性条件对 γ 有限制（单循环），是充分非必要条件
5. 双循环的 M(n) 下界是充分条件，可能不是紧的

**我们读出的额外局限**:
1. 纯理论论文，无任何数值实验、仿真或基准测试来验证理论预测
2. 未与现有 policy gradient 方法 (PPO, TRPO, SAC) 做实验对比
3. Q^π-realisability 是强假设 — 真实 MDP 很难满足。虽然 Linear MDP 满足此假设，但讨论仅限于引用前人工作
4. 未讨论 λ_β > 0 在实际高维特征空间中的可满足性
5. 无代码、无数据、无可复现的实验结果
6. 未分析 critic 步长 h 和 actor 步长 λ 的具体推荐值或调参策略
7. 性能差异引理 (performance difference) 的积分项未给出计算复杂度分析
