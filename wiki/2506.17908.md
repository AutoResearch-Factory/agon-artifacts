# Robust PDE discovery under sparse and highly noisy conditions via attention neural networks (2506.17908)

- **Year**: 2025.06
- **Authors**: Shilin Zhang, Yunqing Huang, Nianyu Yi, Shihan Zhang; Xiangtan University / National Center for Applied Mathematics in Hunan
- **Venue**: arXiv
- **Keywords**: PDE discovery, ANN-PYSR, PySR, symbolic regression, residual attention network, Savitzky-Golay denoising, sparse noisy data, DLGA/R-DLGA
- **One-liner**: 用注意力网络+PySR从稀疏噪声数据识别PDE。

## 精华提炼

### 管线/方法
- 论文目标是从离散时空观测数据自动发现控制系统演化的 PDE。
- 输入是状态观测 \(U_{\text{data}}=\{u(x_i,t_j)\}\)，覆盖空间点和时间点。
- 噪声模型写成 \(\tilde u = u + \text{noise}\cdot \text{std}(u)\cdot N(0,1)\)。
- 第一步是对原始观测做 Savitzky-Golay 相关去噪，得到平滑后的 \(\tilde U_{\text{data}}\)。
- 作者先用 Gaussian convolution 做初始平滑，核函数记为 \(G(x,t;\sigma)\)。
- 然后用 Savitzky-Golay 滑动窗口做局部多项式拟合，窗口长度 \(l=2m+1\)，多项式阶数为 \(n\)。
- SG 参数通过最小化与 Gaussian-filtered signal 的 MSE 来选择，目标是找到 \((\sigma^*,l^*)\)。
- SG 拟合后的多项式也被用来估计导数，降低直接差分在高噪声下的误差传播。
- 第二步是用 residual attention neural network 学习响应函数和高阶偏导。
- 网络将输入 \(X\) 映射到两个静态特征空间 \(U=\phi(XW^1+b^1)\)、\(V=\phi(XW^2+b^2)\)。
- 正文公式说 \(\phi\) 选 sine activation，但实验配置段又说使用 tanh activation；这是一个未解释的不一致。
- 隐层通过 \(Z^{(k)}=\phi(H^{(k)}W^{z,k}+b^{z,k})\) 计算动态注意力权重。
- 隐层更新为 \(H^{(k+1)}=H^{(k)}+(1-Z^{(k)})\odot U+Z^{(k)}\odot V\)。
- 最终输出 \(u_\theta(X)=H^{(L+1)}W+b\)，并通过自动微分/网络导数得到 PDE 所需偏导。
- 第三步是把估计的 \(u_t,u_{tt},u_x,u_{xx},u_{xxx},u_{yy}\) 等候选变量送入 PySR。
- PySR 搜索显式数学表达式，候选表达式用变量集、算子集和常数集组成。
- 损失函数一处写成 MSE + \(\lambda\) complexity，另一处写成 Loss \(\times \exp(\lambda\cdot\) Complexity)，两种描述没有完全统一。
- 常数优化使用 L-BFGS，结构搜索使用 evolve-simplify-optimize 循环。
- PySR 最终按 loss 和 complexity 的 score 选择 Pareto 表达式。
- 论文选择的是在 loss 下降和复杂度增加之间最划算的表达式，而不是单纯最低 loss 的表达式。
- 输出是每个 benchmark 的显式 PDE，例如 Burgers、KdV、Klein-Gordon、Convection-Diffusion、Chaffee-Infante 和 2D Wave。

### 关键技术细节
- PySR population 数 \(n_k=40\)。
- 每个 population 有 \(L=1000\) 个表达式。
- 随机初始表达式 complexity 设为 3。
- 从历史最优集合 \(H\) 替换的比例 \(\alpha_H=0.05\)。
- 从跨 population 组合集合 \(M_i\) 替换的比例 \(\alpha_M=0.05\)。
- 神经网络实验配置说使用 6 层 Transformer。
- 每层有 8 个 attention heads。
- feedforward hidden dimension 扩展到 804。
- dropout rate 为 0.1。
- Burgers 数据：256 个空间点、201 个时间点，总计 51,456；空间 \(x\in(-8,8)\)，时间 \(t\in(0,10)\)。
- Burgers 扩散系数测试 \(\delta=0.02,0.04,0.08,0.1\)，训练/验证按 8:2 切分。
- Burgers 噪声实验覆盖 0% 到 250% 以上；摘要强调最高 200% noise、5000 sampling points。
- KdV 数据：512 个空间点、201 个时间点，总计 102,912；噪声从 0% 到 35%。
- Klein-Gordon 数据：201 个空间点、201 个时间点，总计 40,401；空间 \(x\in[-1,1]\)，时间 \(t\in[0,3]\)。
- Convection-Diffusion 数据：256 个空间点、100 个时间点，总计 25,600；空间 \(x\in[0,2]\)，时间 \(t\in[0,1]\)。
- Convection-Diffusion 噪声从 0.2 到 1.6，即表中最高 160%。
- Chaffee-Infante 数据：301 个空间点、200 个时间点，总计 60,200；噪声从 0% 到 80%。
- 2D Wave 方程测试 \(u_{tt}=u_{xx}+u_{yy}\)，噪声为 0.1、0.2、0.3、0.4。
- 评测指标包括 solution relative error 和 average coefficient error。
- 论文没有给出公开代码、随机种子、完整训练轮数、learning rate、batch size 或独立复现实验。
- 实验全是合成 canonical PDE，没有真实传感器数据或真实实验系统。

### 核心结果
- Burgers \(\delta=0.1\)、200% noise、Exp IC 时，选出的式子为 \(u_t=-0.9898uu_x+0.0981u_{xx}\)。
- Burgers Sine IC、50% noise 时，选出的式子为 \(u_t=-0.9615uu_x+0.0960u_{xx}\)。
- Burgers \(\delta=0.1\) 表中最高到 280% noise，仍报告 \(u_t=-0.9367uu_x+0.1018u_{xx}\)，average coefficient error 0.0427。
- Burgers 30% noise 对比表：ANN-PYSR error 0.0035、time 47.86s。
- 同一 Burgers 对比表：DLGA error 0.4000、time 1973.32s；R-DLGA error 0.0074、time 2162.71s；NN-GA error 0.1611、time 40.58s。
- KdV 10% noise 主表：\(u_t=-1.0065uu_x-0.002489u_{xxx}\)，average coefficient error 0.0654，SR time 111.92s。
- KdV 对比表：ANN-PYSR error 0.0061、time 150.29s；R-DLGA error 0.0064、time 2745.72s。
- KdV 35% noise 时仍恢复结构，但系数误差升到 0.2535。
- Klein-Gordon 30% noise complexity-7 表达式为 \(u_{tt}=0.4816u_{xx}-5.4343u\)，接近真式 \(0.5u_{xx}-5u\)。
- Klein-Gordon 文中称 noise <= 0.4 时 average coefficient error 约 1%，但注释掉的表和正文数值没有完整一致地支撑这个说法。
- Convection-Diffusion 60% noise complexity-7 表达式为 \(u_t=-0.9176u_x+0.2433u_{xx}\)。
- Convection-Diffusion 160% noise 表达式为 \(u_t=-0.9067u_x+0.2336u_{xx}\)，average coefficient error 0.0550。
- Chaffee-Infante 80% noise 表达式为 \(u_t=1.393u_{xx}-0.824u+1.285u^3\)，\(R^2\) 仍报 0.9081。
- Chaffee-Infante 80% noise 的 coefficient error 为 0.285，说明非线性项受噪声影响明显。
- 2D Wave 10% noise 表达式为 \(u_{tt}=1.0766u_{xx}+1.0032u_{yy}\)。
- 2D Wave 40% noise 表达式为 \(u_{tt}=0.7942u_{xx}+1.1258u_{yy}\)。
- 2D Wave 段落声称 40% noise coefficient error 约 3.25%，但表中两个系数相对 1 的平均偏差约 16.6%，这里存在明显可疑之处。
- Discussion 热图结论：低噪声 <=10% 时，约 10,000 samples 可低于 5% error。
- Discussion 建议：sampling density 至少 50%，noise level 低于 20%，更容易把 error 控制在 10% 以下。
- 论文结论又说 as few as 1000 data points，与摘要里的 5000 sampling points 和 discussion 的 10,000/50,000 经验阈值并不完全一致。

### 明确局限
- 最核心局限是验证环境完全合成，缺少真实实验噪声、传感器漂移、缺测、异方差噪声和时空相关噪声。
- 论文没有公开代码和数据，复现实验需要重新实现 SG、attention network、PySR 配置和 benchmark 生成器。
- 论文没有给出多随机种子均值/方差，也没有置信区间。
- 论文没有充分 ablation：SG 去噪、attention network、PySR 分别贡献多少不清楚。
- 论文没有系统比较 weak-SINDy、PDE-LEARN、SINDy-PI、积分形式 PDE discovery 等更强的 noisy-data baseline。
- DLGA/R-DLGA 对比有价值，但都是相近 canonical benchmark；不能证明真实复杂系统有效。
- 部分段落数字不一致，例如 Burgers/KdV 结论和表格中的 error 数值不完全一致。
- 2D Wave 的文字 coefficient error 与表格系数偏差不匹配，需复算。
- 源码 bibliography 中至少两个 arXiv 号与题名错配：`1911.02575` 实际不是 Implicit-SINDy，`2106.10359` 实际不是 Symbolic Genetic Algorithm for PDE Discovery。
- 参考文献错配降低了论文整理质量和可审计性。
- 使用 PySR 后，最终表达式选择仍依赖 score/complexity tradeoff，某些更高 complexity 候选 loss 更低但被舍弃。
- "highly noisy" 的 200%/280% 是按 std(u) 缩放的 synthetic Gaussian noise，不等价于所有真实测量场景的 SNR。
- 论文没有展示 out-of-distribution PDE family、边界条件变化、非均匀网格、真实稀疏 sensor placement 的系统评估。
- 没有验证发现的 PDE 是否能长期 rollout 稳定预测，更多是短期解或系数级评估。
- 没有人类专家独立核查每个 discovered equation 是否物理合理。

- 差异化方向二：把验证者与生成者分离，例如独立数值 solver rollout、物理守恒检查、不同 symbolic engine cross-check。
- 差异化方向三：做强 ablation 和复现包，明确 SG、attention、PySR 各自贡献。
- 差异化方向四：和 2026 年 `2603.22380` 的 graph-message-passing route 区分开，强调局部导数估计、非局部图表示或混合验证机制的不同。

## Read by: 0616-fex (2026-06-16)

从 `0616-fex` 的角度，这篇论文最值得记录的是“SG 去噪 + residual attention derivative surrogate + PySR”的完整工程 pipeline，而不是单个新算法。它的 novelty 更像是把已有部件组合到 sparse/noisy PDE discovery 上，并在六个 synthetic benchmark 上做噪声和采样 sweep。若我们的课题也要宣称在稀疏、高噪声物理数据中自动发现 PDE，则需要明确避开“attention network + PySR + canonical PDE benchmark”这条路线，或者提供更强的验证，例如真实数据、独立 solver rollout、跨 symbolic engine 复核、非高斯/相关噪声和公开复现包。近半年 arxiv_tool 搜索发现 `2603.22380` 用 Symbolic Graph Networks 处理 noisy sparse PDE discovery，它是更接近当前前沿的并行工作；我们的差异化最好放在验证独立性、可复现性和真实场景，而不是只换一个 neural surrogate。
