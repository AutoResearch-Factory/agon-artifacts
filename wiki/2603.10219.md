# A Diffusion Analysis of Policy Gradient for Stochastic Bandits (2603.10219)

- **Year**: 2026.03
- **Authors**: Tor Lattimore (Google DeepMind)
- **Venue**: arXiv preprint
- **Keywords**: policy gradient, stochastic bandits, diffusion approximation, continuous-time SDE, softmax policy, regret analysis, Itô calculus, learning rate sensitivity
- **One-liner**: 用连续时间 SDE 扩散过程逼近 softmax policy gradient 在 k-arm 随机 bandit 上的动力学,证明学习率需为 O(Δ²/log(n)) 才能获得次线性 regret,并构造反例揭示 k>2 时"随机选赢家"现象导致线性 regret。

## 精华提炼

### 管线/方法

**问题设定**：k-arm stochastic Gaussian bandit, 均值 μ∈Rᵏ (μ₁>μ₂≥...≥μ_k≥0), 方差 σ∈Rᵏ (‖σ‖∞≤1), horizon n。离散时间 policy gradient (Alg 1): θ_{t+1,a} = θ_{t,a} + η (𝟙_{A_t=a} - π_{t,a}) Y_t, 其中 π_t = softmax(θ_t)。

**连续时间逼近 (Alg 2)**：用 SDE 替代离散更新:
- 观察过程: dX_t = diag(π_t) μ dt + diag(√π_t) Σ^{1/2} dB_t, 其中 B_t 是 k 维标准布朗运动
- 参数更新: dθ_t = η (I - π_t 𝟙ᵀ) dX_t
- 展开后: dθ_t = η (diag(π_t) - π_t π_tᵀ) μ dt + η diag(√π_t) Σ^{1/2} dB_t - η π_t ⟨√π_t, Σ^{1/2} dB_t⟩

**分析管线**：
1. 建立 basic properties: conservation (∑θ_{t,a}=0 a.s.), lower bound on θ_{t,a} (Lemma 2)
2. 引入关键量 Z_{t,a} = θ_{t,1} - θ_{t,a}, 推导其 SDE (Lemma 3): dZ_{t,a} = η[π_{t,a}Δ_a + (π_{t,1}-π_{t,a})R_t] dt + η√(π_{t,1}+π_{t,a}-(π_{t,1}-π_{t,a})²) dW_t
3. Upper bound: 通过 time change + Itô's formula + ψ-function 构造上界
4. Lower bound: 构造坏实例 + 定义 stopping time τ + SDE comparison + integrating factors

### 关键技术细节

**模型架构**：纯分析性论文,无神经网络。核心数学工具:
- **Itô 微积分**：连续时间 SDE 的 drift-diffusion 分解
- **Time change**：s(t) = ∫₀ᵗ C_u du (其中 C_t = η π_{t,3+}(1-π_{t,3+})), 将 SDE 简化为 drift 关于 C_t 线性的形式
- **Integrating factors**：对形如 dX_t = [A_t + B_t X_t] dt + C_t dW_t 的线性 SDE 使用标准解法
- **Stopping time 构造**：τ = inf{t: min Z_{t,a} ≤ -Δ/2 或 min θ_{t,a} ≤ -log(n/δ)}

**关键参数设定**：
- Upper bound: η ≤ Δ²/(8 log(2n²)) 时, E[Reg_n] = O(k log(k) log(n) / η)
- 2-arm special case (Prop 1): η < Δ (即 a = Δ/η > 1) 时, E[Reg_n] ≤ (a/(2Δ)) log(1+2(a+1)nΔ²/a²) + a²/(2(a-1)Δ)
- Lower bound: Δ = (0, Δ, 1, ..., 1), Σ = diag(1, 1, 0, ..., 0), k ≥ C log(n/Δ), η ∈ [CΔ², c/k] 时, E[Reg_n] = Ω(nΔ)

**ψ-function 构造**：ψ'(u) = 6k log(n/δ)/(u+1+log(n/δ)), ψ(u) = 6k log(n/δ) log((u+1+log(n/δ))/(1+log(n/δ))), 关键性质: ψ''(u) ≥ -ψ'(u), 使得 Itô 公式中二阶项被吸收

**SDE comparison 技术**：将复杂 SDE (Z_t, S_t) 通过 time change 转化为可控形式, 然后与标准线性 SDE 比较, 用 integrating factors 得到显式解

### 核心结果

| 结果 | 条件 | Regret | 关键发现 |
|------|------|--------|---------|
| Prop 1 (2-arm) | η < Δ | ~log(n)/(2Δ) | 2-arm 时 policy gradient 近乎最优 |
| Thm 1 (upper) | η ≤ Δ²/(8log(2n²)) | O(k log k log n / η) | k>2 时需要小得多的学习率 |
| Thm 2 (lower) | η ∈ [CΔ², c/k] | Ω(nΔ) | 存在实例导致线性 regret |
| Remark (equal gaps) | η = cΔ/log(n) | O(k log k log²n / Δ) | 所有 arm gap 相同时可放宽条件 |

**Surprising/反直觉发现**：
1. **k=2 vs k>2 的根本差异**：2-arm 时 Z_t = θ_{t,1}-θ_{t,2} 的 drift 始终为正 → 算法总是朝正确方向学习。但 k>2 时 drift 项含 (π_{t,1}-π_{t,a})R_t, 当 regret 大且 π_{t,1}<π_{t,a} 时 drift 可为负。
2. **"随机选赢家"现象**：当多个 near-optimal arm 几乎不可区分时, 噪声主导动力学, policy gradient 会随机选一个 arm 作为赢家, 选错则导致线性 regret。
3. **学习率需为 O(Δ²) 而非 O(Δ)**：这是比以往认知更严格的约束。即使 k 是对数级别, 也需要极小的学习率。
4. **continuous time 近似的高质量性**：作者声称(但未证明)连续时间逼近是高质量的。实验表明离散时间行为与连续时间预测一致 (Fig. 1)。

### 明确局限

**作者自述的局限**：
1. 连续时间分析是离散时间真实算法的近似, 尚未证明近似的紧致性
2. Upper bound 证明技术"可能"可迁移到离散时间, 但 lower bound 在离散时间"更具挑战性且价值较低"
3. Lower bound 中假设 Σ = diag(e₁+e₂) (仅有前两臂有噪声), 简化了分析
4. 对数因子 log(n) 是否可去除未解决
5. 未建立对 k 的线性依赖下界

**审读发现的局限**：
1. **无真机实验**：纯理论分析, 仅附录有离散时间仿真验证 lower bound 构造
2. **与我们的关系**：本文是 bandit/RL 理论, 与 PDE 基础模型 (0616-fex) 无直接技术重叠。但其 SDE 分析工具 (time change、integrating factors、stopping time 构造) 可能对分析神经算子训练的连续时间动力学有启发
3. **学习率敏感性的普遍启示**：当存在多个竞争性特征/算子且难以区分时, 学习率必须极小 → 这对 PDE foundation model 中 multi-scale operator 训练有潜在警示意义
4. **单一作者**：Tor Lattimore 是 bandit 领域权威, 但单人论文缺少独立验证
