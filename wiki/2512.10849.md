# Bayesian Symbolic Regression via Posterior Sampling (2512.10849)

- **Year**: 2025.12
- **Authors**: G. F. Bomarito, P. Leser; NASA Langley Research Center
- **Venue**: arXiv; published-version metadata found via S2 as Philosophical Transactions A, DOI:10.1098/rsta.2024.0590
- **Keywords**: symbolic regression, Bayesian symbolic regression, Sequential Monte Carlo, posterior sampling, normalized marginal likelihood, uncertainty quantification, Feynman benchmark, SRBench
- **One-liner**: 用 SMC 近似符号表达式后验, 在含噪 Feynman benchmark 上比 GP baseline 更抗过拟合。

## 精华提炼

### 管线/方法
- 输入是监督回归数据集 `D={(x_j,y_j)}`, 目标是发现可解释的符号表达式 `M(x,theta)`。
- 论文把 Bayesian SR 写成模型与参数的联合后验 `pi(theta,M|D)`。
- 为避免在不同维度参数空间之间跳转, 它先对参数 `theta` 积分, 得到模型后验 `pi(M|D)`。
- 这个模型后验正比于模型先验 `pi(M)` 乘以边缘似然。
- 由于 SR 中常用的参数先验是不适定均匀先验, 直接边缘似然会有不确定常数。
- 作者使用 normalized marginal likelihood, NML, 来规避这个常数问题。
- NML 中的 `gamma` 取经验值 `gamma=1/sqrt(N_d)`。
- 实际算法没有用内层 SMC 精确估计 NML, 而是用 Laplace approximation。
- 近似式是 `q_hat = gamma^(N_theta/2) * likelihood(D|theta*,M)^(1-gamma)`。
- 其中 `theta*` 是 MAP/maximum likelihood 点, 由 gradient-based optimization 估计。
- SMC-SR 初始化一个表达式粒子群 `P={M_1,...,M_NP}`。
- 初始表达式从近似均匀模型先验中随机生成, 并做符号化简与 canonical form 去重。
- 每个粒子初始权重是 `1/N_P`。
- 算法通过 likelihood tempering 从先验平滑过渡到后验。
- 第 `t` 个 target 是 `pi_t(M|D) proportional pi(M) q_hat(M)^phi_t`。
- `phi_0=0`, 最终 `phi_T=1`。
- 每轮先自适应更新 `phi_t`, 再基于 `q_hat` 对粒子重加权。
- `Delta phi_t` 由 bisection 选择, 目标是维持指定 ESS。
- 论文正文中 `Delta phi_t = phi_t - phi_t` 明显是排版/符号笔误, 应理解为当前与上一 tempering level 的差。
- 重加权后做 stratified resampling, 高权重表达式被复制, 低权重表达式被淘汰。
- 重采样会减少唯一表达式数量, 所以后接短 MCMC rejuvenation kernel。
- MCMC kernel 用 genetic programming 风格的 crossover 和 mutation 生成 offspring。
- proposal 写作 `P' ~ h(P'|P)`, 使用 BINGO/GP 变异操作。
- 对每个 parent-offspring pair, 接受率为 `alpha=min(1,(q_hat(M')/q_hat(M))^phi_t)`。
- 这和标准 GP deterministic crowding 的硬接受不同: GP 只在 loss 改善时接受 offspring。
- 论文把这种概率接受加 likelihood-tempering 解释为 targeted novelty。
- 它不是追求任意新表达式, 而是在高后验密度附近保持探索。
- 输出包括最终表达式 population, MAP/近似 MAP 表达式, posterior-like 模型集合, 以及模型形式不确定性。

### 关键技术细节
- 论文主算法名是 SMC-SR, 即 Sequential Monte Carlo Symbolic Regression。
- 比较对象是 3 个 GPSR baseline: GP-MSE, GP-NML, GP-agg。
- GP-MSE 使用 MSE loss 和 deterministic crowding selection。
- GP-NML 使用 NML loss 和 deterministic crowding selection。
- GP-agg 使用 MSE loss 和 tournament selection, selection pressure 更强。
- SMC-SR 与 GP baseline 的 population size 都设为 `N_P=2000`。
- SMC 的 MCMC rejuvenation 步数设为 `N_MCMC=10`。
- target ESS 设为 `1900`, 即 population 的 `95%`。
- 作者没有系统调参, 只说明较大的 population size 比小 population 更有利。
- 噪声模型假设 `epsilon_j iid~N(0,sigma)`。
- 模型先验默认是 uniform prior `pi(M) proportional 1`。
- 作者指出 informative priors 可以替换, 例如 Bartlett 等人的物理先验/结构先验。
- benchmark 选择 12 个 Feynman physics equations。
- 这些数据来自 SRBench/Feynman 系列, 作者称是较难子集, 计划进入 SRBench revision。
- 每个 Feynman 数据集总量是 10,000 data points。
- 输入维度 `N_x` 范围是 3 到 8。
- 每次训练抽取 `N_d=500` 个点, 其余作为 test set。
- 每个 dataset 重复训练 20 次。
- 训练集添加 Gaussian noise, 标准差是 `0.1||Y||`。
- `||Y||` 用 `median(|y|)` 计算, 以避免 asymptote 附近极端值影响。
- 主要指标是 NRMSE-train 与 NRMSE-test。
- NRMSE=0.1 大致等于达到训练噪声水平。
- 为保持计算量可比, GPSR generation 数设置为 `T * N_MCMC`, 其中 `T` 是 SMC 自适应 tempering steps。
- 模型选择比较了三种方式: best training error/NML, 20% training-size validation set, final population mode。

### 核心结果
- 在单个示例中, SMC-SR 的 likelihood tempering 先保持较宽 NML 分布, 后逐渐收缩到多峰 posterior。
- demo 数据是 `y = 1/(2*pi) exp(-x^2/2) + epsilon`。
- demo 中 `x~Uniform(-3,3)`, `epsilon~Normal(0,0.16)`, 数据量是 25。
- demo 只允许 `[+, -, times]` operators, 使问题比直接允许 exponential 更难。
- 在 Feynman I-32-17 上, SMC-SR 比三种 GP baseline 取得更低训练 NRMSE。
- 在 Feynman II-36-38 上, GP-agg 更快压低训练误差, 但 test NRMSE 很差, 显示强 selection pressure 容易过拟合。
- GP-NML 相比 GP-MSE 有一定帮助, 说明 NML regularization 有贡献。
- 但 GP-agg 失败说明单纯提高 selection pressure 不足以解释 SMC-SR 的收益。
- 论文的关键解释是概率接受和 likelihood-tempering 共同带来 targeted novelty。
- 在 12 个 Feynman 数据集汇总上, SMC-SR 更少 underfit。
- GPSR 方法几乎在所有数据集都显示 train-test gap, SMC-SR 只在 5/12 数据集上过拟合。
- 最终 population 的最小 NRMSE-test 显示 SMC-SR 更常遇到高质量表达式。
- 这个最小 test error 是 oracle 视角, 实际不能直接用于模型选择。
- 现实模型选择中, final population mode 通常优于 best training loss。
- mode selection 甚至优于使用额外 validation data 的选择。
- ground-truth identification rate 也呈现同样趋势: mode selection 更好。
- 复杂度曲线显示 SMC-SR 早期会像 GP-MSE 一样快速增加 expression complexity。
- 随着 `phi -> 1`, NML regularization 起效, complexity 与参数数量趋近 GP-NML。
- GP-agg 的 complexity 增长过快, 是强 selection pressure 的另一个风险信号。
- diversity 分析里, SMC-SR 遇到的 total unique expressions 约为 GP 方法的 50%。
- 这个结果反直觉: 更好效果不是来自纯粹遍历更多表达式。
- 但 SMC-SR final population 中被接受进 population 的 unique models 更多、更动态。
- 作者据此主张: 有效的是高 posterior density 区域内的新颖性, 不是无差别 novelty。

### 明确局限
- 实验只覆盖 12 个 Feynman/SRBench synthetic datasets, 没有真实工程数据或真机/湿实验验证。
- 所有结果主要来自图, TeX 中没有完整数值表, 复核精确 effect size 需要代码或原始日志。
- SMC-SR 和 GP baseline 的 operator set、mutation/crossover 细节放在 supplementary appendix, 当前 arXiv tex 中未包含 appendix 文件。
- proposal distribution 明确是 asymmetric, 作者承认这对严格 MCMC ergodicity 不利。
- 作者的理由是 SMC 外层可以放松 ergodicity, 但这仍是理论弱点。
- NML 用 Laplace approximation, 作者承认它比内层 SMC 快很多但对常见 SR 模型不够精确。
- 每个 expression 的 `theta*` 优化使用随机初始化, duplicate expression 会重复优化, 有效率损失。
- 作者建议未来缓存 `theta` 并复用优化结果, 引用 2025 gradient-based local search 方向。
- uniform model prior 未必合理, 初始随机表达式生成也只是近似 uniform。
- 没有看到针对 hyperparameters `N_P`, `N_MCMC`, ESS 的系统 ablation。
- 没有和 2025-2026 新的 variational Bayesian SR 方法, 如 DVISR 或 VaSST, 做实验对比。
- 不确定性展示偏模型形式/posterior population, 不是完整 joint posterior credible intervals。
- 若要得到 `pi(theta,M|D)`, 作者还需对 final population 中每个模型做参数 calibration。
- 论文声称 open-source software 是 `github.com/nasa/pysips`, 但本文 TeX 没有给 commit hash 或 artifact DOI。
- 对高维、强噪声、真实 measurement process、非 iid noise 的稳定性仍未验证。

### 跟我们的关系
- 对 `0616-fex` 来说, 这篇是 strong related work, 尤其如果课题涉及 formula/expression discovery、Feynman equations、符号回归或模型形式不确定性。
- 如果我们的 claim 是“在含噪科学数据上用 Bayesian/posterior 方法做 symbolic expression discovery”, 它会直接撞车。
- 具体撞车点是: Bayesian posterior over symbolic expressions, SMC posterior sampling, NML-based model evidence, noisy Feynman benchmark, mode-based model selection。
- 如果我们的方案是 LLM/agent 生成表达式再由 verifier 筛选, 它不直接撞方法, 但会撞任务 setting 与 UQ framing。
- 它可用作 baseline: SMC-SR 或 `pysips` 应作为 Bayesian SR baseline, 至少在 Feynman/SRBench 上比较。
- 它也可用作方法组件: NML/Laplace evidence、posterior population mode、ESS-based tempering、stratified resampling。
- 差异化方向一: 使用外部 verifier 或独立 simulator gate, 提高 R2 verification independence。
- 差异化方向二: 不只在 synthetic Feynman 上做, 增加真实数据、非 iid noise、数据稀缺或外推评测。
- 差异化方向三: 与 2026 DVISR、VaSST 这类 variational Bayesian SR 近作比较, 定位 SMC 与 VI 的 tradeoff。
- 差异化方向四: 若 `0616-fex` 是 agentic/LLM-driven expression search, 应把本文作为非 LLM posterior baseline, 避免声称“首次 UQ symbolic discovery”。
- 该引。相关工作中应同时引用 Jin 2019 Bayesian SR、Guimera 2020 Bayesian machine scientist、Bomarito/Leser 2025 SMC-SR, 以及近期 DVISR/VaSST。
