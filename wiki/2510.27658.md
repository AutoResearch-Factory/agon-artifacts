# What Can One Expect When Solving PDEs Using Shallow Neural Networks? (2510.27658)

- **Year**: 2025.10
- **Authors**: Roy Y. He (City University of Hong Kong), Ying Liang (Duke University), Hongkai Zhao (Duke University), Yimin Zhong (Auburn University)
- **Venue**: arXiv
- **Keywords**: PINN, Deep Ritz Method, shallow neural networks, elliptic PDE, spectral bias, ill-conditioning, random features, boundary regularization
- **One-liner**: 分析浅层网络解椭圆 PDE 时的病态和频率偏置。

## 精华提炼

### 管线/方法
- 论文研究的问题是：用两层浅层神经网络表示 PDE 解时，PINN 和 Deep Ritz Method 到底能期待什么精度、稳定性、收敛速度和计算代价。
- 研究对象限定为二阶椭圆方程，主实验进一步简化到一维 Poisson 方程和一维变系数椭圆方程。
- 网络形式是宽度为 `N` 的浅层网络 `u(x)=sum_i a_i sigma(w_i x-b_i)`。
- 作者先冻结第一层参数 `w_i,b_i`，只优化输出层系数 `a_i`。
- 在冻结第一层后，PINN 和 DRM 都退化为线性随机基/随机特征方法。
- PINN 的域内损失使用强形式残差，等价于最小化二阶导数残差的二次型。
- DRM 的域内损失使用弱形式能量，等价于最小化一阶导数能量的二次型。
- Dirichlet 边界条件采用两种处理方式：硬约束和边界正则化。
- 硬约束形式被写成 KKT 系统，矩阵块为 Gram/stiffness 矩阵加边界约束矩阵。
- 边界正则化形式被写成 `G_F + lambda B^T B` 的正定线性系统。
- 主分析先聚焦 `ReLU^p` 激活，因为它能和 B-spline/FEM 基函数建立明确代数关系。
- 对均匀 bias 的 `ReLU^p` 浅层网络，作者证明其张成空间等价于 p 次样条空间。
- 关键差别是，`ReLU^p` 全局基和 B-spline 局部基虽然张成同一个空间，但 Gram 矩阵谱性质完全不同。
- 作者把 `ReLU^p` Gram 矩阵关联到积分核算子 `K_p`。
- 核函数满足高阶微分方程，进而推出算子特征值按 `k^{-(2p+2)}` 衰减。
- 由核谱再推出有限维 Gram 矩阵的谱衰减和条件数下界。
- 对 PINN 和 DRM 的 KKT 系统，论文进一步给出与激活光滑度相关的谱衰减率。
- 之后用直接求解、SVD 截断、Projected Gradient Descent、边界正则化扫描、Neumann 条件比较来验证这些谱结论。
- 在非齐次激活部分，作者改用 `sin`、`tanh`、GELU，研究初始化 scaling `S` 对谱和误差的影响。
- 最后进入 fully trainable SNN，比较固定随机基的线性 PINN 和所有参数可训练的非线性 PINN。

### 关键技术细节
- PDE 形式是 `Lu=f` in `Omega`，`u=g` on boundary，其中 `L=-sum_ij partial_j(a^ij partial_i u)+c u`。
- 弱形式使用双线性型 `B[u,v]=int a^ij partial_i u partial_j v + int c u v`。
- 浅层网络为 `u(x,a,w,b)=sum_{i=1}^N a_i sigma(w_i^T x-b_i)`。
- PINN 域内损失是 `int_Omega (Lu-f)^2 dx`。
- DRM 域内损失是 `1/2 B[u,u] - int f u dx`。
- 固定第一层后一维化，`Omega=(-1,1)`，只解输出层系数 `a in R^N`。
- Gram 矩阵定义为 `G_sigma^(k)[i,j]=int_{-1}^1 w_i^k w_j^k sigma^(k)(w_i x-b_i) sigma^(k)(w_j x-b_j) dx`。
- 对 PINN，有 `G_PINN=G_sigma^(2)`，`y_PINN=-y_sigma^(2)`。
- 对 DRM，有 `G_DRM=G_sigma^(1)`，`y_DRM=y_sigma^(0)`。
- 边界矩阵 `B` 是两行，分别评估网络基函数在 `x=-1` 和 `x=1` 的取值。
- `ReLU^p` 要满足可微性：PINN 至少 `p>=2`，DRM 至少 `p>=1`。
- `ReLU^p` 与 cardinal B-spline 的关系通过有限差分公式给出：p 次 B-spline 是若干 `ReLU^p(x-k)` 的线性组合。
- 对均匀 grid bias，`span{ReLU^p(x-b_i)}` 等于 p 次样条空间。
- 但变换矩阵 `W` 非正交，所以 `ReLU^p` Gram 与 FEM Gram 只是 congruent，不保持条件数。
- 核算子特征值满足 `mu_{k,p}=O(k^{-(2p+2)})`。
- 准均匀 bias 下，`G_sigma^(k)` 条件数至少是 `Omega(N^{1+2(p-k)})`。
- 均匀 bias 下更尖锐：条件数为 `Theta(N^{2+2(p-k)})`。
- PINN with `ReLU^p` 的 KKT 正特征值按 `Theta(j^{2-2p})` 衰减。
- DRM with `ReLU^p` 的 KKT 正特征值按 `Theta(j^{-2p})` 衰减。
- 这意味着更平滑的激活、更大的 `p` 和更大的宽度 `N` 都会加重病态。
- 作者强调 double precision 约 `1e-16` 会限制可稳定恢复的高频模式数。
- 数值特征分解使用 SciPy `eigh`，`ReLU^p` 积分用附录中的显式公式精确计算。
- 主要直接求解实验使用 `N=100,300,500`，解函数为单 Fourier mode 或两个 Fourier mode 的叠加。
- 两频解典型形式是 `sin(2 pi x + phase)+sin(k_max pi x + phase)`。
- SVD 截断实验用 `N=400`，比较 FEM、PINN `ReLU^3`、DRM `ReLU^2`。
- PGD 实验用 `N=200`，学习率设为 `1.99/lambda_max(P G_F P)`。
- 边界正则化实验中，PINN 用 `ReLU^3` 和 `lambda=0.1 k_max`，DRM 用 `ReLU^2` 和文中图注/正文给出的随 `k_max` 放大的 lambda。
- lambda 扫描固定 `N=300`，比较 `p=2,3,4`，其中 PINN 用 `ReLU^p`，DRM 用 `ReLU^{p-1}`。
- 非齐次激活实验用 `sin`、`tanh`、GELU，`N=300`，`w_i~U(-S,S)`，`b_i~U(-1,1)`，Riemann sum 使用 10000 个 regular grid points。
- fully trainable PINN 的变系数实验使用 `D(x)=sin(pi x)+2`。
- 变系数实验的目标解是带 cutoff 的三个局部振荡 bump 叠加，参数含 `omega_1=60, omega_2=300, omega_3=40`。
- fully trainable 实验网络宽度 `N=100`，`sin` 激活，固定采样 500 点。
- Adam 学习率为 `8e-4`，epoch 数为 20000，边界正则化权重 `lambda=250`。
- 每个 scaling 设置报告 20 次独立随机初始化的均值。

### 核心结果
- 核心结论 1：浅层 `ReLU^p` 网络在 PDE 求解中主要受自身 Gram 矩阵病态和反高频频率偏置支配。
- 核心结论 2：这个网络表征偏置会压过二阶微分算子本身对高频的放大效应。
- 核心结论 3：PINN/DRM 用梯度法时，低频分量先学到，高频分量恢复很慢或受机器精度限制无法稳定恢复。
- 核心结论 4：FEM 的频率偏置方向相反。无预条件迭代 FEM 中高频误差先降，低频误差慢；PINN/DRM 中低频误差先降，高频误差慢。
- 核心结论 5：FEM 和 `ReLU^p` 浅层网络可以张成同样样条空间，但 FEM 局部基给出稀疏、较可控系统，随机/全局神经基给出密集、病态系统。
- 核心结论 6：增加宽度不必然改善可计算精度，因为新加入的模式往往对应更小特征值，先被数值误差吞掉。
- 核心结论 7：SVD 截断在 PINN/DRM 中优先毁掉高频分量，在 FEM 中优先影响低频分量。
- 核心结论 8：边界正则化不会改变渐近谱衰减，还可能因为 `lambda B^T B` 抬高头部特征值而加重条件数。
- 核心结论 9：DRM 对 Dirichlet 边界正则化尤其敏感，因为未必在正确的边界函数空间中最小化能量。
- 核心结论 10：Neumann 条件在 DRM 的弱形式中自然出现，效果比 Dirichlet 边界正则化更稳定。
- 核心结论 11：非齐次激活的 scaling 可以扩大显著谱段，缓解频率偏置。
- 核心结论 12：scaling 太小会欠覆盖高频；scaling 太大则 over-scaling，随机频率/局部 transition 出现空洞，也会失效。
- 核心结论 13：`sin` 激活的线性设置可解释为随机 Fourier basis，`S` 需要覆盖目标 `k_max` 频率范围。
- 核心结论 14：fully trainable SNN 在 over-scaling 初始条件下有一定 adaptivity，但代价高，且性能未明显超过合适 scaling 的线性随机基。
- 反直觉点：网络越宽、激活越光滑，理论表征空间更大，但数值上可能更病态，实际高频恢复更差。
- 反直觉点：PINN/DRM 的 PDE 微分算子本该放大高频，但浅层网络自身的低频偏置更强，使总效果变成反高频。
- 反直觉点：相同 span 不代表相同可计算性；FEM 和神经基张成同一空间时，局部结构/预条件比表达能力更关键。

### 明确局限
- 论文主体是浅层网络，两层 SNN，不覆盖多层深网的完整行为。
- 主要数学分析在一维 Poisson/椭圆问题和固定第一层的线性化设置中完成。
- 多维真实 PDE、复杂几何、时间依赖问题、非线性 PDE 只作为动机，没有系统实验。
- 数值实验是作者自控仿真实验，没有独立复现报告。
- tex 中没有看到公开代码、完整脚本或 artifact 链接。
- 许多图只报告误差曲线，缺少运行时间、内存、矩阵构造成本的量化表格。
- `lambda` 选择在个别位置存在正文和图注不完全一致的迹象，例如 DRM 边界正则化系数一处写 `50 k_max`，一处写 `100 k_max`。
- 对 fully trainable SNN 的实验只用固定采样 PINN、Adam、一个网络宽度和一个构造解，结论应看作现象展示而非广泛 benchmark。
- 没有与现代 Fourier features、SIREN、FNO、DeepONet、多尺度 PINN、XPINN 等更强工程体系做系统比较。
- 作者承认多层 NN 的问题留作 future work。
- 对我们的关系上，这篇更像负结果/诊断论文，而不是一个可直接复制的性能 SOTA 方法。

### 跟我们的关系
- `0616-fex` 的具体题目只给了 slug；以下按可能涉及 feature/frequency/spectral/PDE solver 的方向评估。
- 如果我们的 claim 是“浅层随机特征/PINN 能高效处理高频 PDE”，这篇会直接撞车并形成反证压力。
- 如果我们的 claim 是“需要 operator-aware、spectral-balance、preconditioning 或结构化 basis 来修复 PINN/DRM”，这篇是强支撑文献。
- 它给出了非常明确的 gap：随机全局基的 dense ill-conditioned Gram matrix 缺少有效预条件器。
- 可复用工具 1：用 Gram/KKT 谱衰减诊断一个神经 PDE solver 是否真的能恢复高频。
- 可复用工具 2：用 relative spectral loss 分别跟踪低频/高频误差，而不只报整体 L2 error。
- 可复用工具 3：把边界处理拆成 hard constraint、regularization、Neumann natural boundary 三类分别测。
- 可复用工具 4：对非齐次激活做 scaling sweep，找 under-scaling 与 over-scaling 的临界区。
- 差异化方向 1：做多层网络或结构化多尺度网络，不再停留在 SNN。
- 差异化方向 2：提出可计算的预条件器，尤其是 operator-aware preconditioner，用来平衡网络表征谱和 PDE 算子谱。
- 差异化方向 3：把 dense random basis 替换为局部/稀疏/可分层基，保留 NN 适应性但获得 FEM 式可求解性。
- 差异化方向 4：在多维、复杂几何、真实 benchmark 和 wall-clock cost 上补上本文没有覆盖的工程证据。

## Read by: 0616-fex (2026-06-17)

本次审读结论：这篇不是提出新 solver，而是给出浅层 NN 解 PDE 的负面上界式诊断。它把 PINN/DRM 在固定第一层时化成可分析的线性系统，证明 `ReLU^p` 全局神经基会产生严重谱衰减和反高频偏置；即使表达空间等价于样条空间，数值可求解性也远差于 FEM 局部基。

对 `0616-fex` 的撞车判断取决于我们的主张。如果我们只是做浅层随机特征或 PINN 的高频 PDE 应用，并声称靠宽度/光滑激活即可解决高频，那会被这篇强烈撞车甚至反驳。如果我们做 frequency/spectral diagnostics、feature scaling、预条件器、结构化 basis、多层网络或高频 PDE benchmark，这篇应当作为核心动机文献引用。

最可用的方法是四个诊断：Gram/KKT 谱衰减、RSL 低高频分解、边界 hard/regularized/natural 的拆分实验、以及 activation scaling sweep。最值得接的 gap 是 operator-aware preconditioning：本文准确指出 dense random basis 没有效预条件器，后续反引文 `2603.00904` 正好沿这条线推进，下一轮搜索应重点围绕 spectral balance、PINN preconditioner、random feature PDE ill-conditioning、operator-aware training 展开。
