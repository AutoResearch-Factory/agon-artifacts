# Causal discovery for time series from multiple datasets with latent contexts (2306.12896)

- **Year**: 2023.06
- **Authors**: Wiebke Günther (第一作者), Urmi Ninad, Jakob Runge — German Aerospace Center (DLR) Institute of Data Science, Jena / TU Berlin
- **Venue**: UAI 2023 (Conference on Uncertainty in Artificial Intelligence)
- **Keywords**: constraint-based causal discovery, time series, PCMCI+, latent confounders, context variables, joint causal inference (JCI), dummy variables, multiple datasets, fixed-effects panel, conditional independence testing
- **One-liner**: 提出 J-PCMCI+，把 PCMCI+ 扩展到"多数据集 + 隐藏语境（时间/空间 context）"场景，用 time/space dummy 变量对系统变量去混杂并保持渐近一致，但小样本下有 Nickell 式有限样本偏差导致 FPR 膨胀。

## 精华提炼

### 管线/方法
J-PCMCI+ 是约束型（constraint-based）时序因果发现算法，把 PCMCI+ 扩展到"同一组系统变量在 M 个数据集上被观测"的场景。核心问题设定（以河流径流为例）：M 个流域（catchment）各有一段时序，共享某些因果父节点（大尺度天气 = **temporal context**，随时间变、跨数据集相同）但又各有数据集特有驱动（海拔/植被 = **spatial context**，跨时间恒定、跨数据集变）。这些 context 往往**部分未观测**，会对系统变量造成隐藏混杂。

完整管线（时序版，主算法 Algorithm 2）：
1. **poolData（数据池化）**：把 M 个数据集沿样本维拼接（不是沿时间聚合，而是沿数据集聚合），temporal context 与系统变量一样保留时间结构；spatial context 每数据集重复其单值；构造 **space dummy** $D_{\rm space}$（标记数据集，one-hot 编码 M 维）和 **time dummy** $D_{\rm time}$（标记时间步，one-hot 编码 T 维）作为潜在 context 的代理。
2. **$\text{PC}_1$ lagged phase**：在系统 + 观测 temporal context 上跑 PCMCI+ 的滞后骨架阶段，得到滞后父节点超集 $\hat{\mathcal{B}}^-_t$。
3. **MCI phase 1（context-system）**：对所有 (context, system) 对做 MCI 条件独立检验，条件集取系统+context 子集。**关键：此步刻意忽略 dummy**，因为每个观测 context 是 dummy 的确定性函数，同时放会违反 faithfulness。由 context 对系统外生（假设 2）直接定向 $C\to X$。
4. **MCI phase 2（dummy-system）**：检验 $D \indep X_t^i$，条件在系统子集 + 上一步找到的 context 父节点。这一步"学隐藏 context"——即数据集标签里不能被观测 context 解释的那部分影响。
5. **MCI phase 3（system-system）**：在已发现的滞后 + context + dummy 邻接集条件下检验系统对之间的独立性，去除被隐藏 context 混杂的伪链接。
6. **Orientation**：collider phase + rule phase（Meek 规则），只定向 system-system 边；但会利用 $C\to X^i \oo X^j$ 或 $D\to X^i \oo X^j$ 这类涉及一个 context/dummy + 两个系统变量的 unshielded triple，从而比只看系统三元组定向更多边。

非时序版（Algorithm 1, J-PC）是同一思想 + 标准 PC 算法，只有 space dummy。两步：先学 context-system（忽略 dummy），再学 dummy-system，最后系统-系统 + 定向。

### 关键技术细节
- **两步/多步 skeleton 的动机**：观测 context 与 dummy 是确定性关系（$C=g(D)$），若把两者平等对待会触发 faithfulness 违背（SM 用 $X\to Y\to Z$、$Y=f(X)$ 的例子说明确定性关系产生 Markov 之外的额外独立性，使 PC 不 sound）。分步 + tiered background knowledge 规避此问题。
- **Dummy = 潜在混杂的占位符，本身不是因果变量**：dummy-projection / dummy-deletion 两个图运算把 dummy 边解释为"对未观测 context→system 边的占位"。目标图（target graph）定义为系统节点 + 观测 context 及其到系统的边的诱导子图；算法输出 $\mathcal{G}_{alg}$ 的 dummy-deletion 等于 target graph。
- **假设**（去混杂的代价）：Assumption 0 联合时序 SCM（无环）；Assumption 1 Markov + Faithfulness + 部分因果充分性（允许隐藏 context $\mathbf{L}$ 但不允许隐藏系统混杂）；Assumption 2 = JCI assumption 1（context 对系统外生）+ 观测 context→系统的链接不被隐藏 context 中介；Assumption 3 = JCI assumption 2（无隐藏 context 同时混杂观测 context 和系统）；Assumption 4 context-determinism（spatial context 是数据集索引的确定性、**非可逆**函数；temporal context 是时间索引的确定性非可逆函数）——改编自 CD-NOD 的 pseudo-causal-sufficiency。
- **一致性结果**：Theorem 1（非时序，$M\to\infty$）与 Theorem 2（时序，$M,T\to\infty$）证明 dummy-deletion 收敛到 target graph；Corollary：即使把所有观测 context 也当未观测，仍能恢复正确的系统-系统邻接。证明含 soundness（context-system / dummy-system / system-system 三类边）+ completeness（依赖 tiered background knowledge 下 PC 的完备性，Andrews 2020）。
- **CI 检验**：默认组件式偏相关（partial correlation）；one-hot dummy + 偏相关 ⇒ 退化为 point-biserial 相关，本质在检验"系统变量均值随 context 变化"，等价于固定效应面板回归的组内中心化。非线性版用 GPDC（高斯过程回归 + 距离相关）。
- **实现**：基于开源 Tigramite（GPL）；代码 https://github.com/guenwi/J-PCMCIplus ，并将并入 tigramite 主包。
- **仿真设定**：线性/非线性 SCM 各一套；系统节点 $|\mathcal{I}|=5$，context 节点 3 个（time+space），$\tau_{\max}=2$，$\alpha=0.05$；50 次实现算 TPR/FPR 均值方差；数据生成后方差重缩放到 1 以避免 varsortability（Reisach 2021）；每系统节点至多一个 context 父；frac_observed 控制多少 context 被观测。

### 核心结果
- **同时用观测 context + dummy 的收益**：只用 dummy → 能去混杂但学不到"哪些系统部分依赖 context 属性"；只用观测 context → 学到 context-system 关系但无法去除隐藏 context 混杂（表现为 system-system FPR 升高）。J-PCMCI+ 两者兼得：context-system 链接的 TPR 与"含 context 的 PCMCI+"相当，system-system 链接性能与"只加 dummy"相当。
- **收敛行为**：固定 M，增大 T → FPR 下降、趋于 $\alpha$ 水平（一致）；**固定小 T、增大 M → 链接无法正确恢复，FPR 不收敛**。
- **（反直觉/关键）有限样本偏差**：这与计量经济学固定效应模型的 **Nickell bias**（Nickell 1981，OLS 斜率估计偏差）同源——dummy 相当于固定效应，小 T 时引入不可消除的偏差。简化 SCM 实验（SM）单独复现了此 FPR pattern：固定 T 时 FPR 随 M 上升，固定 M 时 FPR 随 T 下降。
- **仅用系统数据**（简单拼接）的 PCMCI+ 在 edgemark recall/precision 上明显更差，且 FPR 因隐藏 context 混杂而升高。

### 明确局限
- **仅合成数据**：尽管以河流径流为动机，全文没有跑任何真实水文/地学数据；评估全是 toy SCM。（作者自身的真实数据兑现出现在后续工作，见反引文表。）
- **假设强**：JCI assumption 2 与"观测 context→系统链接不被隐藏 context 中介"在真实系统难以证成；SM 讨论了放松 no-mediation 会使 context-system 链接变成 ancestral 而非 direct，且一致性定理不再成立。
- **有限样本偏差不可消除**：小 T 下 Nickell 型偏差导致 FPR 膨胀，作者明确承认无法克服。
- **维度膨胀**：one-hot dummy 把维度抬到 M（space）/T（time），增加检验负担；未针对 dummy 定制 CI 检验（列为 future work）。
- **无环限制 + 无隐藏系统混杂**：只处理隐藏 context 混杂，不处理隐藏系统变量混杂（future work 提到可结合 LPCMCI/gerhardus2020）。
- **无标准 benchmark、无真机/真数据**：纯方法+理论+仿真论文。
