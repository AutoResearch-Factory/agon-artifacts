# Evaluating Bivariate Causal Statements Based on Mutual Compatibility (2606.00278)

- **Year**: 2026.06 (ICML 2026, camera-ready)
- **Authors**: Erik Jahn (California Institute of Technology & Amazon), Dominik Janzing (Amazon Research Tübingen)
- **Venue**: ICML 2026
- **Keywords**: bivariate causal statements, compatibility score, confounding postulate, linear structural equation models, faithfulness-free falsification, graphical incompatibility, ADMG, transitive closure, LLM causal reasoning, gapminder
- **One-liner**: 不依赖 ground truth，用"多元模型不应比其两两边际更混杂"的合理性 postulate 给一组两两因果陈述打兼容性分，量化版和图结构版各一套。

## 精华提炼

### 管线/方法

1. 问题设定：给定 n 个可观测变量，收集覆盖所有变量对的 $\binom{n}{2}$ 条两两因果陈述，目标是不依赖任何因果 ground truth，仅凭陈述集合内部的相互兼容性给出可信度分数。
2. Track A（线性定量陈述）：每条陈述 = 一个方向 $i\to j$ + 一个线性因果系数 $\alpha_{ij}$。证明只要所有陈述方向与某个全局顺序相容（无环），$\binom{n}{2}$ 条陈述总能唯一扩展为一个满足所有边际系数的多元线性 SEM（$\Gamma = I - A^{-1}$，Lemma unique_sem）——因此不存在 Faller2024 意义上的"硬兼容约束"，必须换一种"软"评价方式。
3. 核心 postulate（"confounding postulate"）：一个合理的多元因果模型，不应该比它自己的两两边际模型表现出更多混杂（未被观测因果路径解释的协方差）。直觉：边际化会删掉大量真实因果路径，这些路径原本能解释部分协方差，一旦删掉，协方差若仍要保持不变，就必须转嫁给"未观测混杂"项，因此边际混杂天然应 ≥ 多元混杂，除非陈述集合本身有问题（路径刚好抵消）。
4. 用 Wright's path tracing rules 把协方差 $\Sigma_{ij}$ 分解为"可观测因果路径贡献"+"未观测噪声相关贡献"两部分，后者的平方即为"混杂量" $\mathcal{C}^{biv}/\mathcal{C}^{mult}$ 的正式定义。
5. compatibility score = $\sum_{i<j}$（双变量边际混杂 − 多元诱导混杂）；score < 0 ⟺ 多元模型比每个边际模型都需要更多混杂才能解释数据 ⟹ 陈述集合不合理（证伪证据）；score > 0 不能反推正确性（"inconclusive"，合理但错误的模型也可能得正分）。
6. Track B（图结构定性陈述）：每条陈述 = 一个二变量 ADMG（有/无有向边+方向、有/无双向混杂边），$\binom{n}{2}$ 条陈述的并集构成 "statement graph" $\cG$。复用 Faller2024 的"图兼容性"定义（存在一个大 ADMG，其所有边际投影恰好等于各条陈述），给出等价的三条纯图论判据：无环、传递闭包、confounding-path 闭包（Lemma characterization）。
7. 与 Faller2024 的关键区别：Faller2024 检验"是否与算法自己推断出的某个大图兼容"；本文不假设存在这样的大图，而是直接在陈述图 $\cG$ 上做隐式优化——找到离 $\cG$ 汉明距离最近的、满足三条件的图 $\cG^*$，距离即 incompatibility score。这一泛化把复杂度从多项式（Faller2024）推到 NP-hard（本文，退化到 Acyclic Transitivity Editing 问题）。
8. 因此设计三阶段贪心启发式（GreedyFAS → GreedyTE → GreedyCPC）分别处理无环化、传递闭包编辑、confounding-path 闭包编辑，证明启发式分数 $c(\cG)$ 是 $\incomp(\cG)$ 的上界，且两者同时为零（Lemma heuristics）——精确值不可行，但"是否完全兼容"和"大致多不兼容"仍可信估计。
9. 双轨都先在合成数据（随机线性/图模型 + 注入误差）上验证，再在真实 LLM 输出（10 个模型，经 Amazon Bedrock 调用，gapminder 7 变量数据集）上做实际部署级实验。

### 关键技术细节

- 线性 SEM 设定：$\bX = \Gamma \bX + \bN$，$\Gamma$ 严格下三角（无环），噪声 $\bN$ 允许任意相关（代表未观测混杂），模型在边际化下封闭（Lemma marginalization，边际系数由 Schur complement 给出）。
- compatibility score 计算复杂度：给定协方差矩阵和 $A$，只需一次矩阵求逆 $\Gamma=I-A^{-1}$ 和路径追踪求和，多项式时间可精确计算（区别于图版本的 NP-hard）。
- 理论保证 1（Theorem thm:main）：在"无偏性 + 因果机制独立性 + 非退化性"三条自然假设下（不限定任何具体噪声分布之外的高斯），任意随机线性高斯 SEM 生成的真实陈述集合，其 compatibility score 期望严格 $>0$——为 confounding postulate 提供先验合法性而非仅凭直觉。证明把协方差分解为"可观测回路贡献 $B_{ij}$"+"未观测噪声贡献 $\eps_{ij}$"，用独立性假设证明两者不相关，只剩 $B_{ij}^2\geq0$ 的贡献。
- 理论保证 2（Theorem thm:sensitivity）：有限样本估计误差界 $N = O(n^4(1+a+b)^4V^4/\epsilon^2 \log(n/\delta))$（$a,b$ 为陈述矩阵蕴含的最大直接效应/最大回路贡献），证明基于标准高斯协方差集中不等式。
- 图版本 NP-hardness 归约：计算 $\incomp(\cG)$ 包含 Acyclic Transitivity Editing 问题（全加双向边的特殊情形下等价），该问题已知 NP-hard。
- 三阶段启发式细节：Step1 GreedyFAS（线性时间近似最小反馈弧集）；Step2 GreedyTE（贪心删边最小化到传递闭包的差异，再补齐缺失传递边）；Step3 GreedyCPC（同一贪心模板应用于混杂路径闭包，只操作双向边，固定 Step1-2 得到的有向部分）。
- 数据集：gapminder 7 个国家级发展指标（人口密度、识字率、日均收入、卫生设施可及性、吸烟率、幸福指数、预期寿命），179 国 1950-2025 年跨度，CC-BY 许可。
- LLM 实验：10 个模型经 Amazon Bedrock 统一调用（Claude Opus 4.5、gpt-oss-120b/20b、Gemma 3 27B/4B、Kimi K2 Thinking、Mistral Large 3、Magistral Small 2509、Qwen3 235B A22B/Next-80B A3B），温度 0.6，top-p 0.7，最多 2048 输出 token；每模型独立跑 15 次（线性）/10 次（图结构）取平均；随机基线用系数方差匹配各模型实际输出方差的高斯采样。
- Prompt 设计：先让模型给出 7 变量的因果顺序（root causes → downstream effects），再按此顺序逐对询问标准化总因果效应系数（线性版），或"是否存在因果效应+方向+是否存在混杂"（图版，用随机顺序呈现变量对以同时检验无环一致性）；格式错误时最多重试 5 次。
- 开源代码：github.com/ejahn17/compatibility-scores（含全部合成实验与 LLM 实验复现脚本）。

### 核心结果

- 合成线性实验：随着注入到真实系数上的高斯噪声方差增大，"正 compatibility score 陈述集合"的比例在所有测试的 $(n,m,p)$ 参数组合下都单调下降，说明分数能区分正确/错误陈述；真实陈述集合本身绝大多数得正分，印证 confounding postulate 在合成设定下经验成立。
- 有限样本收敛：少于 100 个样本即可让经验分数与真值分数的符号在 >90% 测试案例中保持一致，远快于 Theorem thm:sensitivity 给出的最坏情形界。
- 合成图实验：注入错误数量增加 → incompatibility score 平均单调上升；小/稀疏图上分数与真实错误数吻合较好，大/密集图上分数系统性高估真实错误数（启发式近似的已知代价）。
- LLM 线性实验（gapminder 7 变量）：更高能力模型倾向获得更高 compatibility score；随机基线本身得正分，但相当一部分被测 LLM 仍得负分（即被证伪）——分数能揭示不同模型间的系统性质量差异。
- LLM 图结构实验：所有派生 statement graph 中只有一个达到 $\incomp=0$（且是完全图，无信息量）；许多低分模型输出极密集图（钻了图兼容性判据的空子）；用边密度 ≤2/3 过滤后重新比较，仍观察到低不兼容分数与模型能力正相关；密度过滤子集中最低分为 4，由 Claude Opus 4.6 一次输出取得（11 有向边 + 14 双向边）。
- 图分数分解（附录）：唯一在双向边一致性上零违规的模型 Gemma 3 27B IT 恰恰是因为它输出的图接近完全图（双向边判据在完全图上平凡满足）——提醒"低分不等于高质量"，必须同时看信息量/密度。

### 明确局限

- 作者自陈（Discussion + Impact Statement）：正 compatibility score 不能证明陈述正确，只能说"通过了合理性检验"；只有负分才是真正的证伪证据。
- 线性 compatibility score 方法论仅覆盖"完整" $\binom{n}{2}$ 陈述列表（覆盖所有变量对且方向一致于某全局顺序）；作者明确列为未来工作：陈述不完整、或只对变量子集给出陈述的推广，图版本相对自然，但线性版本"唯一诱导多元模型"的存在性证明会失效，需要全新思路。
- 图 incompatibility score 精确计算 NP-hard，启发式算法只有单向近似保证（上界 + 零点吻合），在稠密/大规模图上会系统性高估真实错误数，没有下界或紧致近似比保证。
- 图版本方法本质依赖 faithfulness 假设（用于保证边际化图与推断图一致，附录明确承认），而线性版本的核心卖点恰恰是"不需要 faithfulness"——两条路线假设强度不对等，论文未给出图版本去掉 faithfulness 假设的方案。
- 密集图能靠"信息量很低但技术上兼容"刷高分，论文只用事后的密度过滤缓解，没有把陈述的可证伪性/信息量直接整合进分数定义（脚注承认这是待解决问题，引用 Popper 1959）。
- 全文未涉及本课题需要的时间/动态维度：所有实验都是对固定变量集合的单一时间快照做陈述评估，没有滑动窗口、非平稳性或跨时间段陈述比较——这是与前作 Faller2024 (2307.09552) 共享的局限，本文没有在这个方向上推进。
- 真实数据实验里 LLM 输出质量的"因变量"没有独立于 compatibility score 之外的 ground truth 校验；论文承认这正是"缺乏替代验证手段"场景下的核心应用，但也意味着分数本身有效性的验证，最终仍依赖合成数据里"已知真实系数"的间接类比。

### 跟我们的关系

这是 Faller2024 "Self-Compatibility" ([[2307.09552]]) 的直接续作，同作者 Dominik Janzing（新增合作者 Erik Jahn）——[[2307.09552]] 的 wiki 曾明确将本文列为"最高优先级后续精读候选"，专门用来确认该系作者是否已把 self-compatibility 思路扩展到时间/动态场景。**现已确认：没有。** 本文的兼容性检验轴是"变量对"（$\binom{n}{2}$ 条覆盖固定变量集合的两两陈述），不是时间窗口；全部实验都是对固定变量集合的单一时间快照做陈述评估，无滑动窗口、无非平稳性讨论、无跨时间段陈述比较。这意味着 [[2307.09552]] 遗留的"最大未知风险点"已解除——因果-scs 课题（因果效应不稳定性作为强对流预警指标）在时间/动态轴上的定位依然未被这条方法论谱系触及，不构成 claim 级撞车。

具体差异化与可复用点：

1. **变化轴对照**：Faller2024 的兼容性轴是"变量子集"（同一因果发现算法在不同变量子集上的输出是否自洽），本文的轴是"变量对"（一组两两陈述是否共同诱导一个合理的多元模型），因果-scs 课题的轴是"时间窗口"。三者共享同一元方法论直觉——用内部不一致性作为证伪信号而非纯噪声——但轴各不相同，可以放心引用整条谱系作为方法论合法性背书。
2. **新工具（confounding postulate + compatibility score）**：不需要重新对变量子集跑因果发现算法，只需一份协方差矩阵 + 一组两两因果系数陈述即可计算，且线性版本明确不依赖 faithfulness——如果因果-scs 课题未来需要对"某组 SCS 相关变量的两两因果强度声称"（无论来自专家还是 LLM）做静态合理性快检，这是一个比 Faller2024 的 $\kappa^I/\kappa^G$ 更轻量的候选工具，但它检验的是"跨变量对"的一致性，不是"跨时间"的稳定性，直接套用前需要先做时间轴的方法论改造（例如：把"多元模型 vs 两两边际"的比较，替换成"宽时间窗模型 vs 窄时间窗模型"的比较）。
3. **图 incompatibility score 的潜在改造方向**：三阶段启发式（GreedyFAS/GreedyTE/GreedyCPC）检验的是"一组两两图陈述能否被无环化+传递闭包+混杂路径闭包同时满足"，如果因果-scs 课题把"不同时间窗口给出的因果图陈述"当作输入（类比本文的"不同变量对给出的因果图陈述"），理论上可以借用同一套 Hamming 距离 + 贪心近似框架来检验"跨窗口因果图是否自洽"——但这属于我们需要自己做的新扩展，本文没有做。
4. **LLM 因果陈述引出协议可直接复用**：本文附录给出的两套 prompt 模板（先诱导因果顺序，再逐对询问系数/存在性+方向+混杂）、格式重试策略、15 次重复取平均、随机基线构造方式，都是若因果-scs 课题需要用 LLM 产出关于 SCS 前兆变量的因果陈述时的现成模板，可直接借鉴而非重新设计。
5. **同一作者群后续工作仍需追踪**：本文引用列表中出现的 2602.07943 (IV Co-Scientist, 2026) 是另一支"LLM + 因果发现"新工作，值得留意是否涉及时间维度；本文本身未提及任何气象/强对流应用场景，与因果-scs 课题在应用领域上完全不重叠，无需担心应用层面撞车。

本文是 Faller2024 "Self-Compatibility" ([[2307.09552]]) 的直接续作（同作者 Janzing + 新合作者 Jahn），把"兼容性"打分的思路从"因果发现算法在不同变量子集上的输出是否自洽"，扩展到"一组两两因果陈述（尤其是 LLM 产出的）是否能共同构成一个合理的多元因果模型"。核心方法论贡献是两套独立打分：(1) 线性定量版——基于"confounding postulate"（合理的多元模型不应比其两两边际更混杂）的 compatibility score，不依赖 faithfulness，有完整的期望正性定理和有限样本收敛界；(2) 图结构定性版——基于无环+传递闭包+混杂路径闭包三条一致性判据的 incompatibility score，精确计算 NP-hard，用三阶段贪心启发式近似。两套分数都在合成数据和真实 LLM 输出（10 个模型，gapminder 7 变量数据集）上做了系统实验，发现更高能力模型倾向获得更高兼容性分数，但相当一部分 LLM 仍会被负分证伪。
