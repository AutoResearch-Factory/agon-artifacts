# Towards Principled, Practical Policy Gradient for Bandits and Tabular MDPs (2405.13136)

- **Year**: 2024.05
- **Authors**: Michael Lu, Matin Aghaei (Simon Fraser University), Anant Raj (Inria), Sharan Vaswani (Simon Fraser University)
- **Venue**: arXiv (2024-05-21)
- **Keywords**: softmax policy gradient, Armijo line-search, exponentially decreasing step-size, non-uniform smoothness, Łojasiewicz condition, strong growth condition, entropy regularization, bandits, tabular MDP
- **One-liner**: 将优化文献中的实用技术（Armijo line-search、指数衰减步长）引入 softmax 策略梯度，消除对 oracle-like 知识（optimal action、reward gap、full gradient）的依赖，同时保持理论收敛保证。

## 精华提炼

### 管线/方法

**1. Exact Setting（精确梯度）**
- 输入：已知 reward 和 transition 的 tabular MDP/bandit
- PG 更新：θ_{t+1} = θ_t + η_t ∇f(θ_t)（softmax 参数化）
- **Armijo line-search**：每步从 η_max 开始回溯，直到满足 f(θ+η∇f) ≥ f(θ) + h·η·‖∇f‖²（h ∈ (0,1) 超参），保证单调上升且步长 ≥ min{2(1-h)/L, η_max}
- **Log-loss Armijo**：对 ln(f* - f(θ)) 做 line-search，利用 non-uniform smoothness 获得线性收敛，但需知 f*

**2. Stochastic Setting（随机梯度）**
- 输入：未知环境，通过 on-policy importance sampling 估计梯度
- 梯度估计：a_t ∼ π_t，构造 \hat{r}_t(a) = 𝕀{a_t=a}·R_t/π_t(a)，形成无偏有界方差随机梯度
- **指数衰减步长**：η_t = η_0·α^t，α = (β/T)^(1/T)，β ≥ 1 为超参（不依赖任何 problem-dependent 常数）
- "Doubling trick" 变体：小时间窗口开始，每次翻倍重启步长 schedule

**3. Entropy Regularization 多阶段算法**
- 在目标上加 τ·H(π)，使 Łojasiewicz condition 从 ξ=0 变 ξ=1/2（更快收敛）
- 逐阶段减半 τ：τ_{i+1} = τ_i/2，每阶段用固定温度跑梯度上升
- 在 exact setting 得 O(1/ε^p) 率，stochastic setting 得 Õ(1/ε^p + σ²/ε^{2p+1}) 率

### 关键技术细节

- **Uniform smoothness L**：bandits = 5/2，MDP = 8/(1-γ)³
- **Non-uniform smoothness L₁**：bandits = 3，MDP = [3 + (2C_∞-(1-γ))/((1-γ)γ)]·√S
- **Non-uniform Łojasiewicz with ξ=0**：C(θ) = π_θ(a*)（bandits），C(θ) = min_s π(a*(s)|s)/(√S·‖d^π*/d^π‖_∞)（MDP）
- **Reversed Łojasiewicz ν**：bandits = √2/Δ*，MDP = √2/((1-γ)Δ*)
- **SGC constant**：bandits ϱ = 8A^{3/2}/Δ²，MDP ϱ = 4A^{3/2}S^{1/2}/((1-γ)⁴Δ²)，其中Δ为reward gap
- **Variance bound**：bandits σ² ≤ 2，MDP σ² ≤ 2S/(1-γ)⁴
- **Entropy regularized smoothness**：L^τ = 5/2 + 5τ(1+log A)（bandits），L^τ = (8+τ(4+8log A))/(1-γ)³（MDP）
- **Entropy regularized Łojasiewicz**：ξ=1/2, C_τ(θ) = √(2τ)·min_a π(a)（bandits）
- 实验环境：Cliff World (S=21, A=4)，Deep Sea Treasure (S=25, A=2)，Flat Grad (S=22, A=4)；MDP中γ=0.9-0.957
- Bandit实验：A=10，Bernoulli/Gaussian/Beta reward，easy Δ̄=0.5，hard Δ̄=0.1，10⁶ iterations

### 核心结果

**Exact setting:**
- PG-LS（Armijo）：O(1/T) 收敛，比 naive constant step-size (η=1/L) 快得多
- PG-Log-LS：线性收敛 O(log(1/ε))，match GNPG/ PG-A 但不需 concentrability coefficient
- PG-LS 在实验中与 GNPG 和 PG-A 性能相当，PG-Log-LS 更快

**Stochastic setting:**
- SPG-ESS（指数衰减步长）：匹配 Yuan et al. 的 Õ(1/ε³) 率，但不需知道 π*
- SGC 下可插值：η_0 设置大 → "慢" Õ(1/ε³)；η_0 小/T_0=0 → "快" Õ(1/ε)
- 实验结果：SPG-ESS 和 SPG-ESS[D]（doubling trick）与 SPG-O-G（需 full gradient）和 SPG-O-R（需 reward gap）性能相当
- Surprising：SPG-O-R 尽管理论最快 O(1/ε)，但 step-size ∝ Δ²，gap 小则步长极小，实际表现反而差

**Entropy regularization:**
- 多阶段算法 converge 到无偏最优策略（非 biased policy）
- Bad initialization 下熵正则化显著帮助 escape flat regions
- 当 p=1 时，exact 得 O(1/ε) 率，stochastic 得 Õ(σ²/ε³) 率

**Surprising/反直觉发现：**
- Armijo line-search 在 PG 中总能保证 μ>0（因单调改进），这是与 constant step-size 证明的关键区别
- 指数衰减步长不依赖 ϱ（SGC constant），却能自动适应并插值快慢率——"robust to unknown problem-dependent constants"
- Reward gap 对 SGC constant 的依赖被证明是必要的（Proposition 1）
- Entropy 在 exact setting 下理论收敛率其实不优于无 entropy 方法（同样是 O(1/ε)），好处只在 bad initialization

### 明确局限

**作者自述：**
- Log-loss Armijo 需要知道 f*（类似 Polyak step-size），不够实用
- Doubling trick 缺乏理论收敛分析
- Entropy regularization 的 μ := inf [C_τ(θ)]² = τ^p B₁ 假设（Assumption 1）是 benevolent assumption，C_τ ∝ min_a π(a) 可能过于悲观
- τ→0 时从 ξ=1/2 平滑过渡到 ξ=0 的机制未研究
- 仅限 tabular parameterization，未扩展到 function approximation
- SPG-ESS 不保证渐近收敛到最优 arm（因递减步长 → 0），不同于 constant step-size 的 Mei et al. 2023

**审稿人视角的局限：**
- 无真机实验，全是仿真环境和 toy bandit/MDP
- 未与标准 bandit 算法（UCB, Thompson Sampling）实验对比
- 所有理论保证基于 tabular softmax parameterization，扩展到神经网络策略仍遥远
- p 值需要 grid-search（实验中 p=1, B₁=0.01 是搜出来的）
- 无标准 benchmark 对比（如 Atari, MuJoCo）
