# Why Policy Gradient Algorithms Work for Undiscounted Total-Reward MDPs (2510.18340)

- **Year**: 2025.10 (ICML 2026 submission)
- **Authors**: Jongmin Lee (Seoul National University, Dept. of Mathematical Sciences), Ernest K. Ryu (UCLA, Dept. of Mathematics)
- **Venue**: ICML 2026
- **Keywords**: policy gradient, undiscounted MDP, total-reward, recurrent-transient theory, transient visitation measure, natural policy gradient, softmax parameterization, convergence rate, sample complexity, generative model
- **One-liner**: 建立 undiscounted total-reward infinite-horizon MDP 中 policy gradient 的首个收敛理论——用 recurrent-transient 状态分类不变性和 transient visitation measure 替代传统折扣态分布，证明 projected PG 和 natural PG 的 sublinear/linear 收敛率及样本复杂度。

## 精华提炼

### 管线/方法

**问题设定 → 病理诊断 → 理论工具构建 → 算法分析 → 实验结果**：

1. **Input**: undiscounted total-reward infinite-horizon MDP `(S, A, P, r, μ)`，满足 Assumption 1（V^π 有限）
2. **病理学诊断 (Section 3)**：识别两个在 undiscounted 设定下独有的问题：
   - Pathology 1: π → V^π 可能在 Π 边界不连续
   - Pathology 2: Q* 不足以唯一确定 π*（argmax Q* 可能非最优）
3. **Recurrent-Transient 理论工具 (Section 4)**：
   - Proposition 1: 对所有 π ∈ Π+（严格正概率策略），recurrent-transient 状态分类不变
   - Lemma 2: 任何 recurrent state 的 reward 必为 0（否则 V^π 无限）
   - 定义 transient matrix T^π（只保留 transient 间转移）→ 谱半径 < 1
   - Lemma 3: V^π = (I-T^π)⁻¹ r^π（紧凑解析形式）
   - 定义 transient visitation measure δ^π_{s_0}(s) = e_{s_0}^T (I-T^π)⁻¹ e_s
   - Transient Performance Difference Lemma
   - Theorem 1 (Transient Policy Gradient): ∇_θ V^π_μ = Σ_s δ_μ^π(s) Σ_a ∇_θ π_θ(a|s) Q^π(s,a)
4. **算法分析**：
   - Projected PG (direct parameterization) + Π_α（α-shrunken simplex）→ O(1/k) sublinear 收敛
   - Natural PG (softmax parameterization) → 常数步长 O(1/k) sublinear 收敛；自适应步长线性收敛
   - Stochastic NPG (generative model) → Q-estimator sample complexity Õ(t³R²|S||A|/ε²)；整体 Õ(1/ε⁵) constant step，Õ(1/ε²) adaptive step
5. **Output**: ε-optimal policy 的迭代复杂度和样本复杂度上界

### 关键技术细节

**模型架构与假设**：
- 有限状态空间 |S|，有限动作空间 |A|
- 有界奖励 r: S×A → [-R, R]
- **核心假设 (Assumption 1)**：对所有 π 和 s，V_+^π(s) < ∞ 且 V_-^π(s) < ∞（价值函数有限）
  - 在 LLM RLHF/RLVR 场景下自然满足（有限奖励仅在轨迹结束时发放）
- Π_+ = {π | π(a|s) > 0 对所有 s,a}：严格正概率策略集，是 Π 的相对内部
- 非负奖励时 V*_{+,μ} = V*_μ（搜索 Π_+ 足够）

**核心定理与证明技术**：
- **Theorem 1 (Transient Policy Gradient)**：关键证明步骤：(a) 证明在 Π_+ 上 T^π 连续且可微（recurrent-transient 分类不变）；(b) 利用矩阵微积分 ∂A⁻¹/∂θ = A⁻¹(∂A/∂θ)A⁻¹；(c) 对 recurrent states，r(s,a)=0 保证梯度只涉及 transient states
- **Lemma 4 (Smoothness)**：‖∇V^π_μ - ∇V^π'_μ‖₂ ≤ 2RC²_α(C_α+1)|A| ‖π-π'‖₂，其中 C_α = max_{π∈Π_α} ‖(I-T^π)⁻¹‖∞
- **Gradient Domination (Lemma 5)**：V^{π*_α}_μ - V^π_μ ≤ ‖δ^{π*}_μ / μ‖_∞ max_{π̄∈Π_α} (π̄-π)^T ∇_π V^π_μ

**NPG 更新规则**：
- Softmax 参数化后，NPG 等价于 multiplicative weights update：
  π_{k+1}(a|s) ∝ π_k(a|s) exp(η_k Q^{π_k}(s,a))
- 该更新不依赖初始分布 μ（Fisher information 伪逆消除了 μ 依赖）
- 与 mirror descent / Bregman divergence 框架一致

**Transient half-life**：t_{1/2,π} = argmin_{t≥1} ‖(T^π)^t‖_∞ ≤ 1/2，用于 Q-estimator 的 horizon cutoff

**实验环境**：
- FrozenLake 4×4（4 actions，随机转移），CliffWalk 3×7（4 actions，确定性转移）
- Pathological MDP（图1：5 状态，验证 discontinuity 理论）
- 非负奖励（确保 V* = V*_+）
- PPG: α ∈ {0.1, 0.05, 0.01}，NPG: constant + adaptive step sizes

### 核心结果

| 算法 | 收敛率 | 关键条件 | 依赖维度 |
|------|--------|---------|---------|
| Projected PG (direct) | O(1/k) sublinear | α-shrunken Π_α | |S|·|A| |
| Natural PG (constant η) | O(1/k) sublinear | π_0 ∈ Π_+, full support μ | 与 |S|,|A| 无关 |
| Natural PG (adaptive η) | (1-1/ϑ)^k 线性 | 需知道 ϑ^π_μ | 与 |S|,|A| 无关 |
| Stochastic NPG (constant η) | Õ(1/ε⁵) sample | generative model | |S|·|A| |
| Stochastic NPG (adaptive η) | Õ(1/ε²) sample | generative model | |S|·|A| |

**关键发现**：
1. **NPG 收敛率与状态/动作空间大小无关**（vs. PPG 的 O(|S||A|)）
2. **自适应步长 NPG 达到线性收敛**（利用分布 mismatch ϑ^π_μ 信息）
3. **非负奖励下 V*_{+} = V***（搜索 Π_+ 即可找到全局最优）
4. **Pathological MDP 实验**：验证了 discontinuity 理论——V*_μ - V^{π_k}_μ 严格为正且不趋零

### 明确局限

**作者自己说的局限**：
1. Projected PG 只能收敛到 V^{π*_α}_μ 而非 V*_μ（需 α→0 且非负奖励才能 ε-optimal）
2. Projected PG 收敛率依赖 |S||A| 维度，且 sublinear
3. 自适应 NPG 需要知道 ϑ^π_μ（分布 mismatch 系数）来设置步长
4. 所有分析限于 tabular setting（直接参数化或 softmax），未扩展到函数逼近
5. 仅在 toy environments (FrozenLake, CliffWalk) 上验证，无大规模实验

**我们读出来的局限**：
6. **未讨论 softmax 以外的参数化**：FEX controller 使用的可能是非标准 softmax 变体
7. **假设 generative model 可采样任意 (s,a)**：实际 RL 中不现实，且与 FEX 的 sequential tree generation 不同
8. **transient half-life t_{1/2,π} 是 π 依赖的**：在最坏情况下可能随问题规模爆炸
9. **C_α 在 α→0 时可能发散**：例如在 pathological MDP 中，C_α → ∞ as α → 0，导致 PPG 的复杂度常数爆炸
10. **未考虑 partial observability 或 non-stationary transition**：FEX 的表达式树搜索本质上是 non-stationary（每个 token 生成改变状态）
11. **只分析了 expected total-reward，未考虑 risk-sensitive 或 quantile-based reward**：FEX 常使用 top-quantile reward

## Read by: 0616-fex (2026-06-21)

本次审读视角：评估该论文对 FEX search complexity 课题（0616-fex）的 relevance，特别关注其与 claim (iii) conditional PG theorem 的关系。

**不撞车**。本文是 undiscounted total-reward MDP 中 PG convergence 的理论工作，不涉及 FEX/PDE 表达式搜索、quotient counting 或 ∃R-hardness。核心 novelty gap 保持完整。

**核心可用资产**：
1. **Transient visitation measure**：这是 undiscounted setting 中替代 discounted state visitation 的数学构造。对 FEX conditional PG theorem 的直接启示：FEX 的表达式树搜索也在 undiscounted setting（每 episode 生成一个完整表达式），需要类似的 "visit probability replacement" 概念来定义 policy gradient。transient visitation measure 通过 (I-T^π)⁻¹ 解析表达——FEX 可能需要定义 "tree-level visitation measure" 作为类比。

2. **Gradient domination + smoothness → convergence 的证明管线**：Lemma 5 (gradient domination) + Lemma 4 (smoothness) + Fact (Xiao 2022 的 composite optimization) → Theorem 2 (O(1/k) convergence)。这是 conditional PG theorem 最直接的证明模板。FEX 需要 (a) 建立自己的 gradient domination 条件（κ_PG）；(b) 证明 smoothness 常数（依赖 FEX tree grammar 和 PDE residual 结构）；(c) 应用到适当的 optimization framework。

3. **NPG multiplicative weights update 与 softmax controller 的对应**：FEX 使用 softmax controller → 自然对应 NPG 的 multiplicative weights 更新。本文 NPG 的收敛分析（特别是与 |S|,|A| 无关的性质）可直接移植到 FEX NPG controller 的分析。

4. **Recurrent-transient 分类不变性（Proposition 1）**：Π_+ 上状态分类不变的论证方式可类比 FEX 的 quotient cover——在某些等价类划分下，表达式树的结构分类也不变（quotient classes are invariant under sound_ac rewriting）。这提供了 claim (i) 和 claim (iii) 之间的一个理论桥接点。

5. **Transient performance difference lemma**：无需 discounted 因子的 performance difference 公式——FEX 的 undiscounted reward（PDE residual minimization）可直接使用该公式形式。

**差异化与需注意的 gap**：
- FEX 的 action space 是 sequential tree construction（variable-length action sequence），不是 tabular MDP 的固定动作集——本文的全部分析假设固定 tabular structure
- FEX 的 reward 是 PDE residual（内层 Adam/BFGS 优化），不是 bounded immediate reward——smoothness 常数和 gradient domination 条件需要针对 PDE residual 重新推导
- 本文的 transient half-life t_{1/2,π} 和 C_π 定义依赖 MDP transition matrix——FEX 需要定义 tree-grammar 的 "transition operator" 才能类比
- 本文没有处理 quotient/equivalence class 的概念——FEX 的 quotient cover 先于 PG convergence 的结构压缩是独特的贡献
