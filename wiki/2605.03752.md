# Vanishing L2 Regularization for the Softmax Multi Armed Bandit (2605.03752)

- **Year**: 2026.05
- **Authors**: Ștefana-Lucia Anița (CEREMADE, Université Paris Dauphine - PSL), Gabriel Turinici (Octav Mayer Institute of Mathematics, Romanian Academy)
- **Venue**: ICML 2026 (submitted, uses icml2026.sty with [preprint] option)
- **Keywords**: Multi-Armed Bandit, softmax policy gradient, L2 regularization, vanishing regularization, stochastic gradient descent, convergence analysis, regret bounds
- **One-liner**: 首次分析 L2 正则化参数趋于零时 softmax 策略梯度 MAB 的收敛性，证明梯度/regret 收敛到零，且 L2 正则化在数值上优于非正则化和熵正则化。

## 精华提炼

### 管线/方法

1. **问题设定**：k 臂 MAB，每臂 a 有奖励分布 R(a)，均值 q_*(a)。目标最大化平均奖励。
2. **参数化**：偏好向量 H ∈ ℝ^k 通过 softmax 映射 Π_H(a) = e^{H(a)} / Σ_b e^{H(b)} 生成选择分布。
3. **L2 正则化目标**：L_γ(H) = E_{A~Π_H}[R(A)] - (γ/2)||H||^2，其中 γ ≥ 0 为正则化系数。
4. **更新规则（随机梯度上升）**：H_{t+1}(a) = H_t(a) + ρ_t [(R_t - R̄_t)(1_{a=A_t} - Π_{H_t}(a)) - γ_t H_t(a)]
   - R̄_t 是基线奖励（历史平均），引入不引入偏差
   - ρ_t 是学习率，γ_t 是时变正则化系数
5. **核心创新**：允许 γ_t ↘ 0（正则化消失），这在之前工作中未被分析——前人只分析 γ 足够大使得目标函数凸的情况。
6. **新颖假设 Hyp-ργ**：∃ c_{ργ} > 0 s.t. c_{ργ} ρ_t γ_t^2 ≤ γ_t - γ_{t+1}，连接学习率和正则化衰减速度。此假设在文献中从未被使用。

### 关键技术细节

- **奖励分布假设**：二阶矩有界 E[R(a)^2] ≤ C_m（标准）；三阶矩有限 E[|R(a)|^3] < ∞（Hyp-3，部分定理需要）
- **学习率假设 (Hyp-ρ)**：Σ ρ_t = ∞ 且 Σ ρ_t^2 < ∞（标准 Robbins-Monro 条件）
- **正则化假设 (Hyp-γ)**：γ_t 递减，γ_t → 0
- **梯度无偏性**：E[g_t | F_t] = ∇_H L_{γ_t}(H_t)，其中 g_t = u_t - γ_t H_t，u_t 是 REINFORCE 风格的无偏梯度估计
- **梯度有界性**：E[||u_t||^2] ≤ c_{q_*}^2，E[||g_t||^2] ≤ C_g（Lemma 1）
- **关键不等式**：||∇_H L_0(H_t)||^2 ≥ (Δ²/(8(k-1))) · ||Π_{H_t} - D||^2（将梯度范数与到 Dirac 集合的距离关联）
- **regret 定义**：R(Π) = max_a q_*(a) - ⟨q_*, Π⟩
- **代码公开**：https://github.com/gabriel-turinici/regularized_policy_gradient（2025年12月20日版本）
- **实验设置**：
  - M=1000 次独立运行，每次 T=2000 步
  - k=10 臂（默认），扩展到 k=3, k=50
  - q_*(a) ~ N(4, 1)，R(a)|a ~ N(q_*(a), 1)（默认 Gaussian）
  - 偏置初始化 H_0 = (5, 0, ..., 0)
  - 也测试了 Student-t 分布（ν=2.5, ν=1.5）

### 核心结果

**理论（Theorem 1, 时变 ρ_t）**：

| 条件 | 结论 |
|------|------|
| Hyp-3（三阶矩） | ∇_H L_{γ_t}(H_t) → 0 in L² |
| Hyp-ργ（无 Hyp-3） | ∇_H L_{γ_t}(H_t) → 0, ∇_H L_0(H_t) → 0, γ_t H_t → 0 in L², Σ ρ_t E[||∇L_0(H_t)||^2] < ∞ |
| Hyp-ργ + 唯一最优臂 | Π_{H_t} → D（Dirac 集合）in L² |
| Hyp-ργ + 唯一最优臂 + inf Π_{H_t}(a*) > 0 | regret → 0, Π_{H_t} → δ_{a*} in L² |

**理论（Theorem 2, 常数 ρ_t=ρ）**：
- (1/t) Σ E[||∇L_{γ_s}(H_s)||^2] = O(ρ)
- R(Π̄_t) = O(√ρ)，其中 Π̄_t = (1/t) Σ Π_{H_s}
- ||Π̄_t - δ_{a*}|| = O(√ρ)

**数值（核心发现）**：
1. L2 正则化（线性衰减 γ_t = γ_0/(1+0.2t)）在所有场景下 regret 最低，优于无正则化和熵正则化。
2. **反直觉发现**：当 ρ_t 取非最优值时（如 ρ=1 而非最优的 0.1），L2 正则化能"修复"收敛，使非最优学习率表现接近最优情况。无正则化和熵正则化则停滞在高 regret。
3. 对 Student-t 重尾分布（ν=2.5, ν=1.5），L2 正则化仍然有效，而无正则化和熵正则化的 regret 停滞在~0.5。
4. 鲁棒性验证：k=3,10,50 臂均验证；L2 在所有情况下最优。
5. 线性衰减 γ_t = γ_0/(1+c₁t) 在测试的几种衰减方案中表现最好（对比对数衰减和平方根衰减）。

### 明确局限

1. **仅限 MAB 框架**：没有推广到一般 MDP。作者明确说"在此工作中我们聚焦 MAB"。
2. **需要 Hyp-ργ 假设**：虽然作者声称这是新颖且有趣的 regime，但这是一个非标准的额外假设。当 γ_t 衰减不够快时可能不成立。
3. **常数 ρ 情况下的收敛仅在平均分布**：Π̄_t 而非 Π_{H_t} 本身。虽然实验显示 Π_{H_t} 在实践中表现良好，但理论上没有证明。
4. **需要 inf Π_{H_t}(a*) > 0 才能证明 regret → 0**：作者承认这是强假设，但在文献中即使无正则化也需要类似假设（除非假设有界支撑）。
5. **没有与 UCB/Thompson Sampling 等算法比较**：作者明确说这是有意为之——假设用户已经选了 softmax PG 类算法。在附录中给了 UCB 比较（UCB 在高臂数/重尾时表现差），但正文未纳入。
6. **仅在仿真上验证**：无真实应用场景测试。
7. **理论中三阶矩假设 (Hyp-3)**：虽然数值上验证了 ν=1.5 的 Student-t（无有限方差），但理论上仍需要三阶矩来证明部分结果（没有 Hyp-ργ 时）。
8. **没有考虑 adversarial bandit 设置**：仅在 stochastic MAB 下分析。
9. **L2 正则化的超参数选择**：需要调 γ_0 和衰减速率 c₁，文中未给出自动选择方法。

## Read by: 0616-fex (2026-06-21)

### 撞车评估
**不撞车**。本文是 MAB 中 L2 正则化 PG 的渐近收敛理论（t→∞），核心贡献在数学证明。`0616-fex` 课题关注的是 search 的有限样本/计算复杂度，两者在问题设定、分析框架和结论类型上完全不同。

### 可用工具
1. **梯度有界性分析框架**（Lemma 1）：乘 γ_t 缩放后取期望的迭代不等式技术，可迁移到其他递减正则化场景。
2. **Hyp-ργ 假设及其分析的 regime**：连接学习率和正则化衰减速率的新颖假设，若我们的工作涉及此类耦合，可参考。
3. **梯度范数到分布收敛的不等式 (31)**：将 PG 的梯度大小与策略分布到最优解的距离关联，是有限样本分析的潜在工具。
4. **summability 论证模式**（Proposition 1 + Lemma 2/3）：通过级数收敛性证明梯度消失的标准技术路径。

### 差异化
我们的 focus 是 finite-sample / non-asymptotic analysis of search complexity，而本文完全在渐近框架下工作。即使涉及 MAB 或 PG，方向也不同。如果做实验评估，本文的实验 protocol（M=1000 runs, T=2000 steps, biased init）是合理的 baseline 参考。
