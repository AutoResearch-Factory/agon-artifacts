# Overcoming Valid Action Suppression in Unmasked Policy Gradient Algorithms (2603.09090)

- **Year**: 2026.03
- **Authors**: Renos Zabounidis (CMU), Roy Siegelmann (MIT), Mohamad Qadri (CMU), Woojun Kim (CMU), Simon Stepputtis (Virginia Tech), Katia P. Sycara (CMU)
- **Venue**: arXiv (submitted to RLJ)
- **Keywords**: action masking, policy gradient, valid action suppression, feasibility classification, KL-balanced loss, representation learning, softmax policy
- **One-liner**: 揭示了 unmasked policy gradient 中 valid action 被指数级抑制的机制，提出 feasibility classification + KL-balanced loss 实现不依赖 oracle mask 的部署。

## 精华提炼

### 管线/方法

整体 pipeline 分为三个阶段：

1. **问题诊断**：在 unmasked training 中，agent 在已访问状态（visited states）上遇到无效 action 时，policy gradient 会降低其 logit。由于所有 action 共享 prefinal layer 的 feature representation φ(s)，这种 logit 下降通过共享参数传播到未访问状态（unvisited states），导致那些在 visited states 无效但在 unvisited states 有效的 action（如 descend、open_door）在 agent 到达之前就被指数级抑制。

2. **理论分析**：在 linear parameterization z_a(s) = φ(s)ᵀ w_a 下，证明两个条件导致抑制：(i) invalid-action dominance gap — 无效 action 的 Q 值严格低于最优有效 action；(ii) feature alignment — φ(s*) 与 visited states 的 advantage-weighted mean feature 不正交。定理给出 π_T(a|s*) ≤ e^{-K_T}/n 的指数衰减界，并分析了 entropy regularization 的 tradeoff。

3. **解决方案（Feasibility Classification）**：在共享 encoder φ(s) 上增加 per-action binary classification heads h_a，预测 ν(s,a)。训练时用 oracle mask 做 stability + classification loss 做 representation learning；部署时用 learned predictor 替代 oracle mask。分类 loss 有两种变体：Focal loss（处理 class imbalance）和 KL-balanced loss（按 policy sensitivity 加权）。

### 关键技术细节

- **参数化**：logits z_a(s) = φ(s)ᵀ w_a，其中 φ(s) 是神经网络 prefinal layer 的激活（对 deep network 的线性近似）
- **Theorem 1 (Probability Suppression)**：在 uniform init π₀ = 1/n 下，经 T 步梯度后 π_T(a|s*) ≤ e^{-K_T}/n，其中 K_T = Σ κ_τ，κ_τ = η·φ(s*)ᵀ E_{s~d_π}[c_τ(s)·φ(s)]
- **Zero-sum identity**：Σ_j Δz_j(s) = 0 ∀s，结合 Jensen 不等式得 Σe^{σ_j} ≥ n，这是指数衰减界的关键
- **Entropy regularization sandwich bound**：exp(-r_max/(β(1-γ))) ≤ π_∞(a|s*) ≤ e^{-K_∞}/n — entropy 提供下限但不能消除抑制
- **Architecture**：共享 encoder φ(s) → 三个 head：(1) classification head (sigmoid, per-action linear)，(2) policy head (oracle-masked softmax)，(3) value head
- **KL-balanced weight**：w_a(s) = π_θ(a|s) · |log π^oracle(a|s) - log π^pred(a|s)|，衡量 misclassification 对 policy 的影响。用 soft mask (-20 替代 -∞) 避免 KL 发散
- **总 loss**：L_total = L_PPO + λ · L_cls，λ=10
- **训练超参数**：1024 parallel envs, PPO-Hybrid (Transformer-XL + S5), batch size 131K, lr 2e-4, γ=0.999, entropy coef 0.002
- **环境**：Craftax-Full (43 actions, 9 floors, 8268-d symbolic obs), Craftax-Classic (17 actions, 1345-d), MiniHack Corridor-5 (11 actions, 256-d GlyphNet)

### 核心结果

| 指标 | Unmasked | Masked (Oracle) | Masked + KL | Masked + KL (Pred) |
|------|----------|-----------------|-------------|---------------------|
| Craftax PPO-Hybrid Return | 43.9 | 45.6 | 48.8 | 43.2 |
| Craftax PPO-RNN Return | 36.7 | 38.3 | 38.1 | 36.4 |
| Craftax PPO-MLP Return | 26.9 | 30.4 | 28.7 | 28.6 |
| Craftax-Classic PPO-Hybrid | 19.2 | 18.9 | 19.0 | 18.4 |
| Corridor-5 (all arch) | 0.9-1.0 | 0.9-1.0 | 0.9-1.0 | 0.9-1.0 |

**关键发现**：
- **RQ1 — 指数抑制验证**：π(descend|valid) 从 1/43≈0.023 降至 <10⁻⁴（50M frames 内），下降近两个数量级，符合 Theorem 1 预测。Unmasked agent 在早期训练中选择有效 action 的比例不到 15%。
- **RQ2 — Feature correlation**：Oracle masking 保持 feature correlation ≈0.8-1.0（valid vs invalid states），encoder 没有学到区分 valid/invalid 的表示。KL-balanced classification 将 correlation 降至 ≈0.4，学到的 predictor 达 99% accuracy。
- **RQ3 — 即使有 oracle mask，classification 仍提升性能**：Masked + KL 达 48.8 vs Masked only 45.6（+7%），因为 validity-discriminating features 也帮助 policy 学习。
- **RQ4 — Deployment without oracle masks**：Masked only 去掉 oracle 后 collapse 到 -0.9 return。Masked + KL 用 predicted masks 达 43.2，仅比 unmasked 的 GT 评估（43.9）低 2%。KL-balanced 比 Focal loss 好约 2x（43.2 vs 21.2）。
- **Surprising 结果**：MiniHack Corridor-5 虽然所有方法在 GT 评估下都接近最优（0.9-1.0），但 Masked only 去掉 oracle 后跌到 0.1-0.3，说明 masking 的成功掩盖了根本性的表示缺陷。

### 明确局限

- **作者自述**：(1) 理论假设 fixed state features φ(s)，未分析 φ 和 policy weights 联合优化的动态；(2) Predicted mask 部署恢复到 unmasked 性能但不超越之，存在 train-deploy distribution shift；(3) 假设训练时 access to ground-truth validity labels，noisy/partial validity 会降低 predictor 质量；(4) 仅限 discrete action spaces with symbolic state representations，continuous actions 或 pixel-based observations 可能不同；(5) 抑制机制可能与其他已知 failure mode（representation collapse, loss of plasticity, primacy bias）交互。
- **审读者补充**：(1) 仅测试了 Craftax 和 MiniHack 两个游戏环境，缺乏 robotics 或 real-world 验证；(2) 没有和 learned action elimination（如 Zahavy 2018）做直接对比实验；(3) 没有分析 classification head 的计算开销（per-action linear head 在 |A| 很大时可能显著）；(4) λ 的 sensitivity 只在 Craftax 上测，未跨环境验证；(5) 缺乏对 predictor 错误的 failure mode 分析——什么情况下 predictor 会出错，出错后 policy 行为如何退化。

### 审读视角

本文不直接撞车 0616-fex（feature explanation/extraction）课题，但提供了有价值的 representation learning 分析方法和工具。

**撞车评估**：无直接竞争。本文解决的是 RL 中 action masking 的 deployment dilemma 和 suppression 机制，属于 RL training dynamics。与 feature explanation 共享对 "representation 如何影响 behavior" 的关注，但问题设定完全不同。

**可迁移的方法论**：
1. **KL-balanced auxiliary loss**：用 KL divergence 度量 auxiliary task 错误对 main task 的影响来加权——这个 principle 可用于设计 feature explanation 的 training objective，确保解释 faithfulness 和 downstream performance 的平衡。
2. **Feature correlation 诊断**（Section 5.2）：直接测量不同 state group 间的 feature correlation 来诊断 representation quality，这种方法可用于评估 feature extraction 是否学到了有意义的分离表示。
3. **Theory-experiment 闭环**：从理论（Theorem 1）推导出可测量的条件（Condition ii: feature alignment），然后在实验中直接验证（Pearson correlation measurement），这种研究范式值得学习。

**该引不该引**：如果你的工作涉及 (a) auxiliary classification tasks 如何 reshape representation, (b) softmax policy 中 feature 如何通过参数共享影响概率分布, (c) RL training 中 representation learning 的 failure mode，应当引用。如果不涉及这些，作为 related work 中的 "representation learning via auxiliary tasks" 证据引用也合理，但非必须。

**关键数据点**：KL-balanced classification 将 feature correlation 从 0.8 降至 0.4 同时保持 99% classification accuracy；Masked + KL 用 predicted masks 仅比 oracle 评估低 2%。这些数字可作为 auxiliary classification task 有效性的实证参考。
