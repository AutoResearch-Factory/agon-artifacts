# A comprehensive and fair comparison of two neural operators (with practical extensions) based on FAIR data (2111.05512)

- **Year**: 2021.11
- **Authors**: Lu Lu (UPenn), Xuhui Meng, Shengze Cai (Brown), Zhiping Mao (Xiamen), Somdatta Goswami (Brown), Zhongqiang Zhang (WPI), George Em Karniadakis (Brown, corresponding)
- **Venue**: arXiv (Elsevier elsarticle format, likely submitted to a journal)
- **Keywords**: neural operators, DeepONet, FNO, operator regression, scientific machine learning, PDE solvers, FAIR data, feature expansion, POD
- **One-liner**: 在 16 个 benchmark 上系统比较 DeepONet 和 FNO，发现两者在简单设置下性能相当，但 FNO 对噪声极度敏感（0.1% 噪声使误差增 10000 倍），并提出 POD-DeepONet、dFNO+、gFNO+ 和 feature expansion 等实用扩展。

## 精华提炼

### 管线/方法

**整体 pipeline**：输入函数 v(x) 离散化 → 算子网络（DeepONet 或 FNO）→ 输出函数 u(ξ) 的连续预测（DeepONet）或离散预测（FNO）。

**DeepONet 管线**：
1. 输入函数 v 在传感器位置 {x₁,...,xₘ} 上做点估计离散化
2. Branch net 接收 [v(x₁),...,v(xₘ)]，输出 p 维系数向量 {b₁,...,bₚ}
3. Trunk net 接收坐标 ξ，输出 p 维基函数值 {t₁(ξ),...,tₚ(ξ)}
4. 输出 = Σbₖ·tₖ + b₀（两个网络的点积加 bias）
5. 可选 feature expansion：trunk net 输入端做 cos/sin 特征展开；branch net 输入端拼接额外特征函数
6. 可选硬约束 BC：Dirichlet 通过 g(ξ) + ℓ(ξ)·N(v)(ξ) 修正输出；Periodic 通过 Fourier 特征展开强制周期

**FNO 管线**：
1. 输入 v(x) 在 equispaced grid 上离散化
2. P 层（线性层或浅 MLP）将 v(x) 升维到 d_z 维表示 z₀(x)
3. L 个 Fourier 层迭代：每层 z_{l+1} = σ(F⁻¹(R_l · F(z_l)) + W_l·z_l + b_l)，F 为 FFT，R_l 为 Fourier 域复数权重张量
4. Q 层（MLP）将 z_L(x) 投影回标量输出 u(x)
5. 要求输入输出同 domain、同 mesh、equispaced grid

**FNO 扩展 (dFNO+, gFNO+)**：
- dFNO+ Case I (不同 domain)：扩展输入 domain 使其覆盖输出 domain（如加时间坐标 t），或通过 RNN 时间递推
- dFNO+ Case II (输入是输出的子集)：零填充或展开曲线的方法
- gFNO+ (复杂几何)：取几何体的最小包围盒作为 Cartesian domain，域外用"最近邻"填充保持连续性，loss 中 mask 只考虑域内点

### 关键技术细节

**DeepONet 扩展（6 项）**：
1. **Feature expansion in trunk net**：输入 ξ → (ξ, cos(ξ), sin(ξ), cos(2ξ), sin(2ξ), ...)，用于振荡解；也可用历史数据作为 feature
2. **Feature expansion in branch net**：将额外特征函数 f(x) 作为 branch net 的额外输入
3. **硬约束 Dirichlet BC**：输出 = g(ξ) + ℓ(ξ)·N(v)(ξ)，ℓ(ξ) 在边界为 0
4. **硬约束 Periodic BC**：用 Fourier 基 {cos(ωξ), sin(ωξ), cos(2ωξ), sin(2ωξ), ...} 替换 ξ 作为 trunk net 输入。2D 扩展用可训练相位偏移 φ_{i,j}
5. **POD-DeepONet**：对训练数据做 POD 分解得到基函数 {φ₁,...,φₚ}（去除均值），用这些 POD modes 替代 trunk net，branch net 只学系数。输出 = Σbₖ(v)·φₖ(ξ) + φ₀(ξ)
6. **Rescaling (二阶矩分析)**：DeepONet 输出方差正比于 p/8（p 是 branch/trunk 输出维度），建议缩放因子 O(1/√p)；POD-DeepONet 用 1/p 效果更优
7. **多输出处理**：将 branch 和 trunk 的输出分组，每组对应一个输出函数

**FNO 扩展**：
- dFNO+ (不同 domain)：Method 1 扩展输入 domain（加坐标维度）；Method 2 用 RNN 做时间递推
- gFNO+ (复杂几何)：最近邻填充 + mask loss
- Feature expansion：将坐标 x 和特征 f(x) 作为 FNO 的额外输入通道

**网络架构规格（按 problem）**：
- Burgers: Branch Depth 4×128, Trunk Depth 4×128, tanh; POD: Branch Depth 3×128, 32 modes
- Darcy PWC: Branch CNN, Trunk [128,128,128,128,100], ReLU; POD: CNN + 115 modes
- Darcy Cont.: Branch CNN, Trunk [128,128,100], tanh; POD: CNN + 10 modes
- Advection I: Branch Depth 2×256, Trunk Depth 4×256, ReLU; POD: Depth 2×256 + 38 modes
- Instability waves: Branch Depth 6×200, Trunk Depth 7×200, ELU; 4 Fourier features
- Navier-Stokes: Branch CNN, Trunk [128,128,64], tanh; POD: CNN + 29 modes
- Electroconvection: Branch [256,256,128], Trunk [128,128,128], ReLU; POD: Depth 3×256 + 12 modes

**训练细节**：
- Loss: element-wise MSE
- 所有 case 做 5 次独立训练，报告 mean±std 的 L² relative error
- 输出归一化：u(ξ) = std[u](ξ)·N(v)(ξ) + ū(ξ)，使网络输出均值为 0、标准差为 1
- 使用 learning rate decay, L² regularization

**数据规模**（总计 16 个 benchmark）：
- 训练数据量从 10（Cavity Unsteady II）到 40800（Instability waves）不等
- 数据来源：数值求解器（MATLAB PDE Toolbox, LBM, 高阶谱元法）+ 合成数据

### 核心结果

| Benchmark | Best DeepONet variant | Best FNO variant | 关键发现 |
|-----------|----------------------|------------------|---------|
| Burgers | POD-DeepONet 1.94% | FNO+norm 1.93% | 相当，POD-DeepONet 需 1/p 缩放 |
| Darcy PWC | POD-DeepONet 2.32% | FNO+norm 2.41% | POD-DeepONet 略优 |
| Darcy Cont. | FNO+norm 1.19% | POD-DeepONet 1.26% | 相当 |
| Darcy Pentagram | POD-DeepONet 0.82% | dgFNO+ 3.34% | DeepONet 大幅领先（复杂几何） |
| Darcy Triangular | POD-DeepONet 0.18% | dgFNO+ 1.00% | DeepONet 大幅领先 |
| Darcy Notch | POD-DeepONet 1.00% | dgFNO+ 7.82% | DeepONet 大幅领先（缺口几何） |
| Electroconvection | POD-DeepONet 0.14% | FNO+norm 0.43% | DeepONet 更优（刚性边界） |
| Advection I | POD-DeepONet 0.04% | FNO 0.66% | DeepONet 更优 |
| Advection II/III | POD-DeepONet 0.08%/0.40% | dFNO+1 0.22%/0.60% | FNO 原生失败（54%/48%误差），dFNO+ 可用但不如 DeepONet |
| Instability waves | DeepONet 8.90% | FNO 17.8% | DeepONet 大幅领先 |
| Compressible Euler | POD-DeepONet 0.020%/0.012% | FNO 0.076%/0.044% | POD-DeepONet 最优，对间断也有效 |
| Flapping airfoil | DeepONet(feature) 2.87% | dFNO+ 3.56% | 特征扩展有效 |
| Navier-Stokes | POD-DeepONet 1.71% | FNO+norm 1.81% | 相当 |
| Cavity Steady | POD-DeepONet 0.33% | dFNO+ 0.63% | POD-DeepONet 最优 |
| Cavity Unsteady I/II | POD-DeepONet(feature) 0.18%/1.51% | dFNO+(feature) 0.56%/1.78% | 特征扩展是关键 |

**Surprising 发现**：
1. **FNO 对噪声极度敏感**：在 Advection 和 Instability waves 中，仅 0.1% Gaussian 噪声使 FNO 误差增加 4 个数量级（270% → 10000×），而 DeepONet 几乎不受影响
2. **FNO 泛化 gap 大**：Instability waves 中 FNO 训练/测试误差差一个数量级，DeepONet 几乎无泛化 gap
3. **FNO 对不连续性差**：Compressible Euler 中 FNO 依赖 Fourier 变换，在间断处不准确；DeepONet 处理间断良好
4. **POD-DeepONet 几乎一致最优**：在 16 个 benchmark 中大部分达到最佳或接近最佳
5. **特征扩展对非稳态问题至关重要**：Cavity Unsteady 有无 feature expansion 误差从 ~14% 降到 ~0.5%
6. **dFNO+2（RNN 时间递推）极难训练**：10 次实验 4 次卡在 >20% 误差的坏局部最优

### 明确局限

**作者自述的局限**：
1. FNO 噪声敏感性问题需要未来工作解决（"future work should address this important issue"）
2. 多输出 DeepONet 的四种方法未做系统比较
3. POD-DeepONet 需要完整场数据计算 POD modes，不如 vanilla DeepONet 灵活
4. POD-DeepONet 在多尺度问题（如 instability waves 中幅度差 2 个数量级）效果差（20.8% vs DeepONet 8.9%），需要更好的 POD 策略
5. 理论上两者 error estimate 虽类似但可能不是最优的（"the estimation of sizes of networks may be not optimal"）
6. dgFNO+ 在复杂几何上始终不如 DeepONet，最近邻填充策略可能不够好

**我们读出的局限**：
1. 论文发表于 2021 年，FNO 此后已有大量改进（FNO 2D/3D、Geo-FNO、U-FNO 等），结论可能过时
2. 没有真机实验验证，全部为数值仿真数据
3. 没有做标准化 benchmark 上的系统性超参数搜索——架构选择可能偏向 DeepONet
4. 理论比较只覆盖 Burgers 方程，不够一般
5. 没有与同期或更新的 neural operator 架构比较（如 NOMAD、U-NO、LOCA 等）
6. FNO 的 "feature expansion" 部分不够深入——仅把坐标作为额外通道，未见系统消融研究
7. 没有计算成本/推理时间的 head-to-head 比较（虽声称实现快速版 DeepONet）
8. Code 声明 "will be available on Github upon publication"——实际可用性未知（2021 年的论文，code 可能已公开也可能没有）
