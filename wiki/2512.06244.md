# Auto-exploration for online reinforcement learning (2512.06244)

- **Year**: 2025.12
- **Authors**: Caleb Ju, Guanghui Lan (Georgia Tech, inferred from author emails)
- **Venue**: arXiv
- **Keywords**: online RL, auto-exploration, stochastic policy mirror descent, dynamic mixing time, linear function approximation, conditional TD, advantage gap certificate, sample complexity
- **One-liner**: 提出无需预知 mixing 参数的 online discounted RL 自动探索理论框架。

## 精华提炼

### 管线/方法
- 论文研究有限状态、有限动作、无限时域 discounted MDP 的 online RL。
- 数据模型是单条在线轨迹, 不是 generative simulator, 也不是离线 batch。
- 目标是得到 distribution-free 近似最优策略, 即所有状态上的 `V^pi(s)-V^*(s)` 都小。
- 核心矛盾是: online 轨迹不能任意采样状态, 但已有理论常假设所有中间策略都有足够 state-action 覆盖。
- 作者把探索难度抽象成 `D_expl`, 但强调既有 `D_expl` 往往依赖算法轨迹, 因而可能任意大且不可实现。
- 第一层工具是 stochastic policy mirror descent, SPMD。
- SPMD 每轮用估计的 `Q^{pi_t}` 做 mirror descent policy update。
- 论文先证明一个新的高概率 last-iterate SPMD 收敛命题。
- 这个命题只要求两类条件: Q 估计几乎处处有界, 以及加权偏差项足够小。
- 证明技巧借鉴 Jain et al. 的 last-iterate SGD 分组/epoch 分析。
- tabular auto-exploration 分支使用 truncated on-policy Monte Carlo, TOMC。
- TOMC 对非稀有 state-action pair 进行截断 Monte Carlo 估计。
- 如果某 `(s,a)` 的当前策略概率低于阈值 `underline pi`, 估计值直接设成 `(1-gamma)^{-1}`。
- 这个大值会惩罚稀有动作, 防止 SPMD 把概率质量放到没有证据的动作上。
- 新变量是 discounted hitting time `h_m^pi(z)=gamma^{m-tau_m^pi(z)}`。
- `h_m` 完全由在线轨迹可观测 hitting time 得到。
- 动态 mixing time 定义为最小 `m`, 使所有非稀有 pair 的 `h_m` 都低于 `varsigma(1-gamma)`。
- 这避免了预先知道 Markov chain mixing rate 或 stationary distribution。
- tabular 理论的结构假设是存在 deterministic optimal policy `pi*`。
- `pi*` 诱导的 Markov chain 要 irreducible, 有唯一 stationary distribution `nu*`, 且 `nu*(s)>=underline nu*`。
- `pi*` 的混合速度由 `C*<=2` 和 `rho* in [1/2,1)` 控制。
- 在这个假设下, 如果当前策略给所有最优动作至少 `underline pi` 概率, 则当前策略本身也 mixing。
- 关键命题给出当前策略的 stationary lower bound 和 mixing rate bound。
- 这一步把 implicit action exploration 转化为 implicit state exploration。
- policy update 使用 negative Tsallis entropy 诱导的 Bregman divergence。
- Tsallis index 取 `p=1/log((1-gamma)^-1)`。
- 初始策略是每个状态上均匀分布。
- tabular 主定理设置 `k=ceil(36^2 |A|^{1-p}(Qbar^2+M_h^2)/((1-gamma)^2(1-p)p epsilon^2))`。
- TOMC 偏差参数取 `varsigma=(1-gamma)epsilon/36`。
- 动作概率阈值 `underline pi` 随 `|A|`, `1-gamma`, `log(2|S|k/delta)` 和 `log((1-gamma)^-1)` 缩放。
- 定理保证以 `1-delta` 概率, last iterate `pi_k` 在所有状态上达到 `epsilon` 乘 polylog 的误差。
- tabular 总样本量是动态 mixing time 的总和。
- algorithm-dependent bound 仍含中间策略最差 stationary lower bound `underline nu` 和 mixing gap `1-bar rho`。
- algorithm-independent bound 把这些量替换成 `pi*` 的 `rho*`, `nu*` 和 `D_expl(delta)`。
- `D_expl(delta)=((|A| log(|Z|/delta))/(1-gamma))^{2 bar b}/((1-rho*)(underline nu*)^3)`。
- 因此 epsilon 依赖保持 `Otilde(epsilon^-2)`, 但常数可能非常大。
- 作者强调这是在线 discounted finite RL 中首次在较弱探索假设下得到 algorithm-independent `O(epsilon^-2)`。
- tabular 方法的主要缺陷是必须访问并枚举所有状态。
- 第二条主线转向 linear function approximation。
- 对每个 state-action pair 设 feature `phi(z) in R^d`, `Phi` 满列秩。
- 目标不是直接估计全表 Q, 而是在 `span(Phi)` 中求 projected Bellman equation 的解。
- 权重分布不是 stationary distribution, 而是 discounted state visitation distribution。
- 权重 `w(z)=kappa_{s_or,f}^pi(s) * tilde pi^A(a|s)`。
- `kappa_{s_or,f}` 混合了从 origin state 出发的 discounted visitation 和从 uniform state 出发的 visitation。
- `f` 是 state exploration frequency。
- `tilde pi^A` 是轻度 action-exploration policy, 用来保证动作覆盖。
- 作者证明在 `min_s kappa_{s_or,f}^pi(s)>0` 且 `eps_A` 足够小的条件下, projected Bellman equation 有唯一解。
- 对应线性算子 `F(theta)=Phi^T W(Phi theta-c-gamma P^pi Phi theta)` 是 strongly monotone。
- stochastic operator `hat F` 的构造分三步。
- 第一步 sampling origin: 以 `1-f` 选固定 `s_or`, 以 `f` 从 uniform state 选随机目标状态。
- 然后用 state-exploration policy `tilde pi^S` 走到目标状态。
- 第二步从目标状态开始, 用原策略 `pi` 跑一个 `Geo(1-gamma)` 长度的 rollout, 并截断在 mixing time `m`。
- 第三步在到达状态用 `tilde pi^A` 采动作, 观察 cost 和 next state, 再用原策略采下一动作。
- 由这两个 state-action pair 构造 one-step TD operator。
- 因为采样涉及 `tilde pi^S`, `tilde pi^A`, `pi` 三个策略, 这是 hybrid/off-policy sampling。
- CTD 算法用固定步长 `iota` 做 `theta_{t+1}=theta_t-iota hat F(theta_t)`。
- 它跑 `2N` 步, 返回后半段 `N` 个 Q 估计的平均。
- 作者证明 CTD bias 线性下降, variance 可由 `N` 和 `m` 控制。
- 为得到高概率有界 Q 估计, CTD 独立运行 `ceil(log_2(1/delta))` 次。
- robust estimator 选择这些输出中 infinity norm 最小的一个。
- SPMD+CTD 每个 PMD iteration 都运行多次 CTD, 形成 `Breve Q_t`, 再做 policy update。
- 但 SPMD+CTD 需要一个 lower bound `underline kappa`。
- `underline kappa` 必须小于所有中间策略、所有状态上的 `kappa_{s_or^{(t)},f}^{pi_t}`。
- 因此 SPMD+CTD 还不是完全 parameter-free。
- 最后一层算法用 doubling trick 搜索 `underline kappa`。
- 每个 epoch 设 `tilde underline kappa=2^{-ep}`, 运行 SPMD+CTD。
- 然后用 advantage gap certificate 判断输出策略是否足够好。
- advantage gap 定义为 `g^pi(s)=max_p {-psi^pi(s,p)}`。
- 它满足 `g^pi(s) <= V^pi(s)-V^*(s) <= (1-gamma)^-1 max_s g^pi(s)`。
- gap 估计通过两阶段 Monte Carlo 得到。
- Phase I 对非稀有 pair 用动态 mixing time 的 TOMC。
- Phase II 对稀有 pair 用 state-exploration policy 先访问该 pair, 再切回原策略估计 Q。
- 多次重复后用 Hoeffding 型界控制 gap 估计误差。
- 若估计 gap 小于阈值, algorithm early terminate。
- 若不通过, 降低 `underline kappa` 继续。
- 在 optimal-policy mixing 假设下, 取 `underline kappa=(1-gamma)f/|S|` 可保证最终成功。
- 线性函数逼近版本的误差包含 approximation error `epsilon_app,inf`。
- 若 `epsilon_app,inf` 很小, 作者给出可弱化 state-exploration frequency 依赖的 corollary。
- 特别地, 当 feature span 能表达所有 Q 时, `epsilon_app,inf=0`。
- 作者建议实践中可设 `f=0`, 但理论最坏界会变差并含 `D_expl(delta)^5`。

### 关键技术细节
- MDP: finite `S`, finite `A`, discount `gamma in [0,1)`。
- 成本加 regularizer 后假设落在 `[0,1]`。
- 收敛目标: distribution-free last-iterate optimality, 即所有状态同时控制。
- SPMD update: 对每个状态解一个 simplex 上的 mirror descent 子问题。
- Bregman divergence: tabular 主定理用 negative Tsallis entropy。
- 高概率 last-iterate Proposition 的核心输入: `||Q^{pi_t}||_*^2, ||hat Q^{pi_t}||_*^2 <= Qbar^2`。
- 第二输入: `E[<-delta_t, pi_t-pi*>] <= varsigma`。
- 输出 bound 含 `67 alpha(Qbar^2+M_h^2)(log(4k|S|/delta))^2/((1-gamma)sqrt(k))`。
- TOMC 稀有 pair 处理: `Qtilde=(1-gamma)^-1`。
- 动态 mixing time 只检查观测到的 `gamma^{m-tau_m(z)}`。
- optimal mixing assumption 的 `bar b=ceil(log(4/underline nu*)/log(1/rho*))`。
- 当前策略 mixing gap 下界约为 `underline pi^{bar b}(underline nu*)^2/(2bar b)`。
- 当前策略 stationary lower bound 约为 `underline pi^{bar b} underline nu*/2`。
- tabular sample complexity 主项: `Otilde(|A|^{1-p} log(|Z|/delta)^2 D_expl(delta)/((1-gamma)^5 epsilon^2))`。
- linear projected Bellman equation: `Qbar=Phi(Phi^T W Phi)^{-1}Phi^T W(gamma P^pi Qbar+c)`。
- strong monotonicity constant: 至少 `(1-gamma) mu/4`。
- CTD step size: `iota=(1-gamma)/(512 Omega^2)`。
- CTD epoch length proxy: `T=ceil(4096 Omega^2/((1-gamma)^2 mu(ukappa)))`。
- SPMD+CTD policy iterations: `k=ceil(343^2 |A|^{1-p}/((1-gamma)^4(1-p)p epsilon^2))`。
- action exploration in CTD: `eps_A=(1-gamma)underline kappa/4`。
- state exploration in CTD: `eps_S=(1-gamma)/(343^2 log(2k|S|/delta) log((1-gamma)^-1)^2)`。
- robust estimator: 多次 CTD 输出中选 `||Q||_infty` 最小者。
- certificate repetition count: `M=ceil(8/((1-gamma)^2 epsilon^2))`。
- certificate cost 额外含访问所有 rare state-action pair 的 `|A||S|` 因子。

### 核心结果
- 新 SPMD last-iterate theorem: 在有界 Q 和小 weighted bias 下, 得到高概率 last-iterate distribution-free 收敛。
- tabular theorem: 在 optimal-policy mixing 假设下, parameter-free SPMD+dynamic TOMC 达到 algorithm-independent `Otilde(D_expl epsilon^-2)` 样本复杂度。
- tabular 方法不需要预估中间策略的 mixing rate 或 stationary distribution。
- 这改进了需要显式探索 schedule 或依赖 algorithm-dependent exploration constants 的先前 PMD 分析。
- linear theorem: SPMD+CTD 在给定有效 `underline kappa` 时, 收敛到 function approximation error 邻域。
- auto LFA corollary: 用 doubling trick 和 advantage-gap certificate 移除 `underline kappa` 输入。
- 若 feature 表达能力足够好, 可降低对 state-exploration frequency `f` 的依赖。
- surprising 点: tabular 情况下显式探索不是必须的, optimal-policy mixing 加 PMD 的 implicit action exploration 可以带来 state exploration。
- 另一个反直觉点: linear function approximation 下显式 exploration 的作用不是主要为了 state exploration, 而是对冲未知 approximation bias。
- 与经验深度 RL 的联系: 作者把 robust estimator 类比 gradient clipping, 把 discounted visitation sampling 类比 GAE/PPO 中的经验启发。

### 明确局限
- 没有数值实验, 没有 benchmark, 没有真机或实际环境验证。
- 所有结果是理论样本复杂度, 且常数很大。
- tabular bound 有 `(1-gamma)^-5`, linear auto bound 可到 `(1-gamma)^-13`。
- `D_expl(delta)` 可能非常大, 尤其当 optimal policy mixing 慢或 `underline nu*` 小。
- optimal-policy mixing assumption 仍是强结构假设, 不是一般 online RL 保证。
- tabular dynamic mixing time 要枚举并访问非稀有 state-action pairs, 大状态空间不可行。
- linear 方法避免全表枚举, 但需要 feature span 质量好。
- approximation error `epsilon_app,inf` 一般不可方便认证。
- CTD sampling 假设可以识别目标状态并等待 hitting, 对连续状态或高维状态并不直接成立。
- Phase II certificate 需要依次访问 rare state-action pairs, 最坏情况下成本很高。
- 论文的 TeX 中有少量符号/排版瑕疵, 例如个别括号和索引写法不整洁, 但主线可读。
- 对工程应用, 缺少伪代码外的实现细节和默认参数调优经验。
- 对深度非线性函数逼近, 只在结论中提出 general state space/random feature 方向, 没有证明。
