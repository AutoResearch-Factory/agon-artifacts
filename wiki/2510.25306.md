# Hierarchical Physics-Embedded Learning for Prediction and Discovery in Spatiotemporal Dynamical Systems (2510.25306)

- **Year**: 2025.10
- **Authors**: Xizhe Wang, Xiaobin Song, Qingshan Jia, Hao Sun, Hongbo Zhao, Benben Jiang; Tsinghua University, Renmin University of China, UC San Diego
- **Venue**: arXiv
- **Keywords**: HPE-AFNO, physics-embedded learning, adaptive Fourier neural operator, PDE discovery, spatiotemporal dynamics, symbolic regression, sparse noisy data, phase-field models
- **One-liner**: 用两级 AFNO 嵌入已知物理并从稀疏噪声数据预测/发现 PDE。

## 精华提炼

### 管线/方法

1. 输入是稀疏、可能带噪的时空快照 `U_sparse`，典型设置是二维或三维 PDE 场。
2. 论文目标有两个：前向预测完整时空演化，以及从数据中反推可解释的 PDE 组成项。
3. 主框架叫 HPE-AFNO，即 Hierarchical Physics-Embedded Adaptive Fourier Neural Operator。
4. 它把 PDE 右端项 `F(u, ux, uxx, ...)` 拆成两层：第一层抽取基础物理表达式，第二层学习这些表达式的组合规则。
5. 对 Cahn-Hilliard 示例，第一层对应 `M(c)`, `mu_hom(c)`, `kappa nabla^2 c` 等基础项。
6. 第二层对应组合形式，例如 `div(M(c) grad(mu_hom(c) - kappa nabla^2 c))`。
7. 两层都可以使用 AFNO 通道学习未知项，也可以用显式计算通道嵌入已知物理。
8. 已知物理项不再由神经网络拟合，而是直接写入 computational graph，作为额外 channel 进入后续组合。
9. 未知项由 Fourier attention/AFNO 表示，利用频域 token/channel mixing 捕获全局依赖。
10. 模型按 forward Euler 递推：`u_{k+1}=u_k + F_hat(u_k, theta) dt`。
11. 因为是递归结构，训练出的单步动力学可以用于插值窗口内重建，也可以外推到观测窗口之后。
12. AFNO 部分先做 patch 和 position embedding，再经 DFT、频域 channel mixing、IDFT 回到物理域。
13. AFNO 实现使用 block-diagonal frequency weights、共享 MLP、soft-shrinkage、hard-thresholding fraction。
14. 论文声称 Fourier attention layer 对高阶微分算子有 universal approximation 能力。
15. 理论部分给了分层误差分解：总误差由第一层基础项误差和第二层组合误差控制。
16. 误差界形式是 `epsilon <= epsilon_C + L_C sum_i epsilon_E_i`。
17. 这个界解释了为什么嵌入已知基础项或已知组合规则会收紧误差上界。
18. PDE discovery 管线先用 HPE-AFNO 从稀疏噪声观测重建高保真时空场。
19. 然后提取每个 AFNO unknown channel 对应的物理项输出。
20. 对 concentration-dependent 项，使用 concentration binning：按 `c` 的区间聚合模型输出，得到 `D(c)` 或 `mu_hom(c)` 的经验曲线。
21. 为保证未知项对状态变量连续，作者加入 Gaussian kernel feature mapping。
22. kernel 权重形如 `exp(-(u_i-u_j)^2/(2 sigma^2))`，用相似输入值的加权平均输出平滑物理项。
23. 最后用 Deep Symbolic Regression 从 binning 曲线恢复闭式表达。
24. DSR 使用 RNN 逐 token 生成表达式树，token library 包含变量、算子和 `log`, `exp` 等函数。
25. DSR 的 reward 基于 NRMSE，训练使用 risk-seeking policy gradient，强调 top-epsilon 分位的高 reward 表达式。

### 关键技术细节

1. 主要实验系统包括 Cahn-Hilliard, Allen-Cahn, deterministic KPZ, Complex Ginzburg-Landau, 3D Allen-Cahn。
2. CH 主设置：64x64 空间网格，91 个观测快照，观测间隔 `dt'=0.1s`，时间范围 `0-9s`。
3. 模型重建的高分辨率时间步长是 `dt=0.01s`，在 `0-9s` 生成 901 个插值快照。
4. 外推评估维持 `dt=0.01s`，预测 `t>9s`，文中常看到到 `20s` 的结果。
5. CH 方程的物理形式使用 `dc/dt = div(M(c) grad(mu_hom(c) - kappa nabla^2 c))`。
6. 其中 `M(c)=c D(c)`，`mu_hom(c)=log(c/(1-c)) + 3(1-2c)`，`D(c)=1-c`，`kappa=1`。
7. AC 使用非守恒相场动力学，状态依赖 kinetic coefficient `R(c)=c(1-c)`。
8. dKPZ 设置中扩散系数 `nu=0.1`，非线性耦合 `lambda=-0.5`。
9. CGL 设置中 `alpha=-0.5`，`beta=1.07`，输入只用复场 modulus，因此相位信息不完整。
10. 3D-AC 用于检验三维界面演化和 extrapolation 下的拓扑保持。
11. baseline 包括 PeRCNN、FNO-2D/FNO-3D，PDE discovery 还比较 PeRCNN+sparse regression、DSR、genetic programming。
12. 训练 loss 是 MSE，优化器是 Adam。
13. Supplementary Table 1 给出的学习率是 `1e-3`，StepLR `step_size=500, gamma=0.5`。
14. Adam betas 为 `(0.9, 0.999)`，epsilon 为 `1e-8`。
15. AFNO patch size 为 `[4,4]`。
16. 第一层 AFNO embedding dimensionality 为 32，input channels 为 1，output channels 为 3。
17. 第二层 AFNO embedding dimensionality 为 32，input channels 为 3，output channels 为 1。
18. 第一层 number of blocks 为 2，depth 为 1。
19. 第二层 number of blocks 为 4，depth 为 2。
20. 两层 MLP ratio 都是 2.0，dropout rate 都是 0.3。
21. sparsity threshold 都是 0.01，hard thresholding fraction 都是 1.0。
22. 论文说 AFNO 实现来自 NVIDIA Modulus 的 AFNO 代码路径。
23. 噪声设置使用 Gaussian noise：`truth + R * sigma * (max(truth)-min(truth))/2`。
24. 噪声强度包括 5%、10%、20%。
25. 稀疏/噪声鲁棒性实验对 CH 和 AC 各配置用 5 个随机初值重复。

### 核心结果

1. CH 插值区间 `0<=t<=9s`，HPE-AFNO 平均 RMSE 为 0.021。
2. CH 外推区间 `9s<t<=20s`，HPE-AFNO 平均 RMSE 为 0.077。
3. CH 在 `t=2.5s` 的代表性 RMSE 为 0.015。
4. CH 在 `t=18.5s` 的代表性 RMSE 为 0.11。
5. PeRCNN 在 CH 外推后期明显偏离，`t=18.5s` RMSE 约 0.27。
6. CH 上相对 PeRCNN，HPE-AFNO 插值平均 RMSE 降低 71.2%，外推降低 62.8%。
7. CH 上相对 FNO-2D，插值降低 51.3%，外推降低 49.2%。
8. dKPZ 插值阶段 HPE-AFNO 平均 RMSE 为 0.0049，PeRCNN 为 0.018，FNO-2D 为 0.035。
9. dKPZ 外推阶段 HPE-AFNO 平均 RMSE 为 0.006，PeRCNN 为 0.042，FNO-2D 为 0.032。
10. CGL 上 HPE-AFNO 更能恢复 spiral core dynamics，PeRCNN 在 `t>6.5s` 后失去结构相干性。
11. 3D-AC 上 HPE-AFNO 能保持三维界面结构，baseline 在外推阶段边界模糊或时间演化失真。
12. partial physics 实验定义三种情形：black-black、white-black、black-white。
13. black-black 表示基础项和组合规则都未知。
14. white-black 表示基础物理表达式已知，但组合规则未知。
15. black-white 表示组合结构已知，但基础物理表达式未知。
16. CH 上 partial physics 的三种情形在插值区间接近，主要差异出现在外推。
17. white-black 外推平均 RMSE 比 black-black 降低 23.7%。
18. black-white 外推平均 RMSE 比 black-black 降低 17.3%。
19. 在 `t=15s`，white-black 误差降低 25%，black-white 误差降低 18.5%。
20. PDE discovery 中，HPE-AFNO 在 CH 上恢复了 `D(c)` 和 `mu_hom(c)` 两个未知表达式。
21. 即使噪声到 10%，Supplementary Fig. 1/2 显示 binning 曲线仍能贴近 ground truth。
22. 对比方法 PeRCNN+sparse regression、端到端 DSR、GP 在同条件下未能识别 `D(c)` 或 `mu_hom(c)`。
23. 鲁棒性实验显示，20% 噪声和相对 `0.01s` 全分辨间隔的 80 倍降采样下，模型仍保持稳定预测。
24. 反直觉点：black-black 虽误差最大，但跨随机运行 variance 最小，说明可能收敛到稳定但错误的 attractor。
25. 作者把这解释为 partial physics 允许模型在物理约束空间内调整，虽然 variance 稍高但更接近真值。

### 明确局限

1. 实验主要是 synthetic PDE benchmark，不是实验室真实观测数据。
2. 作者在结论中说未来可扩展到 multi-physics coupling、irregular geometries、real-world experimental data，说明本文尚未覆盖这些场景。
3. 虽然引言强调 electrochemistry 和 biology，核心量化评估仍是 CH/AC/dKPZ/CGL/3D-AC 等标准或合成系统。
4. 没有看到公开代码、训练数据或独立复现实验；R4 复现性受限。
5. baseline 对比集中在 PeRCNN 和 FNO，缺少更多 2024-2026 operator learning / transformer PDE 模型的系统比较。
6. PDE discovery 主要展示 CH 上的 `D(c)` 和 `mu_hom(c)`，尚未证明能稳健发现更复杂多物理耦合系统。
7. 物理嵌入需要用户知道哪些项已知、哪些组合已知；现实系统中的 prior 质量会强烈影响效果。
8. Gaussian kernel continuity mapping 适合一元 `c -> term` 关系，多变量或历史依赖 constitutive relation 未充分验证。
9. DSR 的 token library 和 reward 仍会影响表达式搜索空间，论文没有给出足够的 sensitivity 分析。
10. CGL 只用 modulus 输入是困难设置，但也让问题和标准 CGL 全状态预测不完全等价。
11. 理论误差分解依赖理想分解和 Lipschitz combiner 假设，更多是解释性 bound，不是泛化保证。
12. 论文源在 arxiv tool 中解析为 PDF-only artifact，非真正 LaTeX source；全文审读基于该 47 页 PDF 的完整文本抽取。
13. Semantic Scholar 只解析出 16 条 references，少于正文编号到 59 的参考文献，说明 citation metadata 不完整。

### 跟我们的关系

1. 对 `0616-fex`，如果课题涉及 feature extraction、PDE discovery、operator learning 或 physics-guided surrogate，这篇是强相关背景。
2. 可能撞车的 claim 是“两级结构：先抽取基础物理项/feature，再学习组合规则”。
3. 另一个可能撞车点是“已知物理项作为 deterministic channel 直接嵌入 computational graph，而未知项由 neural operator channel 表示”。
4. 第三个撞车点是“每个 unknown AFNO channel 对应一个待发现物理项，之后用 symbolic regression 得到闭式表达”。
5. 如果我们的方案也做 PDE governing law discovery，需要明确区别：是不同结构分解、不同 operator backbone、不同数据 regime、真实实验数据，还是不同 verification/gating 机制。
6. 如果我们的 FEX 指的是 feature extraction，本论文的第一层 AFNO 可以视作 physics-aware feature extractor；可借用“feature channel 对齐物理项”的 framing。
7. 如果我们的 FEX 指的是 forward extrapolation，这篇的关键启发是 partial physics 对 extrapolation 的贡献比 interpolation 更明显。
8. 该文适合作为 PeRCNN/FNO/DSR 之间的桥接引用：它把 operator learning、physics embedding 和 symbolic discovery 连成一条管线。
9. 不建议直接复用其 benchmark 数字作为强证据；需要在我们的数据和 baselines 上重跑。
10. 差异化方向可以是：真实实验数据、irregular mesh/geometry、多变量 constitutive law、uncertainty quantification、独立 verifier、或 claim-level audit trail。

## Read by: 0616-fex (2026-06-16)

本次审读重点看它是否会撞到 `0616-fex` 的 feature/extraction 或 physics-discovery 方向。结论是强相关但不一定完全撞车：它的主 claim 是 HPE-AFNO 的两级物理结构，第一层抽取基础物理表达式，第二层学习组合规则，并允许把已知物理作为硬编码 channel 嵌入。若我们的课题也声称“从稀疏噪声数据中抽取可解释物理 feature 并用 symbolic regression 发现 PDE”，需要把差异化讲清楚，尤其是是否有真实数据、不同的 feature factorization、不同的 verifier、或更广的多物理/非规则几何场景。

可用的方法点有三个：第一，把 feature channel 显式绑定到物理项，可以提高可解释性；第二，把 known physics 和 learned residual 分开，能显著改善外推；第三，在 symbolic regression 前做 concentration binning/consistency smoothing，可以降低点预测噪声。需要警惕的是，本文主要在 synthetic PDE 上验证，未见代码/数据 release，也没有 independent reproduction；因此适合作为设计启发和相关工作，不足以直接支持强可信 claim。
