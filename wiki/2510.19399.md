# Iterative Training of Physics-Informed Neural Networks with Fourier-enhanced Features (2510.19399)

- **Year**: 2025.10
- **Authors**: Yulun Wu, Miguel Aguiar, K. H. Johansson, Matthieu Barreau (KTH Royal Institute of Technology / Digital Futures)
- **Venue**: arXiv
- **Keywords**: PINN, spectral bias, Random Fourier Features, bi-level optimization, high-frequency PDE, linear regression, physics-informed learning, feature basis
- **One-liner**: 用隐藏层 RFF 扩展和两阶段训练缓解 PINN 高频谱偏置。

## 精华提炼

### 管线/方法
- 输入是一类带边界/初值条件的 PDE：内部算子 $\mathfrak{F}[u](x)=f(x)$，边界算子 $\mathfrak{B}[u](s)=g(s)$。
- 训练数据由边界/初值点 $X_u$ 和内部 collocation 点 $X_f$ 组成，损失仍是标准 PINN 的边界项加 PDE residual 项。
- 第一步先训练一个普通 PINN $\tilde{u}_{\omega,W}(x)=W h_\omega(x)$，让隐藏层 $h_\omega$ 学到一个 nominal latent basis。
- 这个 warm start 不一定每个实验都用：低频 Helmholtz 和高频 Helmholtz表中为 No，convection、Burgers、convection-diffusion 为 Yes。
- 论文的核心改动不是把 RFF 直接加在原始输入 $x$ 上，而是加在最后隐藏层特征 $h_\omega(x)$ 上。
- RFF 映射定义为 $\psi_D(x)=\gamma_D(h_\omega(x))=\frac{1}{\sqrt{D}}[\cos(2\pi B_D h_\omega(x)), \sin(2\pi B_D h_\omega(x))]$。
- $B_D \in \mathbb{R}^{D\times p}$ 从 $\mathcal{N}(0,\sigma^2)$ 独立采样，实验中 $\sigma=1$ 固定不调。
- 最终预测不是普通线性层 $W h_\omega$，而是 $u_{\omega,\theta}(x)=\psi_D(x)^\top\theta$。
- 训练被拆成两个问题：上层学习 basis generator 参数 $\omega$，下层在固定 basis 上求线性 readout 系数 $\theta$。
- 对线性 PDE 和线性边界算子，下层损失可以写成二次型 $\frac12\theta^\top Q(\omega)\theta+c(\omega)^\top\theta+b$。
- $Q$ 来自边界设计矩阵 $B_u^\top B_u$ 和 residual 设计矩阵 $R_f^\top R_f$ 的加权和。
- 如果 stacked design matrix $M=[\sqrt{2/N_u}B_u;\sqrt{2\lambda/N_f}R_f]$ 满列秩，则 $Q$ 正定。
- 满列秩的必要条件是 $N_u+N_f\ge 2D$，所以 RFF 特征数量不能脱离采样预算无限增大。
- 当 rank 不足或病态时，作者使用 Tikhonov regularization：$Q_{\mathrm{reg}}=Q+\gamma I$。
- 下层闭式解写作 $\theta^\star=-(Q+\gamma I)^{-1}c$；正文算法中无正则时写作 $-Q^{-1}c$。
- 每个 epoch 的算法是：构造 $\psi_D$，形成 BLO 二次目标，下层解 $\theta_{k+1}$，上层对 $\omega_k$ 做一次梯度下降。
- 上层更新为 $\omega_{k+1}=\omega_k-\eta\nabla_\omega\mathfrak{L}_{\mathrm{upper}}(\omega_k\mid\theta_{k+1})$。
- 论文通过 IFT 给出 hypergradient 形式，依赖 $Q^{-1}$、$\partial(Q\theta^\star)/\partial\omega$ 和 $\partial c/\partial\omega$。
- 理论结论主要是：下层强凸时 $\theta^\star(\omega)$ 局部 Lipschitz；上层 L-smooth 且步长合适时收敛到 stationary point。
- 因此“全局最优”的说法只对固定 $\omega$ 后的线性下层更准确；完整 bi-level 神经网络训练只证明到 stationary point。
- 表达能力证明走 RKHS/RFF 路线：复合 RFF 空间的闭包包含原始隐藏层线性 readout 空间。
- 论文声称当神经元数和 RFF 特征数足够时，$\mathcal{H}_{\mathrm{RFF}}$ 对 PDE 解的 projection error 可以任意小。

### 关键技术细节
- 基础网络全用 MLP、tanh activation、Adam optimizer，网络参数学习率 $10^{-3}$。
- IFeF-PD 额外使用 primal-dual weight balancing，动态 physics weight 的 Adam 学习率为 $10^{-4}$。
- Vanilla PINN 的 physics loss 权重在正文实验中简化设为 $\lambda=0.01$。
- 高/低频 Helmholtz、convection、Burgers、convection-diffusion 都用 relative $L^2$ error 评估。
- 所有方法每个实验跑 5 个 random seeds；正文说展示/比较 convergence 后结果，并提到 best predictions。
- 硬件是单张 NVIDIA GeForce RTX 4090 GPU；实现为 PyTorch；代码承诺论文接收后 release。
- 低频 2D Helmholtz：$a_1=1,a_2=4$，domain $[-1,1]^2$，uniform sampling，边界点 1000，physics 点 $71\times71$。
- 高频 2D Helmholtz：$a_1=a_2=100$，domain $[0,0.2]^2$，LHS sampling，边界点 3000，physics 点 23000。
- 1D convection：$u_t+\beta u_x=0$，periodic boundary，解 $u(x,t)=\sin(x-\beta t)$。
- 低频 convection 用 $\beta=50$，高频 convection 用 $\beta=200$。
- Convection-diffusion：$c=1,d=0.00005,A_{\text{low}}=1,A_{\text{high}}=0.1,k_{\text{low}}=4\pi,k_{\text{high}}=60\pi$。
- Burgers：$u_t+uu_x=\nu u_{xx}$，$\nu=0.01/\pi$，domain $[0,1]\times[-1,1]$。
- IFeF 超参：低频 Helmholtz $D=800,\lambda_{\mathrm{LL}}=10^{-2},\gamma=10^{-7}$。
- 高频 Helmholtz $D=2400,\lambda_{\mathrm{LL}}=10^{-7},\gamma=10^{-4}$。
- 低频 convection $D=800,\lambda_{\mathrm{LL}}=10^{-2},\gamma=10^{-7}$。
- 高频 convection $D=1600,\lambda_{\mathrm{LL}}=10^{-2}$，表中 $\gamma$ 写作 `le-4/1e-7`，明显有排版或拼写问题。
- Burgers $D=800,\lambda_{\mathrm{LL}}=10^{-1},\gamma=0$。
- Convection-diffusion $D=800,\lambda_{\mathrm{LL}}=10^{-2},\gamma=10^{-7}$。
- 网络结构：低频 Helmholtz 是 3 层隐藏层，宽度 [50,50,20]。
- 高频 Helmholtz、两个 convection、convection-diffusion 都是 6 层、宽度 64。
- Burgers 是 8 层、宽度 20。
- Spectrum analysis 使用 8 层、每层 64 hidden units、tanh 的 FCN。
- Spectrum analysis 的频率集合是 $\{1,2,5,10,30,40,50,60,70,80\}$，所有幅值 1。
- Spectrum analysis 中 vanilla PINN 先用 Adam $10^{-3}$ 训练 40000 epochs。
- 然后不做上层迭代，只把 basis 扩展成 $D_j\in\{400,800,1600,2400,3200,4000\}$ 并解下层问题。
- 该谱实验还强制 $D_i<D_j$ 时 $B_{D_i}\subset B_{D_j}$，使不同 D 的 RFF 矩阵嵌套，比较更干净。

### 核心结果
- 低频 benchmark 上，IFeF/IFeF-PD 在 Helmholtz、convection、Burgers 三个任务中取得最低 median error 和更小 variability。
- 低频 Helmholtz 最好结果：IFeF-PD relative $L^2$ error 为 $3.5\times10^{-5}$。
- 低频 convection 最好结果：IFeF relative $L^2$ error 为 $4.3\times10^{-5}$。
- Burgers 是 nonlinear PDE；作者只给出“lowest median error”的定性说法，正文没有在 tex 中列出对应数值表。
- 低频 convection 上移除 RFF、只保留类似二阶段优化，relative $L^2$ error 为 $1.4923\times10^{-2}$，只比 vanilla 略好。
- 高频 Helmholtz 表中 vanilla、PINNsformer、NTK 都未收敛，PIG 为 $1.6884\pm0.2775$。
- 高频 Helmholtz 上 IFeF 为 $0.0156\pm0.0055$，IFeF-PD 为 $0.0092\pm0.0031$。
- 高频 convection $\beta=200$ 上 vanilla 为 $0.9024\pm0.0239$，PINNsformer 为 $1.2278\pm0.2010$，NTK 为 $0.8685\pm0.0318$，PIG 为 $1.0009\pm0.0003$。
- 高频 convection 上 IFeF 为 $0.0027\pm0.0010$，IFeF-PD 为 $0.0025\pm0.0005$。
- 多尺度 convection-diffusion 上 vanilla/PINNsformer/NTK/PIG 约在 $0.050$ 到 $0.056$，IFeF 为 $0.0009\pm0.0003$，IFeF-PD 为 $0.0010\pm0.0002$。
- 高频 Helmholtz 和高频 convection 的 ablation 中，去掉 RFF basis extension 后训练不收敛。
- Spectrum analysis 显示 vanilla PINN 对高频 components 的 FFT magnitude 明显不足。
- 只做 RFF basis extension 和一次下层求解时，随着 D 增大，高频 components 的拟合显著改善。
- 最有说服力的结果不是低频 benchmark，而是高频 Helmholtz/convection 和多尺度 convection-diffusion 中 baselines 接近失败、IFeF 明显可解。
- 反直觉点：作者固定 $\sigma=1$，仍在多种频率任务上取得强结果；但这也留下调参敏感性问题。
- 另一个关键点：RFF 不加在输入层，而加在 learned hidden basis 上，意图是把 vanilla PINN 已学到的低频/几何表示扩展成更丰富的 stationary-kernel feature space。

### 明确局限
- 作者明确说 nonlinear PDE 下下层问题不再凸，不能一部闭式求全局解，只能用迭代二阶段梯度法，可能卡在局部极小。
- 作者明确说方法对 resampling 敏感，因为 bi-level objective 依赖固定 dataset 上的下层最优解。
- 作者明确说 high memory footprint；从 $2D$ features 和 $Q\in\mathbb{R}^{2D\times2D}$ 看，$D=2400$ 或 4000 时矩阵开销会很快变大。
- rank 条件要求 $N_u+N_f\ge2D$，采样点不足时可能得到 aliased solution；这不是普通 PINN 必然面对的约束。
- $\sigma$ 的选择没有系统 study，论文只说固定 $\sigma=1$，未来会分析。
- 实验全是解析解 PDE benchmark，没有真机、真实物理数据或跨平台 simulator 验证。
- 与 PDEBench、operator-learning benchmark 或更大规模工程 PDE 的关系没有建立。
- 低频 benchmark 主要靠 boxplot 图，不是所有任务都给完整数值表。
- “best predictions”和 5 seeds 的描述需要小心：若按 best rather than mean/median 选图，可能高估视觉结果。
- 代码未随 arXiv 源码公开，只承诺接收后开源；当前第三方复现成本较高。
- 表格中高频 convection 的 $\gamma$ 写作 `le-4/1e-7`，属于可复现性瑕疵。
- 理论部分有一个容易被误读的点：下层 linear QP 可以有唯一全局解，但完整 $\omega$ 学习仍是非凸 neural optimization。
- 对比 baselines 使用“default settings”，但没有看到对所有 baseline 做同等强度调参的证据。
- 没有和专门的 Fourier-feature PINN 变体做很细的 apples-to-apples 消融，例如 input RFF vs hidden-layer RFF、learned Fourier features、SIREN 类网络。
- RFF 随机性对 seed、D、rank、regularization 的稳定性没有系统置信区间分析。

### 跟我们的关系
- 对 `0616-fex` 来说，这篇高度相关，甚至可能是直接撞车论文。
- 如果 `fex` 指 Fourier feature expansion / Fourier-enhanced features for PINN，那么该文已经覆盖“hidden feature 上做 RFF 扩展 + linear readout regression + iterative upper/lower training”的核心 claim。
- 该文的差异化点很明确：不是只做 input encoding，而是在 learned latent basis $h_\omega$ 后做 RFF；不是纯 end-to-end PINN，而是把 readout coefficient fitting 单独变成下层问题。
- 如果我们的课题也打算声称“RFF 缓解 PINN 高频谱偏置”，这篇必须引用，而且 novelty 需要收缩到更具体的 claim。
- 可借鉴工具：下层 QP/regularized least squares formulation、rank 条件 $N_u+N_f\ge2D$、nested RFF spectrum ablation、high-frequency Helmholtz/convection benchmark。
- 可借鉴实验：高频 Helmholtz $a_1=a_2=100$，高频 convection $\beta=200$，multi-scale convection-diffusion $k_{\text{low}}=4\pi,k_{\text{high}}=60\pi$。
- 可借鉴叙事：把 spectral bias 解释成“feature learning 和 coefficient fitting 耦合”导致的问题，再用 classical basis solver 类比拆开。
- 需要避开的撞车 claim：单纯“在 PINN 里加入 Fourier features 捕捉高频”不新；“隐藏层特征 RFF + 两阶段训练”也已被这篇占住。
- 可能的差异化方向一：把下层解法扩展到 nonlinear PDE 时仍保留可验证的独立检验或 convex surrogate，而不是停在 local minimum。
- 可能的差异化方向二：解决 resampling sensitivity，把 adaptive sampling 和下层 QP/rank conditioning 统一起来。
- 可能的差异化方向三：做 $\sigma$、D、rank、regularization 的系统理论和自动选择，而不是固定 $\sigma=1$。
- 可能的差异化方向四：把方法放到真实工程 PDE、PDEBench 或 operator generalization setting，而不是只在单实例 analytic PDE 上比较。
- 可能的差异化方向五：提供可复现 artifact 和 independent evaluator，把 R-score 从 B 提到 A。

## Read by: 0616-fex (2026-06-16)

从 `0616-fex` 视角看，这篇是强相关、强撞车论文。它已经把 “Fourier-enhanced feature expansion + PINN spectral bias + high-frequency PDE” 这条线讲得比较完整，且不是浅层 input Fourier encoding，而是对隐藏层 learned basis 做 RFF 扩展，再把最后 readout 单独解成下层回归/QP。若我们的计划只是做 Fourier feature 扩展来提升 PINN 高频拟合，这篇会直接压缩 novelty。

可用部分很明确：下层 QP 形式、rank 条件、Tikhonov regularization、nested RFF spectrum ablation、高频 Helmholtz/convection/convection-diffusion 三组 benchmark 都值得复用或对照。需要警惕的是，它的理论保证只稳在 fixed basis 的线性下层；full bi-level training 仍只到 stationary point，nonlinear PDE 下也没有 convex guarantee。我们的差异化最好落在 resampling-aware 的 feature expansion、自动 D/σ/γ 选择、非线性下层的可验证 surrogate，或更强 artifact/独立复现实验上。
