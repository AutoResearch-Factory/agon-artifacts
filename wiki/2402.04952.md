# Separation-Based Distance Measures for Causal Graphs (2402.04952)

- **Year**: 2024-02
- **Authors**: Jonas Wahl (DFKI, Saarbrücken), Jakob Runge (U. Potsdam / ScaDS.AI Dresden-Leipzig / TU Berlin)
- **Venue**: arXiv preprint (generic arxiv-style template; no confirmed peer-reviewed venue in the source)
- **Keywords**: causal graph distance, Markov equivalence class, d-/m-separation, structural Hamming distance, maximal ancestral graph, adjustment identification distance, CPDAG/PAG, causal discovery evaluation
- **One-liner**: 提出基于分离/连接语句而非边差异的因果图距离族 (s/c-metric、可扩展 SD 家族)，DAG/MAG/CPDAG/PAG 通用且多项式时间可算。

## 精华提炼

### 管线/方法

这是一篇纯理论 + 玩具实验/合成实验论文，没有数据管线，核心是"定义 + 证明 + 复杂度分析 + 小规模仿真验证"四步:

1. **问题设定**: 现有比较两个因果图 G, H 的指标 (SHD、AID/SID) 只看边的存在/方向差异或下游可辨识效应差异，但许多因果发现算法 (PC, GES, PCMCI+, FCI...) 理论上只能恢复到 Markov 等价类 (MEC)，即图的全部分离/连接语句集合。作者认为评估应该直接比较分离语句本身。
2. **s/c-metric (穷举法)**: 枚举所有可能的三元组 (X,Y,S)（S 为分离集），对每个三元组比较两图中是否分离/连接，取加权平均的不一致率。理论上是 MEC 集合上的严格数学度量 (metric)，但三元组数量随节点数指数增长，仅适合极小图或需 Monte-Carlo 采样近似。
3. **Separation Distance (SD, 可扩展法)**: 借用 Henckel et al. (gadjid, AID) 的思路——不穷举，而是按一个"分离策略" 𝔖 为 H 中每对不相邻节点 (X,Y) 选定唯一一个分离集 𝔖_H(X,Y)，再验证这个分离集在 G 中是否仍然分离 (X,Y)；不满足则罚 1。归一化后得到不对称距离 d^𝔖(G,H)。
4. **具体分离策略**: parent-SD/ancestor-SD (仅 DAG 有效)、p-parent-SD (基于 possible parent，对 CPDAG 有效)、ZL-SD (基于 van der Zander & Liśkiewicz 2020 的线性时间最小分离集算法，对 MAG/PAG 有效，本文证明该分离集是"极小且最近"两个性质唯一同时满足的分离集，并且在 Markov 等价图间不变)。
5. **MB (Markov blanket) 增强**: 对任意策略 𝔖，用节点 X 的 Markov blanket 替代 𝔖(X,Y) 作为默认分离集 (只有 Y 恰好落在 MB(X) 内的"例外"情形才退回原策略)，借助 Bayes-Ball 算法把验证步骤从对 (X,Y) 的双重循环降为对 X 的单重循环，在稀疏 DAG/CPDAG 上把复杂度从 O(N⁴) 降到 O(N²)，与 gadjid 的 AID 复杂度打平。
6. **合成实验**: Erdős–Rényi 图上做单次/多次边删除+反转扰动、以及不同密度下距离间相关性分析；不涉及任何真实因果发现算法在真实数据集上的评测。

### 关键技术细节

- **记号**: 因果图类 𝔾 (DAG/MAG/CPDAG/PAG)，节点集 V，|V|=N；三元组集合 𝒞 = {(X,Y,S): X≠Y∈V, S⊆V\{X,Y}}，按 |S|=k 分层为 𝒞^k。
- **s/c-metric 定义**: d_{s/c}^k(G,H) = (1/|𝒞^k|) Σ_{(X,Y,S)∈𝒞^k} |ι_G(X,Y,S) − ι_H(X,Y,S)|，其中 ι 是分离指示函数 (1=连接, 0=分离)；d_{s/c}^{≤K} 是 k=0..K 的平均；K=N-2 时为 full s/c-metric。附录证明其嵌入到一个赋范向量空间，因此是 MEC 集合上真正意义的数学度量 (三角不等式成立)。
- **𝔖-SD 定义**: d^𝔖(G,H) = (1/N(N-1)) Σ_{(X,Y)∈nadj(H)} ι^𝔖_{G,H}(X,Y)，与 AID 一样不对称；对称化用两方向的算术平均 (正文提"geometric mean"但给出的公式实为算术平均，是原文的一处笔误，精读时发现)。
- **ZL-separator 关键定理**: ZL_G(X,Y) 是唯一同时满足"极小"与"最近 (nearest，本文给出的新刻画)"两性质的分离集，且对 Markov 等价的 MAG 不变 (ZL_G = ZL_{G'})——这是 ZL-SD 能在 PAG/MAG 的 MEC 上良定义的理论基础，不是简单套用已有算法。
- **反例 (附录 Fig)**: 给出一个 5 节点 MAG，其中 parent-separation、ancestor-separation、possible-parent-separation 全部失效 (都不能真正分离一对不相邻节点)——证明这三种在 DAG 上有效的策略不能直接推广到一般 MAG，这正是引入 ZL-separation 的必要性证据而非便利选择。
- **复杂度表** (Appendix ALGORITHMS): 分离集计算 O(N³) 一般 / O(N²) 稀疏 (parent/ancestor/p-parent)，ZL 为 O(N⁴) 一般 / O(N³) 稀疏；验证步骤朴素实现 O(N⁴) 一般 / O(N²) 稀疏，MB-enhanced + Bayes-Ball 可降到 O(N²) (稀疏 DAG/CPDAG)，比朴素实现快一个数量级但仍比 AID 慢一档 (除非用 MB-enhanced)。
- **实测耗时**: N=1000 节点、约 10N 条边的 Erdős–Rényi 图，100 对节点，MB-enhanced parent-SD 平均 0.59s，非增强版 4.35s (Apple M1 Max 64GB)。
- **定理 (SD=0 的含义)**: 若 d^𝔖(G,H)=0（对某个分离策略），则 (i) sk(G)⊆sk(H)（骨架子集关系）、(ii) H 中的 unshielded collider 在两图共有的邻接三元组中也是 G 的 collider、(iii) 共享的 discriminating path 上 collider 归属一致；且对称化距离 d_sym^𝔖(G,H)=0 当且仅当 G,H 严格 Markov 等价——这是一个干净的完备性保证。
- **代码**: 两个 GitHub 仓库，`JonasChoice/sep_distances` (方法实现) 与 `JonasChoice/SDs_experiments` (论文实验复现)。
- **附录提到的关键工具事实**: d-separation/m-separation 的 oracle 在 Python 包 **Tigramite** 中已经实现，且明确支持 MAG、tsMAG、tsDAG (即 PCMCI+/LPCMCI 使用的时间序列因果图)，不局限于静态 DAG。

### 核心结果

- **Example 1 (SHD 失灵的直接证据)**: 两个仅相差一条边方向的 DAG，SHD 几乎为 0；但其中一个图任意两节点间因果路径全通，另一个图因为一个 collider 把节点集切成互相 d-分离的左右两半——SHD 完全看不出这种"整体连通性坍塌"。
- **Example 2 (AID 对 MEC 不一致的反例)**: 两条变量顺序完全不同的因果链 (共享 0 条边，SID/parent-AID 在 DAG 层面拉满)，但它们的 CPDAG 都是无向链、没有可辨识因果效应，导致 CPDAG-level parent-AID = 0——证明"MEC 间 AID"不被"其成员间 AID 的 min/max"所界定，是 AID 推广到 MEC 时的一个真实不一致，而 SD 因为按定义就是 MEC 不变量所以不受此困扰。
- **合成实验**: ZL-SD 对边删除/反转最敏感；MB-enhanced parent-SD 的敏感度模式与 parent-AID 高度相似；但在稠密混合图 (mixed graph) 下 ZL-SD 反而可能低于 SHD 差异 (因为几乎没有节点对还能被分离)。
- **反直觉发现 (相关性实验，附录)**: 在密度 p∈[0.25, 0.85] 区间，parent-SD 与 SHD 出现**负相关**——不只是"测的东西不同"，而是在这个密度区间里两个指标的排序会系统性反着来。
- **SD 与完整 s-metric 的相关性**: 稀疏图上接近 1，但随图密度增加而明显下降，说明"只选一个分离集"的近似质量在稠密图上会打折扣。

### 明确局限

- **无真实数据集实验**: 全文没有在任何真实因果发现算法输出 (如 Sachs 数据集) 上跑评测；结论段明确把"across a broad range of measures and data sources" 的系统性重新评测列为未来工作。
- **SD 是有损代理**: 只选一个分离集验证，必然漏掉其他分离集能揭示的分离/连接差异；与完整 s-metric 的相关性随密度上升而下降 (见上)。
- **不对称性 + 对称化不精细**: 和 AID 一样默认不对称；对称化只是简单平均两个方向，作者自己也提到调和平均等其他方式"也可以"，没有给出选择依据。
- **无法区分同一 MEC 内的成员**: 这是按设计的取舍——若某方法输出比 MEC 更细 (如 LiNGAM 的全定向 DAG)，SD 会丢失这部分信息，必须配合 SHD/AID 使用。
- **MB-enhanced 的 O(N²) 保证是 DAG/CPDAG 专属**: 推广到一般 MAG 需要额外假设 (collider path 长度有界)，论文里没有对这一假设做实证检验。
- **Full s/c-metric 指数级**: 只有 Monte-Carlo 采样一条路径可行，其采样量-精度权衡只有实证曲线 (N=10)，没有理论误差界。
- **全程假设"分离 oracle 完美"**: ι_G(X,Y,S) 全文都当作精确的图论计算或已知 ground-truth 图，完全没有讨论分离关系本身需要从有限、含噪数据估计时的误差——这恰好是 causal-scs-indicator 课题最关心的场景 (见下)。
- **静态图formalism**: 主文没有显式的时间序列/滑动窗口版本，虽然附录提到 Tigramite 的 oracle 已支持 tsDAG/tsMAG，但本文自己从未在时间序列因果图上做过 SD 实验，这条推广路径是"可信但未验证"。

### 跟我们的关系

**不构成 novelty 撞车**: 本文研究的是"如何比较两个静态因果图的准确度差异 (相对 ground truth)"，属于因果发现算法的**事后评估工具**；`0710-causal-scs` 关心的是"因果效应估计在滑动时间窗口下的**不稳定性本身**能否作为强对流预警信号"，是完全不同的问题 (一个是评估指标论文，一个是 EWS 诊断论文)，两者的核心 claim 没有交集，不需要在 novelty quick-check 中对比。

**可直接用的工具/方法**:
1. **J 通道的结构变化度量可以升级**: idea v2 的 J (突变) 通道明确列出"相邻窗口 DAG edge 变化"作为候选信号之一，目前隐含的实现大概率是朴素边计数/Hamming 式比较。本文 Example 1 恰好是对这种做法的直接警告——SHD 很小时因果连通结构可能整体坍塌，反之 Appendix 相关性实验显示 parent-SD 与 SHD 在 p∈[0.25,0.85] 区间甚至负相关。建议把朴素边计数换成/补充为一个 separation distance (PCMCI+ 输出的 CPDAG-like 结构用 MB-enhanced parent-SD 或 p-parent-SD，若未来换用允许隐变量的方法则用 ZL-SD)，作为更具因果意义的 J 通道统计量。
2. **工具链现成可用**: 本文附录明确指出 **Tigramite** (idea 计划使用的 PCMCI+/LPCMCI/J-PCMCI+ 所在的包) 的 m-separation oracle 已经支持 MAG、tsMAG、tsDAG，意味着把 SD 框架接到 idea 已经选定的工具链上，理论上不需要额外实现一个新的分离 oracle，工程成本较低。
3. **计算预算匹配**: idea 真实数据阶段预算 500–1500 CPU-hours 用于滑窗因果估计；MB-enhanced SD 在稀疏图上是 O(N²)，与 gadjid 的 AID 同阶，加进每对相邻窗口的比较开销可忽略，而全量 s/c-metric (指数级) 应明确避免。
4. **同作者的相邻工作值得补充检索**: Wahl & Runge 另有一篇 "Internal incoherency scores for constraint-based causal discovery algorithms" (arXiv:2502.14719，在本文自己的参考文献列表中)，内容是检验 PCMCI+ 这类约束型方法自己的 CI 检验结果与其输出图隐含分离语句是否一致——这与 causal-scs-indicator 的 "estimator fragility" 框架概念上相邻，可能是 (a) 一个互补信号来源，也可能是 (b) 一个需要排除的混淆解释 (如果窗口间不稳定性其实只是 PCMCI+ 自身已知的内部不一致伪影，而非新发现的 EWS 信号)。目前 landscape 文件未见收录，建议下一轮 deep-lit 精读此文。
5. **不解决 idea 当前最大缺口**: v2 review 明确指出的核心缺口——"尚未证明因果估计器的不稳定性优于同复杂度非因果预测模型的不稳定性"——本文完全没有涉及，它是图比较工具而非"因果 vs 非因果脆弱性可分性"的论证。因此这篇论文只能加固 idea 的某个测量组件 (J 通道的结构距离定义)，不能替代 idea 仍然欠缺的核心机制论证。
