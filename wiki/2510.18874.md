# Retaining by Doing: The Role of On-Policy Data in Mitigating Forgetting (2510.18874)

- **Year**: 2025.10
- **Authors**: Howard Chen, Noam Razin, Karthik R. Narasimhan, Danqi Chen (Princeton Language and Intelligence, Princeton University)
- **Venue**: arXiv (submitted to ICLR 2026 based on style file)
- **Keywords**: catastrophic forgetting, on-policy data, reinforcement learning, supervised fine-tuning, KL divergence, mode-seeking, post-training, continual learning
- **One-liner**: RL 比 SFT 更少遗忘不是因为 KL 正则化或 advantage estimation，而是因为 RL 使用 on-policy 数据；approximately on-policy 数据也能显著减少遗忘。

## 精华提炼

### 管线/方法

1. **实验设定**：用 SFT、Self-SFT、RL（GRPO）三种方法在目标任务上 fine-tune LM，评估目标任务的 gain 和非目标任务的 drop。
   - SFT：用 Llama-3.3-70B-Instruct 生成的 ground truth responses 进行交叉熵训练（fully off-policy）
   - Self-SFT：用初始模型自己生成的正确 responses（从 initial policy 采样，filter 后保留正确样本）
   - RL：使用 GRPO（Group Relative Policy Optimization），每步从当前 policy 采样 group size=5 的 responses，用 verifiable reward（正确=1，错误=0）训练

2. **评估指标**：
   - Target task gain Δ_g：目标任务准确率变化
   - Non-target tasks drop Δ_d：非目标任务的平均准确率下降
   - 非目标任务包括：MATH、WildJailbreak、WildGuardTest，加上非当前目标的其他任务

3. **高斯仿真**：建模 optimal policy 为两个 Gaussian 的混合（old mode = prior knowledge, new mode = target task），对比 forward KL（SFT）和 reverse KL（RL）在 uni-modal 和 bi-modal training policy 下的遗忘模式。

4. **消融验证**：
   - KL regularization 消融：GRPO β=0.05 vs β=0
   - Advantage estimation 消融：GRPO vs REINFORCE
   - On-policy 程度实验：Self-SFT（initial only）→ Iterative-SFT（per-epoch）→ RL（per-step）→ SFT on RL traces

### 关键技术细节

- **模型**：Llama-3.2-1B-Instruct, Llama-3.1-8B-Instruct, Qwen-2.5-1.5B-Instruct, Qwen-2.5-7B-Instruct
- **任务**：
  - IFEval：instruction following，训练集 13,000 / 评测集 1,972
  - MMLU：general knowledge，训练集 12,000 / 评测集 2,042（多选题格式，要求先推理后回答 "The answer is: <option>"）
  - Countdown：arithmetic reasoning，训练集 10,000 / 评测集 1,000（用给定数字通过 +-*/ 构造目标数字）
- **训练配置**：
  - Optimizer: AdamW，cosine scheduler，warmup ratio 0.03
  - LR: 1e-4（1B/1.5B 模型），5e-6（7B/8B 模型）
  - Batch size: 128（IFEval/MMLU），64（Countdown）
  - Epochs: 2
  - Max sequence length: 4096
  - GRPO group size: 5，KL coefficient β=0.05，无 advantage clipping
  - 硬件：最多 8×H100 GPU，bfloat16 mixed precision
- **高斯仿真参数**：
  - Uni-modal setting：π_θ 初始化为 N(-3.2, 1.0)，π* = 0.75·N(-3.0, 1.0) + 0.25·N(3.5, 0.7)，LR=0.05
  - Bi-modal setting：π_θ 初始化为 0.75·N(-3.5, 1.0) + 0.25·N(0.5, 0.7)，π* = 0.75·N(-3.0, 1.0) + 0.25·N(3.5, 0.7)，FKL LR∈{0.15, 0.01}，RKL LR=0.01
  - 每次梯度更新用 1000 个样本，每 100 步评估一次

### 核心结果

**主实验（Figure 2 / gain_drop）**：
- 以 Llama 3.1 8B Instruct 为例（粗略估算自图表）：
  - IFEval: SFT gain ≈25%, drop ≈28%; Self-SFT gain ≈11%, drop ≈8%; GRPO gain ≈18%, drop ≈3%
  - MMLU: SFT gain ≈11%, drop ≈39%; GRPO gain ≈15%, drop ≈-0.2%
  - Countdown: SFT gain ≈26%, drop ≈36%; GRPO gain ≈60%, drop ≈-0.5%
- 一致的 pattern：SFT 要么 gain 高但遗忘严重，要么 gain 低（Self-SFT）；RL 能同时达到高 gain 和低 drop

**REINFORCE vs GRPO（Table 1）**：
- Llama 3.1 8B: REINFORCE gain 17.8%/drop 7.7%，GRPO gain 18.4%/drop 3.4%（IFEval 上 GRPO 更好但 REINFORCE 遗忘也远比 SFT 低）
- Qwen 2.5 7B: REINFORCE gain 5.7%/drop 2.9%，GRPO gain 17.0%/drop 0.2%
- 说明：advantage estimation 主要影响 gain 的大小，而 on-policy data 才是减轻遗忘的关键

**KL regularization 消融（Figure 3）**：
- GRPO β=0 和 β=0.05 在大多数设置下 gain-drop tradeoff 接近
- 唯一例外：Llama family on IFEval，KL regularization 有帮助
- 总体结论：KL regularization 不是 RL 抗遗忘的主要原因

**Approximately on-policy 实验（Figure 5 / raft_gain_drop）**：
- Iterative-SFT（每 epoch 初用当前模型生成数据）能在 IFEval 和 MMLU 上达到接近或超过 SFT 的 gain，同时保持低 drop
- SFT on RL traces（appendix）：用 GRPO 训练过程中产生的 on-policy traces 做 SFT，遗忘也显著低于 fully off-policy SFT

**Surprising/反直觉发现**：
1. **最大反直觉点**：直觉上 mode-seeking 的 reverse KL（RL）应该更快丢掉旧 mode，但实验和仿真都表明在 multi-modal 设定下 reverse KL 反而遗忘更少。这是因为 bi-modal policy 可以保持 old mode 不变而只移动 new mode 去拟合 target。
2. **Forward KL 在 uni-modal 设定下确实遗忘更少**——与直觉一致，但在真实 LM（multi-modal）场景下结论反转。
3. **KL divergence from initial policy 与 forgetting 的 Pearson correlation 仅 0.52**（appendix Table 2），且 Self-SFT vs SFT 之间关系非单调——大 KL 不一定意味更严重的遗忘。
4. **RL 并非对遗忘免疫**：当 target distribution 距离 old mode 很远时（simulation_diff_dist），RL 也会开始遗忘（appendix 高斯仿真）。

### 明确局限

- **规模限制**：最大只到 8B 参数模型。作者明确说"beyond our compute budget"的更大规模实验是未来工作。
- **理论不足**：虽然通过高斯仿真提供了 intuition，但缺少关于 on-policy data 如何减轻遗忘的严格理论证明。作者明确把"theoretically establish the role of on-policy data"列为未来方向。
- **任务覆盖有限**：只测试了 instruction following、general knowledge、arithmetic reasoning 三类任务。代码生成、多语言、长文本等其他能力未测。
- **训练方法局限**：RL 方法只用 GRPO 和 REINFORCE；SFT 只用标准的交叉熵。未测试 DPO、KTO 等偏好优化方法。
- **遗忘度量单一**：只用 accuracy drop 度量遗忘。未考察 generation quality、diversity、calibration 等更细粒度的能力变化。
- **未做真机/部署评估**：这是纯离线评测，没有在真实部署环境中验证。
- **与 concurrent work 有分歧**：Lai et al. (2025) 认为是 advantage estimator 减轻遗忘，Shenfeld et al. (2025) 认为是 KL divergence 隐式正则化。本文驳斥了这两种说法并给出了自己的解释（on-policy data），但三篇 concurrent work 的结论不完全一致。

## Read by: 0616-fex (2026-06-22)

本文与 0616-fex 课题（推测涉及 feature extraction / representation learning）的方向不同——本文完全在 LM post-training 的 forgetting 问题上，不涉及 representation 层面的分析。但它提供了以下对课题有用的参考：

1. **实验范式**：gain-drop tradeoff 的量化框架可以直接复用来评估任何 fine-tuning 方法的 stability-plasticity 特性。如果 0616-fex 涉及 LM fine-tuning 场景，可以用同一套 metrics（IFEval/MMLU/Countdown 作为目标任务，MATH/safety tasks 作为非目标任务）。

2. **On-policy 原理的可迁移性**：本文的核心发现——用当前模型自己生成的数据训练能更好地保留已有能力——可能与 feature extraction 中的某些 principle 相通。例如，如果用当前 feature extractor 生成的 representations 来做 self-distillation，可能比用 external teacher 生成的 representations 更少破坏原有 feature space。

3. **高斯混合建模思路**：用 mixture of distributions 建模 multi-modal knowledge 的方法可能可以迁移到 feature space 的 multi-modal structure 分析。

4. **不撞车**：本文的核心贡献（on-policy data mitigates forgetting in LM post-training）与 feature extraction 研究没有直接重叠。如果课题需要在论文中讨论 fine-tuning 导致的 feature degradation，本文可以作为实证支撑引用。
