# Logit Dynamics in Softmax Policy Gradient Methods (2506.12912)

- **Year**: 2025.06
- **Authors**: Yingru Li (independent, no institutional affiliation)
- **Venue**: arXiv preprint
- **Keywords**: Softmax Policy Gradient, Logit Dynamics, Collision Probability, Rényi Entropy, Self-Regulation, Update Magnitude, Exploration-Exploitation
- **One-liner**: 推导 softmax 策略梯度中 logit 更新向量 L2 范数的解析公式,揭示策略置信度通过碰撞概率自动调节学习强度的自调控机制。

## 精华提炼

### 管线/方法

本文是一个纯数学分析工作,没有实验管线。分析流程为:

1. **输入**: 策略梯度定理框架 — 参数化策略 π_θ, 优势函数 A(s,a), 学习率 η
2. **软最大化策略**: P_i = e^{z_i} / Σ_k e^{z_k}, 其中 z_i 为 logit (logit)
3. **分数函数推导**: ∂log π(a_i)/∂z_j = δ_ij - P_j (Kronecker delta 减去动作概率)
4. **Logit 更新公式导出**:
   - 选中动作: Δz_c = η(1-P_c)A
   - 其他动作: Δz_o = -ηP_o A
5. **守恒性质证明**: Σ_j Δz_j = 0, 即 logit 更新总和为零,概率重新分配而非均匀偏移
6. **更新量级公式导出**: ∥Δz∥₂ = η|A|√(1-2P_c + C(P)), 其中 C(P)=Σ_a P_a² 为碰撞概率
7. **碰撞概率与熵的关系**: 附录中证明 C(P) = e^{-H₂(P)}, H₂(P) ≤ H(P), C(P) ≥ e^{-H(P)}

### 关键技术细节

- **核心数学工具**: 策略梯度定理 (Sutton et al. 1999), 链式法则分解参数梯度到 logit 梯度
- **碰撞概率 C(P)** = Σ_a P_a²: 两个独立采样动作相同的概率, 范围 [1/n, 1]
  - Uniform 分布: C(P) = 1/n (最小值, 最大熵)
  - Deterministic 分布: C(P) = 1 (最大值, 最小熵)
- **Rényi 熵 H₂(P)** = -log C(P): 与碰撞概率的直接对数关系
- **熵排序定理**: H₂(P) ≤ H(P), 即二阶 Rényi 熵是 Shannon 熵的下界
- **敏感度因子**: √(1-2P_c + C(P))
  - 探索时 (P_c ≈ 0): ≈ √(1+C(P)), 可高达 √2·η|A|
  - 利用时 (P_c ≈ 1): ≈ 0, 因为 C(P) ≈ 1
- **动作数 n**: 文中以 4 动作系统为例做数值演示,但公式适用于任意离散动作空间
- **无任何超参数调优**, 无训练数据, 无模型架构 — 纯解析推导

### 核心结果

1. **Logit 更新 L2 范数解析公式**:
   ∥Δz∥₂ = η|A|√(1 - 2P_c + C(P))
   这是论文的核心贡献,将更新强度分解为动作特定项 (1-2P_c) 和分布整体项 C(P)

2. **自调控学习机制**: 策略置信度自动调节学习强度:
   - 高置信度策略 (P_c ≈ 1) → 极小更新 → 稳定性
   - 低置信度策略 (P_c ≈ 0) → 极大更新 (可达 η|A|√2) → 快速适应
   - 这种机制实现了从探索到利用的自然过渡

3. **碰撞概率作为学习状态变量**: C(P) 同时捕捉策略集中度和更新灵敏度
   - C(P) 高 → 策略集中 → 探索时更新大(但探索本身少)
   - C(P) 低 → 策略分散 → 更新中等(但探索频繁)

4. **数值演示 (4 动作系统)**:
   | 分布 | H(P) | C(P) | ∥Δz∥₂/η|A| |
   |------|------|------|-------------|
   | [.25,.25,.25,.25] | 1.386 | 0.25 | 0.866 |
   | [.8,.1,.05,.05] | 0.722 | 0.655 | 0.245 |
   | [1,0,0,0] | 0 | 1.0 | 0.0 |

5. **Surprising 发现**: 低熵策略在被迫探索时反而产生最大更新 (∥Δz∥₂ → √2),但这很少发生;而高熵策略的更新总是中等但频繁。这构成了自然的 adaptive curriculum。
