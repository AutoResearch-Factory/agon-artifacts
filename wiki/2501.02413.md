# Semantic foundations of equality saturation (2501.02413)

- **Year**: 2025.01
- **Authors**: Dan Suciu (UW), Yisu Remy Wang (UCLA), Yihong Zhang (UW)
- **Venue**: ICDT (LIPIcs)
- **Keywords**: equality saturation, E-graph, tree automata, the chase, termination, partial congruence, weak term acyclicity, TGD, EGD
- **One-liner**: 用 tree automata 和 chase 理论为 EqSat 建立形式语义，证明三种终止问题的复杂度（R.E.-complete / Π2-complete / undecidable）。

## 精华提炼

### 管线/方法

1. **E-graph 的形式定义** (Def ~\ref{def:egraph})：E-graph = 可达的(reachable)、确定性的(deterministic) bottom-up tree automaton，无 final states。
   - State = E-class
   - Transition = E-node: f(c1,...,ck) → c
   - 树自动机的决定性与可达性正好对应 E-graph 的必然属性

2. **PCR 语义** (~\ref{def:PCR})：E-graph G 的语义是一个 Partial Congruence Relation ≈_G：t1≈t2 当存在某个 E-class c 同时接受 t1 和 t2。证明了任意 PCR 都对应唯一的 E-graph。

3. **EqSat 操作的形式化**：
   - **E-matching**：对规则 lhs→rhs，找出所有 (σ, c) 使得 lhs[σ] →*_G c
   - **Insertion**：插入 (t, c) = G ∪ FL(t →* c)，其中 FL 是 flattening/normalization
   - **Rebuilding** (CC)：将非确定性树自动机 A 转成确定性 E-graph CC(A)
   - **Least Upper Bound**：任意 E-graph 族有最小上界 ⊔

4. **ICO 和最小不动点** (Eq.~\ref{eq:def:ico})：
   - ICO_R(G) = CC ∘ T_R(G)
   - T_R(G) = G ∪ ∪{FL(rhs[σ] →*_G c) | ...}
   - EqSat(R,G) = ⊔_{i≥0} ICO_R^{(i)}(G)
   - 证明 EqSat(R,G) 是 R,G 的 universal model

5. **EqSat ⇔ Chase 的双向归约**：
   - **Skolem Chase ⇒ EqSat** (Thm ~\ref{thm:skolem-to-eqsat})：用 ⊤, ∧ 编码关系原子，Skolem 函数自然对应 term
   - **EqSat ⇒ Standard Chase** (Thm ~\ref{thm:eqsat-to-chase})：E-class → marked null，R_f(c1,...,cn, c) 编码 E-node，match/apply → TGD，FD → EGD（确定性约束）

### 关键技术细节

- **同态唯一性** (Lem ~\ref{lem:unique-morphism})：任意两个 E-graph 之间至多存在一个同态。这意味着 E-graph 是 rigid tree automata，且 ⊑ 构成偏序（同构意义下）。
- **Finite Convergence Lemma** (Lem ~\ref{lemma:convergence})：若有限 E-graph 的上升链 G1⊏G2⊏... 的极限 G∞ 是有限的，则该链本身必有限。证明用 E-class rank 的精细计数论证。
- **Representation Lemma** (Lem ~\ref{lemma:representation})：R*(w) ⊆ [w]_≈_H ⊆ [w]_{≈_R ∨ ≈_G}。对 variable-preserving R，取 Sym(R)=R∪R^{-1} 可得等号。
- **EGD-fairness** (Def ~\ref{dfn:egd-fairness})：chase 序列每步要么终止，要么之后某步 EGD 子集被满足。EqSat 终止 ⇔ 一条 chase 序列终止 ⇔ 所有 EGD-fair chase 序列终止。

### 核心结果

**终止复杂度（三个问题）**：

| 问题 | 复杂度 | 证明方法 |
|------|--------|---------|
| 单实例终止 T^EqSat_G | R.E.-complete | 从 halting problem 归约（Narendran 编码） |
| 全项实例终止 T^EqSat_∀t | Π2-complete | 从 universal halting problem 归约 |
| 全 E-graph 实例终止 T^EqSat_∀G | undecidable（上界未知） | 从 Post correspondence problem 归约 |

**复杂度对比表**（与 chase 的对比）：

| 问题 | EqSat | Skolem Chase | Standard Chase |
|------|-------|-------------|----------------|
| single-instance | R.E.-complete | R.E.-complete | R.E.-complete |
| all-instance | Π2-complete (T^EqSat_∀t) | R.E.-complete | Π2-complete (T^StdChase_∀,∃)，T^StdChase_∀,∀ 上界 open |

**关键反直觉结果**：T^EqSat_∀t 是 Π2-complete，而 T^SklChase_∀ 是 R.E.-complete，说明 EqSat 的全项终止问题**严格难于** Skolem chase 的全实例终止问题。这是因为 EqSat 的输入是任意的 term（无 ground identity 约束），而 Skolem chase 的输入是数据库实例（ground facts）。

**Weak Term Acyclicity** (Def ~\ref{def:weak-term-acyclicity})：
- 从 classic weak acyclicity (Fagin et al. 2005) 适配而来
- 比"先译成 TGD/EGD 再套用 weak acyclicity"更强（Ex ~\ref{ex:acyclicity-ex1} 展示了不 weak acyclic 的 TGD 翻译但 weakly term acyclic 的原始规则）
- 保证 EqSat 在多项式步数内终止，步数上界为 |Q|^{ar+2}

### 明确局限

1. **纯理论论文**：无实验、无真机运行、无代码。所有结果是定理证明。
2. **无有限性之外的终止研究**：只关心 EqSat 是否终止；不涉及资源限制下的行为。
3. **不处理 extraction**：EqSat 的最后一步（从 E-graph 提取最优程序）完全未涉及。
4. **Chase 归约的单向性**：EqSat ⇒ Standard Chase 存在于 encoding，但反之 Standard Chase ⇒ EqSat 不存在（因 non-determinism 与 determinism 的本质差异）。
5. **Weak term acyclicity 仅为充分条件**：不是必要条件。很多 terminating EqSat 不满足该条件。
6. **All-E-graph-instance 的精确上界未解**：只证明 undecidable，但未知是否 co-R.E. 或更高。
7. **只考虑 ground term**：E-graph 仅接受 ground term，不处理带变量的 pattern 作为输出。
8. **无查询优化场景的具体分析**：虽在 intro 中提到 query optimization 应用，但无对应实验或 case study。

### 跟我们的关系

**对 0616-fex (search complexity in equality saturation) 课题：**

- **不撞车**：该论文是 EqSat 的纯理论基础，不涉及搜索策略、heuristics、搜索空间的组织或剪枝。0616-fex 关注的是如何在 EqSat 中控制 search/exploration，而本文定义了"搜索在做什么"的形式语义。

- **可用工具/方法**：
  1. **ICO 不动点语义**：可以用 ICO 框架形式化定义搜索复杂度的度量（如 |ICO^{(i)}(G)| 随 i 的增长曲线）。
  2. **Finite Convergence Lemma**：如果搜索能保证 fixpoint 有限，则迭代必收敛——这对设计有界搜索策略有指导意义。
  3. **Weak term acyclicity**：可以扩展为"部分 acyclicity"来在保证终止的前提下限定搜索空间大小。
  4. **Chase 归约**：可以用 database theory 中已知的 chase termination 结果（如 stickiness, shyness 等）来推理 EqSat 搜索空间的有限性。
  5. **Representation Lemma** 中的上界 [w]_≈_H ⊆ [w]_{≈_R ∨ ≈_G}：可为搜索提供上限。

- **该引**：该论文定义了 EqSat 搜索的形式框架，任何关于 EqSat 搜索复杂度的工作都应在 related work 中引用此文作为理论基础。

- **差异化**：我们的工作聚焦于**如何度量、预测和控制 EqSat 的搜索成本**（在给定资源下），而本文只关心**是否终止**。这是两种不同的复杂度关切——他们是 asymptotic/decision problem，我们是 practical/resource-bounded。

## Read by: 0616-fex (2026-06-21)

本篇对 0616-fex 课题的核心价值在于提供了 **EqSat 搜索的形式语义基础**。我们的课题关注 search complexity（搜索过程本身的代价），而本文定义了 EqSat 在做什么的形式描述（ICO 不动点、tree automata 语义），这是我们度量搜索复杂度的起点。

具体可用的几个切入点：
1. **用 ICO 迭代步骤作为搜索复杂度的代理**：|ICO^{(i)}| - |ICO^{(i-1)}| 可衡量每步新增的搜索空间，见 Finite Convergence Lemma.
2. **Weak term acyclicity 的多项式上界**可被扩展为 partial acyclicity —— 对规则的子集施加 acyclicity 条件来 bound 搜索复杂度而非完全终止。
3. **Chase connection**：database theory 中已知的 chase termination subclass（sticky, shy, guarded, linear）可直接翻译为 EqSat 的搜索有界条件。
4. **Representation Lemma** 的上界约束可用于设计搜索剪枝策略：任何被 EqSat 等价的 term 必须落在 [w]_{≈_R ∨ ≈_G} 内。

不撞车的最终判断：本文是 "what is EqSat" 的语义理论，而我们做的是 "how costly is EqSat search" 的度量与控制。两者互补——本文提供定义，我们提供分析该定义的复杂度工具。
