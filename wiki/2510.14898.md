# Convergence of an actor-critic gradient flow for entropy regularised MDPs in general spaces (2510.14898)

- **Year**: 2025.10 (submitted to ICLR 2026)
- **Authors**: Denis Zorba, David Šiška, Lukasz Szpruch (School of Mathematics, University of Edinburgh; Alan Turing Institute; Simtopia)
- **Venue**: ICLR 2026 (conference bib)
- **Keywords**: Reinforcement Learning, Actor-Critic, Entropy Regularisation, Two-Timescale, Policy Mirror Descent, Fisher-Rao Gradient Flow, TD Learning, Global Convergence, Function Approximation, Continuous State-Action Spaces
- **One-liner**: 证明了连续时间 actor-critic 梯度流在熵正则化 MDP 中的稳定性和全局指数收敛，适用于一般 Polish 状态-动作空间和线性函数逼近。

## 精华提炼

### 管线/方法

完整的 actor-critic 连续时间动力学系统：

1. **输入**：MDP (S, A, P, c, γ)，其中 S、A 为 Polish 空间，参考测度 μ ∈ P(A)，正则化参数 τ > 0，初始分布 ρ ∈ P(S)，特征映射 φ: S×A → ℝᴺ。

2. **Critic 更新（TD learning 的连续时间极限）**：
   - 参数化 Q 函数：Q(s,a;θ) = ⟨θ, φ(s,a)⟩
   - 半梯度 (semi-gradient) 流：dθ_t/dt = -η_t · g(θ_t, π_t)
   - 其中 g(θ,π) = ∫(Q(s,a;θ) - T^π Q(s,a;θ)) φ(s,a) d_β^π(da,ds)
   - η_t: [0,∞) → [1,∞) 是 timescale separation 函数，非递减

3. **Actor 更新（Policy Mirror Descent 的连续时间极限）**：
   - 近似 Fisher-Rao 梯度流：∂_t π_t(da|s) = -A(s,a;θ_t) π_t(da|s)
   - 其中 A(s,a;θ) 是近似 soft advantage function
   - A(s,a;θ) = Q(s,a;θ) + τ ln(dπ/dμ)(s,a) - ∫_A (Q + τ ln(dπ/dμ)) π(da|s)

4. **输出**：策略 π_t 和 critic 参数 θ_t 的轨迹，收敛到最优正则化策略 π*。

### 关键技术细节

**模型架构**：
- 线性函数逼近 Q(s,a;θ) = ⟨θ, ϕ(s,a)⟩，θ ∈ ℝᴺ
- 策略类 Π_μ：π(da|s) = exp(f(s,a)) μ(da) / ∫ exp(f(s,a)) μ(da)，其中 f ∈ B_b(S×A)
- 状态-动作占据测度 d_β^π 由 β ∈ P(S×A) 导出

**核心假设**：
1. Q^π_τ-realisability（Assumption 1）：∀π ∈ Π_μ, ∃θ_π s.t. Q^π_τ(s,a) = ⟨θ_π, ϕ(s,a)⟩ ← 关键简化假设，排除函数逼近误差
2. 特征有界性（Assumption 2）：|ϕ(s,a)| ≤ 1 ∀(s,a)
3. 特征协方差正定性（Assumption 3）：λ_β = λ_min(∫ ϕϕᵀ dβ) > 0，且与策略 π 无关

**Loss 函数**：
- MSBE(θ,π) = ½ ∫ (Q(s,a;θ) - T^π Q(s,a;θ))² d_β^π(da,ds)
- Squared loss: L(θ,π;ζ) = ½ ∫ (⟨θ,ϕ⟩ - Q^π_τ)² dζ
- L(·,π;d_β^π) 是 λ_β(1-γ)-强凸的

**关键常数**：
- Γ = λ_β(1-γ)(1-√γ)
- K_t = sup_{s∈S} KL(π_t(·|s) | μ)
- Timescale separation 条件：η₀ > τ/Γ（稳定性）、η₀ > 1/Γ（收敛性）
- 一般 γ 下需要 η_t = η₀ e^{k₁t}，其中 k₁ > τ/2 + α₂

**核心证明技术**：
1. Lyapunov 分析：½η_t⁻¹ d/dt |θ_t|² ≤ -(Γ/2)|θ_t|² + (τ²γ²K_t²)/Γ + |c|²/Γ
2. Grönwall 不等式串联 KL 散度与 critic 参数
3. Duhamel 原理处理近似 Fisher-Rao 流的对数密度
4. Performance difference lemma 做收敛桥梁
5. 通过 Q^π_τ 沿流的时间导数刻画 θ_π_t 的变化率

### 核心结果

| 结果 | 内容 | 条件 |
|------|------|------|
| Theorem (Grönwall-KL) | K_t² ≤ a₁ + a₂∫₀ᵗ e^{-τ(t-r)} K_r² dr | Assumptions 2,3; η₀ > τ/Γ |
| Corollary (KL stability) | KL(π_t(·\|s)\|μ)² ≤ a₁e^{a₂t} | 同上，∀γ∈(0,1) |
| Corollary (θ stability) | \|θ_t\| ≤ r₁e^{r₂t} | 同上，dη_t/dt ≤ αη_t |
| Theorem (Convergence 1) | min_{r∈[0,t]} [V^π_r_τ(ρ) - V^*_τ(ρ)] ≤ τ/(2(1-γ)(1-e^{-τt/2})) · [e^{-τt/2}∫KL(π*\|π₀) + (1/2τ)∫e^{-τ(t-r)/2}\|θ_r-θ_{π_r}\|²dr] | Assumptions 1,2 |
| Theorem (Main) | ∫ e^{-τ(t-r)/2} \|θ_r-θ_{π_r}\|²dr ≤ b₁e^{-τt/2} + b₂∫ e^{-τ(t-r)/2} η_r⁻¹\|dθ_{π_r}/dr\|²dr | Assumptions 1,2,3; η₀ > 1/Γ; τ<1 |
| **Theorem (Global convergence)** | min_{r∈[0,t]} [V^π_r_τ(ρ) - V^*_τ(ρ)] ≤ O(e^{-τt/2}) | 同上; η_t = η₀e^{k₁t} |
| Corollary (Small γ) | KL uniformly bounded, θ uniformly bounded | γ 满足 64γ²/(Γ²-Γτ/η₀) < 1 |
| Corollary (Small γ convergence) | min_{r∈[0,t]} [V^π_r_τ(ρ) - V^*_τ(ρ)] ≤ (τ/(2(1-γ)(1-e^{-τt/2}))) · [e^{-τt/2}∫KL(π*\|π₀) + d₁∫e^{-τ(t-r)/2}η_r⁻¹dr] | 同上, 更宽松的 η_t 条件 |

**Surprising/反直觉的发现**：
- 在一般动作空间中，即使参考测度 μ 有 full support，KL 散度也可能无界（与有限动作空间不同，那里 KL ≤ log|A|）
- Timescale separation 不仅加速收敛，更是**稳定性**的关键：如果 actor 更新太快，KL 散度可能在有限时间内 blow up
- 文中证明需要指数增长的 timescale separation η_t = η₀e^{k₁t} 来保证全局收敛（除非 γ 足够小）
- 正则化的存在使得最优策略有 full support，这让 KL(π*|π₀) 在合理选择 π₀ 时有限——而 unregularised 情形下这几乎不可能

### 明确局限

**作者明确指出的局限**：
1. 只分析了**连续时间**动力学，离散时间对应物仍需严格处理
2. Critic 逼近仅限**线性函数逼近**，实际中使用非线性神经网络
3. 所有积分假定精确计算（特别是 semi-gradient），实际中需从样本估计，引入 Monte Carlo 误差

**审读发现的额外局限**：
4. **无任何数值实验或仿真验证**：纯理论论文，没有任何 empirical validation
5. **Q^π_τ-realisability 是极强假设**：要求线性函数族能精确表示所有可能策略下的 Q 函数，这在实践中几乎不成立
6. **无真机/仿真环境测试**：不涉及任何具体 RL 环境（如 Gym、MuJoCo 等）
7. **无与现有算法的比较**：如 SAC、PPO 等实际使用的 actor-critic 算法
8. **指数增长的 timescale separation 不实用**：η_t = η₀e^{k₁t} 意味着 critic 更新速度指数增长，实际中无法实现
9. **没有讨论 sample complexity**：连续时间分析回避了采样效率问题
10. **半梯度（semi-gradient）而非真梯度**：TD learning 的 semi-gradient 性质带来的偏差在连续时间极限下被抹平，但离散情况下这是核心困难
