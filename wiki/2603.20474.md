# From Data to Laws: Neural Discovery of Conservation Laws Without False Positives (2603.20474)

- **Year**: 2026.03
- **Authors**: Rahul D Ray, BITS Pilani Hyderabad Campus (single author)
- **Venue**: arXiv preprint
- **Keywords**: conservation-law discovery, neural-symbolic pipeline, multi-restart variance minimization, symbolic regression, false-positive elimination, dynamical systems, polynomial Lasso, PySR
- **One-liner**: NGCG decouples dynamics learning from invariant discovery, uses 10-restart variance minimizer + system-specific symbolic extraction + strict constancy gate to achieve DR=1.0/FDR=0.0 on 4 true-law systems and zero false positives on 5 no-law systems.

## 精华提炼

### 管线/方法

NGCG（Neural-Guided Conservation-law Generator）是一个四阶段解耦 pipeline：

**Stage 1: 神经动力学模型（独立训练，冻结）**
- 输入：轨迹数据 `{x_t}` (N条轨迹，每条T步，维度D)
- 训练 MLP f_θ: R^D → R^D，最小化一步预测 MSE
- 架构：2层隐藏层，256单元，Tanh激活，Adam + one-cycle LR + early stopping
- 冻结后仅用于报告 MSE@16 rollout 误差，不参与守恒律发现
- 目的：避免此前方法中 dynamics 学习和 invariant 发现互相干扰的灾难性遗忘

**Stage 2: 多重启方差最小化器（核心发现模块）**
- 小网络 φ_ψ: R^D → R，3层×64单元 Tanh
- Loss = mean(intra-var) / (inter-var + ε) + λ‖ψ‖²，其中 ε=1e-4, λ=1e-4
  - intra-var：每条轨迹内部 φ 的时间方差 → 鼓励守恒
  - inter-var：跨轨迹 φ 均值的方差 → 防止坍塌到常数
- R=10 次独立随机初始化重启，每次训练最多300 epochs（Adam + cosine annealing）
- 选择验证集 constancy = mean(σ_intra / |mean(φ)|) 最低的网络
- 关键创新：多重启解决了 Lotka-Volterra 等系统的局部极小值陷阱

**Stage 3: 系统特定符号提取（四种并行策略）**
- Polynomial Lasso：对 ≤4 次单项式构建协方差矩阵 C = (1/N)Σ P̃_i^T P̃_i，取最小特征值的特征向量作为守恒量系数。凸问题，全局最优。对 Hénon-Heiles 直接恢复精确 Hamiltonian
- Log-Basis Lasso（Lotka-Volterra 专用）：基函数 {x, y, ln(x+ε), ln(y+ε)}，同样用最小特征向量方法。真实不变量在此 span 中
- PDE Explicit Candidates：对 KS/Burgers 直接检查空间均值是否为守恒量
- PySR Fallback（通用）：对所有系统，在 φ 网络输出上运行遗传编程符号回归（50轮，种群15），算子集含 {+,-,×,÷,²,³,√,exp,log,sin,cos}

**Stage 4: 严格验证门 + 多样性过滤器**
- Constancy Gate：测试集 constancy < τ=0.01。真守恒量通常在 1e-6 量级，混沌系统上的伪常数在 0.01-0.1
- Diversity Filter：ρ = std_i(mean_t(C)) / mean_i(std_t(C)) > 10。要求表达式在不同初始条件/参数下取不同值，拒绝平凡常数。在 Lorenz 系统上至关重要——PySR 可产生 constancy 低至 0.00046 但 ρ≈0.16 的虚假表达式

### 关键技术细节

- **数据集**：9个系统（7 ODE + 2 PDE），ODE 500条轨迹×500步，PDE 1000条×500步（64点谱方法降维至mean/variance/skewness 3维）
- **参数变化**：4个有守恒律系统（mass-spring、Lotka-Volterra、coupled springs、Hénon-Heiles）的物理参数在轨迹间随机采样，构造分布偏移测试
- **真值不变量**：
  - mass-spring: E = p²/(2m) + ½kq²
  - Lotka-Volterra: V = δx - γln x + βy - αln y
  - coupled springs: 总能量（二次型）
  - Hénon-Heiles: H = ½(p_x²+p_y²) + ½(x²+y²) + x²y - ⅓y³
- **无守恒律系统**（预期输出 "no law"）：double pendulum, Lorenz, three-body, Burgers, KS
- **对比基线**：HNN, MLP+PySR, SINDy, IRAS
- **噪声鲁棒性**：σ=0.01, 0.05, 0.10 高斯噪声下 DR 保持 1.0
- **样本效率**：polynomial系统50条轨迹足够，Lotka-Volterra需100条
- **运行时间**：45-50秒/系统（单GPU），基线20-27秒

### 核心结果

| System | NGCG F1/Const | Best Baseline F1/Const |
|--------|---------------|------------------------|
| mass_spring | 1.00 / 1e-4 | HNN 1.00 / 2.3e-3 |
| lotka_volterra | 1.00 / 4e-4 | ALL FAILED |
| coupled_springs | 1.00 / 1e-4 | HNN 1.00 / 2.6e-3 |
| henon_heiles | 1.00 / 0.0 (exact!) | HNN 1.00 / 7.3e-4 |
| double_pendulum | 0.00 / — (correct reject) | HNN FDR=0.8 |
| lorenz | 0.00 / — (correct reject) | correct |
| three_body | 0.00 / — (correct reject) | HNN FDR=0.89 |
| burgers | 0.00 / — (correct reject) | SINDy FDR=0.67 |
| ks | 0.00 / — (correct reject) | SINDy FDR=0.67 |

**核心发现**：
1. NGCG 是唯一在 Lotka-Volterra 上成功的方法（对数不变量此前无解）
2. constancy 比最佳基线低 2-3 个数量级
3. 在5个无守恒律系统上零假阳性——HNN 在混沌系统上频繁产生虚假守恒量
4. Poly-Lasso 在 Hénon-Heiles 上直接恢复精确 Hamiltonian（constancy=0.0，机器精度）
5. Diversity Filter 是消除 Lorenz/Burgers/KS 上伪常数（constancy 低至 1e-5 但无跨轨迹变化）的关键
6. 消融实验表明：单组件在多数系统上冗余，但完整组合确保全线鲁棒；Diversity Filter 在 Lorenz 上去掉即产生 FDR=1.0

### 明确局限

**作者自述**：
1. PDE 降维到 3 个空间矩丢弃了细粒度空间结构，仅适用于积分型不变量
2. 发现的表达式不总是最简形式（如 harmonic oscillator 找到三角表达式而非二次能量）
3. 多重启计算开销是单重启的 10 倍
4. Diversity Filter 阈值 τ=10 是经验设置，缺乏统计原理推导
5. 基线只用单种子评测，NGCG 用 3 种子（但声称基线表现稳定）

**审读发现的额外局限**：
1. **单作者学生论文**：来自 BITS Pilani 本科/硕士生，无导师署名，学术监督和审稿质量控制不确定
2. **无真机实验**：全部为合成数值数据（ODE/PDE 数值积分），无真实物理实验数据验证
3. **代码未公开链接**：声称"code and datasets are publicly available"但论文中无具体 URL
4. **写作质量有问题**：多处出现 "new achieves strong performance relative to prior methods" 等语法错误和重复句式，可能由 LLM 辅助写作未仔细校对
5. **对比基线使用不一致**：HNN 不适用于非 Hamiltonian 系统、SINDy/IRAS 不产生预测模型，"—"标记大量缺项，部分比较不公平
6. **参数变化范围窄**：Lotka-Volterra 参数变化范围很小（α∈[0.15,0.35], β∈[0.065,0.085]），真实生态系统的参数范围通常大得多
7. **PDE 降维是潜在的 confounder**：将 64 维 PDE 场降到 3 维可能在降维过程中丢失了真实守恒律的信息
