# Some remarks on gradient dominance and LQR policy optimization (2507.10452)

- **Year**: 2025.07
- **Authors**: E.D. Sontag (Northeastern University, Electrical and Computer Engineering & BioEngineering)
- **Venue**: arXiv; short paper based on a 2025 L4DC keynote
- **Keywords**: gradient dominance, Polyak-Lojasiewicz inequality, continuous-time LQR, policy optimization, input-to-state stability, perturbed gradient flow, linear feedforward neural network, overparameterization
- **One-liner**: 梳理 CT-LQR 的 PLI 缺口、ISS 鲁棒性和线性 NN 过参数化加速。

## 精华提炼

### 管线/方法

1. 论文不是一篇常规算法实验论文，而是 L4DC 2025 keynote 的理论综述式短文。
2. 主线是从一般 loss function 的 gradient flow 出发，解释哪些 gradient dominance 条件足以保证 convergence rate。
3. 输入对象是一个 open domain `D` 上的非凸 loss `L(k)`，例如 stabilizing LQR feedback matrices 的集合。
4. 基础动力系统是连续时间梯度流 `dot{k} = -grad L(k)^T`，学习率为 1。
5. 作者先定义 target set、critical set、strict saddle set，并用 LaSalle、Lojasiewicz gradient theorem 和 center-stable manifold argument 得到几乎处处收敛到 target 的条件。
6. 第一层技术是 classical global Polyak-Lojasiewicz inequality：`||grad L||^2 >= lambda (L-Lmin)`。
7. 若 global PLI 成立，relative loss `l(t)=L(k(t))-Lmin` 指数下降：`l(t) <= exp(-lambda t) l(0)`。
8. 第二层技术是 semiglobal PLI：任意 compact subset 上有自己的 `lambda_C`，但 `lambda_C` 可随集合变大趋近 0。
9. 论文用一维 continuous-time LQR integrator `dot{x}=u, u=-kx` 作为最小反例。
10. 该例中 `L(k)=(1+k^2)/(2k)`，domain 是 stabilizing feedback `k>0`，最优点是 `k=1`。
11. 当 `k -> infinity` 时，gradient 有界而 regret 无界，因此 global PLI 不可能成立。
12. 该例给出 CT-LQR 的典型行为：远离 optimum 时近似线性下降，靠近 optimum 后转为指数下降。
13. 为表达这种 mixed global-linear/local-exponential 行为，作者引入 generalized gradient dominance。
14. 统一形式是 `||grad L(k)|| >= alpha(L(k)-Lmin)`，其中 `alpha` 可以是 positive definite、class K、class K-infinity 或 saturated class。
15. 核心新类别是 saturated PLI：`alpha(r) = sqrt(a r/(b+r))`。
16. saturated PLI 在小 regret 时近似 classical PLI，提供局部指数收敛。
17. saturated PLI 在大 regret 时饱和到常数，提供全局近线性下降。
18. 文中给出关系链：global PLI => saturated PLI => semiglobal PLI => local PLI。
19. 另一个关系链是：K-infinity => class K => positive definite。
20. 第二个大模块是 classical continuous-time LQR policy optimization。
21. LQR loss 是对初始状态分布取期望的 infinite-horizon quadratic cost。
22. 搜索变量是 feedback matrix `K`，domain `D_(A,B)` 是所有使 `A-BK` Hurwitz 的 stabilizing matrices。
23. 论文强调这个 domain 可以是非凸的；二维 slice 的 heat map 图展示了 stabilizing set 的非凸形状。
24. 对 CT-LQR，gradient flow 可写成 `dot{K} = -2(RK-B^T P)Y`。
25. 这里 `P` 和 `Y` 分别由 closed-loop Lyapunov equations 确定。
26. 该 gradient flow 不是简单凸优化，且 domain boundary 与稳定性强相关。
27. 论文对比 discrete-time LQR：DT-LQR 已知满足 global PLI，因此有全局指数收敛。
28. CT-LQR 不满足 global PLI；早期已知结果只能给 semiglobal PLI。
29. 作者引用 Cui-Jiang-Sontag 的结果：一般 CT-LQR loss 满足 saturated PLI。
30. 这解释了 CT-LQR 为什么不能有 uniform exponential rate，但仍有可控的 linear-exponential convergence。
31. 论文还用 Euler discretization 解释 CT/DT 差异：固定步长 `h` 下离散化 loss 的 domain 是 `(0,2/h)`。
32. 对任意固定 `h`，离散化问题可获得 global PLI，但其常数 `lambda_h -> 0` as `h -> 0`。
33. 这说明 DT 的 global PLI 不稳定地依赖离散步长，不能直接传回连续时间极限。
34. 第三个大模块是 perturbed gradient systems。
35. 实际 policy optimization 中，gradient oracle 会受到攻击、仿真误差、早停、stochastic computation、digital twin 不准和有限采样影响。
36. 作者把这些误差建模为 additive input：`dot{k} = -grad L(k)^T + B(k)u(t)`。
37. `B(k)` 被假设为 bounded locally Lipschitz。
38. 分析工具是 input-to-state stability (ISS) 及其 small-input / integral variants。
39. 因为 LQR stabilizing feedback domain 是 open set，ISS 需要用相对 open set 和 target set 的 size function 来表述。
40. 若 `L-Lmin` 是 size function，且 gradient dominance 的 `alpha` 属于 K-infinity，则 perturbed gradient system 是 ISS。
41. 若 `alpha` 属于 class K，则得到 small-input ISS。
42. 若 `alpha` 只是 positive definite，则得到 integral ISS。
43. 因此 saturated PLI 给出 small-input ISS 和 iISS 的组合，文中称为 strong iISS。
44. 对 CT-LQR，Cui-Jiang-Sontag 结果直接推出 perturbed gradient flow strongly iISS。
45. 这比 Sontag 2022 只基于 semiglobal PLI 得到 iISS 的结果更强。
46. 论文还指出 natural gradient flow 和 Gauss-Newton flow 也能得到相近的 ISS-like properties。
47. 第四个大模块是 linear feedforward neural network (LFFNN) feedback controller。
48. 作者把 LQR feedback `u=Kx` 过参数化为矩阵乘积 `K_N ... K_1 x`。
49. 这里激活函数取 identity，因此网络整体仍表示线性 feedback，但参数空间显著扩大。
50. 过参数化问题的 domain 是所有使乘积 feedback stabilizing 的 tuple `K=(K_1,...,K_N)`。
51. 该部分研究对每层矩阵参数做 gradient flow：`dot{K_i}=-grad_{K_i} L(K)`。
52. 已知 well-posedness 结果包括：solution 对所有 `t>=0` 存在、precompact、保持在 stabilizing set、并收敛到 critical points。
53. 在单隐藏层 `N=2` 情况，每个 equilibrium 要么是 global minimum，要么是 strict saddle。
54. 单隐藏层 critical point 的乘积 feedback 对应最优 feedback 的 low-rank approximation。
55. 除零测集初始化外，`K_2 K_1` 会收敛到 optimal feedback `K*`。
56. 论文强调了一个 conserved quantity：相邻层的 Gram imbalance `K_i K_i^T - K_{i+1}^T K_{i+1}` 沿梯度流保持不变。
57. 对每个 imbalance matrix，作者定义 eigenvalue concentration measure `c_i`。
58. 直观结果是 imbalance 越大，convergence 越快，ISS 意义下鲁棒性越强。
59. 对 scalar/vector 简化问题，论文给出显式 phase-plane 图、stable manifold 和 boundary of stabilizing region。
60. scalar `n=m=1, N=2` 情况下，gradient dynamics 可写成 `dot{k_1}=f(k_2 k_1) k_2^T` 与 `dot{k_2}=f(k_2 k_1) k_1^T`。
61. `f(k)` 由 LQR 参数 `a,q,r` 给出，equilibria 是 `f(k_2 k_1)=0` 的 hyperbola。
62. 稳定 manifold `k_1=-k_2^T` 是收敛失败的 measure-zero 集合。
63. Imbalance speed-up theorem 的说法是：两个 solution 若 initial cost 相同但 imbalance `c` 较大，则较大 imbalance 的 loss 在所有 `t>0` 更低。
64. 这说明过参数化不仅改变参数维度，还改变 loss landscape 上 gradient flow 的有效 geometry。
65. 最关键的 claim：CT-LQR 原始 one-matrix policy optimization 没有 global PLI，但过参数化后，在 uniformly imbalanced forward-invariant sets 上可以恢复 global PLI。
66. 对 scalar `q=r=1,a=-1`，作者写出 `||grad L||^2 >= rho min(||k_1||^2+||k_2||^2,1)(L-Lmin)`。
67. 如果 imbalance 下界 `c>=sqrt(mu)` 成立，就得到 `||grad L||^2 >= rho_tilde (L-Lmin)`。
68. 因此在避开 spurious critical points 且 imbalance 有下界的区域，loss 可以全局指数下降。
69. 这也带来 ISS asymptotic gain：足够小的 input perturbation 只会导致足够小的 asymptotic regret。
70. 数值例子使用 `n=5,m=3,N=2,kappa=10`，展示 overparameterized trajectory 比 classical one-matrix solution 更接近指数下降。
71. 图中也展示了 near saddle slow-down：过参数化会带来加速，但也引入额外 saddle geometry。

### 关键技术细节

1. 主 loss domain 是 open subset `D subset R^n`，后续 LQR domain 是 stabilizing feedback set。
2. General CT-LQR system：`dot{x}=Ax+Bu`，feedback `u=-Kx`。
3. LQR cost：`int_0^infty x^T Q x + u^T R u dt`，对 `x_0` 分布取期望。
4. Stabilizing domain：`D_(A,B)={K | A-BK Hurwitz}`。
5. Gradient formula：`dot{K}=-2(RK-B^T P)Y`。
6. `P` solves closed-loop Riccati/Lyapunov equation with `Q+K^T R K` term。
7. `Y` solves closed-loop Lyapunov equation with initial covariance term，文中例子取 identity。
8. One-dimensional integrator example：`L(k)=(1+k^2)/(2k)`，`k_min=1`，`L_min=1`。
9. 该例的 `|grad L(k)|^2` 在 `k->infty` 时趋向 `1/4`，但 regret 发散。
10. saturated comparison function：`alpha(r)=sqrt(a r/(b+r))`。
11. Perturbation model：`dot{k}=-grad L(k)^T + B(k)u(t)`。
12. ISS size function 要在 `k -> partial D` 或 `|k| -> infinity` 时 proper/coercive。
13. LFFNN product form：`u=K_N K_{N-1} ... K_1 x`。
14. 单隐藏层 critical products 是 `K*` 的 low-rank approximations。
15. Critical varieties 的维度写成 `(kappa-r)(n+m)+r^2`，其中 `r=rank(hat{K})`。
16. Imbalance invariant：`K_i(t)K_i(t)^T - K_{i+1}(t)^T K_{i+1}(t) == C_i`。
17. scalar two-layer imbalance measure 简化为 `c=sqrt(|K_1^2-K_2^2|)`。
18. 过参数化恢复 PLI 的关键是 forward-invariant region 中 `||k_1||^2+||k_2||^2` 有正下界。
19. 仿真例子包括 `n=5,m=3,N=2,kappa=10`，比较 classical red curve 和不同 imbalance 的 overparameterized curves。
20. 论文没有给新的 FEX/SR 实验，也没有标准 benchmark；它的贡献是理论关系和已发表/待发表结果的组织。

### 核心结果

1. Global PLI 足以推出 loss 的全局指数收敛。
2. CT-LQR 一般不能满足 global PLI；一维 integrator 已给出反例。
3. CT-LQR 满足 saturated PLI，因此有全局线性到局部指数的 mixed convergence。
4. DT-LQR 满足 global PLI，但 Euler step `h -> 0` 时 PLI constant 退化。
5. Gradient dominance 的强弱直接对应 perturbed gradient system 的 ISS / siISS / iISS。
6. saturated PLI 足以给出 strong iISS，这对 noisy oracle 或 disturbed policy optimization 很重要。
7. CT-LQR perturbed gradient flow 由 saturated PLI 推出 strong iISS。
8. 线性 NN feedback 的过参数化 gradient flow 保持 stabilizing domain，不会从可稳定 feedback 区域跑出去。
9. 单隐藏层 LFFNN-LQR 除零测初始化外收敛到 optimal feedback product。
10. Imbalance 是 conserved quantity，不只是初始化细节。
11. 更大 imbalance 在 scalar/vector 分析中带来严格更快的 loss convergence。
12. 原始 CT-LQR 无 global PLI，但 overparameterized LFFNN 版本可在 uniformly imbalanced invariant sets 上恢复 global PLI。
13. 这提供了一个有趣机制：过参数化并不改变 representable controller class，却改变 optimization geometry。
14. 对 FEX/RL 搜索而言，最可迁移的是“representation 改写可以把非 PLI 的优化问题变成局部或分区 PLI”的思路，而不是具体 LQR 公式。

### 明确局限

1. 论文是短文/讲稿式综述，不是完整证明论文；不少 theorem 需要回到引用的已发表或待发表论文。
2. 多处结果依赖 real-analytic、precompactness、target/critical/strict saddle 分解等条件。
3. CT-LQR 的 saturated PLI 与 FEX 的离散表达式树 RL 搜索没有直接数学等价。
4. ISS 分析使用连续时间 gradient flow；FEX controller 通常是 policy-gradient over discrete grammar actions，不能直接套用。
5. LFFNN 部分的强理论主要在 `N=2`、scalar/vector 简化或 uniformly imbalanced invariant region 中。
6. `n=5,m=3,kappa=10` 是数值展示，不是大规模 benchmark。
7. 论文没有公开代码或复现实验脚本；source 只有 tex 和 figures。
8. 对 stochastic gradient、finite-sample oracle 和真实 model-free rollout 的 sample complexity 只作为动机出现，没有完整定理。
9. 对 spurious critical points 的处理依赖避开 measure-zero stable manifolds 或选定 invariant sets。
10. 对 FEX 来说，本文不能作为“RL 表达式搜索收敛”的 prior；只能作为 gradient dominance / perturbation robustness / overparameterization geometry 的理论背景。

## Read by: 0616-fex (2026-06-19)

从 `0616-fex` 视角看，这篇是理论背景，不是直接竞品。它完全不处理 FEX 的有限表达式空间、PDE residual reward、symbolic regression benchmark、expression-tree RL controller 或 operator pool 设计；因此不会撞到 FEX 的方法 claim。

可用价值主要在三个地方。第一，global PLI / saturated PLI / semiglobal PLI 的分层非常适合提醒我们：FEX 搜索复杂度如果要写 gradient-dominance 条件，不能笼统说“满足 PLI”，必须说明是全局、局部、半全局，还是 saturated comparison function。第二，perturbed gradient + ISS 框架可作为 `fex-active-collocation` 和 noisy residual reward 的理论类比：collocation set 变化、finite-sample residual、early stopping 和 candidate evaluation error 都是 oracle perturbation。第三，过参数化 LFFNN 恢复 PLI 的故事对 FEX 表示学习有启发：LLM skeleton、dimension-lift skeleton、e-graph quotient 或初始化先验也可能不是简单“加速搜索”，而是改变 reward landscape 的有效 geometry。

但引用时必须严格限定范围。本文的定理是连续时间 gradient flow 和 LQR control 的结果，不能直接推出离散 grammar action policy gradient 的 sample complexity 或 convergence rate。对 `fex-search-complexity`，它适合作为 conditional PG theorem 的背景和反例来源，而不是 prior work preempting FEX-specific quotient-count / sampler lower bound / coefficient feasibility analysis。建议列入 theoretical background，不放入直接竞品矩阵。
