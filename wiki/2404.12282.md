# Investigating Guiding Information for Adaptive Collocation Point Sampling in PINNs (2404.12282)

- **Year**: 2024.04
- **Authors**: Jose Florido (U. Leeds), He Wang (UCL), Amirul Khan (U. Leeds), Peter K. Jimack (U. Leeds)
- **Venue**: arXiv / LNCS format (likely conference proceedings, uses llncs.cls)
- **Keywords**: PINNs, adaptive collocation points, resampling, PDE residual derivatives, solution geometry, Burgers' equation, Allen-Cahn equation
- **One-liner**: Proposes using mixed second derivatives of the PDE residual (PDE_xt) and solution estimate (U_xt) as guiding information for adaptive collocation point resampling in PINNs, showing PDE_xt consistently outperforms residual-only methods, especially for complex problems with sharp features.

## 精华提炼

### 管线/方法

**整体流程**：
1. **输入**: PDE 定义（Burgers' 或 Allen-Cahn）+ 边界/初始条件 + 超参数 N（配点数）
2. **网络架构**: 简单前馈 NN，3 个中间层各 64 节点，tanh 激活
3. **初始训练**: ADAM 优化器（lr=0.001）训练 15,000 步 → L-BFGS 1,000 步
4. **自适应重采样循环**（重复至指定次数）:
   - 在细网格候选点集 X 上评估信息源 Y（例如 PDE_xt）
   - 构建概率密度函数: P(X) = Y(X)^k / mean(Y(X)^k) + c
   - 归一化后按概率从 X 中抽取 N 个配点
   - 继续训练: ADAM 1,000 步 + L-BFGS 1,000 步
5. **输出**: 预测解 u(x,t)，用细网格上的 L^2 相对误差评估

**硬边界条件**: 通过输出变换强加 BC/IC，使损失函数仅包含 L_PDE 项，排除 L_data 和 L_BCs。

**比较的 6 种采样策略**:
- Fixed, R: 固定均匀随机分布
- Fixed, H: 固定 Hammersley 拟随机分布
- PDE, R: 基于局部残差的 adaptive，随机初始化
- PDE, H: 基于局部残差的 adaptive，Hammersley 初始化
- U_xt: 基于解估计的混合二阶偏导 ∂²u/∂x∂t
- PDE_xt: 基于残差的混合二阶偏导 ∂²(PDE)/∂x∂t

### 关键技术细节

- **网络**: 3×64 全连接，tanh 激活，所有权重可训练
- **优化器**: ADAM (lr=0.001) 用于初始化和重采样间的粗调；L-BFGS 用于精调
- **训练调度**: 15,000 ADAM → 1,000 L-BFGS → 每 2,000 步 (1,000 ADAM + 1,000 L-BFGS) 后重采样
- **重采样参数**: k 和 c 为常数超参数（未报告具体值，继承自 Wu et al. 2023）
- **评估指标**: L^2 相对误差 = sqrt(Σ(u_i - u_gt,i)²) / sqrt(Σ u_gt,i²)，在远多于 N 的细背景网格上计算
- **重复实验**: 每种配置至少 20 次不同随机种子，报告平均误差
- **PDE 测试案例**:
  - Burgers' Equation: u·u_x + u_t = ν·u_xx, x∈[-1,1], t∈[0,1]
  - 默认 ν = 1/(100π) ≈ 0.00318
  - 变体: ν = 0.01 (平滑), ν = 0.001 (锐激波)
  - 5 种不同初始条件
  - Allen-Cahn Equation: u_t = D·u_xx + 5(u-u³), D=0.001

### 核心结果

**主要发现**:
1. **PDE_xt 几乎总是最优**: 在 Burgers' 方程的大多数 IC 和 ν 变体中，PDE_xt 给出最低 L^2 误差。对最锐激波 (ν=0.001, N=2000)，仅 PDE_xt 达到 <15% 误差 (14.25%)，而其他方法 >37%。
2. **自适应方法节省配点**: 从 N=2000 减半到 1000 时，PDE_xt 误差仅增加 19.3%，而固定 Hammersley 增加 338.4%。
3. **存在临界 N**: 对每个问题存在一个 N 阈值——高于此值，固定 Hammersley 分布足够且更简单；低于此值，自适应方法有显著优势。Allen-Cahn N=500 时自适应明显优于固定，N≥1000 时固定 Hammersley 反而更好。
4. **初始分布重要**: Hammersley 初始化一致优于随机初始化，无论是固定还是自适应方法。
5. **问题依赖性强**: IC3 案例所有方法均表现不佳，Hammersley 固定分布反而是"最不差"的——说明 PINN 训练难度高度依赖问题本身。

**Surprising 结果**:
- U_xt（解曲率）作为信息源的效果不稳定，远不如 PDE_xt，说明残差的高阶导数比解的高阶导数更富含信息。
- 对 Allen-Cahn N=500，自适应方法显著优于固定，但 N=1000/2000 时差异消失——说明问题复杂度与配点数的交互存在阈值效应。
- 对 ν=0.01 的平滑 Burgers，所有自适应方法行为相似且固定 Hammersley 表现最好——可能因为 L-BFGS 步数更多（固定方法多跑 L-BFGS）。

### 明确局限

**作者自述的局限**:
- 仅测试了 2 个 benchmark PDE (Burgers', Allen-Cahn)，均为 1D 时间依赖问题
- PINN 的计算成本不具竞争力（相比传统离散化方法），主要是 L-BFGS 太贵
- 未充分探索不同训练调度的影响；cosine annealing 等方法可能改善效率
- 未探索网络架构变化的影响

**审读者观察到的局限**:
- **无理论分析**: 论文完全是经验性的，没有解释为什么 PDE_xt 比残差本身更好。没有信息论或学习理论的分析。
- **仅 1D 空间**: 所有实验都是 (x,t) 二维域，没有扩展到 2D/3D 空间问题
- **仅前向问题**: 没有测试逆问题或含噪声数据的情况
- **固定网络架构**: 只用了 3×64 的简单网络，没有测试架构敏感性
- **硬 BC 仅需**: 未比较弱 BC 与硬 BC 在自适应采样下的交互
- **无开源代码**: 复现需要从零实现全部方法
- **重采样成本**: 虽然引用 Wu et al. 说重采样仅占 3% 开销，但未提供 PDE_xt 计算的具体成本（计算二阶混合导数比一阶代价高得多）
- **k, c 超参数**: 未报告具体值，这些对 PDF 形状有显著影响
