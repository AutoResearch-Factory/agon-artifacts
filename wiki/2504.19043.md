# MiniMax Learning of Interpretable Factored Stochastic Policies from Conjoint Data, with Uncertainty Quantification (2504.19043)

- **Year**: 2025.04
- **Authors**: Connor T. Jerzak (UT Austin, Government), Priyanshi Chandra (USI Lugano), Rishi Hazra (Harvard, Statistics & Economics)
- **Venue**: ICML 2026 (accepted)
- **Keywords**: Conjoint Analysis, Offline Policy Learning, Stochastic Interventions, Factored Policies, Minimax Optimization, Strategic Candidate Selection, Uncertainty Quantification, Delta Method, Multi-Agent Reinforcement Learning, Combinatorial Action Spaces
- **One-liner**: 从联合调查实验数据中学习可解释的因子化随机策略，扩展到制度感知的 minimax 博弈，并提供 Delta-method 不确定性量化。

## 精华提炼

### 管线/方法

整体 pipeline：
1. **输入**：forced-choice conjoint 数据 $(C_i, \bT_i^A, \bT_i^B)_{i=1}^n$（受访者从两个随机生成的候选人 profile 中强制选择）
2. **Step 1 — 结果建模 (Outcome Model)**：用 Bernoulli GLM（logistic link, main effects + pairwise interactions）或 Bayesian Transformer 神经网络拟合 $C_i \mid (\bT_i^A, \bT_i^B)$。GLM 中采用 sum-to-zero 约束保证 identifiability；强制选择反对称条件 $\eta(\bt,\bu)=-\eta(\bu,\bt)$ 确保相同 profile 胜率 1/2
3. **Step 2 — 策略优化**：
   - **Average-case**：固定对手分布（如 uniform），最大化期望胜率 + L2 正则化（shrink toward logging policy $\bp$）。在 two-way interaction linear-probability 近似下有闭式解（Proposition 1）；更一般地用 logit-parameterized gradient ascent 保证 simplex 约束
   - **Adversarial (minimax)**：两玩家零和博弈，同时优化 $\max_{\bpi^A}\min_{\bpi^B} Q(\bpi^A,\bpi^B)$。嵌入制度约束（初选 + 大选两阶段），通过 gradient ascent-descent 求解 restricted stationary point
4. **Step 3 — 不确定性量化 (UQ)**：Delta method 传播 outcome model 的渐近协方差到 $\hat{\bpi}^*$ 和 $\widehat{Q}(\hat{\bpi}^*)$。Jacobian 通过 implicit differentiation（对收敛解求 KKT 条件导数）或 unrolling 整个优化 trace 的 autodiff 获得
5. **输出**：每个 factor 的 Categorical 概率 $\hat{\bpi}^*$（可解释的策略）、期望胜率 $\widehat{Q}$、标准误、策略分歧度量 $\mathcal{D}_\varepsilon(\bt)$

### 关键技术细节

- **策略参数化**：product-of-Categoricals — 每个 factor $d$ 的各 level 有独立概率 $\pi_{dl}$，完整 profile 概率为 $\prod_d \pi_d(t_d)$。保持 attribute-level 可读性，与 conjoint 的随机化分布同构
- **闭式解**（Proposition 1）：在 baseline-coded linear-probability 近似 + two-way interactions + L2 正则化下，最优策略是线性系统 $\bpi^* = \mathbf{C}^{-1}\mathbf{B}$ 的解。矩阵 $\mathbf{C}$ 的 off-diagonal 由交互系数 $\bar\gamma_{dl,d'l'}$ 填充，对角线和同 factor 内 off-diagonal 由 $-\lambda$ 项控制
- **变分近似解释**：KL 正则化下，全单纯形最优策略是 Gibbs 分布 $\sigma^*(\bt) \propto p(\bt)\exp\{u(\bt)/\lambda\}$；限制到 product-of-Categoricals 等价于 mean-field 变分近似（reverse-KL projection）。Theorem 1 给出值函数 gap 的上界：$\frac{1}{8\lambda}\sum_d c_d^2$，其中 $c_d$ 是 per-coordinate bounded-differences 常数
- **训练数据**：US presidential conjoint from Ono & Burden (2019)，forced-choice 设计，受访者按党派分层
- **超参数**：$\lambda$（L2 正则化强度，实验中固定以确保最大 factor 概率 ≤0.9 但与 uniform 有实质偏离）；梯度步数 $S$（默认 10000）；Monte Carlo 样本数（默认 64）
- **实现**：开源 `strategize` 包（https://github.com/cjerzak/strategize-software），JAX 后端，支持 CPU/GPU
- **运行时间**：average-case 6–14 分钟（随 $D$ 和 $n$ 变化），adversarial case 4–5 分钟（RuntimeScaling_macros.tex）
- **Neural outcome model**：共享 Transformer encoder 处理每个 profile → 表示 $\phi_\theta(\bt) \in \mathbb{R}^{64}$ → 标量 utility $u_\theta(\bt)$。Pairwise logit 含 antisymmetrized bilinear interaction term $\omega\{\phi_A^\top M_\times\phi_B - \phi_B^\top M_\times\phi_A\}$，embedding dim=64, depth=8, RMS-norm pre-activation。Bayesian 估计（MCMC 或 variational inference）

### 核心结果

**仿真结果（Average Case）**：
- $\hat{\bpi}^*$ 和 $\widehat{Q}(\hat{\bpi}^*)$ 的 bias 小（$n=500$ 时已可忽略），RMSE 以 variance 为主，随 $n$ 快速下降
- Coverage 在所有 $K \in \{5,10,20\}$ 和 $n$ 下接近 nominal 95%
- Optimal SI 的 $Q$ 均值 1.550 vs Max-AMCE baseline 0.783，在高维（$D=20$）时优势最大
- 模型误设鲁棒性（Table RR）：正确设定时 GLM coverage 0.94 / RMSE 0.007 极优；强非线性误设时 Transformer RMSE 更小但 coverage 偏低

**仿真结果（Adversarial Case）**：
- RMSE 主要由 sample size 决定，对 $p_R$（共和党选民比例）不敏感
- $n=1000$ 时 coverage 低于名义值，$n \ge 5000$ 接近 95%

**实数据结果（US Presidential Conjoint）**：
- Average-case 最优策略：各子群（全样本/民主党/共和党/独立选民）在移民、堕胎、政策优先级上有分歧；人格特质上趋同
- Adversarial restricted-equilibrium：民主党在 average-case 中 deprioritize 移民，但在对抗设定中反击共和党的 guest-worker 立场
- 历史比较（Fig. HistoricalComp）：average-case optimizer（uniform 对手）预测的 vote share 超出 1976 年以来两党历史范围；adversarial restricted-equilibrium 预测与 2016 实际结果接近且在历史范围内 → **对抗设定比平均设定更贴近现实**
- Strategic divergence：民主党候选人整体略高于共和党
- 候选人对数概率（GLM）：Democrats avg case −16.05, adversarial −17.84；Republicans −15.78, −17.14。神经模型下差距更大（−14.92 vs −23.33, −17.29 vs −24.29）

### 明确局限

- **作者自述局限**：两步估计器依赖 accessible variance-covariance matrices；factorized policy class 是可解释近似而非无限制最优；不确定性估计不考虑偏好形成过程；推断策略依赖制度设计知识（可能难以获取）
- **读出的额外局限**：
  - 闭式解仅在 linear-probability 近似 + two-way interactions 下成立，真实选择概率是 logistic 非线性，闭式解只是近似
  - 对抗设定中 restricted factored class 非凸，gradient ascent-descent 只能找 stationary point，不保证 Nash equilibrium；exploitability 诊断只是事后检查
  - L2 正则化没有 KL 正则化那样的变分解释 — mean-field gap bound（Theorem 1）仅适用于 KL，L2 没有对应理论保障
  - 制度建模假设 primary pushforward 是 affine 的（以获得 affine payoff），现实初选机制更复杂（多轮、多候选人、策略投票）
  - 仅在一份 US presidential conjoint 数据上验证（Ono & Burden 2019），缺乏跨数据集/跨文化的泛化证据
  - 没有真机实验 — 是纯离线策略学习，没有在线部署和 A/B 验证
  - Transformer outcome model coverage 偏低（0.43–0.56），说明 Bayesian neural net 的不确定性校准不足
  - 没有与标准 benchmark（如 OpenML 的 conjoint 数据集集合）系统比较

### 撞车评估
**不撞车。** 本文是 conjoint analysis（联合调查实验）+ offline policy learning + minimax game theory 的组合，关注从 forced-choice 偏好数据中学习可解释的策略分布。0616-fex 关注 feature extraction / representation learning，方法论完全不同。

### 可用工具
1. **Delta method + implicit differentiation UQ pipeline**：两步估计器的不确定性传播方法（outcome model fit → downstream optimization → Jacobian via implicit diff），可复用到 fex 中从 learned representation 到 downstream performance 的 uncertainty 传播
2. **Logit-parameterized simplex optimization**：$\alpha_{dl}$ 重参数化保证 simplex 约束的技巧，对 fex 中带约束的分布优化有用
3. **Exploitability decomposition**：restricted-vs-full 博弈论诊断，如果 fex 涉及 adversarial feature learning 可参考
4. **Variational approximation gap analysis**（Theorem 1）：mean-field 近似 gap 的分析方法，对理解 factorized vs joint representation 的 trade-off 有参考价值

### 差异化
完全不同的 problem setting。本文处理的是离散组合动作空间上的策略优化（来自调查数据），fex 处理的是连续表示学习。不存在直接竞争或重叠。
