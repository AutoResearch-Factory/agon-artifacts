# Discovery under Hypothesis Redundancy: A Geometric Theory of Discovery Bottlenecks (2606.14386)

- **Year**: 2026.06
- **Authors**: Li C. Xia / Li Xia (Tsinghua University), Baoxun Wang (Tencent Platform & Content Group)
- **Venue**: arXiv preprint; source uses NeurIPS 2026 preprint style, acceptance not established
- **Keywords**: hypothesis redundancy, spectral compression, effective rank, directed non-local search, residual signal alignment, predictive novelty, LLM-guided factor discovery, symbolic regression
- **One-liner**: 提出 Search Compression Hypothesis：LLM 非局部搜索只有在谱压缩、逃逸距离、残差信号对齐同时成立时才有用。

## 精华提炼

### 管线/方法

1. 论文把“发现”抽象为一个已有假设 archive 上的搜索问题：已有假设的相关矩阵如果谱集中，名义候选很多但独立方向很少。
2. 核心对象是 archive 相关矩阵 $\Sigma_{\mathcal{A}}$ 的 effective rank：$r_{\text{eff}}=\exp(-\sum_i p_i\log p_i)$，其中 $p_i=\lambda_i/\sum_j\lambda_j$。
3. 谱压缩定义为 $r_{\text{eff}}\ll N$ 且最大特征值占比高，表示候选表达式/因子集中在少数方向。
4. 本文提出 Search Compression Hypothesis：非局部探索能帮忙，必须同时满足三项几何条件。
5. 第一项是 compression severity，即 $1-r_{\text{eff}}/N$ 足够大；接近 full rank 时混合搜索优势应消失。
6. 第二项是 escape distance，即新 seed 相对结构化 span 的正交残差 $d_\perp=\|(I-P_{\mathcal{S}})z\|_2$ 足够大。
7. 第三项是 residual signal alignment (RSA)，即逃逸出来的分量不仅正交，而且和目标信号方向对齐。
8. RSA 定义为 $\langle f_\perp,y\rangle/(\|f_\perp\|\|y\|)$；实证估计用 PredNovelty，即残差因子的 ICIR。
9. 整体 discovery operator 写作 $\mathcal{T}_{verify}\circ\mathcal{T}_{local}\circ\mathcal{T}_{jump}$。
10. $\mathcal{T}_{jump}$ 用 LLM 生成非局部 seed；$\mathcal{T}_{local}$ 在 seed 周围做结构化扩展；$\mathcal{T}_{verify}$ 过滤非预测、重复和复杂候选。
11. 在因子发现实例中，LLM deepseek-v4 接收因子 taxonomy 和已接受因子知识库，输出 Python 候选表达式。
12. 系统解析每个表达式中的 base factors 和 operators，然后结构化枚举 product、ratio、difference、20-day rolling correlation。
13. 还加入 segment-conditioned variants，包括 percentile rank 和 z-score。
14. 对 $n$ 个 base factors，结构化引擎约产生 $2\binom{n}{2}+2n$ 个候选。
15. 因子按市值分段评估，用 weekly forward returns 上的 Pearson IC。
16. 接受条件为 $|\text{ICIR}|>0.3$ 且 $|\text{IC}|<0.95$，并用 archive correlation 衡量 novelty。
17. 全部评估放在 GPU-accelerated sandbox 中，有 timeout 和 memory management。
18. 理论上，Proposition 1 给出 local yield scaling：结构化搜索的非冗余发现数被 stable/effective rank 控制。
19. 该 bound 依赖 bounded leverage、bounded dependence、$r_{\text{stable}}\le r_{\text{eff}}\le N$ 等假设。
20. Theorem 1 不是充分条件，而是必要条件：压缩、逃逸、RSA 任一趋近 0，混合收益 $\Delta_{\text{yield}}$ 应趋近 0。
21. 论文把经验诊断写成乘法形式：$\widehat{\Delta}_{yield}\propto(1-r_{\text{eff}}/N)\cdot d_\perp\cdot RSA$。
22. 作者明确说该公式是 qualitative diagnostic，不是逐点预测律。

### 关键技术细节

23. A 股主数据为 Tushare Pro 日频 OHLCV，覆盖 5,647 只 A 股，时间为 2010-01 到 2026-05。
24. 光谱分析用全样本；因子发现训练 2010--2018、验证 2019--2021、测试 2022--2026。
25. 股票 universe 排除 ST/*ST、交易天数少于 60 天、IPO 后 30 天内股票。
26. 100-factor structured archive 上，pre-eligibility 的 $r_{\text{eff}}$ 为 31.5，post-eligibility 降为 28.6。
27. 同一 archive 的最大特征值占比从 0.187 升到 0.215，平均绝对相关从 0.118 升到 0.140。
28. 这意味着 post-eligibility 后少于 29% 名义维度携带独立信号。
29. 合成实验控制 $r_{\text{eff}}/N\in\{0.1,0.3,0.5,0.8,0.95,0.98\}$，做 100 次 Monte Carlo。
30. signal planting 实验用 $N=40,T=300$，压缩水平为 0.15、0.30、0.50、0.80，每个条件 300 MC reps。
31. directed-vs-random 实验在 weak eigenspace signal、$r_{\text{eff}}/N=0.15$ 下比较 random、shuffled、retrieval-guided、oracle。
32. real factor compression sweep 使用 3,638 个真实 A 股因子和预计算 IC。
33. LLM seed ablation 显示 LLM seed 的平均 $d_\perp=0.657$，高于 random symbolic 的 0.352、shuffled LLM 的 0.448、structured 的 0.213。
34. LLM seed 的 PredNovelty/RSA 估计为 0.032，高于 random symbolic 0.014、shuffled LLM 0.019、structured 0.008。
35. Symbolic regression 评测包含 25-equation custom benchmark 和 LLM-SRBench 158 problems。
36. 25-equation benchmark 使用 Feynman 10、Nguyen 7、Keijzer 5、LSR-Transform 3；每题 500 train 和 200 test samples。
37. PySR baseline 设置为 10 iterations、5 populations、30 individuals；hybrid 为半预算 PySR 加半预算 LLM proposals。
38. OpenML panel 不是压缩律直接验证，而是预算分配 sanity check：12 public tabular datasets、19 held-out splits、48 provider calls。
39. 论文报告 prompt/response hashes、parser-failure logs、固定 train/validation/test split 和 equal verified-candidate budgets。

### 核心结果

40. Synthetic spectral experiment：$r_{\text{eff}}/N=0.1$ 时 structured yield 15.5，hybrid 20.3；接近 full rank 0.98 时 structured 1.7，hybrid 1.8。
41. synthetic 中 random search 在全部压缩水平 yield 均为 0.0，支持“正交但不对齐没用”。
42. signal planting 主表显示 hybrid gain 在 weak eigenspace 为正，平均约 +2.7；在 top eigenspace 为负，约 -1.7。
43. 具体例子：$r_{\text{eff}}/N=0.30$ 且 signal 在 weak 时 structured 12.5、hybrid 15.3、$\Delta=+2.8$。
44. directed-vs-random 中 random seed 的 $d_\perp=0.969$ 最高，但 yield 只有 0.01。
45. shuffled 与 retrieval-guided 的逃逸距离几乎一样，0.692 vs 0.688；但 yield 是 8.03 vs 12.30。
46. retrieval-guided 的 PredNovelty/RSA 为 0.470，高于 shuffled 的 0.254，说明方向比距离更关键。
47. real factor compression sweep 中 high-corr subset 的 structured 38.8、hybrid 49.8、$\Delta=+11.0$。
48. real factor sweep 的 Spearman $\rho=-0.72$，说明 $r_{\text{eff}}/N$ 越低，hybrid advantage 越强。
49. HybridFactor full run：Pure LLM 100 candidates 接受 9 个、best ICIR 0.52；Pure Structured 55 candidates 接受 10 个、best ICIR 0.42。
50. HybridFactor 155 candidates 接受 19 个，acceptance rate 12%，best ICIR 0.52，breadth 19。
51. top factor `gap_up_ratio_20d` 的 Mean IC 为 +0.0418、ICIR 为 0.52；其他 top factors 包括 `mom_vol`、`ix_ret_5_x_ret_10`、`ret_5_plus_vol`、`vol_ratio`。
52. risk neutralization 后 5 个 top factors 均保持 Romano--Wolf $p<0.01$。
53. graph-guided factor combination 相比 single-factor Sharpe 从 2.34 提高到 2.63，提升 12%，最大回撤降低 36%，平均 pairwise correlation 0.083。
54. LLM-SRBench synthetic 子集：GP-only median $R^2=0.980$，hybrid full 为 0.984，基本持平或略好。
55. LLM-SRBench transform 子集：GP-only median $R^2=-0.011$，hybrid full 为 -0.065；两者都弱，但 hybrid rescue 25/46 个 GP catastrophic failures。
56. LLM-SRBench overall：GP-only median $R^2=0.875$，hybrid full 为 0.838，full-budget hybrid 与 GP parity，报告 $p=0.789$。
57. OpenML operational check：diagnostic validation-proxy test score 0.411，高于 local-only 0.380、fixed 50/50 hybrid 0.352、forced nonlocal 0.221。
58. OpenML 上 diagnostic proxy 相对 local-only 提升 0.045 的 clustered paired difference，95% CI 为 [0.003, 0.106]，Wilcoxon $p=0.008$。
59. cross-domain stress test 覆盖 drug discovery、climate、genomics、NAS、code generation；强压缩 $r_{\text{eff}}/N=0.18$ 平均 $\Delta=+3.0$。
60. qualitative scaling diagnostics 中 pooled $R^2=0.13$，leave-one-domain-out 为负，说明定量指数不跨域稳定；作者承认只支持定性规律。

### 明确局限

61. Theorem 1 是必要条件而不是充分条件；满足三条件并不保证正收益。
62. local-yield proposition 是 stylized scaling result，不给 tight universal constants。
63. 乘法模型 individual-level $R^2=0.21$，condition-level $R^2=0.83$；逐点预测能力有限。
64. OpenML 被作者明确标注为 operational sanity check，不是第三个 compression-law validation。
65. LLM-SRBench 缺少每次 run 的完整 candidate-archive spectra，因此主要是 failure-rescue evidence。
66. 小型 SR archive-spectra audit 只有 11 个问题，且有 2 个 errors，不能支撑强泛化声明。
67. A 股实验依赖 Tushare Pro 和可能不可再分发的商业/受限市场数据。
68. 代码、scripts、cached tables 和 manifests 说是 acceptance 后 release；当前论文文本本身不足以第三方完全复现。
69. 因子 discovery 的 LLM 使用 deepseek-v4，但 prompt 细节、候选失败日志和完整 archive 未在 tex 中展开。
70. 结果中存在一些强合成设定，例如 signal planting 和 weak eigenspace oracle/retrieval guidance，外部真实任务中不一定能拿到同等 guidance。
71. random non-local 被设计为无目标对齐，0 yield 说明 alignment 重要，但也可能低估更强随机/进化式 explorer。
72. 论文把“LLM 风格 seed”与“真实 LLM 发现能力”部分混用，需要看 release artifact 才能确认所有候选如何生成和筛选。

## Read by: 0616-fex (2026-06-16)

从 `0616-fex` 视角看，这篇不是普通 related work，而是一个强 framing competitor。它把 LLM 生成表达式/因子的问题重新解释为“压缩 archive 中寻找 target-bearing residual direction”，并给出了可直接工程化的三个诊断量：effective rank、escape distance、PredNovelty/RSA。若我们的工作也主张 LLM 能在因子或表达式发现中提升搜索，必须避免只说“LLM 更有创造性”；需要说明我们的 proposal 如何被独立验证、如何避免冗余、如何在压缩变化时分配预算。

可用的方法包括：对候选 archive 做相关矩阵谱分析；每轮搜索前计算 $r_{\text{eff}}/N$；对新表达式做 ridge projection 得到 residual；用 residual ICIR 或其他目标对齐指标做 verification gate；当 full-rank、escape 低或 alignment 低时停止花 LLM budget。可引用的核心结论是 random orthogonal novelty 不足，必须是 directed predictive novelty。

差异化建议：不要复刻 HybridFactor 的 A 股实验主张，除非我们能提供更强的 artifact 和可复现性。更好的路线是把三指标纳入 agent-factory 的在线调度/critic，做跨任务的 feature-expression search，并用完整日志证明每个 claim 由独立 evaluator veto 过。当前论文的弱点是 artifact 未公开、A 股数据受限、theorem 是必要非充分、OpenML 只是 sanity check；这些正好是我们可以提升可信度的空间。
