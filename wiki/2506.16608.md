# Distributions as Actions: A Unified Framework for Diverse Action Spaces (2506.16608)

- **Year**: 2025.06
- **Authors**: Jiamin He, A. Rupam Mahmood, Martha White (University of Alberta, Amii)
- **Venue**: ICLR 2026 (submitted)
- **Keywords**: distributions-as-actions, policy gradient, action space unification, deterministic policy gradient, actor-critic, interpolated critic learning, DA-PG, DA-AC
- **One-liner**: 将策略的参数化分布参数重新定义为连续动作空间，从而使同一个连续控制算法（DA-AC）能统一处理离散、连续和混合动作空间。

## 精华提炼

### 管线/方法

**核心思路：重新定义 agent-environment 边界**

传统 RL 中，agent 输出分布参数 → 采样动作 → 环境执行。本文把「采样」这一步从 agent 移到 environment，agent 直接输出分布参数作为「动作」，环境负责采样并在 MDP 中做随机转移。

**完整 pipeline：**
1. **输入**: 状态 s
2. **Actor 网络** (π_θ): s → 分布参数 Θ = [离散: 概率向量 p_1...p_N; 连续: 均值和 log-std 的 tanh 变换值]
3. **环境采样** f(·|Θ): Θ → 实际动作 A（categorical sampling / Gaussian + clip）
4. **Critic 网络** (TQ_φ): (s, Θ) → 期望回报的估计（输入是分布参数，不是原始动作）
5. **DA-PG 梯度**: ∇_θ π_θ(s)^T · ∇_Θ TQ_φ(s, Θ)|_{Θ=π_θ(s)} （形式上与 DPG 相同，但操作的语义不同）
6. **ICL 更新 critic**: 在 Θ_t 和 Θ_{A_t}（采样动作对应的一-hot/Dirac-delta 参数）之间线性插值，用插值点 Û_t 做 TD 学习

**DA-AC 算法**基于 TD3 构建：
- 双 Q 网络（clipped double Q-learning）
- 去除 actor target network（实验表明无影响）
- ICL 替换标准 TD critic update
- 统一的 exploration：连续场景均匀采样 distribution parameters ∈ [-1,1]²

### 关键技术细节

**分布参数化（每种动作空间不同，但接口统一）：**
- **离散**: 概率向量 [p_1,...,p_N]，Θ_A = onehot(A)
- **连续**: Θ = [Θ_μ, Θ_σ] ∈ [-1,1]²，μ = squash(Θ_μ) 映射到 [a_min, a_max]，σ ∈ [σ_min, σ_max]（可学习），Θ_A = [2A/(a_max-a_min), -1]（逼近 Dirac delta）
- **混合**: 同时包含概率向量 + 均值和 log-std

**Critic 架构：** 2层 MLP，hidden=(256,256)，ReLU。对于 MinAtar 的 CNN 观测，使用 CNN 特征和 action representation 的 flattened outer product 做 joint encoding。

**超参数（沿用 TD3 默认值）：**
- Batch size: 256（Gym 128, MinAtar 32）
- Actor/Critic LR: 3e-4
- τ = 0.005, γ = 0.99
- Replay buffer: 1e6（Gym 1e4, MinAtar 1e5）
- Policy update delay N_d = 2
- Uniform exploration: 25k steps
- Learnable σ range: [0.05, 0.2]

**ICL 关键细节：**
- Û_t = ω Θ_t + (1-ω) Θ_{A_t}，ω ~ Uniform[0,1]
- 不做 importance sampling 校正（biased but works）
- 直观解释：迫使 critic 在策略参数到最优动作参数之间的路径上学习平滑的 value landscape

**理论贡献：**
1. DA-MDP 的价值函数与原始 MDP 等价（Prop 3.1）
2. DA-PG 是 DPG 的严格推广（Prop 4.1: δ-distribution 时退化）
3. DA-PG 是 LR 和 RP 估计量的条件期望 → 严格更低的方差（Prop 4.2, 4.3）
4. 在 on-policy linear function approximation 下的收敛性（直接从 DPG 收敛理论推广）

### 核心结果

**连续控制（20 tasks: 5 MuJoCo + 15 DMC）：**
- DA-AC vs TD3: 在更多环境上优于 TD3（Fig 3 col 1）
- 聚合性能：在 MuJoCo 和 DMC 上均优于 TD3、RP-AC、SAC、PPO（DMC 上尤其显著）
- 高维动作空间（Humanoid, Quadruped）: DA-AC 显著优于所有 baseline（Fig 5）

**离散控制（Gym 4 + MinAtar 5）：**
- DA-AC 与 DQN 性能相当，是 best-performing 算法之一
- 优于 LR-AC、ST-AC、EAC、DSAC、PPO

**高维离散控制（离散化 MuJoCo/DMC，每维 7 bins）：**
- 离散化 Humanoid: action space 7^17 ≈ 2.3e14，DQN/DSAC/EAC 不可行
- DA-AC 在所有 baseline（ST-AC、LR-AC、PPO）中最佳
- DA-AC 离散版 vs 连续版性能差距不大（Fig A.10）

**混合控制（PAMDP 7 tasks）：**
- DA-AC 与 PATD3 竞争，优于 PDQN 和 HHQN
- 使用统一算法即可，无需像 PATD3 那样专门设计

**ICL ablation：**
- DA-AC w/o ICL 在各方面均低于 DA-AC
- Paired t-test: 7 个 setting 中 4 个显著（MuJoCo p=0.0018, Disc MuJoCo p=1.25e-6, Disc DMC p=0.0333, PAMDP p=0.0059）
- 跨 setting 聚合: mean diff=0.0508, t=3.820, p=0.0088

**Bandit 可视化实验（K-Armed + Bimodal）：**
- DA-PG (true critic) vs RP-PG: 方差更低，收敛略快
- DA-AC w/o ICL: 严重欠拟合，收敛到次优解
- DA-AC (with ICL): 初期慢于 LR-AC/RP-AC，但最终匹配或超越

### 明确局限

1. **Critic 学习难度**: 分布参数空间比原始动作空间高维，critic 更难学（作者明确承认 bias-variance trade-off）
2. **ICL 是 heuristic**: 不做 importance sampling 校正，有偏；作者留待未来工作做更 principled 的处理
3. **离散动作空间的结构信息丢失**: 离散空间的枚举/比较优势被放弃（作者在 Appendix 明确提及）
4. **无真机实验**: 所有实验在仿真环境（MuJoCo, DMC, MinAtar, Gym Classic Control, PAMDP）
5. **仅与 TD3-based 方法对比**: 没有与 SAC-Discrete、Dreamer 等更广泛的 baseline 比较
6. **CPU-only 训练**: 所有实验用 CPU 单核（< 2GB RAM），未在 GPU 上验证可扩展性
7. **没有 entropy regularization**: DA-AC 缺少 SAC 那样的自适应探索机制（作者在 Appendix 讨论了扩展方向）
8. **没有 model-based 实验**: 虽然讨论了与 MCTS/TD-MPC 结合的可能性，但没有实验
