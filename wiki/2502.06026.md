# A Multimodal PDE Foundation Model for Prediction and Scientific Text Descriptions (2502.06026)

- **Year**: 2025.02
- **Authors**: Elisa Negrini, Yuxuan Liu, Liu Yang, Stanley J. Osher, Hayden Schaeffer (UCLA Math, NUS Math)
- **Venue**: arXiv preprint
- **Keywords**: PDE foundation model, multimodal learning, operator learning, transformer, GPT-2, scientific text generation, cross-attention, in-context learning
- **One-liner**: 提出 MOL-LLM，用 GPT-2 backbone + cross-attention 同时输出 PDE 数值解和科学文本描述，首次做到真正的多模态（数值+文本）双向 PDE 基础模型。

## 精华提炼

### 管线/方法
完整 pipeline：
1. **多模态输入**：文本描述（方程公式/物理过程描述）+ 数值数据（方程参数、初始条件）
2. **Custom Tokenizer**：文本用 GPT-2 tokenizer 编码，数值用一个小 MLP（multilayer perceptron）编码为连续特征向量（类似 xVal [golkar2023xval] 但加了 MLP）
3. **LLM Backbone**：pretrained GPT-2 处理 unified token sequence，做 token mixing
4. **双解码路径**：
   - Data Decoder（数值输出）：cross-attention-based transformer，query locations 作为 evaluation points，类似 DeepONet 的 branch/trunk 结构
   - Text Decoder（文本输出）：标准 autoregressive next-token prediction，greedy decoding
5. **联合训练**：Loss = α·相对平方误差（数值）+ β·交叉熵（文本），α,β 为超参数

### 关键技术细节
- **Backbone**: GPT-2（pretrained），具体参数量未明确说明
- **数据集**：52 个参数化微分方程 × 100 参数采样 = 5200 个 distinct equations
  - 1D ODE × 5, 2D ODE × 5 (Van der Pol, Lotka-Volterra, FitzHugh-Nagumo, Brusselator, Duffing), 3D ODE × 2 (SIR, Neural Dynamics)
  - PDE × 13 (Heat, Porous Medium, Klein-Gordon, Sine-Gordon, Cahn-Hilliard, KdV, Advection, Wave, 4×Reaction-Diffusion, Fokker-Planck)
  - Conservation Laws × 27 (Burgers, Inviscid Burgers, various fluxes, shock/rarefaction variants)
- **参数采样范围**：[Q-0.1Q, Q+0.1Q]，即 10% 相对变化
- **ODE 数据生成**：SciPy `solve_ivp`，t ∈ [0,5]，每方程 50 个初始条件
- **PDE 数据生成**：沿用 sun2024towards 方法，(t,x) ∈ [0,5]×[0,2]，空间离散 128 点，每方程 100 个初始条件
- **文本标注**：GPT-4 生成，每方程 50 条不同描述（方程性质、数值方法、物理意义）
- **数值 Loss**：relative squared error，对 padded 位置不计算 loss
- **文本 Loss**：标准 cross-entropy，训练时 mask future tokens
- **数值评测**：Relative Error = ||û - u|| / ||u||
- **文本评测**：BERTScore（Precision/Recall/F1），+ 人工抽查 100 例
- **训练/测试**：同分布（10% range），仅给 t=0 初始条件，预测 t∈[0,5]
- **时间外推**：给模型自己预测的 t=5 解作为新初始条件，预测 t∈[5,10]

### 核心结果
| 类别 | In-Distribution Error | OOD 20% Error | OOD 30% Error | BERTScore F1 |
|------|----------------------|---------------|---------------|-------------|
| 1D ODE | 3.00% | 7.19% | 12.56% | 0.917 |
| 2D ODE | 5.36% | 15.15% | 25.71% | 0.950 |
| 3D ODE | 4.43% | 6.46% | 8.86% | 0.916 |
| PDE | 1.85% | 3.56% | 5.82% | 0.948 |
| Conservation Laws | 1.41% | 3.54% | 6.66% | 0.952 |
| **Total Avg** | **3.21%** | **7.78%** | **11.92%** | **0.937** |

- **文本生成**：人工抽查 100 例，100% 正确描述
- **时间外推**：Heat 8.65%, Advection 29.3%, Diffusion-Reaction 9.08%, Inviscid CL Cubic 7.94%, CL Sine 12.9%
- **Surprising**：2D ODE 的 OOD 误差暴涨（25.71%），远超其他类。原因是 Duffing/Van der Pol 等系统对参数极度敏感（混沌/振荡 regime 切换），仅给初始条件不足以推断轨迹
- **激波/稀疏波识别**：模型能从纯输入数据判断 conservation law 解是否含 shock/rarefaction，且位置和强度正确。F1 > 0.94
- **与 FNO/DeepONet 对比**（引用前作 sun2024towards Table 5）：FNO 和 DeepONet 在同 PDE 数据集上即使给了多个时间快照，误差仍分别达 32.07% 和 34.86%，远高于本方法

### 明确局限
- 作者自述：
  1. 仅给初始条件（单时间步）限制了 OOD 泛化；给更多时间快照可改善
  2. 文本描述多样性有限（训练集 50 条/方程），且是 GPT-4 生成的 synthetic text，非人类专家标注
  3. 不支持用户指定文本输出焦点（如"描述长时间行为"还是"描述数值方法"），留作 future work
  4. 时间外推仅在解 t=5 与训练初始条件相似的方程上测试（Advection 误差 29.3% 很糟）
  5. 未涉及高维 PDE（最大 1D 空间+时间）
- 我读出的局限：
  1. **无真机/无真实数据**：全部合成数据，无实验物理系统验证
  2. **文本是 GPT-4 生成的**：训练信号来自另一个 LLM，本质上是 distillation，未证明模型真正"理解"了物理，只是在模仿 GPT-4 的描述风格
  3. **52 个方程覆盖面窄**：没有 2D/3D 空间 PDE、没有流体力学（Navier-Stokes）、没有多物理场耦合
  4. **GPT-2 backbone 很小**：相比现代 LLM（LLaMA 等），GPT-2 能力有限。未 scaling 实验
  5. **无消融实验**：没有验证 MLP encoding vs 直接 tokenization、没有验证 GPT-2 pretrained vs random init、没有验证 α/β loss weight 对结果的影响
  6. **周期边界假设**：所有实验假设 periodic boundary conditions
  7. **文本评测粗糙**：BERTScore 测的是语义相似度而非事实正确性；"100% 正确"来自 100 例人工抽查，样本太小
