# TS-CausalNN: Learning Temporal Causal Relations from Non-linear Non-stationary Time Series Data (2404.01466)

- **Year**: 2024.04
- **Authors**: Omar Faruque, Sahara Ali, Xue Zheng, Jianwu Wang（UMBC Information Systems + LLNL Climate Science Section）
- **Venue**: arXiv preprint, Springer LNCS 格式排版（llncs.cls + splncs04 引用样式），具体会议/期刊名称正文中未声明，未核实，不臆测
- **Keywords**: temporal causal discovery, non-stationary, non-linear, NOTEARS acyclicity, 2D causal convolution, augmented Lagrangian, score-based, Arctic Sea Ice / TKE
- **One-liner**: 用并行自定义 Causal Conv2D 层 + NOTEARS 式迹指数无环约束, 单次前向同时学出滞后与同期因果图, 在非线性非平稳合成与真实地球科学数据上优于 PCMCI(+)/NOTEARS-MLP/DAG-GNN/DYNOTEARS/NTS-NOTEARS。

## 精华提炼

### 管线/方法

1. **输入构造**：多变量时间序列 $X=\{x^1,...,x^n\}$，$T$ 个时间步，设定最大滞后 $l_{max}$。所有变量先归一化到 [0,1]，然后把每个时间步 $t$ 之前 $l_{max}$ 步的数据拼接到当前步前面，构成形状 $(T, n, l_{max}+1)$ 的张量，直接喂给 2D 卷积输入层。
2. **第一层（Latent 表征）**：一层标准 Conv2D 把原始输入映射到隐表征，用于捕捉数据里的非线性关系（这一步不承担因果结构学习任务，纯粹是非线性特征提取）。
3. **Causal Conv2D 块（核心创新）**：为每个目标变量 $x^i$ 单独设一条并行的自定义 Causal Conv2D 支路，输入是 $x^i$ 以外全部变量的滞后项 + 同期项（$x^i$ 自身被强制权重置零，排除自因果），输出 $E[x^i \mid PA_{x^i}]$（Eq. 2）。$n$ 个变量就是 $n$ 条并行支路，一次前向同时算完全部变量的预测。
4. **邻接矩阵读出**：沿用 NOTEARS-MLP / DAG-GNN 的套路——每条支路卷积层的权重 $W^{x^i}$ 直接被解释为该变量的因果强度（不是先训练再单独跑一次结构搜索），权重非零即认为存在边、正负号仅表示强度不表示方向额外含义。
5. **无环约束（关键简化）**：把总邻接矩阵 $W$ 拆成滞后部分（天然无环，因为箭头只能从过去指向未来，不需要约束）和同期部分 $W^t$（可能成环，需要约束）。只对 $W^t$ 套用 Zheng et al. (NOTEARS, 2018) 的 $h(W^t)=tr(e^{W^t \circ W^t})-n=0$，而不是对整个 $(n\times(l_{max}+1))\times n$ 矩阵套约束——这是相对原始 NOTEARS 家族的主要效率/简洁性改动。
6. **优化目标**：增广拉格朗日法，$\min_W [L(W) + \frac{\rho}{2}|h(W^t)|^2 + \alpha h(W^t) + \lambda\|W\|_1]$，其中 $L(W)=\frac{1}{T}\|X-WX\|_F^2$ 是最小二乘重构损失，$\lambda=0.1$ 控制稀疏度。$\rho,\alpha$ 按标准对偶上升规则迭代更新（$h(W^i) > \gamma h(W^{i-1})$ 则 $\rho \leftarrow (1+\beta)\rho$，否则不变；$\alpha \leftarrow \alpha + \rho h(W^i)$），经验最优 $\beta=0.1,\gamma=0.25$。
7. **后处理**：收敛后对最终权重做阈值化 $W_\omega = (W>\omega)$，$\omega=0.3$，剪掉弱边得到最终二值因果图；可进一步把滞后+同期的 full graph 坍缩成按变量对聚合的 summary graph。
8. **输出**：既有完整的时序因果图（区分每条边来自哪个滞后）也有 summary graph（变量对层面的聚合图），两种粒度都参与和 baseline 的定量比较。

### 关键技术细节

- **实现与算力**：TensorFlow Keras 实现，全部实验跑在 **Google Colab CPU runtime**（不是 GPU），固定随机种子保证可复现——相对于典型深度因果发现工作，算力footprint 明显更轻。
- **无环约束的部分应用是全文最具体的技术贡献点**：只约束同期子矩阵 $W^t$，滞后部分利用时间先后关系天然免疫环路，省掉了对高维矩阵算迹指数的开销。
- **Loss 构成**：LS 重构 + 增广拉格朗日无环惩罚（仅同期部分）+ L1 稀疏惩罚（$\lambda=0.1$ 固定超参，未做敏感性扫描）。
- **合成数据**：Dataset-1（4 变量，含 $\cos$ 季节项 + $\log$ + 指数非线性，Gaussian 噪声，稀疏真图，滞后 1/2/5 步依赖并存）；Dataset-2（同结构去掉 cos 季节项，噪声换成 Poisson，用来测试对非高斯噪声的鲁棒性）；Dataset-1.1/1.2/1.3 是 Dataset-1 的三档信噪比（SNR）变体，专门做鲁棒性研究。
- **真实数据集 1（TKE，湍流动能）**：DOE ARM Southern Great Plains 场址的 LES/LASSO 理想化数值模拟数据，变量含 SH（切变生成项）、BU（浮力生成项）、TR（湍流+压力输运项，隐含）、TEND（净倾向项）、TKE 本身，滞后 $l_{max}=5$（对应边界层日循环 05:00–18:00 的快演化）。
- **真实数据集 2（Arctic Sea Ice，北极海冰）**：1980–2018 月均值，60N 以北，11 个 ERA-5 大气变量（RH/SLP/U10m/V10m/HFLX/PRE/CC/CW/SW/LW/GH）+ 海冰范围，滞后 $l_{max}=12$（年周期）；ground-truth 因果图沿用 Huang et al. 2021 (arXiv 未提供，S2 命中) 的物理/微物理文献综述图。
- **Baseline 集合**：PCMCI、PCMCI+（约束型，tigramite）、NOTEARS-MLP、DAG-GNN、DYNOTEARS、NTS-NOTEARS（后三者本非为时序设计，作者把滞后+同期数据展平成长向量后套用）。
- **评测指标**：SHD（结构汉明距离，边增/删/转向的最小修正次数）、F1（precision/recall 调和均值）、FDR（预测边中错误的比例，考虑方向）。
- **非平稳性不是假设而是实测验证**：TKE 和 Arctic Sea Ice 都跑了 ADF + KPSS 双检验（Table 8/9），TKE 全部 4 变量双检验一致判非平稳；Arctic Sea Ice 的 12 个变量里两种检验结果不完全一致（ADF 判 4 个非平稳，KPSS 判 3 个），说明是非平稳/平稳混合数据——这个统计验证步骤比很多同类论文单纯口头宣称"处理非平稳数据"更扎实。
- **消融**：Causal Conv1D（把隐表征展平后套 1D 版本的自定义因果层）vs Causal Conv2D（本文提出的 2D 版本），2D 在全部 4 个数据集上大幅胜出（如 Synthetic-1 summary graph：SHD 7→3，F1 0.46→0.77，FDR 0.57→0.28）。

### 核心结果

- **合成数据 full graph**：Dataset-1 上 F1 最优(0.60)，SHD/FDR 次优；Dataset-2 上 SHD(7)/F1(0.46)/FDR(0.25) 三项全优。
- **TKE summary graph**：与 PCMCI+ 打平最优（SHD=5, F1=0.44, FDR=0.66），且定性上正确识别出关键单向边 BU→TEND、TEND→TKE，而 PCMCI+ 把这两条预测成双向、NOTEARS-MLP 漏掉其中一条。
- **Arctic Sea Ice summary graph**：F1(0.49)/FDR(0.56) 最优（SHD 非最优，PCMCI+ 的 50 更低），且从 ICE 节点出发的 4 条真实因果边正确恢复 3 条（"什么导致海冰融化"这一科学问题方向上的定性验证）。
- **SNR 鲁棒性**：SNR 从低到高（Synthetic-1.1→1.3），summary SHD 单调下降 5→4→2，F1 单调上升 0.54→0.66→0.83——性能随信噪比平滑退化，没有低 SNR 下的灾难性失效。
- **一个值得注意的"反直觉"细节，体现作者的方法论细心**：DAG-GNN 在部分表格里 FDR=0.00（看似完美），但同时 F1=0.00，即它几乎不预测任何边，FDR 为零只是"没边可错"的假象；论文正文明确指出不能只看 FDR 不看 F1，这是易被忽略的指标陷阱。
- **评测协议的一个弱点（也算"核心结果"的解读前提）**：论文写明"每个方法多次运行，报告最佳结果"（best-of-N），而不是均值±方差——所有 SHD/F1/FDR 数字都是各方法各自的最优发挥，不是典型运行表现，横向比较存在乐观偏差。

### 明确局限

- **未来工作段落在最终版里被注释掉**：tex 源码中"考虑隐藏混杂因子""扩展到时空 3D 场景"这两条未来方向文字仍在 `%` 注释里，没有出现在正文可见的 Conclusion 中——即公开版本实际上没有写局限/未来工作段落，只有一句笼统的方法总结。
- **无不确定性量化**：所有 SHD/F1/FDR 都是单点估计（且是 best-of-N 而非均值），没有跨 run 的方差、置信区间或显著性检验。
- **无规模/复杂度实验**：最大真实数据集也只有 12 个变量（Arctic Sea Ice），合成数据 4 个变量，完全没有测试变量数增长（50+/100+）时的可扩展性或计算复杂度曲线。
- **阈值 $\omega=0.3$、稀疏惩罚 $\lambda=0.1$ 均为全局固定超参**，真实数据集上没有报告敏感性分析。
- **应用域是湍流动能 + 北极海冰，不是强对流/极端天气事件**：地球科学时序因果发现这个大方向相关，但数据集本身与强对流预警场景有距离。
- **与"因果估计不稳定性作为信号"这条研究路线完全不相交**：本文的全部目标是让点估计的因果图本身更准（SHD/F1/FDR 越好越好），没有做滑动窗口、超参扰动或 bootstrap 意义下的估计稳定性分析，也没有把"图不稳定"当一种信号来利用或讨论。
- **代码链接是双盲评审期间的匿名 anonymous.4open.science 短链**，长期可达性、是否迁移到正式 GitHub 组织未知。

### 跟我们的关系

课题 `0710-causal-scs`（因果效应不稳定性作为强对流预警指标）的核心主张是**反过来利用**因果发现方法在滑动窗口/假设违背下的估计不稳定性（当前候选方法：PCMCI+/J-PCMCI+、LKIF，topic 文件里还写了"视资源情况纳入 score-based 方法"）。TS-CausalNN 与本课题**不构成新颖性撞车**——它是一篇标准的"让因果图点估计更准"论文，完全没有做窗口滑动、超参扰动或 bootstrap 意义下的稳定性分析，与 v3 idea 里 C1a/C1b/C2/C3/C4 的核心 claim 没有重叠，v3 review 里列出的最强撞车项（RCV-PCMCI/VarLiNGAM 2410.19412、VCDF 2602.21381、2606.01214、2502.14719）里也不包含本文。

但它对课题有两点直接可用价值：

1. **它恰好是 topic 文件里提到但尚未列入具体候选的"score-based 方法"的一个现成实例**——用 NOTEARS 式迹指数无环约束 + 神经网络，天然支持非线性、非平稳、任意噪声分布，且轻量（CPU 可跑）。如果 v3/v4 决定真的纳入 score-based 方法做跨估计器复现（Pilot 里 LKIF 是当前唯一计划中的第二估计器），TS-CausalNN 可以作为第三候选，用它在滑动窗口下重跑同一套 (D,J) 不稳定性协议，检验"条件化丢信号"这一现象是否只是 PCMCI+ 的 constraint-based 特性还是跨范式（constraint-based vs score-based）的一般现象。
2. **它直接把 PCMCI/PCMCI+ 当 baseline，在真实非平稳大气数据（TKE、北极海冰）上给出了 PCMCI+ 的具体 SHD/F1/FDR 表现**，可以作为"PCMCI+ 在真实地球科学非平稳数据上大致处在什么准确度水平"的外部校准参考，帮助判断 v3 review 里质疑的"PCMCI+ summary 统计量弱于 marginal Pearson"这一结果是否与文献里 PCMCI+ 的一般表现相符还是异常。
