# The Finite Expression Method for Turbulent Dynamics with High-Order Moment Recovery (2605.10687)

- **Year**: 2026.05
- **Authors**: Xingjian Xu (U Florida), Di Qi (Purdue), Chunmei Wang (U Florida)
- **Venue**: arXiv (submitted to SIAM journal)
- **Keywords**: Symbolic discovery, Finite Expression Method (FEX), turbulent dynamical systems, generative models, high-order moments, Training-Free Diffusion Model (TFDM), stochastic triad model, two-stage framework
- **One-liner**: 提出 FEX + 生成模型的两阶段框架，从湍流数据中同时恢复确定性动力学符号表达式并预测高阶统计矩（最高五阶）。

## 精华提炼

### 管线/方法
完整的两阶段管线：
1. **Stage I — FEX 符号发现**：对每个系统维度 i=1,...,d，用一个深度 L=3 的二叉树表示候选符号表达式，叶子节点接收输入（状态分量 u_j 和时间 t），每个节点从一元算子集合 U={sin, cos, exp, 0, Id, (·)², (·)³, (·)⁴} 和二元算子集合 B={+, −, ×} 中选择操作符。通过 RL controller（单层 NN + ReLU）生成候选算子序列，用 policy gradient + ε-greedy 搜索，reward = (1+√L(e_i))^{-1}，loss 为 MSE over 轨迹数据。维护 candidate pool（size=100），用 Adam（lr=8e-3）20 iter + L-BFGS 10 iter 做 score 优化，再用 Adam + cosine decay 做 50000 iter fine-tuning。最终为每个维度找到 C_i(u,t) 的闭式表达。
2. **Residual 计算**：R_n = u(t_{n+1}) − u(t_n) − h·C*(u(t_n))，即观测增量减去 FEX 确定性预测。
3. **Stage II — 生成模型建模残差**：三种方案：
   - **TFDM**：Forward SDE (dZ = b(τ)Z dτ + q(τ)dW)，α_τ=1-τ, β²_τ=τ。用 Monte Carlo 估计 score function V(Z,τ)=∇_z log p_τ(z)，反向 ODE flow (dZ = [bZ − ½q²V]dτ)，从 Z₁~N(0,I) 积分到 τ=0 得到 y。用 FAISS 最近邻选取代表性样本。训练 NN G_{θ1}(ξ₁)→y，MSE loss。网络：单隐层 50 神经元 tanh，Adam lr=1e-2，weight decay=1e-6，2000 iter。
   - **SRAN**：直接 NN G_{θ2}(ξ₂~N(0,I))→R，loss = MSE + λ||E[GG^T] − hI||²_F，λ=0.1。用协方差正则化保证二阶统计量正确。
   - **VAE**：encoder G_{φ3}(R)→(μ,Σ)，decoder G_{θ3}(ξ₃)→R。Loss = α_mean·MSE + α_var·CovReg + β_KL·D_KL。α_mean=0.1, α_var=1.0, β_KL=1e-3。隐维度 9（每维 3 个隐变量）。网络同 TFDM。
4. **推理**：给定初态，用 FEX 计算 h·C*(u)，采样高斯噪声驱动生成模型得到残差 R̂，u ← u + h·C*(u) + R̂，逐步推进。

### 关键技术细节
- **FEX 树结构**：统一树架构分为三部分：Linear Part（线性项：u_j 的一维 tree block）、Nonlinear Part（非线性交互：多维度 tree block 的乘积组合）、Time-dependent Part（时间相关：以 t 为输入的 tree block）。每个一维 tree block 深度 L=3，含 4 个算子位置 e_{ij}=(e_{ij1}, e_{ij2}, e_{ij3}, e_{ij4})，整体 aligned operator sequence e_i = (e_{ij})_{j=1}^d。
- **训练数据规模**：M=1000 条轨迹，每条 N=1001 时间步，h=0.01，T=10.0。重组为 M×N 个 scalar input-output pair。测试：500,000 条轨迹，T_test=20.0。
- **RL Controller**：单隐层 NN + ReLU，输出维度 = |B|+3|U|。Policy gradient 只使用 top v-fraction 的候选序列（reward ≥ quantile）。Batch size=200，Adam lr=2e-3，迭代 100 次（复杂 regime）。
- **Candidate Pool**：N=100，防止错误表达式替换正确候选。Fine-tuning 使用 Adam + cosine decay 50000 iter。
- **理论假设**：(1) 遍历性（ergodicity）— 时间平均收敛到不变测度下的期望；(2) 强 Itô-Taylor 展开 — 随机增量可分解为 σ√h·z + R，余项 E[R]=0, E[|R|²]=O(h²)；(3) 基函数线性独立于 L²(ρ)；(4) 逼近残差可任意小；(5) Gaussian mixture initialization for TFDM。
- **核心定理 5.1**：FEX 系数估计量的相合性 — 在 T→∞, h→0 下，α̂^i →_P α^i。
- **核心定理 5.2**：系数误差界 — ||E[α̂−α]|| ≤ C(ε+h+o(1))；Var(α̂) ≤ C̃/T。
- **核心定理 5.3**（Lemma 5.3）：残差训练集有界 — ||R_n||₂ ≤ M_R。
- **核心定理 5.4**：TFDM 总误差界 — E[||y*−N_{θ1}||₂] ≤ C₀√d + C₁dh + C₃（smoothing error + discretization error + network approximation error）。

### 核心结果
**Stage I — FEX vs Weak SINDy 表达式恢复**：
- FEX 在所有 5 个 regime 下均准确恢复闭式表达式，系数误差通常在 10⁻²~10⁻⁵ 量级（详见 Table tab:multi_regime_expression）。
- Weak SINDy 对 forcing term 只能做多项式逼近（如用 t,t²,t³,t⁴ 代替 sin(πt/4)），在强随机性下无法识别交互项。
- FEX 在 Random Cascade 中用 exp(−kt) 近似 OU 过程 m(t) 的确定性平均行为。
- 系数估计满足能量守恒约束 B₁+B₂+B₃=0（Forward Cascade: 2.035−1.001−0.999≈0.035；Dual Cascade: 1.985−1.000−1.000≈−0.015）。
- 噪声鲁棒性：scaling factor c∈[0,2] 下系数误差在 10⁻⁶~10⁻²，semi-log 尺度稳定。
- 样本量 scaling：1000→4000 改进最大（尤其 Forward/Dual Cascade），4000→10000 边际递减。

**Stage II — 高阶矩预测（1-5 阶）**：
- SRAN 在所有 regime 下最接近 ground truth，尤其在 Equipartition 和 Dual Cascade。
- TFDM 有轻微偏差但整体良好。
- VAE 保持趋势但在高阶有较大振幅波动。
- 6-7 阶矩（Supplement）：SRAN 仍一致，TFDM 和 VAE 下降，但密度估计仍准确。
- 一阶矩（均值）准确：主要反映 FEX 质量，生成模型贡献小。
- 能量演化：超出训练时域（T=10→20）仍有良好泛化。

**Time-independent vs Time-dependent TFDM**：
- Time-dependent 短期（T<10）协方差更准，尤其振荡行为，但 T>10 后不稳定（ill-conditioned）。
- Time-independent 长期稳定但存在系统偏差和丢失细粒度变化。
- 短/长期准确性的 trade-off。

### 明确局限
- **仅限低维系统**：只测试了 d=3 的 triad model。高维系统（如完整 Navier-Stokes 截断）的计算代价和搜索复杂度未验证。
- **TFDM 计算代价**：每时间步需重新训练，总代价 ∝ 时间步数 × 数据量。作者称为"重要实践问题"并留给 future work。
- **随机强迫恢复有限**：Random Cascade 中 FEX 用 exp(−kt) 近似 OU 过程 m(t)，无法真正恢复随机过程的完整分布特征。
- **Step size 敏感性**：TFDM 对步长选择敏感（正向/反向 SDE 离散化 2000 步），trade-off 未充分探索。
- **没有真机实验**：纯数值仿真，没有真实物理系统数据。
- **没有与其他 DL 方法对比**：仅对比了 Weak SINDy 和 3 种生成模型间的比较，未与 Physics-informed Neural Networks, Neural ODE 等方法对比。
- **生成模型 Stage II 依赖 Stage I 质量**：FEX 的表达误差通过 residual 传递给生成模型，两阶段的耦合方式较简单（只是 residual 相减）。
- **只在 additive noise 假设下证明理论**（Assumption 4.3），multiplicative noise 未考虑。
- **代码虽开源但无独立复现报告**。

### 跟我们的关系
对 0616-fex 课题：
- **不撞车**：本文聚焦湍流动力学系统 + 高阶统计矩恢复，是 FEX 在流体物理中的应用。0616-fex 如果是 FEX 方法本身改进或新应用场景，不直接冲突。
- **可用的工具/方法**：
  1. **两阶段框架范式**：FEX deterministic + generative stochastic 的分解思路可用于任何含噪声的动力学系统，不限于湍流。如果我们的课题涉及噪声数据下的符号回归，这是一个直接可参考的解决方案。
  2. **FEX 实现细节**：controller 架构（单层 NN+ReLU）、ε-greedy 探索、candidate pool（size=100）、score 计算（Adam+L-BFGS hybrid）、fine-tuning（cosine decay 50000 iter）— 这些超参数和工程经验可以直接复用。
  3. **三种生成模型对比**：SRAN 简单且效果最好（MSE + Covariance Regularization），其 loss 设计（λ=0.1 的协方差正则化）值得学习。
  4. **Triad model 基准**：是一个好的低维湍流测试平台，参数化清晰，覆盖 5 种统计 regime。
  5. **残差 boundedness 理论框架**：Lemma 5.3 的证明技术可用于类似的两阶段方法的收敛性分析。
- **差异化**：
  1. FEX 的树搜索策略可改进（目前只是 RL controller + ε-greedy），可以探索 MCTS 等更高效的搜索。
  2. Stage I 和 Stage II 的耦合是单向的（FEX→residual→generative），可以做迭代式 refinement（Stage II 反馈矫正 Stage I 的系数）。
  3. 当前只支持 additive noise，可以扩展到 multiplicative noise 或 state-dependent diffusion。
  4. 扩展到高维系统（d≫3）的 component-wise + 结构化搜索是未来方向。
  5. 0616-fex 可以探索将两阶段框架应用到 new domain（非湍流）或改进 FEX 本身的搜索效率。
