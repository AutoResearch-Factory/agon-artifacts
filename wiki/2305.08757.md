# Physics Informed Token Transformer for Solving Partial Differential Equations (2305.08757)

- **Year**: 2023.05
- **Authors**: Cooper Lorsung, Zijie Li, Amir Barati Farimani (Carnegie Mellon University, MechE/BME/MLD)
- **Venue**: arXiv (formatted for ACS journal submission)
- **Keywords**: Neural Operators, Physics-Informed Machine Learning, Transformers, PDE Solving, Equation Tokenization, Linear Attention, Fourier Neural Operator, Surrogate Modeling
- **One-liner**: 将 PDE 控制方程 tokenize 成文本序列，用 self-attention 学习方程嵌入，再用 linear attention 将其作为解析驱动的修正叠加到数据驱动的 neural operator 输出上。

## 精华提炼

### 管线/方法
完整的 PITT pipeline：
1. **输入**：数值场（多帧 snapshot）+ 网格信息 + tokenized 控制方程 + 目标时间差
2. **Equation Tokenization**：将 PDE 的每个数学符号（偏导、变量、运算符、数值参数、边界条件类型）解析为 token，用预定义词表映射为 token index。例如 `∂/∂t u(x,t)` → `[Derivative, (, u, (, x, ,, t, ), ,, t, )]` → `[6,0,3,15,0,16,33,14,1,33,1]`。数值截断到 15 位精度。1D 方程 padding 到 500 tokens，2D 到 100 tokens
3. **Token Transformer**（图 1a）：对 tokenized 方程做 Multi-Head Self-Attention，输入时 token 先 shift+scale 到 [-1,1]。两种 embedding 方法：
   - 方法1（† novel embedding）：直接用可学习权重矩阵 W_T1, W_T2, W_T3 将 tokens 映射为 KQV，利用数值 token 间的相关性
   - 方法2（* standard embedding）：标准位置编码 + lookup table embedding，不引入非常规 token 间相关
   - 单层 self-attention，单头（1D 实验）
4. **Linear Attention Update**（图 1b，Algorithm 1）：Token Transformer 输出 T_h 经权重矩阵得到 T_h1, T_h2，作为 attention 的 keys 和 queries；Neural Operator（FNO/DeepONet/OFormer）的输出作为 values V_0。使用 softmax-free Linear Attention：z = Q(K̃^T Ṽ)/n（K̃, Ṽ 做 instance normalization）。Attention 输出与 fractional timestep embedding 拼接后经 MLP 计算更新量：V_l = V_{l-1} + MLP([X_l, t_l])
5. **多层数值更新**：L 层迭代，每层处理 l·t/L 的时间片段。长目标时间（如 fixed-future）用多层（2-20层），短时间步（next-step）用单层
6. **输出**：最终 V_L，经过全连接投影层匹配目标维度

### 关键技术细节
- **底层 Neural Operator**：FNO（4 modes, hidden dim 64 而非原始 256）、DeepONet（branch net [10,128,128], trunk net [100,128,128]）、OFormer（encoder/decoder depth 2, latent channels 64-256）。PITT 使用缩小版的 underlying operator，参数量显著更少
- **训练数据规模**：
  - Heat: 60,000 samples (6 β 值 × 10,000)
  - Burgers': 90,000 samples (6α × 6β × 2,500)
  - KdV: 15,000 samples (6 γ 值 × 2,500)
  - Navier-Stokes: 370 parameter combos × 1-5 initializations, 120-200 frames
  - Poisson: 5,000 samples, 所有 16 种边界条件组合
- **优化器**：Adam，L1 loss（全部实验）
- **学习率调度**：FNO/DeepONet baseline 用 step scheduler（γ=0.5, step=50/100/200），PITT 变体和 OFormer 用 One Cycle LR
- **训练配置**：60-20-20 train-val-test split，5 random splits，early stopping 选最低 validation loss
- **关键超参数**：1D next-step: batch 32-256, lr 1e-3~1e-4, weight decay 1e-2~1e-8；2D NS: batch 4-64, lr 1e-3~1e-4
- **硬件**：GeForce 2080 Ti GPU, PyTorch 1.13.0
- **数值方法对比**：Heat/Burgers 用 RK4(Dormand-Prince) 自适应步长（平均步长 0.035s），KdV 平均步长仅 0.00032s（需 7.33 次自适应细化）

### 核心结果
**1D Next-Step Prediction（MAE ×10⁻³）**：
| | Heat | Burgers' | KdV |
|---|---|---|---|
| FNO (2.4M) | 4.80 | 8.22 | 11.28 |
| PITT FNO† (0.2M) | **0.38** | **0.23** | **8.77** |
| PITT OFormer† (0.4M) | **0.06** | **0.22** | **0.46** |
| PITT DeepONet† (0.2M) | **0.03** | **0.24** | **1.78** |

- PITT 在 Heat/Burgers 上比 FNO 好 10-20 倍，参数量仅 1/12
- PITT DeepONet 在 Heat 上 MAE 仅 0.03×10⁻³，比原始 DeepONet 好 20+ 倍
- KdV 上提升相对较小（方程更复杂，非线性更强）

**1D Fixed-Future + Multi-Equation Training**：PITT 多方程训练（同时学 Heat+Burgers+KdV）在所有三个方程上均优于单方程训练，证明 PITT 能从多系统中学习通用知识。

**2D Navier-Stokes**：PITT 在 next-step MAE 上改进幅度不大（error bar 重叠），但 rollout 稳定性显著提升——在 T=25s, 30s 时仍保持大尺度特征，而 FNO 已产生明显偏离。PITT FNO rollout 最终 MAE 为 0.065（FNO 0.125，近 2x 改进）。

**2D Poisson**：PITT 实现 ~10x 改进（FNO 9.79 vs PITT FNO* 0.85，MAE ×10⁻³）。

**2D Fixed-Future (T=20s, 30s)**：PITT 一致优于 baseline，T=30s 时 PITT FNO* 7.46 vs FNO 8.11（MAE ×10⁻²）。

**Rollout 误差累积**：PITT 在所有 time-dependent 数据集上的长期 rollout 误差显著更低。

**Time-to-Solution**：所有神经网络模型均比数值方法快（Heat: 0.0053s numerical vs 0.00028s DeepONet），KdV 尤其明显（122.6s numerical vs 0.0013s FNO）。

**Attention Map 分析**：修改 viscosity 和 forcing term amplitude 时，self-attention weights 产生明显不同的变化模式，证明 PITT 确实从 tokenized 方程中学习了物理参数信息。修改目标时间时 attention 模式保持近似恒定（novel embedding）或线性叠加（standard embedding）。

### 明确局限
1. **无真机验证**：纯数值模拟，无物理实验验证
2. **PDE 类型有限**：仅测试了 Heat（线性抛物）、Burgers'（非线性双曲）、KdV（非线性色散）、Navier-Stokes（2D 不可压缩）、Poisson（稳态椭圆），无 3D 系统
3. **方程结构冗余**：实验中只有参数变化（viscosity, forcing amplitude 等），方程结构不变。多方程训练中方程结构变了但 token 冗余度高（continuity equation 对所有 NS 样本相同）
4. **Tokenization 粗糙**：novel embedding 方法在数值 token 间引入了非自然的关联（因为直接用可学习矩阵映射而非传统 embedding），standard embedding 则没有利用数值间的有用关联。两种方法各有缺陷
5. **OFormer 在 rollout 中不稳定**：论文直接省略了 OFormer 的 rollout 结果
6. **未使用 recurrent rollout 训练**：仅 next-step 训练 + autoregressive rollout 测试，未采用已知可稳定 rollout 的 recurrent 训练策略
7. **发表场所弱势**：投在 ACS 化学期刊格式而非 ML 顶会（NeurIPS/ICML/ICLR），同行评审标准不同
8. **无消融实验**：没有系统地消融 tokenization 各部分（边界条件 token、时间 token、数值精度等）的贡献
9. **1D 数据集来自 Brandstetter 2022，2D NS 来自 Li 2021 FNO**：数据非自创，且参数范围有限

## Read by: 0616-fex (2026-06-24)

本次审读从 **0616-fex** 视角分析 PITT：

**撞车评估**：不撞车。PITT 的核心贡献是"将 PDE 方程文本作为额外输入，通过 attention 机制融合进 neural operator"，本质上是 **multi-modal operator learning（数值场 + 方程文本）**。0616-fex 如果聚焦于特征提取（从数据中提取物理特征或发现方程），则是互补方向——PITT 需要已知方程作为输入，而 fex 可能是产出方程作为输出。

**可复用的技术要素**：
1. **Equation Tokenization pipeline**：符号解析 + token 词表构建 + padding 策略。如果 fex 需要将方程表示为模型可处理的序列，这是直接可用的方案
2. **Linear Attention numerical update**：将 attention 用作数值积分式更新的机制（z = Q(K̃^T Ṽ)/n），可作为 building block
3. **Attention map 分析方法**：通过修改 token 观察 attention weight 差异来验证模型是否学到物理——这是一种通用的 interpretability 工具，可用于任何将物理信息编码为 token 的模型
4. **PDEBench 数据管线**：快速生成大规模 PDE 训练数据的成熟方案

**差异化方向**：
- PITT 的 tokenization 是 flat token list，没有利用方程的树形/图结构（operator tree）。更结构化的方程表示可能带来更好的泛化
- PITT 假设方程完全已知，不涉及方程发现或特征选择。如果 fex 涉及从候选特征中筛选或发现控制方程，这是 PITT 未触及的
- PITT 的 novel embedding 方法有理论缺陷（引入虚假 token 关联），standard embedding 又丢失了数值间的有效关联——在方程表示方面有明显改进空间
