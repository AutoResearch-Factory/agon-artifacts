# When is a System Discoverable from Data? Discovery Requires Chaos (2511.08860)

- **Year**: 2025.11
- **Authors**: Zakhar Shumaylov, Peter Zaika, Philipp Scholl, Gitta Kutyniok, Lior Horesh, Carola-Bibiane Schonlieb; University of Cambridge, LMU Munich, IBM Research, MCML
- **Venue**: arXiv
- **Keywords**: dynamical systems, discoverability, identifiability, chaos, symbolic discovery, neural operators, analytic functions, conservation laws
- **One-liner**: 有限轨迹能否唯一发现系统，关键取决于混沌和物理先验。

## 精华提炼

### 管线/方法
- 论文研究对象是有限维自治动力系统：\(\dot u = F(u)\), \(u:\mathbb{R}\to\mathbb{R}^d\)。
- 目标不是提出新算法，而是判定从有限条轨迹唯一恢复 \(F\) 的数学条件。
- 作者把传统 parameter identifiability 扩展为非参数函数类中的 discoverability。
- 函数类主要分两类：连续函数 \(C^0\) 与实解析函数 \(C^\omega\)。
- 第一层 reduction：轨迹数据给出 \(F\) 在轨迹像集 \(\mathcal{O}=u(U)\) 上的值。
- 若存在非零函数 \(G\) 在整条轨迹上为 0，则 \(F+G\) 与 \(F\) 对同一数据不可区分。
- 因此 discoverability 被化约为 set of uniqueness 问题。
- 对连续函数，set of uniqueness 等价于轨迹像集在整个状态空间中稠密。
- 对实解析函数，set of uniqueness 更微妙；开集和正 Lebesgue 测度集足够，但不是必要条件。
- 作者引入解析几何与 sheaf 语言，描述轨迹闭包何时能强迫解析函数全局为零。
- 对混沌，作者采用 topological transitivity 加 sensitive dependence 的吸引子定义。
- 真正用于 discoverability 的核心性质是 topological transitivity，因为它通过 Birkhoff lemma 给出稠密轨迹。
- 连续情形的流程是：topological transitivity -> hypercyclic/dense trajectory -> 连续函数在稠密集为 0 则全局为 0 -> 单轨迹可发现。
- 多轨迹连续情形的流程是：若 \(n\) 条轨迹可发现，则这些轨迹闭包必须覆盖空间。
- 若系统不整体 topologically transitive，则空间可分解为若干 proper closed invariant cells。
- 每个 cell 上限制流又是 topologically transitive。
- 因此有限连续可发现性对应一种 Smale-like cell decomposition：每个 cell 至少需要一条代表轨迹。
- 解析情形的流程更复杂：轨迹 \(\gamma\) 的闭包 \(Z=\overline{\gamma}\) 是否为解析函数的 uniqueness set。
- 作者分析 analytic trajectory 的 subanalytic 性质。
- 若轨迹是 subanalytic，则不是解析 uniqueness set。
- 潜在可发现性来自非 subanalytic locus，尤其是轨迹无限接近某个吸引子导致的 type II 点。
- 对来自解析流的轨迹，作者证明 type I non-subanalytic 点不存在。
- 若轨迹落在某个 basin of attraction 中，则 non-subanalytic locus 是轨迹闭包与 attractor 的交。
- 主解析定理：若系统在吸引子 \(A\) 上混沌，且 \(A\) 是解析 uniqueness set，则从一条稠密于 \(A\) 的轨迹可解析发现。
- 一个可检查充分条件是 \(\dim_H(A)>d-1\)，因为非零实解析函数的零集 Hausdorff 维数至多 \(d-1\)。
- Lorenz 系统作为 \(d=3\) 例子，吸引子 Hausdorff 维数约 \(2.06\pm0.01\)，并借助 geometric Lorenz flow 的严格结果推出 \(>2\)。
- 因此经典 Lorenz attractor 在实解析函数类中可发现。
- 论文还处理非混沌系统：若有 analytic first integral，则单条轨迹永远困在一个解析 level set 上。
- 这会构造出 \(H(u)=G(u)-c_0\) 这样的非零解析函数，在轨迹上恒为 0。
- 结论是：存在非平凡 global analytic first integral 的系统不可从轨迹数据解析发现。
- 作者用简单谐振子说明 non-discoverability：\(\dot x=y,\dot y=-x\) 与半径依赖角速度变体可共享同一圆轨迹。
- 但论文没有停在否定结论上，而是展示某些 conservation laws 可以恢复唯一性。
- 第一种恢复方式：若已知解析 law \(H(v,u)\)，且 \(\nabla_v H(F(u),u)=0\)，并在轨迹上一点 \(p\) 处 \(\nabla_v^2H(F(p),p)\) 非奇异，则 implicit function theorem 给出局部唯一，再由解析 identity theorem 推到全局。
- 第二种恢复方式：若限制向量场 Jacobian 为 symmetric 或 skew-symmetric，同时 law 的 Hessian kernel 被轨迹所在解析流形的切空间包含，也能推出全局唯一。
- 对谐振子，作者给出 law \(H=\frac12(x\dot{x}+y\dot{y})^2+\frac12(\dot{x}^2+\dot{y}^2-x^2-y^2)^2\)，在原点外 Hessian determinant 为 \(4\|u\|^4\)。
- 对 skew-symmetric Jacobian 类，只需 \(H=\frac12(x\dot{x}+y\dot{y})^2\) 即可在圆轨迹上恢复唯一性。

### 关键技术细节
- 主要定理依赖 Picard-Lindelof 流存在唯一性、Birkhoff topological transitivity lemma、set of uniqueness、real analytic geometry。
- 连续函数关键命题：\(\mathcal{D}\) 稠密当且仅当连续函数在 \(\mathcal{D}\) 上为零蕴含全局为零。
- 解析函数关键命题：正 \(d\)-维 Lebesgue measure、包含开集、或 Hausdorff 维数 \(>d-1\) 的集合都是 uniqueness set。
- 解析轨迹部分引入 ideal sheaf \(\mathcal{I}_Z\)，表示在轨迹闭包 \(Z\) 上消失的解析函数芽。
- 使用 Cartan Theorem A 与 Oka coherence theorem 来联系 coherent subsheaf 和 global analytic sections。
- 使用 pushforward sheaf \(\phi_*\mathcal{O}_D\) 判断轨迹像的 subanalytic 性质。
- 论文区分 type I 点：存在无收敛子列的时刻序列反复精确回到同一点。
- 论文区分 type II 点：存在无收敛子列的时刻序列趋近某一点但不精确重复。
- 对解析流，periodicity 会让 type I 轨迹变成 proper image，从而 subanalytic，所以 type I 不提供可发现性来源。
- Lorenz 结论依赖两步外部事实：geometric Lorenz flow Hausdorff dimension \(>2\)，以及 Tucker 对原始 Lorenz 方程的 geometric model 有效性。
- 该论文几乎没有数值实验、训练数据、优化器或 benchmark 表；可信度主要来自数学证明。
- 文中 figure 主要起解释作用：Lorenz ambiguity/uniqueness、pendulum vs double pendulum、sets of uniqueness、cell decomposition、first-integral foliation。

### 核心结果
| 结论 | 条件 | 输出 |
|------|------|------|
| 连续单轨迹可发现 | 系统在整个域上混沌/拓扑传递，有稠密轨迹 | \(C^0\) 类中唯一确定 \(F\) |
| 连续单轨迹反向 | 单轨迹在 \(C^0\) 中可发现 | 蕴含 topological transitivity |
| 连续多轨迹 | \(n\) 条轨迹可发现 | 空间分解为 \(k\le n\) 个 closed invariant cells，每个 cell 拓扑传递 |
| 解析吸引子可发现 | chaotic attractor 是 uniqueness set | 一条稠密吸引子轨迹足够 |
| 解析维数充分条件 | \(\dim_H(A)>d-1\) | attractor 是 uniqueness set |
| Lorenz 例子 | \(d=3\), Lorenz attractor dimension \(>2\) | 经典 Lorenz analytically discoverable |
| First integral 负结果 | 存在非平凡 global analytic first integral | 任何轨迹都不是解析 uniqueness set |
| Conservation law 正结果 | Hessian 非奇异或 Jacobian 类受限 | 非混沌系统可由先验恢复唯一性 |

- 最反直觉结果：通常认为 chaos 使预测困难，但这里 chaos 反而是 trajectory data 含信息量足够的条件。
- 对 scientific ML 的解释：天气、流体等混沌 benchmark 可能天然满足唯一性，使方法看起来更可靠。
- 对 digital twins 的警告：工程系统常被设计成稳定、可控、非混沌，纯轨迹数据可能根本不足以唯一恢复模型。
- 对 neural operators 的解释：在黑箱模型中，discoverability 更像 out-of-distribution generalization 的上界条件。
- 对 symbolic discovery 的解释：若 uniqueness 不成立，任何 symbolic regression 成功拟合训练轨迹都不能保证发现了真实物理方程。

### 明确局限
- 论文主要解决 uniqueness，没有解决 Hadamard well-posedness 中的 stability。
- 噪声、有限采样、数值微分误差、模型 misspecification 基本不在主定理范围内。
- 连续情形的结论对状态空间覆盖要求极强，实践中有限离散采样只能近似 dense trajectory。
- 解析情形虽然比连续情形弱，但依赖真实系统属于 \(C^\omega\) 函数类。
- Lorenz discoverability 依赖外部严格动力系统结果，不是本文从零证明 Lorenz attractor 维数。
- 作者承认 PDE 和高阶 ODE 扩展仍未形式化；operator learning 的常见 PDE setting 与本文有限维 ODE setting 有错位。
- Conservation law 恢复唯一性的条件较强：非奇异 Hessian 意味 law 已经把候选 \(F\) 限到离散/局部唯一。
- 对称或反对称 Jacobian 限制覆盖 gradient flows 与 separable Hamiltonian systems，但不覆盖一般工程系统。
- 反向解析定理和 attractor/basin 部分有较强动力系统假设，实际系统是否满足需要单独证明。
- 论文没有提供可运行算法，也没有给出如何从 noisy finite observations 验证 attractor dimension 或 topological transitivity 的 pipeline。

## Read by: 0616-fex (2026-06-16)

从 `0616-fex` 视角看，这篇论文不是一个可直接复现的 discovery 算法，而是一条理论边界：有限轨迹能否支持可靠发现，首先取决于轨迹像集是不是候选函数类的 uniqueness set。它最可能撞车的 claim 是“只靠有限轨迹数据就能可靠发现非混沌/稳定系统的真实方程”；本文给出简单谐振子和 first integral 的负结果，说明这类 claim 需要物理先验或更强实验设计支撑。它能为 `0616-fex` 提供三个可用抓手：一是用 chaos/state-space coverage 解释 benchmark 偏差；二是把守恒律写成 \(H(v,u)\) 并检查 Hessian/Jacobian 限制来恢复唯一性；三是把验证问题前置为“生成模型与检验 gate 是否能排除等价拟合但物理错误的模型”。差异化空间在 noisy finite samples、主动选择初值/控制输入、PDE setting、以及把 uniqueness 理论落成可操作的实验协议。
