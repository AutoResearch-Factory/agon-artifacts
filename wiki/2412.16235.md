# Utilizing Causal Network Markers to Identify Tipping Points ahead of Critical Transition (2412.16235)

- **Year**: 2024.12 (arXiv); published in *Advanced Science* (Wiley)
- **Authors**: Shirui Bian, Zezhou Wang, Siyang Leng, Wei Lin, Jifan Shi\* (通讯) — Fudan University (School of Mathematical Sciences / Shanghai Center for Mathematical Sciences / Research Institute of Intelligent Complex Systems / Shanghai AI Lab)
- **Venue**: arXiv preprint + *Advanced Science*
- **Keywords**: early-warning signal, critical transition, tipping point, causal network marker, Granger causality, transfer entropy, dynamical network biomarker (DNB), bifurcation, iEEG / epileptic seizure
- **One-liner**: 用 Granger 因果 / 转移熵构造"因果网络 marker"(CNM), 理论证明系统趋近分岔点时 dominant→non-dominant 方向因果强度确定性趋于 0、marker 发散, 作为比传统 DNB 更抗噪的早期预警信号。

## 精华提炼

### 管线/方法
CNM 框架的完整流程（对应论文 Figure 1）：

1. **输入**：多节点系统的多元时间序列 $\boldsymbol Z^t=(z_1^t,\dots,z_n^t)$（benchmark 由 Langevin/反应扩散方程仿真产生；真实数据为 iEEG 多导联电压）。
2. **分组（K-means on variance）**：对给定时间窗内每个节点的**平均方差**做 $K$-means 聚类，把节点分成 **DG (dominant group)** 和 **NDG (non-dominant group)**。理论依据：临界点附近 dominant 变量方差趋于无穷。基因/生态网络里 DG 可由特征值分析先验确定（对应最大特征值 $\lambda_{\max}$ 的节点）；Turing / 真实数据里 DG 未知，纯靠 K-means 划分（作者明确承认此划分"有主观性"）。
3. **算因果强度 cs(x_j→x_i)**：对 DG→NDG 的每条有向边算因果强度，可选两种指标：
   - **CNM-GC**（线性因果）：Granger causality，$\text{GC}_{x_j\to x_i}=\log\frac{\E[\xi_1^2]}{\E[\xi_2^2]}$，比较 $H_0$（只用 $x_i$ 自回归）与 $H_1$（加入 $x_j$）的残差方差比。
   - **CNM-TE**（非线性因果）：transfer entropy，$\text{TE}_{X\to Y}=H(Y_{t+1}\mid Y_t)-H(Y_{t+1}\mid Y_t,X_t)$。高斯过程下 TE 退化为 GC。
4. **合成 marker**：
   $$\text{CNM(Net)} := \frac{|\text{DG}|\cdot|\text{NDG}|}{\sum_{j\in\text{DG},\,i\in\text{NDG}}\text{cs}(x_j\to x_i)}$$
   分母中只要有一条 DG→NDG 因果强度趋于 0，整个 marker 就**爆炸（blow up）**。
5. **输出/判读**：把 marker 的显著上升作为 tipping point 的预警信号。真实数据里用 5 秒 / 12 秒滑动平均平滑原始 marker。

### 关键技术细节
- **核心理论（"因果消失"定理）**：离散动力系统 $\boldsymbol Z^{t+1}=\boldsymbol f(\boldsymbol Z^t;\boldsymbol P)$ 在不动点 $\bar{\boldsymbol Z}$ 处线性化 $\boldsymbol X^{t+1}=\boldsymbol A\boldsymbol X^t+\boldsymbol\Gamma^t$，对 Jacobian $\boldsymbol A=\boldsymbol S\boldsymbol\Lambda\boldsymbol S^{-1}$ 做对角化，令 $\boldsymbol Y^t=\boldsymbol S^{-1}\boldsymbol X^t$。设 $y_1$ 为唯一 dominant 变量（$\boldsymbol P\to\boldsymbol P_c$ 时 $\text{Var}(y_1)\to\infty$，且 $y_1$ 与其它变量的 PCC → 0）。对原空间变量 $x_i=\sum_k s_{ik}y_k$，用 $H_0/H_1$ 模型分四种情形讨论 GC：
  - (a) $s_{i1}=0, s_{j1}\neq0$（$j$ 连 dominant、$i$ 不连）→ $\text{GC}_{x_j\to x_i}\to 0$（**这是 marker 发散的来源**）；
  - (b) $s_{i1}\neq0, s_{j1}=0$ → GC 有界；
  - (c) $s_{i1}\neq0, s_{j1}\neq0$ → GC 有界；
  - (d) $s_{i1}=0, s_{j1}=0$ → GC 不变。
  结论：临界点处 **DG→NDG 的因果 → 0，NDG→DG 有界变化，NDG 内部不变**。这是一个**确定性**（deterministic）结论，不是统计涨落。
- **Benchmark 1 — 五基因调控网络**：5 维 Langevin 方程，稳定不动点 $\bar{\boldsymbol Z}=(1,0,1,3,2)$，离散线性化特征值 $(0.74^{|P|},0.55,0.37,0.20,0.14)$，$P\to0$ 时 $\lambda_{\max}=0.74^{|P|}\to1$ 触发相变。理论 DG = $\{z_1,z_2\}$。单参数分岔。
- **Benchmark 2 — 生态互惠网络**：pollinator/plant 二部图 → 两个单类网络（$A_{ij}$ 邻接）。Barabasi–Barzel–Barabási 的普适韧性降维（mean-field 得到一维 $x_\text{eff}$ 与 $\beta_\text{eff}$ 韧性函数）。分岔由全局扰动 $A_{ij}\to w_{ij}A_{ij}$ 驱动（$w_{ij}=\text{Debuff}$，噪声下阈值在 0.52 附近波动）——**多因子/交互驱动的分岔，tipping point 不固定**，是传统 EWS 难处理的场景。
- **Benchmark 3 — Turing 扩散网络**：Beddington–DeAngelis 捕食-被捕食反应扩散系统，$11\times11=121$ 格点，每格点当一个变量。参数 $r=0.5,\varepsilon=1,\beta=0.6,B=0.4,\eta=0.25,\omega=0.4,D_2=1$，$D_1$ 与 $K$ 控制空间 pattern，$D_1$ 增大穿越 Turing 分岔（$K\approx2$）。仿真跑 800 秒，1 秒时间窗聚合快照。**空间分岔**——DG 完全未知，靠 K-means。
- **真实数据 — iEEG 癫痫**：短时程颅内脑电，多个患者多次发作（如 ID1Sz4, ID7Sz2, ID5Sz2, ID16Sz1）。三个候选 marker：DNB、CNM-GC、CNM-TE，统一用 DG/NDG 框架 + 统一评估标准。
- **DNB 基线公式**：$\text{DNB(Net)}=\frac{\text{SD}_d\cdot\text{PCC}_d}{\text{PCC}_o}$（DG 内平均标准差 × DG 内平均相关 / DG-NDG 间平均相关）。DNB 直接受节点自身方差与 PCC 影响，对噪声敏感；CNM 只看节点间因果，抗噪。

### 核心结果
- **iEEG 准确率对比（Table 1）**：All-three-agree 43% | **CNMs 组合 92%** | DNB 79% | CNM-GC only 68% | CNM-TE only 74%。→ CNM 组合显著优于 DNB。
- **三类 benchmark 全部成功**：基因（内部动力学分岔）、生态（交互分岔）、Turing（空间分岔）——都在分岔点附近 CNM-GC 与 CNM-TE 显著上升，且 Turing 里只在 $K\approx2$ 分岔处 peak（有选择性，不是恒升）。
- **抗噪性**：DNB 的方差项在高噪声下剧烈涨落导致失效（例 ID7Sz2），CNM 因不含节点自身动力学而稳定给出预警峰。
- **反直觉/有趣发现（对本课题最关键）**：**CNM-GC 与 CNM-TE 可能不一致**——有的发作只有 GC 报警（线性因果主导，例 ID7Sz2），有的只有 TE 报警（非线性因果主导，例 ID5Sz2）。作者据此提出可用"哪个 marker 报警"来**给发作做因果类型分类**（线性 / 非线性 / 混合）。少数罕见个例还观察到 CSD（critical slowing down）现象（例 ID5Sz1），说明因果模式不止两种。

### 明确局限
- **无真机/无标准 benchmark 的统一评估**：作者自己承认 DG/NDG 的 K-means 划分"无论怎么分都有主观性"；"92% vs 79%"的评估标准（怎么算一次"成功预警"）在正文里没有严格定义，判据偏软。
- **无代码/artifact 公开**：正文只说伪代码在 SI；iEEG 说是"开源癫痫数据集"但未指明具体版本/来源。基准模型给了完整方程和参数（可复现），但真实数据管线欠详。
- **无置信区间/无正式统计检验**：只有点估计的准确率，没有 CI、没有显著性检验、没有 surrogate data 对照。
- **理论假设是"慢速接近平衡分岔"**：整套理论建立在系统缓慢逼近 codimension-one 分岔、可线性化、Jacobian 可对角化、存在唯一 dominant 特征值之上——**这是慢变、近平衡的假设**。
- **失败模式**：GC/TE 之外的因果爆炸无法捕捉；高维临界动力学无法被低维刻画；部分个例三个 marker 全升（43%）说明区分度有限。

## Read by: 0710-causal-scs (2026-07-10)

**审读视角**：本课题（因果效应不稳定性作为强对流预警指标）把这篇当作头号 closest prior work 精读。结论如下。

**撞车评估（细化到 claim）**：
- CNM 的核心 claim = "临界点附近 DG→NDG 的因果强度**确定性**趋于 0，据此构造的 marker 发散，可预警 tipping point"。这是**动力学 + 确定性**结论，有解析证明（四情形 GC + 特征值分解）。
- 本课题的核心 claim = "因果效应**估计**在假设违背/方法差异/滑窗下的**统计不稳定性**（方差/突变/DAG 结构变化）本身可作预警信号"。这是**认识论 + 统计**结论。
- **两条 claim 在现象层面重叠**（分岔附近因果量都会剧烈变化），**在机制层面可分**（确定性真值变化 vs 估计过程统计脆弱性）。但：CNM 的机制直接蕴含"分岔附近滑窗因果估计必然变化"，因此本课题观测到的"不稳定性升高"**极可能被 CNM 机制解释掉**——这正是 idea 自身列出的 strongest objection，也是 idea 现稿最大的未证成缺口。

**尺度错配（本课题独有的机会 + 风险）**：CNM 全部案例（基因/生态/癫痫）+ 本课题 Stage 1 的 Lorenz pilot 都是**慢变、近平衡、缓慢参数漂移穿越分岔**的设定，正是 CNM 确定性定理成立的前提。但**强对流启动是小时级快变、非平衡过程**——CNM 的线性化 + 唯一 dominant 特征值假设很可能失效。
