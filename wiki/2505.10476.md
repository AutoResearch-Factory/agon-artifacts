# Causal discovery on vector-valued variables and consistency-guided aggregation (2505.10476)

- **Year**: 2025.05
- **Authors**: Urmi Ninad (TU Berlin), Jonas Wahl (DFKI, equal contrib.), Andreas Gerhardus (DLR, equal contrib.), Jakob Runge (Uni Potsdam / TU Berlin / ScaDS.AI)
- **Venue**: TMLR-formatted preprint (arXiv, `\usepackage[preprint]{tmlr}`; no accepted-venue confirmation found in source)
- **Keywords**: causal discovery, vector-valued variables, spatio-temporal causal discovery, causal abstraction, aggregation consistency, conditional independence testing, PC algorithm, PCMCI, SAVAR
- **One-liner**: Formalizes three approaches to causal discovery over vector-valued variables, proves when each is sound, and introduces testable "aggregation consistency" scores plus an adaptive wrapper (Adag) to make dimension-reduced causal discovery reliable without ground truth.

## 精华提炼

### 管线/方法

The paper is a theory + synthetic-experiment paper, not a learned pipeline; its "pipeline" is a decision procedure over three competing strategies for causal discovery (CD) when the causal variables of interest are themselves vectors (e.g. a spatial grid, a set of indices):

1. **Problem setup**: $N$ vector variables $\mathbf{X}^i$ (dimension $d_i$ each) generated by a vector-level SCM $\mathbf{X}^i := f^i(Pa(\mathbf{X}^i), \boldsymbol{\eta}^i)$ entailing an acyclic macro graph $\mathcal{G}$, assumed Markov + faithful + causally sufficient at the vector level (Assumption "CD assumptions"). The internal relationship between the scalar components of a single $\mathbf{X}^i$ is left deliberately unconstrained — it can be a DAG, contain latent confounding, contain cycles, violate faithfulness, or be "acausal" (MRF / Bayesian network with no causal reading). This taxonomy of "internal dynamics" is the paper's organizing device for Contribution 1.
2. **Component-wise / "S2V" approach** (Algorithm `S2V(A, E)`): run a scalar CD algorithm $\mathcal{A}$ (e.g. PC) over *all* scalar components jointly to get a micro graph $\widehat{\mathcal{G}}_{micro}$, then coarsen it to a macro graph via an **edge-aggregation map** $\mathcal{E}$ — either a *majority* vote over directed vs. undirected edge counts between component pairs, or a *conservative* variant. A second meta-algorithm `S2V-2` (only skeleton phase shared, orientation redone at macro level via a triple-based collider rule + Meek rules) is relegated to the appendix.
3. **Aggregation approach**: apply a dimension-reducing **aggregation map** $g_i: \mathbf{X}^i \mapsto \mathbf{Z}^i$ (e.g. elementwise average, first principal component) to each vector variable, then run CD directly on the low-dimensional $\mathbf{Z}$.
4. **Vectorized approach** (the paper's recommended default): run CD directly on the vector variables themselves using a *multivariate* conditional-independence (CI) test — no coarsening, no dimension reduction. Framed as "deceptively simple" but under-used relative to (2)/(3).
5. **Aggregation-consistency layer** (Contribution 2, the operational core): since (4) needs reliable high-dimensional CI tests (hard in finite samples) and (3) has no soundness guarantee without ground truth, the paper defines three data-computable **consistency scores** ($c_{ind}$, $c_{dep}$/$\bar c_{dep}$, joint $AC$) that estimate how "valid" a chosen aggregation map $g$ is, purely from the CI statements already produced by running the CD algorithm on $\mathbf{Z}$, cross-checked against the vector-level data $\mathbf{X}$.
6. **Adag wrapper** (Algorithm `Adag(q, α_q, A, g^m)`): given a *tunable* aggregation map $g^{\mathbf{m}}(\cdot)$ (a family indexed by a dimensionality knob $\mathbf{m}$, e.g. "first $m$ principal components," monotonically more expressive as $m$ grows, up to $m=d_i$ where it becomes the identity/vectorized case), a chosen consistency score $q$, and a target value $\alpha_q$: initialize $\mathbf{m}=0$, run $\mathcal{A}$ on $g^{\mathbf{m}}(\mathbf{X})$, compute $\widehat\alpha_q$; while $\widehat\alpha_q < \alpha_q$ and $\mathbf{m}$ has not reached its per-variable max, increment $\mathbf{m}$ and repeat; otherwise stop and return the current output graph. This is a closed loop between "aggregate → run CD → score it → aggregate less" with no learning involved — it is a search over a 1-parameter family of hand-specified aggregation maps, not a learned representation.
7. **Experiments**: synthetic linear-Gaussian SCMs over vector variables with tunable internal (Markov-random-field) and external (DAG) edge densities compare all approaches by adjacency/edgemark precision-recall (PC algorithm, non-time-series and PCMCI+ time-series variants), then an application to the SAVAR synthetic spatio-temporal climate benchmark (Tibau et al.) comparing vectorized-PCMCI vs. Adag-PCA-PCMCI.

### 关键技术细节

- **Internal-dynamics soundness table** (Section 4): for `S2V`/`S2V-2` combined with PC-family algorithms —
  - *Internal DAG*: sound (reduces to standard vector-CD correctness, stated as "Remark: Correctness of Vector CD" — any sound+complete scalar CD algorithm remains sound+complete at the vector level under the paper's assumptions).
  - *Internal latent confounding (ADMG, no cycles)*: **not provably sound** for PC — spurious edges can survive via inducing paths / possible-d-separating sets (`Poss(X,Z)`) that fall outside adjacency sets; a worked counterexample ($X_3 \not\perp Z_3 \mid \{X_1,X_2,Z_1,Z_2\}$ but $X_3 \perp Z_3 \mid Y$) shows PC can get lucky "for the wrong reasons." Formal fix requires FCI-style non-local separating-set search (more CI tests). Empirically (Fig. `PC_conflicts`), orientation *conflicts* across vector variables from majority edge-aggregation are rare in practice — the paper's Lemma (`no_inducing`, proved by contradiction via acyclicity of $\mathcal{G}$) rules out certain inducing-path configurations across (but not within) vector variables.
  - *Internal cycles (no confounding)*: **sound**, leaning on Mooij & Claassen's σ-separation result that PC is sound/complete for the acyclification of a cyclic SCM (Bongers et al.'s ADMG-equivalent acyclification operator); worked example via the cyclic SCM $Y_1 := X\cdot Y_2+\eta_{Y_1}, Y_2 := Z\cdot Y_1 + \eta_{Y_2}$.
  - *Internal faithfulness violations (determinism/XOR)*: **not sound** — concrete counterexample: $\mathbf{X}=(X_1,f(\eta_{X_1}))$ deterministic, $\mathbf{Y}:=g(\mathbf{X},\eta_Y)$; `S2V`+PC returns a fully disconnected macro graph (misses the edge) while vectorized-PC correctly detects $\mathbf{X}\not\perp \mathbf{Y}$ via the graphoid-axiom argument $X_1\not\perp Y \Rightarrow \mathbf{X}\not\perp\mathbf{Y}$.
  - *Acausal internal structure (MRF/Bayes net)*: soundness "cannot be proven generally," case-by-case.
- **Aggregation faithfulness / sufficiency** (Section 5, Defs.): $g$ is *faithful* if $\mathbf{Z}^i\perp\mathbf{Z}^j\mid \mathcal{S}_Z \Rightarrow \mathbf{X}^i\perp\mathbf{X}^j\mid\mathcal{S}_X$ (no accidental independence-by-cancellation — counterexample given with a linear combination $-\beta X^2_1+\alpha X^2_2$ that exactly zeroes out a real dependency); *sufficient* if the converse holds (no loss of separating information — counterexample from Wahl & Ninad's own prior work, §8.2.1, reused directly). A *valid* aggregation map satisfies both, equivalently $\mathcal{I}(g(\mathbf{X}))\equiv\mathcal{I}(\mathbf{X})$ (independence models coincide).
- **Independence score** $c_{ind}=|\mathfrak{C}^{ind}|/(|\mathfrak{C}^{ind}|+|\mathfrak{I}^{ind}|)$: precision-style — of the CI statements the CD algorithm found true on $\mathbf{Z}$, what fraction also verify true on the corresponding $\mathbf{X}$. Proven $=1$ iff $g$ is aggregation-faithful, *given* the CD algorithm is **causal input complete** (a completeness notion borrowed/adapted from Verma & Pearl's "causal input list"/"stratified protocol" — informally, every local-Markov-property CI statement was actually tested). Any sound+complete algorithm can be trivially augmented to be causal-input-complete (Remark, `augmented_alg`) by testing the residual local-Markov statements post hoc.
- **Dependence score** $c_{dep}$ (complete, generally intractable) and its practical proxy $\bar c_{dep}$ (**Algorithm** `cdep_eff`): for each adjacent pair in the aggregate skeleton, take the *most significant* (lowest p-value) dependence statement found by the CD algorithm and test its vector-level counterpart; increment score on agreement, normalize by number of adjacent pairs. $c_{dep}=1$ iff $g$ is aggregation-sufficient (given a *complete* testing strategy — $\bar c_{dep}$ only approximates this).
- **Joint $AC$ score** = arithmetic mean of $c_{ind}$ and $c_{dep}$ (or $\overline{AC}$ using $\bar c_{dep}$). **Theorem**: $AC=1 \Leftrightarrow g$ valid (given causal-input-completeness + complete dependence strategy). **Corollary**: $c_{ind}=1 \Rightarrow sk(\mathcal{G})\subseteq sk(\mathcal{G}_Z)$ (aggregate skeleton is a conservative superset of the truth) — useful even when full validity can't be certified.
- **Adag soundness theorem**: in the infinite-sample limit, with a causal-input-complete $\mathcal{A}$ and a tunable $g^{\mathbf{m}}$ satisfying "aggregation graph existence," targeting $AC=1$ recovers the exact ground-truth CPDAG; targeting $c_{ind}=1$ recovers at least the correct skeleton.
- **Multivariate CI test benchmark** (Appendix): compares Max-Corr (Bonferroni-corrected max pairwise correlation of OLS residuals), GCM (Generalized Covariance Measure, Shah & Peters, with Gaussian multiplier bootstrap), HSIC (kernel), and a Box-M-test adaptation; Box-M dropped for being unreliable in high dimensions; **GCM+OLS residualization chosen as the paper's default multivariate CI test** for all vectorized/aggregate experiments, citing its best type-I-error/power trade-off at higher dimensions among the options tried.
- **Experimental configuration**: synthetic linear-Gaussian SCMs, internal dynamics = Markov random field with density $int_{dens.}$, cross-vector DAG with edge probability $1/(d_{macro}-1)$ (density $ext_{dens.}$), coefficients drawn from $[c,0.5]$ with $c\in[-0.5,0]$ (controls effect-cancellation risk under averaging), 100 repetitions with standard-error bars. SAVAR experiment: PCMCI (lag-only PCMCI+) as the CD algorithm, $d_{macro}=3$ climate modes, $d_{micro}=49$ ($7\times7$ grid) per mode, $T=200$, $\tau_{max}=1$, Adag targets $c_{ind}\ge0.8$ with first-$m$-PCA as the tunable map.

### 核心结果

- Averaging aggregation degrades relative to vectorized-PC specifically as the coefficient interval becomes more *symmetric* around zero (more cancellation opportunity) — direct empirical confirmation of the aggregation-faithfulness failure mode.
- PCA aggregation degrades relative to vectorized-PC as cross-vector edge density increases, because the retained principal component increasingly fails to align with the causally relevant subspace; adjacency precision of `pca-PC` decreases with $ext_{dens.}$ while `vec-PC` is unaffected.
- Component-wise (`S2V`) has systematically lower adjacency precision, marginally lower edgemark recall, and much higher runtime than vectorized-PC as internal density grows (many more, mostly irrelevant, CI tests).
- $c_{ind}$ **empirically tracks adjacency recall and SHD of the ground-truth-unknown aggregate graph** across a range of internal dimensions — this is the paper's main "it actually works" evidence for the central claim that the score is a usable proxy for validity without ground truth.
- Adag beats plain vectorized CD at low sample sizes / high internal dimension for all three scores; at higher sample sizes vectorized CD catches up unless the target $\alpha_q$ is retuned upward — i.e. Adag's advantage is sample-size-dependent, not universal. Independence-score-tuned Adag outperforms dependence/AC-tuned Adag in their experiments, which the authors trace to the incompleteness of the practical $\bar c_{dep}$ strategy rather than a fundamental property of the dependence score.
- SAVAR result: Adag-PCA-PCMCI gains adjacency/edgemark precision over vectorized-PCMCI "while not sacrificing severely in recall" — the one result that most resembles a realistic (if still fully synthetic) spatio-temporal use case.

### 明确局限

- **No real observational data anywhere** — every experiment, including the climate-flavored one, uses the synthetic SAVAR generator; the paper's own figure caption admits the SAVAR summary graph shown is "a toy-model... no real teleconnections are shown."
- Authors explicitly flag that the reliability of *all three* consistency scores is bottlenecked by multivariate CI test quality, and that this is exactly worst in the regime (very high-dimensional vectors) where aggregation is most needed — called an "immitigable" challenge in their own conclusion, not just an incidental caveat.
- The practical dependence score $\bar c_{dep}$ is an *incomplete* heuristic (single most-significant statement per pair); only the theoretical "complete" $c_{dep}$ carries the soundness guarantees, so the deployed Adag pipeline's guarantees are weaker than the theorems suggest.
- Constraint-based CD only (PC/PCMCI family with explicit CI-test bookkeeping); score-based, FCI-family-for-general-latent-confounding, and non-constraint-based approaches are only gestured at ("the general philosophy is easily extendable") without proof or experiment.
- Cross-vector-variable latent confounding is excluded by assumption (causal sufficiency required *between* vector variables); only *within*-vector confounding is analyzed.
- Aggregation maps must belong to a hand-specified "tunable" family (monotone-dimension, invertible-reparametrization-equivalence-stable, Remark `bijective`); no representation-learning / causal-representation-learning approach is considered — explicitly out of scope ("we do not focus on causal representation learning").
- The paper's central conceptual move — arguing that CD-task "aggregation validity" (a purely observational, independence-model-equivalence notion) can be decoupled from the stronger interventional "causal abstraction consistency" (Rubenstein et al.) — is argued pragmatically/philosophically, not proven as a formal bridging theorem between the two literatures. This is presented as a stance the reader must accept, not a corollary.
- No public code repository is referenced for the paper's own implementation (Adag, the three scores, the vectorized/component-wise/aggregate experiment harness); only the external SAVAR generator repo is linked. Reproducibility rests on re-implementing from pseudocode.

### 跟我们的关系

Direct methodological overlap risk for any `causal-scs-indicator`-style idea: this paper's core deliverable — a **data-computable, ground-truth-free indicator of whether a dimension-reduced/aggregated causal representation is trustworthy**, built from precision-style ($c_{ind}$) and recall-style ($c_{dep}$) verification of CI statements against the underlying finer-grained data — is conceptually the same move as building a "summary causal graph" consistency indicator, just applied to *spatial/dimensional* aggregation (vector → scalar/lower-dim proxy) rather than *temporal* aggregation (time series → summary/window-marginalized graph). If the target idea is about scoring/validating *summary causal graphs* specifically (à la Assaad's line, see `2407.07934` below), the collision is at the framing level ("test aggregate-vs-fine-grained independence-model equivalence without ground truth") rather than at the exact construction — this paper's scores are defined over *contemporaneous* vector aggregation, not over the lag/time-marginalization that defines a summary causal graph in the time-series literature. Differentiation angles worth checking against the idea: (a) does the idea's indicator handle latent confounding *between* the summary nodes (this paper assumes causal sufficiency at the macro level, matching Ferreira & Assaad's gap that they instead target); (b) does the idea reuse or extend this paper's independence/dependence/AC score triplet vs. proposing a genuinely different statistic; (c) does the idea's setting allow a "tunable" dimension knob the way Adag needs, or is the summary-graph aggregation intrinsically discrete/non-tunable. The Adag wrapper pattern (iteratively de-aggregate until a target score is hit) is a reusable algorithmic template regardless of exact score choice. This paper, its precursor `2306.07047` (Wahl/Ninad/Runge foundations), and `2407.07934` (Ferreira & Assaad) together should be treated as the closest prior art cluster for the topic and read together, not in isolation.

## Read by: 0710-causal-scs (2026-07-11)

Read for the `causal-scs-indicator` topic. This paper is high-relevance prior art at the framing level: it builds a ground-truth-free, data-computable indicator (independence score / dependence score / joint AC score) for whether a coarsened causal representation is trustworthy, plus an adaptive wrapper (Adag) that walks a tunable-dimension aggregation map toward a target score. If the current idea proposes a "causal-scs-indicator" for *summary causal graphs* in the time-series sense (temporal marginalization over lags/clusters, à la Assaad), the overlap is conceptual/structural (same "test aggregate-vs-fine-grained independence-model equivalence" move) rather than a literal duplicate, since this paper's aggregation is spatial/dimensional and contemporaneous, assumes causal sufficiency at the macro level throughout, and never engages the lag-marginalization structure specific to summary causal graphs. Recommend explicitly citing this paper (and its precursor `2306.07047`) as related work and using it as the yardstick for what "differentiation" needs to look like: our indicator should either (a) target the lag/window-marginalization setting this paper does not cover, (b) handle latent confounding at the macro/summary level (this paper assumes sufficiency, `2407.07934` is the paper that instead targets that gap), or (c) propose a genuinely different statistic rather than a relabeled independence/dependence-score pair. The Adag "iteratively de-aggregate until score target is hit" pattern is worth citing as a reusable algorithmic template independent of which score is used.
