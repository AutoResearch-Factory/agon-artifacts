# Symbolic recovery of PDEs from measurement data (2602.15603)

- **Year**: 2026.02
- **Authors**: Erion Morina (University of Graz IDea_Lab), Philipp Scholl (Aleph Alpha Research), Martin Holler (University of Graz IDea_Lab / NAWI Graz / BioTechMed Graz)
- **Venue**: arXiv
- **Keywords**: symbolic recovery, PDE discovery, rational neural networks, ParFam, all-at-once inverse problems, identifiability, noisy incomplete measurements, function-space analysis
- **One-liner**: 用有理符号网络从噪声测量恢复可解释PDE律。

## 精华提炼

### 管线/方法
- 论文目标是同时恢复未知状态 `u` 和未知物理律 `f`，形式为 `partial_t u(t,x) = f(t, J_kappa u(t,x))`。
- 输入不是完整干净场，而是间接、低分辨、带噪测量 `y^m ~= K^m u^\dagger`。
- 这里 `K^m` 是 reduced measurement operator，理想极限是 injective full measurement operator `K^\dagger`。
- 作者采用 all-at-once 反问题形式，不先拟合状态再拟合方程，而是在同一个优化问题里同时优化状态和物理律参数。
- 核心优化问题是最小化三项：PDE residual、measurement mismatch、regularization。
- 形式上是 `lambda^m ||partial_t u - S_theta(J_kappa u)||_W^q + mu^m ||K^m u - y^m||_Y^r + R(u,theta)`。
- 物理律由 symbolic network `S_sigma^theta` 表示，而不是普通黑箱 MLP。
- symbolic network 的层间映射是有理函数，也就是多项式分子除以正多项式分母。
- 激活函数是用户给定 base functions，例如 `sin`、`cos`、`exp`，用于表达非代数关系。
- 最后一层有 rational skip connection，把输入变量直接送到输出 rational map，覆盖 ParFam 的实际设计。
- 该框架把 ParFam 和 EQL-type symbolic networks 都纳入统一形式。
- 分母多项式被限制为全域正，并用 `Q^epsilon(d,n)` 和分母系数归一化保证参数集闭性与数值稳定性。
- 训练时用 `L^1` 风格参数正则倾向选择稀疏、可解释的符号表达式。
- 理论层面先在函数空间中处理 Nemytskii operator，再讨论离散数值实验。
- 这点是论文主张的关键差异：重建保证不是只在离散网格或有限样本回归层面成立。
- 读法上要把它看成 inverse problems + symbolic regression 的交叉论文，而不是单纯 PDE-FIND benchmark 论文。
- 主 theorem 要求真实物理律能被用户选定的 symbolic network architecture 表示。
- 若满足这个 representability 假设，则在 noiseless/full-measurement 极限中，解序列恢复一个与 PDE 相容的物理律。
- 如果物理律不唯一，恢复结果不保证是真实生成律本身，而是正则化最小的 admissible parameterization。
- 如果附加 identifiability 条件成立，即 `f1(Ju)=f2(Ju)` 蕴含 `f1=f2`，则能恢复唯一真实物理律。
- 数值实现用 ParFam 作为 `S_theta`，状态 `u` 直接作为离散网格上的函数值训练。
- 实验中的 `K^m` 采用低通滤波或截断傅里叶观测，模拟 incomplete measurement。
- 噪声采用 multiplicative noise `y^m = K^m(u^\dagger) * epsilon^{delta^m}`，其中 `epsilon ~ Unif(0.5,1.5)`。
- 训练采用 alternating optimization：固定 `u` 优化 `theta`，再固定 `theta` 优化 `u`。
- `theta` 由 ParFam 过程优化，使用 basin-hopping 加 BFGS。
- `u` 用 ADAM 优化，学习率为 `4e-4`。
- 整体实验设计是先用可解析 1D 线性 PDE 验证 theorem 现象，再扩展到 Strogatz ODE 和 2D PDE 系统。
- 2D PDE 包括 Brusselator reaction-diffusion 和 membrane transport。

### 关键技术细节
- 符号网络定义为 `S_sigma(x)=r_{L+1}((sigma_L o r_L o ... o sigma_1 o r_1)(x),x)`。
- `r_i` 是 degree `d_i` 的 rational functions，`sigma_i` 是固定 base functions。
- rational function 定义要求分母多项式 `q(x)>0` 对所有 `x in R^n` 成立，避免 poles。
- 参数集 `Theta` 把分子多项式系数放在欧氏空间，把分母系数限制在闭集 `bar Q^epsilon`。
- 作者证明 bounded domain 上 rational map 关于参数在 `C^1` 意义下连续。
- 在函数空间中，symbolic network 被扩展为 Nemytskii operator，作用在 `J_kappa u` 的点值上。
- 主要空间假设包括 reflexive separable Banach spaces、`V` 到 `W^{kappa,p}` 的紧嵌入、以及 uniform state regularity。
- uniform state regularity 写作 `||J_kappa v||_{L^\infty((0,T)xOmega)} <= c_V ||v||_V`。
- 这个假设用于控制 symbolic network 层间输入落在 bounded set 内，从而使用局部 Lipschitz 连续性。
- measurement 假设要求 `K^\dagger` injective 且 weak-strong continuous。
- reduced operators `K^m` 只需 weak-weak continuous，并满足 `K^m u^m - K^\dagger u^m -> 0`。
- 正则项写作 `R(u,theta)=||u||_V^p + R_0(theta)`，其中 `R_0` proper、coercive、weakly lower semicontinuous。
- theorem 的参数条件是 `lambda^m -> infinity`、`mu^m -> infinity`、`mu^m delta(m) -> 0`。
- 实验中固定 `q=r=2`。
- 实验超参为 `lambda^m=m^3/M`、`mu^m=m/M`、`delta^m=0.1 M/m^3`。
- 最大 `M=100`，线性 PDE 取 `m=50,...,99`。
- ParFam 默认使用单隐层 `L=1`。
- 默认 numerator polynomial degree 为 3，denominator polynomial degree 为 2。
- 默认 base functions 包括 sine 和 exponential。
- ParFam regularization 为 `1e-3`。
- basin-hopping 和 BFGS 都运行 100 iterations，其他设置默认。
- alternating optimization 运行 100 full cycles。
- 1D PDE 和 ODE 中每个 state-update 最多 300 ADAM iterations；2D PDE 最多 100 iterations。
- early stopping patience 为 10。
- 1D PDE 时间导数用 central finite differences。
- ODE 和 2D PDE 时间导数用 Savitzky-Golay filtering，以降低噪声敏感性。
- ODE benchmark 包括 7 个 Strogatz systems：Bacterial Respiration、Bar Magnets、Glider、Lotka-Volterra、Predator-Prey、Shear Flow、Van der Pol。
- ODE 实验每个系统用 10 到 80 条随机初值轨迹，取决于复杂度。
- ODE 每个 `(m, system)` 组合跑 15 seeds。
- 2D Brusselator forward solve 使用 Strang splitting：Fourier-space spectral diffusion 加 4th-order Runge-Kutta reaction integration。
- Brusselator 网格为 `64 x 64`，`L=50`，`dt=0.01`，80 snapshots，8 initial conditions。
- Brusselator 三个 regime：easy `(a=1,b=3,D_u=1,D_v=4)`，medium `(2,5,1,5)`，hard `(3,8,1,8)`。
- Brusselator unconstrained recovery 给 ParFam 的特征是 `Delta u,u,v` 或 `Delta v,u,v`。
- Brusselator constrained additive 版本强制 `D Delta u + r(u,v)`，避免 Laplacian 和 state variables 的非物理 cross-terms。
- membrane transport 使用 `32 x 32` 网格，周期边界，8 个 perturbed-equilibrium 初值。
- membrane 方程含 rational term `v/(1+u)` 和 exponential term `e^{-u}v`，用于测试非多项式表达能力。

### 核心结果
- 主 theorem 证明：若真实物理律可由 architecture 表示，则任意收敛参数子序列的极限仍是该 architecture 中的 admissible physical law。
- 状态满足 `u^m` 弱收敛到 `u^\dagger`。
- 参数子序列对应的 symbolic networks 在局部 `L^\infty` 上收敛。
- 极限 law 满足 `partial_t u^\dagger = S_sigma^{tilde theta}(J_kappa u^\dagger)`。
- 极限参数是正则化最小解，因此 `L^1` 正则会偏向稀疏符号表达。
- 若 identifiability condition 成立，则整列 symbolic networks 收敛到唯一真实 law。
- 1D uniquely identifiable PDE 为 `u_t = a u + b u_x`，取 `a=1,b=2`。
- 对应解析解为 `u^\dagger(t,x)=(x+bt) exp(at)`。
- 学到的公式示例为 `1.006u - 0.005u_x^2 + 2.116u_x - 0.535`，接近真实 `u+2u_x`，但有小的 spurious terms 和常数偏移。
- 非唯一 PDE 例子中 `u(t,x)=exp(x-at)` 对任意 `a1+a2=-a` 都成立。
- 该非唯一设置下学到 `-0.982u - 0.016u_x`，体现 theorem 里的 regularization-minimizing admissible law，而非唯一真值恢复。
- Strogatz ODE 中 polynomial systems 表现最好。
- Lotka-Volterra best seed 恢复约 `-0.97x^2 - 1.97xy + 2.91x` 和 `-0.99xy - 0.94y^2 + 1.84y`，`L_infty` errors 约 0.14/0.11。
- Van der Pol 中 `f_y=-0.1x` 被精确恢复，`f_x` 近似 `-3.36x^3 + 3.36x + 10.02y`，但 reported error 为 1.50。
- Bar Magnets 和 Glider 的 trig/rational 结构也能恢复到较合理表达。
- Bacterial Respiration 和 Shear Flow 的 rational/trig 组合更不稳定，median variability 明显更大。
- Brusselator unconstrained recovery 对 `v` 方程尤其困难，因为 `D_v` 大且空间梯度更尖锐。
- Brusselator hard regime 中 degree-2 small architecture 不能表示 `u^2 v`，导致高且平的 model error。
- Brusselator extended architecture 加入 `sin` 和 rational denominator 虽更 expressive，但搜索空间变大，只在高 `m` 和少数 seeds 下成功。
- 默认 degree-3 polynomial architecture 被作者认为是 expressiveness 和 optimization reliability 的折中。
- Brusselator constrained additive structure 明显更物理，能在 easy/medium regime 中较好恢复 diffusion coefficients。
- hard regime 中 `D_v` convergence 仍失败，说明高非线性和大扩散系数下该数值 pipeline 不稳。
- membrane transport 中 `v` 方程恢复很稳，典型结果 `0.198 Delta v - 0.975 v exp(-0.968u) + 0.370` 接近真实 `0.2 Delta v - e^{-u}v + e^{-1}`。
- membrane `u` 方程的 rational term 在 best runs 中能识别，但 median 受 local minima 影响明显。
- 作者给出公开代码链接，复现性比纯理论预印本好。
- 2026 近期检索中，最接近的并发方向是 SGN 的 noisy sparse PDE discovery、constitutive closure 的 forward validation、diffusion-based data enrichment，而不是同一函数空间重建 theorem。

### 明确局限
- 最大理论限制是 representability：真实 `f^\dagger` 必须落在固定 symbolic network architecture 中。
- 如果 architecture 选错，theorem 不保证恢复真实 law，只能借助后续 growing architecture 或 approximation 扩展。
- uniform state regularity 很强，实际 PDE forward solutions 未必满足。
- measurement 极限依赖 `K^\dagger` injective；实际 sparse sensors 可能永远不达到这种信息量。
- theorem 是极限 reconstructibility，不提供有限样本误差率。
- 论文没有统计置信区间或 sample complexity。
- 数值优化是高度非凸的 alternating scheme，成功率依赖 seeds、architecture 和 time limit。
- 实验主要是 synthetic systems，没有真实传感器数据或真实物理实验闭环。
- 2D PDE 实验显示在 hard Brusselator 中仍会失败，尤其是 `v` equation 和大 diffusion coefficient。
- 结果展示 median/IQR 和 best seed，但没有外部独立复现报告。
- 后处理把小于 0.015 的系数置零，这有助解释性，但也说明原始拟合仍会产生 spurious terms。
- 论文承认目前是 analysis-guided framework，不是成熟、鲁棒的数值框架。
- `arxiv_shared.tex` 暴露的 affiliation/email 写法略混乱，但不影响技术内容。
- Appendix 采样算子证明里有一处明显疑似排版笔误：`u^m(s)-u^m(s)` 应为与上下文一致的 `u(s)-u^m(s)`；需人工核对正式版本。
- 跟 PDE-FIND/SINDy 类方法相比，这篇更重理论保证和函数空间，不是大规模 benchmark winner。
- 跟神经算子类方法相比，它牺牲预测黑箱容量来换取符号可解释性。
- 对 `0616-fex` 来说，如果我们主张从 incomplete/noisy fields 中恢复可解释 PDE law，需要把这篇作为强相关 prior art。
- 如果我们的 novelty claim 只是工程上用 LLM/agent 自动选库或验证公式，则不完全撞车，但这篇已经覆盖了 rational symbolic network + all-at-once inverse reconstruction 的理论核心。

- 若 `0616-fex` 的目标是发现可解释 governing equations，这篇是必须引用的理论近邻。

## Read by: 0616-fex (2026-06-16)

本次从 `0616-fex` 视角看，这篇论文是 “可解释物理方程恢复” 方向的强相关理论 prior art。它已经覆盖了一个相当具体的 claim：用 rational symbolic networks/ParFam 表示 PDE right-hand side，在 all-at-once inverse-problem objective 中同时恢复 state 和 law，并在完整测量极限、representability、identifiability 条件下证明恢复到 architecture 内的 admissible physical law。若我们的题目也声称 “noisy/incomplete measurement + symbolic PDE law + reconstruction guarantee”，会有直接撞车风险。

可利用的部分是它的建模语言和评测分层：`K^m` 低通测量、`lambda/mu/delta` 的极限条件、`L^1` 稀疏参数正则、unique/non-unique identifiability 对照、Strogatz ODE、Brusselator、membrane transport。差异化最好不要只说 “我们也恢复 PDE”，而要落在 finite-sample 可靠性、真实传感器数据、自动 architecture selection、独立 forward simulation verifier、失败检测、或生成/验证分离上。近期检索到的 2026 相邻工作包括 `2603.22380` 的 Symbolic Graph Networks for noisy sparse PDE discovery、`2604.01335` 的 constitutive closure bias inheritance、`2606.00988` 的 diffusion data enrichment for symbolic regression、`2604.23867` 的 LatentPDE reconstruction；这些应作为下一轮 novelty check 的重点。
