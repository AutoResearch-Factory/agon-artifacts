# Agentic Symbolic Search: Characterizing PDEs Beyond Hand-crafted Expressions, Meshes, and Neural Networks (2606.20467)

- **Year**: 2026.06
- **Authors**: Zongmin Yu, Liu Yang (National University of Singapore)
- **Venue**: arXiv preprint
- **Keywords**: agentic symbolic search, PDE characterization, differentiable symbolic programs, evolutionary search, interpretable representations, self-similarity, finite-time blow-up
- **One-liner**: LLM-coding-agent 驱动的符号搜索框架 ASYS，将 PDE 理论和物理先验转化为可微符号程序，通过演化搜索自动发现 PDE 解的显式数学结构（缩放律、界面几何、相似变量）。

## 精华提炼

### 管线/方法

ASYS 接收已知 PDE + 初始/边界条件 + 问题常量，输出一个可微符号程序（其数学结构可直接阅读）。管线分两层嵌套循环：

1. **外层循环（结构搜索）**：Agent 提出数学 ansatz 形式——坐标变换、分支分解、可训练参数的选择。外层由 EvE (Evolutionary Ensemble of Agents) 框架驱动，维护候选程序种群和搜索策略种群。每次迭代：从种群采样强候选作为 context → 两个并行 worker（coding-agent sessions）各自提出一个新候选 → Evaluator 用 L-BFGS 拟合参数并打分 → 候选加入种群。无候选被丢弃（append-only）。

2. **内层循环（参数拟合）**：L-BFGS 优化每个候选的连续参数，固定 wall-clock 预算（有界问题 30s，blow-up/free-boundary 问题 300s）。Agent 可自由设计训练 loss，但评测使用固定四维 score。

**关键设计：目标分离**——Agent 控制的训练 loss 与 Evaluator 的固定四维 score 完全解耦。Agent 可以 engineering 不同的 loss landscape，但评估标准保持绝对。

3. **种群选择**：每维独立排名，exponentially decaying weight，四维加和得到采样概率，不放回抽样。

4. **离线验证**：相对 L^2 error against 独立数值参考解，仅在搜索结束后用于验证，不参与搜索评分。

### 关键技术细节

- **四维 Score**：
  1. $s_{\rm phys}$: PDE residual（有界问题用 raw MSE；blow-up 问题用 scale-free geometric-mean relative residual）
  2. $s_{\rm ic}$: initial value MSE at $t=0$
  3. $s_{\rm bc}$: boundary condition residual
  4. $s_{\rm comp}$: compatibility condition —— 首次引入的关键创新，比较 one-sided finite-difference $\partial_t u_\theta$ 与 PDE RHS 作用于初始条件，防止解落入 trivial branch（如 $u\equiv 0$）

- **硬件**：单 Apple M4 laptop CPU，无 GPU
- **内层优化器**：L-BFGS（SciPy 实现）
- **Agent 接口**：subscription-covered interactive coding-agent interface（非 per-token API）
- **EvE 参数**：2 workers/iteration，每 worker 4 examples，有界 5 iter / blow-up 10 iter

### 核心结果

| Case | Type | ASYS 相对 L^2 | ASYS 参数量 | SS-PINN L^2 | SS-PINN 参数量 |
|------|------|-------------|-----------|------------|-------------|
| NLS | bounded | 0.0059 | 5 | -- | -- |
| Allen-Cahn 2D | bounded | 0.0107 | 23 | -- | -- |
| Keller-Segel | blow-up | 0.188 | 9 | 0.258 | 49,921 |
| Graveleau | free boundary | 0.00132 | 13,133 | -- | -- |
| gCLM | blow-up | 0.465 | 13,060 | 0.196 | 49,921 |

**Surprising 发现**：
1. **Keller-Segel**：ASYS 仅用 9 个可解释标量参数就超过了使用 49,921 个权重的 SS-PINN baseline（L^2=0.188 vs 0.258）。这是这篇论文最关键的实证贡献——证明了结构化符号搜索在 blow-up 场景下可以超越"先验坐标变换 + 大 MLP"的方法。
2. **Allen-Cahn 2D**：ASYS 仅用几何符号公式（无神经网络层）构建了花生形到椭圆形的界面演化公式，23 个参数都有直接几何含义（lobe shrink rates, center relaxation, oval blend, neck-fill scales）。
3. **NLS breather**：ASYS 恢复了 Satsuma-Yajima 呼吸子闭式解，并添加了 5 参数 endpoint correction 处理周期边界——既展示了 recall 能力，也展示了自适应修正能力。
4. **Graveleau**：ASYS 实例化了第二类自相似框架，学到了 $\beta_{\rm model}=0.928$（参考值 $\beta_{\rm ref}=0.877$），残差引导的符号搜索逼近了正确的非线性本征值。
5. **gCLM 失败诊断**：nonlocal Hilbert 变换使 compact local ansatz 搜索变成 global-mode 问题，10 次迭代未构建出 self-similar reduction。这隔离了当前方法的边界——非局部耦合 PDE。

**Score-L^2 divergence 现象**：
- Keller-Segel：迭代 8 后 physics loss 继续下降但 validation L^2 不改善→ 9 参数家族达到 profile-matching 极限
- Graveleau：初始 unstructured 候选 raw residual 最低但 L^2 较差；引入 rigid self-similar 框架后 raw residual 上升但 L^2 改善→ 说明 residual-only scoring 不能完全替代结构验证

### 明确局限

1. **非 amortized solver**：ASYS 是单轨迹结构表征器，不是跨初始条件族的泛化求解器。需要多个初始条件时，neural operators 仍然是合适工具。
2. **Agent 结构触及范围受限**：gCLM 展示了 agent 在 10 次迭代内未发现 global self-similar reduction。nonlocal/强耦合/高维 PDE 可能需要 hybrid representation（符号奇异核心 + neural operator）。
3. **内层非凸优化**：L-BFGS 在奇点附近 conditioning 恶化；可能需要 continuation、relaxed gradient flows 或 Hessian-free 方法。
4. **仅基于 equation residual + public constraints 评分**：无观测数据参与（这是有意为之的 design choice），扩展到逆问题需要额外 score dimension。
5. **计算成本依赖 LLM agent 迭代**：10 次迭代 × 2 workers × 300s = 约 6000s wall-clock per blow-up case。Agent 能力的提升直接影响搜索质量。
6. **无真机实验**——纯计算方法。
7. **无标准 benchmark 比较**：没有在 PDEBench 等公开 benchmark 上与 SOTA neural solver 比较。

**撞车评估**：
- ASYS 的核心贡献是 agent-driven structural search for PDE solution representation，与 FEX（Finite Expression Method）同属"用符号/表达式表征 PDE 解"的范式。
- **不撞车的关键差异**：
  1. FEX 搜索表达式树，ASYS 搜索可微符号程序（更广泛：支持 multi-branch decomposition, learned coordinate rescaling, smooth gates）
  2. FEX 用 RL/evolutionary search over expression grammar；ASYS 用 LLM agent 注入 PDE 理论先验作为 inductive bias
  3. FEX 从 data/trajectory 学习；ASYS 仅用 equation residual + public constraints（无观测数据）
  4. 目标不同：FEX 目标是通用 PDE solver；ASYS 目标是单轨迹结构表征

**可用工具/方法**：
1. **四维 scoring 系统（尤其是 compatibility condition）**：可作为 FEX 类方法的 evaluation protocol 增强，防止 trivial solution
2. **Scale-free geometric-mean relative residual**：FEX 处理 blow-up 问题时可直接采用
3. **EvE 框架**：如果考虑 LLM 增强 FEX 搜索，EvE 是现成的 population-based agent orchestration
4. **SS-PINN baseline 设计**：FEX 做 blow-up 问题时可用同样的 SS-PINN 作为对比
5. **Agent workspace 设计**：README.md + strategy.md + references.md 的结构可作为 FEX agent 化时的 prompt engineering 模板

**该引不该引**：
- 必须引：这是 2026 年 agentic symbolic search for PDE 的最新工作，直接相关
- 差异化论述：ASYS 用 LLM prior 指导搜索，FEX 用弱先验（expression grammar）；ASYS 不需要观测数据，FEX 通常需要

## Read by: 0616-fex (2026-06-27)

### 撞车判断：不撞，但高度相关

ASYS 与 FEX 类方法的核心差异在于**搜索机制的哲学不同**：

1. **ASYS = LLM 先验驱动的结构搜索**：Agent 从 PDE 文献知识（profiles, scalings, similarity structures）出发提出 ansatz，搜索是 inductive-bias injection 的自动化。适合对单个困难轨迹做深度结构表征。

2. **FEX = 弱先验的表达式搜索**：在 expression grammar 定义的空间中用 RL/evolutionary 方法搜索，不依赖 LLM 编码的领域知识。适合做通用 PDE solver。

### 对 fex 的 actionable insights

1. **Compatibility condition 是 killer feature**：FEX 如果只用 residual + IC/BC loss，同样会面临 trivial solution（constant-in-time）问题。应该立即把 compatibility condition 加到 FEX 的 evaluation 中。

2. **Scale-free residual for blow-up**：FEX 处理 blow-up 问题时，raw MSE 在 singularity 附近会 dominated by large values。ASYS 的 geometric-mean relative residual 是更好的选择。

3. **ASYS 的 gCLM 失败 = FEX 的机会**：ASYS 在 nonlocal coupling PDE 上失败，因为 LLM agent 没能在 10 轮内构建 global self-similar reduction。FEX 的 grammar-based search 可能在 nonlocal 问题上反而有优势（不需 LLM 理解非局部结构，只需搜索到正确的表达式形式）。

4. **Hybrid 方向最有价值**：ASYS 提出的"符号核心 + neural operator correction" hybrid 方向，与 FEX 的"有限项表达式 + MLP correction"天然对接。可以考虑 FEX 搜核心结构，用 ASYS 的 workspace 设计做 agent 辅助。

5. **FEX 可以用 ASYS 的实验 protocol**：五个问题（NLS, Allen-Cahn 2D, Keller-Segel, Graveleau, gCLM）构成了一套很好的 benchmark，覆盖 bounded dynamics → blow-up → free-boundary → nonlocal stress test。FEX 可以直接跑同一套问题做 head-to-head 比较。
