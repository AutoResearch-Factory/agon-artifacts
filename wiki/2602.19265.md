# Spectral bias in physics-informed and operator learning: Analysis and mitigation guidelines (2602.19265)

- **Year**: 2026.02
- **Authors**: Siavash Khodakarami, Vivek Oommen, Nazanin Ahmadi Daryakenari, Maxim Beekenkamp, George Em Karniadakis; Brown University
- **Venue**: arXiv

## 精华提炼

### 管线/方法

1. 论文把 neural PDE solver 的 spectral bias 定义为：低频误差比高频误差更快衰减，导致解场看似满足低阶残差但丢失 sharp gradient、localized structure 和 high-frequency modes。
2. 统一表示从 Fourier/basis expansion 出发：误差 `e(x)=u_theta(x)-u(x)` 被分解成 mode coefficients `e_k`，再比较不同频率的学习速度。
3. 第一层理论管线是 activation analysis：Tanh 的 Fourier transform 指数衰减，高频表示需要很大权重；SIREN 的 sine activation 对应 Dirac-like spectral support，不天然压低高频。
4. 第二层理论管线是 optimizer analysis：一阶优化在 NTK 线性化下满足 `dot e_k=-eta lambda_k e_k`，若 `lambda_k` 随频率下降就产生 spectral bias。
5. 对二阶优化，作者把 function-space dynamics 写成 `dot f=-J(J^T J)^{-1}J^T e`；在 residual 落入 Jacobian column space 的近似下得到 `dot e_k=-e_k`，即 mode 收敛率近似与频率无关。
6. 这给出全文主 claim：spectral bias 不是单纯 representation limitation，而是 optimizer/loss/architecture 耦合导致的 dynamical phenomenon。
7. 度量管线包含三个互补指标：frequency-resolved error、Barron norm、前四阶 statistical moments。
8. frequency-resolved error 定义为 `e_F,p = 1/N_x^2 sum_k |k|^p |hat e_k|^2`；`p=0` 对应 field L2，`p=2` 对应 gradient，`p=4` 对应 Laplacian。
9. Barron norm 用 `int ||omega|| |hat f(omega)| d omega` 度量振荡含量；预测 Barron norm 偏低意味着高频被过度平滑。
10. moments 诊断中，mean/variance 多由低频主导，skewness/kurtosis 更敏感于 localized/high-frequency structure。
11. 实验管线先做纯 data-driven function approximation，隔离 architecture effect。
12. Case 1 是 piecewise discontinuous function：`x<0` 用 `5+sum_{k=1}^4 sin(kx)`，`x>=0` 用 `cos(10x)`，80 个训练样本。
13. Case 2 是 smooth multi-frequency signal：`sin(2pi*0.01t)+0.5sin(2pi*0.05t)+0.2sin(2pi*0.1t)`，300 个训练样本。
16. PINN 表示是普通 MLP，activation 比较 Tanh 与 SIREN；SIREN 用 `sin(omega_i W X+b)`，最佳设置为第一层 `omega_0=30`、其余层 `omega_i=1`，低频 wave case 中 `omega_0=5` 更合适。
20. KdV 设置为 `u_t + eta u u_x + mu/2 u_xxx = 0`，`x in [-1,1]`，`t in [0,1]`，初值 `cos(pi x)`，周期边界，`eta=1`，`mu=0.022`。
21. Wave 设置为 1D wave equation，`c=1`，`T=1`，左边界输入是逐步增加频率的 sinusoidal superposition；最难 case 包含 1、5、10、20、40 Hz-like components 并带 Gaussian envelope。
22. Reaction-diffusion 设置为 2D periodic domain `[-1,1]^2`，方程 `Delta u - k_r u^2=f`，`k_r=0.1`，manufactured solution 为 `sin(3pi x)cos(3pi y)`。
23. Reaction-diffusion 中周期性不是靠 boundary residual，而是通过 trigonometric feature map hard-embed 到神经表示中。
25. BSP loss 训练目标是 field error 加 log-binned energy spectrum error；field 和 BSP 都用训练集 min/max normalize 到相近量级。
26. Neural operator 的第一个任务是 impinging-jet Schlieren super-resolution：1000 个 `[128,256]` snapshots，800/100/100 train/val/test，low-resolution 输入来自空间下采样 8 倍。
27. Jet task 中模型学习 `u_LR -> u_HR`，重点是恢复 shock/turbulence fine-scale density-gradient features。
28. Neural operator 的第二个任务是 earthquake structural response：6 层 RC frame，有限元矩阵 `M,C,K in R^{504x504}`，输入地震加速度，输出 roof displacement。
29. Earthquake 数据来自 144 条 PEER NGA-West2 records，100 train、44 test；每条 80 秒、50 Hz、4000 timesteps，Butterworth filter 0.1-24.9 Hz。
30. Earthquake task 有 2000 个可解析 frequency bins，比 jet spatial dimension 的 64 modes 更难，作者用它作为强 spectral-bias 压力测试。
32. 作者还测量 L2 field-loss 与 BSP spectral-loss 的 gradient cosine similarity，用来解释 BSP 在不同 architecture 上为什么有时帮忙、有时冲突。

### 关键技术细节

33. KdV 模型规模：除 SS-Broyden 外约 40k parameters；SS-Broyden 模型约 12.9k parameters。
34. KdV optimizer 比较包括 Adam、Adam+L-BFGS、SOAP、SS-Broyden；SS-Broyden 使用 Wolfe line search。
35. KdV 最佳 PINN 结果是 SS-Broyden+SIREN：relative L2 `7.54e-8`，Barron norm relative error `5.51e-8`，`log(e_F,p=4)=-7.66`。
36. KdV SOAP+SIREN 已经显著好于 Adam+SIREN：relative L2 `1.06e-4` vs `6.35e-2`。
37. KdV Adam+Tanh 基本失败：relative L2 `2.47e-1`，`log(e_F,p=4)=5.34`，且 time to 1% error 为 NA。
40. Wave hardest case 中 Adam-Tanh 与 Adam-SIREN relative L2 都约 `1.00`，说明一阶优化在高频 hyperbolic problem 上完全不够。
41. Wave hardest case 中 SOAP+SIREN relative L2 `4.34e-3`，SS-Broyden+SIREN `1.16e-5`，SS-Broyden 比 SOAP 低约 2-3 个数量级。
43. Reaction-diffusion 中 PINN SS-Broyden+Tanh 最好：relative L2 `1.27e-6`，Barron norm error `7.08e-10`，convergence time `0.06 hr`。
44. Reaction-diffusion 中 Adam+Tanh relative L2 `1.41e-1`，Adam+SIREN `3.14e-3`，说明 SIREN 可部分补偿一阶优化，但 derivative-level error 仍有 structured oscillation。
49. Jet baseline L2 下 field errors 都约 `0.05`，但 energy-spectrum error 差异很大，说明 field loss 掩盖高频失败。
51. Jet BSP 对 field error 基本无收益甚至略差，说明它的价值在 spectral fidelity 而非 pointwise RMSE。
52. Jet CNO+BSP 最低 spectral error `0.0211` 和 Barron norm error `0.0792`；DeepONet 对 BSP 不敏感甚至 spectral error 变差。
53. Earthquake architecture comparison 中 DeepONet(SIREN) baseline field error `0.0008`，DeepONet(Tanh+BSP) field error `0.0006`，是该表最佳。
54. Earthquake 中 BSP 对 FNO/CNO 的 log spectral error 有益：FNO `0.1485->0.0819`，CNO `0.1323->0.0758`。
56. Gradient cosine 解释很清楚：DeepONet(SIREN) causal final cosine `-0.482`，FNO non-causal `+0.282`；负 alignment 对应目标冲突。
58. Earthquake activation 表明 SIREN 本身提供 implicit spectral mitigation；Tanh 需要 BSP 才能收敛，但 Tanh+BSP 可得到最佳 field 与 acceleration error。
59. 作者承认 SS-Broyden 目前不能像 SOAP 一样 mini-batch，难以扩展到大规模 neural operator，这是未来方向。
60. Data availability：代码和除 Schlieren 外的数据将在发表后公开；Schlieren 数据需联系 Tsinghua SMC Lab/He Feng。

### 核心结果

61. 论文最强结论是 optimizer dominates representation：二阶/准二阶优化能改变高频学习顺序，activation 的作用在 Adam 下最明显，在 SS-Broyden 下显著减弱。
62. KdV 中从 Adam+Tanh 到 SS-Broyden+SIREN，relative L2 从 `2.47e-1` 降到 `7.54e-8`，约 6 个数量级。
63. KdV 中 SOAP+SIREN 相比 SOAP+Tanh 进一步降低 error，但 SS-Broyden 下 Tanh/SIREN 的训练轨迹相近，说明 curvature preconditioning 已经解决主要频率瓶颈。
64. Wave high-frequency case 中 Adam 不管 Tanh 还是 SIREN 都失败，SS-Broyden+SIREN 达 `1.16e-5`，显示高频 hyperbolic PDE 对 optimizer 尤其敏感。
65. Reaction-diffusion 说明 Fourier feature embedding 可以在 representation level 覆盖 true spectral support，但优化曲率仍决定最终 derivative accuracy。
67. Neural operator 中 spectral bias 不是 FNO 独有；即使显式用 Fourier layer，plain L2 仍会压低 high-wavenumber energy。
68. BSP loss 是低成本改善 spectral fidelity 的有效工具，因为它只改训练 loss，不增加 inference cost。
71. 近半年相关检查显示，`2603.00904` 的 operator-aware preconditioners for PINNs 是最接近的 2026 同题方向之一；它同样把频率偏置视作可由 preconditioning rebalance 的问题。
72. 反引文中 `2606.02427` 继续把本文思路推进到 in-context operator networks 的 spectral audit，说明本文的 spectral diagnostics 正在形成后续研究线。

### 明确局限

73. 理论分析依赖 linearization、over-parameterization 和 residual 落在 Jacobian column space 等强假设；它解释现象，但不是严格证明所有非线性训练都会频率无关收敛。
74. 实验覆盖多个 PDE 和 operator tasks，但仍是同一组作者主导的 benchmark suite，没有独立第三方复现。
75. Schlieren dataset 不完全公开，复现 neural operator 部分需要额外联系数据所有方。
76. 代码声明为 publication 后公开；当前 arXiv 版本不能直接审计实现细节、random seeds、hardware precision 和 optimizer hyperparameters。
78. BSP loss 在 causal branch-trunk architecture 上可能产生 gradient conflict；它不是 universal fix。
80. 本文不是 symbolic regression 或 expression-tree search 方法，不解决 FEX 的 RL 搜索复杂度、表达式等价剪枝、LLM skeleton 或 PDE closed-form recovery 问题。
81. 高频谱指标提升不一定等价于可解释公式正确；对 FEX 课题来说，spectral fidelity 只能作为 reward/diagnostic，不能替代 symbolic exactness。
82. 作者强调 field RMSE 可能掩盖高频失败，但没有建立这些 spectral metrics 与实际 PDE downstream quantities 的统一标准。

### 跟我们的关系

83. 对 `0616-fex`（FEX: RL for Symbolic Regression）来说，本文不直接撞车：它研究 neural PDE solvers/operator learning 的 spectral bias，不生成符号表达式树，也不做 RL-based symbolic regression。
84. 间接相关度很高，因为 FEX 方法族也常处理 PDE/high-frequency/multiscale 解，Multi-Scale FEX 已明确暴露 frequency separation 导致 RL dynamics unstable 的问题。
85. 本文最可借鉴的是诊断指标：`e_F,p`、Barron norm、gradient/Laplacian error、skewness/kurtosis 可直接用于评估 FEX 是否真的恢复高频结构，而不是只在 pointwise L2 上好看。
86. 如果 `0616-fex` 做 multi-scale/high-frequency FEX，建议把 spectral reward 或 BSP-like objective 加到 FEX 的 scoring function 中，尤其用于区分“低频拟合好但高频错”的表达式。
87. 本文支持一个重要 framing：高频失败不一定来自表达空间不够，也可能来自优化/搜索 dynamics。因此 FEX 需要诊断 RL controller 是 representation-limited 还是 search/optimization-limited。
88. SOAP/SS-Broyden 的结果提示：FEX 的 inner continuous parameter optimization 可能比当前 Adam+BFGS pipeline 还有改进空间，尤其在高频表达式参数和 stiffness 较强时。
89. 但 SS-Broyden 不能 mini-batch 的限制也提醒：如果 FEX 扩到大规模数据或 operator setting，准二阶方法可能比全二阶更实际。
90. SIREN/Fourier feature 结论可作为 FEX input-layer/operator-set design 的参考，但它不是 symbolic expression discovery 本身。
92. 反引文 `2606.02427` 的 spectral audit 思想可作为 FEX 的 verifier：对候选表达式不只测 output error，也测局部 frequency response 或 operator tangent。
93. 近半年 prior `2603.00904` 已经把 PINN 的 frequency bias 明确解释为 operator/representation bias 的竞争，并提出 operator-aware preconditioning；如果 FEX 声称“频率平衡训练/求解”必须与它区分。
94. 差异化建议：FEX 可强调 symbolic exactness、expression compactness、operator-set interpretability 和 closed-form PDE solution；本文只能提供 neural approximation/diagnostic。
