# Bias Inheritance in Neural-Symbolic Discovery of Constitutive Closures Under Function-Class Mismatch (2604.01335)

- **Year**: 2026.04
- **Authors**: Hanbing Liang, Ze Tao, Fujun Liu (Changchun University of Science and Technology, School of Physics, Nanophotonics and Biophotonics Key Laboratory of Jilin Province)
- **Venue**: arXiv preprint (targeting Journal of Computational Physics)
- **Keywords**: closure discovery, weak form, symbolic regression, reaction-diffusion, bias inheritance, identifiability, function-class mismatch, neural-symbolic pipeline, constitutive modeling, forward validation
- **One-liner**: 提出三阶段 neural-symbolic 框架从反应扩散系统数据中恢复未知本构闭包，发现 symbolic compression 会原样继承（而非修复）neural surrogate 的本构偏差（BIR≈1），指出瓶颈在 Stage-1 数值反问题而非 Stage-2 符号压缩。

## 精华提炼

### 管线/方法

整体三阶段 pipeline:
1. **Stage 1 -- Weak-form numerical constitutive recovery**: 输入 spatiotemporal 观测数据 u(x,t)，使用 weak-form-driven hybrid objective 训练 dual-branch MLP（扩散分支 + 反应分支），输出数值替代模型 D_hat(u) 和 R_hat(u)。扩散分支强制正值，反应分支通过 anchor loss 在下边界软约束。
2. **Stage 2 -- Restricted symbolic compression**: 在 dense support grid 上评估 Stage-1 surrogate，然后用受限符号族（多项式 ≤ degree 3-4、有理函数 (2,2)、指数衰减、饱和形式）拟合，选最低复杂度且拟合误差在最优候选 tolerance 内的表达式。
3. **Stage 3 -- Forward validation**: 将符号闭包重新插入 PDE，在 unseen initial conditions 上 forward simulation，输出 ε_roll^neural 和 ε_roll^sym 两个 rollout 指标。

数据流: observations (M trajectories, 可能带 noise/sparsity) → Stage 1 训练 (150 epochs, Adam lr=1e-3) → numerical surrogate → Stage 2 压缩 → symbolic law → Stage 3 前向验证。

### 关键技术细节

**模型架构**:
- MLP backbone: hidden width 64, depth 2, tanh 激活
- Explicit polynomial-residual parameterization: diffusion 用 degree-2 polynomial backbone + residual MLP, reaction 用 degree-3 polynomial backbone + residual MLP
- 两分支共享输入 u，分别输出标量 D(u) 和 R(u)

**训练目标 (6-term hybrid loss)**:
```
L = L_weak + α·L_roll + β·L_mass + γ·L_strong + η·L_anchor + λ·L_reg
```
默认权重: α=0.1, β=1, γ=1, η=1, λ=1e-4
- L_weak: space-weak form 残差 (4 Fourier modes + 4 bump functions 作 test functions)
- L_roll: auxiliary rollout loss
- L_mass: 显式加权质量守恒 (φ≡1 退化的全局平衡)
- L_strong: 辅助 pointwise consistency (仅 synthetic regime)
- L_anchor: 在低 state 边界软锚定 reaction 分支
- L_reg: 闭包变化量正则化

**数值 protocol**:
- 1D periodic domain Ω=[0,1), nx=64, Δt=1e-4, T=0.1, snapshots saved every 10 steps
- 8 training trajectories, 4-mode random Fourier ICs
- 3 seeds average, RK4 integrator, conservative finite-volume discretization
- "5% noise": additive Gaussian σ=0.05·std(u)
- "sparse x+t": spatial stride 2 + temporal stride 2

**三个 benchmark cases**:
- Case A (matched): D(u)=0.01+0.05u, R(u)=u(1-u) -- 线性扩散 + Fisher-KPP
- Case B (matched): D(u)=0.01+0.03u², R(u)=u-1.5u²+0.5u³ -- 二次扩散 + 三次反应
- Case Exp (mismatch): D(u)=0.01+0.035(1-e^{-2.5u}), R(u)=u(1.1e^{-1.4u}-0.22) -- 非多项式，是 smooth non-polynomial stress test

**符号压缩候选族**:
- Diffusion: polynomials ≤ degree 3, (2,2) rational, exponential-decay, saturation
- Reaction: polynomials ≤ degree 4, (2,2) rational, u·exp(·), u/(1+·) saturation
- 选择标准: lowest-complexity candidate with fit MSE within 5% relative tolerance and 1e-8 absolute tolerance

**评价指标 (三个层次)**:
1. Trajectory level: weak residual + unseen rollout error ε_roll
2. Closure level: relative discrete L² error ErrD, ErrR (on common support grid)
3. Neural-symbolic level: surrogate error ||f_hat - f_sym||, BIR (bias inheritance ratio)

### 核心结果

**1. 多项式 baseline 在 matched-library setting 下极强** (Table 1):
- Case A clean: strong_poly ErrD=4.1e-4, ErrR=2.1e-3, Unseen=1.8e-5 -- 全面碾压 neural surrogate (ErrD=6.9e-2, ErrR=3.1e-1, Unseen=4.8e-3)
- Case B 同样模式。这说明在正确指定的 regime 下，neural surrogate 不应被假设为 uniformly outperform classical bases。

**2. 偏差继承 (Bias Inheritance) 是核心发现** (Table 3):
- BIR_D 和 BIR_R 在所有 setting 下都接近 1（0.92-1.00）
- 即使是 mismatch Case Exp: clean BIR_D=0.999, BIR_R=1.000
- 噪声下更极端: BIR ≈ 1.000，说明 Stage-2 完全不修正 Stage-1 的 constitutive bias
- 反直觉发现: 符号压缩非常精确地复现了 surrogate 的偏差，而不是"修复"它

**3. 低残差不等于正确恢复** (Table 2, Proposition 3):
- low-excitation regime: weak loss 1.9e-5 (vs high-excitation 9.5e-4)，但 closure error 更差 (ErrD=1.3e-1 vs 5.5e-2)
- 这是因为 observation operator 在 limited excitation 下不是 injective -- 不同闭包组合可产生相似 trajectory-level predictions

**4. PySR 替换检查** (Table 4):
- 将 Stage 2 从 restricted family 换成 PySR（更强大的符号回归引擎），结论不变
- PySR mean complexities: clean (17, 20.7) vs restricted (5,5)，噪声下 (17, 22) vs (3,5)
- 更 expressive 的符号引擎也不修复继承的偏差 -- 这强化了"瓶颈在 Stage 1"的核心论点

**5. 噪声敏感性** (Figure 2):
- 1%-3% noise 之间出现急剧退化（degradation knee），不是单一病态点
- 原因: Stage 1 的 hybrid design 包含多个显式时间导数通道 (strong residual + mass balance)，不光是 space-weak residual

**6. Cross-resolution anti-inverse-crime check** (Table 5):
- same-grid unseen rollout 与 cross-grid rollout 非常接近 → 结论不是 trivial 的 inverse-crime artifact
- 空间降采样远比时间降采样更有害: spatial stride 2 alone accounts for almost all sparse-setting degradation

**7. 2D preliminary check** (Appendix Table):
- 在 2D periodic domain 上 (32×32 grid, 4 training trajectories, 120 epochs)，BIR 仍然接近 1
- 2D clean Case A: BIR_D=0.975, BIR_R=1.000; 2D noisy Case Exp: BIR_D=1.000, BIR_R=0.999
- 初步验证核心机制在维度提升后仍然成立

**8. Stage-1 objective ablation** (Appendix Table A):
- pure weak-form only: 在 mismatch stress test 上不 competitive (ErrR=7.6e-1, Unseen=7.2e-3)
- full hybrid: best diffusion recovery, smallest weak loss, best rollout (Unseen=3.3e-3)
- 但有 metric-dependent tradeoff: 去掉 strong residual 反而改善 reaction recovery

### 明确局限

**作者自述的局限**:
1. 仅 1D periodic reaction-diffusion，2D 只是 preliminary stress test
2. 仅 synthetic data
3. 受控的 closure families（非 nonlocal/singular/history-dependent）
4. Stage 1 的 hybrid objective 在噪声下表现不佳（1-3% noise degradation knee）
5. 未宣称 exhaustive hyperparameter tuning
6. 不声称 bitwise determinism across hardware

**我读出的额外局限**:
1. 没有真机实验数据，全合成。这对声称"physical discovery"是严重局限。
2. 没有跟 SINDy/PDE-FIND 等主流方法的系统对比（只跟 polynomial baselines 比）
3. BIR≈1 的发现本质上是 reverse triangle inequality 的直接推论（Proposition 5），理论贡献较薄
4. 没有讨论如果 Stage 1 surrogate 本身就很差怎么办？BIR≈1 意味着 garbage-in-garbage-out -- 但没有讨论诊断方法
5. 时间尺度极短 (T=0.1)，长期动力学行为未验证
6. 所有 case 的 diffusion 都很小（0.01+...），未讨论 stiff/convection-dominated 情形
7. 没有跟 PINN-based inverse methods 的直接数值对比
8. GitHub repo 存在但未验证其可复现性
9. Stage 2 "restricted family" 的选取没有理论指导，只是 heuristic
10. Weak form test function 的选取 (4 Fourier + 4 bump) 对其他 PDE 类型的泛化性未讨论

## Read by: 0616-fex (2026-06-28)

### 审读视角

这是一篇对 FEX (Function Expression) 课题高度相关的论文。它不跟 FEX 直接竞争，而是**诊断了 neural-symbolic pipeline 中一个被普遍忽视的问题**：符号压缩阶段不会修复神经网络代理模型的偏差，而是原样继承。

### 核心洞察

**Bias Inheritance Ratio (BIR) ≈ 1** 是本文最重要的经验发现。它意味着：
- 如果 Stage 1 (neural surrogate) 学错了，Stage 2 (symbolic regression) 必然也错 -- 即使换成 PySR 也一样
- 因此，投入精力改进符号回归算法（Stage 2）是**边际收益递减**的
- 真正需要改进的是 Stage 1 的数值反问题求解质量

这对 FEX 的启示是两面的：
1. **如果 FEX 采用 end-to-end 直接符号发现**（不经 neural surrogate），则本文的 BIR 问题不直接适用，但本文提出的 excitation coverage / identifiability 问题仍然关键
2. **如果 FEX 采用两阶段方法**，则本文的发现是重要警告：不要指望第二步修复第一步的错误

### 具体可用部件

1. **Excitation benchmark protocol**: 通过改变 IC amplitude range 来测试 identifiability -- FEX 可以直接复用这个实验设计
2. **Weak-form hybrid objective**: 可以作为 FEX 数据预处理的参考
3. **三层次评估 (trajectory/closure/neural-symbolic)**: 比单纯 report rollout error 更全面
4. **BIR 诊断指标**: FEX 可以把 BIR 作为内部 sanity check
5. **Restricted symbolic family 设计**: 参考其 candidate library 构建策略

### 关于 novelty 的提醒

本文的"bias inheritance"本质上是一个 reverse triangle inequality 的直接推论（Proposition 5），理论深度有限。但其**实验证据的系统性**（3 cases × 3 regimes × PySR 替换 × noise sweep × 2D check）使得结论有说服力。FEX 如果要 claim novelty，需要在理论或实验上做出比"反向三角不等式 + 实验验证"更强的贡献。
