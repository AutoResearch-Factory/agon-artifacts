- **Year**: 2017.06
- **Authors**: Niklas Pfister (ETH Zürich, Seminar for Statistics), Peter Bühlmann (ETH Zürich), Jonas Peters (University of Copenhagen)
- **Venue**: 期刊论文格式（blind-review 宏、chicago 引用格式、"Supplementary materials... available online" 措辞均为 JASA 风格；tex 源码未显式写出刊名）
- **Keywords**: invariant causal prediction, change point model, Chow statistic, instantaneous causal effects, Granger causality, monetary policy, sequential/time-ordered data, resampling test
- **One-liner**: 把 Peters2016 的 ICP（已知环境）推广到无需已知环境的时序数据，用 change-point resampling 检验替代条件独立图检验来做因果发现。

## 精华提炼

### 管线/方法

核心问题：从响应变量 $Y_t$ 和协变量 $X_t\in\R^{1\times d}$ 的时间序列 $(Y_t,X_t)_{t=1}^n$ 中，估计因果预测变量集合 $S^*$，且不需要像 Peters2016 的原始 ICP 那样预先知道"环境"划分。

管线（与 PCMCI+ 这类图上条件独立检验完全不同的范式）：

1. **不变性假设**：假设存在集合 $S^*$，使得 $Y_t = X_t^{S^*}\beta + \epsilon_t$，且 $\epsilon_t \iid \mathcal N(0,\sigma^2)$ 独立于 $X_t^{S^*}$，对所有 $t$ 成立（Def. 1，结构不变性）。这是"无环境版"的 Peters2016 不变性假设 Eq (1)。
2. **假设检验框架**：对每个候选集合 $S\subseteq\{1,\dots,d\}$ 构造原假设 $H_{0,S}$："S 是不变集"。用 OLS 残差 $R^S=(I-P_X^S)Y$ 标准化为 scaled residual $\tilde R^S$，在 $H_{0,S}$ 成立时其分布不依赖未知参数，可用高斯白噪声精确重采样（不是渐近近似）得到检验的 cutoff。
3. **检验统计量的选择（对应不同"conditioning"策略）**：
   - **分块比较类**（change-point 假设）：把时间轴按 grid 切成多个"伪环境" $\mathcal E$，两两比较：
     - $T^1_{e,f}$：回归系数差 $\|\hat\gamma_e-\hat\gamma_f\|$（检测 $\beta$ 漂移）
     - $T^2_{e,f}$：残差方差比 $\hat s_e^2/\hat s_f^2-1$（检测 $\sigma^2$ 漂移）
     - 两者做 Bonferroni 合并 → **decoupled test**
     - $T^3_{e,f}$（类 Chow 统计量）同时检测两者 → **combined test**（作者证明其对 $\beta$ 漂移的检测率比 decoupled 慢，见下）
     - $T^4,T^5$：仅用残差均值/方差漂移，不做回归，计算更快但检测力更弱（无法检测某些结构变化，见 Example 2.6 的反例）
   - **光滑类**：GAM 平滑残差对 time 回归，检测渐变漂移；或 HSIC(残差, time) 检测任意形式的依赖漂移。
4. **多重比较的组合方式**：把所有 pairwise $(e,f)$ 的统计量用 $\max$ 或 $\sum$ 聚合（Eq 11），聚合窗口 $\mathcal F^1$（不相交环境两两比）或 $\mathcal F^2$（每环境 vs 补集）。
5. **时序扩展（本文核心卖点，Section 6）**：把每个候选集 $S$ 的回归自变量从 $X_t^S$ 换成 $Z_t^{S,p}=(X_t^S, Y_{t-1},X_{t-1},\dots,Y_{t-p},X_{t-p})$，即把过去 $p$ 阶滞后作为"永远纳入"的控制变量，只对 $S$（瞬时效应）做假设检验。这样可以在 SVAR 框架下检测**瞬时**（instantaneous）因果效应，是 Granger causality 做不到的（Granger 天然排除同期效应）。滞后阶数 $p$ 未知时給三种处理方案：取足够大的固定 $p$（损失功效）、用能让 $\hat S$ 最大的 $p$（数据驱动但有 post-selection 问题）、或对多个候选 $p\in L$ 取并集并做 Bonferroni（$|L|$ 倍降低 $\alpha$，规避 post-selection 但更保守）。
6. **输出**：$\hat S=\bigcap_{S:\,H_{0,S}\text{ 不被拒绝}} S$，覆盖性质 $\P(\hat S\subseteq S^*)\ge 1-\alpha$ 精确成立（非渐近，因为用了精确重采样，Prop. 1）。
7. **实现**：R 包 `seqICP`（CRAN），子程序见 Algorithm 1（分块比较流程）。

### 关键技术细节

- **精确 vs 渐近**：Prop. 1（无滞后）和 Prop. 2（含滞后 $p$）均证明检验在**任意有限样本**下精确达到水平 $\alpha$（只要 Gaussian 假设成立），不是渐近逼近——这是相对 Peters2016 原始 ICP 的一个技术贡献（他们的检验在多数情形下也是精确的，这里是把该精确性从"已知环境"场景搬到"change-point 隐藏"场景）。
- **检测率理论**（Section 4，全部基于渐近 change-point 模型，n 个观测中 change point 位置按比例 $\alpha_i^*\in(0,1)$ 增长）：
  - decoupled test 检测 $\beta$ 漂移的 minimax 速率是 $n^{-1/2}$（up to $\log n$ 因子）
  - combined test（Chow-like 单检验）检测 $\beta$ 漂移的速率只有 $n^{-1/4}$ ——**反直觉的核心发现**：同时检验两个量（均值/方差）的单一统计量，比分别检验再 Bonferroni 合并的组合，在检测均值漂移上明显更慢。两者检测方差漂移的速率都是 $n^{-1/2}$。
  - 这个"combined 检验力弱于 decoupled"的结论在有限样本模拟中也复现（Appendix A.1, Fig. chowbon）。
- **环境网格选择与计算复杂度权衡**（Section 5.1-5.2）：环境数 $|\mathcal E|=O(\log n)$ 时可同时满足一致性条件 (C1)-(C3) 且总复杂度 $O(Bn\log n)$（低维情形）。环境网格越细（比较越多），检测率里的 $|\mathcal E_n|^{1/k}$ 惩罚项越大，构成 grid 精细度 vs 统计功效的 explicit trade-off。
- **实验设计（Appendix A）**：
  - A.1（exptheo）：$n\in\{100,...\}$，单变化点在 $n/2$，验证 combined 比 decoupled 慢的理论速率。
  - A.2（exppower）：4 变量 SCM（$X^1\to X^2\to Y\to X^3$，$X^1\to Y$），$n\in\{100,...,500\}$，1000 次重复，比较 seqICP（未知 change point）、cp known（已知 change point 网格）、原始 ICP（Peters2016，仅用两个已知环境）——所有方法都随 n 增长识别出真因果集，**已知 change point 略好于未知，但差距不大**；sum 聚合略优于 max 聚合。
  - A.2.1（expsplitting）：反直觉现象——固定两个已知 change point 划分 3 个环境后，**故意把最大的环境再切细**（引入本不存在的额外网格点）反而能显著提升 sum 检验的功效，直到过细导致方差爆炸而性能反转下降。
  - A.3（expshocks）：3 变量 VAR(1)（$X,Y,Z$，Fig. timeseries 给出完整 DAG，含瞬时和滞后边），在随机时间点注入单点 shock，随冲击强度增大功效上升；用 outlier（对 $Y$ 单点设极端值，非真实 SCM 干预）做鲁棒性检查——outlier 越大，方法保持保守（输出空集），不会误报，体现"违反假设时保守而非误导"的性质。
- **Remark B.3（rem:changepoint，附录，对应 idea 主题最相关的一段）**：构造一个 $Y\leftarrow X^1,X^2$ 的线性 SCM，两次干预（$t_1=t_2=200$，即两个变化点重合于同一时刻但分别作用于 $X^1,X^2$ 的不同子区间），使得**只有第二次干预（对 $X^2$ 的干预）在 $Y$ 的边际分布上可见，第一次干预（对 $X^1$）在 $Y$ 上不可见**。此时任何"先在 $Y$ 上做 change-point 检测、再套用标准 ICP"的两阶段流程必然失败（因为 $Y$ 上只能分出 2 个环境），而作者的细粒度网格方法能利用第一段数据里 $X^1$ 的分布差异，把 $\{X^1\}$ 的 $p$-value 从 0.554（不显著，无法拒绝）压到 0.039（显著拒绝）。**这里的因果方向是"用得上的信息越多（细粒度时间网格）→ 检验力越强"，与本课题要研究的"条件化越多→功效跌到 chance"方向相反**。
- **真实数据实验（monetary policy, Section 7.2）**：SNB（瑞士央行）1999-2017 月度数据，$Y$=欧元兑瑞郎汇率对数收益率，$X^1$-$X^6$ 为 SNB 资产负债表相关变量，$X^7$-$X^9$ 为瑞士/欧元区宏观变量（GDP、CPI）。用滞后 $p\in\{1,...,9\}$ 跑 5 种因果检验（variance/decoupled/combined/smooth-variance/HSIC）+ 1 种非因果基线（**全模型**：同时纳入全部瞬时+全部滞后变量做标准 t-检验，即"最大化条件集"而非"边际/存在性检验"）。结果：variance/decoupled/smooth-variance 在 4 滞后左右显著拒绝 $X^2,X^7$ 的非因果性，combined 和 HSIC 检测力不足；**所有基于不变性的检验的 p-value 都随滞后数增加而单调上升**（因为更多滞后=更多待估参数=功效被稀释），而"全模型"非因果 t-检验基线的 p-value 不随滞后增加而系统性上升。

### 核心结果

| 实验 | 关键数字/发现 |
|---|---|
| 检测率理论 | decoupled 检测 $\beta$ 漂移速率 $O(n^{-1/2})$；combined 只有 $O(n^{-1/4})$ |
| 模拟功效对比 | seqICP（未知 change point）funcpower 略低于已知 change point 版本，但差距小；均远优于原始两环境 ICP |
| 环境切分实验 | 故意切细最大环境 → sum 检验功效可从约 20% 提升到接近 100%（Fig. powersim2），随后过细会反转下降 |
| Shock 实验 | 单点冲击强度增大 → 拒绝空集/检出 $\{Z\}$ 的比例上升；outlier 增大 → 方法保守，输出空集比例上升（不误报） |
| 真实数据 | $X^2$（外币投资占比）、$X^7$（瑞士 GDP）在 4 阶滞后附近被 3/5 种因果检验拒绝非因果性；combined/HSIC 无功效；因果检验 p-value 随滞后数单调上升，非因果全模型基线不呈现此现象 |
| Remark B.3 反例 | 粗粒度两阶段流程：$p=0.554$（不显著）；细粒度网格方法：$p=0.039$（显著）——**细粒度条件化win，而非输** |

### 明确局限

- 作者自陈局限：(1) 仅覆盖线性高斯模型，非线性/非高斯只给出启发式的置换检验方案（Remark B.1），且明确说"turns out to be rather difficult to get precise results about the asymptotic level"；(2) 检验对模型误设保守但不给出诊断信息，只输出空集；(3) 真实数据例子被作者自己称为"only an illustration"，样本量、变量选择均为示范性，未做严格 benchmark 对比；(4) 无标准公开 benchmark（不是在标准因果发现 benchmark 数据集上评测，而是自建模拟 + 一个真实经济数据集）。
- 读出的局限（与本课题相关）：
  - **完全没有真机/物理动力系统实验**——所有时序例子都是线性 SVAR 玩具模型或宏观经济月度数据，没有涉及本课题关心的"fragility regime"这类非线性/混沌/多尺度动力系统场景。
  - **没有 PCMCI+ 或任何 PC 家族条件独立图算法的直接对比**——本文的"因果 vs 非因果"对比轴是"不变性检验 vs 全模型 t-检验"，而不是"条件独立检验（图算法）vs 边际检验（存在性阈值）"。这是完全不同的方法论谱系，无法直接迁移其结论到 PCMCI+ 语境。
  - **样本量都不大**（$n\le 500$ 模拟，真实数据约 217 个月度点），且理论速率虽是 $n^{-1/2}$ 但常数依赖 $|\mathcal E_n|^{1/k}$ 和矩条件 $k$，论文没有给出与"conditioning set 维度 $d$"直接相关的速率下界讨论（即没有回答"条件集越大，检测率如何随 d 退化"这个我们更关心的问题）。

### 跟我们的关系

**结论先行：这是一篇与 causal-scs-indicator 课题（PCMCI+ 条件化在 fragility regime 判别力落 chance、而非因果 marginal 基线通过存在性门槛）方法论谱系完全不同的论文，未能提供所需的"时序/动力系统原生理论刻画"，本轮属于重复确认（甚至连重复确认都谈不上——是切题外围），无实质新增量。**

具体展开回应 dispatcher 提出的两个问题：

**(a) 是否提供了"因果条件化系统性弱于非因果预测"在时序/动力系统场景下的直接理论刻画？** 否。本文的核心方法（seqICP）不是条件独立图检验算法，而是"不变性假设检验"：对每个候选因果集 $S$ 单独做 change-point resampling 检验，输出所有不被拒绝的 $S$ 的交集。它与 PCMCI+ 的"conditioning on separating sets 做条件独立检验"完全是两套范式。本文中唯一的"因果 vs 非因果"对比，是"不变性检验（causal）vs 全模型 t-检验（non-causal，且是**更大**的条件集，包含全部瞬时+滞后变量）"——这与本课题"marginal existence 检验 vs conditional 检验"的对比方向是**反的**：本文的非因果基线是最大化条件化，而不是不做条件化。并且，本文里唯一体现"更多条件化 → 功效下降"的证据（滞后数增加导致 p-value 单调上升）是标准的 nuisance-parameter/multiple-testing dilution 效应，这已经是我们 7 种机制里大概率覆盖过的"条件集维度增长稀释功效"机制的一个变种，不构成动力系统原生的新增量。更值得注意的是，本文附录 Remark B.3 展示的方向恰恰相反：**更细粒度地利用时间结构信息（更多"条件化"在环境网格意义上）反而把 p-value 从 0.554 压到 0.039**，即精细条件化提升而非损害检测力——这与本课题的核心现象在方向上不一致，若引用需非常谨慎地做范围限定。

**(b) 是否只是重复确认？** 是，且强度上是最弱的一种——本文甚至没有直接实例化目标现象，只是在"lag 数增加导致检验力下降"这一点上有表面的类比空间，而这个空间本身也是已知机制（参数维度/多重检验代价）的又一次具体化，不是 fragility regime/PCMCI+ 场景的原生新证据。

**可用的工具/方法**：
- `seqICP` R 包提供了严格的（非渐近）不变性检验实现，如果 causal-scs-indicator 未来需要"用非渐近精确检验代替 PCMCI+ 的渐近 CI 检验"这个角度，这里有现成的精确重采样检验设计模板（尤其是 decoupled test 比 combined test 快的证明思路，对应"分别检验多个属性再合并 vs 单一联合统计量"的一般性教训，在设计新指标时可以借鉴）。
- Remark B.3 的反例构造（两个变化点，一个在 $Y$ 边际不可见）是一个很好的"为什么边际/粗粒度方法会漏检"的教学案例，可以反过来在我们的叙事里当作"反向对照"：这里是"粗粒度 ICP 漏检、细粒度条件化才能救回来"，正好可以对比出 PCMCI+ 现象（细粒度条件化反而失败）的反常性/独特性——**用作对比案例的价值大于作为同源支持证据的价值**。

**差异化**：causal-scs-indicator 关注的是 PCMCI+ 类图算法在 fragility regime 下条件化过度导致条件独立检验退化到 chance，这是"conditioning set 内某个变量的信息几乎被完全屏蔽/或有效样本量崩溃"式的退化；而 seqICP 的功效退化机制是标准的"参数越多、检验统计量方差越大、多重检验校正越保守"，是统计学教科书级别的效应，不需要、也不特别涉及"fragility"这个物理/动力系统概念。二者不撞车，但也谈不上互相支持——引用时应明确说明这是"时序因果推断的另一分支（不变性检验），非 PCMCI+ 同类方法"，避免读者误以为这是同一现象在时序场景的验证。

**结论：重复确认，无实质新增量。** 本论文（seqICP）是一个与 PCMCI+ 完全不同方法论谱系（不变性假设检验 vs 条件独立图检验）的时序因果推断框架，其"因果 vs 非因果"对比轴与本课题关注的"conditional 检验退化到 chance vs marginal 检验通过存在性门槛"方向不一致（本文非因果基线是**更大**的条件集，而非 marginal）。本文中唯一可类比的功效退化现象（滞后数增加→p-value 单调上升）是标准的参数维度/多重检验稀释效应，已被已有 7 种机制覆盖，非动力系统原生新证据。更重要的是，本文附录 Remark B.3 的反例展示的是**细粒度条件化提升检测力**（p: 0.554→0.039），与本课题现象方向相反，若要引用需谨慎限定范围、避免误用为支持证据。
