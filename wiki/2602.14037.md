# Affine Rank Minimization is ER Complete (2602.14037)

- **Year**: 2026.02
- **Authors**: Angshul Majumdar, Indraprastha Institute of Information Technology Delhi
- **Venue**: arXiv preprint; manuscript template lists `Signal Processing`
- **Keywords**: affine rank minimization, fixed-rank feasibility, ETR, exists-R completeness, rank-constrained matrix completion, arithmetic circuits, determinant forcing, gauge fixing
- **One-liner**: 固定秩 affine rank feasibility 被声称为 exists-R 完全。

## 精华提炼

### 管线/方法
- 论文研究的决策问题是 ARM(k)：给定有理矩阵 `A_1,...,A_q` 和有理标量 `b_1,...,b_q`，问是否存在实矩阵 `X` 满足所有线性等式 `<A_l, X>=b_l` 且 `rank(X)<=k`。
- 目标不是给出算法，而是给出复杂性分类：固定常数秩约束下，纯 affine 输入是否已经表达出 ETR 的全部难度。
- membership 方向有两种写法：一是用 rank factorization `X=UV` 把 `rank(X)<=k` 写成二次多项式存在量词；二是对 `k=3` 用所有 `4x4` minors vanish 的 determinantal encoding。
- hardness 方向声称从 ETR 归约到 ARM(3)。
- 归约第一步把 ETR 公式整理为 polynomial equality form，再进一步整理为 arithmetic circuit gate-equality form。
- circuit gate 包括 input、constant、addition、negation、multiplication；inequality/disequality 通过 square slack 和 inverse gadget 转为 equality。
- 每个 circuit quantity 被分配到矩阵 `X` 的一个 designated entry，也就是 carrier `X_{r_i,c_i}`。
- 常数、copy、fan-out、addition、negation 都直接用 affine equality enforce，例如 `X_z-X_x-X_y=0`。
- 唯一非线性 gate 是 multiplication `z=xy`。
- multiplication 被声称由 rank-forcing gadget enforce，而不是显式加入 polynomial/minor constraints。
- gadget 的核心形式是固定一个 `2x2` block 为 `[[1, x], [y, z]]`。
- 论文声称在全局 `rank(X)<=3` 和固定 `3x3` gauge block 的帮助下，若 `z != xy`，就会诱导一个 nonzero `4x4` determinant，从而违反 rank bound。
- gauge block 设为某个 full-rank `3x3` submatrix `X[I_0,J_0]=B`，通常可取 `B=I_3`。
- 论文的意图是用 gauge block 消除 `X=UV` factorization 中的 `GL(3)` ambiguity，使不同 gadget 的 local factorization 可以组合成一个 global factorization。
- 每个 gadget 被分配 fresh auxiliary rows/columns，避免不同 gadget 直接竞争同一行列。
- fan-out 采用 occurrence-based carrier：一个 wire 的不同使用位置各有 carrier，再用 copy constraint 显式相等。
- soundness 方向：若 ARM(3) instance feasible，则从 `X` 的 designated entries 解码 circuit values；每个 gate gadget 保证 gate semantics；因此原 ETR/circuit satisfiable。
- completeness 方向：若 circuit 有 satisfying assignment，则给每个 gadget 构造 local rank-3 completion；再通过 global embedding lemma 拼成一个矩阵 `X=UV`。
- size accounting 声称每个 gate 只引入 `O(1)` rows/columns 和 `O(1)` affine constraints，总约束数和 bit length 都是 polynomial，甚至线性于 circuit size。
- 最终 claim 是 ARM(3) exists-R-hard，结合 membership 得到 ARM(3) exists-R-complete。
- 论文还声称固定 rank 3 是 determinant-forcing mechanism 的最小 rank，因为 rank 2 下所有 `3x3` minors 已经 vanish，无法用这种 nonzero minor obstruction。

### 关键技术细节
- ARM(k) 输入系数全部是 rational，bit model 按 numerator/denominator 总长度计。
- membership 的 factorization witness：若 `rank(X)<=k`，存在 `U in R^{m x k}`、`V in R^{k x n}` 使 `X=UV`。
- 每个 affine constraint `<A_l,X>=b_l` 代入 `X=UV` 后变成 degree-2 polynomial equality。
- 对 `k=3` 的 determinantal membership：所有 `4x4` minors 为零；minor 数为 `binom(m,4) binom(n,4)`，每个 determinant 展开有 `4! = 24` 项。
- ETR inequality encoding：`p>=0` 用 `p-u^2=0`；`p>0` 用 `p-u^2=0` 且 `uv-1=0`；`p!=0` 用 `pv-1=0`。
- arithmetic circuit exactness lemma：gate variables `z_g` 满足 GateEq 且 output 为 0 iff circuit output polynomial 为 0。
- affine entry constraint：`X_{r,c}=a` 对应一个 elementary matrix inner product；`X_{r,c}=X_{r',c'}` 对应两个 elementary entries 的差。
- addition gadget 只有一个 affine constraint：`X_z-X_x-X_y=0`。
- negation gadget 只有一个 affine constraint：`X_z+X_x=0`。
- multiplication gadget 的 claimed local pattern 是 `X[{r1,r2},{c1,c2}] = [[1,x],[y,z]]`。
- determinant obstruction 的 claimed formula 是 `det(Y)=det(B_sub)*(z-xy)`。
- proof 里要求 gauge block 至少提供一个 nonzero `2x2` subminor，且 gadget block 与 gauge block 组成近似 block diagonal `4x4` submatrix。
- local-to-global composition 依赖 fresh auxiliary index discipline：除 gauge block 和显式 copy carriers 外，gadget 行列 disjoint。
- bit complexity：除 fixed constants 和 input circuit constants 外，gadget coefficients 都在 `{0, +/-1}`。
- 论文没有实证实验、没有代码、没有生成 benchmark；这是纯理论 reduction paper。

### 核心结果
- Main claim 1：对每个固定 `k>=1`，ARM(k) 在 exists-R 中。
- Main claim 2：ARM(3) exists-R-hard。
- Main claim 3：因此 ARM(3) exists-R-complete。
- Main claim 4：hardness instance 只含 affine equalities 和单个全局 `rank(X)<=3`，不含显式 determinant/minor constraints。
- Main claim 5：multiplication gate 可以通过 constant-size rank-forcing gadget 表达。
- Main claim 6：整个 reduction 的 dimensions、constraints 数、rational encoding length 都 polynomial bounded。
- surprising 点：如果成立，固定到 3 的 rank bound 加上线性等式就已经能表达全部 real algebraic feasibility。
- 对低秩优化的含义：nuclear norm 和 factorization heuristics 的 worst-case barrier 不是离散 NP-hardness，而是 continuous/semi-algebraic hardness。
- 对 matrix completion 的含义：即使 observation model 是 affine，rank constraint 本身也可能携带全部非线性表达力。
- 对算法的含义：通用 exact feasibility solver 不应期待 polynomial-time，除非复杂性层级发生重大塌缩。

### 明确局限
- 这篇论文没有实验，也不需要实验；可信度完全取决于 reduction proof 的严谨性。
- 论文没有提供 formal proof assistant artifact、reference implementation 或 machine-checkable reduction generator。
- 最关键的 determinant-forcing proof 看起来存在严重缺口。
- 问题一：固定 `X[I0,J0]=B` 只固定了 product submatrix，并不会消除 `X=UV` 的 `GL(3)` gauge；对任意 invertible `G`，`(UG)(G^{-1}V)` 仍给出同一个 `X` 和同一个 pinned block。
- 论文中 gauge-fixing lemma 从 `U[I0,:]V[:,J0]=U[I0,:]G(G^{-1}V)[:,J0]` 推出 `G=I`，这个推理不成立，因为等式对任意 `G` 都成立。
- 问题二：rank obstruction lemma 假设能取到一个 block diagonal 形状的 `4x4` submatrix，其 off-diagonal blocks 为 0。
- 但 construction 只明确固定了 gauge block 和 gadget 的 `2x2` block，并没有看到 affine constraints 强制 gauge rows/gadget columns、gadget rows/gauge columns 的 off-diagonal entries 为 0。
- 如果这些 off-diagonal entries 是自由的，`2x2` block 的 determinant `z-xy` 不必单独造成 rank 4 obstruction。
- 问题三：`2x2` block `[[1,x],[y,z]]` 自身 rank 为 2 时并不违反 `rank(X)<=3`；必须依赖与 gauge block 的特殊耦合，但耦合条件没有被充分建立。
- 问题四：completeness proof 中把 gadget rows 写成 gauge rows 的 linear combinations，但没有严谨说明所有被约束 entries 和 carrier entries 在同一个 global factorization 中如何同时满足。
- 问题五：文中多处 lemma reference/notation 有明显错位，例如引用 Definition 当 Lemma、section reference 指向不匹配，降低可审计性。
- 问题六：论文重复证明了若干 membership/hardness statement，结构上像不同草稿拼接，增加了隐藏不一致的风险。
- 作者自己的局限披露较少；主要局限由审读发现，而不是论文主动承认。
- 因此，结论目前应被视为“有趣但需要独立复核的 claim”，不能直接当作可靠 theorem 使用。
