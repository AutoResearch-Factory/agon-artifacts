# On the Convergence of Single-Timescale Actor-Critic (2410.08868)

- **Year**: 2024.10
- **Authors**: Navdeep Kumar (Technion), Priyank Agrawal (Columbia University), Giorgia Ramponi (University of Zurich), Kfir Yehuda Levy (Technion), Shie Mannor (Technion)
- **Venue**: arXiv
- **Keywords**: single-timescale actor-critic, policy gradient, discounted MDP, softmax policy, global convergence, Lyapunov recursion, ODE tracking, sample complexity
- **One-liner**: 证明 vanilla single-timescale AC 以 O(eps^-3) 达全局最优。

## 精华提炼

### 管线/方法
- 论文研究的是有限状态、有限动作、无限时域 discounted MDP。
- 策略使用 tabular softmax 参数化：每个 state-action 有一个参数 theta(s,a)。
- 目标是最大化 discounted return J^pi，并分析期望意义下的全局 sub-optimality J* - E[J^pi_k]。
- 算法是 single-timescale actor-critic：actor 和 critic 每轮都用类似量级的步长更新。
- 每轮采样 s ~ d^{pi_theta_k}，a ~ pi_theta_k(.|s)，再采样 s' ~ P(.|s,a)，a' ~ pi_theta_k(.|s')。
- actor 用当前 critic 的 advantage 更新单个被采样坐标：
- theta_{k+1}(s,a) = theta_k(s,a) + eta_k (1-gamma)^-1 A(s,a)。
- critic 用 TD 形式更新 Q(s,a)：
- Q(s,a) <- Q(s,a) + beta_k [R(s,a) + gamma Q(s',a') - Q(s,a)]。
- 论文核心不是提出新算法，而是给这个 vanilla AC 一个更强的全局收敛分析。
- 关键变量有三个：a_k = E[J* - J^k] 表示 actor sub-optimality。
- z_k = sqrt(E ||Q_k - Q^k||^2) 表示 critic tracking error。
- y_k = sqrt(E ||nabla J^k||^2) 表示 policy gradient norm。
- 作者先分别推 actor recursion 和 critic recursion。
- actor recursion 说明 sub-optimality 的下降受三项影响：真实梯度下降项、critic 偏差项、actor 随机更新方差项。
- critic recursion 说明 Q tracking error 有 contraction 项，也有 TD 方差、policy moving target 和 actor 梯度耦合项。
- 单独看 a_k 或 z_k 都不单调，直接 telescoping 很难给全局 rate。
- 关键转折是构造 Lyapunov 量 x_k = a_k + z_k^2。
- 通过选择 beta_k = c_beta eta_k，把 actor 与 critic 两个递推合并。
- 合并后得到形如 x_{k+1} <= x_k - c eta_k x_k^2 + C eta_k^2 的递推。
- 这里用到了 gradient domination lemma，把 sub-optimality 与 policy gradient norm 连接起来。
- 论文再用 ODE tracking 解这个非线性递推。
- ODE 直觉是选 eta_k 与 u_k^2 同阶，把 du/dk = -c u^4 的轨迹作为上界。
- 该 ODE 的解给出 u_k = O(k^-1/3)。
- 因为 a_k <= x_k，最终得到 J* - E[J^pi_k] = O(k^-1/3)。
- 因此达到 eps-global-optimal policy 的样本复杂度是 O(eps^-3)。
- 推荐步长是 eta_k, beta_k = O(k^-2/3)，而不是常见的 k^-1/2。
- 数值实验只用于说明不同步长下的趋势，主体贡献是理论证明。

### 关键技术细节
- MDP 假设：finite state space S、finite action space A、discount factor gamma in [0,1)。
- reward 假设：||R||_infty <= 1。
- policy 参数化：softmax pi_theta(a|s) = exp(theta(s,a)) / sum_a exp(theta(s,a))。
- policy gradient 公式：partial J / partial theta(s,a) = (1-gamma)^-1 d^pi(s) pi(a|s) A^pi(s,a)。
- smoothness 常数：J^pi_theta 对 theta 的 smoothness L = 8 / (1-gamma)^3。
- global guarantee 依赖 gradient domination lemma。
- GDL 常数包括 C_PL = max_k ||d^{pi*} / d^{pi_theta_k}||_infty。
- GDL 还需要 c = min_k min_s pi_theta_k(a*(s)|s) > 0。
- 这意味着初始分布需要覆盖状态，softmax policy 也不能让最优动作概率退化到 0。
- critic 的核心假设是 sufficient exploration assumption。
- 该假设要求 Bellman/Q 评估方向与 Q error 有正内积下界 lambda ||Q^pi - Q||^2。
- 作者指出这类探索假设是 single/two-timescale AC 理论中常见前提。
- 固定 policy 下，critic Q update 在该假设下以 alpha = sqrt(1 - lambda^2/2) 收敛。
- actor recursion 的形式是 a_{k+1} <= a_k - c1 eta_k y_k^2 + c2 eta_k y_k z_k + c3 eta_k^2。
- appendix 中更具体地给出 a_{k+1} <= a_k - eta_k/(1-gamma) y_k^2 + 2 eta_k/(1-gamma) y_k z_k + 4L eta_k^2/(1-gamma)^4。
- critic recursion 的形式是 z_{k+1}^2 <= (1-c4 beta_k)z_k^2 + c5 beta_k^2 + c6 eta_k^2 + c7 eta_k y_k z_k。
- appendix 中还保留 eta_k^2 z_k 的平滑项，并用 z_k <= c_z 吸收。
- 关键耦合项是 y_k z_k：actor 受 critic error 拖累，critic 又受 changing policy 拖累。
- 传统局部收敛分析会把 y_k^2 放到 telescoping sum，得到 stationary policy rate。
- 本文利用 GDL 直接对 global sub-optimality a_k 做控制。
- Lyapunov x_k = a_k + z_k^2 是本文 claim 的核心对象。
- beta_k / eta_k 需要取特定比例 c_beta，足以用平方项吸收 y_k z_k cross term。
- ODE tracking lemma 给出了一个显式上界：x_k 被 O(k^-1/3) 轨迹控制。
- 该上界对应 eta_k = O(k^-2/3)。
- 结果中 actor sub-optimality rate 是 O(k^-1/3)。
- critic error z_k rate 是 O(k^-1/6)，反而慢于一些 local-convergence 分析里的 O(k^-1/4)。
- 作者强调这是一个 tradeoff：为了更快的 global actor convergence，critic 方差下降更慢。
- 数值实验使用随机生成 transition kernel 和 reward 的 MDP。
- 实验包含 state/action 规模如 50/5、5/2、20/5。
- 图中比较 eta_k = beta_k = k^-a 或 10 eta_k = beta_k = k^-a 的不同 a。
- 实验观察是 k^-2/3 最终表现最好，慢衰减步长前期较好但后期被超过。
- 实验采样为了简单使用 uniform samples，而不是理论假设里的 discounted occupation measure。
- 代码位置在文中给的是 anonymous 4open science 链接，未在 tex 中提供正式长期仓库。

### 核心结果
- 主结果：single-timescale AC 在 softmax finite discounted MDP 中达到 J* - E[J^pi_k] <= O(k^-1/3)。
- 等价样本复杂度：达到 eps-global-optimal policy 需要 O(eps^-3) samples。
- 对比 prior local convergence：已有 single-timescale AC 常见结果是 O(eps^-2) 达 eps-stationary。
- 通过 GDL 从 eps-stationary 转 global optimality 会变成 O(eps^-4)。
- 本文绕开“先 local 再 global”的路径，直接分析 global sub-optimality。
- 对比表中列出 Chen and Zhao 2022、Olshevsky and Gharesifard 2022 等 single-timescale AC 分析，多为 local stationarity。
- 两时间尺度或 double-loop AC 可以更容易分析，但算法形式和实践中的 single-loop AC 有差距。
- Gaur et al. 相关 actor-critic variant 也有 O(eps^-3) 新样本量，但需要 replay/buffer 和重复使用样本，计算/内存复杂度更高。
- NAC 类方法在理论上可更快，但需要 Fisher information matrix inversion，不是 vanilla AC。
- 本文 claim 的差异化是“不换算法，只换分析”。
- surprising result 1：best theoretical step size 是 k^-2/3，不是非凸优化里常见的 k^-1/2。
- surprising result 2：actor global rate 提升时，critic MSE rate 可以变慢。
- surprising result 3：作者认为当前 variance-based 分析可能卡在 O(eps^-3)，进一步到 O(eps^-2) 需要转向 critic bias 分析。
- 论文明确提出未来方向：只控制 bias ||Q^k - E Q_k||，可能允许 constant 或慢衰减 critic step size。
- 这点对所有 actor-critic 乃至 bilevel/minimax 两模块耦合算法都有方法论价值。

### 明确局限
- 理论只覆盖 finite state/action discounted MDP。
- 策略必须是 tabular softmax 参数化。
- 采样需要每轮能从 d^{pi_theta_k} occupation measure 抽样。
- 作者说明可通过 geometric termination 链实现，但平均多一个 1/(1-gamma) 复杂度因子。
- 真实 online Markovian sampling 未在主定理中处理。
- sufficient exploration assumption 很强，且是保证 critic 固定 policy 评估收敛的关键。
- GDL 依赖 C_PL 有界和 c > 0；如果初始分布不覆盖状态或最优动作概率退化，该理论可能失效。
- 没有 function approximation、deep policy network 或连续状态动作空间结果。
- 没有真实 RL benchmark 或标准 control task 实验。
- numerical simulations 是随机 MDP 上的趋势图，且采样方式和理论假设不完全一致。
- 代码仅通过匿名链接声称可用，长期可复现性一般。
- appendix 中有一些排版/推导层面的疑点需要二次核查，例如部分常数定义和证明文字有明显笔误。
- c_l 的定义依赖 a_k 的写法看起来不够干净，实际应理解为由 GDL 常数和 critic bound 诱导的下界常数，需要复核原证明。
- 论文不提供 lower bound，k^-2/3 的“最优”更多是作者基于当前递推结构的判断。
- 论文没有独立验证，也没有和现有实现做工程级 benchmark。
- 对 FEX 来说，最大局限是它的 MDP/AC setting 与 expression-tree symbolic regression controller 差距很大。
- FEX 通常涉及树生成、离散算子 grammar、连续系数优化、PDE residual reward、可能的 REINFORCE/risk-seeking objective。
- 本文没有处理表达式等价类、无效表达式、参数内层优化噪声、PDE collocation residual 或 symbolic simplification。
- 因此不能直接拿来证明 FEX controller 收敛，只能作为理论模板。

### 跟我们的关系
- 对 `0616-fex`，本文不与 FEX 的 PDE symbolic regression、expression tree search、LLM skeleton 或 dimension-lift claims 直接撞车。
- 它研究的是 vanilla actor-critic 在有限 MDP 中的 global convergence，不是符号回归，也不是 PDE solver。
- 但它正好对应 FEX landscape 中的理论缺口：“FEX 有 approximation theory，但 RL controller 的收敛率/采样复杂度缺失”。
- 可用点 1：把 FEX tree generation 看成 finite-horizon/episodic MDP 后，本文的 actor-critic coupling 分析可作为起点。

- 可用点 3：ODE tracking 解耦合递推的方法可复用到 FEX 的 policy-gradient controller 或 learned evaluator。
- 可用点 4：k^-2/3 step-size 结论可作为 FEX controller 训练 schedule 的理论候选，但需要实验验证。
- 可用点 5：作者关于 bias-vs-variance 的讨论对 FEX 很重要，因为 FEX 的 reward/evaluator 方差可能来自 collocation sampling 和 inner optimizer instability。
- 该引不该引：如果写 FEX 搜索复杂度或 RL controller 理论，该引；如果只写 PDE symbolic solver 实验，放 related work 边角即可。
- 差异化：FEX 的关键理论难点不是 standard AC 的 Q tracking，而是表达式 grammar、continuous coefficient fitting、PDE residual landscape、等价表达式和维度灾难。
- 撞车风险：低。它不解决 FEX 的 SR/PDE 任务，也没有提供 expression-search-specific 收敛结论。
- 机会：可以把本文作为“现有 RL 理论能处理 simple AC，但离 FEX controller 仍有显著 gap”的证据。
- 对 claim 级 novelty：若我们的 claim 是“首次证明 FEX-style expression-tree RL search 的 sample complexity”，本文是近邻但不杀 novelty。
- 对 method claim：若我们仅使用 vanilla actor-critic + finite MDP + softmax + GDL，则会与本文方法高度重叠；必须引入 FEX-specific structure 才有差异。
- 对 experimental setting：本文没有 PDE、SRBench、LLM-SRBench、FEX benchmark 或高维 Poisson/Schrodinger 实验。
- 对可靠性使用：可以引用它的 O(eps^-3) single-timescale AC rate，但不要把它说成适用于 general symbolic regression controller。
